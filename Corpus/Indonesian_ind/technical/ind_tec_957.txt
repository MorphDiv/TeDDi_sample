Teks Bantuan File Browser

 File browser digunakan untuk membrowse struktur direktori untuk memilih file untuk dibaca atau ditulis. Anda bisa menggunakan tombol panah atau Page Up/Down untuk membrowse melalui file-file, dan S atau Enter untuk memilih file atau memasuki direktori terpilih. Untuk berpindah satu level ke atas pilih direktori bernama ".." yang ada di bagian atas daftar file.
