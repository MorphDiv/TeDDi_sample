Tombol tetikus yang digunakan untuk menangkap pointer
Tombol tetikus ini harus ditekan saat pointer melewati area penangkapan, untuk menangkap pointer. Nilai valid adalah "0" jika tombol tidak diperlukan, "1" untuk tombol kiri, "2" untuk tombol tengah, dan "3" untuk tombol kanan.
Tombol tetikus yang digunakan untuk melepaskan pointer
Tombol tetikus ini harus ditekan untuk melepaskan pointer dari area penangkapan. Nilai valid adalah "1" untuk tombol kiri, "2" untuk tombol tengah, dan "3" untuk tombol kanan.
Menangkap pointer hanya jika Shift ditekan
Apakah tombol Shift harus ditekan agar ponter ditangkap saat melewati area ambil. Jika opsi ini ditetapkan sebagai benar, dan tombol tetikus ditetapkan dalam kunci "capture_buttonl", maka tombol Shift harus ditekan saat tombol tetikus ditekan.
Menangkap pointer hanya jika Alt ditekan
Apakah tombol Alt harus ditekan agar ponter ditangkap saat melewati area ambil. Jika opsi ini ditetapkan sebagai benar, dan tombol tetikus ditetapkan dalam kunci "capture_button", maka tombol Alt harus ditekan saat tombol tetikus ditekan.
Menangkap pointer hanya jika Ctrl ditekan
Apakah tombol Ctrl harus ditekan agar ponter ditangkap saat melewati area ambil. Jika opsi ini ditetapkan sebagai benar, dan tombol tetikus ditetapkan dalam kunci "capture_button", maka tombol Ctrl harus ditekan saat tombol tetikus ditekan.
Melepaskan pointer hanya jika Shift ditekan
Apakah tombol Shift harus ditekan dengan tombol tetikus ditetapkan dalam "release_button" agar pointer dilepaskan dari area penangkapan.
Melepaskan pointer hanya jika Alt ditekan
Apakah tombol Alt harus ditekan dengan tombol tetikus ditetapkan dalam "release_tombol" agar pointer dilepaskan dari area penangkapan.
Melepaskan pointer hanya jika Ctrl ditekan
Apakah tombol Ctrl harus ditekan dengan tombol tetikus ditetapkan dalam "release_button" agar pointer dilepaskan dari area penangkapan.
