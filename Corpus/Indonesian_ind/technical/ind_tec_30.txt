Kuki Kuki berisi informasi yang Konqueror (atau aplikasi KDE lain yang menggunakan protokol HTTP) simpan di komputer anda, dijalankan oleh server Internet jarak jauh. Ini berarti bahwa server web dapat menyimpan informasi tentang anda dan aktivitas perambanan anda di mesin anda untuk digunakan nanti. Anda mungkin menganggap ini adalah bentuk pelanggaran privasi. Tapi, kuki berguna dalam situasi tertentu. Misalnya, kuki sering digunakan oleh toko- toko Internet, sehingga anda dapat 'menaruh barang di keranjang belanja'. Beberapa situs mensyaratkan anda menggunakan peramban yang mendukung kuki. Karena banyak orang ingin berkompromi antara privasi dan keuntungan yang kuki tawarkan, KDE menawarkan kepada anda kemampuan untuk menyesuaikan bagaimana KDE menangani kuki. Jadi anda mungkin ingin mengatur kebijakan standar KDE untuk menanyakan kepada anda saat server ingin mengatur kuki, mengizinkan anda untuk memutuskan. Untuk situs web belanja favorit anda yang anda percayai, anda mungkin ingin mengatur kebijakan untuk menerima, sehingga anda dapat mengakses situs web tanpa ditanyai setiap kali KDE menerima kuki.

Proxy Server proxy adalah program antara yang terletak di antara mesin anda dan Internet yang menyediakan layanan seperti tembolok laman dan/ atau penyaringan. Tembolok server proxy memberikan anda akses yang lebih cepat ke situs yang telah anda kunjungi dengan menyimpannya secara lokal atau tembolok isi dari halaman tersebut; penyaringan server proxy, sebaliknya, menyediakan kemampuan untuk menangkal permintaan terhadap iklan, spam, atau apapun lainnya yang ingin anda halangi. Catatan: Beberapa server proxy menyediakan kedua layanan tersebut.

Dalam keadaan apapun, alamat broadcast (antarmuka di smb. conf) harus diatur jika diterka secara salah atau anda mempunyai banyak kartu. Server WINS biasanya meningkatkan performa, dan mengurangi beban jaringan secara signifikan. Pengikat digunakan untuk menugaskan pengguna standar untuk server yang diberikan, kemungkinan dengan menyesuaikan sandi, atau untuk mengakses kongsi spesifik. Jika anda memilih, pengikat baru akan dibuat untuk log masuk dan kongsi yang diakses selama meramban. Anda dapat menyunting semuanya dari sini. Sandi akan disimpan secara lokal, dan diacak sehingga membuatnya tidak terbaca bagi mata manusia. Demi alasan keamanan, anda mungkin tidak ingin melakukan hal tersebut, karena entri dengan sandi jelas mengindikasikan seperti itu.

Identifikasi Peramban Modul identifikasi peramban memungkinkan anda untuk memiliki kontrol penuh terhadap bagaimana Konqueror akan mengidentifikasi dirinya sendiri ke situs web yang anda ramban. Kemampuan untuk memberikan identifikasi palsu diperlukan karena beberapa situs web tidak ditampilkan secara benar jika mereka mendeteksi bahwa mereka tengah berhubungan dengan versi terkini atau versi apapun dari Netscape Navigator atau Internet Explorer, bahkan jika peramban sebenarnya mendukung semua fitur yang diperlukan untuk render halaman tersebut dengan benar. Untuk situs seperti itu, anda dapat menggunakan fitur ini untuk coba meramban ke situs tersebut. Tolong pahami bahwa hal ini mungkin tidak selamanya dapat bekerja, karena situs seperti itu mungkin menggunakan protokol web atau spesifikasi non- standar. CATATAN: Untuk mendapatkan bantuan spesifik tentang bagian tertentu dari kotak dialog, cukup klik di tombol bantuan cepat di batang judul jendela, kemudian klik di bagian yang ingin anda cari bantuannya.
