CREATE TABLE servers (name string, location string);
我们假设表里含有下列信息：
&gt; select * from servers ;
name | location
---------+---------
moon | room1 
darkstar | room2 
flag | room3 
alf | room2 
vador | room3 
(5 rows)
&gt;

$ cat problems.csv 
Server name,Error code,Error description

vador,1,过期的AV签名
darkstar,2,过载
flag,0,文件系统已满
