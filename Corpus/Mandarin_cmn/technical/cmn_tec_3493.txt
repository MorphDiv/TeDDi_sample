终端文字的默认颜色
终端文字的默认颜色，必须是颜色指定(可以是 HTML 风格的十六进制数码，也可以是像“red”这样的颜色名称)。
终端背景的默认颜色

终端加粗文字的默认颜色
终端加粗文字的默认颜色，必须是颜色指定(可以是 HTML 风格的十六进制数码，也可以是像“red”这样的颜色名称)。如果bold_color_same+as_fg是真，这个选项将被忽略。
粗体字是否应该与普通字的颜色相同
如果是真(true)，粗体字将与普通字显示同样的颜色。
如何对待动态标题
如果终端中的程序设置了标题(最典型的就是用户自己的 shell 可能执行这一操作)，动态设置的标题可以删除配置好的标题，或者在原有标题前/后插入。可供选择的值有“替换”、“之前”、“之后”和“忽略”。
终端标题
终端窗口或标签页显示用的标题。根据 title_mode 设置的不同，此标题可替换为终端内程序所指定的标题，或与之组合。
是否允许粗体字
如果为真(true)，就允许终端中的程序将文字以粗体显示。
是否禁用终端铃声
如果为真(true)，在应用程序发送终端响铃的转码序列时就闷声不响。
被认为是“单词的一部分”的字符
逐词选中文字时，这些字符的序列将被认为是一个单词。范围可以给为“A-Z”。连字符(不表示范围)必须是给出的第一个字符。
是否对新窗口使用自定义终端大小
如果是真(true)，新创建的终端窗口将使用default_size_columns和default_size_rows所指定的自定义大小。
默认列数
新建窗口的列数。如果use_custom_default_size选项没有被启用，这个选项没有效果。
默认行数
新建窗口的行数。如果use_custom_default_size没有被启用，这个选项没有效果。
滚动条的位置
将终端滚动条放在哪里。可供选择的值是“左”、“右”和“隐藏”。
回滚的行数
要保留的回滚行数。您可以往回滚动终端到这么多行；不在回滚之内的行将被丢弃。如果scrollback_unlimited被设置为真(true)，这个值将被忽略。
是否保存无限行的回滚内容
如果是真(true)，回滚的行将不被丢弃。回滚历史记录临时保存在磁盘上，所以如果终端输出的内容很多，这个选项可能导致磁盘空间耗尽。
是否在按键时滚动到底部
如果为真(true)，按键就把滚动条滚动到底部。
是否在有新输出时滚动到底部
如果为真(true)，在终端有新输出时滚动条就会滚动到底部。
如何对待子命令退出时
可供选择的值中，“关闭”可关闭终端，“重新启动”可重新启动命令。
是否以登录 shell 在终端调用命令
如果为真(true)，终端中的命令将会以登录 shell 被调用。(argv[0] 将在前面有一个连字符。)
是否在调用终端命令时更新登录日志
如果为真(true)，将会在终端内的命令调用时更新登录记录 utmp/wtmp。
是否运行自定义命令而不是 Shell
如果为真(true)，将会使用用户自定义的设置来取代 shell。
是否闪烁光标
可供选择的值包括“system”可使用全局光标闪烁设置，“on”或“off”可显式设置模式。
光标外观
可能值为 "block" 以使用块状鼠标，"ibeam" 以使用垂直线鼠标，或者 "underline" 以使用下划线鼠标。
取代 shell 的自定义命令
在 shell 的位置运行此命令，如果 use_custom_command 为 true。
终端窗口的图标
包含此配置文件的标签页/窗口所用的图标。
终端程序调色板
终端使用 16 色的调色板，以供终端中的程序使用。这就是该调色板，格式为冒号分隔的颜色名称。颜色名必须是十六进制格式，即“#FF00FF”
字体
Pango 字体名称。例如“Sans 12”或“Monospace Bold 14”。
背景类型
终端背景的类型。可以是“纯色”代表单一颜色，“图像”代表图像，以及“透明”代表真透明(如果一个复合窗口管理器在运行)，否者为假透明。
背景图像
背景图像的文件名。
是否滚动背景图像
如果为真(true)，就和前景文字一起滚动背景图像；如果为 false，就将图像保留在固定位置，而只滚动其上的文字。
背景图像的灰暗程度
介于 0.0 和 1.0 之间的值，表明背景图像的灰暗程度。0.0 表示不作处理，1.0 表明全黑。在目前的实现方案中，只有两种级别的灰暗程度，所以这里的设置和布尔值类似，即 0.0 禁用灰暗效果。
退格键效果
设置退格键生成的代码。可供选择的值有，“ascii-del”代表 ASCII DEL 字符，“control-h”代表 Ctrl+H(即 ASCII BS 字符)，“escape-sequence”代表 backspace 或 delete 代表的转码序列。“ascii-del”通常是退格键正确的设置。
删除键效果
设置 Delete 键生成的代码。可供选择的值有，“ascii-del”代表 ASCII DEL 字符，“control-h”代表 Ctrl+H(即 ASCII BS 字符)，“escape-sequence”代表 backspace 或 delete 代表的转码序列。“ascii-del”通常是 Delete 键正确的设置。
是否为终端部件使用主题中的颜色
如果为真(true)，终端就会使用文本输入框的主题颜色配对，而不是用户给出的颜色。
是否使用系统字体
如果为真(true)，终端就将使用与桌面相同的标准字体，假如该字体为等宽字体的话(否则就是用最为接近的字体)。
