代理
代理服务器
可以根据需要手动设置用于访问 Internet 的选项。
设置
代理服务器
的设置。
代理服务器
指定代理定义的类型。
无
不使用代理服务器访问 Internet。用于在计算机上直接建立与不使用代理服务器的 Internet 供应商的连接。
手动
用于手动输入代理服务器。根据可用的 Internet 服务，指定代理服务器。请向系统管理员询问需要输入的代理服务器和端口。
键入不含协议前缀的服务器名称。例如，请键入 &quot; www.sun.com &quot; 而不是 &quot; http://www.sun.com &quot; 。
系统
在使用 GNOME 或 KDE 的 Windows 或 UNIX 系统上，此选项通知 %PRODUCTNAME 使用系统设置。必须重新启动 %PRODUCTNAME 以启动此设置。
Http 代理
HTTP
代理服务器的名称。
在右边的字段中键入端口号。
HTTPS 代理服务器
输入 HTTPS 代理服务器的名称。在右边的字段中输入端口号。
FTP 代理服务器
FTP
代理服务器的名称。
在右边的字段中键入端口号。
不使用代理服务器
指定不需要任何代理服务器的服务器名称，用分号分隔。
例如，局部网络中使用的服务器，以及用于处理视频和音频流的服务器。
在主机名称和域名称中，也可以使用占位符。例如，键入 &quot; *.sun.com &quot; 可以定位到 sun.com 域中不使用代理服务器的所有主机。
端口
键入相应代理服务器的端口号。
端口号的最大值为 65535。
