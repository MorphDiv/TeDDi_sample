求解器
打开“求解器”对话框。求解器通过单变量求解方法使用多个未知变量来求解方程式。
target cell
输入或单击目标单元格的单元格引用。该字段接受其值将要被优化的单元格地址。
maximum
试着求解目标单元格最大值方程式。
minimum
试着求解目标单元格最小值方程式。
value of
试着求解目标单元格内接近给定值方程式。
value of text field
输入值或单元格引用。
by changing cells
输入能被更改的单元格区域。
cell reference
输入单元格引用。
operators
从列表中选择一个运算符。
value
输入一个值或单元格引用。
shrinker
单击来收缩或还原对话框。您可以单击或选择工作表中的单元格。您可以在输入框中手动输入单元格引用。
delete row
单击删除列表中的行。该行下的行均上移。
options button
打开“选项”对话框。
solve button
单击求解为当前设置的方程式。对话框设置将会保留直到关闭当前文档。
使用求解器求解方程式
目标单元格
（也称为“目标”）中产生最佳值。您可以选择目标单元格中的值为最大值、最小值、或接近一个给定的值。
通过更改单元格
框输入的矩形单元格区域中。
您可以定义一系列限制条件来设置某些单元格的约束。例如，您可以设置约束为某个变量或单元格不能比另外某个变量、或某个给定值大。您也可以定义约束为必须有一个或多个变量为整数（不含小数的值），或二进制数值（仅为 0 和 1）。
默认求解器引挚仅支持线性方程。
