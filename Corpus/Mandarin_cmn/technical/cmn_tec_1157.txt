以 HTML 格式保存文本文档
为了在 Web 浏览器中进行查看，您可以将 $[officename] Writer 文档保存为 HTML 格式，另外，也可以根据需要将分页与特定的标题段落样式相关联，这样每当文档中显示该样式时，就会产生一个单独的 HTML 页面。$[officename] Writer 会自动创建一个页面，并在该页面中含有与这些页面中每一页的超链接。
当使用 HTML 格式保存文本文档时，其中含有的所有图形都会转换为 JPEG 格式，并使用 HTML 文档的名称作为图形文件名称的前缀。JPEG 图像与 HTML 文档保存在同一个文件夹中，并通过 HTML 代码中的 < IMG > 标记来引用。
对要产生新 HTML 页的段落应用一种默认 $[officename] 标题段落样式（如“标题 1”）。
