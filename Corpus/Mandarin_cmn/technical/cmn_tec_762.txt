Java 启动参数
可以使用此对话框为 Java 运行时环境 (JRE) 输入可选的启动参数。在此对话框中指定的设置对您启动的任何 JRE 均有效。
Java 启动参数
如同针对命令行操作一样，输入 JRE 的启动参数。单击“指定”即可将参数添加到可用启动参数的列表。
不要在路径名中使用转义符或引号。
例如，要将系统属性 &quot; myprop &quot; 指向文件夹，请输入以下参数：
-Dmyprop=c:\program files\java
要在 JRE 中启用调试，请输入以下参数：
-Xdebug
-Xrunjdwp:transport=dt_socket,server=y,address=8000
这些修改在重新启动 %PRODUCTNAME 后生效。
