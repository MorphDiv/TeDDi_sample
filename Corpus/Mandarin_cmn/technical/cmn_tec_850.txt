Web 向导 - 预览
指定要在何处发布网站和预览网站。
预览
在操作系统的默认 Web 浏览器中打开网页。
本地目录
将索引页和文件上载到本地目录。索引页保存到指定位置。所有其他文件保存到包含索引页的 &quot; myWebsite &quot; 目录。
...
打开一个对话框，可在其中选择文件夹。
存档文件
将索引页和文件添加到压缩的存档文件，然后将文件上载到网站。索引页保存到指定位置。所有其他文件保存到包含索引页的 &quot; myWebsite &quot; 目录。
视操作系统而定，可用的存档文件格式有 zip、gzip 和 war。
...
打开一个对话框，您可以在该对话框中指定存档文件的位置。
通过 FTP 的 Web
将文件上载到 FTP 服务器。索引页保存到指定位置。所有其他文件保存到包含索引页的 &quot; myWebsite &quot; 目录。
如果您通过代理服务器连接到 Internet，将无法使用 FTP 选项。
配置
FTP 连接
对话框，您可以在该对话框中编辑和测试 FTP 服务器的连接设置。
保存设置
保存在此向导中指定的设置。
另存为
为设置文件输入名称。
