类路径
可使用此对话框将文件夹和档案添加到 Java 类路径中。这些路径对您启动的任何 JRE 均有效。
指定的文件夹和档案
指定 Java 类或 Java 类库的位置。
新的类路径在您重新启动
%PRODUCTNAME
后生效。
通过类路径访问的 Java 类不接受安全检查。
添加档案
选择 jar 或 zip 格式的存档文件，然后将该文件添加到类路径中。
添加文件夹
选择文件夹并将其添加到类路径中。
删除
在列表中选择档案或文件夹，然后单击“删除”从类路径中删除对象。
