语法：
ChDir Text As String
参数：
Text:
用于指定目录路径或驱动器的任意字符串表达式。
如果只需要修改当前驱动器，请输入驱动器号，并在后面加上冒号。
示例：
Sub ExampleChDir
Dim sDir1 as String , sDir2 as String
sDir1 = &quot; c:\Test &quot;
sDir2 = &quot; d:\private &quot;
ChDir( sDir1 )
msgbox CurDir
ChDir( sDir2 )
msgbox CurDir
end sub
