浮动框架属性
修改选定浮动框架的属性。浮动框架最适用于当含有一个 html 文档时，或当被插入到其他 html 文档中时的情况。
名称
输入浮动框架的名称。名称中不能含有空格、特殊字符，也不能以下划线 ( _ ) 开头。
内容
...
按钮，然后找到要显示的文件。
例如，可以输入：
http://www.sun.de
file:///c|/readme.txt
...
打开
。
滚动条
向选定的浮动框架添加滚动条，或者从中删除滚动条。
打开
显示浮动框架的滚动条。
关闭
隐藏浮动框架的滚动条。
自动
如果当前活动浮动框架在需要时可能会具有滚动条，请选中此选项。
边框
显示或隐藏浮动框架的边框。
打开
显示浮动框架的边框。
关闭
隐藏浮动框架的边框。
至内容的间距
假设浮动框架内、外的文档均为 HTML 文档，则可以定义浮动框架的边框和内容之间的间隔。
宽度
输入浮动框架的左、右边缘与框架内容之间的水平间隔。浮动框架内、外的文档必须均为 HTML 文档。
高度
输入浮动框架的上、下边缘与框架内容之间的垂直间隔。浮动框架内、外的文档必须均为 HTML 文档。
默认
应用默认间距。
