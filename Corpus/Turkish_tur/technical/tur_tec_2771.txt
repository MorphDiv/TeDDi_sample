XL. iconv functions

To be able to use the functions defined in this module you must compile you PHP interpreter using the --with-iconv option.
In order to do so, you must have iconv() function in standard C library or libiconv installed on your system. libiconv library is available from http: / /www.gnu.org / software / libiconv /.
iconv library function converts files between various encoded character sets.
The supported character sets depend on iconv() implementation on your system.
Note that iconv() function in some system is not work well as you expect.
In this case, you should install libiconv library.
