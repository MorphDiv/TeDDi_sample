BCMath Arbitrary Precision Mathematics Functions

Requirements
Due to changes in the licensing, the BCMATH library is distributed separate from the standard PHP source distribution.
You can download the tar-gzipped archive at the url: http: / /www.php.net / extra / number4.tar.gz.
Read the file README.BCMATH in the PHP distribution for more information.
Installation

In PHP 3, these functions are only available if PHP was not configured with --disable-bcmath.
