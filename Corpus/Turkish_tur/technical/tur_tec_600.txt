ftp_connect
(PHP 3 = 3.0.13, PHP 4 = 4.0.0)
Description
Returns a FTP stream on success, FALSE on error.
ftp_connect() opens up a FTP connection to the specified host.

If it is omitted or zero, then the default FTP port, 21, will be used.


The timeout can be changed and queried anytime with ftp_set_option() and ftp_get_option().
Not:
This parameter is only available in CVS.
