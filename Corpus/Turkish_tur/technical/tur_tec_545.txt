fdf_open
(PHP 3 = 3.0.6, PHP 4 = 4.0.0)
Description

This file must contain the data as returned from a PDF form.
Currently, the file has to be created 'manually' by using fopen() and writing the content of HTTP_FDF_DATA with fwrite() into it.
A mechanism like for HTML form data where for each input field a variable is created does not exist.
