pg_numrows
(PHP 3, PHP 4 = 4.0.0)
Description
pg_numrows() will return the number of rows in a PostgreSQL result resource. result is a query result resource returned by pg_exec().
This function will return -1 on error.
Not:
Use pg_cmdtuples() to get number of rows affected by INSERT, UPDATE and DELETE query.
See also pg_numfields() and pg_cmdtuples().
