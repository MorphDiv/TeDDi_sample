คีย์นี้เก็บรายชื่อโฮสต์ที่ให้ติดต่อโดยตรง แทนที่จะผ่านพร็อกซี (ถ้าเปิดทำงานอยู่) สามารถตั้งเป็น ชื่อโฮสต์ โดเมน (ใช้ wildcard ข้างหน้าเช่น *.foo.com), ที่อยู่ IP (ทั้ง IPv4 และ IPv6) หรือที่อยู่บนเครือข่ายพร้อมกับ netmask (เช่น 192.168.0.0/24)
โหมดการตั้งค่าพร็อกซี
เลือกโหมดการตั้งค่าพร็อกซี สามารถตั้งเป็น "none", "manual", "auto"
ชื่อโฮสต์พร็อกซี Secure HTTP
ชื่อเครื่องที่จะติดต่อ Secure HTTP ผ่าน
พอร์ตพร็อกซี Secure HTTP
พอร์ตบนเครื่องที่ตั้งไว้ใน "/system/proxy/secure_host" ที่จะใช้ติดต่อผ่าน
ชื่อโฮสต์พร็อกซี FTP
ชื่อเครื่องที่จะติดต่อ FTP ผ่าน
พอร์ตพร็อกซี FTP
พอร์ตบนเครื่องที่ตั้งไว้ใน "/system/proxy/ftp_host" ที่จะใช้ติดต่อผ่าน
ชื่อโฮสต์พร็อกซี SOCKS
ชื่อเครื่องที่จะติดต่อ socks ผ่าน
พอร์ตพร็อกซี SOCKS
พอร์ตบนเครื่องที่ตั้งไว้ใน "/system/proxy/socks_host" ที่จะใช้ติดต่อผ่าน
URL สำหรับตั้งค่าพร็อกซีโดยอัตโนมัติ
URL ที่จัดส่งค่าตั้งพร็อกซี
เวิร์กกรุป SMB
เวิร์กกรุปหรือโดเมนของเครือข่ายวินโดวส์ที่ผู้ใช้อาศัยอยู่ ผู้ใช้อาจต้องออกจากระบบแล้วเข้าใหม่เพื่อจะเริ่มใช้เวิร์กกรุปใหม่ได้
