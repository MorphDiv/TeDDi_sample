Καλώς ήρθατε στην εισαγωγή του KPhotoAlbum Αυτός ο μάγος θα σας βοηθήσει στα βήματα της λειτουργίας εισαγωγής εικόνων. Τα βήματα είναι: Πρώτα πρέπει να επιλέξετε τις εικόνες που επιθυμείτε να εισάγετε από το αρχείο εξαγωγής. Αυτό το κάνετε επιλέγοντας το πλαίσιο ενεργοποίησης δίπλα σε κάθε εικόνα. Έπειτα πρέπει να ορίσετε τον κατάλογο στον οποίο θα τοποθετήσει το KPhotoAlbum τις εικόνες. Αυτός ο κατάλογος φυσικά πρέπει να βρίσκεται κάτω από τον ανώτερο κατάλογο που χρησιμοποιεί το KPhotoAlbum για τις εικόνες. Το KPhotoAlbum θα επιλύσει τυχόν συγκρούσεις με την ονομασία των αρχείων Το επόμενο βήμα είναι η επιλογή των κατηγοριών που επιθυμείτε να εισάγετε (Πρόσωπα, Τοποθεσίες,...) και επίσης πως ακριβώς το KPhotoAlbum θα αντιστοιχίσει τις κατηγορίες του αρχείου με τις δικές σας υπάρχουσες κατηγορίες. Φανταστείτε ότι φορτώνετε ένα αρχείο, στο οποίο κάποια κατηγορία ονομάζεται Flower (η αγγλική λέξη για το λουλούδι), πιθανότατα επιθυμείτε την αντιστοίχηση αυτής της κατηγορίας με τη δικιά σας κατηγορία, η οποία μπορεί να ονομάζεται Λουλούδια. Κατηγορία για εικονικά άλμπουμ:
Το KPhotoAlbum θα προσπαθήσει να διαβάσει την ημερομηνία της εικόνας από πληροφορίες EXIF μέσα στην εικόνα. Αν αυτό αποτύχει θα προσπαθήσει να ανακτήσει την ημερομηνία από τη χρονική σφραγίδα του αρχείου. Ωστόσο, αυτή η πληροφορία μπορεί να είναι λανθασμένη αν η εικόνα σαρώθηκε (ζητάτε την ημερομηνία που τραβήχτηκε η φωτογραφία, και όχι την ημερομηνία της σάρωσης). Αν σαρώνετε φωτογραφίες, αντί να χρησιμοποιείτε ψηφιακή φωτογραφική μηχανή, θα πρέπει να απαντήσετε όχι. Αν δε σαρώνετε ποτέ φωτογραφίες, θα πρέπει να απαντήσετε ναι, σε άλλη περίπτωση απαντήστε ερώτηση. Αυτό θα σας επιτρέπει να αποφασίζετε αν οι εικόνες προέρχονται από σαρωτή ή φωτογραφική μηχανή, για κάθε συνεδρία.
Οι εικόνες JPEG μπορεί να περιέχουν πληροφορίες σχετικά με την περιστροφή. Αν έχετε κάποιο λόγο να μη χρησιμοποιείτε αυτές τις πληροφορίες για να ανακτήσετε τον προσανατολισμό των εικόνων σας, ενεργοποιείστε αυτό το πλαίσιο επιλογής. Σημείωση: Η ψηφιακή σας φωτογραφική μηχανή μπορεί να μην καταγράφει καθόλου αυτή την πληροφορία.
Οι εικόνες JPEG μπορεί να περιέχουν μια περιγραφή. Ενεργοποιώντας αυτό το πλαίσιο επιλογής ορίζετε ότι θέλετε να χρησιμοποιήσετε αυτή την περιγραφή ως προκαθορισμένη για τις εικόνες σας.
Το KPhotoAlbum είναι ικανό να αναζητεί για νέες εικόνες όταν εκκινηθεί, αυτό ωστόσο διαρκεί κάποιο χρόνο, έτσι μπορεί να επιθυμείτε να ορίσετε στο KPhotoAlbum τη χειροκίνητη αναζήτηση για νέες εικόνες χρησιμοποιώντας το Συντήρηση - > Επανασάρωση για νέες εικόνες
Το KPhotoAlbum έχει τη δυνατότητα να διαβάζει κάποια είδη ακατέργαστων μορφών εικόνων RAW. Μερικές φωτογραφικές μηχανές αποθηκεύουν τόσο την εικόνα RAW όσο και μια αντίστοιχη JPEG ή TIFF εικόνα. Αυτό έχει σαν αποτέλεσμα διπλά αντίγραφα της κάθε εικόνας να αποθηκεύονται στο KPhotoAlbum, πράγμα που μπορεί να μην είναι επιθυμητό. Αν ενεργοποιήσετε αυτή την επιλογή το KPhotoAlbum δε θα διαβάζει τα αρχεία RAW όταν υπάρχουν επίσης τα αντίστοιχα αρχεία εικόνας.
Το KPhotoAlbum μοιράζεται πρόσθετα με άλλες εφαρμογές επεξεργασίας εικόνας, που έχουν την έννοια των άλμπουμ. Το KPhotoAlbum δεν έχει αυτή την έννοια, αλλά για να δουλέψουν μερικά πρόσθετα το KPhotoAlbum συμπεριφέρεται σαν να τα υποστηρίζει. Το KPhotoAlbum ορίζει ως τρέχον άλμπουμ την τρέχουσα προβολή - δηλαδή όλες τις εικόνες που ο περιηγητής εμφανίζει. Επιπλέον του τρέχοντος άλμπουμ, το KPhotoAlbum πρέπει να δώσει και μια λίστα όλων των άλμπουμ. Αυτή ορίζεται ω ς εξής: Όταν το KPhotoAlbum εμφανίζει το περιεχόμενο μιας κατηγορίας, για παράδειγμα όλα τα πρόσωπα, τότε κάθε αντικείμενο της κατηγορίες θα εμφανίζεται ως άλμπουμ στο πρόσθετο. Σε άλλη περίπτωση, η κατηγορία που ορίζετε με αυτή την επιλογή θα χρησιμοποιηθεί. Για παράδειγμα αν ορίσετε τα πρόσωπα με αυτή την επιλογή τότε το KPhotoAlbum θα ενεργεί σαν να επιλέξατε την εμφάνιση των προσώπων πριν από την κλήση του πρόσθετου. Οι περισσότεροι χρήστες πιθανώς θα ήθελαν να συμπληρώσουν εδώ γεγονότα.

Περίληψη Παρακάτω μπορείτε να δείτε μια λίστα των χαρακτηριστικών κατά τη μεταγλώττιση του KPhotoAlbum καθώς και την κατάστασή τους:% 1
Τί μπορώ να κάνω αν λείπει κάποιο χαρακτηριστικό; Αν μεταγλωττίσατε το KPhotoAlbum μόνοι σας, τότε διαβάστε τα παρακάτω για να δείτε τι πρέπει να εγκαταστήσετε ώστε να βρείτε το χαρακτηριστικό που επιθυμείτε. Αν από την άλλη εγκαταστήσατε το KPhotoAlbum από ένα δυαδικό πακέτο, παρακαλώ ενημερώστε το δημιουργό του πακέτου για την έλλειψη του χαρακτηριστικού που επιθυμείτε περιλαμβάνοντας τις πληροφορίες της παρακάτω παραγράφου. Στη περίπτωση που σας λείπει κάποιο χαρακτηριστικό και δε μεταγλωττίσατε μόνοι σας το KPhotoAlbum, θα ήταν καλό να το κάνετε. Δεν είναι τόσο δύσκολο. Αν χρειάζεστε βοήθεια για τη μεταγλώττιση του KPhotoAlbum, ζητήστε ελεύθερα στη Λίστα αλληλογραφίας του KPhotoAlbumΤα βήματα για τη μεταγλώττιση του KPhotoAlbum μπορούν να βρεθούν στην Αρχική σελίδα του KPhotoAlbum. Αν ποτέ δεν έχετε μεταγλωττίσει μια εφαρμογή του KDE, σιγουρευτείτε ότι έχετε εγκατεστημένα τα πακέτα ανάπτυξης: Στις περισσότερες διανομές έχουν ονόματα όπως το kdelibs- devel
Υποστήριξη πρόσθετωνΤο KPhotoAlbum περιέχει ένα σύστημα πρόσθετων με πολλές επεκτάσεις. Μπορείτε να βρείτε μεταξύ των άλλων, πρόσθετα για: Την εγγραφή εικόνων σε cd ή dvd Τη διόρθωση των χρονικών σφραγίδων στις εικόνες σαςΤη δημιουργία ενός ημερολογίου με τις εικόνες σαςΤην αποστολή των εικόνων σας στο flickrΤην αποστολή των εικόνων σας στο facebookΗ βιβλιοθήκη προσθέτων ονομάζεται KIPI, και μπορείτε να την κατεβάσετε από την Αρχική σελίδα του KIPI
Υποστήριξη EXIF Οι εικόνες αποθηκεύουν πληροφορίες όπως την ημερομηνία που τραβήχτηκε η φωτογραφία, την γωνία λήψης, την απόσταση εστίασης, και το χρόνο του διαφράγματος, οι οποίες είναι γνωστές ως πληροφορίες EXIF. Το KPhotoAlbum χρησιμοποιεί τη Βιβλιοθήκη EXIV2 για την ανάγνωση των πληροφοριών EXIF από τις εικόνες
Υποστήριξη βάσης δεδομένων SQL Το KPhotoAlbum σας επιτρέπει να αναζητήσετε με τη χρήση ορισμένου αριθμού ετικετών EXIF. Για αυτήν τη λειτουργία το KPhotoAlbum χρειάζεται μια βάση δεδομένων Sqlite. Ταυτόχρονα θα πρέπει να έχετε εγκαταστήσει το πακέτο της qt για υποστήριξη της sqlite (π. χ. qt- sql- sqlite).
Υποστήριξη εικόνων επισκόπησης βίντεοΤο KPhotoAlbum ζητάει τη βοήθεια του υποσυστήματος προσθέτων του KDE για τη δημιουργία εικόνων επισκόπησης βίντεο. Για την ώρα το KDE4 δεν εγκαθιστά κανένα πρόσθετο για τη λειτουργία αυτή, οπότε θα πρέπει να το εγκαταστήσετε χειροκίνητα MPlayerThumbs.
Υποστήριξη βίντεο Το KPhotoAlbum εξαρτάται από την αρχιτεκτονική Qt Phonon για την προβολή βίντεο, το οποίο με τη σειρά του εξαρτάται από το GStreamer. Αν αυτό το χαρακτηριστικό δεν είναι ενεργοποιημένο στο σύστημά σας, μπορείτε να πάρετε βοήθεια από το άρθρο στο wiki του KPhotoAlbum για υποστήριξη βίντεο.
Δε βρέθηκαν τύποι mime βίντεο, το οποίο υποδηλώνει ότι η Qt δε μεταγλωττίστηκε με υποστήριξη Phonon, ή δεν υπάρχουν κάποιοι κωδικοποιητές
Το Phonon έχει τη δυνατότητα αναπαραγωγής ταινιών αυτών των τύπων mime:% 1

Γενικά σχόλια
Κάθε φορά που ο πελάτης αλληλογραφίας μου που δηλώνει ότι μια νέα φόρμα έχει σταλεί, ανυπομονώ να τελειώσει η δουλειά για να συνεχίσω τον προγραμματισμό του KPhotoAlbum. Ένα σημαντικό πρόβλημα με την ανάπτυξη λογισμικού είναι ότι οι προγραμματιστές συνήθως μαθαίνουν σχετικά με προβλήματα και σπάνια για τις ευκολίες που προσφέρει η εφαρμογή τους. Εδώ λοιπόν σας δίνεται η ευκαιρία να μεταδώσετε λίγη από τη θετική σας ενέργεια σε μένα λέγοντας μου πόσο σας άρεσε η εφαρμογή.
Σας ευχαριστώ πολύ για το χρόνο σας, ελπίζω να συνεχίσετε να χρησιμοποιείτε το KPhotoAlbum και ότι οι μελλοντικές εκδόσεις του θα ταιριάζουν ακόμη πιο πολύ στις ανάγκες σας. Τέλος επιτρέψτε μου να σας ζητήσω να σκεφτείτε κάποια δωρεά. Δείτε τη Βοήθεια - > Δωρεά στο μενού για τον τρόπο που μπορείτε να προσφέρετε χρήματα. Για μια άλλη φορά σας ευχαριστώ που συμπληρώσατε αυτό το ερωτηματολόγιο - Jesper
