Ο προεπιλεγμένος κατάλογος στον οποίο θα κλωνοποιούνται τα νέα αποθετήρια.
Προσανατολισμός της κύριας διεπαφής (κάθετα ή οριζόντια)
Ρύθμιση που ορίζει τον προσανατολισμό της κύριας διεπαφής.
Πότε να συμπτύσσονται οι ανενεργές γραμμές
Ρύθμιση που ορίζει πότε μια ανενεργή γραμμή πρέπει να συμπτύσσεται. Έγκυρες τιμές είναι μεταξύ 0 - 4, όπου το 0 υποδεικνύει τις 'νωρίς' και το 4 τις 'αργά'.
Ρύθμιση που ορίζει αν θα συμπτύσσονται οι γραμμές ιστορικού που δεν εμφανίζουν δραστηριότητα. Η ενεργοποίηση αυτής της ρύθμισης μπορεί να προσφέρει μια πιο καθαρή προβολή ιστορικού όταν υπάρχει πολλή παράλληλη ανάπτυξη. Δείτε τις ανενεργές γραμμές σύμπτυξης για να ελέγξετε πότε οι γραμμές θα πρέπει να συμπτύσσονται.
Εμφάνιση ιστορικού με τοπολογική σειρά
Ρύθμιση που ορίζει αν θα εμφανίζεται το ιστορικό με τοπολογική σειρά.
Εμφάνιση αλλαγών στην κρύπτη
Ρύθμιση που ορίζει αν θα εμφανίζονται στοιχεία για την κρύπτη στο ιστορικό.
Εμφάνιση έτοιμων αλλαγών
Ρύθμιση που ορίζει αν θα εμφανίζεται ένα εικονικό στοιχείο για τις τρέχουσες έτοιμες αλλαγές στο ιστορικό.
Εμφάνιση αλλαγών που ανακλήθηκαν
Ρύθμιση που ορίζει αν θα εμφανίζεται ένα εικονικό στοιχείο για τις τρέχουσες αλλαγές που έχουν ανακληθεί στο ιστορικό.
Εμφάνιση δεξιού περιθωρίου στη προβολή υποβολής μηνύματος
Εμφάνιση δείκτη δεξιού περιθωρίου στην προβολή υποβολής μηνύματος. Αυτό μπορεί να χρησιμοποιηθεί για να δείτε εύκολα πού να σπάσει η υποβολή μηνύματος σε μια συγκεκριμένη στήλη.
Στήλη στην οποία προβάλλεται το δεξιό περιθώριο
Η στήλη στην οποία θα εμφανίζεται το δεξιό περιθώριο, αν η προτίμηση για την εμφάνιση δεξιού περιθωρίου ορισθεί σε ΑΛΗΘΗ.
Εμφάνιση θέματος περιθωρίου στη προβολή υποβολής μηνύματος
Επισήμανση του κειμένου θέματος του μηνύματος υποβολής, όταν περνά το περιθώριο που ορίσθηκε από τη θέση-περιθωρίου-θέματος.
Στήλη στην οποία προβάλλεται το περιθώριο θέματος
Η στήλη στην οποία θα εμφανίζεται το περιθώριο θέματος, αν η προτίμηση για την εμφάνιση περιθωρίου θέματος ορισθεί σε ΑΛΗΘΗ.
Ενεργοποίηση ορθογραφικού ελέγχου
Ρύθμιση που ορίζει αν θα είναι ενεργοποιημένος ή όχι ο ορθογραφικός έλεγχος, κατά τη συγγραφή του μηνύματος υποβολής.
Γλώσσα ορθογραφικού ελέγχου
Η γλώσσα που θα χρησιμοποιηθεί όταν είναι ενεργοποιημένος ο ορθογραφικός έλεγχος, για τη συγγραφή του μηνύματος υποβολής.
Αγνόηση αλλαγών κενών διαστημάτων
Ρύθμιση που ορίζει αν θα πρέπει να αγνοούνται αλλαγές σε κενά διαστήματα κατά την εμφάνιση των διαφορών μιας υποβολής.
Ενσωματωμένη εμφάνιση αλλαγών
Ρύθμιση που υποδεικνύει αν οι αλλαγές μέσα σε γραμμές θα πρέπει να εμφανίζονται ενσωματωμένες
Αριθμός των πριν/μετά γραμμών περιεχομένου
Ρύθμιση που καθορίζει πόσες γραμμές του περιεχομένου (πριν και μετά) θα πρέπει να εμφανίζονται κατά την εμφάνιση των διαφορών μιας υποβολής.
Το πλάτος απεικόνισης του χαρακτήρα στηλοθέτη
Ρύθμιση που καθορίζει πόσο διάστημα θα πρέπει να καταλαμβάνει ένας χαρακτήρας στηλοθέτη κατά την εμφάνιση των διαφορών μιας υποβολής.
