Πότε να συμπτύσσονται οι ανενεργές γραμμές
Η ρύθμιση που ορίζει πότε μια ανενεργή γραμμή πρέπει να συμπτύσσεται. Έγκυρες τιμές είναι μεταξύ 0 - 4, όπου το 0 υποδεικνύει τις 'νωρίς' και το 4 τις 'αργά'.
Ρύθμιση που δείχνει αν θα συμπτύσσονται οι γραμμές ιστορικού που δεν εμφανίζουν δραστηριότητα. Η ενεργοποίηση αυτής της ρύθμισης μπορεί να προσφέρει μια πιο καθαρή προβολή ιστορικού όταν υπάρχει πολλή παράλληλη ανάπτυξη. Δείτε τις ανενεργές γραμμές σύμπτυξης για να ελέγξετε πότε οι γραμμές θα πρέπει να συμπτύσσονται.
Εμφάνιση ιστορικού με Τοπολογική Σειρά

Εμφάνιση Δεξιού περιθωρίου στη προβολή Υποβολής μηνύματος
Εμφάνιση δείκτη δεξιού περιθωρίου στην προβολή υποβολής μηνύματος. Αυτό μπορεί να χρησιμοποιηθεί για να δείτε εύκολα πού να σπάσει η υποβολή μηνύματος σε μια συγκεκριμένη στήλη.
Στήλη στην οποία προβάλλεται το Δεξιό περιθώριο
Η στήλη στην οποία θα εμφανίζεται το Δεξιό περιθώριο, αν η προτίμηση για την εμφάνιση δεξιού περιθωρίου ορισθεί σε ΑΛΗΘΗ.
Εμφάνιση θέματος περιθωρίου στη προβολή Υποβολής μηνύματος
Επισήμανση του κειμένου θέματος του μηνύματος υποβολής, όταν περνά το περιθώριο που ορίσθηκε από τη θέση-περιθωρίου-θέματος.
