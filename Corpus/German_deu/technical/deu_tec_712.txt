Browser Plug-in
Gibt an, dass das Mozilla-Plugin zum Anzeigen von %PRODUCTNAME-Dokumenten verwendet werden soll.
Wenn Sie in Ihrem Mozilla- oder Netscape-Browser auf einen Hyperlink zu einem %PRODUCTNAME-Dokument klicken, kann einer der folgenden Vorgänge durchgeführt werden:
Der Browser fragt, ob das Dokument auf der Festplatte gespeichert werden soll oder ob eine Anwendung zum Öffnen des Dokuments verwendet werden soll.
Dieser Anforderungsdialog wird standardmäßig angezeigt, wenn Sie zum ersten Mal auf einen Hyperlink zu einem %PRODUCTNAME-Dokument klicken und das Browser-Plugin nicht aktiviert ist.
Der Browser lädt das Dokument in einen temporären Ordner herunter und weist Ihr System an, %PRODUCTNAME auszuführen und das Dokument in %PRODUCTNAME zu öffnen.
Dies ist der Standardvorgang, wenn Sie das Browser-Plugin nicht aktiviert haben und bei der ersten Abfrage festgelegt haben, das Dokument mit %PRODUCTNAME zu öffnen.

Anzeigen von Dokumenten im Browser
Aktiviert das Browser-Plugin zum Anzeigen von %PRODUCTNAME-Dokumenten in einem Browserfenster.
Aktivieren, dann auf &quot; OK &quot; klicken und den Browser neu starten.
Klicken Sie dann im Browser-Fenster auf einen Hyperlink zu einem %PRODUCTNAME-Dokument.
Für das Plugin muss eine Version von Mozilla, Netscape oder Firefox auf Ihrem System installiert sein.
