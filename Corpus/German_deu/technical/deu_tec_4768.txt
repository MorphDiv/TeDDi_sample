Ausführliche Anleitungen
Dieses Kapitel soll zeigen, wie die häufigsten Aufgaben in & kpresenter; durchgeführt werden.
Erstellung einer HTML -Präsentation
Erstellt HTML -Seiten mit zugehörigen Navigationselementen aus einer vorhandenen Präsentation.
Zuerst erstellen Sie eine neue oder öffnen eine vorhandene Präsentation. Dann klicken Sie auf Datei HTML-Präsentation erstellen...
Menüeintrag HTML-Präsentation erstellen...
Menüeintrag HTML-Präsentation erstellen...
In einem Dialog können Sie bereits vorher gespeicherte Einstellungen laden:
Dialog für bereits vorher gespeicherte Einstellungen
Dialog für bereits vorher gespeicherte Einstellungen
Klicken Sie auf Nein, wenn Sie diese Funktion zum ersten Mal verwenden, oder wenn Sie Einstellungen ändern möchten. Mit Ja benutzen Sie bereits früher eingegebene Einstellungen für Ihre HTML -Präsentation, damit können Sie ohne Bearbeitung von Einstellungen schnell eine Präsentation erstellen.
In einem Dialog geben Sie dann einige Einstellungen ein:
Eingabe der Informationen für die HTML -Präsentation
Eingabe der Informationen für die HTML -Präsentation
Füllen Sie alle Textfelder aus. Einige Felder enthalten bereits Ihre im & kcontrolcenter; eingetragenen Daten. Überprüfen Sie ob, die Pfadangabe richtig ist, da im angegebenen Ordner von & kpresenter; die HTML-Dateien und Bilder gespeichert werden. Wenn alle Eingaben richtig sind, klicken Sie auf Weiter.
Falls der eingetragene Ordner nicht vorhanden ist, werden Sie von & kpresenter; gefragt, ob er erstellt werden soll.
Nun können Sie die Einstellungen für HTML angeben:
Eingabe weiterer Informationen für die HTML -Präsentation
Eingabe weiterer Informationen für die HTML -Präsentation
Hier können Sie die Kodierung und den Dokumententyp zusammen mit der Vergrößerung festlegen. Falls die Präsentation ursprünglich für ein LCD-Vollbild mit 1280x1024 Pixel gedacht war, sollten Sie die Folien auf 50% verkleinern, damit die Präsentation in das viel kleinere Fenster eines Browsers passt.
In den meisten Fällen sind die Standardeinstellungen passend, so dass Sie dann nur auf Weiter klicken müssen, um auf die nächste Seite zu kommen.
Nun können Sie die Farben für die Webseiten festlegen:
Stil der Webseiten für die HTML -Präsentation
Stil der Webseiten für die HTML -Präsentation
Bei der Erzeugung der Webseiten verwendet & kpresenter; die eingestellten Farben für den Navigationsbereich. Um sie zu ändern, klicken Sie auf die Farbfelder neben den Texten. Die Farben der Folien selbst werden unverändert übernommen.
Wählen Sie Farben, die sich vom Inhalt Ihrer Präsentation unterscheiden, damit die Navigationselemente gut erkennbar sind.
Mit Klicken auf den Knopf Weiter kommen Sie zum nächsten Dialog, in dem Sie den Titel jeder Folie für die HTML -Seiten ändern können:
Eingabe der Folientitel
Eingabe der Folientitel

Unbeaufsichtigte Präsentationen
Unbeaufsichtigte Präsentationen
Stellen Sie die Zeit für den Übergang zur nächsten Folie ein und geben Sie an, ob die Präsentation am Ende automatisch wieder von vorn beginnen soll.
Nun können Sie den Ihre HTML -Präsentation erstellen, diesen Vorgang sehen Sie in diesem Dialogfeld:
Speicherung der Einstellungen
Speicherung der Einstellungen
Bevor Sie den Dialog schließen, können Sie noch die Einstellungen mit Einstellung speichern... sichern.
Damit wurden am vorher festgelegten Pfad neue html / - und pics / -Ordner angelegt:
Die neu angelegten Ordner der HTML -Präsentation
Die neu angelegten Ordner der HTML -Präsentation
Die neu erstellte Datei index.html dient als Einstiegspunkt zum Start der Präsentation.
Für einen Test öffnen Sie mit dem Dateimanager oder Browser die Datei index.html. Klicken Sie auf den Start, so wird die erste Seite der Präsentation angezeigt. Der weitere Ablauf entspricht einer & kpresenter;-Präsentation.
Mit den blauen Pfeilen können Sie in den Folien blättern, ein Klick auf das Foliensymbol bringt Sie zum Inhaltsverzeichnis der Präsentation.
Um Ihre Präsentation zu veröffentlichen, laden Sie die neu erstellten Seiten ins Internet, ohne dabei die Ordnerstruktur zu verändern.
Export von & kpresenter;-Präsentationen auf einen Memory Stick
Einige Sony-Projektoren können eine Präsentation direkt von einem Memory Stick ablaufen lassen, ohne dass dabei ein Computer angeschlossen sein muss. & kpresenter; kann Präsentationen in dem Format für diese Projektoren exportieren. Diese Anleitung zeigt Ihnen, wie Sie Ihre Präsentation exportieren können.
Jede exportierte Präsentation besteht aus einer Serie von Bilddateien (eine für jede Folie der ursprünglichen Präsentation, sowie zwei Titelfolien) und einer Indexdatei. Diese werden in einer ziemlich komplizierten Ordnerstruktur angelegt, die die Speicherung von mehr als einer Präsentation auf einem Memory Stick ermöglicht. Ein aus zwei Präsentationen bestehendes Beispiel sehen Sie hier:
Memory Stick -Ordnerstruktur
Wenn Sie die für den Export bestimmte Präsentation noch nicht geöffnet haben, wählen Sie sie mit Datei Öffnen... im Menü aus.
Wählen Sie jetzt im Menü Datei Memory-Stick-Präsentation erstellen....
Menüeintrag Memory-Stick-Präsentation erstellen...
Menüeintrag Memory-Stick-Präsentation erstellen...
Damit öffnen Sie den Dialog Memory-Stick-Präsentation.
Der Dialog Memory-Stick-Präsentation
Der Dialog Memory-Stick-Präsentation
Der Pfad ist der Ordner, in dem die Stammverzeichnisse (DCIM und MSSONY, siehe dazu das Orderstruktur-Diagramm weiter oben) angelegt werden. Wenn Sie die Präsentation direkt auf den Memory-Stick exportieren wollen, geben Sie in diesem Textfeld den Ordner an, an dem das Wechselmedium eingebunden ist.
Der Titel wird sowohl als Name für die Index-Datei wie auch als Überschrift für das Titelblatt der Präsentation verwendet. Beachten Sie, dass dieser Titel keinen Einfluss auf die Titel der anderen Präsentationsfolien hat; die Einleitungsfolie ist nur bei der Einrichtung des Sony-Projektors vor dem Start der eigentlichen Präsentation sichtbar.
Wählen Sie die Einstellung Textfarbe festlegen, um den Dialog zu erweitern:
Der erweiterte Dialog Memory-Stick-Präsentation
Der erweiterte Dialog Memory-Stick-Präsentation
Mit den beiden Farbwahlfelder stellen Sie die Farben für die Einleitungsfolie ein. Diese Einstellung kann es Ihnen erleichtern, bei der Bedienung des Projektors die eigentliche Präsentation leichter zu erkennen, aber Sie können diese Einstellungen ohne Bedenken unverändert lassen.
Nun drücken Sie den Knopf OK, um den Export zu starten.
Ihre Präsentation wird nun im richtigen Format exportiert, der folgende Dialog zeigt diesen Vorgang:
Fortschrittsdialog Memory-Stick-Präsentation
Fortschrittsdialog Memory-Stick-Präsentation
Wenn der Exportvorgang abgeschlossen ist, können Sie den Knopf Fertig drücken, und der Dialog wird geschlossen.
Wenn Sie die Präsentation nicht direkt auf den eingebundenen Memory-Stick exportiert haben, müssen Sie die Dateien kopieren, bevor Sie den Projektor benutzen können. Wie auch bereits vorher angemerkt, müssen Sie die richtigen Ordner und Dateien, angefangen mit DCIM und MSSONY in den Wurzelordner (/) des Memory Stick kopieren. Es ist wichtig, dass die Ordnerstruktur erhalten bleibt, und es wird wahrscheinlich am einfachsten sein, wenn Sie die Ordner DCIM und MSSONY als Ganzes kopieren.
Wenn nötig, können Sie die Index-Datei umbenennen (die als Vorgabe den Namen Vorführung.SPP erhält), aber das gilt nicht für die Namen der Bilddateien oder Ordner, da diese in der Index-Datei eingetragen sind.
Vorlagen für & kpresenter; erstellen
Die & kpresenter;-Bibliothek kann auf einfache Weise um eigene Vorlagen erweitert werden.
Wenn Sie eine von Ihnen erstellte Vorlage wiederverwenden wollen, können Sie sie als Vorlage speichern.
Dieses Anleitung erläutert die Erstellung und Abspeicherung einer eigenen Vorlage.
Als erstes starten Sie & kpresenter; und erstellen ein neues Dokument.
Um die Folie ein bisschen ansprechender zu gestalten, ändern Sie den Folienhintergrund im Menü mit Format > Folienhintergrund... oder durch Auswahl des gleichen Eintrags aus dem Kontextmenü der rechten Maustaste:
Der & kpresenter;-Dialog Folienhintergrund.
Der & kpresenter;-Dialog Folienhintergrund.
Der Hintergrund soll attraktiv aussehen, daher stellen Sie einen Farbverlauf im Auswahlfeld ein. Wählen Sie dann die Farben für den Verlauf durch Klicken auf die Farbfelder. Natürlich können Sie auch mit Bild eine Grafik als Hintergrund auswählen. Probieren Sie einfach verschiedene Einstellungen aus, bis das Ergebnis zufriedenstellend ist.
Auswahl eines Folienhintergrundes
Auswahl eines Folienhintergrundes
Damit ist gewünschten Hintergrund für die Seite eingestellt. Erstellen Sie ein Textfeld und geben Sie eine Überschrift für die Folie ein:
Eine Überschrift zur neuen Vorlage hinzufügen
Eine Überschrift zur neuen Vorlage hinzufügen
Wenn Sie mit Ihrer Vorlagen zufrieden sind, klicken Sie auf Datei Vorlagenverwaltung:
Datei Vorlagenverwaltung
Datei Vorlagenverwaltung
In diesem Dialog können Sie festlegen, wo Ihre Vorlage gespeichert wird:
Die neue Vorlage speichern
Die neue Vorlage speichern
Im Beispiel wird die Folie in der Gruppe Folien-Präsentationen unter dem Namen Vorlagen-Anleitung gespeichert. Drücken Sie auf OK, um Ihre Präsentation in die Sammlung der Vorlagen einzufügen.
Nun können Sie das gerade bearbeitete Dokument schließen, Sie müssen es nicht speichern. Wenn Sie das nächste Mal eine neue Präsentation erstellen, können Sie Ihre selbst erstellte Vorlage im Startdialog auswählen.
