Befehlsreferenz
Das Menü Datei
Ctrl; N Datei Neu...
Erstellt ein neues Projekt in einem neuen Fenster. Das aktuell bearbeitete Projekt bleibt weiter geöffnet.
Ctrl; O Datei Öffnen...
Öffnet ein vorhandenes Objekt in einem neuen Fenster. Das aktuell bearbeitete Projekt bleibt weiter geöffnet.
Datei Beispieldatenbanken herunterladen...
Öffnet den Dialog "Neue Sachen", um Beispieldatenbanken aus dem Internet herunterzuladen. Dieser Menüeintrag ist zurzeit für MS Windows nicht verfügbar.
Ctrl; S Datei Speichern
Speichert alle Änderungen an Objekten im geöffneten Fenster.
Datei Importieren Tabellendaten aus Datei...
Importiert eine Datentabelle aus einer Datei im CSV-Format (durch Komma getrennte Werte).
Datei Exportieren Tabellen- oder Abfragedaten in Datei...
Exportiert Daten aus der geöffneten Tabelle in eine Datei im CSV-Format (durch Komma getrennte Werte).
Ctrl; P Datei Drucken...
Druckt Daten der geöffneten Tabelle oder Abfrage.
Anmerkung für KDE: Überprüfen Sie, ob das richtige Drucksystem im Abschnitt Aktuelles Drucksystem: ausgewählt ist. Diese Einstellung wird angezeigt, wenn Sie auf Optionen & gt;gt; klicken.
Datei Druckvorschau...
Zeigt eine Druckvorschau der geöffneten Tabelle oder Abfrage.
Datei Seiteneinstellungen...
Schriftart festlegen... für den Seitentitel:, Ändern... von Seitengröße & Ränder und Tabellenumrandungen hinzufügen.
Datei Projekt schließen
Schließt das aktuell geöffnete Projekt ohne & kexi; zu beenden.
Ctrl; Q Datei Beenden
Beendet & kexi;.
Das Menü Bearbeiten
Ctrl; Z Bearbeiten Rückgängig
Macht eine Aktion rückgängig. Damit kommen Sie zum Zustand vor Ihrer letzten Änderung zurück.
Ctrl; & Shift; Z Bearbeiten Wiederherstellen
Stellt eine rückgängig gemachte Aktion wieder her. Damit wird die letzte Änderung wieder hergestellt.
Ctrl; X Bearbeiten Ausschneiden
Löscht den aktuell ausgewählten Inhalt eines Eingabefeldes oder einzelne Oberflächenelemente bzw. eine Gruppe im Formularentwurf und verschiebt sie in die Zwischenablage. Dieser Menüeintrag ist nur dann aktiviert, wenn Text in einem Eingabefeld oder Oberflächenelemente im Formularentwurf ausgewählt sind.
Ctrl; C Bearbeiten Kopieren
Kopiert den aktuell ausgewählten Inhalt eines Eingabefeldes oder einzelne Oberflächenelemente bzw. eine Gruppe im Formularentwurf in die Zwischenablage. Dieser Menüeintrag ist nur dann aktiviert, wenn Text in einem Eingabefeld oder Oberflächenelemente im Formularentwurf ausgewählt sind.
Ctrl; V Bearbeiten Einfügen
Fügt den Inhalt der Zwischenablage am Einfügepunkt ein. Dieser Menüeintrag ist nicht aktiviert, wenn die Zwischenablage leer ist. Wird diese Aktion im Formularentwurf benutzt und enthält die Zwischenablage Oberflächenelemente, werden diese in das Formular eingefügt.
Bearbeiten Kopieren Spezial Tabelle oder Abfrage als Datentabelle...
Kopiert die ausgewählte Tabelle oder Abfrage in die Zwischenablage.
Bearbeiten Einfügen Spezial Als Datentabelle...
Fügt Daten aus der Zwischenablage in eine neue Tabelle im aktuellen Projekt ein.
Ctrl; A Bearbeiten Alles auswählen
Wählt alle Zeichen in einem bearbeiteten Textfeld oder alle Oberflächenelemente im Formularentwurf aus.
Bearbeiten Löschen
Löscht das aktuell ausgewählte Objekt.
Ctrl; Entf Bearbeiten Zeile löschen
Löscht die ausgewählte Zeile aus einer Tabelle.

Schaltet in die Entwurfsansicht um.

Alt; 1 Ansicht Projekt-Navigator

Alt; 3 Ansicht Eigenschaften-Editor

Ansicht Eigenschaften anzeigen/ausblenden

Ansicht Projekt-Navigator anzeigen/ausblenden
Zeigt den Projekt-Navigator an bzw. blendet ihn aus.
Das Menü Einfügen
Einfügen Tabelle...
Fügt einen neuen, leeren Tabellenentwurf ein, ohne ihn zu speichern. Das Fenster Tabellenentwurf wird angezeigt.
Einfügen Abfrage...
Fügt einen neuen, leeren Abfrageentwurf ein, ohne ihn zu speichern. Das Fenster Abfrageentwurf wird angezeigt.
Einfügen Formular...
Fügt einen neuen, leeren Formularentwurf ein, ohne ihn zu speichern. Das Fenster Formularentwurf wird angezeigt.
Einfügen Skript...
Fügt einen neuen, leeren Skriptentwurf ein, ohne ihn zu speichern. Das Fenster Skriptentwurf wird angezeigt. Dieser Eintrag ist nur dann verfügbar, wenn Skripte in & kexi; aktiviert sind.
Das Menü Format
Format Schriftart...
Ändert die Schriftart für ausgewählte Objekte. Kann nur im Formularentwurf zur Änderung der Schriftart von Oberflächenelementen verwendet werden.
Format An Gitter ausrichten
Wenn diese Einstellung eingeschaltet ist, werden Oberflächenelemente auf einem Formular mit der linken oberen Ecke am nächsten Gitterpunkt angeordnet. Damit sind die Möglichkeiten zur Anordnung auf der Formularoberfläche eingeschränkt, aber die Oberfächenelement lassen sich dann genau an einer Linie ausrichten.
Format Oberflächenelement anordnen
Erstellt eine Anordnung von Oberflächenelementen. Diese können waagrecht, senkrecht und am Gitter angeordnet werden.
Format Anordnung auflösen
Löst die ausgewählte Anordnung auf.
Format Oberflächenelemente ausrichten
Richtet die Anordnung der aktuell ausgewählten Oberflächenelemente aus: Nach links, Nach rechts, Nach oben, Nach unten, An Gitter.
Format Größe der Oberflächenelemente anpassen
Passt die Größe der Oberflächenelemente an: Passend, An Gitter, An das Kürzeste, An das Längste, An das Schmalste, An das Breiteste.
Format Oberflächenelement nach vorne holen
Holt das aktuell ausgewählte Oberflächenelement nach vorne.
Format Oberflächenelement nach hinten stellen
Stellt das aktuell ausgewählte Oberflächenelement nach hinten.
Das Menü Daten
Shift; Eingabe Daten Zeile speichern
Speichert die Daten in der aktuell ausgewählten Datenzeile.
Daten Zeilenänderungen verwerfen
Verwirft alle Änderungen an den aktuell ausgewählten Tabellenzeilen.
Daten Sortieren Aufsteigend
Sortiert Daten in aufsteigender Reihenfolge (von A bis Z und von 0 bis 9). Sortiert wird nach den Daten in der aktuell ausgewählten Spalte.
Daten Sortieren Absteigend
Sortiert Daten in absteigender Reihenfolge (von Z bis A und von 9 bis 0). Sortiert wird nach den Daten in der aktuell ausgewählten Spalte.
Das Menü Extras
Extras Datenbank importieren...
Startet den Datenbank-Import-Assistenten, mit dem Sie eine vorhandene Datenbank in & kexi; importieren können.
Extras Skriptdatei ausführen...
Öffnet den Dateidialog, um ein vorhandenes Skript zu laden.
Extras Skriptverwaltung...
Öffnet & kexi;s Dialog Skriptverwaltung, in dem Sie Skripte ausführen, laden, entladen, installieren, deinstallieren und mehr Skripte holen können.
Extras Skripte
Führt bereits geladene Skripte aus.
