Text eingeben
So geben Sie direkt Zeichenketten ein, die nicht interpretiert werden
*
(ein Buchstabe mit einem hochgestellten Stern) schreiben wollen, wird der Stern als ein Multiplikationsoperator interpretiert werden.
Schließen Sie direkten Text innerhalb doppelter Anführungszeichen ein oder fügen Sie leere Gruppierungsklammen hinzu.
Beispiele:
Eine importierte MathType-Formel enthält die folgende Zeichenkette
W rSup { size 8{*} }
Falls Sie Math eingerichtet haben, dass es importierte MathType-Formeln (in Extras - Optionen - Laden/Speichern - Microsoft Office) konvertiert, sehen Sie die Formel mit einem Platzhalter anstatt des Sterns.
Ändern Sie {*} in {} * {} wie in der folgenden Formel:
W rSup { size 8{} * {} }
Sie können auch W^ &quot; * &quot; zur Eingabe der Zeichen als direkten Text benutzen.
Einige Formeln beginnen mit einen =-Zeichen.
Benutzen Sie &quot; = &quot; zur Eingabe dieses Zeichens als direkten Text.
