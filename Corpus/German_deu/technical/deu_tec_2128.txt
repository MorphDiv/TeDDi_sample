KOMBINATIONEN (Funktion)
Anzahl an Kombinationen
mw added one entry
KOMBINATIONEN
Gibt die Anzahl von Kombinationen für Elemente ohne Wiederholung zurück.
Syntax
KOMBINATIONEN(Count1; Count2)
Count1
ist die Anzahl von Elementen in der Gruppe.
Count2
ist die Anzahl von Elementen, die aus der Gruppe ausgewählt werden.
KOMBINATIONEN gibt die geordneten Möglichkeiten zur Auswahl diese Elemente zurück.
Wenn beispielsweise 3 Elemente A, B und C in einer Gruppe zusammengefasst sind, haben Sie 3 verschiedene Möglichkeiten, um 2 Elemente auszuwählen, nämlich AB, AC und BC.
KOMBINATIONEN führt folgende Formel aus:
Count1!/(Count2!*(Count1-Count2)!)
Beispiel
=KOMBINATIONEN(3;2)
ergibt 3.
see also COMBINA

mw added one entry
KOMBINATIONEN2
Gibt die Anzahl von Kombinationen einer Teilmenge von Elementen einschließlich Wiederholung zurück.
Syntax
KOMBINATIONEN2(Count1; Count2)
Count1
ist die Anzahl von Elementen in der Gruppe.
Count2
ist die Anzahl von Elementen, die aus der Gruppe ausgewählt werden.
KOMBINATIONEN2 gibt die Anzahl von eindeutigen Möglichkeiten zur Auswahl dieser Elemente zurück, wobei die Reihenfolge der Auswahl irrelevant und die Wiederholung von Elementen erlaubt ist.
Wenn beispielsweise 3 Elemente A, B und C in einer Gruppe zusammengefasst sind, haben Sie 6 verschiedene Möglichkeiten, um 2 Elemente auszuwählen, nämlich AB, BA, AC, CA, BC und CB.
KOMBINATIONEN2 führt folgende Formel aus:
(Count1+Count2-1)! / (Count2!(Count1-1)!)
Beispiel
=KOMBINATIONEN(3;2)
ergibt 6.
see also COMBIN
