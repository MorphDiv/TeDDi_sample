Datenbankinhalt; als Tabellen einfügen
Fügt die im Datenquellen-Browser ausgewählten Daten als Tabelle in das Dokument ein.
Wählen Sie im Dialog
Datenbankspalten einfügen
die Option
Tabelle
, damit die ausgewählten Daten als Tabelle in das Dokument eingefügt werden.
In diesem Dialog entscheiden Sie, welche Datenbankfelder oder -spalten übertragen und wie die Texttabelle formatiert werden soll.
Tabelle
Tabelle
wählen Sie mithilfe der Pfeilschaltflächen die Spalten der Datenbanktabelle aus, die in die Texttabelle übertragen werden sollen.
Datenbankspalten
Hier geben Sie die in die Texttabelle einzufügenden Datenbankspalten an.
Hier sind alle Spalten der Datenbanktabelle aufgeführt, die nicht in das Listenfeld
Tabellenspalte(n)
übernommen wurden.
Die Einträge sind in alphabetischer Reihenfolge aufgeführt.
Tabellenspalte(n)
Hier sind alle in das Dokument einzufügenden Spalten aus der Datenbank aufgeführt.
Jedem Eintrag in der Tabelle wird eine Spalte zugewiesen.
Die Reihenfolge der Einträge im Listenfeld
Tabelle
gibt die Reihenfolge der Daten in der Texttabelle vor.

Format
Hier können Sie das Format bestimmen, in dem die Inhalte der Datenbankfelder in das Dokument eingefügt werden sollen.
Aus Datenbank
Die Datenbankformate werden übernommen.
Auswahl
Hier wählen Sie ein Format für Datenfelder, deren eigene Formatinformationen nicht akzeptiert werden.
Die hier angebotenen Formate sind nur für bestimmte Datenbankfelder, z.
B. numerische oder Boolesche Felder, verfügbar.
Für Datenbankfelder in Textformat kann keines der Formate in der Liste gewählt werden.
In diesem Fall wird das Textformat automatisch beibehalten.
Zahlenformat

Tabellenüberschrift einfügen
Hier geben Sie an, ob eine Überschriftzeile für die Spalten in der Texttabelle eingefügt werden soll.
Spaltennamen übernehmen
Die Feldnamen der Datenbanktabelle werden als Überschrift für die einzelnen Spalten der Texttabelle verwendet.
Nur Zeile erstellen
Es wird eine leere Überschriftzeile in die Texttabelle eingefügt.
Die Option
Nur Zeile erstellen
lässt Ihnen die Möglichkeit, von den Feldnamen der Datenbank abweichende Überschriften für die Tabelle im Dokument zu wählen.
