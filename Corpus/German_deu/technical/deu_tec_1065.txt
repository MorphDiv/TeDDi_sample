Web-Assistent - Vorschau
Geben Sie an, wo Ihre Website veröffentlicht werden soll.
Außerdem können Sie eine Vorschau Ihrer Site anzeigen.
Vorschau
Öffnet Ihre Webseite im Standardwebbrowser Ihres Betriebssystems.
Lokales Verzeichnis
Lädt Ihre Indexseite und Dateien in ein lokales Verzeichnis.
Die Indexseite wird an dem von Ihnen angegebenen Speicherort gespeichert.
Alle anderen Dateien werden innerhalb des Verzeichnisses, das die Indexseite enthält, im Verzeichnis &quot; meineWebseite &quot; gespeichert.
...
Öffnet einen Dialog zum Auswählen eines Ordners.
Archivdatei
Fügt Ihre Indexseite und Dateien einer komprimierten Archivdatei hinzu und lädt diese Datei auf Ihre Website.
Die Indexseite wird an dem von Ihnen angegebenen Speicherort gespeichert.
Alle anderen Dateien werden innerhalb des Verzeichnisses, das die Indexseite enthält, im Verzeichnis &quot; meineWebseite &quot; gespeichert.
In Abhängigkeit von Ihrem Betriebssystem stehen als Archivdateiformate &quot; zip &quot; , &quot; gzip &quot; und &quot; war &quot; zur Verfügung.
...
Öffnet einen Dialog, in dem Sie den Speicherort der Archivdatei angeben können.
Das Web über FTP
Lädt Ihre Dateien auf einen FTP-Server.
Die Indexseite wird an dem von Ihnen angegebenen Speicherort gespeichert.
Alle anderen Dateien werden innerhalb des Verzeichnisses, das die Indexseite enthält, im Verzeichnis &quot; meineWebseite &quot; gespeichert.
Wenn Sie eine Internetverbindung über einen Proxy-Server herstellen, können Sie die FTP-Option nicht verwenden.
Konfigurieren
FTP-Verbindung
, in dem Sie die Verbindungseinstellungen für den FTP-Server bearbeiten und testen können.
Einstellungen speichern
Speichert die Einstellungen, die Sie in diesem Assistenten festgelegt haben.
Speichern als
Geben Sie den Namen für die Einstellungsdatei ein.
