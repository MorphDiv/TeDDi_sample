& kblocks; ist ein klassisches Tetris-Spiel. Die herunterfallenden Blöcke müssen so gestapelt werden, dass sie komplett gefüllte Zeilen bilden. Eine solche Zeile verschwindet dann, und im Spielfeld ist wieder mehr Platz. Wenn es keinen Platz mehr für herunterfallende Blöcke gibt, ist das Spiel zu Ende.
Spielanleitung
Ziel des Spiels:
Drehen und stapeln Sie die herunterfallenden Blöcke so, dass die Zeilen durchgehend gefüllt werden und sich das Spielfeld leert.
Per Voreinstellung können die herunterfallenden Blöcke mit den Pfeiltasten Links und Rechts bewegt werden. Die Pfeiltaste Runter erhöht die Fallgeschwindigkeit des Blockes, die Hoch -Taste dreht ihn. Die Tasteneinstellungen können über Einstellungen Kurzbefehle festlegen... angepasst werden.
Spielregeln, Spielstrategien und Tipps
Spielregeln
Die Regeln sind einfach: Schaffen Sie Platz für neue herunterfallende Blöcke!
Wenn ein neuer Block nicht mehr fallen kann, ist das Spiel vorbei.
Spielstrategien und Tipps
Behalten Sie das Vorschaubild im Auge: es zeigt Ihnen den nächsten Block.
Wenn Sie mehrere Zeilen auf einmal entfernen, erhalten Sie zusätzliche Punkte.
Befehle und Tastenkürzel
Im nächsten Abschnitt wird jeder Menüeintrag kurz beschrieben.
Das Menü Spiel
Das Menü Spiel besteht aus folgenden vier Einträgen.
Ctrl; N Spiel Neu
Startet ein neues Spiel.
P Spiel Pause
Hält das Spiel an oder setzt es fort.
Ctrl; H Spiel Bestenliste anzeigen
Zeigt Ihnen die höchste erreichte Punktzahl für jede Schwierigkeitsstufe.
Ctrl; Q Spiel Beenden
Beendet & kblocks;
Das Menü Einstellungen
Einstellungen Werkzeugleiste anzeigen
Blendet die Werkzeugleiste ein bzw. aus. In der Voreinstellung wird sie angezeigt.
Einstellungen Statusleiste anzeigen
Blendet die Statusleiste ein bzw. aus. In der Voreinstellung ist sie eingeblendet.
Einstellungen Schwierigkeit
Hier können Sie den Schwierigkeitsgrad aus einem Untermenü wählen.
Es gibt drei voreingestellte Schwierigkeitsstufen: Einfach (Spiel startet bei Stufe 1), Mittel (startet bei Stufe 5), und Schwer (startet bei Stufe 10).
Einstellungen Kurzbefehle festlegen...
In diesem Menüeintrag können Sie die Tastenkürzel anpassen, die von & kblocks; verwendet werden. Dazu zählt auch die Einstellung, ob Blöcke im oder gegen den Uhrzeigersinn gedreht werden.
Einstellungen Werkzeugleisten einrichten...
Öffnet den & kde;-Standarddialog zur Änderung der Werkzeugleisten für & kblocks;.
Einstellungen kblocks; einrichten...
Hier können Sie das Spiel einrichten. Weitere Informationen dazu finden Sie im Abschnitt Spieleinstellungen.

Spieleinstellungen
Um die Einstellungen für das Spiel zu ändern, öffnen Sie mit Einstellungen kblocks; einrichten... den Einrichtungsdialog.
Im Karteireiter Design können Sie das Design einstellen. Klicken Sie auf eins der verfügbaren Designs aus der Liste, um auf der rechten Seite eine Vorschau zu sehen. Sie können es über Anwenden und OK auswählen.

Wie starte ich ein neues Spiel?
Drücken Sie den Knopf Neu, wählen Sie in der Menüleiste Spiel Neu oder drücken Sie Ctrl; N.

Um das Spiel anzuhalten, drücken Sie den Knopf Pause, wählen Sie im Menü Spiel Pause oder drücken Sie P.
