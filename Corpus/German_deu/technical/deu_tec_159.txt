des Dialogs
Hyperlink
bearbeiten Sie Hyperlinks mit WWW- oder FTP-Adressen.
Die Felder für Loginname, Kennwort und Anonymer Benutzer stehen nur für FTP-Adressen zur Verfügung.
Art des Hyperlinks
Web
Erzeugt einen HTTP-Hyperlink.
FTP
Erzeugt einen FTP-Hyperlink.
Telnet
Erzeugt einen Telnet-Hyperlink.
Sprungziel
Geben Sie eine URL für die Datei ein, die beim Klick auf den Hyperlink geöffnet werden soll.
Falls Sie keinen Zielrahmen angeben, wird die Datei im aktuellen Dokument oder Rahmen geöffnet.
WWW-Browser
Öffnet einen Webbrowser, in dem Sie den gewünschten URL laden können.
Diesen URL können Sie dann kopieren und in das Feld
Sprungziel
einfügen.

Sprungziel im Dokument

Übernehmen
an den Ziel-URL anfügen.
Hier legen Sie fest, an welche Stelle im Zieldokument der Hyperlink führen soll.

in den Dialog
Hyperlink
ein.
Schließen
, um den Hyperlink so zu übernehmen und den Dialog zu schließen.
Loginname
Hier geben Sie Ihren Anmeldenamen für die Arbeit mit FTP-Adressen ein.
Kennwort
Hier geben Sie Ihr Kennwort für die Arbeit mit FTP-Adressen ein.
Anonymer Benutzer
Ermöglicht eine Anmeldung bei der FTP-Adresse als anonymer Benutzer.
Weitere Einstellungen
Frame
Geben Sie die Bezeichnung für den Rahmen ein, in dem die Zieldatei geöffnet werden soll, oder wählen Sie einen der vordefinierten Rahmen in der Liste.
Lassen Sie dieses Feld leer, so wird die Zieldatei im aktuellen Browser-Fenster geöffnet.
Form
Hier geben Sie an, ob der Hyperlink als Text oder Schaltfläche eingefügt werden soll.

Name
Geben Sie einen Namen für den Hyperlink ein.
