Kill-Anweisung [Laufzeit]
Löscht eine Datei von einem Datenträger.
Syntax:
Kill Datei As String
Parameter
:
Datei:
Ein beliebiger Zeichenkettenausdruck, der eine eindeutige Dateiangabe enthält.
Sie können auch die
URL-Notation
verwenden.
Beispiel:
sub ExampleKill
Kill &quot; C:\datafile.dat &quot; REM Datei sollte vorher angelegt werden
end sub
