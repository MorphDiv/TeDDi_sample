Der Einrichtungsdialog von kspread;
Wenn Sie Einstellungen kspread; einrichten... wählen, wird ein Dialog mit mehreren Seiten geöffnet. Klicken Sie auf ein Symbol links im Dialog, um eine Seite auszuwählen. Auf diesen Seiten können Sie viele Einstellungen von & kspread; ändern.
Regionaleinstellungen
Einrichtung der Regionaleinstellungen
Diese Seite des Einrichtungsdialogs von & kspread; zeigt, in welchem Format Daten wie Zahlen, Datum, Zeit und Währung angezeigt werden.
Wenn Sie eine Tabelle geladen haben, die mit einer anderen Regionaleinstellung erzeugt wurde, ändert ein Klick auf Regionaleinstellungen aktualisieren die Einstellungen der Tabelle auf die Regionaleinstellungen aus dem & kcontrolcenter;.
Oberfläche
Einrichtung der Oberfläche
Auf dieser Seite des Einrichtungsdialogs von & kspread;, den Sie mit kspread; einrichten... im Menü Einstellungen aufrufen, können Sie weitere Eigenschaften von & kspread; einstellen:
Anzahl der am Anfang geöffneten Tabellen:
Legt die Anzahl der Tabellen fest, wenn Sie & kspread; mit der Option Leeres Arbeitsblatt starten.
Anzahl der Einträge in der Liste der zuletzt geöffneten Dateien:
Bestimmt die maximale Anzahl der Dateinamen, die im Menü Datei Zuletzt geöffnete Dateien angezeigt werden.
Automatische Sicherung (Min.):
Hier können Sie den Zeitraum zwischen automatischen Sicherungen festlegen oder die automatische Sicherung ganz ausschalten. Dazu schieben Sie den Schieberegler ganz nach links, bis im Fenster Nicht automatisch sichern angezeigt wird.
Sicherungskopien erstellen
Aktivieren Sie diese Einstellung, wenn Sicherungskopien erstellt werden sollen. Standardmäßig ist diese Einstellung eingeschaltet.
Senkrechte Bildlaufleiste anzeigen
Aktivieren bzw. deaktivieren Sie diese Einstellung, um die senkrechte Bildlaufleiste in allen Tabellen anzuzeigen oder auszublenden.
Waagerechte Bildlaufleiste anzeigen
Aktivieren bzw. deaktivieren Sie diese Einstellung, um die waagerechte Bildlaufleiste in allen Tabellen anzuzeigen oder auszublenden.
Spaltenüberschrift anzeigen
Aktivieren Sie diese Einstellung, um die Spaltenbuchstaben als Kopfzeile über allen Tabellen anzuzeigen bzw. auszublenden.
Zeilenbeschriftung anzeigen
Aktivieren Sie dieses Feld, um die Zeilennummern auf der linken Seite der Tabelle anzuzeigen.
Reiter anzeigen
Mit diesem Ankreuzfeld legen Sie fest, ob die Kartenreiter der Tabellen am unteren Rand des Arbeitsblattes angezeigt werden.
Formelleiste anzeigen
Hier können Sie wählen, ob die Formel leiste angezeigt oder ausgeblendet werden soll.
Statusleiste anzeigen
Deaktivieren Sie diese Einstellung, um die Statusleiste auszublenden.
Diverses
Einrichtung für Diverses
Die Seite Diverses des Dialogs kspread; einrichten enthält die folgenden Einstellungsmöglichkeiten:
Vervollständigungsmodus:
Hier können Sie den Modus der Vervollständigung von Text aus einer Reihe von Möglichkeiten im Auswahlfeld einstellen.
Drücken der Eingabetaste bewegt den Zellcursor nach:
Wenn Sie eine Zelle ausgewählt haben und danach die Taste Eingabe drücken, wird der Cursor Nach unten, Nach oben, nach Rechts, nach Links oder Nach unten, erste Spalte verschoben, je nachdem, was Sie hier ausgewählt haben.
Berechnungsmethode:
Hier können Sie wählen, welche Berechnung standardmäßig vom Ergebnisfeld in der Statuszeile angezeigt wird.
Einrückungsschritt (cm):
Hier können Sie die Breite der Einrückung einstellen, die auf Zellen angewandt wird, wenn Sie Einrückung vergrößern im Menü Format wählen.
Hier wird die Einheit benutzt, die im Einrichtungsdialog von & kspread; im Seitenlayout als Standard-Längeneinheit: eingestellt ist.
Bei ungültigen Formeln Fehlermeldung anzeigen
Wenn dieser Eintrag aktiviert ist, wird ein Fenster mit einer Fehlermeldung angezeigt, wenn Sie eine ungültige Formel eingeben, die von & kspread; nicht ausgewertet werden kann.
Farbe
Farbeinstellung
Auf dieser Seite des Einrichtungsdialogs von & kspread; können Sie die Farbe des Gitterlinien in der Tabelle wählen. Wenn das Gitter nicht angezeigt werden soll, deaktivieren Sie Gitter anzeigen im Dialog Format Tabelle Tabellen-Eigenschaften.
Hier können Sie auch die Farbe der Linien wählen, die in der Tabelle die Seitenränder der gedruckten Seiten anzeigen, wenn Seitenränder anzeigen im Menü Ansicht ausgewählt ist.
Klicken Sie auf das Farbfeld, um den Standarddialog von KDE zur Farbauswahl zu öffnen.
Gitterfarbe:
Klicken Sie hier, um die Gitterfarbe, d. h. die Farbe der Ränder aller Zellen, auszuwählen.
Seitenränder:
Wenn im Menü Ansicht Seitenränder anzeigen ausgewählt ist, werden die Seitenränder angezeigt. Klicken Sie auf das Farbfeld, um für die Ränder eine andere Farbe als das voreingestellte Rot auszuwählen.
Seitenlayout
Einstellung des Seitenlayout
In dieser Seite des Dialogs kspread; einrichten können Sie die Standard-Seitengröße, die Seitenausrichtung und die Längeneinheit einstellen, die vom Drucker und im Dialog Seitenlayout, das Sie über SeitenLayout... im Menü Format erreichen, verwendet werden.
Standard-Seitengröße:
Wählen Sie eine Standard-Seitengröße für Ihre Tabellen aus einer Liste gebräuchlicher Formate. Beachten, das Sie die Seitengröße für die aktuelle Tabelle im Dialog Format Seitenlayout... überschreiben können.
Standard-Seitenausrichtung:
Wählen Sie die Standard-Seitenausrichtung: Hochformat oder Querformat. Beachten, das Sie die Seitenausrichtung für die aktuelle Tabelle im Dialog Format Seitenlayout... überschreiben können.
Standard-Längeneinheit:
Wählen Sie die Standard-Längeneinheit für alle Tabellen. Beachten, das Sie die Längeneinheit für die aktuelle Tabelle im Dialog Format Seitenlayout... überschreiben können.
Rechtschreibung
Einstellung der Rechtschreibung
Auf dieser Seite können Sie das Verhalten der Rechtschreibprüfung in & kspread; einstellen.
Kombination aus Wortstamm+Zusätzen erstellen, die nicht im Wörterbuch stehen
Wenn diese Einstellung ausgewählt ist, wird die Rechtschreibprüfung in & kspread; ein unbekanntes Wort im Dokument, das aber aus einem bekannten Stammwort und bekannter Vor- oder Nachsilbe besteht, als richtig erkannt. Ansonsten wird es dagegen von der Rechtschreibprüfung als falsch erkannt.
Zusammengezogene Wörter als Rechtschreibfehler betrachten
Wenn dieses Feld markiert ist, werden normale Wörter, zwischen denen das Leerzeichen fehlt, als Rechtschreibfehler betrachtet, z. B. kannnicht.
Wörterbuch:
Hier können alternative Wörterbücher gewählt werden.
Kodierung:
Hier wählen Sie die verwendete Kodierung der Zeichen.
Programm:
Mit diesem Auswahlfeld können Sie zwischen verschiedenen Programmen zur Rechtschreibprüfung wählen, die auf Ihrem Computer vorhanden sind.
Durchgehend groß geschriebene Wörter überspringen
Aktivieren Sie dieses Feld, wenn komplett groß geschriebene Worte ignoriert werden sollen. Dies sind normalerweise Akronyme wie z. B. & kde;.
Die Groß-/Kleinschreibung des Titels nicht überprüfen
Aktivieren Sie diese Einstellung, wenn die Rechtschreibprüfung die Groß-/Kleinschreibung des Titels ignorieren soll. Ist diese Einstellung deaktiviert, erwartet die Rechtschreibprüfung Großbuchstaben für Nomina im Titel.
