Adress-Datenquelle
Dieser Assistent meldet ein bestehendes Adressbuch als eine Datenquelle bei $[officename] an.
Sie können Adressdaten und andere Datenquellen jederzeit in $[officename] anmelden:
Bitte wählen Sie den Typ Ihres externen Adressbuches
Nicht alle Typen sind auf allen Systemen verfügbar.
Mozilla / Netscape
Wählen Sie diese Option, wenn Sie bereits in Mozilla oder Netscape mit einem Adressbuch arbeiten.
Thunderbird
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Thunderbird verwenden.
LDAP Adress-Bestand
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch auf einem LDAP Server verwenden.
Outlook Adressbuch
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Microsoft Outlook (nicht Outlook Express).
Windows System-Adressbuch
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Microsoft Outlook Express nutzen.
KDE Adressbuch
Wählen Sie diese Option, wenn Sie bereits ein KDE-Adressbuch nutzen.
OS X Adressbuch
Wählen Sie diese Option, wenn Sie bereits ein OS X Adressbuch nutzen.
Evolution
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Evolution nutzen.
Evolution LDAP
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Evolution LDAP.
Groupwise
Wählen Sie diese Option, wenn Sie bereits ein Adressbuch in Groupwise nutzen.
andere externe Datenquelle
Wählen Sie diese Option, wenn Sie jetzt eine andere Datenquelle als Adressbuch in $[officename] anmelden wollen.
