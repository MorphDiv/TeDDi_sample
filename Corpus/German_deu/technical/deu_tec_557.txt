Linienstile
Sie können gepunktete oder gestrichelte Linienstile bearbeiten oder erstellen.
Eigenschaften
Linienstil
Geben Sie den gewünschten Linienstil an.
Typ
Wählen Sie eine Kombination aus Punkten und Strichen.
Anzahl
Geben Sie an, wie viele Punkte und Striche hintereinander erscheinen sollen.
Länge
Geben Sie eine Strichlänge ein.
Abstand
Geben Sie an, wie groß der Abstand zwischen den einzelnen Punkten oder Strichen sein soll.
An Linienbreite anpassen
Passt die Einträge automatisch an die Zeilenlänge an.
Hinzufügen...
Erzeugt einen Linienstil mit den aktuellen Einstellungen.
Name
Geben Sie einen Namen für den neuen Linienstil ein.
Ändern...
Überträgt die aktuellen Einstellungen auf den ausgewählten Linienstil.
Zum Umbenennen des ausgewählten Linienstils geben Sie bei Aufforderung einen neuen Namen ein.
Linienstiltabelle laden
Importiert eine Linienstiltabelle.
Linienstiltabelle speichern
Speichert die aktuelle Linienstiltabelle, damit sie künftig wieder eingeladen werden kann.
