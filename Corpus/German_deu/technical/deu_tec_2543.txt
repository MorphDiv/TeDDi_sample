Uniform Resource Identifier.
Eine standardisierte M\xF6glichkeit, eine Ressource (also Datei auf Ihrem Rechner, eine WWW-Adresse, eine E-Mail-Adresse usw.) zu benennen.
Internet-Stichw\xF6rter machen es Ihnen einfacher, im Internet Markennamen, ber\xFChmte Pers\xF6nlichkeiten etc. zu finden.
Zum Beispiel kann man, statt http://www.kde.org / einzugeben, einfach KDE oder auch K Desktop Environment eintippen.
Der eingegebene Name wird dann in eine URI \xFCbersetzt und man wird an die entsprechende Stelle im Internet weitergeleitet.
Web-Kurzbefehle sind benutzerdefinierte Pseudo- URI s, die z.B. als Abk\xFCrzung f\xFCr das Abfragen bestimmter Suchmaschinen dienen k\xF6nnen und den Umgang mit den verschiedenen Suchparametern dieser viel benutzten Suchmaschinen vereinfachen.
Wenn man zum Beispiel die Google-Suchmaschine benutzen will, kann man KDE so einrichten, dass die Eingabe einer Pseudo- URL wie gg eine Suche bei Google ausl\xF6st.
Es ist leicht ersichtlich, warum diese Eingaben Pseudo- URL genannt werden.
Sie werden zwar wie eine URL benutzt, aber die Eingabe ist nicht im korrekten URL -Format gehalten, man gibt zum Beispiel google:kde apps und nicht google:kde+apps ein.
Man kann Internet-Stichw\xF6rter in KDE in verschiedenen Anwendungen wie zum Beispiel Konqueror benutzen.
Man kann die Web-Kurzbefehle generell anstelle von URI s verwenden.
Die Kurzbefehle f\xFCr viele Suchmaschinen sind bereits vorkonfiguriert, aber man kann nach Belieben neue Kurzbefehle definieren und bestehende \xE4ndern oder l\xF6schen.
All dies geschieht im Kontrollzentrum im Modul Erweitertes Web-Browsen.

Internet-Stichw\xF6rter

Nur wenn dieses Feld markiert ist, kann man Internet-Stichw\xF6rter in KDE verwenden.

Diese Zeichensequenz ist ein Platzhalter, und wird durch dasjenige ersetzt, was bei einer Suchanfrage hinter dem Doppelpunkt steht.
Damit dieser Teil etwas klarer wird, sollte man sich die folgenden Beispiele ansehen.
Angenommen, dass bei Such-URI steht: http://www.google.com/search?q=\{@} und dass gg ein Kurzbefehl f\xFCr diese URI ist.

Nur der \{@} -Teil der Such-URI wird konvertiert.
Vom Rest wird von vornherein angenommen, dass er sich bereits in einem korrekten URL -codierten Format befindet.
Man kann auch Kurzbefehle ohne Parameter erstellen.
Angenommen, die URI ist file:/home/ich/meinedocs/kofficedateien/kword und der dazugeh\xF6rige Kurzbefehl ist meinkword.
Dann f\xFChrt die Eingabe von meinkword: zum selben Ergebnis wie die komplette URI.
Es ist zu beachten, dass man zwar nichts hinter dem Doppelpunkt eingeben muss, dieser aber trotzdem notwendig ist, damit die Eingabe als Kurzbefehl akzeptiert wird.
Sp\xE4testens jetzt sollte klar sein, dass die Kurzbefehle nicht nur f\xFCr Suchmaschinen, sondern f\xFCr alle Arten von parametrisierten URI s verwendet werden k\xF6nnen.
Diese k\xF6nnen nicht nur auf Internetseiten sondern auf alles verweisen, worauf man mit einer URI verweisen kann.
