Java-Unterstützung
$[officename] unterstützt Java-Plattform zur Ausführung von Anwendungen und Komponenten die auf der JavaBeans-Architektur basieren.
Voraussetzung für die Unterstützung von Java durch $[officename] ist, dass die Software Java 2 Runtime Environment von Sun Microsystems auf dem System installiert ist.
Werden diese Dateien bei der Installation von $[officename] nicht auf dem System gefunden, so wird Ihnen automatisch die Gelegenheit geboten, sie zu installieren.
Falls erforderlich, können Sie diese Dateien auch jetzt installieren.
Die ausführbare Installationsdatei finden Sie im Ordner $[officename]/program und auf der $[officename]-CD.
Der Name dieser Version für Windows lautet &quot; j2re-1_4_1_01-windows-i586-i.exe &quot; .
Für andere Betriebssysteme ändert sich der Name der Datei entsprechend.
Um Java-Anwendungen ausführen zu können, muss die Java-Unterstützung in $[officename] aktiviert werden.
Extras - Optionen - $[officename] - Sicherheit
.
Bevor Sie einen JDBC-Treiber verwenden können, müssen Sie dessen Klassenpfad hinzufügen.
Wählen Sie &quot; Extras - Optionen - %PRODUCTNAME- Java &quot; , und klicken Sie auf die Schaltfläche &quot; Class Path &quot; .
Nachdem Sie die Pfadinformationen hinzugefügt haben, starten Sie %PRODUCTNAME neu.
