Sicherheitsstufe
Makrosicherheit
sstufe aus unter vier Optionen aus.
Die Optionen unterscheiden sich in ihrer Sicherheitsstufe.
Makros, die bei einer höheren Sicherheitsstufe ausgeführt werden dürfen, dürfen ebenfalls bei allen niedrigeren Stufen ausgeführt werden.
Sehr hoch
Nur Makros von vertrauenswürdigen Dateispeicherorten dürfen ausgeführt werden.
Alle anderen Makros werden, unabhängig davon, ob sie signiert sind oder nicht, deaktiviert.
Vertrauenswürdige Dateispeicherorte können auf dem Register &quot; Vertrauenswürdige Quellen &quot; festgelegt werden.
Jedes Makro, das von einem vertrauenswürdigen Dateispeicherort stammt, darf ausgeführt werden.
Hoch
Nur signierte Makros von vertrauenswürdigen Quellen dürfen ausgeführt werden.
Nicht signierte Makros werden deaktiviert.
Vertrauenswürdige Quellen können auf dem Register &quot; Vertrauenswürdige Quellen &quot; festgelegt werden.
Nur signierte Makros von einer vertrauenswürdigen Quelle dürfen ausgeführt werden.
Außerdem darf jedes Makro, das von einem vertrauenswürdigen Dateispeicherort stammt, ausgeführt werden.
Mittel
Bestätigung erforderlich, bevor Makros von unbekannten Quellen ausgeführt werden.
Vertrauenswürdige Quellen können auf dem Register &quot; Vertrauenswürdige Quellen &quot; festgelegt werden.
Signierte Makros von einer vertrauenswürdigen Quelle dürfen ausgeführt werden.
Außerdem darf jedes Makro, das von einem vertrauenswürdigen Dateispeicherort stammt, ausgeführt werden.
Alle anderen Makros müssen von Ihnen durch Bestätigung genehmigt werden.
Niedrig (nicht empfohlen)
Alle Makros werden ohne Bestätigung ausgeführt.
Verwenden Sie diese Einstellung nur, wenn Sie sicher sind, dass alle Dokumente, die geöffnet werden, sicher sind.
Ein Makro kann zum automatischen Starten eingerichtet werden, und es kann potentiell schädliche Aktionen wie z.
B. das Löschen oder Umbenennen von Dateien ausführen.
Diese Einstellung wird nicht empfohlen, wenn Sie Dokumente von anderen Autoren öffnen.
