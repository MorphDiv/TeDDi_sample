Kombinieren von nummerierten Listen
Sie können nummerierte Listen in eine gemeinsame fortlaufend nummerierte Liste kombinieren.
So kombinieren Sie aufeinander folgende nummerierte Listen:
Wählen Sie alle Absätze in den Listen aus.
Format
klicken Sie zweimal auf das Symbol
Nummerierung an/aus
So erstellen Sie eine nummerierte Liste aus nicht aufeinander folgenden Absätzen:
Halten Sie Strg gedrückt und ziehen Sie eine Auswahl im ersten nummerierten Absatz.
Es reicht, wenn Sie ein einzelnes Zeichen auswählen.
Halten Sie Strg weiterhin gedrückt und ziehen Sie eine Auswahl in jedem nummerierten Absatz der Listen, die Sie kombinieren möchten.
Format
klicken Sie zweimal auf das Symbol
Nummerierung an/aus
