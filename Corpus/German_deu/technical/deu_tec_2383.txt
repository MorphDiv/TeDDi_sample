Einstellungen
Hier kann man Werte f\xFCr verschiedene Zeitlimits angeben.
Man kann diese Werte anpassen, falls die Verbindung sehr langsam wird.
Die Standardwerte sollten allerdings in den allermeisten F\xE4llen angemessen sein.
Die Zeitlimit-Werte geben die Zeitspannen an, die eine Anwendung auf bei einem Netzwerkzugriff auf eine Antwort warten soll.
Man kann die folgenden Zeitlimits festlegen:
Socket-Daten
Legt fest, wie lange auf Socket-Daten gewartet wird.
Proxy-Verbindung
Legt fest, wie lange auf die Verbindung zum Proxy gewartet wird, falls ein Proxy eingerichtet ist.
Server-Verbindung
Legt die Wartezeit f\xFCr die Verbindung zu einem entfernten Server fest.
Server-Antwort
Legt die Wartezeit f\xFCr die Antwort eines entfernten Servers fest.
Hier kann man FTP-Optionen einrichten.
Zum augenblicklichen Zeitpunkt gibt es nur eine Einstellm\xF6glichkeit, n\xE4mlich Passven Modus aktivieren (PASV).
Passives FTP ist hinter einem Firewall h\xE4ufig notwendig.
Viele Firewalls erlauben nur Verbindungen, die vom inneren Netzwerk ausgehen.
Passives FTP wird vom Client kontrolliert, was die Verwendung durch einen Firewall erm\xF6glicht.
