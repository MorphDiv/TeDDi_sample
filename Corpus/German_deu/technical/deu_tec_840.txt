Start-up Wizard
Der Start-up Wizard wird geöffnet, wenn Sie %PRODUCTNAME zum ersten Mal starten.
Der Assistent besteht aus folgenden Seiten:
Willkommen
Der Start-up Wizard hilft Ihnen dabei, die Benutzereinstellungen für %PRODUCTNAME anzugeben.
Dieser Assistent wird nur einmal ausgeführt.
Lizenz
Lesen und akzeptieren Sie den Lizenztext.
Sie müssen einen Bildlauf bis zum Ende des Lizenztexts durchführen, bevor Sie fortfahren können.
Wenn Sie die Lizenz ablehnen, wird die Installation abgebrochen.
Überführung
Auf dieser Seite können Sie Benutzerdaten aus einer zuvor installierten Version von %PRODUCTNAME importieren.
Persönliche Daten kopieren
Übernimmt (kopiert) den Benutzernamen sowie andere Daten aus einer vorherigen Installation von %PRODUCTNAME.
Benutzername
Der Benutzername wird für die Dokumenteigenschaften, Vorlagen und bei der Aufzeichnung von Änderungen am Dokument verwendet.
Diese Seite wird nur angezeigt, wenn auf dem Computer keine vorherige Installation von %PRODUCTNAME gefunden wird.
Geben Sie Ihre persönlichen Daten in die Textfelder ein.
