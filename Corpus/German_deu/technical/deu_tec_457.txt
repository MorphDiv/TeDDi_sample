ImageMap-Editor
Hier können Sie speziellen Bereichen, den so genannten Hotspots, in einer Grafik oder einer Gruppe von Grafiken URLs zuweisen.
Eine ImageMap ist eine Gruppe von Hotspots.
Sie können drei Arten von Hotspots zeichnen:
Rechtecke, Ellipsen und Polygone.
Beim Klicken auf einen Hotspot wird die URL in dem von Ihnen angegebenen Browserfenster oder Frame geöffnet.
Außerdem können Sie einen Text eingeben, der angezeigt wird, während der Mauszeiger über dem Hotspot liegt.
