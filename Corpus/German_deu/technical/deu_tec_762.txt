Java-Startparameter
Mit diesem Dialog können Sie optionale Startparameter für die Java-Laufzeitumgebung (JRE) eingeben.
Die in diesen Dialog eingegebenen Einstellungen gelten für alle JREs, die Sie aufrufen.
Java-Startparameter
Geben Sie JRE-Startparameter genau wie auf der Befehlszeile ein.
Klicken Sie auf &quot; Zuweisen &quot; , um den Parameter zur Liste verfügbarer Startparameter hinzuzufügen.
Verwenden Sie in Pfadnamen keine Escape- oder Anführungszeichen.
Damit der Systemparameter &quot; myprop &quot; beispielsweise auf einen Ordner zeigt, geben Sie den folgenden Parameter ein:
-Dmyprop=c:\Programme\java
Zum Aktivieren von Debugging-Funktionen in JRE müssen Sie die folgenden Parameter eingeben:
-Xdebug
-Xrunjdwp:transport=dt_socket,server=y,address=8000
Die Änderungen werden nach einem Neustart von %PRODUCTNAME wirksam.
