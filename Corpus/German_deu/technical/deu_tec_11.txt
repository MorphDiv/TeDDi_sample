Ändern von Diagrammdaten
Wenn Sie ein auf Standarddaten basierendes Diagramm erstellen oder ein Diagramm in ein Dokument kopieren, können Sie den Dialog &quot; Datentabelle &quot; öffnen und eigene Daten eingeben.
Die eingegebenen Daten werden in einer Live-Vorschau im Diagramm dargestellt.
Bearbeiten - Rückgängig
können Sie die Änderungen verwerfen.
Fügen Sie ein Diagramm ein, das nicht auf vorhandenen Zelldaten basiert, oder wählen Sie eins aus.
Ansicht - Diagrammdatentabelle
, um den Dialog &quot; Datentabelle &quot; aufzurufen.
Die Datenreihen werden in Spalten angeordnet.
Als Rolle für die Spalte ganz links sind Kategorien oder Datenbeschriftungen festgelegt.
Der Inhalt der Spalte ganz links ist immer als Text formatiert.
Klicken Sie im Dialog auf eine Zelle und ändern Sie den Inhalt.
Wenn Sie auf eine andere Zelle klicken, wird der geänderte Inhalt in der Vorschau angezeigt.
Geben Sie den Namen der Datenreihe in das Textfeld über der Spalte ein.
Mit den Symbolen über der Tabelle können Sie Zeilen und Spalten einfügen und löschen.
Bei Datenreihen mit mehreren Spalten können nur ganze Datenreihen eingefügt oder gelöscht werden.
Datenreihen nach rechts verschieben
können Sie die aktuelle Spalte mit der Spalte rechts daneben vertauschen.
Zeile nach unten verschieben
können Sie die aktuelle Zeile mit der darunter liegenden Zeile vertauschen.
insert row
Fügt unter der aktuellen Zeile eine neue Zeile ein.
insert column
Fügt hinter der aktuellen Spalte eine neue Spalte ein.
delete row
Löscht die aktuelle Zeile.
Die Beschriftungszeile kann nicht gelöscht werden.
delete column
Löscht die aktuelle Spalte.
Die Beschriftungsspalte kann nicht gelöscht werden.
move series right
Vertauscht die aktuelle Spalte mit der Spalte rechts daneben.
move row down
Vertauscht die aktuelle Zeile mit der darunter liegenden Zeile.
