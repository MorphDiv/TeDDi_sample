Join-Funktion [Laufzeit]
Gibt aus einer Anzahl von Unterzeichenketten in einem Zeichenketten-Array eine einzige Zeichenkette zurück.
Syntax:
Join (Text As String Array, Trenner)
Rückgabewert:
String
Parameter:
Text:
Ein Zeichenketten-Array.
Trenner (optional):
Trennerzeichen zwischen den einzelnen Textabschnitten in der Ergebniszeichenkette.
Standardmäßig wird das Leerzeichen verwendet.
Ist der Trenner eine leere Zeichenkette ( &quot; &quot; ), so werden die Unterzeichenketten ohne Trenner aneinandergefügt.
