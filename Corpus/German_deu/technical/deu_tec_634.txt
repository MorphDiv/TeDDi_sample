ADO-Verbindung
Legt die Optionen für das Hinzufügen einer ADO-Datenbank (Microsoft ActiveX Data Objects) fest.
Die ADO-Schnittstelle ist ein proprietärer Microsoft Windows-Container zum Herstellen einer Verbindung mit Datenbanken.
$[officename] benötigt die Microsoft Data Access Components (MDAC), um die ADO-Schnittstelle verwenden zu können.
Microsoft Windows 2000 und XP enthalten diese Komponenten standardmäßig.
Bei älteren Versionen von Windows müssen Sie MDAC separat installieren.
Sie können MDAC von der Microsoft-Website herunterladen.
Datenquellen-URL
Geben Sie die Datenquellen-URL ein.
Beispiel-URLs
Um eine Verbindung mit einer Access 2000-Datei herzustellen, verwenden Sie das Format:
PROVIDER=Microsoft.Jet.OLEDB.4.0;DATA SOURCE=c:\Access\nwind2000.mdb
Um mit einem Namen eine Verbindung mit einem Katalog auf einem Microsoft SQL-Server herzustellen, der einen Namen &quot; Turner &quot; enthält, geben Sie Folgendes ein:
PROVIDER=sqloledb;DATA SOURCE=Turner;INITIAL CATALOG=First
So greifen Sie auf einen ODBC-Treiber als Anbieter zu:
DSN=SQLSERVER
