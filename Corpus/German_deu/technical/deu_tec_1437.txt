Online Update
Legt Optionen für automatische Benachrichtigung und Downloads von Online-Updates für %PRODUCTNAME fest.
Automatisch nach Updates suchen
Markieren Sie diese Option, um regelmäßig nach Online-Updates zu suchen, und wählen Sie dann das Zeitintervall, in dem %PRODUCTNAME nach Online-Updates suchen soll.
%PRODUCTNAME führt die Prüfung wahlweise täglich, wöchentlich oder monatlich aus, sobald eine funktionierende Internetverbindung besteht.
Wenn Sie die Verbindung zum Internet über einen Proxy-Server herstellen, geben Sie diesen unter
Extras - Optionen - Internet - Proxy
an.
Sobald ein Update zur Verfügung steht, werden über ein Symbol in der Menüleiste einige Erläuterungen angezeigt.
Klicken Sie zum Fortfahren auf das Symbol.
Wenn Sie die Überprüfung ausschalten, wird das entsprechende Symbol von der Menüleiste entfernt.
Die Online-Aktualisierung ist ein Modul, das bei der Installation ausgewählt werden kann.
Wählen Sie die benutzerdefinierte Installation im Setup von %PRODUCTNAME.
Täglich
Prüfung wird einmal täglich durchgeführt.
Wöchentlich
Eine Prüfung wird einmal wöchentlich durchgeführt.
Dies ist die Standardeinstellung.
Monatlich
Prüfung wird einmal pro Monat durchgeführt.
Jetzt suchen
Prüfung wird jetzt durchgeführt.

Updates automatisch downloaden
Wählen Sie diese Option, wenn eine verfügbare Online-Update-Datei automatisch in den angegebenen Ordner heruntergeladen werden soll.
Downloadverzeichnis
Zeigt den zum Speichern der heruntergeladenen Dateien ausgewählten Ordner an.
Ändern
Klicken Sie, um einen Dialog zur Auswahl eines anderen Verzeichnisses aufzurufen.
