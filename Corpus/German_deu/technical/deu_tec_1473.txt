UFI: new file see i68756
Es gibt verschiedene Möglichkeiten, eine Bildschirmpräsentation zu starten.
Sobald eine Bildschirmpräsentation gestartet wurde, können Sie sie mithilfe der Tastatur oder der Maustasten steuern.
In der Standardeinstellung startet eine Bildschirmpräsentation mit der ersten Folie.
Dann durchlaufen Sie manuell alle Folien bis zu letzten Folie.
Sie können diese Einstellungen ändern.
