Diagramm-Assistenten
können Sie auswählen, welche Diagrammelemente angezeigt werden.
titles title
Geben Sie einen Titel für das Diagramm ein.
subtitle
Geben Sie einen Untertitel für das Diagramm ein.
x axis
Geben Sie eine Beschriftung für die X-Achse (horizontal) ein.
y axisx axis
Geben Sie eine Beschriftung für die Y-Achse (vertikal) ein.
z axis
Geben Sie eine Beschriftung für die Z-Achse ein.
Diese Option steht nur für 3D-Diagramme zur Verfügung.
legend
Zeigt eine Legende im Diagramm an.
legend left
Diese Option wählen Sie, wenn die Legende links neben dem Diagramm zu sehen sein soll.
top
Diese Option wählen Sie, wenn die Legende über dem Diagramm zu sehen sein soll.
right
Diese Option wählen Sie, wenn die Legende rechts neben dem Diagramm zu sehen sein soll.
bottom
Diese Option wählen Sie, wenn die Legende unter dem Diagramm zu sehen sein soll.
grid x axis
Blendet senkrecht zur X-Achse verlaufende Gitterlinien ein.
y axis
Blendet senkrecht zur Y-Achse verlaufende Gitterlinien ein.
z axis
Blendet senkrecht zur Z-Achse verlaufende Gitterlinien ein.
Diese Option steht nur für 3D-Diagramme zur Verfügung.
secondary x axis ( &quot; 2nd x axis &quot; in spec doc)
Geben Sie eine Beschriftung für die zweite X-Achse ein.
Diese Option ist nur für Diagramme verfügbar, die eine zweite X-Achse unterstützen.
secondary y axis ( &quot; 2nd y axis &quot; in spec doc)
Geben Sie eine Beschriftung für die zweite Y-Achse ein.
Diese Option ist nur für Diagramme verfügbar, die eine zweite Y-Achse unterstützen.
So geben Sie Diagrammelemente ein
Geben Sie Titel ein oder klicken Sie auf die Elemente, die im aktuellen Diagramm angezeigt werden sollen.
Titel
Wenn Sie Text für einen Titel, Untertitel oder eine Achse eingeben, bleibt neben dem Diagramm ausreichend Platz für die Textanzeige frei.
Wenn Sie keinen Text eingeben, ist dies nicht der Fall, so dass mehr Platz für die Anzeige des Diagramms zur Verfügung steht.
Der Titeltext kann nicht mit einer Zelle verknüpft werden.
Sie müssen den Text direkt eingeben.
Format
festlegen.
Legende
Als Legende werden die Beschriftungen aus der ersten Zeile oder Spalte oder aus dem Bereich, den Sie im Dialog &quot; Datenreihe &quot; festgelegt haben, angezeigt.
Wenn das Diagramm keine Beschriftungen enthält, wird als Legende &quot; Zeile 1, Zeile 2, ... &quot; oder &quot; Spalte A, Spalte B, ... &quot; angezeigt, je nach Zeilennummer oder Spaltenbuchstabe der Diagrammdaten.
Sie können den Text nicht direkt eingeben, denn er wird automatisch aus dem Namenszellenbereich generiert.
Wählen Sie eine der Positionsoptionen.
In einem fertigen Diagramm können Sie weitere Positionen über das Menü &quot; Format &quot; festlegen.
Gitter
grid x axis
Blendet senkrecht zur X-Achse verlaufende Gitterlinien ein.
y axis
Blendet senkrecht zur Y-Achse verlaufende Gitterlinien ein.
z axis
Blendet senkrecht zur Z-Achse verlaufende Gitterlinien ein.
Diese Option steht nur für 3D-Diagramme zur Verfügung.
Die sichtbaren Gitterlinien unterstützen Sie beim Einschätzen der Datenwerte im Diagramm.
Der Abstand der Gitterlinien entspricht dem im Register &quot; Skalierung &quot; der Achseneigenschaften eingestellten Intervall.
Gitterlinien können für Kreisdiagramme nicht ausgewählt werden.
Weitere Elemente
Weitere Elemente können Sie über das Menü &quot; Einfügen &quot; des Diagramms im Bearbeitungsmodus hinzufügen.
Folgende Elemente können Sie auswählen:
Sekundäre Achsen
Hilfsgitter
Datenbeschriftung
Statistische Angaben, wie zum Beispiel Mittelwerte, Y-Fehlerbalken und Trendlinien
