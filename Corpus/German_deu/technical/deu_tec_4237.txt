(PHP 4)
Wenn mode FALSE / falsch ist, wird die angegebene Socket - Bezeichnung auf Nicht-Block-Modus gesetzt, anderenfalls auf Block-Modus.
Dies beeinflusst Aufrufe von fgets(), die über diesen Sockets lesen.
Im Nicht-Block-Modus wird ein fgets() -Aufruf sofort zum Script zurückkehren, während er im Block-Modus auf eingehende Daten (auf diesem Socket) warten wird.
Hinweis::
Diese Funktion hiess früher set_socket_blocking(), dieser Name sollte jedoch nicht länger verwendet werden.
