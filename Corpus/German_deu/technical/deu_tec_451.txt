Ebenen; Definition
Über Ebenen
Ebenen stehen Ihnen in $[officename] Draw, nicht aber in $[officename] Impress zur Verfügung.
Mit den Ebenen können Sie verwandte Elemente auf einer Seite anordnen.
Stellen Sie sich Ebenen als einzelne Arbeitsbereiche vor, die aus der Ansicht oder für das Drucken ausgeblendet oder gesperrt werden können.
Controls
, die stets vor anderen Ebenen liegt, haben die Ebenen keinen Einfluss auf die Stapelreihenfolge von Objekten auf einer Seite.
Ändern - Anordnung
können Sie die Stapelreihenfolge ändern.
Die Bereiche auf einer Ebene, die keine Objekte enthalten, sind transparent.
$[officename] Draw bietet drei Standardebenen:
Layout
Controls
Maßlinien
Einfügen - Ebene
können Sie eigene Ebenen hinzufügen.
Layout
ist der voreingestellte Arbeitsbereich.
Die Ebene
Layout
bestimmt die Position von Titel, Text und Objektplatzhaltern in Ihrer Seite.
Controls
ist für Schaltflächen gedacht, denen eine Aktion zugewiesen wurde und die nicht gedruckt werden sollen.
Deaktivieren Sie die Eigenschaft &quot; Druckbar &quot; der Ebene.
Die Objekte auf der Ebene
Controls
befinden sich stets vor den Objekten auf anderen Ebenen.
Maßlinien
Ebene dient zum Zeichnen von Maßlinien.
Durch das Wechseln der Ebene können Sie diese Linien leicht an- und ausschalten, um sie zu verstecken oder anzuzeigen.
