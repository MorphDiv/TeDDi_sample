Media-Player-Fenster
Medienwiedergabe
Öffnet das Fenster der Medienwiedergabe.
Hier können Sie sich eine Vorschau von Video- und Audiodateien anzeigen lassen und diese Dateien in das aktuelle Dokument einfügen.
Die Medienwiedergabe unterstützt viele verschiedene Dateiformate.
Sie können auch Mediendateien aus dem Medienwiedgabefenster in Ihr Dokument einfügen.
Auf Linux- oder Solaris-basierten Systemen braucht die Medienwiedergabe das Java Media Framework API (JMF).
Laden Sie die JMF-Dateien von http://java.sun.com/javase/technologies/desktop/media/jmf/index.jsp herunter und fügen Sie die Pfade zu der installierten jmf.jar dem Class Path unter Extras - Optionen - %PRODUCTNAME - Java hinzu.
Auf Windows-Computern verwendet der Media-Player DirectShow, das auf Ihrem Computer standardmäßig installiert sein sollte.

Audiodatei zur Vorschau.

Audiodatei als Medienobjekt in das aktuelle Dokument ein.
Abspielen

Pause
Beginnt die Wiedergabe der aktuellen Datei bzw. hält die Wiedergabe an.
Stop
Beendet die Wiedergabe der aktuellen Datei.
Letzter Befehl
Gibt die Datei mehrmals wieder.
Mute
Schaltet die Stummschaltung ein bzw. aus.
Schieberegler zur Lautstärkeeinstellung
Stellt die Lautstärke ein.
Zoom
Passt die Größe der Videowiedergabe an.
Positionsschieberegler
