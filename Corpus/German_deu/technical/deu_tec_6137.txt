KMyMoney-Dateien
Ihre .KMY-Datendatei
Alle Ihre finanziellen Informationen, die & kappname; kennt, werden in einer Datei mit der Endung .KMY gespeichert. Sie können beliebig viele solcher Dateien haben, wobei jede einen unterschiedlichen Namen haben muss. Sobald Sie das Programm starten, wird automatisch die zuletzt geladene Datei geöffnet.
Bearbeiten der .KMY-Datendatei
Die .KMY Datei ist eine mit gzip komprimierte XML Datei. Wenn Sie mehr über die Struktur der Datei lernen möchten, können Sie sie mit gunzip entpacken und in einem Texteditor öffnen.
Erstellen einer anonymen Datendatei
Wenn Sie einen Fehler feststellen, ist es manchmal hilfreich für die Entwickler, Ihre Datendatei zu erhalten, um das Problem zu reproduzieren und zu beheben. Das stellt natürlich ein Problem dar, da Sie sicherlich nicht den Entwicklern Ihre privaten finanziellen Details zusenden möchten.
Um dies zu lösen, erlaubt es Ihnen & kappname; eine & quot;anonymisiertequot; Version Ihrer Daten abzuspeichern. Dadurch werden sämtliche persönliche Informationen aus der Datei entfernt, aber die Struktur bleibt erhalten.
Um eine solche anonyme Kopie der Datei zu erzeugen, öffnen Sie das Menü Datei aus der Menüleiste und wählen Sie dort Speichern unter.... Als Dateityp wählen Sie & quot;anonyme Dateienquot;. Hiermit speichern Sie Ihre Datei in anonymer Form.
Als nächstes sollten Sie prüfen, ob das Problem mit der anonymen Datei weiterhin besteht. Nicht alle Probleme treten mit dieser anonymisierten Form auf. Schließen Sie Ihre Daten und laden Sie die anonymisierte Datei. Probieren Sie die gleichen Aktionen, die das Problem verursachten. Hoffentlich lässt sich das Problem auch mit der anonymisierten Datei reproduzieren.
Wenn dies der Fall ist, können Sie das Problem nun an die Entwickler Mailing-Liste melden und anmerken, dass Sie eine anonymisierte Datei haben, mit der das Problem reproduziert werden kann. Wenn Sie dazu aufgefordert werden, senden Sie die Datei als private E-Mail an denjenigen Entwickler, der diese anfordert.
