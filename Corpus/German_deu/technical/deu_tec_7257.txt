ZEIT kann auf verschiedene Art und Weise angegeben werden, der gebräuchlichste Wert ist das Wort »now«, damit das System sofort herunterfährt. Andere gültige Formate sind »+m«, wobei m die Anzahl der Minuten bis zum Herunterfahren angibt, und hh:mm (eine Uhrzeit im 24-Stunden-Format).

Angemeldete Benutzer werden mit einer an ihr Terminal gesendeten Meldung gewarnt, Sie können optional eine NACHRICHT dazu angeben. Nachrichten können ohne tatsächliches Herunterfahren mittels der Option -k versandt werden.

Falls ZEIT angegeben wurde, wird der Befehl im Vordergrund verbleiben, bis das Herunterfahren beginnt. Er kann durch Strg+C oder von einem anderen Benutzer mittels der Option -c abgebrochen werden.

Das System wird standardmäßig in einen Wartungszustand (Einbenutzer-Modus) versetzt. Sie können dieses Verhalten durch Angabe der Optionen -h (für Herunterfahren) oder -r (für Neustart) ändern. Die Option -h kann mittels -H oder -P verfeinert werden, um anzugeben, ob das System nach dem Herunterfahren angehalten (-H) oder ausgeschaltet (-P) werden soll. Der Standardwert wird den shutdown-Skripten überlassen.
