UCase-Funktion [Laufzeit]
Konvertiert alle Kleinbuchstaben in einer Zeichenkette in Großbuchstaben.
LCase-Funktion
Syntax
:
UCase (Text As String)
Rückgabe
:
String
Parameter:
Text:
Ein beliebiger Zeichenkettenausdruck, den Sie konvertieren möchten.
Beispiel:
Sub ExampleLUCase
Dim sVar As String
sVar = &quot; Las Vegas &quot;
print LCase(sVar) REM liefert &quot; las vegas &quot;
print UCase(sVar) REM liefert &quot; LAS VEGAS &quot;
end Sub
