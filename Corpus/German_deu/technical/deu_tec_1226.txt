Linie
Hier legen Sie die Formatierungsoptionen für die ausgewählte Linie oder die Linie fest, die gleich darauf gezeichnet werden soll.
Außerdem können Sie Linien mit Linienenden versehen oder Diagrammsymbole ändern.
Linieneigenschaften
Stil
Wählen Sie den gewünschten Linienstil.
Farbe
Wählen Sie eine Linienfarbe.
Breite
Wählen Sie eine Linienstärke.
Dabei können Sie zusätzlich eine Maßeinheit angeben.
Die Linienstärke Null führt zu einer Haarlinie mit einer Breite von einem Pixel auf dem jeweiligen Ausgabemedium
Transparenz
Geben Sie die gewünschte Transparenz für die Linie ein.
Bei 100 % ist die Linie vollständig durchsichtig, bei 0 % vollständig undurchsichtig.
