Zusätzliche Einstellungen
Legt zusätzliche Optionen für eine Datenquelle fest.
Bearbeiten - Datenbank - Eigenschaften
, und klicken Sie auf das Register
Zusätzliche Eigenschaften
Die Verfügbarkeit der folgenden Steuerelemente hängt vom Datenbanktyp ab:
Rechnername
Geben Sie den Host-Namen des Servers ein, auf dem die Datenbank installiert ist, z.
B. &quot; ldap.Server.com &quot; .
Portnummer
Geben Sie die Portnummer für den Server ein, auf dem die Datenbank installiert ist.
MySQL JDBC Treiberklasse
Geben Sie den Namen des JDBC-Treibers für die MySQL-Datenbank ein.
Zeichensatz
Wählen Sie einen Zeichensatz für die Anzeige der Datenbank in $[officename] aus.
Diese Einstellung hat keine Auswirkungen auf die Datenbank.
Wählen Sie &quot; System &quot; , um den Standardzeichensatz Ihres Betriebssystems zu verwenden.
Text- und dBase-Datenbanken sind auf Zeichensätze mit fester Zeichenlänge beschränkt, in welchen alle Zeichen mit derselben Anzahl von Byte codiert sind.
Oracle JDBC Treiberklasse
Geben Sie den Namen des JDBC-Treibers für die Oracle-Datenbank ein.
Treibereinstellungen
Festlegen zusätzlicher Treiberoptionen.
Katalog verwenden bei dateibasierten Datenbanken
Verwendet die aktuelle Datenquelle des Katalogs.
Diese Option ist hilfreich, wenn die ODBC-Datenquelle ein Datenbankserver ist.
Wählen Sie diese Option nicht, falls die ODBC-Datenquelle ein dBase-Treiber ist.
Base DN
Geben Sie den Ausgangspunkt für die Suche in der LDAP-Datenbank an, z.
B. &quot; dc=com &quot; .
Maximale Anzahl der Datensätze
Geben Sie hier an, wie viele Datensätze beim Zugriff auf den LDAP-Server maximal geladen werden sollen.
Auch gelöschte Datensätze anzeigen
Zeigt alle Datensätze einer Datei an, einschließlich der als gelöscht gekennzeichneten.
Wenn Sie dieses Markierfeld auswählen, ist das Löschen von Datensätzen nicht möglich.
Im dBase-Format verbleiben gelöschte Datensätze in der Datei.
Wenn Sie die an der Datenbank vorgenommenen Änderungen sehen möchten, schließen Sie die Verbindung zur Datenbank und stellen sie direkt wieder her.
Indizes
Öffnet den Dialog Indizes, in dem Sie die Tabellenindizes der aktuellen dBase-Datenbank verwalten können.
UFI:
Commercial means Adabas
Datenpuffergröße (MB)
Geben Sie die gewünschte Größe des Datenpuffers für die Datenbank ein.
Die Einstellung wird erst nach einem Neustart der Datenbank wirksam.
Zuwachsgröße (MB)
Geben Sie die gewünschte Zuwachsgröße für die automatische Vergrößerung der Datenbank an.
Die maximal zulässige Zuwachsgröße beträgt 100 MB.
Die Einstellung wird erst nach einem Neustart der Datenbank wirksam.
Control Benutzername
Der hier angegebene Benutzer erhält eine beschränkte Berechtigung zum Ändern einiger Datenbankparameter.
Control-Kennwort
Geben Sie das Kennwort für den Control-Benutzer ein.
Service beim Beenden von %PRODUCTNAME herunterfahren
Der Adabas-Datenbankserver wird beim Beenden von $[officename] ebenfalls beendet.
Diese Option steht nur dann zur Verfügung, wenn der Datenbankserver von $[officename] aus gestartet wurde und sowohl ein Control-Benutzer als auch dessen Kennwort angegeben wurden.
Erweitert
Datenbankstatistiken
mit statistischen Angaben zur Adabas-Datenbank.
Text enthält Kopfzeile
Aktivieren Sie dieses Markierfeld, wenn die erste Zeile der Textdatei Feldnamen enthält.
Feldtrenner
Geben Sie das Trennzeichen für Datenfelder in der Textdatei ein oder wählen Sie es aus.
Texttrenner
Geben Sie den Bezeichner für Textfelder in der Textdatei ein oder wählen Sie ihn aus.
Dieses Zeichen muss sich vom Feldtrennzeichen unterscheiden.
Dezimaltrennzeichen
Geben Sie das in der Textdatei verwendeten Dezimaltrennzeichen, z.B. einen Punkt (0.5) oder ein Komma (0,5), ein oder wählen Sie es aus.
Tausendertrennzeichen
Geben Sie das in der Textdatei verwendete Tausendertrennzeichen, z.B. einen Punkt (1.000) oder ein Komma (1,000), ein oder wählen Sie es aus.
Dateierweiterung
Wählen Sie ein Format für die Textdatei aus.
Die ausgewählte Dateinamenserweiterung wirkt sich auf einige der Standardeinstellungen in diesem Dialog aus.
