Erstellen und Anwenden von Seitenvorlagen
$[officename] verwendet Seitenvorlagen, um das Layout einer Seite festzulegen.
Dazu gehören Ausrichtung, Hintergrund, Ränder, Kopf-/Fußzeilen und Textspalten der Seite.
Um das Layout einer bestimmten Seite in einem Dokument zu ändern, müssen Sie eine benutzerdefinierte Seitenvorlage erstellen und auf die Seite anwenden.
So definieren Sie eine neue Seitenvorlage:
