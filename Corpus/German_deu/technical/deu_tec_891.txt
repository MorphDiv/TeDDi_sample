Klickbarer Hotspots zu Grafiken hinzufügen
Eine ImageMap bietet die Möglichkeit, bestimmten Bereichen in Bildern eines Dokuments, den so genannten Hotspots, URLs zuzuweisen.
Eine ImageMap ist eine Gruppe von Hotspots.
Es können drei Typen von Hotspots eingezeichnet werden:
Rechtecke, Ellipsen und Polygone.
Wenn Sie auf einen Hotspot klicken, wird der URL in dem Browserfenster oder Frame geöffnet, den Sie angeben.
Sie können auch bestimmen, welcher Text angezeigt wird, wenn die Maus auf dem Hotspot liegt.
So fügen Sie einer Grafik einen anklickbaren Hotspot hinzu
Setzen Sie den Cursor an die Stelle im Dokument, an der die ImageMap eingefügt werden soll.
Einfügen - Bild - Aus Datei
, wählen Sie eine Bitmap-Grafik aus und fügen Sie sie ein.
Bearbeiten - ImageMap
.
Der
ImageMap-Editor
öffnet sich, in dem die Grafik im Hintergrund angezeigt wird.
Ziehen Sie mit den Symbolen im ImageMap-Editor eine Hotspot-Form (z.
B. ein Rechteck) über dem Bild im Hintergrund auf.
Extras - Optionen - %PRODUCTNAME - Allgemein
die Option &quot; Erweiterte Tipps &quot; aktivieren, erhalten Sie ausführlichere Hilfetexte zu den Funktionen der einzelnen Symbole.
Geben Sie als &quot; Adresse &quot; den URL ein, der beim Klicken auf den Hotspot in einem Webbrowser angezeigt wird.
Sie haben auch die Möglichkeit, den &quot; Text &quot; einzugeben, der als Tipp angezeigt werden soll, wenn die Maus über den Hotspot geführt wird.
Klicken Sie auf die Schaltfläche &quot; Zuweisen &quot; , um die Änderungen zu übernehmen, und schließen Sie den ImageMap-Editor.
Speichern Sie das Dokument im %PRODUCTNAME- oder HTML-Format.
Sie können die ImageMap als Datei speichern und beispielsweise auf einen Webserver hochladen.
