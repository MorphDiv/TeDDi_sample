Die folgenden Tipps sind keine vollkommene Garantie dafür, dass Sie das Spiel gewinnen. Sie sind vielmehr als eine einfache Hilfe gedacht.
Schauen Sie nach einer allein stehenden Zahl
Schauen Sie beim Start des Spiels nach einer Zahl,die nur einmal vorkommt, oder nach einer Zahl, zu der es keine ähnlichen Steine gibt, die daneben platziert werden könnten.
Falls Sie eine solche Zahl finden, ist dies möglicherweise der Einstieg in das Spiel, sofern Sie wissen, wo sie platziert werden muss.
Denken Sie daran: Je größer das Spielfeld ist, umso schwieriger ist es, eine solche Zahl zu finden.

Grundlegende Bedienung
Das Fenster des Spiels ist in zwei Bereiche untergliedert: Rechts befinden sich die Spielsteine, die als Ausgangspunkt zum Anordnen und Verschieben auf der linken Seite zur Verfügung stehen. Jeder Spielstein wiederum ist in vier Dreiecke untergliedert, die jeweils eine Zahl enthalten.
Ziel des Spiels ist es, die Steine so anzuordnen, dass zwei gleiche Zahlen jeweils einander berühren.
Es ist nicht möglich, die Spielsteine so abzulegen, dass zwei verschiedene Zahlen einander berühren.
Das Spiel ist beendet, sobald Sie alle Spielsteine an deren richtigen Positionen abgelegt haben.

Bewegen der Spielsteine
Bewegen eines Spielsteins
Um einen Spielstein zu bewegen, klicken Sie darauf und halten die Maustaste gedrückt. Dann ziehen Sie ihn aus seiner aktuellen Position an den gewünschten Ort im linken oder rechten Bereich des Spielfelds und lassen die Maustaste los.
Diese Züge sind nicht permanent und können rückgängig gemacht werden.
Die Steine im linken Spielfeldbereich können alle gleichzeitig in eine bestimmte Richtung verschoben werden. Weitere Informationen finden Sie in
»Klicken und gedrückt halten« vermeiden
Wenn Sie es nicht praktisch finden, zum Bewegen von Spielsteinen die Maustaste gedrückt zu halten, können Sie das Verhalten so einstellen, dass Sie jeweils nur einen Spielstein anklicken müssen, woraufhin ein weiterer Klick an der gewünschten Position den Stein dort ablegt.
