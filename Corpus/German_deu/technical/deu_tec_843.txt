Web-Assistent - Dokumente
Wählen Sie die Dateien, die auf der Website veröffentlicht werden sollen.
Die Dateien können auf einen Server oder ein lokales Verzeichnis geladen werden.
Dokumentliste
Listet die Dokumente auf, die auf der Website veröffentlicht werden sollen.
Der Assistent kann %PRODUCTNAME-Dokumente vor dem Hochladen auf den Server in das HTML- und PDF-Format sowie (in einigen Fällen) in das Flash-Format konvertieren.
Alle anderen Dateitypen werden in ihrem jeweiligen Format auf den Server geladen.
Hinzufügen...
Öffnet einen Dialog, in dem Sie die Dateien auswählen können, die auf der Website veröffentlicht werden sollen.
Die Reihenfolge in dieser Liste legt die Reihenfolge fest, in der Hyperlinks zu den Dokumenten auf der Indexseite Ihrer Website angezeigt werden.
Aufheben
Entfernt die ausgewählte Datei aus der Liste.
Exportieren im Dateiformat
Wählen Sie das Dateiformat, in das die ausgewählte Datei exportiert werden soll.
Titel
Geben Sie für das ausgewählte Dokument einen Titel ein.
Der Titel erscheint auf der Indexseite Ihrer Website als Hyperlink zum ausgewählten Dokument.
Zusammenfassung
Geben Sie eine Beschreibung für das ausgewählte Dokument ein.
Autor
Geben Sie den Autorennamen für das ausgewählte Dokument ein.
