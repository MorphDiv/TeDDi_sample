Text in Tabelle umwandeln
Hiermit wandeln Sie den ausgewählten Text in eine Tabelle bzw. die ausgewählte Tabelle in einen Text um.
Die in diesem Dialog verfügbaren Optionen richten sich nach der Art der Umwandlung.
Getrennt
Trennzeichen wie z.
B.
Tabulatoren legen die Spaltenunterteilung im ausgewählten Text fest.
Jeder Absatz innerhalb der Auswahl wird in eine Tabellenzeile umgewandelt.
Wenn Sie eine Tabelle in Text umwandeln, werden dementsprechend die Spaltenmarkierungen in das festgelegte Zeichen und jede Zeile in einen eigenen Absatz umgewandelt.
Tabulator
Hiermit wandeln Sie den Text in eine Tabelle um; dabei werden Tabulatoren als Spaltentrennzeichen verwendet.
Semikolon
Hiermit wandeln Sie den Text in eine Tabelle um; dabei werden Semikolons (;) als Spaltentrennzeichen verwendet.
Absatz...
Hiermit wandeln Sie den Text in eine Tabelle um; dabei werden Absatzzeichen als Spaltentrennzeichen verwendet.
Andere:
Hiermit wandeln Sie den Text in eine Tabelle um; dabei wird das in das Feld eingegebene Zeichen als Spaltentrennzeichen verwendet.
Textfeld
Geben Sie das Zeichen ein, das Sie als Spaltentrennzeichen verwenden möchten.
Gleiche Breite aller Spalten
Hiermit erstellen Sie unabhängig von der Position des Spaltentrennzeichens gleich breite Spalten.
AutoFormat
Zusätze
Überschrift
Hiermit formatieren Sie die erste Zeile der neuen Tabelle als Überschrift (Spaltenkopf).
Überschrift wiederholen
Diese Option wiederholt die Tabellenkopfzeile auf jeder Seite, über die sich die Tabelle erstreckt.
Die ersten ...
Zeilen
Wiederholt die ersten n Zeilen als eine Kopfzeile.
Tabelle nicht trennen
Hiermit verhindern Sie, dass die Tabelle über mehrere Seiten aufgeteilt wird.
Umrandung
Hiermit versehen Sie die Tabelle und Tabellenzellen mit einem Rahmen.
