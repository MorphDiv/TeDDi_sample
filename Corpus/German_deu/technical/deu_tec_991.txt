Brief-Assistent - Seitendesign
Hier geben Sie an, ob ein Privat- oder ein Geschäftsbrief erstellt werden soll.
Die auf den folgenden Seiten verfügbaren Optionen hängen von dieser Auswahl ab.
Wählen Sie die gewünschte Briefdokumentvorlage.
Wählen Sie hier, ob Sie die Vorlage für einen privaten oder einen geschäftlichen Brief erstellen möchten.
Geschäftsbrief
Gibt an, dass ein Geschäftsbrief erstellt werden soll.
Förmlicher Privatbrief
Gibt an, dass Sie einen förmlichen Privatbrief erstellen möchten.
Privatbrief
Gibt an, dass ein Privatbrief erstellt werden soll.
Für Privatbriefe wird ein weniger formales Format verwendet als für Geschäftsbriefe.
Welche Stilart möchten Sie verwenden?
Wählen Sie den Stil für die Gestaltung Ihrer Briefvorlage.
Geschäftliches Briefpapier verwenden
Gibt an, ob Papier verwendet wird, das bereits ein aufgedrucktes Logo, Adresse und/oder Fußzeile enthält.
Der Assistent zeigt als nächstes die Seite &quot; Briefpapier-Layout &quot; an.
