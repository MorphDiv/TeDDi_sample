Symbolleiste Format
/text/schart/main0202.xhp
Format
Die Symbolleiste Format wird angezeigt, wenn Sie sich ein Diagramm im Bearbeitungsmodus befindet.
Doppelklicken Sie auf ein Diagramm, um in den Bearbeitungsmodus zu wechseln.
Klicken Sie außerhalb des Diagramms, um den Bearbeitungsmodus zu beenden.
Sie können die Formatierung der Diagrammelemente mithilfe der Symbolleiste Format ändern.
Wählen Sie ein Diagrammelement
Wählen Sie das Diagrammelement, das Sie formatieren möchten.
Das Element wird in der Voransicht ausgewählt.
Klicken Sie auf Auswahl formatieren, um den Eigenschaftsdialog für das gewählte Element zu öffnen.
Auswahl formatieren
Öffnet den Eigenschaftsdialog für das gewählte Element.
Diagrammtyp
Öffnet den Dialog Diagrammtyp.
Diagrammdatentabelle
Öffnet den Dialog Datentabelle, in dem Sie die Diagrammdaten bearbeiten können.
Horizontales Gitter An/Aus
Das Symbol Gitter horizontal ein/aus auf der Symbolleiste Format blendet das Hauptgitter für die Y-Achse ein bzw. aus.
Hinweis:
Wenn das Markierfelder für das Hilfsgitter der Y-Achse im Dialog Einfügen - Gitter markiert ist, wird das Hilfsgitter immer angezeigt
Legende An/Aus
Um die Legende anzuzeigen oder auszublenden, klicken Sie auf das Symbol Legende ein/aus in der Symbolleiste Format.
Textskalierung
Passt die Textgröße an, wenn Sie die Diagrammgröße ändern.
Automatisches Layout
Setzt alle Diagrammelemente auf Ihre Standardposition im aktuellen Diagramm zurück.
Diese Funktion ändert nur die Elementpositionen, aber nicht den Diagrammtyp oder andere Eigenschaften.
