Arbeiten mit Gruppen
Sie können mehrere Grafikobjekte in einer Gruppe kombinieren, so dass Sie diese wie ein einzelnes Objekt nutzen können.
Sie können alle Objekte einer Gruppe gemeinsam verschieben, umwandeln, vergrößern, verzerren oder konvertieren, und Sie können die Gruppe jederzeit betreten, um die einzelnen Objekte zu ändern.
Sie können die Eigenschaften (Linienstärke, Füllfarbe, usw) aller Objekte einer Gruppe gemeinsam verändern und Sie können die Gruppe jederzeit betreten, um die einzelnen Objekte zu ändern.
Gruppen können auch verschachtelt werden, um Gruppen innerhalb anderer Gruppen zu bilden.
So gruppieren Sie Objekte
Wählen Sie die Objekte gleichzeitig aus, die Sie gruppieren möchten.
Halten Sie die Umschalt-Taste gedrückt, während Sie die einzelnen Objekte anklicken.
Rechtsklicken Sie auf ein beliebiges der selektierten Objekte, um das Kontextmenü zu öffnen.
Gruppieren
.
In Calc oder Writer, wählen Sie
Gruppe - Gruppieren
.
Um Objekte zu markieren, ziehen Sie mit gedrückter Maustaste einen Rahmen um die Objekte auf.
Sie können zum Beispiel sämtliche Objekte eines Firmenlogos gruppieren, um das Logo wie ein einzelnes Objekt zu verschieben oder die Größe zu ändern.
Nachdem Sie die Objekte gruppiert haben, wird bei Selektion eines beliebigen Teilobjekts die ganze Gruppe ausgewählt.
So betreten Sie eine Gruppe
Rechtsklicken Sie auf ein beliebiges Objekt der Gruppe.
Wählen Sie in Impress oder Draw 'Gruppierung betreten'.
In Calc oder Writer wählen Sie 'Gruppe - Betreten'.
Nun können Sie in der Gruppe ein einzelnes Objekt auswählen und bearbeiten.
In diesem Modus können Sie Objekte einer Gruppe hinzufügen oder aus einer Gruppe löschen.
Die Objekte, die nicht Teil der Gruppe sind, werden mit gedämpften Farben dargestellt.
So verlassen Sie eine Gruppe
Rechtsklicken Sie auf ein beliebiges Objekt der Gruppe.
Wählen Sie in Impress oder Draw 'Gruppierung verlassen'.
In Calc oder Writer wählen Sie 'Gruppe - Verlassen'.
Sie können eine Gruppe in Draw oder Impress auch verlassen, indem Sie an eine beliebige Stelle außerhalb der Gruppe doppelklicken.
So heben Sie eine Gruppierung auf
Rechtsklicken Sie auf ein beliebiges Objekt der Gruppe.
Wählen Sie in Impress oder Draw 'Gruppierung aufheben'.
In Calc oder Writer wählen Sie 'Gruppe - Aufheben'.
Sie können nun alle Objekte einzeln auswählen und bearbeiten.
