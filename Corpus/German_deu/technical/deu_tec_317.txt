Filter für mobile Geräte (Palm / Pocket-PC)
$[officename]-Dokumente können in verschiedenen Mobilgeräte-Dateiformaten gespeichert und geöffnet werden.
Installieren Sie die Filter mithilfe des $[officename]-Installationsprogramms.
Derzeit sind die folgenden Filter erhältlich:
AportisDoc für Palm OS-kompatible Geräte
Pocket Excel für Pocket-PC-kompatible Geräte
Pocket Word für Pocket-PC-kompatible Geräte
Bei der Installation der Pocket-PC-Filter unter Windows wird außerdem eine DLL-Datei für die Unterstützung der Synchronisierungssoftware Ihres Pocket-PCs installiert.
Zur Verwendung von Filtern für mobile Geräte wird JRE (Java Runtime Environment) benötigt.
