ImageCharUp() zeichnet das erste Zeichen (Charakter) des Strings c in das Bild, auf das per id gezeigt wird.
Die Koordinaten des Zeichens werden durch x / y bestimmt.
Dabei wird von den Koordinaten 0 / 0 als Zeichnungsanfang links-oben ausgegangen.
Die Farbe des Zeichens wird mit col bestimmt.
Ist font als 1, 2, 3, 4 oder 5 definiert, wird ein eingebauter Font genutzt (je grösser die Nummer, desto grösser der Font).
Beachten Sie auch imageloadfont().
