Task Pane
Task Pane
Schaltet das %PRODUCTNAME Impress Task Pane ein bzw. aus.
Der Aufgabenbereich enthält Seiten, auf welchen Sie Masterseiten, Layouts, benutzerdefinierte Animation sowie Folienübergang festlegen können.
Klicken Sie auf einen anderen Titel, um eine andere Seite zu öffnen.
Master Page
Öffnet das Register &quot; Masterseiten &quot; , über das Sie entweder auf alle Folien (Mausklick links) oder die ausgewählten Folien (Mausklick rechts) eine Masterseite (Hintergrund) anwenden können.
Used in this presentation
Zeigt eine Liste der derzeit verwendeten Masterseiten an.
Recently used
Zeigt eine Liste der zuletzt verwendeten Masterseiten an.
Available for Use
Zeigt eine Liste aller verfügbaren Masterseiten an.
Klicken Sie mit der linken Maustaste, um die Masterseite auf alle Folien anzuwenden.
Klicken Sie mit der rechten Maustaste, um ein Untermenü aufzurufen.
Wendet die Masterseite auf alle Folien an.
Wendet die Masterseite oder das Foliendesign auf die ausgewählten Folien an.
Verändert die Größe für die Vorschau der Masterseiten.
Layouts
Öffnet das Register &quot; Layouts &quot; , über das Sie ein Foliendesign auf die ausgewählten Folien anwenden können.
Klicken Sie hier, um ein Foliendesign auf alle ausgewählten Folien anzuwenden.
Klicken Sie mit der rechten Maustaste, um ein Untermenü aufzurufen.
Custom Animation
Öffnet das Register &quot; Benutzerdefinierte Animation &quot; , über das Sie Effekte auf die ausgewählten Objekte einer Folie anwenden können.
Slide Transition
Öffnet das Register &quot; Folienübergang &quot; zum Festlegen von Übergangseffekten für die ausgewählten Folien.
Table Design
Öffnet den Dialog Tabelle einfügen.
Klicken Sie doppelt auf eine Vorschau, um eine neue Tabelle einzufügen.
