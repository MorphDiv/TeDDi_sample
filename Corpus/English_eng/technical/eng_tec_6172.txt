kappfinder
1
An app to add installed applications to the & kde; menu.
kappfinder --dir directory KDE Generic Options Qt Generic Options
Description
The KDE application findern (kappfinder;) looks for legacy and third party applications that are installed on your system and adds them to the KDE menu system.
Options
--dir directory
Install. desktop files into directory directory
