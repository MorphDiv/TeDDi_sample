ovrimos_execute() is used to execute an SQL statement.
ovrimos_execute() executes a prepared statement.
Returns TRUE or FALSE.
If the prepared statement contained parameters (question marks in the statement), the correct number of parameters should be passed in an array.
Notice that I don 't follow the PHP convention of placing just the name of the optional parameter inside square brackets.
I couldn 't bring myself on liking it.
