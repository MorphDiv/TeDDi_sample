Arranging Objects
Each object that you place in your document is successively stacked on the preceding object.
To rearrange the stacking order of a selected object proceed as follows.
Click the object that you want to change the position of.
Modify - Arrange
to bring up the context menu and choose one of the arrange options:
Bring to Front
places the object on top of all other objects
Bring Forwards
places the object one place forwards in the stack of objects
Send Backwards
places the object one place backwards in the stack of objects
Send to Back
places the object behind all other objects
Behind Object
places the object behind another object that you select
Arranging an Object Behind Another Object
Click the object whose position you want to change.
Modify - Arrange
to open the context menu and choose
Behind Object
.
The mouse pointer changes to a hand.
Click the object behind which you want to place the selected object.
Reversing The Stacking Order of Two Objects
Shift-click both objects to select them.
Modify - Arrange
to open the context menu and choose
Reverse
.
Aligning Objects
Alignment
function enables you to align objects relative to each other or relative to the page.
Select an object to align it to the page or select multiple objects to align them relative to each other.
Modify - Alignment
and select one of the alignment options.
Distributing Objects
Distribution
command to distribute the vertical and horizontal spacing evenly between the objects.
Select three or more objects to be distributed.
Modify - Distribution
.
OK
.
Distribution
command is applied.
