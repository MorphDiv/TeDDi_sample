(PHP 4 = 4.3.0)
Returns FALSE on error, and an array upon success.
This function will return information about the active connection following the information from within the DSN.
The connection_id is required to be a valid ODBC connection.
The fetch_type can be one of two constant types:
SQL_FETCH_FIRST, SQL_FETCH_NEXT.
Use SQL_FETCH_FIRST the first time this function is called, thereafter use the SQL_FETCH_NEXT.
