Using Smart Tags
Smart Tags provide additional information and functionality to specified words in a Writer document.
The available features can be different for different Smart Tags extensions.
Installing Smart Tags
extensions
to %PRODUCTNAME Writer.
To install a Smart Tag, do one of the following:
Tools - Extension Manager
to open the Extension Manager, click Add and browse to the file.
Click a Smart Tag *.oxt file link on a web page and open the link with the default application.
This requires a properly configured Web browser.
Smart Tags menu
Tools - Options - %PRODUCTNAME - Appearance
.
When you point to a Smart Tag, a tip help informs you to Ctrl-click to open the Smart Tags menu.
If you don't use a mouse, position the cursor inside the marked text and open the context menu by Shift+F10.
