Print
Determines the printer settings for spreadsheets.
Tools - Options - %PRODUCTNAME Calc - Print
defines the settings for all spreadsheets.
To define settings for the current document only, choose
File - Print
, then click the
Options
button.
Pages
Suppress output of empty pages
Specifies that empty pages that have no cell contents or draw objects are not printed.
Cell attributes such as borders or background colours are not considered cell contents.
Empty pages are not counted for page numbering.
Sheets
Print only selected sheets
File - Print
dialogue box.
If you have entered several &quot; pages &quot; as a print range in the
File - Print
dialogue box, only the pages (sheets) that have been selected are printed.
To select multiple sheets, click on each desired sheet name on the bottom margin of the workspace while keeping the Ctrl key pressed.
