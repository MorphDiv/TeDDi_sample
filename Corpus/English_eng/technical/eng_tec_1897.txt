Right Function [Run-time]
Returns the rightmost &quot; n &quot; characters of a string expression.
Left Function
.
Syntax:
Right (Text As String, n As Long)
Return value:
String
Parameters:
Text:
Any string expression that you want to return the rightmost characters of.
n:
Numerical expression that defines the number of characters that you want to return.
If
n
= 0, a zero-length string is returned.
The maximum allowed value is 65535.
The following example converts a date in YYYY-MM-DD format to the US date format (MM/DD/YYYY).
Example:
Sub ExampleUSDate
Dim sInput As String
Dim sUS_date As String
sInput = InputBox( &quot; Please input a date in the international format 'YYYY-MM-DD' &quot; )
sUS_date = Mid(sInput, 6, 2)
sUS_date = sUS_date & &quot; / &quot;
sUS_date = sUS_date & Right(sInput, 2)
sUS_date = sUS_date & &quot; / &quot;
sUS_date = sUS_date & Left(sInput, 4)
MsgBox sUS_date
End Sub
