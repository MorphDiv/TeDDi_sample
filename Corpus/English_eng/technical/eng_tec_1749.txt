Second Function [Run-time]
Returns an integer that represents the seconds of the serial time number that is generated by the TimeSerial or the TimeValue function.
Syntax:
Second (Number)
Return value:
Integer
Parameters:
Number:
Numerical expression that contains the serial time number that is used to calculate the number of seconds.
TimeSerial
function.
It returns the seconds of a serial time value that is generated by the
TimeSerial
or
TimeValue
functions.
For example, the expression:
Print Second(TimeSerial(12,30,41))
returns the value 41.
Example:
Sub ExampleSecond
MsgBox &quot; The exact second of the current time is &quot; & Second( Now )
End sub
