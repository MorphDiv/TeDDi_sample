Introduction
PDF Mod is a simple tool for doing basic manipulations of PDF documents.
With this tool, you can rotate, remove, or extract pages in a PDF document. You can also add pages from another document.
You can start PDF Mod from your applications menu or launcher, or by right clicking on one or more PDF documents in the file browser, and choosing to open with PDF Mod.
Like a normal editor application, you must save the changes you make using PDF Mod. If you want to keep the original document and save your changes to a new file, use Save As.
