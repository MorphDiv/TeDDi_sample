SECOND function
SECOND
Returns the second for the given time value.
The second is given as an integer in the range 0 to 59.
Syntax:
SECOND(Number)
Number
, as a time value, is a decimal, for which the second is to be returned.
Examples:
=SECOND(NOW())
returns the current second
=SECOND(C4)
returns 17 if content of C4 =
12:20:17
.
