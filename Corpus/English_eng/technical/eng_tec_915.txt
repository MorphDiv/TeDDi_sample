LDAP Connection
Specifies the settings for importing a database using LDAP
(
Lightweight Directory Access Protocol)
.
This page is only available if you registered an LDAP server as an address database.
Server URL
Enter the name of the LDAP server using the format &quot; ldap.server.com &quot; .
Base DN
Enter the starting point to search the LDAP database, for example, &quot; dc=com &quot; .
Port number
Enter the port of the LDAP server, normally 389.
Use secure connection (SSL)
Creates a secure connection to the LDAP server through the Secure Sockets Layer (SSL).
By default, a SSL connection uses port 636.
A standard connection uses port 389.
