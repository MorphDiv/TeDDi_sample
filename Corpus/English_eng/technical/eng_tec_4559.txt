The behavior of this function is almost identical to preg_replace(), except for the fact that instead of replacement parameter, one should specify a callback The behavior of this function is almost identical to preg_replace(), except for the fact that instead of replacement parameter, one should specify a callback that will be called and passed an array of matched elements in the subject string.
