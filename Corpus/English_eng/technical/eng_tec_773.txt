Comparison Operators
Condition
in the
Default Filter

Comparative operator
Effect
Equal (=)
Shows values equal to the condition.
Less than ( < )
Shows values less than the condition.
Greater than ( > )
Shows values greater than the condition.
Less than or equal to ( < =)
Shows values that are less than or equal to the condition.
Greater than or equal to ( > =)
Shows values that are greater than or equal to the condition.
Not equal ( < > )
Shows the values not equal to the condition.
Largest
Shows the N (numerical value as parameter) largest values.
Smallest
Shows the N (numerical value as parameter) smallest values.
Largest %
Shows the largest N% (numerical value as parameter) of the total values.
Smallest %
Shows the smallest N% (numerical value as parameter) of the entire values.
