Applying AutoFilter
AutoFilter
function inserts a combo box on one or more data columns that lets you select the records (rows) to be displayed.
Select the columns you want to use AutoFilter on.
Data - Filter - AutoFilter
.
The combo box arrows are visible in the first row of the range selected.
Run the filter by clicking the drop-down arrow in the column heading and choosing an item.
Only those rows whose contents meet the filter criteria are displayed.
The other rows are filtered.
You can see if rows have been filtered from the discontinuous row numbers.
The column that has been used for the filter is identified by a different arrow button colour.
When you apply an additional AutoFilter on another column of a filtered data range, then the other combo boxes list only the filtered data.
Standard Filter
dialogue box appears, allowing you to set up a standard filter.
Choose &quot; Top 10 &quot; to display the highest 10 values only.
