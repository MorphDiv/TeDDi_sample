Selecting Tables, Rows, and Columns
You can select a table in a text document with a keyboard or with a mouse.
Command
Ctrl
+A until all the cells are selected.
To select a table with the mouse, move the mouse pointer to a position just above and left of the table.
The mouse pointer becomes a diagonal arrow.
Click to select the table.
To select a row or column with the mouse, point to a position just left of the row or above the column.
The mouse pointer becomes an arrow.
Click to select the row or column.
