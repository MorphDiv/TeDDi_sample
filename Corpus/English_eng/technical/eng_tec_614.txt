Execute SQL statement
Opens a dialogue box in which you can enter an SQL command for administering a database.
You can only enter administration commands in this dialogue box, such as Grant, Create Table, or Drop Table, and not filter commands.
The commands that you can enter depend on the data source, for example, dBASE can only run some of the SQL commands list here.
