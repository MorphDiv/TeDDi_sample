When you click the Check for Updates button or choose the Update command, the Extension Update dialogue box is displayed and the check for availability of updates starts immediately.
While checking for updates, you will see a progress indicator.
Wait for some messages to show up in the dialogue box, or click Cancel to abort the update check.
If no updates are available, the message in the dialogue box tells you there are no updates.
Close the dialogue box.
If updates are available, the updates can either be installed automatically, or you must respond with some action:
The Extension Update dialogue box may contain entries which are not selectable and hence no automatic update can be performed.
Dependencies are not fulfilled (the update needs some more or newer files to be installed).
Extension Manager
for details.
A manual update is necessary.
When you click the Install button the Download and Installation dialogue box is displayed.
All extensions which can be directly downloaded are downloaded now.
The progress is shown in the Download and Installation dialogue box.
If an extension cannot be downloaded, a message is displayed.
The operation continues for the remaining extensions.
Some extensions may be marked with the phrase &quot; browser based update &quot; .
These extensions cannot be downloaded by the Extension Manager.
A web browser must be opened to download the extension update from a particular web site.
That site may require several more user interaction to download the extension.
After downloading you must install the extension manually, for example by double-clicking the extension's icon in a file browser.
For extensions marked as &quot; browser based update &quot; , the Extension Manager will open your web browser on the respective web site.
This happens when you close the dialogue box, after downloading any other extension updates.
If there are no extensions which can be directly downloaded then the web browser is started immediately.
After the last extension has been downloaded, the installation begins.
First all installed extensions for which an update could be downloaded successfully, are removed.
Then the updated extensions are installed.
If an error occurs, a message that the installation failed is displayed, but the operation proceeds.
