WeekDay function
WeekDay Function [Run-time]
Returns the number corresponding to the weekday represented by a serial date number that is generated by the DateSerial or the DateValue function.
Syntax:
WeekDay (Number)
Return value:
Integer
Parameters:
Number:
Integer expression that contains the serial date number that is used to calculate the day of the week (1-7).
The following example determines the day of the week using the WeekDay function when you enter a date.
Example:
Sub ExampleWeekDay
Dim sDay As String
REM Return and display the day of the week
Select Case WeekDay( Now )
case 1
sDay= &quot; Sunday &quot;
case 2
sDay= &quot; Monday &quot;
case 3
sDay= &quot; Tuesday &quot;
case 4
sDay= &quot; Wednesday &quot;
case 5
sDay= &quot; Thursday &quot;
case 6
sDay= &quot; Friday &quot;
case 7
sDay= &quot; Saturday &quot;
End Select
msgbox &quot; &quot; + sDay,64, &quot; Today is &quot;
End Sub
