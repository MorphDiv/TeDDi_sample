Move or Copy a Sheet
Moves or copies a sheet to a new location in the document or to a different document.
date values
between different spreadsheets, both spreadsheet documents must be set to the same date base.
If date bases differ, the displayed date values will change!
To Document
Indicates where the current sheet is to be moved or copied to.
Select
new document
if you want to create a new location for the sheet to be moved or copied.
Insert Before
The current sheet is moved or copied in front of the selected sheet.
The
move to end position
option places the current sheet at the end.
Copy
Specifies that the sheet is to be copied.
If the option is unmarked, the sheet is moved.
Moving sheets is the default.
