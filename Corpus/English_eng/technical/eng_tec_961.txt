Colour Resolution
Colour Resolution
list you will find the following colour resolution variants:
Original
Retains the original colour resolution.
1-Bit Threshold
Converts the object to black and white.
All brightness levels below the middle value become black, while all above the middle value become white.
1 Bit Dithered
Allows the object to be converted to a grid of black and white dots.
4-Bit Greyscale
Allows the object to be converted to a palette of 16 greyscale tones corresponding to the brightness of the original.
4-Bit Colour Palette
Allows the object to be converted to a palette of 16 colours that match the original colours as closely as possible.
8-Bit Greyscale
Allows the object to be converted to a palette of 256 greyscale tones corresponding to the brightness of the original.
8-Bit Colour Palette
Allows the object to be converted to a palette of 256 colours that match the original colours as closely as possible.
24-Bit True Colours
Allows the object to be converted to a full colour gamut containing approximately 16 million colours.
