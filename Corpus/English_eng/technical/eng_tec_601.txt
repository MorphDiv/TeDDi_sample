Navigating in the Data Source Browser
Use the Form Navigation bar at the bottom of the Data Source view to navigate between different records.
First record
Icon
Go to the first record in the table.
Previous record
Icon
Go to the previous record in the table.
Record number
Type the number of the record that you want to display, and then press Enter.
Next record
Icon
Go to the next record in the table.
Last record
Icon
Go to the last record in the table.
New record
Icon
Inserts a new record into the current table.
To create a record, click the asterisk (*) button at the bottom of the table view.
An empty row is added at the end of the table.
Number of records
The Form Navigation bar also displays information about the number of records in the current table.
For example, &quot; Record 7 of 9(2) &quot; indicates that two records (2) are selected in a table containing 9 records, and that the cursor is in record number 7.
