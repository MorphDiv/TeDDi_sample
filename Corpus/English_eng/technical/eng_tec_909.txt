Mobile Devices Filters for Pocket Device Appliances
You can save and open documents to and from several mobile devices file formats.
Currently the following filters are available:
AportisDoc for Palm OS compatible devices
Pocket Excel for Pocket PC compatible devices
Pocket Word for Pocket PC compatible devices
Installing the Pocket PC filters under Windows will also install a DLL file that supports the synchronisation software of your Pocket PC.
The Java Runtime Environment is required to use the mobile device filters.
