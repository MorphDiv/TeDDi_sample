Caption
Adds a numbered caption to a selected graphic, table, frame, text frame, or drawing object.
You can also access this command by right-clicking the item that you want to add the caption to.
Properties

Category
Select the caption category, or type a name to create a new category.
The category text appears before the caption number in the caption label.
Each predefined caption category is formatted with a paragraph style of the same name.
For example, the &quot; Illustration &quot; caption category is formatted with the &quot; Illustration &quot; paragraph style.
