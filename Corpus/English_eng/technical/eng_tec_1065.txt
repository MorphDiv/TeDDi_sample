Web Wizard - Preview
Specify where you want to publish your web site as well as preview your site.
Preview
Opens your web page in the default web browser of your operating system.
Local directory
Uploads your index page and files to a local directory.
The index page is saved to the location that you specified.
All other files are saved to the &quot; myWebsite &quot; directory in the directory that contains the index page.
...
Opens a dialogue box to select a folder.
Archive file
Adds your index page and files to a compressed archive file and uploads the file to your web site.
The index page is saved to the location that you specified.
All other files are saved to the &quot; myWebsite &quot; directory in the directory that contains the index page.
Depending on your operating system, the available archive file formats are zip, gzip, and war.
...
Opens a dialogue box in which you can specify the location of the archive file.
The web via FTP
Uploads your files to an FTP server.
The index page is saved to the location that you specified.
All other files are saved to the &quot; myWebsite &quot; directory in the directory that contains the index page.
You cannot use the FTP option if you connect to the Internet through a proxy server.
Configure
FTP Connection
dialogue box in which you can edit and test the connection settings for the FTP server.
Save settings
Saves the settings that you specified in this wizard.
Save as
Enter the name for the settings file.
