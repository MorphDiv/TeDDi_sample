(PHP 3 = 3.0.13, PHP 4)
ftp_fget() retrieves remote_file from the FTP server, and writes it to the given file pointer, fp.
The transfer mode specified must be either FTP_ASCII or FTP_BINARY.
Note:
The resumepos parameter was added in PHP 4.3.0.
Returns TRUE on success or FALSE on failure.
See also ftp_get().
