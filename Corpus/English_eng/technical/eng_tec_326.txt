Data labels
dialogue box, which enables you to set the data labels.
If an element of a data series is selected, this command works on that data series only.
If no element is selected, this command works on all data series.
Show value as number
Displays the absolute values of the data points.
Number format
Opens a dialogue box to select the number format.
Show value as percentage
Displays the percentage of data points in each column.
Percentage format
Opens a dialogue box to select the percentage format.
Show category
Shows the data point text labels.
Show legend key
Displays the legend icons next to each data point label.
Separator
Selects the separator between multiple text strings for the same object.
Placement
Selects the placement of data labels relative to the objects.
