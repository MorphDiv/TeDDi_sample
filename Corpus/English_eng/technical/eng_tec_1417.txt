Default colours
Assigns colours to the data rows.
The settings only apply for all newly created charts.
Chart colours
Displays all the colours available for the data series.
Select a data series to change its colour.
Select the desired colour from the adjacent colour table.
Colour table
This table is used as a means of replacing the chart colours for the selected data rows.
For example, if you selected data row 6 and then click on the colour green 8, the old colour of the data row is replaced by green 8.
The name of the selected colour is shown below the colour table.
Default
Restores the colour settings that were defined when the program was installed.
