This wizard is visible for the evaluation version of %PRODUCTNAME.
If you are using a time-limited evaluation version of %PRODUCTNAME, you can purchase a licence on-line.
You will get a serial number to change your %PRODUCTNAME to a full version.
The wizard can be called any time from the Help menu.
The wizard will be offered automatically starting five days before the evaluation period expires.
The wizard consists of the following steps:
Purchase %PRODUCTNAME
This is the welcome page of the wizard.
If you want to buy a serial number on-line, click the button.
If you already have a serial number, click Next to enter the number on the next page.
Get your serial number on-line
Click to enter a web page, where you can purchase a serial number on-line.
Unlock %PRODUCTNAME
Enter your serial number or copy and paste the number from the on-line web page.
Licence Agreement
Please read the licence text (scroll down the text accordingly), then accept the licence to continue unlocking.
If you do not accept the licence, the software will not be unlocked.
Personal Data
This step of the wizard appears when the unlocking program finds an installed version of %PRODUCTNAME where some user settings can be copied to the new installation.
Copy personal data
Mark this check box to copy your user name and other data from a previous installation to your new installation of %PRODUCTNAME.
Summary
This is the last page of the Purchase %PRODUCTNAME Wizard.
