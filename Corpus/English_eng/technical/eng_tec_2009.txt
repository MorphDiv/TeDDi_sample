Referencing Cells by Drag-and-Drop
With the help of the Navigator you can reference cells from one spreadsheet to another sheet in the same document or in a different document.
The cells can be inserted as a copy, link or hyperlink.
The range to be inserted must be defined with a name in the original file so that it can be inserted in the target file.
Open the document that contains the source cells.
Insert - Names - Define
.
Save the source document, and do not close it.
Open the sheet in which you want to insert something.
Navigator
.
In the lower box of the Navigator select the source file.
In the Navigator, the source file object appears under &quot; Range names &quot; .
Drag Mode
icon in Navigator, choose whether you want the reference to be a hyperlink, link, or copy.
Click the name under &quot; Range names &quot; in the Navigator, and drag into the cell of the current sheet where you want to insert the reference.
This method can also be used to insert a range from another sheet of the same document into the current sheet.
Select the active document as source in step 4 above.
