Date and Time
Report Builder
by choosing
Insert - Date and Time
.
Shift-F1
and point with the mouse at an input box to see a help text for this input box.
Enable Include Date to insert a date field into the active area of the report.
The date field displays the current date when the report is executed.
Select a format to display the date.
Enable Include Time to insert a time field into the active area of the report.
The time field displays the current time when the report is executed.
Select a format to display the time.
Click OK to insert the field.
You can click the date or time field and drag to another position within the same area, or edit the properties in the Properties window.
