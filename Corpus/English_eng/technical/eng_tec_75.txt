Importing and Exporting dBASE Files
You can open and save data in the dBASE file format (*.dbf file extension) in $[officename] Base or a spreadsheet.
In %PRODUCTNAME Base, a dBASE database is a folder that contains files with the .dbf file extension.
Each file corresponds to a table in the database.
Formulae and formatting are not transferred when you open and save a dBASE file from %PRODUCTNAME.
