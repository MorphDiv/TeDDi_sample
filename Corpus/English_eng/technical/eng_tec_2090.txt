Insert as Hyperlink
Inserts a hyperlink when you drag-and-drop an object from the Navigator into a document.
You can later click the created hyperlink to set the cursor and the view to the respective object.
If you insert a hyperlink that links to an open document, you need to save the document before you can use the hyperlink.
