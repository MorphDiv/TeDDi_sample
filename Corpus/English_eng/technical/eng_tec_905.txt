Connection Type Wizard
Changes the type of connection for the current database.
Edit - Database - Connection Type
The Connection Type Wizard consists of three pages.
You cannot transfer all settings from one database type to another.
For example, you can use the wizard to open a database file that is in a format that is usually not recognised by an installed database.
Database type
Select the type of database to which you want to connect.
