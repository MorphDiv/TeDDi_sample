Protecting Sheet
Protects the cells in the current sheet from being modified.
Choose
Tools - Protect Document - Sheet
to open the
Protect Sheet
dialogue box in which you then specify sheet protection with or without a password.
Protected
check box must be checked on the
Format - Cells - Cell Protection
tab page or on the
Format Cells
context menu.
Tools - Protect Document - Sheet
and
Format - Cells - Cell Protection
menus:
Select the cells that will be unprotected
Format - Cells - Cell Protection
.
Unmark the
Protected
box and click
OK
.
Tools - Protect Document - Sheet
menu, activate protection for the sheet.
Effective immediately, only the cell range you selected in step 1 can be edited.
Format - Cells - Cell Protection
tab page, mark the
Protected
check box.
Finally, choose the
Tools - Protect Document - Sheet
menu.
The previously editable range is now protected.
Delete
and
Move/Copy
commands cannot be selected.
If a sheet is protected, you will not be able to modify or delete any Cell Styles.
