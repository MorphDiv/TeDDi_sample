Trusted Sources
macro security
settings for trusted certificates and trusted file locations.
Trusted certificates
Lists the trusted certificates.
View
Opens the View Certificate dialogue box for the selected certificate.
Remove
Removes the selected certificate from the list of trusted certificates.
Trusted file locations
Trusted file locations are used only if the security level is set to &quot; very high &quot; .
With this setting, document macros are only executed if they have been opened from one of the following locations.
Add
Opens a folder selection dialogue box.
Select a folder from which all macros are allowed to execute.
Remove
Removes the selected folder from the list of trusted file locations.
