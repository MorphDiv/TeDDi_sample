(PHP 3 = 3.0.6, PHP 4)
Returns the width of text using the last font set by pdf_setfont().
If the optional parameters font and size are specified, the width will be calculated using that font and size instead.
Please note that font is a font handle returned by pdf_findfont().
Note:
Both the font and size parameters must used together.
See Also: pdf_setfont() and pdf_findfont().
