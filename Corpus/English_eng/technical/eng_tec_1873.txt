DimArray function
DimArray Function [Run-time]
Returns a Variant array.
Syntax:
DimArray ( Argument list)
Array
If no parameters are passed, an empty array is created (like Dim A() that is the same as a sequence of length 0 in Uno).
If parameters are specified, a dimension is created for each parameter.
Parameters:
Argument list:
A list of any number of arguments that are separated by commas.
Example:
DimArray( 2, 2, 4 ) is the same as DIM a( 2, 2, 4 )
