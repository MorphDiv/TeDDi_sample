HOUR function
HOUR
Returns the hour for a given time value.
The hour is returned as an integer in the range 0 to 23.
Syntax:
HOUR(Number)
Number
, as a time value, is a decimal, for which the hour is to be returned.
Examples:
=HOUR(NOW())
returns the current hour
=HOUR(C4)
returns 17 if the contents of C4 =
17:20:00
.
