Lof Function [Run-time]
Returns the size of an open file in bytes.
Syntax:
Lof (FileNumber)
Return value:
Long
Parameters:
FileNumber:
A numerical expression that contains the file number that is specified in the Open statement.
FileLen
function.
Example:
Sub ExampleRandomAccess
Dim iNumber As Integer
Dim sText As Variant REM must be a Variant
Dim aFile As String
aFile = &quot; c:\data.txt &quot;
iNumber = Freefile
Open aFile For Random As #iNumber Len=32
Seek #iNumber,1 REM Position at start
Put #iNumber,, &quot; This is the first line of text &quot; REM Fill with text
Put #iNumber,, &quot; This is the second line of text &quot;
Put #iNumber,, &quot; This is the third line of text &quot;
Seek #iNumber,2
Get #iNumber,,sText
Print sText
Close #iNumber
iNumber = Freefile
Open aFile For Random As #iNumber Len=32
Get #iNumber,2,sText
Put #iNumber,, &quot; This is a new line of text &quot;
Get #iNumber,1,sText
Get #iNumber,2,sText
Put #iNumber,20, &quot; This is the text in record 20 &quot;
Print Lof(#iNumber)
Close #iNumber
end sub
