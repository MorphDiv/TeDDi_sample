Templates and Styles
A template is a document that contains specific formatting styles, graphics, tables, objects, and other information.
A template is used as the basis for creating other documents.
For example, you can define paragraph and character styles in a document, save the document as a template, and then use the template to create a new document with the same styles.
Unless you specify otherwise, every new $[officename] text document is based on the default template.
