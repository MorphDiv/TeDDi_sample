Class Path
You use this dialogue box to add folders and archives to the Java class path.
These paths are valid for any JRE that you start.
Assigned folders and archives
Specifies the location of Java classes or Java class libraries.

Java classes that are accessed through the classpath do not undergo a security check.
Add Archive

Add Folder

Remove
