Add / Edit
Adds a new item or edits the selected item in the XForms Data Navigator.
Items can be elements, attributes, submissions, or bindings.
%PRODUCTNAME inserts a new item directly after the currently selected item in the Data Navigator.
A new attribute is added to the currently selected element.
Name
Enter the name of the item.
The attribute names of elements or groups must be unique within the same group.

You cannot change the type of an edited item.
