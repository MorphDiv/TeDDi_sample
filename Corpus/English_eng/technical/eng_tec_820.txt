Specifies the graphics type and the target screen resolution.
Save graphics as
Determines the image format.
You can also define the compression value for the export.
PNG - Portable Network Graphics format
The files are exported as PNG files.
PNG files are compressed without loss of data, and can contain more than 256 colours.
GIF - Graphics Interchange Format
The files are exported as GIF files.
GIF files are compressed without loss of data, and have a maximum of 256 colours.
JPG - Compressed file format
The files are exported as JPEG files.
JPEG files are compressed, with adjustable compression and can contain more than 256 colours.
Quality
Specifies the compression factor of the JPEG graphic.
A 100% value offers the best quality for a large data range.
The 25% factor indicates small files with inferior image quality.
Monitor resolution
Defines the resolution for the target screen.
Depending on the selected resolution, the image will be displayed in a reduced size.
You can specify a reduction of up to 80% from the original size.
