Hour function
Hour Function [Run-time]
Returns the hour from a time value that is generated by the TimeSerial or the TimeValue function.
Syntax:
Hour (Number)
Return value:
Integer
Parameters:
Number:
Numerical expression that contains the serial time value that is used to return the hour value.
TimeSerial
function.
It returns an integer value that represents the hour from a time value that is generated by the
TimeSerial
or the
TimeValue
function.
For example, the expression
Print Hour(TimeSerial(12,30,41))
returns the value 12.
Example:
Sub ExampleHour
Print &quot; The current hour is &quot; & Hour( Now )
End Sub
