(PHP 4 = 4.0.1)
This function takes data compressed by gzcompress() and returns the original uncompressed data or FALSE on error.
The function will return an error if the uncompressed data is more than 32768 times the length of the compressed input data or more than the optional parameter length.
See also gzdeflate(), gzinflate(), gzcompress(), gzencode().
