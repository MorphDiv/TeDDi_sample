자동 방향
EXIF 방향 정보에 따라 그림을 자동으로 회전할 지 여부.
그림 인터폴레이션
확대/축소에서 그림을 인터폴레이션을 할지 여부. 인터폴레이션을 하면 화질이 좋아지지만 하지 않는 경우보다 느립니다.
그림 엑스트라폴레이션
축소에서 그림을 엑스트라폴레이션을 할지 여부. 엑스트라폴레이션을 하면 화질이 좋아지지만 하지 않는 경우보다 느립니다.
투명한 부분 나타내기
투명한 부분을 어떻게 나타낼 지 지정합니다. 가능한 값은 CHECK_PATTERN, COLOR, 그리고 NONE입니다. COLOR의 경우 trans_color 키에서 어떤 색을 쓸 지 지정합니다.
휠로 확대/축소
스크롤 휠로 확대/축소를 할지 여부.
확대 비율
스크롤 휠로 확대/축소를 사용할 때 적용되는 비율. 각각의 스크롤 이벤트가 생길 때 적용되는 확대/축소 비율입니다. 0.05로 설정되면 각각의 스크롤 이벤트가 발생될 때 5%씩 확대증가가 되고, 1.00이면 100% 확대증가가 됩니다.
투명 색
transparency 키 값이 COLOR일 경우, 이 키 값이 투명한 부분에 사용될 색이 됩니다.
그림 순서를 반복합니다
무한 루프에서 그림 시퀀스를 반복해서 표시할 지 여부.
최초에 100%보다 크게 허용합니다
거짓이면, 최초에 작은 이미지를 화면에 맞게 늘리지 않습니다.
다음 그림을 표시할 때까지의 지연 시간 (초 단위)
0보다 큰 값이면 다음 그림으로 자동으로 넘어갈 때까지 유지할 시간입니다. 0이면 자동 넘어가기를 끕니다.

물어보지 않고 휴지통에 그림 버리기
참이면 그림을 휴지통에 옮길 때 확인 질문을 하지 않습니다. 참이라고 해도 파일 중에 휴지통에 옮기지 못하고 삭제하는 경우에는 확인 질문을 합니다.
그림을 읽어들이지 않았을 때 파일 선택 창에서 그림 폴더를 표시할지 여부.
참이고 창에서 그림을 읽어들이지 않은 경우, 파일 선택 창에서 해당 XDG 사용자 디렉터리를 사용해 사용자의 그림 폴더를 표시합니다. 거짓이거나 그림 폴더를 설정하지 않은 경우에는 현재 디렉터리를 표시합니다.
속성 창의 메타데이터 목록을 별도 페이지에 표시할지 여부.
참이면 속성 창에서 자세한 메타데이터 목록을 별도 페이지에 표시합니다. 이렇게 하면 넷북처럼 화면이 작은 경우에 속성 창 크기가 잘 맞습니다. 거짓이면 위젯이 "메타데이터" 페이지에 포함됩니다.
사용중인 플러그인
사용중인 플러그인 목록. 사용중인 플러그인의 "위치"는 포함하지 않습니다. 주어진 플러그인의 "위치"는 .eog-plugin 파일을 참고하십시오.
