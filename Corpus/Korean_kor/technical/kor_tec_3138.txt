마우스 왼쪽 버튼 선택
이것이 설정되면, 아이템을 왼쪽 클릭하는 것이 선택하는 것입니다.
왼쪽 마우스 버튼이 선택을 유지합니다.
이것이 설정되면, 아이템을 왼쪽 클릭하면 선택하는 것이지만 이미 선택된 다른 아이템은 선택해제 됩니다.
Shift/ Ctrl- 마우스 왼쪽 버튼 선택
이것이 설정되면, shift/ ctrl 과 왼쪽 마우스 클릭으로 아이템을 선택합니다. 알림: 만일 '마우스 왼쪽 버튼 선택' 이 선택되어 있으면, 이 설정은 아무 의미가 없습니다.
마우스 오른쪽 버튼 선택
이것이 설정되면, 아이템을 오른쪽 클릭하면 선택됩니다.
오른쪽 마우스 버튼이 선택을 유지합니다.
이것이 설정되면, 아이템을 오른쪽 클릭하면 선택됩니다, 하지만 이미 선택되었던 다른 아이템은 선택이 해제됩니다.
Shift/ Ctrl- 마우스 오른쪽 버튼 선택
이것이 설정되면, shift/ ctrl 과 오른쪽 마우스 클릭으로 아이템을 선택합니다. 알림: 만일 '마우스 오른쪽 버튼 선택' 이 선택되어 있으면, 이 설정은 아무 의미가 없습니다.
스패이스바로 선택 후 하이라이트바 아래로 이동
이것이 설정되면, 스패이스바를 누르면 현재 아이템을 선택하고 하이라이트바가아래로 이동하게됩니다. 설정되지 않으면, 현재 아이템이 선택되어지고, 하이라이트바는 현재 아이템에 그래로 머물게 됩니다.
스패이스바로 디스크 사용량 계산
이것이 설정되면, 선택 아이템이 폴더일 때 스패이스바를 누르면 (폴더 선택을 제외하고) 폴더가 디스크를 점유하여 사용하는 양을 계산해 줍니다. (재귀적으로)
아래로 이동 삽입
이것이 설정되면, INSERT 키를 누르면 현재 아이템을 선택할 수 있고, 하이라이트바가 다음 아이템으로 이동하게 됩니다. 설정되지 않으면, 현재 아이템이 변경되지 않습니다.
오른쪽 버튼 클릭을 하면 컨텍스트 메뉴가 바로 나타납니다.
이것이 설정되면, 마우스 오른쪽 버튼 클릭을 하면 곧바로 컨텍스트 메뉴가 보여기게 됩니다. 설정되지 않으면, 사용자는 마우스 오른쪽 버튼 클릭 상태에서 500ms( 0. 5초) 동안 기다려야 합니다.
로컬 복사/ 이동 시에 속성을 보존 (매우 느림)
Krusader 는 사용자 권한에 따른 로컨 파일의 모든 속성 (시간, 소유자, 그룹) 을 유지하려 할 것입니다: 당신이 Root( 관리자) 라면 사용자 속성을 유지하고, 그룹의 관리자 또는 멤버이면 그룹을 유지하며, 타임스탬프를 유지합니다. 알림: 이는 복사 진행률을 느리게 합니다.
