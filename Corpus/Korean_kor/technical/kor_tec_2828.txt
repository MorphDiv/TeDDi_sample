IP 주소는 TCP/ IP 네트워크의 유일한 주소입니다
네트워크 장치의 IP 주소
이 곳에 네트워크 인터페이스의 IP 주소를 입력하십시오. 메모: 네트워크가 인터넷에 연결되어 있으면 서비스 공급자가 할당한 IP 주소를 입력하십시오. 그렇지 않으면 개인 사용으로 예약되어 있는 다음 중 하나의 IP 주소를 입력하십시오. 대부분의 작은 사설 네트워크에서는 C 클래스를 사용하며, 255대의 컴퓨터를 사용할 수 있습니다. 192. 168. 1. 1, 192. 168. 1. 2, 192. 168. 1. 171과 같은 주소를 컴퓨터에 할당할 수 있습니다. 클래스 C 네트워크: 192. 168. 0. 0에서 192. 168. 255. 255까지. 예를 들어 192. 168. 0. 13. 클래스 B 네트워크: 172. 16. 0. 0에서 172. 31. 255. 255까지. 예를 들어 172. 28. 2. 5 클래스 A 네트워크: 10. 0. 0. 0에서 10. 255. 255. 255까지. 예를 들어 10. 5. 12. 14 컴퓨터에 할당한 IP 주소는 유일해야 합니다. 같은 주소가 하나 이상의 장치에 할당되어 있다면 문제가 생길 수 있습니다.
네트워크 장치의 서브넷 마스크
인터페이스의 서브넷 마스크를 입력하십시오. 작은 개인 네트워크에는 255. 255. 255. 0을 사용하십시오. 아래에 고급 설정을 입력하면 이 필드는 콤보 상자로 바뀝니다.
255. 255. 255. 0

255. 0. 0. 0
