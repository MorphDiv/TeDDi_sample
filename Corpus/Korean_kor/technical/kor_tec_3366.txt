LDAP 그룹 필터
이것은 그룹 항목에 사용할 필터를 지정합니다.
LDAP 사용자 RDN 접두사
이것은 사용자 항목에 사용할 접두사를 지정합니다.
사용자의 전체 이름이 cn( Canonical Name) 속성에 저장될 지 여부를 선택하십시오.
GECOS 필드 업데이트하기
GECOS 속성을 업데이트하려면 선택하십시오.
LDAP shadowAccount 개체 클래스 관리하기
사용자 항목에 shadowAccount 개체를 사용해야 할 지 여부를 선택하십시오. 이것은 암호 변경/ 만료 정책을 적용할 수 있도록 해 줍니다.
LDAP 구조적 개체 클래스
이 옵션을 사용하면 구조적 개체 클래스를 사용자 항목과 연결할 수 있습니다. 이 항목들을 인증뿐만 아니라 주소록에도 사용한다면 inetOrgPerson을 선택하십시오.
LDAP 그룹 RDN 접두사
이것은 그룹 항목에 사용할 접두사를 설정합니다.
LDAP 암호 해시 방법
이것은 암호 해시 방법을 지정합니다. 가장 좋은 방법은 SSHA입니다.
삼바 계정 관리 사용하기
삼바 도메인의 사용자/ 그룹 항목을 사용하려면 선택하십시오. 이렇게 하면 각각 항목에 대해서 sambaSamAccount 개체 클래스를 생성할 것이며, 삼바 버전 3. 0 이상에서 ldapsam 암호 DB 백엔드를 사용하는 경우 사용할 수 있습니다.
삼바 도메인 이름
삼바 도메인 이름을 지정합니다.
삼바 도메인 SID
이것은 도메인 보안 식별자( SID) 를 지정합니다. 각각 도메인 안에서 모두 같아야 합니다. 도메인 SID 값을 알아내려면 'net getlocalsid 도메인_ 이름' 을 실행하십시오.
수치적 RID 기본값
이 값은 RID와 UID, GID를 수치적으로 일치시키는 오프셋입니다. 기본값 및 최소값은 1000이며, 짝수여야 합니다. LDAP 데이터베이스와 smb. conf에도 같은 값을 지정해야 합니다.
삼바 로그인 스크립트
윈도 머신에 로그인할 때 실행될 (`Netlogon` 공유 안에 있는) 로그인 스크립트의 이름입니다.
삼바 홈 드라이브
윈도 머신에 로그인했을 때 사용자의 홈 디렉터리를 자동으로 매핑할 드라이브 문자를 지정하십시오.
삼바 프로필 경로 템플릿
사용자의 로밍 프로필 경로를 지정합니다. '% U' 매크로는 현재 사용자 이름으로 대체됩니다.
삼바 홈 경로 템플릿
사용자 홈 디렉터리의 위치를 지정합니다. 이 필드는 윈도 머신에서만 의미있는 필드입니다. '% U' 매크로는 현재 사용자 이름으로 대체됩니다.
LanManager 해시 암호를 sambaLMPassword 속성에 저장합니다. 네트워크에 윈도 9x 및 그 이전의 클라이언트가 있다면 이 옵션을 사용하십시오.
