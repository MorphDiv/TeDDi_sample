Параметры полей данных
сводной таблице
можно указать дополнительные параметры для полей данных столбцов, строк и страниц.
Сортировать по
Выберите поле данных, по которому будут упорядочиваться строки или столбцы.
По возрастанию
Сортировка значений от наименьшего значения до наибольшего.
Если выбранное поле совпадает с полем, для которого был открыт диалог, элементы сортируются по имени.
Если выбрано поле данных, элементы сортируются по итоговому значению выбранного поля данных.
По убыванию
Сортировка значений в порядке убывания от наибольшего значения до наименьшего.
Если выбранное поле совпадает с полем, для которого был открыт диалог, элементы сортируются по имени.
Если выбрано поле данных, элементы сортируются по итоговому значению выбранного поля данных.
Вручную
Значения сортируются в алфавитном порядке.
Параметры отображения
Можно указать параметры отображения для всех полей строки, за исключением последнего, самого внутреннего поля строки.
Разметка
В этом списке можно выбрать режим разметки для поля.
Пустая строка после каждого элемента
Этот параметр добавляет пустую строку после данных для каждого элемента в сводной таблице.
Показывать автоматически
Отображает верхние или нижние nn элементов при сортировке по указанному полю.
Показать
Включение функции автоматического показа.
элементы
Введите максимальное число элементов для автоматического показа.
От
Показ верхних или нижних элементов в указанном порядке сортировки.
С помощью поля
Выберите поле данных, по которому будут упорядочиваться данные.
Скрыть элементы
Выберите элементы, которые требуется скрыть от вычислений.
Иерархия
Выберите нужную иерархию.
Сводная таблица должна быть основана на внешнем источнике данных, содержащем иерархии данных.
