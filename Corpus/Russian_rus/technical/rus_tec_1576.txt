Параметры
Эффект
можно задать параметры.
Направление
Определяет направление для эффекта.
Ускоренный запуск
Этот параметр можно включить для назначения постепенно возрастающей скорости запуска эффекта.
Замедленное завершение
Этот параметр можно включить для назначения постепенно уменьшающейся скорости завершения эффекта.
Улучшение
Задание улучшений для текущего эффекта.
Звук
Выберите звук из галереи или выберите один из дополнительных пунктов.
Без звука
— анимация без звукового сопровождения.
Прекратить звук
— звук предыдущего эффекта останавливается при выполнении текущего эффекта.
Другой звук
— отображает диалоговое окно открытия файла для выбора звукового файла.
Кнопка &quot; Звук &quot;
Воспроизведение выбранного звукового файла.
После анимации
Выберите цвет, отображаемый после завершения анимации, или выберите другой элемент завершения эффекта из списка
:
Затемнение цветом
- после анимации фигура заполняется цветом затемнения.
Не затемнять
— никакого действия после завершения эффекта.
Скрыть после анимации
— скрывает фигуру после завершения анимации.
Спрятать при следующей анимации
- скрывает фигуру при следующей анимации.
Приглушить цвет
Выберите приглушенный цвет.
Анимация текста
Выберите режим анимации для текста текущей фигуры
:
Все вместе
— анимация всего текста сразу.
Слово за словом
— анимация текста пословно.
Буква за буквой
— побуквенная анимация текста.
Задержка между буквами
Указывает в процентах задержку между анимациями слов или букв.
