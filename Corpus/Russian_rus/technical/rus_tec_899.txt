База данных Adabas D
Adabas D компании Software AG устанавливается при установке коммерческого офисного программного обеспечения компании Sun Microsystems, Inc.
Если используется программное обеспечение OpenOffice.org, программный пакет Adabas D можно загрузить и установить отдельно (см. www.adabas.com).
База данных Adabas D, которую можно установить одновременно с $[officename], является ограниченной версией базы данных.
Дополнительные сведения об Adabas см. на веб-сайте Software AG http://www.softwareag.com/adabas/.
О программе Adabas
База данных Adabas D, которую можно установить одновременно с $[officename], ограничена размером 100 МБ и количеством пользователей в сети (3).
Дополнительные сведения см. в файле License.txt в каталоге Adabas.
База данных Adabas должна быть установлена в каталоге, отличном от каталога $[officename].
Имя файла Adabas может содержать не более 8 символов.
Путь к файлу Adabas, включая имя файла, не может содержать больше 30 символов, а также должен быть совместим с 7-разрядным кодом ASCII.
Как путь, так и имя файла не должны содержать пробелов.
В Windows программа установки Adabas добавляет переменную среды DBROOT, которая содержит сведения о пути к базе данных.
Если на момент установки такая переменная существует, база данных не устанавливается.

Чтобы удалить базу данных Adabas:
В Windows эти переменные определяются системными параметрами или в файле c:\Autoexec.bat.
В Unix эти переменные определяются скриптом запуска.
В каталоге $DBCONFIG$ удалите три файла с другими расширениями, имя которых совпадает с именем базы данных.
Например, для базы данных под именем &quot; Big &quot; удалите файлы &quot; Big.sys &quot; , &quot; Big.trs &quot; и &quot; Big.dat &quot; из каталога &quot; C:\Adabas\ &quot; .
В каталоге $DBCONFIG$\Config удалите файл, имя которого совпадает с именем базы данных, например &quot; C:\Adabas\Sql\Config\Big &quot; .
В каталоге $DBCONFIG$\Wrk удалите каталог, имя которого совпадает с именем базы данных, например &quot; C:\Adabas\Sql\Wrk\Big &quot; .
