Параметры показа слайдов
Определяет параметры показа слайдов, включая начальный слайд для показа, способ перехода между слайдами, тип презентации и параметры указателя.
Диапазон
Указывает слайды, включаемые в показ слайдов.
Все слайды
В показ слайдов включаются все слайды.
С:
Введите номер начального слайда.
Настраиваемый показ слайдов
Показ слайдов - Настраиваемый показ слайдов
.
Тип
Выберите тип показа слайдов.
По умолчанию
Слайд отображается во весь экран.
Окно
Показ слайдов осуществляется в окне программы $[officename].
Автоматически
Показ слайдов начинается заново после паузы указанной длины.
Между последним и начальным слайдами отображается слайд паузы.
Для прекращения показа используется клавиша ESC.
.
Длительность паузы
Укажите продолжительность паузы перед повторным показом слайдов.
Если ввести нулевое значение, показ начнется немедленно, без демонстрации слайда паузы.
Логотип в паузах
На слайде паузы отображается логотип $[officename].
Этот логотип не может быть заменен.
Параметры
Смена слайда вручную
Если этот флажок установлен, слайды никогда не сменяются автоматически.
Показывать указатель мыши
Указатель мыши отображается во время показа слайдов.
Указатель мыши в виде карандаша
Указатель мыши принимает вид пера, которым можно рисовать на слайдах во время презентации.
Все написанное пером теряется при выходе из показа слайдов.
Цвет пера не может быть изменен.
Показывать навигатор
Навигатор
.
Разрешить анимацию
Отображает все кадры анимированного GIF во время презентации.
Если этот параметр отключен, то отображается только первый кадр анимированного GIF.
Смена слайда по щелчку
Переход к очередному слайду происходит по щелчку в области фона слайда.
Презентация всегда на переднем плане
Окно $[officename] остается во время презентации на переднем плане.
Окна других программ не будут отображаться перед окном презентации.
see http://qa.openoffice.org/issues/show_bug.cgi?id=12719
Несколько мониторов
По умолчанию в режиме демонстрации используется основной монитор.
При отображении текущего рабочего стола на нескольких мониторах можно выбрать, какой монитор использовать в полноэкранном режиме демонстрации.
Если текущий рабочий стол охватывает только один монитор или если текущей системой не поддерживается функция использования нескольких мониторов, выбор другого монитора невозможен.
Монитор презентации
Выберите монитор для использования в полноэкранном режиме демонстрации.
Если в системе задана возможность охвата окном всех доступных мониторов, можно также выбрать &quot; Все мониторы &quot; .
В этом случае презентация охватывает все доступные мониторы.
Эта настройка сохраняется в конфигурации пользователя, а не в документе.
