Функция EqualUnoObjects [время выполнения]
Возвращает значение True, если два указанных объекта Basic Uno представляют один и тот же экземпляр объекта Uno.
Синтаксис:
EqualUnoObjects( Объект1, Объект2 )
Возвращаемое значение:
Bool
Пример:
// Копии объектов являются одним и тем же экземпляром
oIntrospection = CreateUnoService( &quot; com.sun.star.beans.Introspection &quot; )
oIntro2 = oIntrospection
print EqualUnoObjects( oIntrospection, oIntro2 )
// Копия структуры по значению является новым экземпляром
Dim Struct1 as new com.sun.star.beans.Property
Struct2 = Struct1
print EqualUnoObjects( Struct1, Struct2 )
