LDAP-подключение
Задает параметры импорта базы данных с помощью LDAP
(
Lightweight Directory Access Protocol - облегченный протокол доступа к каталогу
).
Эта страница доступна, только если в качестве адреса базы данных зарегистрирован сервер LDAP.
URL-адрес сервера
Введите имя сервера LDAP в формате &quot; ldap.server.com &quot; .
Доменное имя базы
Введите начальную точку поиска базы данных LDAP, например &quot; dc=com &quot; .
Номер порта
Укажите порт сервера LDAP, обычно 389.
Использовать безопасное подключение (SSL)
Создает безопасное подключение к серверу LDAP с помощью протокола безопасных соединений (Secure Sockets Layer, SSL).
По умолчанию SSL-подключение использует порт 636.
Обычное подключение использует порт 389.
