Память
Задание параметров для графического кэша и числа возможных шагов отмены.
Отменить
Определяет максимально-допустимое количество отменяемых операций.
Число шагов
В этом списке можно выбрать, сколько операций можно отменить.
Кэш графических объектов
Кэш графических объектов сохраняет содержащиеся в документе графические объекты в основной памяти компьютера.
Это означает, что атрибуты графического объекта, сохраненного в кэше, не нужно пересчитывать при возвращении на страницу с этим графическим объектом после прокрутки документа.
Использовать для $[officename] (МБ)
Указывает общий размер кэша для всех графических объектов.
Памяти на объект (МБ)
Указывает, что объекты, размер которых в мегабайтах превышает указанный, сохраняться в кэше не будут.
Удалить из памяти через (чч:мм)
Указывает время (часы и минуты), в течение которого графический объект остается в памяти.
Кэш для вставленных объектов
Количество объектов
Выберите максимальное число объектов OLE, которые могут находиться в кэше.
$[officename] Quickstarter
Загружать $[officename] во время запуска системы
Использовать быстрый запуск
Установите этот флажок, если необходимо активировать быстрый запуск $[officename].
Этот параметр доступен только при установленном модуле Quickstart
