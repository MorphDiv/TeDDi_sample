Панель форматирования
Панель форматирования появляется при переходе в режим изменения диаграммы.
Для перехода в режим изменения диаграммы дважды щелкните диаграмму.
Для выхода из режима изменения щелкните область за пределами диаграммы.
Изменить формат диаграммы можно с помощью элементов управления и значков на панели форматирования.
Выбор элемента диаграммы
Выберите элемент диаграммы, для которого требуется форматирование.
Этот элемент выделяется в окне предварительного просмотра диаграммы.
Для открытия диалогового окна свойств выбранного элемента щелкните &quot; Выбор формата &quot; .
Выбор формата
Используется для вызова диалогового окна свойств выбранного элемента.
Тип диаграммы
Используется для вызова диалогового окна &quot; Тип диаграммы &quot; .
Таблица данных диаграммы
Используется для вызова диалогового окна &quot; Таблица данных &quot; , в котором можно изменить данные диаграммы.
Показать/спрятать горизонтальную сетку
Значок &quot; Показать/спрятать горизонтальную сетку &quot; на панели форматирования используется для включения и выключения видимости сетки для оси X.
Примечание:
Это возможно только в том случае, если по пути &quot; Вставка – Сетки &quot; отменен выбор флажка &quot; Дополнительная сетка &quot; .
Показать/спрятать легенду
Для отображения или скрытия легенды щелкните &quot; Показать/спрятать легенду &quot; на панели форматирования.
Масштаб текста
Используется для изменения масштаба текста в диаграмме при изменении ее размера.
Автоматическая разметка
Перемещает все элементы диаграммы в положения по умолчанию на текущей диаграмме.
Эта функция не затрагивает тип диаграммы и другие атрибуты, кроме положения элементов.
