Выполнение команд SQL
С помощью команд SQL можно управлять базой данных напрямую, а также создавать и изменять таблицы и запросы.
Не все базы данных поддерживают полный набор инструкций SQL.
При необходимости можно узнать, какие команды SQL поддерживаются системой баз данных пользователя.
