Область настройки анимации
Назначение эффектов выбранным объектам.
Список анимаций
Перечень всех анимаций, используемых для текущего слайда.
На каждом слайде имеется одна основная анимация, которая запускается при показе слайда.
Можно включать дополнительные анимации, запускаемые при показе соответствующей фигуры.
Если на слайде есть такие анимированные фигуры, их перечень приводится в нижней части списка анимаций.
На ярлычках показаны имена фигур, для которых предусмотрена анимация.
Каждый элемент списка может содержать следующие поля (слева направо).
Знак плюс (+), указывающий на анимированные абзацы текста (может отсутствовать).
Во втором столбце показан порядковый номер выполнения анимации.
В третьем столбце содержится значок мыши, если анимация начинается по щелчку мыши, или значок часов, если анимация запускается по окончании предыдущей анимации.
В четвертом столбце содержится значок анимационного эффекта.
В пятом столбце показано имя фигуры для данного анимационного эффекта и первые символы анимированного текста (если таковой имеется).
Добавить
Настройка анимации
, где можно добавить еще один анимационный эффект для выбранного объекта слайда.
Изменить
Настройка анимации
, в котором можно изменить анимационный эффект для выбранного элемента списка анимаций.
Удалить
Удаляет выбранные анимационные эффекты из списка анимаций.
Начать
Способ запуска выбранного анимационного эффекта.
Возможны следующие способы запуска.
По щелчку
- анимация запускается щелчком мыши.
С предыдущей
- анимация запускается сразу.
После предыдущей
- анимация запускается, как только заканчивается предыдущая анимация.
