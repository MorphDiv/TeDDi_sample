Основные операции
Это главное окно Kscd. Вы должны увидеть что-то вроде этого после запуска Kscd. Описание кнопок приведено ниже.
Панель управления
Это главная панель управления для Kscd. Функции этих кнопок должны быть знакомы любому, кто хоть раз использовал CD плейер.
Верхняя кнопка переключает режим воспроизведения и паузы CD. Левая кнопка во втором ряду останавливает воспроизведение CD. Правая кнопка во втором ряду включает непрерывное воспроизведение, при достижении конца последней дорожки CD начнется воспроизведение диска с начала. Кнопки в третьем ряду перемещают воспроизведение CD назад (налево) или вперёд (направо) на 30 секунд. Кнопки в четвертом ряду устанавливают воспроизведение следующей дорожки (направо) или предыдущей (налево).
Кнопка Извлечь
Эта кнопка извлекает CD. Заметьте, что кнопка извлечения CD на вашем CD-ROM е может не работать, если в нем находится аудио CD и запущен Kscd. Некоторые очень старые CD-ROM 'ы могут не поддерживать программную кнопку извлечения. Также, возможно, вам понадобится вручную закрыть CD-ROM после вставки нового диска.
Экран состояния
Это экран состояния. Вверху, справа налево, показывается основное время (см. ниже описание различных режимов показа времени), состояние CD-ROM 'а, общее время воспроизведения аудио CD, текущая установка уровня громкости, номер текущей дорожки и общее число дорожек (текущ. / общ.). На двух нижних рядах показывается: артист и название CD, название дорожки, если соответствующие записи будут найдены в локальной или сетевой базе CDDB (CD Data Base.)
Режимы показа времени
Эта кнопка переключает режимы показа времени. По умолчанию, Kscd показывает оставшееся время текущей дорожки, если CD воспроизводится, и: или 00:00 если CD не воспроизводится. При нажатии на эту кнопку режимы показа переключаются в следующем порядке: оставшееся время дорожки, всего прошло времени и всего осталось времени.

Кнопка CDDB

Эта кнопка поможет вам найти информацию об артисте в интернете. Вы можете найти информацию о датах выступления артиста, о ценах и другое, нажав эту клавишу и выбрав соответствующую опцию в всплывающем меню.
Кнопка выхода

Уровень громкости
Этим регулятором громкости настраивается выход аудио CD. Направо - громче, налево - тише. Если вы слушаете ваш CD через звуковую карту, то микшер звуковой карты тоже будет влиять на уровень громкости.

Выбор дорожки

Закладка freedb
Закладка freedb позволяет настроить функцию CDDB для Kscd.
Опция Разрешить доступ к freedb включает или выключает просмотр CDDB в Интернете. Можно задать время, в течение которого программа будет пытаться искать запись в Интернете. Для использования HTTP -прокси, отметьте Доступ к freedb через HTTP-прокси и введите URL в поле ниже.
Текстовое поле Локальный каталог для хранения данных freedb: позволяет вам выбрать каталог, где бы вы хотели хранить записи CDDB.
Раскрывающийся список Сервер freedb: показывает список известных freedb серверов. Выберите запись из этого листа, и она появится в текстовом поле под раскрывающимся списком. Нажмите кнопку Обновить для обновления списка серверов. Нажмите кнопку По умолчанию чтобы ввести значение по умолчанию, freedb.freedb.org cddbp 888 -, которое позволит вам обновить список cddb серверов, перечисленных на главном сайте freedb freedb.freedb.org. Нажмите кнопку - для удаления выбранной записи из списка.
Поле Адрес для отправки дополнений freedb: содержит e-mail адрес для отправки записей CDDB. Адрес по умолчанию freedb-submit@freedb.org. Как и в версии 0.8, записи CDDB из Kscd принимаются для добавления в базу данных. Нажмите клавишу - для удаления выбранной записи из списка. Кнопка По умолчанию устанавливает все значения заданными по умолчанию.Кнопка Помощь открывает файл помощи по Kscd (этот документ).
Кнопка OK сохраняет текущие настройки и закрывает окно; Кнопка Отмена закрывает окно без сохранения.
Закладка Настройки SMTP
Закладка настроек SMTP служит для настройки соединения с почтовым сервером, чтобы отправлять новые записи CDDB. Это полезно, если ваша система не является сервером. Отметьте флажок Отправлять дополнения по почте для включения этой функции. Введите адрес SMTP -сервера в верхнем левом текстовом поле, а номер порта (обычно 25) в верхнем правом поле. Введите адрес вашей электронной почты в нижнем текстовом поле.
Закладка Параметры KSCD
Поля Цвет индикатора: и Цвет фона: показывают цвета выбранные для индикатора и фона дисплея. Нажмите кнопку Изменить рядом с полем цвета для выбора нужного цвета.
Поле Устройство CDROM: содержит наименование устройства CD-ROM используемого для проигрывания аудио CD. Значение по умолчанию - / dev / cdrom. Данное устройство должно быть доступно только для чтения. Изменение прав доступа к этому устройству потребует прав администратора и может быть сделано из командной строки или в режиме администратора в Konqueror.
Флажок Показывать подсказки включает или выключает подсказки (маленькие окна, которые появляются, когда курсор мыши останавливается над каким-либо элементом окна на некоторое время).
Флажок Встраивать пиктограмму в панель позволяет вставлять управление Kscd в панель Kicker.
Флажок Начинать воспроизведение при вставке CD позволяет воспроизводить CD сразу после вставки, без необходимости нажимать кнопку Воспроизвести.
Флажок Останавливать воспроизведение при выходе позволяет останавливать воспроизведение CD при выходе из Kscd.
Флажок Извлечь CD по окончании воспроизведения позволяет извлечь CD по окончании воспроизведения.
Случайное воспроизведение без повтора дорожек Означает что, когда используется режим Случайная дорожка, каждая дорожка будет воспроизводится только один раз.
Кнопка Помощь открывает справочник по Kscd.
Кнопка OK сохраняет текущие настройки и закрывает окно; Кнопка Отмена закрывает окно без сохранения.

Редактор базы данных CD
Если существует запись в вашей локальной базе данных CDDB (смотрите подраздел CDDB в главе Настройка) для CD находящемся в CD-ROM, или, если диск будет найден в CDDB, вы увидете имя артиста и название CD в поле Артист / название диска и список дорожек с названиями в поле Дорожка / время / название. Иначе, вы увидете список дорожек и время воспроизведения без названий.
Вы можете добавить комментарий для всего диска, нажав на кнопку Комментарий рядом с полем Артист / название диска, или для выбранной дорожки в поле Дорожка / время / название нажав на кнопку Комментарий, находящуюся рядом. Если вы выберете дорожку в поле Дорожка / время / название, название, если оно есть, появится в поле Название выделенной дорожки. Вы можете написать название дорожки в этом поле или отредактировать запись так, как вам хочется. Нажмите кнопку Ввод на клавиатуре, и текст появится в соответствующей линии списка дорожек.
Когда будут даны названия всем дорожкам и поле Артист / название диска будет заполнено, вы можете нажать кнопку Отправить для отправки вашей записи по электронной почте на freedb.
Экран Полное время: показывает общее время воспроизведения CD.
Редактор списка воспроизведения позволяет изменить порядок воспроизведения диска. Введите номера дорожек в нужном порядке, разделенные запятыми.
Кнопка Сохранить сохраняет запись на диск. Укажите категорию, под которой сохранить запись. Выберите каталог и нажмите Ok.
Кнопка Загрузить позволяет указать каталог для поиска записей CDDB, отличных от находящихся на сервере CDDB или в локальной базе данных CDDB. Кнопка Закрыть закрывает редактор базы данных CD без сохранения.
Использование Kscd в панели KDE
Когда флажок Встраивать пиктограмму в панель отмечен, маленький аплет Kscd появится на Панели KDE. В отличие от главного окна Kscd, этот аплет доступен на любом рабочем столе. При нажатии правой кнопки мыши на аплете появляется меню управления воспроизведением CD. При нажатии левой кнопки мыши на аплете открывается главное окно Kscd. Если главное окно свернуто, то оно развернется.
Поддержка CDDB
freedb Data Base - это распределенная сетевая база данных, доступная через Интернет, которая содержит информацию практически обо всех аудио CD. Если у вас есть доступ к Интернету, очевидно, вам не придется вводить вручную информацию о ваших CD, конечно, если вы все правильно настроили. См. подраздел Закладка freedb в главе о конфигурации Kscd для получения информации по настройке этого сервиса и раздел Редактор базы данных CD для получения информации по редактированию записей CDDB.
Использование CDDB свободное. Поощряются дополнения от пользователей.
Когда вы подготавливаете записи для CDDB, помните следующее:
Используйте стандартные латинские буквы в записях. Некоторые специальные символы поддерживаются, но, например, кириллица или греческие символы недопустимы.
Используйте только символ / в поле Артист / название диска редактора базы данных CD.
Для CD с классикой, поставьте имя композитора в разделе Артист (перед наклонной чертой) и имя исполнителя в разделе Название диска (после наклонной черты).
Если вы отправляете запись, которая уже существует в базе данных, то любая дополнительная информация, которую вы введете, может быть добавлена к существующей записи.
По умолчанию Kscd устанавливает стандартные категории CDDB в $KDEDIR / share / apps / kscd / cddb. Вы можете создать столько категорий, сколько вам нужно. Однако, когда вы отправляете запись, будут показаны только общеупотребительные категории CDDB. Адрес для отправки по умолчанию - это freedb-submit@freedb.org. Для получения более детальной информации по freedb посетите домашнюю страницу freedb.
Вопросы и ответы
Я вижу этот диалог после старта Kscd. Что неправильно?
Это означает, что Kscd не смог открыть ваше устройство CD-ROM. Имя устройства в поле Устройство CDROM в Настройка Kscd должно соответствовать блочному устройству, связанному с вашим CD-ROM приводом. Чаще всего, это ссылка на соответствующее IDE (/ dev / hdx) или SCSI (/ dev / sdx) устройство.
Файл устройства обычно принадлежит пользователю root в группе root, и обычный пользователь не имеет права открывать его для чтения, записи или исполнения непосредственно. Это не имеет ничего общего с SUID -программой (Set User ID программы используют функцию setuid() в стандартной библиотеке Un*x, oтвечающую за идентификацию пользователя) mount, которая не имеет проблем с правами доступа; Kscd должен иметь возможность получать дескриптор файла только для чтения при обращении к CD устройству для управления CD-ROM привода и читать raw-данные с диска.
Также, см. раздел Настройка KSCD
Я не могу заставить работать CDDB. Могу я получить более детальную информацию, что работает не так, как надо?
Если у вас есть проблемы с работой CDDB, попробуйте запустить Kscd из командной строки с опцией -d и просмотрите вывод отладочной информации.
