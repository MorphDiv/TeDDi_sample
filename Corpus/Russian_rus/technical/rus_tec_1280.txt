Геометрия
Определяет свойства формы выделенного трехмерного объекта.
Скругленные края
Введите значение, определяющее степень скругления углов выделенного трехмерного объекта.
Масштаб глубины
Введите значение, на которое необходимо увеличить или уменьшить область передней стороны выделенного трехмерного объекта.
Угол поворота
Ввод угла в градусах для поворота выбранного трехмерного объекта.
Глубина
Введите расстояние, на которое должен выступать выделенный трехмерный объект.
Этот параметр недействителен для трехмерных тел вращения.
Сегменты
Можно изменить число сегментов, используемое для рисования трехмерного тела вращения.
Горизонтально
Введите число горизонтальных сегментов для использования в выбранном трехмерном теле вращения.
Вертикально
Введите число вертикальных сегментов для использования в выбранном трехмерном теле вращения.
Нормаль
Позволяет изменить стиль визуализации поверхности трехмерного объекта.
Для объекта
Отображает трехмерную поверхность в соответствии с формой объекта.
Например для визуализации круглой формы используется сферическая поверхность.
