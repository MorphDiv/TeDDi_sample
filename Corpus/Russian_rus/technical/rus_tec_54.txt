мастера диаграмм
можно выбрать отображаемые элементы диаграммы.
titles title
Введите заголовок диаграммы.
subtitle
Введите подзаголовок для диаграммы.
x axis
Введите подпись для оси X (горизонтальной).
y axisx axis
Введите подпись для оси Y (вертикальной).
z axis
Введите подпись для оси Z.
Этот параметр доступен только для трехмерных диаграмм.
legend
Отображение легенды для диаграммы.
legend left
Легенда располагается слева от диаграммы.
top
Легенда располагается над диаграммой.
right
Легенда располагается справа от диаграммы.
bottom
Легенда располагается под диаграммой.
grid x axis
Отображаются линии сетки, перпендикулярные оси X.
y axis
Отображаются линии сетки, перпендикулярные оси Y.
z axis
Отображаются линии сетки, перпендикулярные оси Z.
Этот параметр доступен только для трехмерных диаграмм.
secondary x axis ( &quot; 2nd x axis &quot; in spec doc)
Ввод подписи для дополнительной оси X.
Этот параметр доступен только для диаграмм, поддерживающих дополнительную ось X.
secondary y axis ( &quot; 2nd y axis &quot; in spec doc)
Ввод подписи для дополнительной оси Y.
Этот параметр доступен только для диаграмм, поддерживающих дополнительную ось Y.
Ввод элементов диаграммы
Введите заголовки или щелкните по элементам, которые необходимо вывести на текущей диаграмме.
Заголовки
При вводе текста для заголовка, подзаголовка или оси будет зарезервировано необходимое пространство для отображения текста рядом с диаграммой.
Если текст не вводится, то пространство не резервируется, и остается больше места для отображения диаграммы.
Связать текст заголовка и ячейку невозможно.
Следует ввести текст напрямую.
Формат
.
Легенда
Легенда отображает подписи из первого ряда или столбца или из диапазона, который был задан в диалоговом окне &quot; Ряд данных &quot; .
Если диаграмма не содержит подписи, в легенде будет отображаться текст в виде &quot; Строка 1, Строка 2 ... &quot; или &quot; Столбец A, Столбец B ... &quot; в соответствии с номером ряда или буквой столбца данных диаграммы.
Ввести текст напрямую невозможно; он генерируется автоматически на основе имени диапазона ячеек.
Выберите один из вариантов расположения.
После завершения настройки диаграммы ее позицию можно изменить с помощью меню Формат.
Сетки
grid x axis
Отображаются линии сетки, перпендикулярные оси X.
y axis
Отображаются линии сетки, перпендикулярные оси Y.
z axis
Отображаются линии сетки, перпендикулярные оси Z.
Этот параметр доступен только для трехмерных диаграмм.
Видимые линии сетки позволяют более точно оценить расположение значений в диаграмме.
Расстояние линий сетки соответствует настройкам интервала на вкладке &quot; Масштаб &quot; свойств оси.
Для круговых диаграмм линии сетки недоступны.

Для дополнительных элементов используйте меню диаграммы &quot; Вставить &quot; в режиме редактирования.

Statistics, for example mean values, y error bars and trend lines
