Этот параметр для режима синхронизации. Режим синхронизации может использоваться для внесения изменений в две ветки одновременно. Укажите для этого параметра путь, который соответствует корневому каталогу ветки проекта, и вторичная синхронизация будет автоматически открывать файлы ветки. Затем, каждый раз, как вы будете вносить изменения в файлы основной ветки, они будут автоматически скопированы в ветку (конечно, если они содержат те же английские строки). « Справочные » переводы: this message contains text from documentation, so use its translation when you' re translating docs
Папка с файлами- источниками альтернативных переводов: Укажите тут путь к папке со структурой, схожей с корневой. При каждом открытии файла, в каталоге с альтернативным переводом будет производится поиск соответствующего файла, и, если он будет найден, он будет использован для панели альтернативных переводов. Возможно вы захотите использовать переводы другого языка (например другого подпроекта), близкого к вашему. Также вы можете использовать переводами другого подпроекта или немедленно или предварительно перевести их с помощью компьютера. Смотрите документацию для подробностей.
Для облегчения перевода KDE, Lokalize может захватывать текст из приложений пользователя и искать файлы, содержащие этот текст. Когда отмечено, захват можно осуществить нажатием средней кнопки мыши на нередактируемом элементе графического интерфейса пользователя (виджете). После этого, переключитесь на окно Lokalize и выберете уведомление с текстом. Даже если несколько файлов содержат этот текст, в большинстве случаев будет выбран правильный. Помните, что для использования этой функции, вам надо иметь актуальную память переводов для всех файлов.
