Панели ошибок для Y
/text/schart/01/04050000.xhp
Y Error Bars
i85330
Планки погрешностей для Y
указывается необходимость отображения планок погрешностей для двумерных диаграмм.
Панель ошибки - это линия-указатель, охватывающая область от y - NegativeErrorValue до y + PositiveErrorValue.
Здесь y - значение точки данных.
При выбранном &quot; стандартном отклонении &quot; y - среднее значение ряда данных.
NegativeErrorValue и PositiveErrorValue - это величины, рассчитанные функцией панели ошибки или указанные явно.
Вставка - Панели ошибок для Y
доступна только для двумерных диаграмм.
Категория погрешностей
Категория погрешностей
можно выбрать различные виды отображения категории погрешностей.
Нет
Не показывать панели значений погрешностей.
Постоянное значение
Показывает постоянные значения, назначенные в области &quot; Параметры &quot; .
Процент
Отображение процента.
Отображаемые данные относятся к соответствующей точке данных.
Процент задается в поле &quot; Параметры &quot; .
Функции
Выберите функцию для расчета панелей погрешностей.
Стандартная погрешность:
Показывает стандартную погрешность.
Отклонение:
Показывает отклонение, рассчитанное на основе количества точек данных и соответствующих значений.
Стандартное отклонение:
Показывает стандартное отклонение (квадратный корень отклонения).
Граница погрешностей:
Отображает наивысшую границу погрешности в процентах в соответствии с максимальным значением группы данных.
Процент задается в поле &quot; Параметры &quot; .
Диапазон ячейки
Нажмите на область ячеек и укажите тот диапазон ячеек, откуда будут браться положительные и отрицательные значения погрешностей.
Нажмите на кнопку, чтобы свернуть диалоговое окно, затем с помощью мыши выделите диапазон ячеек в листе.
Нажмите на эту кнопку снова, чтобы развернуть диалоговое окно.
Из таблицы данных
При использовании графика с собственными данными, значения погрешностей можно вводить в таблицу данных графика.
В диалоговом окне &quot; Таблица данных &quot; показываются дополнительные столбцы &quot; Панели положительных погрешностей Y &quot; и &quot; Панели отрицательных погрешностей Y &quot; .
Positive (+)
Введите значение, которое необходимо добавить к отображаемому значению в качестве положительного значения погрешности.
Negative (-)
Введите значение, которое необходимо вычесть из отображаемого значения в качестве отрицательного значения погрешности.
Positive (+) when Cell Range is selected
Введите диапазон адресов для получения положительных значений погрешности.
Используйте кнопку &quot; Свернуть &quot; , чтобы выбрать диапазон в листе.
Negative (-) when Cell Range is selected
Введите диапазон адресов для получения отрицательных значений погрешности.
Используйте кнопку &quot; Свернуть &quot; , чтобы выбрать диапазон в листе.
Одинаковое значение для обоих значений
Установите флажок в поле, чтобы использовать положительные значения погрешностей в качестве отрицательных.
Изменить это значение можно только в окне &quot; Положительные (+) &quot; .
Это значение автоматически копируется в поле &quot; Отрицательные (-) &quot; .
Индикатор погрешностей
Указывает индикатор погрешностей.
Все значения
Показывать панели с положительными и отрицательными значениями погрешностей.
Положительные
Показывать только панели положительных значений погрешностей.
Отрицательные
Показывать только панели отрицательных значений погрешностей.
