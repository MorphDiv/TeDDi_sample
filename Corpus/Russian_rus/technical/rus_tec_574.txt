Текстовые эффекты
Панель инструментов текстовых эффектов открывается при выделении объекта текстового эффекта.
Галерея текстовых эффектов
Открывает галерею текстовых эффектов, где можно выбрать другое представление.
Нажмите кнопку &quot; ОК &quot; , чтобы применить новый набор свойств к объекту текстового эффекта.
Форма текстового эффекта
Открывает панель инструментов &quot; Форма текстового эффекта &quot; .
Щелкните форму, чтобы применить форму ко всем выбранным объектам текстовых эффектов.
Текстовый эффект с одинаковой высотой букв
Изменение высоты букв выделенных объектов текстовых эффектов с обычной на одинаковую высоту для всех объектов.
Выравнивание текстовых эффектов
Открывает окно &quot; Выравнивание текстовых эффектов &quot; .
Щелкните, чтобы применить выравнивание к выделенным объектам текстовых эффектов.
Межсимвольный интервал для текстового эффекта
Открывает окно &quot; Межсимвольный интервал для текстового эффекта &quot; .
Щелкните, чтобы применить межсимвольный интервал к выделенным объектам текстовых эффектов.
Настройка
Открывает диалоговое окно &quot; Межсимвольный интервал для текстового эффекта &quot; , где можно ввести новое значение для межсимвольного интервала текстового эффекта.
Значение
Введите значение для межсимвольного интервала текстового эффекта.
