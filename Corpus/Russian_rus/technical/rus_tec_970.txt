Параметры экспорта EPS
Определяет несколько параметров экспорта для файлов EPS.
Необходимо печатать файл EPS с помощью принтера PostScript.
Другие принтеры будут печатать только вставленный предварительный просмотр.
Предварительный просмотр
Определяет параметры для вставленного предварительного просмотра для файла EPS.
Предварительный просмотр картинки (TIFF)
Указывает, будет ли предварительный просмотр изображения экспортироваться в формате TIFF вместе с фактическим файлом PostScript.
Обмен (EPSI)
Указывает, будет ли монохромный предварительный просмотр графического объекта в формате EPSI экспортироваться вместе с файлом PostScript.
Этот формат содержит только печатаемые символы из 7-разрядного кода ASCII.
Версия
Определяет уровень PostScript для экспорта файла EPS.
Уровень 1
Уровень 1
, если принтер PostScript не предлагает возможности уровня 2.
Уровень 2
Уровень 2
, если устройство вывода поддерживает цветные растровые изображения, графические изображения в цветовой палитре и сжатые графические изображения.
Цветной формат
Цветной формат
предоставляет возможность выбора между экспортом в цвете или в оттенках серого.
Цвет
Экспортирует файл в цвете.
Оттенки серого
Экспортирует файл в оттенках серого.
Сжатие
Указывает, экспортировать ли данные со сжатием.

Кодировка LZW

Нет
