& Название:
Имя будет показано в меню 'Инструменты - > Внешние'
Сценарий:
Сценарий для выполнения вызова инструмента. Сценарий будет передан для выполнения / bin/ sh. Следующие макросы будут развёрнуты:% URL - URL текущего документа.% URLs - список URL всех открытых документов.% directory - URL каталога, содержащего текущий документ.% filename - имя файла текущего документа.% line - текущая строка, где находится курсор в данном просмотре.% column - колонка, где находится курсор в данном просмотре..% selection - выделенный текст в данном просмотре.% text - текст текущего документа.
& Исполняемый файл:
Программа, выполняющая команду. Используется, чтобы проверить доступность инструмента. Если не заполнено, будет использовано первое слово команды.
Типы & MIME:
Список типов MIME, разделённый точкой с запятой. Этот список будет ограничивать применение инструмента только для обработки файлов определённых типов. Если поле оставить пустым, инструмент можно использовать для любых документов. Для выбора типов MIME нажмите кнопку справа.
Щёлкните для открытия диалога, который может помочь вам создать список типов MIME.
& Сохранить:
Нет
Текущий документ
Все документы
Вы можете выбрать сохранение текущего документа или всех (изменённых) документов перед запуском команды. Это полезно, если вы хотите передать URL таким приложениям как, например, клиент FTP.
& Команда правки:
При нажатии F7 появляется строка для команд правки. Вы сможете вызвать этот внешний инструмент, указав exttool- название_ команды. Не используйте пробелы и табуляцию в названии.
