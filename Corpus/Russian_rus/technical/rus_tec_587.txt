Поддержка платформы Java
В $[officename] поддерживается платформа Java, используемая для выполнения приложений и компонентов на основе архитектуры JavaBeans.
Для работы с Java в $[officename] следует установить виртуальную машину Java 2 корпорации Sun Microsystems.
После окончания установки $[officename] автоматически появится возможность установки этих файлов, если они еще не были установлены.
При необходимости можно установить эти файлы прямо сейчас.
Запускаемый файл установки расположен в папке $[officename]/program и на компакт-диске $[officename].
Чтобы запускать приложения Java, необходимо включить поддержку платформы Java в $[officename].

.

После добавления информации о пути поиска перезапустите %PRODUCTNAME.
