параметры прокси-сервера
Прокси
Прокси-сервер
для доступа к Интернету можно указать вручную.
Параметры
прокси-сервера
.
Прокси-сервер
Выбор типа определения прокси-сервера.
Нет
Доступ к Интернету без прокси-сервера.
Позволяет настроить прямое подключение компьютера к поставщику услуг Интернета без использования прокси-сервера.
Вручную
Позволяет указать прокси-сервер вручную.
Укажите прокси-серверы в соответствии со службой доступа к Интернету.
Информацию о прокси-серверах и портах можно получить у системного администратора.
Введите имена серверов без префикса протокола.
Например, введите www.sun.com, а не http://www.sun.com.
Системный
В системах Windows или UNIX с использованием Gnome этот параметр указывает %PRODUCTNAME использовать системные параметры.
Для инициализации этого параметра необходимо перезапустить %PRODUCTNAME.
HTTP-прокси
HTTP
Введите номер порта в поле справа.
HTTPS-прокси
Выберите имя прокси-сервера для HTTPS.
Введите порт в правое поле.
FTP-прокси
FTP
Введите номер порта в поле справа.
Без прокси для
Перечисление через точку с запятой имен серверов, для которых не требуется использование прокси-серверов.
Это серверы локальной сети и серверы, используемые, например, для потокового видео и аудио.
Кроме того, можно использовать подстановочные знаки в именах узлов и доменов.
Например, введите *.sun.com для обращения ко всем узлам в домене sun.com без прокси.
Порт
Укажите порт для соответствующего прокси-сервера.
Максимальное значение для номера порта - 65535.
