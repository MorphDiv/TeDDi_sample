Управление
Задайте параметры для выбранного стиля.
Имя
Отображение имени выбранного стиля.
При создании или изменении пользовательского стиля введите имя этого стиля.
Имя предопределенного стиля менять нельзя.
Автообновление
Обновление стиля при применении прямого форматирования к абзацу, который был отформатирован этим стилем.
Форматирование всех абзацев с таким стилем автоматически обновляется.
Обновление стиля при применении прямого форматирования к абзацу, который был отформатирован с использованием этого стиля.
Форматирование всех абзацев с таким стилем автоматически обновляется.
Следующий стиль
Выберите имеющийся стиль, который будет использоваться следующим за текущим стилем в документе.
Для стилей абзацев следующий стиль применяется к создаваемому абзацу при нажатии клавиши ВВОД.
Для стилей страницы следующий стиль применяется при создании новой страницы.
Связан с
Выберите имеющийся стиль, на котором будет основан новый стиль, или выберите &quot; Нет &quot; , чтобы определить собственный стиль.
Категория
Отображается категория текущего стиля.
При создании или изменении нового стиля выберите из списка значение &quot; Пользовательский стиль &quot; .
Категорию предопределенного стиля изменять нельзя.
Содержит
Описание форматирования, применяемого в текущем стиле.
Назначение сочетаний клавиш
Сервис - Настройка - Клавиатура
, где можно назначить сочетание клавиш для текущего стиля.
