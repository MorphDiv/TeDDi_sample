Функция CONVERT_ADD
CONVERT_ADD
Преобразует значение в одной единице измерения в соответствующее значение в другой единице измерения.
Единицы измерения можно вводить как текст в кавычках или как ссылку.
При вводе единиц измерения в ячейки они должны совпадать с указанными ниже единицами измерения.
При вводе учитывается регистр.
Например, чтобы ввести в ячейку строчную &quot; l &quot; (литр), введите после нее апостроф ( ' ).
Свойство
Единицы
Масса
g, sg, lbm, u, ozm, stone, ton, grain, pweight, hweight, shweight
Длина
m, mi, Nmi, in, ft, yd, ang, Pica, ell, parsec
Время
yr, day, hr, mn, sec
Давление
Pa, atm, mmHg, Torr, psi
Сила
N, dyn, pond
Энергия
J, e, c, cal, eV, HPh, Wh, BTU
Мощность
W, HP, PS
Напряженность поля
T, ga
Температура
C, F, K, Reau, Rank
Объем
l, tsp, tbs, oz, cup, pt, qt, gal, m3, mi3, Nmi3, in3, ft3, yd3, ang3, Pica3, barrel, bushel, regton, Schooner, Middy, Glass
Площадь
m2, mi2, Nmi2, in2, ft2, yd2, ang2, Pica2, Morgen, ar, acre, ha
Скорость
m/s, m/h, mph, kn, admkn
Каждой единице измерения может предшествовать префикс из следующего списка:
Разрешенные префиксы
10^( < 0)
d, c, m, u, n, p, f, a, z, y
10^( > 0)
e, h, k, M, G, T, P, E, Z, Y
Для единиц &quot; bit &quot; and &quot; byte &quot; также можно использовать следующие префиксы IEC 60027-2/IEEE 1541:
Кбайт киби 1024
Мбайт меби 1048576
Гбайт гиби 1073741824
Тбайт теби 1099511627776
Пбайт пеби 1125899906842620
Эбайт эксби 1152921504606850000
Збайт зеби 1180591620717410000000
Йбайт йоби 1208925819614630000000000
Синтаксис
CONVERT_ADD(Число; &quot; Из_единиц &quot; ; &quot; В_единицы &quot; )
Число
: число для преобразования.
Из_единиц
: исходная единица измерения.
В_единицы
: целевая единица измерения.
Обе единицы должны быть одного типа.
Примеры
=CONVERT_ADD(10; &quot; HP &quot; ; &quot; PS &quot; )
возвращает значение 10,14 (с округлением до двух знаков после запятой).
10 HP равно 10,14 PS.
=CONVERT_ADD(10; &quot; km &quot; ; &quot; mi &quot; )
возвращает значение 6,21 (с округлением до двух знаков после запятой).
10 километров равно 6,21 мили.
Префикс &quot; k &quot; соответствует коэффициенту 10^3.
функция FACTDOUBLE
факториалы; числа с приращением 2

FACTDOUBLE
Возвращает двойной факториал числа.
Синтаксис
FACTDOUBLE(Число)
число
!!
, т.е. двойной факториал
числа
, где
число
является целым числом, большим или равным нулю.
Для четных чисел FACTDOUBLE(n) возвращает:
2*4*6*8* ... *n
Для нечетных чисел FACTDOUBLE(n) возвращает:
1*3*5*7* ... *n
FACTDOUBLE(0) по определению возвращает 1.
Пример
=FACTDOUBLE(5)
возвращает значение 15.
=FACTDOUBLE(6)
возвращает значение 48.
=FACTDOUBLE(0)
возвращает значение 1.
see also FACT, MULTINOMIAL, PRODUCT
