Полупрозрачные панели инструментов при перемещении
Отбрасывание тени недоступно.
Если включить этот параметр, можно установить различные эффекты для выпадающих списков, меню и подсказок.
Отключить: Не использовать эффекты выпадающих списков. Анимировать: Анимировать выпадающие списки.
Отключить: Не использовать эффекты подсказок. Анимировать: Анимировать подсказки. Плавное исчезновение: Использовать альфа- канал для плавного исчезновения подсказок.
Отключить: Не использовать эффекты меню. Анимировать: Анимировать меню. Плавное исчезновение: Fade in menus using alpha- blending. Сделать полупрозрачным: Сделать меню полупрозрачными (только для стилей KDE)
Если установлен этот параметр, выпадающие меню будут отбрасывать тень. Этот эффект доступен только для стилей KDE.
Перемешением этого движка можно настроить степень прозрачности меню.
Примечание: все элементы в этом выпадающем списке не применимы к приложениям Qt, собранным без поддержки KDE.
Если установлен этот параметр, панели инструментов при перемещении будут полупрозрачными.
Если установлен этот параметр, некоторые выпадающие меню будут иметь так называемые рукоятки отделения. Если щелкнуть на этой рукоятке, вы получите меню в отдельном элементе. Это может быть очень полезным при многократном выполнении одного и того же действия.
