Параметры шифрования GPG
На этой вкладке вы можете настроить параметры шифрования содержимого файла KMyMoney посредством GPG. Если GPG недоступна в вашей системе, параметры будут неактивны. В этом случае проверьте правильность работы GPG для текущего пользователя. Возможность дополнительного шифрования доступна если для пользователя есть ключ kmymoney- recover@ users. sourceforge. net с id 0x8AFDDC8E.
Шифровать содержимое файла ключом GPG
Шифрование ключом GPG
Ключ
В этом списке показаны все закрытые ключи, которые есть у текущего пользователя. Выберите один из этих ключей для шифрования содержимого файла при его сохранении.
Введите идентификатор дополнительного ключа, которым вы хотите дополнительно шифровать данные. Это может быть как адрес электронной почты, так и идентификатор в шестнадцатеричном виде, начинающийся с 0x.
Дополнительные ключи
Если указанный ключ найден, индикатор показывается ярко- зелёным. Если нет — тёмно- зелёным.
Если ключ для восстановления найден, показывается ярко- зелёным. Если нет — тёмно- зелёным.
Доступно шифрование ключом восстановления KMyMoney
Вы можете дополнительно шифровать данные ключом kmymoney- recover. Только основные разработчики KMyMoney имеют закрытую часть этого ключа для чтения зашифрованных данных. Этот механизм предназначен для случаев потери вашего ключа и, как следствие, невозможности использования данных. При включении этого параметра разработчики KMyMoney могут расшифровать данные и отправить их вам. Перед восстановлением данных вам будут заданы несколько вопросов о содержимом файла.

Тип счёта
Для иностранных валют необходимо указать курс на дату создания счёта. Курсом является цена иностранной денежной единицы в базовой валюте на дату создания счёта.
Название счёта
Пластиковая карта Тип « Пластиковая карта » предназначен для учёта средств на чековой книжке или пластиковой карте. Банковский вклад Тип « Банковский вклад » предназначен для учёта сбережений на банковских депозитах. Кредитная карта Тип « Кредитная карта » предназначен для учёта средств на вашей кредитной карте. Наличные Тип « Наличные » предназначен для учёта наличных средств. Кредит Тип « Кредит » предназначен для учёта средств в виде займов под проценты (например, кредит на покупку автомобиля, недвижимости и т. п.). Инвестиции Тип « Инвестиции » предназначен для учёта средств в виде акций, паёв и прочих инвестиционных вложений. Активы Тип « Активы » предназначен для учёта вашего имущества: дома, автомобиля или коллекции картин. Пассивы Тип « Пассивы » предназначен для учёта ваших обязательств (кроме кредитов). Например, налоги или взятие в долг.
