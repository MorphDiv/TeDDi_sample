Пузырьковая диаграмма
С помощью пузырьковой диаграммы можно представить отношения между тремя переменными.
Две переменные применяются для определения положения по оси X и по оси Y, в то время как третья переменная представляется в качестве относительного размера каждого пузырька.
В диалоговом окне &quot; Ряд данных &quot; для пузырьковой диаграммы можно определить диапазон данных для размеров пузырьков.
