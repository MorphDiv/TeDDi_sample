Активность по дискуссиям
Этот тип группирует дискуссии по периодам дат последних сообщений в них. Дискуссии с самыми последними сообщениями показываются вверху. Например, в группе « Сегодня » будут расположены все сообщения, которые получены сегодня и дискуссии, содержащие эти сообщения.
Активность по датам
Этот тип группирует сообщения по периодам. Последние полученные сообщения показываются вверху. Например, в группе « Сегодня » будут расположены все сообщения, которые получены сегодня.
По датам и дискуссиям
Этот тип группирует дискуссии по датам. Дискуссии с самыми последними сообщениями показываются вверху. Например, в группе « Сегодня » будут расположены все сообщения, которые получены сегодня и дискуссии, содержащие эти сообщения.
По датам
Этот тип группирует сообщения по датам. Последние полученные сообщения показываются вверху. Например, в группе « Сегодня » будут расположены все сообщения, которые получены сегодня.
Стандартный список рассылки
Стандартный вид списка почтовой рассылки: с группировкой только по дискуссиям, в которых последние полученные сообщения показываются внизу.
По дате без группировки
Обычный список сообщений, без групп и дискуссий, отсортированный по дате, когда последние полученные сообщения показываются вверху.
По отправителю или получателю
Этот тип группирует сообщения по их отправителям или получателям (в зависимости от типа папки). Группы сортируются по именам отправителей или получателей или по их адресам электронной почты. Сообщения не группируются по дискуссиям, с показом последних сообщений внизу.
По инициаторам дискуссий
Этот тип группирует сообщения по дискуссиям, а дискуссии — по их инициаторам. Группы сортируются по имени или адресу электронной почты. Default theme name
