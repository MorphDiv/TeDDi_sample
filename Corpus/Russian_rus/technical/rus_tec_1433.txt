Путь класса
Это диалоговое окно используется для добавления папок и архивов в путь класса Java.
Эти пути доступны для любой среды выполнения Java (JRE).
Назначенные папки и архивы
Указывает размещение классов Java или библиотек классов Java.
Новый путь класса вступает в силу после перезапуска
%PRODUCTNAME
.
Для классов Java, обращение к которым осуществляется с помощью пути класса, не выполняется проверка безопасности.
Добавить архив
Выберите файл архива в формате jar или zip и добавьте этот файл в путь класса.
Добавить папку
Выберите папку и добавьте ее в путь класса.
Удалить
Выберите архив или папку в списке и нажмите кнопку &quot; Удалить &quot; для удаления объекта из этого пути класса.
