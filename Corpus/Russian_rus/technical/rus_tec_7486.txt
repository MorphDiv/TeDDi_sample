Приглушать ли звуковые сигналы терминала
Если установлено, не производить шума, когда приложения отправляют escape-последовательность для терминального гудка.
Количество строк в области прокрутки
Количество строк прокрутки. Вы можете прокручивать окно терминала на указанное количество строк; неуместившиеся строки будут отброшены. Если установлен ключ scrollback-unlimited, это значение будет проигнорировано.
Хранить ли неограниченное количество строк прокрутки
Если установлено, строки прокрутки никогда не будут отбрасываться. История прокрутки временно сохраняется на диск, что может привести к дефициту свободного места на диске, если в терминале находится слишком длинный вывод.
Прокручивать ли вниз при нажатии на клавишу
Если установлено, нажатие клавиш приводит к перемещению полосы прокрутки вниз.
Прокручивать ли вниз при поступлении нового вывода
Если установлено, каждый новый вывод в терминале будет прокручивать экран вниз.
Разрешить ли полужирный текст
Если установлено, разрешить приложениям в терминале делать текст полужирным.
Символы как часть слова
При выделении текста по словам последовательности этих символов считаются отдельными словами. Диапазоны могут быть заданы как «A-Z». Литера «дефис» должна указываться в начале (чтобы она не была похожа на обозначение диапазона).
Цвет текста по умолчанию для терминала
Цвет текста по умолчанию для терминала, цвет можно указывать в HTML-стиле шестнадцатиричными цифрами или указать название цвета, например «red».
Цвет фона по умолчанию для терминала
Цвет фона по умолчанию для терминала, цвет можно указывать в HTML-стиле шестнадцатиричными цифрами или указать название цвета, например «red».
Палитра для приложений терминала
Терминалы используют 16-цветовую палитру, которую могут использовать приложения, запущенные в терминале. Палитра — список названий цветов, разделённых двоеточиями. Названия цветов записываются в hex-формате, например «#FF00FF»
Использовать ли цвета из темы оформления для виджета терминала
Если установлено, цветовая схема оформления полей ввода текста будет использована для терминала вместо цветов, выбранных пользователем.
Использовать ли мигающий курсор
Допустимые значения: «system» — системные параметры мигания курсора; «on» или «off» — непосредственная установка режима.
Внешний вид курсора
Допустимые значения: «block» — блочный курсор; «ibeam» — курсор в виде вертикальной линии; «underline» — курсор в виде подчёркивания.
Использовать ли системный шрифт
Если установлено, терминал будет использовать стандартный системный моноширный шрифт (если системный шрифт не моноширный, то будет использоваться наиболее подходящий шрифт).
Шрифт
Название Pango-шрифта. Примеры: «Sans 12» или «Monospace Bold 14».
