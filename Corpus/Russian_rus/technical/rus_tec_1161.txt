Использование смарт-тегов
Смарт-теги предоставляют дополнительную информацию и функциональные возможности для указанных слов в документе Writer.
Для различных расширений смарт-тегов доступны различные функции.
Установка смарт-тегов
расширения
.
Для установки смарт-тега выполните одно из следующих действий:
Сервис - Управление расширениями
для вызова менеджера расширений, нажмите &quot; Добавить &quot; и выберите файл.
Щелкните по ссылке на файл смарт-тегов *.oxt на веб-странице и откройте ссылку приложением по умолчанию.
Для этого требуется надлежащая настройка веб-браузера.
Меню &quot; Cмарт-теги &quot;
Сервис - Параметры - %PRODUCTNAME - Внешний вид
.
При наведении курсора на смарт-тег появляется подсказка с рекомендацией нажать клавишу Ctrl для открытия меню смарт-тегов.
Если мышь не используется, установите курсор внутри отмеченного текста и вызовите контекстное меню с помощью Shift+F10.

Сервис - Параметры автозамены
появится страница
Смарт-теги

Это диалоговое окно используется для активации или деактивации смарт-тегов, а также настройки установленных тегов.
Текст, который распознается как смарт-тег, не подлежит автоматической проверке орфографии.
