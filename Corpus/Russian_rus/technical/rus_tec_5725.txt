ОПИСАНИЕ
& kpat; - это подборка нескольких популярных пасьянсов, начиная от Косынки и Свободной ячейки, и заканчивая Дедушкиными часами и Mod3, а всего тринадцать вариантов чем- нибудь занять себя в свободное время.
КЛЮЧИ
Программа использует стандартный синтаксис командной строки GNU, с длинными ключами, начинающимися с двух дефисов ('--'). Ниже описаны все ключи, распознаваемые kpat.
Общие параметры
-- author
Информация об авторах программы
-- help
Краткая справка о возможных ключах
-- help- qt
Список ключей, специфичных для программ Qt
-- help- kde
Список ключей, специфичных для программ & kde;
-- help- all
Список всех ключей
-- license
Информация о лицензировании программы
- v, -- version
Версия программы.
Ключи Qt
-- display имя_ монитора
Использовать монитор X- сервера имя_ монитора.
-- session Id_ сеанса
Продолжить по переданному идентификатору сеанса
-- cmap
Установить собственную цветовую схему на 8- битном экране.
-- ncols количество
Ограничить число цветов, расположенных в цветном кубе на 8- битном экране, но только при использовании спецификации цветов через QApplication:: ManyColor.
-- nograb
Никогда не перехватывать информацию с клавиатуры и мыши.
-- dograb
Отладка может привести к эффекту ключа - nograb, для форсирования перехвата используйте - dograb.
-- sync
Переключиться в режим синхронизации (для отладки).
-- fn, -- font имя_ шрифта
Определить шрифт, используемый в программе.
-- bg, -- background цвет
Установить цвет фона, а следовательно и палитру приложения (цвет тёмных и светлых теней подсчитывается автоматически).
-- fg, -- foreground цвет
Цвет переднего плана.
-- btn, -- button цвет
Цвет кнопок.
-- name имя
Установить имя приложения.
-- title заголовок
Установить заголовок приложения.
-- visual TrueColor
Форсировать использование TrueColor на 8- битном экране.
-- inputstyle
Установить стиль ввода XIM (X Input Method). Возможные значения этого параметра: onthespot, overthespot, offthespot и root.
-- im сервер XIM
Установить имя сервера XIM.
-- noxim
Выключить XIM.
Ключи KDE
-- caption заголовок
Установить заголовок окна приложения.
-- icon значок
Установить значок приложения.
-- miniicon значок
Установить значок окна приложения.
-- dcopserver сервер
Использовать указанный сервер DCOP.
-- nocrashhandler
Отключить отслеживание сбоев, чтобы получить аварийный дамп.
-- waitforwm
Ожидать менеджера окон, совместимого с WM_ NET.
-- style стиль
Установить визуальный & GUI; стиль.
-- geometry геометрия
установить геометрию главного окна.
