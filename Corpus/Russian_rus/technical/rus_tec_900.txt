Параметры пользователя
Администратор
Введите имя администратора базы данных.
Имя и пароль администратора назначаются при создании базы данных Adabas.
Помимо администратора, к базе данных Adabas могут иметь доступ еще два пользователя.
Управляющий пользователь
Введите имя пользователя, которому требуется предоставить ограниченное право изменения параметров базы данных.
Обычно параметры по умолчанию для имени и пароля управляющего пользователя не изменяются.
Введение управляющего пользователя не влияет на ограничение числа пользователей тремя.
Пользователь домена
Введите имя пользователя домена, которое будет применяться внутри Adabas.
Обычно параметры по умолчанию для имени и пароля пользователя домена не изменяются.
Введение пользователя домена не влияет на ограничение числа пользователей тремя.
Пароль
Введите пароль.
Параметры базы данных
Диски, содержащие части серверной базы данных, называются DEVSPACE.
Для повышения быстродействия выделите для каждого файла DEVSPACE отдельный диск.
Длина пути к файлу для DEVSPACE, включая имя файла, не должен превышать 40 символов.
SYSDEVSPACE
Введите путь для системного файла DEVSPACE.
SYSDEVSPACE управляет данными конфигурации базы данных.
Размер файла SYSDEVSPACE зависит от размера базы данных.
TRANSACTIONLOG
Введите путь к файлу журнала транзакций.
В этот файл записываются все изменения, внесенные в базу данных в ходе транзакции.
По завершении транзакции они перезаписываются в базу данных.
Файл TRANSACTIONLOG также используется при отмене транзакции.
DATADEVSPACE
Введите путь для файла данных DEVSPACE.
В этом файле хранятся данные пользователя, такие как таблицы и указатели, а также каталог SQL (сведения о схеме).
Данные, содержащиеся в одной таблице пользователя, в равной мере распределяются по всем файлам DATADEVSPACE.
Общий размер всех файлов DATADEVSPACE соответствует размеру базы данных.
Максимальный размер файлов DATADEVSPACE в ограниченной версии базы данных Adabas составляет 100 МБ.
Если емкость файлов DATADEVSPACE исчерпывается во время работы с базой данных, программа Adabas выключается, размер файла DATADEVSPACE увеличивается (в пределах 100 МБ), затем база данных снова загружается.
Сохраненные данные при этом не теряются.
