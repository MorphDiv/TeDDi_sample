Текстовый редактор
Редактор текстовых файлов
Текстовый редактор gedit
Использовать шрифт по умолчанию
Следует ли использовать исходный системный шрифт для отображения редактируемых документов. Если этот параметр отключён, для области редактирования будет использоваться шрифт, определённый в параметре «Шрифт редактора».
Шрифт редактора
Другой шрифт, который будет использован для области редактирования. Будет иметь эффект только если параметр «Использовать исходный шрифт» выключен.
Стилевые схемы
Идентификатор стилевой схемы GtkSourceView для подсветки текста.
Создавать резервные копии
Следует ли создавать резервные копии сохраняемых файлов. Расширение имен резервных файлов устанавливается параметром "Расширение резервных файлов".
Автосохранение
Следует ли автоматически сохранять изменённые файлы через промежуток времени. Интервал между сохранениями устанавливается параметром "Интервал автосохранения".
Интервал автосохранения
Интервал в минутах, после которого все изменённые файлы будут автоматически сохранены. Будет иметь эффект, только если параметр "Автосохранение" включён.
Лимит операций отмены (УСТАРЕВШИЙ)
Максимальное число действий, которые можно отменить или вернуть. "-1" означает неограниченное количество действий. Считается устаревшим после 2.12.0
Максимальное количество действий Undo
Максимальное число действий, которые можно отменить или вернуть. "-1" означает неограниченное количество действий.
Режим переноса строк
Определяет, как переносить длинные строки в области редактирования. «none» — не переносить, «word» — переносить по словам, «char» — переносить посимвольно. Т. к. значения чувствительны к регистру, проверьте, правильно ли они записаны.
Размер табуляции
Определяет число пробелов, которые следует отображать вместо символа табуляции.
Вставлять пробелы
Следует ли вставлять пробелы вместо табуляций.
Автоматический отступ
Должен ли gedit автоматически расставлять отступы.
Отображать номера строк
Следует ли отображать номера строк в области редактирования.
Подчёркивать текущую строку
Подсвечивать ли текущую строку.
Подсвечивать парные скобки
Подсвечивать ли парные скобки.
Отображать правую границу
Следует ли отображать правую границу в области редактирования.
Положение правой границы
Указывает положение правой границы.
Умные Home и End
Определяет способ перемещения курсора при нажатии клавиш «HOME» и «END». «disabled» — перемещать курсор в начало и конец строки, «after» — при первом нажатии перемещать курсор в начало или конец строки, при втором — в начало или конец текста без учёта пробельных символов, «before» — перемещать курсор сперва в начало текста, и только затем, в начало или конец строки, «always» — перемещать курсор в начало или конец текста.
Восстанавливать предыдущую позицию курсора
Должен ли gedit восстанавливать предыдущую позицию курсора после загрузки файла.
Включить подсветку синтаксиса
Следует ли включить подсветку синтаксиса.
Включить подсветку поиска
Должен ли gedit подсвечивать все найденные при поиске совпадения.
Панель инструментов видима
Будет ли видна панель инструментов в окне редактирования.
Режим вкладок
Определяет, когда показывать вкладки. «never» — не показывать вкладки, «always» — всегда показывать вкладки, «auto» — показывать вкладки при наличии более одной вкладки. Т. к. значения чувствительны к регистру, проверьте, правильно ли они записаны.
Видимость строки состояния
Будет ли видна строка состояния окна редактирования.
Показывать боковую панель
Будет ли видна боковая панель слева от окна редактирования.
Максимальное число последних файлов
Определяет число недавно использованных файлов, которые будут отображаться в подменю "Открыть последние файлы".
Печатать подсветку синтаксиса
Следует ли печатать подсветку синтаксиса при печати документа.
Печатать заголовки страниц
Следует ли печатать заголовок документа.
Режим переноса строк при печати
Определяет, как переносить длинные строки при печати. «none» — не переносить, «word» — переносить по словам, «char» — переносить посимвольно. Т. к. значения чувствительны к регистру, проверьте, правильно ли они записаны.
Печатать номера строк
Если значение установлено в 0, номера строк не будут вставлены при печати документа. Иначе номера строк будут печататься для каждой строки.
Основной шрифт для печати
Определяет шрифт, используемый при печати тела документов.
Шрифт заголовка для печати
Определяет шрифт, используемый при печати заголовков. Будет иметь эффект, только если параметр "Печатать заголовки страниц" включён.
Шрифт номера строк для печати
Определяет шрифт, используемый при печати номеров строк. Будет иметь эффект, только если параметр "Печатать номера строк" не равен нулю.
Кодировки, определяемые автоматически
Сортированный список кодировок, используемых редактором для автоопределения кодировки файла. «ТЕКУЩАЯ» — это кодировка локали. Используются только распознаваемые кодировки.
Кодировки, отображаемые в меню
Список кодировок, отображаемый в меню "Кодировка символов" диалога выбора файлов. Используются только распознаваемые кодировки.
Активные модули
Список активных модулей. Он содержит "Адрес" активного модуля. См. файл .gedit-plugin для получения "адреса" данного модуля.
