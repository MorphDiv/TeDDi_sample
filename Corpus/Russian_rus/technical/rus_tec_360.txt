Формирование ряда данных
В поле списка &quot; Ряд данных &quot; перечислены все ряды данных в текущей диаграмме.
Ряды данных столбца располагаются в верхней части списка, ряд данных линии − в нижней части списка.
Для формирования ряда данных выберите запись в списке.
Нажмите &quot; Добавить &quot; для добавления другого ряда данных ниже выбранной записи.
Новый ряд данных имеет тот же тип, что и выбранная запись.
Нажмите &quot; Удалить &quot; для удаления выбранной записи из списка &quot; Ряд данных &quot; .
Для перемещения выбранной записи в списке вверх или вниз используйте кнопки &quot; Вверх &quot; и &quot; Вниз &quot; .
Этот способ можно использовать для преобразования ряда данных столбца в ряд данных списка и обратно.
При этом меняется лишь расположение в диаграмме, а не порядок в таблице-источнике данных.
Редактирование ряда данных
Щелкните по записи в списке для просмотра и редактирования свойства этой записи.
В поле списка &quot; Диапазон данных &quot; перечислены имена ролей и диапазоны ячеек компонентов ряда данных.
Щелкните по записи, затем отредактируйте содержимое в текстовом поле ниже.
Подпись рядом с текстовым полем указывает на текущую выбранную роль.
Выберите диапазон данных
для сворачивания диалогового окна и выбора диапазона с помощью мыши.
Диапазон для роли данных (например, Y-значения) не должен включать ячейку подписи.
Редактирование категорий или подписей данных
Введите или выберите диапазон ячеек, который должен использоваться в качестве текста для категорий или подписей данных.
Значения в диапазоне &quot; Категории &quot; будут отображаться как подписи на оси X.

Здесь можно определить следующие элементы:
Дополнительные оси

Statistics, for example mean values, y error bars and trend lines
