По умолчанию показывать панель инструментов
Панель инструментов — это верхняя панель, которая содержит средства для навигации и управления масштабированием. Два логических параметра: true — делает панель инструментов по умолчанию видимой, false — делает панель невидимой.
По умолчанию показывать панель состояния
Панель состояния — это нижняя панель, в которой показываются вспомогательные сообщения. Два логических параметра: true — делает панель состояния по умолчанию видимой, false — делает панель невидимой.
Выбрать режим масштабирования для просмотра
Существует два режима масштабирования: «1» — размеры комикса подгоняются под размеры окна, «2» — размеры комикса подгоняются под ширину окна.
