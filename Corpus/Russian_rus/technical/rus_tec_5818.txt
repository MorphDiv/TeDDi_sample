Типичное задание требует лимит фильтра менее 200. Лимит меньше минимума, требуемого для задания, преобразуют статус отдельного задания для печати в любое время. По умолчанию лимит фильтра 0 (без лимита). Пример: 200 Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)

Вы можете указать несколько строк для прослушивания с нескольких серверов или для ограничения доступа. Примечание: к сожалению, большинство веб- браузеров не поддерживают TLS или обновления шифрования HTTP. Если вы хотите использовать шифрование для передачи по интернету, возможно, вам нужно установить порт 443 (порт "HTTPS"). Адреса просмотра (BrowseAddress) Укажите широковещательный адрес. По умолчанию широковещательное сообщение посылается на все активные интерфейсы. Примечание: HP- UX 10. 20 и более ранние версии некорректно поддерживают широковещательные запросы по маске сети класса A, B, C или D netmask (так как нет поддержки CIDR). Например: x. y. z. 255, x. y. 255. 255 Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Включить/ выключить просмотр (BrowseAllow/ BrowseDeny) BrowseAllow: указывает маску адреса, с которого принимаются запросы на просмотр состояния. По умолчанию со всех адресов. BrowseDeny: указывает маску адреса, с которого игнорируются запросы на просмотр состояния. По умолчанию таких адресов нет. "BrowseAllow" и "BrowseDeny" принимают следующие значения адресов: All None *. domain. com. domain. com host. domain. com nnn. * nnn. nnn. * nnn. nnn. nnn. * nnn. nnn. nnn. nnn nnn. nnn. nnn. nnn/ mm nnn. nnn. nnn. nnn/ mmm. mmm. mmm. mmm Ограничения по имени узла/ домена работают только при включённом поиске имени узла! Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)

Информация о состоянии посылается при изменении состояния принтера, поэтому этот параметр указывает максимальное время между обновлениями. Укажите 0 для отключения исходящих широковещательных пакетов. Таким образом ваш локальный принтер не будет объявлять себя по сети, однако вы можете просмотреть остальные принтеры в сети. Пример: 30 Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)

Классы принтеров могут быть указаны явно в файле classes. conf, использованы неявные принтеры в сети или применены оба параметра. При включении поддержки ImplicitClasses принтеры в сети с такими же именами (например, Acme- LaserPrint- 1000) будут помещены в класс с таким же именем Это позволит настроить дополнительные очереди в сети без особых трудностей. При посылке пользователем задания на Acme- LaserPrint- 1000 оно будет помещено в первую доступную очередь. Включено по умолчанию. Do not translate the keyword between brackets (e. g. Класс (AuthClass) Класс авторизации; поддерживаются классы Анонимный, Пользователь, Системные (действительный пользователь в группе SystemGroup), и Группа (действительный пользователь в указанной группе). Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Имена пользователей/ групп, которые имеют доступ к ресурсу. Формат: список, разделённый запятыми. Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Удовлетворяет условиям (Satisfy) Этот параметр устанавливает проверку условия для доступа к ресурсу Если выбран "all", для доступа должны быть удовлетворены все условия аутентификации и прав доступа. Если выбран "any" то пользователь получает доступ если удовлетворены условия аутентификации или прав доступа. Например, вы можете требовать аутентификацию для удалённого доступа, но позволить локальным пользователям не использовать аутентификацию. По умолчанию установлен "all". Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)

Допустимые значения: Always - Всегда использовать шифрование (SSL) Never - Не использовать шифрование Required - Использовать шифрование TLS IfRequested - Использовать шифрование, если этого требует сервер Значение по умолчанию "IfRequested". Автоматическая очистка заданий (AutoPurgeJobs) Автоматическая очистка заданий если они не нужны для квот. По умолчанию No( Нет). Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Протоколы просмотра (BrowseProtocols) Указывает, какой протокол используется для просмотра. Могут быть следующие из указанных, разделённые пробелом и( или) запятой: all - Все поддерживаемые протоколы. cups - Протокол просмотра CUPS. slp - Протокол SLPv2. По умолчанию: cups. Примечание: если вы выбрали SLPv2, очень рекомендуем запустить хотя бы один агент каталогов SLP (SLP Directory Agent) (DA) в вашей сети. В противном случае, просмотр ресурсов может занять несколько секунд, пока сервер не получит ответ на запрос клиенту. Do not translate the keyword between brackets (e. g. При включении пользователи могут печатать страницу- разделитель до или после самого задания, но не могут полностью отменить вывод информации о классификации. По умолчанию выключено. Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Скрыть неявных членов (HideImplicitMembers) Показывать или нет пользователей неявных классов. При включении этого параметра любой удалённый принтер, который является членом неявного класса, будет скрыт от пользователя, который будет видеть только одну очередь даже если неявным классом поддерживается несколько очередей. Включено по умолчанию. Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Использовать & quot; любыеquot; классы (ImplicitAnyClasses) Создавать или нет неявные классы типа Любой принтер (AnyPrinter). Если параметр ImplicitAnyClasses включён и локальная очередь имеет существующее имя, например, "printer", "printer@ server1", "printer@ server1", тогда вместо неё создаётся класс названный "Anyprinter". Если параметр ImplicitAnyClasses выключен, неявные классы не создаются, когда имеется локальная очередь с таким же именем. Выключено по умолчанию. Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Макс. количество заданий (MaxJobs) Максимальное количество заданий, которые находятся в памяти (активные и выполненные). По умолчанию 0 (без ограничений). Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Макс. количество заданий для пользователя (MaxJobsPerUser) Максимальное количество активных заданий, которые позволены каждому пользователю. При превышении этого параметра, следующие задания будут отвергнуты, пока активные задания не будут завершены, остановлены, отменены или прерваны. Значение 0 отключает проверку количества заданий. По умолчанию 0 (без ограничений). Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
Макс. количество заданий на каждый принтер (MaxJobsPerPrinter) Максимальное количество активных заданий, которые позволены каждому принтеру или классу. При превышении этого параметра, следующие задания будут отвергнуты, пока активные задания не будут завершены, остановлены, отменены или прерваны. Значение 0 отключает проверку количества заданий. По умолчанию 0 (без ограничений). Do not translate the keyword between brackets (e. g. ServerName, ServerAdmin, etc.)
