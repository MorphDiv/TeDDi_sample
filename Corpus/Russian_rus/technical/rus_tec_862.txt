Просмотр базы данных
В %PRODUCTNAME для просмотра базы данных можно использовать два различных метода.
Файл - Открыть
, чтобы открыть файл базы данных.
Файл базы данных
предоставляет полный доступ к таблицам, запросам, отчетам и формам.
Можно изменить структуру таблиц и содержимое записей данных.
Вид - Источник данных
для просмотра зарегистрированных баз данных.
Представление источника данных
может использоваться для перетаскивания полей таблиц из зарегистрированных баз данных в документы и для выполнения слияний.
