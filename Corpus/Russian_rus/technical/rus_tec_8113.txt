Сохранять данные об использовании
Когда вы используете HUD, он сохраняет результаты для того, чтобы ваши будущие запросы обрабатывались лучше. Некоторые пользователи не хотят, чтобы эта информация была сохранена. При желании они могут отключить эту функцию.
Уведомление будет отображено в списке меню как индикатор
Для того, чтобы меню приложений отображалось выше результатов поиска, краткое уведомление появится в одном из пунктов меню индикатора. Это значение представляет собой процент уведомления, например значение 50 - это 50% дополнительно к расчетному расстоянию.
Уведомление о лишних символах добавлено в строку поиска
Уведомление о каждом лишнем символе в строке поиска, который не отображается в списке меню приложений.

Уведомление будет принято, если пользователь напечатал "фаайл" вместо поиска "Файл", например.
Уведомление будет принято, если символ будет удален
Уведомление о каждом символе удалено из строки поиска, в отличие от текста в списке меню. Это относиться только к недостающим символам, которые находятся не в конце поисковой строки.

Уведомление будет принято, если пользователь напечатал "фаайл" вместо поиска "Файл", например.
Уведомление будет принято, если символ был удален в конце слова
Уведомление о каждом пропущенном символе в конце объекта поиска.

Это уведомление будет принято, если пользователь , например, написал "фай" когда хотел найти объект меню "файл".
Уведомление будет одобрено, если символы не соответствуют друг другу
Уведомление о каждом замененном символе в поисковом запросе.

Это уведомление будет принято, если пользователь, например, написал "фацл" когда хотел найти объект меню "файл".
Максимальное расстояние для включения в результат
Если рассчитанное расстояние (включая штраф для индикаторов) превышает это значение, то соответствующий элемент не будет включен в результат. Также для этих элементов не будут просматриваться их истории.
