Цвета по умолчанию
Назначение цветов рядам данных.
Эти параметры будут использоваться только для всех новых диаграмм.
Цвета диаграммы
Отображает все цвета, используемые в рядах данных.
Выберите ряды данных для изменения их цвета.
Выберите необходимый цвет из соседней таблицы цветов.
Таблица цветов
Эта таблица используется для замены цветов диаграммы для выбранных рядов данных.
Например, если выбрать ряд данных 6, а затем щелкнуть зеленый цвет 8, прежний цвет ряда данных будет заменен на зеленый 8.
Имя выбранного цвета отображается под цветовой таблицей.
По умолчанию
Восстанавливает параметры цвета, которые были определены при установке программы.
