О переменной среды DISPLAY
При запуске приложения с графическим интерфейсом из командной строки сначала нужно присвоить значение переменной среды DISPLAY. Эта переменная сообщает вашему приложению, какой X-сервер нужно использовать. Обычно вы будете использовать тот, который установлен на вашем собственном компьютере. В этом случае, присвоить значение переменной DISPLAY нужно следующим образом:
DISPLAY=:0.0
где :0.0 означает экран 0 X-сервера 0 на этом компьютере.
Чтобы отобразить приложение на удалённом экране, можно указать IP-адрес удалённого X-сервера:
DISPLAY=192.168.0.25:0.0
Установка переменной DISPLAY для заданий, которые будут выполнены однократно
При создании или изменении задания можно задать значение переменной DISPLAY в текстовом поле сценария. Сделать это нужно в начале сценария.
