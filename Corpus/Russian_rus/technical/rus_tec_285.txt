Отправка документов электронной почтой
Работая в $[officename], можно отправить текущий документ в виде вложения электронной почты.
Файл - Отправить - Документ как сообщение
.
Сервис - Параметры - Интернет - Эл. почта
.
В почтовой программе введите получателя, тему и текст сообщения, если требуется.
Затем отправьте электронное письмо.
Файл - Отправить - Документ как файл Microsoft Word
.
Для электронных таблиц выберите команду
Файл - Отправить - Документ как файл Microsoft Excel
.
Для презентаций выберите команду
Файл - Отправить - Документ как файл Microsoft PowerPoint
.
Файл - Отправить - Документ как файл PDF
При использовании этих команд активный документ не изменяется.
Производится создание и отправка временной копии.
