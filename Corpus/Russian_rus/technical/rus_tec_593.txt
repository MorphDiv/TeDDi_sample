Параметры
Тип
Тип выбранной ссылки.
В некоторых базах данных поддерживается только подмножество возможных типов.
внутреннее соединение
При использовании внутреннего соединения таблица результатов содержит только те записи, содержимое связанных полей которых совпадает.
В $[officename] SQL ссылки этого типа создаются с помощью соответствующего выражения WHERE.
Соединение слева
При использовании соединения слева таблица результатов содержит все поля левой таблицы и только те связанные поля правой таблицы, содержимое которых совпадает.
В $[officename] SQL этот тип ссылок соответствует команде LEFT OUTER JOIN.
Соединение справа
При использовании соединения справа таблица результатов содержит все поля правой таблицы и только те связанные поля левой таблицы, содержимое которых совпадает.
В $[officename] SQL этот тип ссылок соответствует команде RIGHT OUTER JOIN.
Полное соединение
При использовании полного соединения таблица результатов содержит все поля обеих таблиц.
В $[officename] SQL этот тип ссылок соответствует команде FULL OUTER JOIN.
Параметры естественного объединения
Используется для вставки ключевого слова NATURAL в SQL-оператор, определяющий отношение.
Отношение объединяет все столбцы, имеющие одинаковые имена в обеих таблицах.
Полученная соединенная таблица содержит только по одному столбцу из каждой пары столбцов с одинаковыми именами.
