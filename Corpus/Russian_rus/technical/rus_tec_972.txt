Параметры фильтра ASCII
Здесь можно выбрать, какие параметры (простой шрифт, язык, набор символов, коды конца строки) импортируются или экспортируются с текстовым документом.
Такое диалоговое окно появляется при попытке загрузить файл ASCII с применением фильтра &quot; Кодированный текст &quot; , сохранить документ впервые или сохранить его под другим именем.
Свойства
Определяет параметры для импорта или экспорта файла.
Во время экспорта можно задавать только набор символов и код конца абзаца.
Набор символов
Указывает набор символов файла для экспорта или импорта.
Стандартные шрифты
Установка шрифта по умолчанию определяет шрифт, используемый для отображения текста.
Выбор шрифтов по умолчанию возможен только при импорте.
Язык
Если этого еще не было сделано, определите язык текста.
Эта возможность доступна только во время импорта.
Разрыв абзаца
Определяет тип конца абзаца для текстовой строки.
ВОЗВРАТ КАРЕТКИ И ПЕРЕВОД СТРОКИ
Производит возврат каретки и перевод строки.
Этот параметр задан по умолчанию.
ВОЗВРАТ КАРЕТКИ
Производит возврат каретки для обозначения конца абзаца.
ПЕРЕВОД СТРОКИ
Производит перевод строки для обозначения конца абзаца.
