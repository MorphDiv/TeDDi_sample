Работа с таблицами
Данные сохраняются в таблицах.
Например, адресная книга системы, которая используется для адресов электронной почты пользователя, представляет собой таблицу базы данных этой адресной книги.
Каждый адрес - это запись данных, представленная в виде строки в этой таблице.
Эти записи данных включают поля данных, например поля имени, фамилии и поле адреса электронной почты.

Для большинства операций баз данных представление можно использовать так же, как и таблицу.
