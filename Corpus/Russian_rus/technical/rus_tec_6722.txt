Группы шрифтов
Имеется 4 специальных группы шрифтов:
Все шрифты Показывать все шрифты - как шрифты пользователя, так и общие для всех.
Шрифты пользователя Показываются шрифты находящиеся в вашем личном распоряжении. Они не недоступны другим пользователям.
Системные шрифты Шрифты доступные всем пользователям в системе. Для установки шрифта как системного или удаления системного шрифта требуются права администратора. Системные шрифты выделяются полужирным текстом, чтобы можно было отличать их от шрифтов пользователя.
Неклассифицированные Показывает список шрифтов, которые не принадлежат ни к одной из пользовательских групп.
Чтобы добавить шрифт в группу, перетащите его из списка шрифтов в группу. Чтобы удалить шрифт из группы, перетащите его в группу Все шрифты
Включение и выключение
Это будет полезно пользователям, использующим много шрифтов, для активации определённых шрифтов в разное время. Данный модуль позволяет отключать отдельные шрифты или их группы. Выключение шрифта не удаляет его из системы - оно просто скрывает его от приложений.
Дублирующиеся шрифты
Если в вашей системе установлено много шрифтов, то вполне возможно, что среди них найдутся одинаковые шрифты. Поиск дублирующихся шрифтов... - это простой инструмент для сканирования системы на предмет шрифтов, которым соответствует несколько файлов. Например, если у вас есть times. ttf и times. TTF, установленные в / usr/ local/ share/ fonts, то базовый механизм шрифтов (называемый FontConfig) увидит только один из них. Так, если вы удалили этот шрифт, он снова появится в списке, поскольку только один из файлов будет удалён и система переключится на второй. При запуске этого инструмента появится список шрифтов, имеющих несколько файлов. Чтобы выбрать файл для удаления, щёлкните на столбце со значком корзины.
Текстовое поле для фильтрации списка шрифтов. Можно фильтровать по семье и названию стиля.

Этот модуль отвечает за установку, удаление и управление шрифтами. Главное окно содержит список установленных шрифтов, сгруппированных по семейству - число в квадратных скобках показывает количество установленных шрифтов для данной семьи. Например, шрифт The Times может отображаться как:
Times [4]

Курсив
Полужирный
Полужирный курсив
Чтобы установить шрифт, нажмите кнопку Добавить и выберите желаемый шрифт в файловом диалоге. Значение выпадающего списка, упомянутого выше, определит место установки шрифта.
Для того чтобы удалить шрифты, выберите их из списка и нажмите кнопку Удалить.
KControl
Этот модуль имеет два режима работы:
Установка шрифтов
Управление шрифтами
Установка шрифтов
Этот режим разработан для пользователей, которые только хотят ограничиться установкой и удалением шрифтов. В нём присутствуют следующие элементы управления:
Кнопка меню Настройка. Используйте эту кнопку для изменения режима этого модуля, переключения панели демонстрации больших размеров шрифтов и изменения текста, используемого для демонстрации.
Выпадающий список для изменения набора показываемых шрифтов и соответственно места их установки.
Управление шрифтами
Включение этого режима активирует следующие дополнительные функции: Включение и выключение шрифтов. Организация шрифтов в группы. Дополнительные критерии фильтрации шрифтов. « Шрифтолитейная ». Автоподбор шрифтов FontConfig. Позволяет узнать семейство шрифтов, которая использовалась бы в FontConfig фактически, для данного семейства. Файл шрифта. Местоположение файла шрифта. Письменная система. Поиск дублирующихся шрифтов В этом режиме выпадающий список содержит группы шрифтов.
