Ускоренная печать с сокращенными данными
Существует возможность сокращения данных, необходимых для печати документа.
Для непосредственной печати на принтере и для печати в файл параметры задаются по-разному.
Сервис - Параметры - $[officename] - Печать
.
Выберите один из следующих вариантов настройки:
Принтер
-
чтобы определить параметры сокращения данных при печати непосредственно на принтере
Печатать в файл
-
чтобы определить параметры сокращения данных при печати в файл
ОК
.
С этого времени для всех документов, отправляемых на печать, будут использоваться эти измененные параметры.
Напечатайте свой документ.
Можно сократить данные, касающиеся прозрачности, градиентов или точечных рисунков.
На многих принтерах сокращение данных не вызовет заметного ухудшения качества печати.
При этом время печати существенно уменьшится, а при печати в файл получится файл гораздо меньшего размера.
