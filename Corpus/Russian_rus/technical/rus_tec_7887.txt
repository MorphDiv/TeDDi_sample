Пути, исключённые из быстрого поиска
Этот ключ определяет пути, которые будут исключены из быстрого поиска. Поддерживаются шаблоны «*» и «?». Значения по умолчанию /mnt/*, /media/*, /dev/*, /tmp/*, /proc/* и /var/*.
Запретить второй проход быстрого поиска
Этот ключ определяет, отключено ли использование команды find после быстрого поиска.
Пути, исключённые из второго прохода быстрого поиска
Этот ключ определяет пути, которые средство поиска будет исключать при втором проходе быстрого поиска. Второй проход использует команду «find» для поиска файлов. Цель второго прохода — найти файлы, которые не были проиндексированы. Поддерживаются шаблоны «*» и «?». Значение по умолчанию — /.
Порядок столбцов результата поиска
Этот ключ определяет порядок столбцов в результатах поиска. Это ключ не должен изменяться пользователем.
Ширина окна по умолчанию
Этот ключ определяет ширину окна и используется для сохранения размера средства поиска между сеансами. Если установить в значение -1, то будет использоваться ширина по умолчанию.
Высота окна по умолчанию
Этот ключ определяет высоту окна и используется для сохранения размера средства поиска между сеансами. Если установить в значение -1, то будет использоваться высота по умолчанию.
Окно развернуто по умолчанию
Этот ключ определяет, должно ли окно средства поиска появляться в развёрнутом виде.
