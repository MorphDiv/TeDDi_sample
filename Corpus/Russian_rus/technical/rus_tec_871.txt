Перетаскивание в представлении источника данных
Наиболее быстрым способом копирования данных из источника данных в текстовый документ или документ электронной таблицы, а также создания форм на основе источника данных является метод перетаскивания.

Копирование перетаскиванием
Правка - Отменить
.
Перетаскиванием также можно копировать данные из документа в источник данных.
Текстовую таблицу или выделенную область электронной таблицы можно перетащить в хранилище таблиц навигатора источников данных.
Обычный текст можно копировать перетаскиванием из документа в поле данных в представлении источника данных.
Использование данных в текстовом документе
Можно вставить поле базы данных в текстовый документ, перетащив имя поля из заголовка столбца представления источника данных в документ.
Это особенно полезно при создании стандартных писем.
Просто перетащите нужные поля - домашний адрес, форму адреса и т.д. - в свой документ.
Вставка столбцов базы данных
, где можно указать, следует ли использовать все поля базы данных и следует ли копировать данные в документ в виде текста, таблицы или полей.
При этом будут вставлены все выделенные записи.
Применение данных к документу таблицы
В текущий лист электронной таблицы можно вставить одну или несколько записей, выделив строки в представлении источника данных и перетащив их в электронную таблицу.
Данные будут вставлены там, где будет отпущена кнопка мыши.
Вставка элементов управления в текстовую форму
При создании текстовой формы, связанной с базой данных, можно создавать элементы управления с помощью перетаскивания из представления источника данных.
