Источник данных адресов
Этот мастер регистрирует существующую адресную книгу в качестве источника данных в $[officename].
Кроме того, можно в любое время регистрировать источники данных адресов и других данных в $[officename] без применения мастера:
Выберите тип внешней адресной книги
Некоторые типы недоступны в определенных системах.
Mozilla / Netscape
Выберите этот параметр, если адресная книга уже используется в Mozilla или Netscape.
Thunderbird
Select this option if you already use an address book in Thunderbird.
Адреса LDAP
Выберите этот параметр, если адресные данные уже есть на сервере LDAP.
Адресная книга Outlook
Выберите этот параметр, если уже используется адресная книга в Microsoft Outlook (но не в Outlook Express).
Адресная книга Windows
Выберите этот параметр, если уже используется адресная книга в Microsoft Outlook Express.
Адресная книга KDE
Выберите этот параметр, если уже используется адресная книга KDE.
Адресная книга ОС X
Выберите этот параметр, если уже используется адресная книга ОС X.
Evolution
Выберите этот параметр, если уже используется адресная книга в Evolution.
Evolution LDAP
Выберите этот параметр, если уже используется адресная книга в Evolution LDAP.
Groupwise
Выберите этот параметр, если уже используется адресная книга в Groupwise.
Другой внешний источник данных
Выберите этот параметр, если требуется зарегистрировать другой источник данных в качестве адресной книги в $[officename].
