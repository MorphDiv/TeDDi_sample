Параметры отладки
Автоматический прогрев
Разогревать пока яркость лампы не станет постоянной, вместо простого 60-тисекундного ожидания разогрева.
Полное сканирование
Сканировать полностью всю область сканирования, включая калибровочную полосу. Будьте осторожны. Не выбирайте полную высоту. Только для тестирования.
Грубая калибровка
Настраивает усиление и смещение для сканирования автоматически. Если этот параметр выключен, будут предоставлены параметры для ручной аналоговой настройки. Параметр включен по умолчанию. Только для тестирования.
Грубая калибровка только для первого сканирования
