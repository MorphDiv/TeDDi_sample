Параметры запуска Java
Этот диалоговое окно можно использовать для ввода дополнительных параметров запуска для среды выполнения Java (JRE).
Параметры, указанные в этом диалоговом окне, действительны для любой среды выполнения Java.
Параметры запуска Java
Введите параметр запуска для среды выполнения Java так, как это делается в командной строке.
Нажмите кнопку &quot; Назначить &quot; для добавления этого параметра в список назначенных параметров запуска.
В именах путей не следует использовать &quot; escape-символы &quot; и кавычки.
Например, чтобы указать системное свойство &quot; myprop &quot; для папки, введите следующий параметр:
-Dmyprop=c:\program files\java
Для включения отладки в среде выполнения Java введите следующие параметры:
-Xdebug
-Xrunjdwp:transport=dt_socket,server=y,address=8000
Эти изменения вступят в силу после перезапуска %PRODUCTNAME.
