Символ форматирования
Используется для вызова подменю для добавления специальных символов форматирования.
Для расширения списка команд активируйте режим сложной системы письменности.
Неразрывный пробел
Вставляет пробел, который объединяет границы слов при переводе строк.
Неразрывный дефис
Вставляет дефис, который объединяет границы слов при переводе строк.
Мягкий перенос
Вставляет невидимый дефис внутри слова, который появится и создаст перевод строки в случае, если символ будет последним в строке.
Невидимая связка
Вставляет невидимый пробел внутри слова, благодаря которым в случае, если символ будет последним в строке, вставляется разрыв строки.
Доступно при активации сложных систем письменности (CTL).
Невидимый мягкий разделитель
Вставляет невидимое пространство внутри слова, которое позволяет избежать разрыва слова в конце строки.
Доступно при активации сложных систем письменности (CTL).
Значок ввода слева направо
Inserts a text direction mark that affects the text direction of any text following the mark.
Available when complex text layout (CTL) is enabled.
Значок ввода справа налево
Inserts a text direction mark that affects the text direction of any text following the mark.
Available when complex text layout (CTL) is enabled.
