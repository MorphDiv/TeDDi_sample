Перевод элементов управлений в редакторе диалоговых окон
На панели инструментов &quot; Язык &quot; в редакторе диалоговых окон IDE Basic представлены элементы управления, которые можно использовать для активации и управления локализуемыми диалоговыми окнами.
По умолчанию любое созданное диалоговое окно содержит строковые ресурсы только для одного языка.
Можно создать диалоговые окна, в которых автоматически будут отображаться локализованные строки, выбираемые в соответствии с языковыми настройками пользователя.
current language listbox
Выберите язык для строк, которые требуется изменить.
Для добавления языков щелкните по значку &quot; Управление языками &quot; .
Manage Language icon is in sbasic/shared/02/20000000.xhp
present languages box in manage user interface language dialog
Щелкните по языку, затем нажмите &quot; По умолчанию &quot; для установки языка в качестве значения по умолчанию, либо нажмите &quot; Удалить &quot; для удаления языка из списка.
add
Открытие диалогового окна, в котором можно добавить язык к списку.
delete
Выберите язык в списке и нажмите &quot; Удалить &quot; для удаления этого языка.
При удалении всех языков строковые ресурсы для локализуемых диалоговых окон удаляются из всех диалоговых окон в текущей библиотеке.
default
Выберите язык в списке и нажмите &quot; По умолчанию &quot; для установки языка в качестве языка по умолчанию.
set default user interface dialog
Язык по умолчанию будет использоваться как источник для всех других языковых строк.
available languages box
Добавление языков пользовательского интерфейса для строк диалогового окна.
Активация локализуемых диалоговых окон
Вид - Панели инструментов - Язык
.
Если текущая библиотека уже содержит локализуемое диалоговое окно, панель инструментов &quot; Язык &quot; отображается автоматически.
Управление языками
значок &quot; Управление языками &quot;
на панели инструментов &quot; Язык &quot; или в области &quot; Панель инструментов &quot; .
Появится диалоговое окно &quot; Управление языками пользовательского интерфейса &quot; .
В этом диалоговом окне осуществляется управление языками для текущей библиотеки.
Имя текущей библиотеки отображается в строке заголовка.
Для добавления языка нажмите &quot; Добавить &quot; .
Это приведет к активации всех новых диалоговых окон, которые должны содержать локализуемые строковые ресурсы.
При первом нажатии &quot; Добавить &quot; появится диалоговое окно &quot; Установить язык по умолчанию для пользовательского интерфейса &quot; .
При последующих нажатиях &quot; Добавить &quot; появляется диалоговое окно &quot; Добавить язык пользовательского интерфейса &quot; .
Кроме того, в диалоговом окне &quot; Управление языками пользовательского интерфейса &quot; можно изменить язык по умолчанию.
Выберите язык.
Это приведет к добавлению строковых ресурсов, которые должны содержать переведенные версии всех строк, к свойствам диалогового окна.
Набор строк диалогового окна языка по умолчанию копируется в новый набор строк.
Впоследствии можно переключиться к новому языку и затем перевести строки.
Закройте диалоговое окно или добавьте дополнительные языки.
Редактирование локализуемых элементов управления в диалоговом окне
После добавления ресурсов для локализуемых строк в диалоговых окнах в поле списка &quot; Текущий язык &quot; на панели инструментов &quot; Язык &quot; можно выбрать текущий язык.
В списке &quot; Текущий язык &quot; выберите язык по умолчанию.
Вставьте в диалоговое окно любое количество элементов управления и введите все требуемые строки.
Выберите другой язык в списке &quot; Текущий язык &quot; .
С помощью диалоговых окон свойств элементов управления отредактируйте все строки на другом языке.
Повторите эти действия для всех добавленных языков.
Для пользователей, работающих с этим диалоговым окном, будут отображаться отредактированные строки пользовательского интерфейса данной версии %PRODUCTNAME.
Если язык, соответствующий версии пользователя, отсутствует, то отображаются строки на языке по умолчанию.
Если пользователь использует старую версию %PRODUCTNAME без локализуемых ресурсов строки для диалоговых окон Basic, то отображаются строки на языке по умолчанию.
