Задание диапазона подписей
В открывающемся диалоговом окне можно задавать диапазоны подписей.
Содержимое ячеек диапазона подписей можно использовать в формулах в качестве имен - $[officename] распознает эти имена так же, как и заранее определенные имена дней недели и месяцев.
При вводе в формулу с клавиатуры эти имена завершаются автоматически .
Кроме того, имена, определенные по диапазонам подписей, имеют приоритет над именами, установленными по автоматически созданным диапазонам.
Можно определить диапазоны подписей, которые будут содержать одни и те же подписи на разных листах. $[officename] сначала выполняет поиск в диапазонах подписей текущего листа, а при неудаче переходит к диапазонам на других листах.
Область
Здесь отображается ссылка на ячейки для каждого диапазона подписей.
Чтобы удалить диапазон подписей из списка, выделите его и затем щелкните
Удалить
.
Заголовки в первой строке
Выберите этот параметр, чтобы включить подписи столбцов в текущий диапазон подписей.
Содержит заголовки строк
Выберите этот параметр, чтобы включить подписи строк в текущий диапазон подписей.
Для диапазона данных
Здесь задается область данных, для которой действует выбранный диапазон подписей.
Чтобы изменить ее, щелкните на листе и выберите мышью другую область.
Добавить
Добавление текущего диапазона подписей к списку.
