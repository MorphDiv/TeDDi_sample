Функция IsObject [время выполнения]
Проверяет, является ли данная переменная типа object объектом OLE.
Функция возвращает значение True, если эта переменная является объектом OLE, иначе возвращается значение False.
Синтаксис:
IsObject (Переменная_типа_Object)
Возвращаемое значение:
Bool
Параметры:
Переменная_типа_Object
.
Любая переменная, подлежащая проверке.
Если переменная типа Object содержит объект OLE, функция возвращает значение True.
