Цвет новых записок по умолчанию
Цвет по умолчанию для новых липких записок. Цвет должен соответствовать шестнадцатеричной спецификации html, например «#30FF50».
Цвет шрифта по умолчанию
Цвет шрифта по умолчанию для новых липких записок. Цвет должен соответствовать шестнадцатеричной спецификации html, например «#000000».
Шрифт по умолчанию для новых записок
Шрифт по умолчанию для новых липких записок. Название шрифта должно указываться в формате Pango, например «Sans Italic 10».
Прилипчивость записок к рабочим местам
Указывает, видимы липкие записки на ВСЕХ рабочих местах или нет.
Состояние фиксации липких записок
Указывает, зафиксированы липкие записки (недоступны для изменений) или нет.
Формат даты в заголовках записок
По умолчанию липким запискам при создании даётся заголовок с текущей датой. Используется этот формат; всё, что может быть корректно проанализировано функцией strftime() является правильным.
Использовать ли системный цвет по умолчанию
Если этот ключ не установлен, то цвет пользователя может быть использован как цвет по умолчанию для всех липких записок.
Использовать ли системный шрифт по умолчанию
Если этот ключ не установлен, то шрифт пользователя может быть использован как исходный шрифт для всех липких записок.
Применять ли цвет и шрифт по умолчанию ко всем запискам
Если этот ключ установлен, то будут игнорироваться цвет и шрифт пользователя, назначенные отдельным запискам.
Запрашивать ли подтверждение на удаление записки
Пустые записки всегда удаляются без подтверждения.
