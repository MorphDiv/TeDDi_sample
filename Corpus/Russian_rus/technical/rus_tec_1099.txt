Печать
Печать текущего документа, выделения или указанных страниц.
Для текущего документа также можно задать параметры печати.
Параметры печати могут отличаться в зависимости от принтера и используемой операционной системы.
Принтер
Указание принтера по умолчанию для текущего документа.
Сведения для принтера отображаются под этим полем.
Если выбирается другой принтер, сведения обновляются.
Имя
Список установленных принтеров в операционной системе.
Чтобы изменить принтер по умолчанию, выберите имя принтера из списка.
Состояние
Описание текущего состояния выбранного принтера.
Тип
Отображение типа выбранного принтера.
Расположение
Отображение порта для выбранного принтера.
Комментарий
Отображение дополнительных сведений для принтера.
Свойства
Открытие диалогового окна свойств принтеров.
Свойства принтера изменяются в соответствии с выбранным принтером.
Печатать в файл
Печать в файл вместо принтера.
Выберите это поле и место для сохранения файла.
