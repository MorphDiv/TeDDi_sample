Входными данными для функции являются одиночный текстовый аргумент, от которого зависят возвращаемые данные.

Значение параметра &quot; Тип &quot;
Возвращаемое значение

Тип

функция CURRENT
CURRENT
Эта функция возвращает результат частичного вычисления формулы, в которую она входит (другими словами, результат выполненной части вычислений).
В основном она используется совместно с функцией STYLE() для применения выбранных стилей к ячейке в зависимости от ее содержимого.
Синтаксис
CURRENT()
Пример
=1+2+CURRENT()
=1+2+CURRENT()-5
будет возвращено значение 1, так как к моменту вызова CURRENT() вычислен результат 1+2.
Соответственно, CURRENT() возвращает 3; 3+3-5 вернет 1.
=A2+B2+STYLE(IF(CURRENT() > 10; &quot; Красный &quot; ; &quot; По умолчанию &quot; ))
STYLE
.
= &quot; choo &quot; & CURRENT()
Пример возвращает choochoo.

ISREF
Проверяет, является ли данное выражение ссылкой.
Возвращает значение TRUE, если аргумент является ссылкой; в противном случае возвращается FALSE.
Если в эту функцию передается ссылка, значение, на которое она ссылается, не анализируется.
Синтаксис
ISREF(Значение)
Значение
: значение для проверки с целью определения наличия ссылки.
Пример
=ISREF(C5)
возвращает результат TRUE, поскольку C5 - действительная ссылка.
=ISREF( &quot; abcdef &quot; )
всегда возвращает FALSE, поскольку текст не может являться ссылкой.
=ISREF(4)
возвращает FALSE.
=ISREF(INDIRECT( &quot; A6 &quot; ))
возвращает TRUE, поскольку функция INDIRECT возвращает ссылку.
=ISREF(ADDRESS (1; 1; 2; &quot; Sheet2 &quot; ))
возвращает FALSE, так как функция ADDRESS возвращает текст, который можно принять за ссылку.

функция N
N
Возвращает числовое значение выбранного параметра.
Возвращает 0, если параметр является текстом, FALSE или #NA.
Если происходит ошибка, отличная от #NA, функция возвращает значение ошибки.
Синтаксис
N(Значение)
Значение
: параметр для преобразования в число.
N() возвращает числовое значение, если возможно.
Логические значения TRUE или FALSE возвращаются как 1 и 0 соответственно.
Текст и ошибки возвращаются как 0.
Пример
=N(123)
возвращает значение 123.
=N(TRUE)
возвращает значение 1.
=N(FALSE)
возвращает значение 0.
=N( &quot; abc &quot; )
возвращает значение 0.
=N(1/0) возвращает значение #DIV/0!

CELL
Возвращает информацию об адресе, форматировании или содержимом ячейки.
Синтаксис
CELL( &quot; Тип информации &quot; ; Ссылка)
Тип информации
- строка символов, указывающая на тип информации.
Эта строка всегда содержит текст на английском языке.
Верхний и нижний регистры не учитываются.
Тип информации
Значение
COL
Возвращает номер столбца в ссылке.
=CELL( &quot; COL &quot; ;D2)
возвращает значение 4.
ROW
Возвращает номер строки в ссылке.
=CELL( &quot; ROW &quot; ;D2)
возвращает значение 2.
SHEET
Возвращает номер листа в ссылке.
=CELL( &quot; Sheet &quot; ; Sheet3.D2)
возвращает значение 3.
ADDRESS
Возвращает абсолютный адрес ячейки.
=CELL( &quot; ADDRESS &quot; ;D2)
возвращает значение $D$2.
=CELL( &quot; ADDRESS &quot; ;Sheet3.D2)
возвращает значение $Sheet3.$D$2.
=CELL( &quot; ADDRESS &quot; ;'X:\dr est.sxc'#$Sheet1.D2)
возвращает значение 'file:///X:/dr/test.sxc'#$Sheet1.$D$2.
FILENAME
Возвращает имя файла и номер листа для ячейки.
=CELL( &quot; FILENAME &quot; ;D2)
возвращает значение 'file:///X:/dr/own.sxc'#$Sheet1, если формула в текущем документе X:\dr\own.sxc расположена на листе Sheet1.
=CELL( &quot; FILENAME &quot; ;'X:\dr est.sxc'#$Sheet1.D2)
возвращает 'file:///X:/dr/test.sxc'#$Sheet1.
COORD
Возвращает полный адрес ячейки в нотации Lotus(TM).
=CELL( &quot; COORD &quot; ; D2)
возвращает $A:$D$2.
=CELL( &quot; COORD &quot; ; Sheet3.D2)
возвращает значение $C:$D$2.
CONTENTS
Возвращает содержимое ячейки без форматирования.
TYPE
Возвращает тип содержимого ячейки.
b = пусто.
Пустая ячейка
l = подпись.
Текст, результат формулы в виде текста
v = значение.
Значение, результат формулы в виде числа
WIDTH
Возвращает ширину заданного столбца.
Ширина измеряется числом нулей (0) в столбце стандартного размера.
ПРЕФИКС
Возвращает выравнивание заданного столбца.
' = выравнивание по левому краю
&quot; = выравнивание по правому краю
^ = выравнивание по центру
\ = повторение (в текущей версии недоступно)
PROTECT
Возвращает состояние защиты для ячейки.
1 = ячейка защищена
0 = ячейка не защищена
FORMAT
Возвращает строку символов, обозначающую формат числа.
, = число с разделителем разрядов
F = число без разделителя разрядов
C = денежный формат
S = экспоненциальное представление, например 1,234+E56
P = процент
В перечисленных выше форматах количество знаков после запятой дается в виде числа.
Например, формат числа # ##0,0 возвращает ,1, а формат числа 00,000% возвращает P3
D1 = MMM-D-YY, MM-D-YY и аналогичные форматы
D2 = DD-MM
D3 = MM-YY
D4 = DD-MM-YYYY HH:MM:SS
D5 = MM-DD
D6 = HH:MM:SS AM/PM
D7 = HH:MM AM/PM
D8 = HH:MM:SS
D9 = HH:MM
G = все остальные форматы
- (знак минуса) в конце = отрицательные числа выделяются цветом
() (скобки) в конце = в коде формата содержится открывающая скобка
COLOR
Возвращает значение 1, если отрицательные значения выделяются цветом; в противном случае возвращает значение 0.
PARENTHESES
Возвращает значение 1, если код формата содержит открывающую скобку; в противном случае возвращает значение 0.
Ссылка
(список параметров).
Позиция ячейки для проверки.
Если список
Ссылка
содержит диапазон, ячейка перемещается в левый верхний угол диапазона.
Если список
Ссылка
отсутствует, $[officename] Calc использует позицию ячейки, в которой содержится эта формула.
Microsoft Excel использует ссылку на ячейку, в которой находится курсор.
