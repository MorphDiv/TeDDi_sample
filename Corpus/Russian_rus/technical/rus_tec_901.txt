Метки
выберите формат листов меток, которые требуется распечатать.
Выберите базу данных и таблицу, из которых требуется извлечь данные.
Выберите поле базы данных, содержимое которого требуется напечатать.
Нажмите кнопку с изображением направленной влево стрелки для вставки поля базы данных в поле &quot; Текст метки &quot; .
Если для каждой метки требуется большее количество полей, то можно выбрать и вставить дополнительные поля базы данных.
Для вставки новой строки нажмите клавишу Enter, для вставки фиксированного текста введите любые символы.
