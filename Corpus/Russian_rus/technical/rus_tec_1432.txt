Java
Установка параметров поддержки приложений Java в %PRODUCTNAME, в том числе используемых Java Runtime Environment (JRE).
Параметры Java
Использовать среду выполнения Java
Позволяет выполнять приложения Java в %PRODUCTNAME.
При попытке доступа к жесткому диску со стороны приложения Java выводится запрос.
Установленные среды выполнения Java (JRE):
Выберите требуемую JRE.
В некоторых системах необходимо ожидать заполнения списков в течение непродолжительного времени.
В некоторых системах для использования измененных настроек необходимо перезапустить %PRODUCTNAME.
Путь к JRE отображается под полем списка.
Добавить

Путь задается в следующем диалоговом окне.
