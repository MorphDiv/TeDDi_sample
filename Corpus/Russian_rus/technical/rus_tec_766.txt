Уровень безопасности
безопасности макросов
.
Эти варианты отличаются степенью безопасности.
Макросы, для которых разрешен запуск на высоком уровне безопасности, также разрешены на всех более низких уровнях.
Очень высокий
Разрешено запускать макросы только из расположений надежных файлов.
Все остальные подписанные и неподписанные макросы запрещены.
Расположения надежных файлов задаются на вкладке Надежные источники.
Разрешено запускать любой макрос из расположений надежных файлов.
Высокий
Разрешено запускать только подписанные макросы из доверенных источников.
Неподписанные макросы запрещены.
Доверенные источники задаются на вкладке Надежные источники.
Разрешено запускать только подписанные макросы из доверенного источника.
Кроме того, разрешено запускать любой макрос из расположений надежных файлов.
Средний
Требуется подтверждение на запуск макросов из неизвестных источников.
Доверенные источники задаются на вкладке Надежные источники.
Разрешено запускать только подписанные макросы из доверенного источника.
Кроме того, разрешено запускать любой макрос из расположений надежных файлов.
Для всех остальных макросов требуется подтверждение.
Низкий (не рекомендуется)
Все макросы выполняются без подтверждения.
Этот параметр следует использовать только при полной уверенности в безопасности всех открываемых документов.
Для макроса можно задать автозапуск, что может привести к потенциально опасным действиям, например, к удалению или переименованию файлов.
Этот параметр настройки не рекомендуется использовать при открытии документов от других авторов.
