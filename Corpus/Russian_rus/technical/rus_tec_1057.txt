Мастер запуска
Окно мастера запуска открывается при первом запуске %PRODUCTNAME.
Мастер состоит из следующих страниц:
Добро пожаловать
Мастер запуска поможет установить пользовательские параметры для %PRODUCTNAME.
Этот мастер запускается только один раз.
Лицензия
Прочтите и примите текст лицензии для продолжения.
Для продолжения необходимо прокрутить текст лицензии до конца.
В случае отказа от принятия лицензии установка прекращается.
Перенос данных
Эта страница служит для импорта пользовательских данных в предварительно установленную версию %PRODUCTNAME.
Копирование личных данных
Копирование имени пользователя и других данных из предыдущей установки %PRODUCTNAME.
Имя пользователя
Имя пользователя указывается в свойствах документа и шаблонах, а также при сохранении изменений в документах.
Эта страница отображается только в случае, если предыдущая установка %PRODUCTNAME не найдена.
Введите личные данные в текстовые поля.
