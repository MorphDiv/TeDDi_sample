Добавление текстур для столбцов графиков
С помощью графических растровых изображений можно добавить текстуру для панелей диаграммы (вместо цветов по умолчанию).
Переключитесь в режим редактирования, дважды щелкнув диаграмму.
Дважды щелкните по любому столбцу ряда, который требуется отредактировать.
Будут выделены все столбцы этого ряда.
Для редактирования только одной полосы снова дважды щелкните по полосе.
