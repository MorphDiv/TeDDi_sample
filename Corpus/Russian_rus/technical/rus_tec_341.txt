Назначение сценариев в %PRODUCTNAME
В %PRODUCTNAME существует возможность назначения пользовательских сценариев (макросов) пунктам меню, значкам, элементам управления в диалоговых окнах и событиям.
В %PRODUCTNAME встроена поддержка следующих языков сценариев:
%PRODUCTNAME Basic
JavaScript
BeanShell
Python (входит не во все дистрибутивы)
Кроме того, для внешнего управления %PRODUCTNAME разработчики могут использовать языки более высокого уровня, например язык программирования Java.
См. проект API в OpenOffice.org.
