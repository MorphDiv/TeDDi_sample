Медиаплеер
Открывает окно &quot; Медиаплеер &quot; , в котором можно просмотреть звуковые и видеофайлы, а также вставить их в текущий документ.
Медиаплеер поддерживает множество различных медиаформатов.
Можно также вставить медиафайлы из медиаплеера в документ.
В системах Linux и Solaris для медиаплеера требуется API Java Media Framework (JMF).
Загрузите и установите файлы JMF с веб-сайта http://java.sun.com/javase/technologies/desktop/media/jmf/index.jsp и добавьте путь к установленному файлу jmf.jar к пути к классу в меню &quot; Сервис – Параметры – %PRODUCTNAME – Java &quot; .
В системах Windows медиаплеер использует программу DirectShow, которая должна быть установлена в системе по умолчанию.
Открыть
Открывает видео- или звуковой файл для просмотра.
Применить
Вставляет текущий видео- или звуковой файл как медиаобъект в текущий документ.
Воспр.
Воспроизводит текущий файл.
Пауза
Приостанавливает или возобновляет воспроизведение текущего файла.
Остановить
Останавливает воспроизведение текущего файла.
Повторить
Воспроизводит файл повторно.
Без звука
Выключает и включает звук.
Регулятор громкости
Регулирует громкость.
Масштаб
Регулирует размер воспроизводимого видеофайла.
Регулятор положения
Перемещается в различные положения в файле.
