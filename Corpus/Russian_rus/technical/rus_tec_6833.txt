Модуль Галерея изображений
Как пользоваться модулем Галерея Изображений
Модуль Галерея изображений можно запустить из меню & konqueror; Сервис Создать галерею изображений, или нажав Ctrl; I.
При запуске модуля Галерея изображений, открывается диалог, предназначенный для настройки отображения Галереи.
Вид страницы
Снимок экрана Диалога просмотра
Снимок экрана
Диалог просмотра предназначен для настройки внешнего вида странички галереи.
Заголовок страницы
Опция Заголовок страницы содержит строку ввода текста для выбора названия странички Галереи изображений.
Изображений в ряд
Опция Изображений в ряд содержит ползунок и окно числового ввода для выбора количества эскизов, отображаемых в ряд.
Показать имя файлов изображений
Флажок Показать имя файлов изображений предназначен для выбора - отображать или не отображать имя файла под эскизом.
Показывать размер файла изображения
Флажок Показывать размер файла изображения служит для выбора - отображать или не отображать размер файла изображения под эскизом.
Показывать размер файла изображения
Флажок Показывать размер файла изображения служит для выбора - отображать или не отображать размер файла изображения под эскизом.
Шрифт
Список Шрифт служит для выбора шрифта, которым будут отображаться надписи в Галерее.
Размер шрифта
Окошко Размер шрифта предназначено для выбора размера выбранного шрифта.
Цвет надписей
Опция Цвет надписей вызывает окно выбора цвета надписей.
Цвет фона
Опция Цвет фона вызывает окно выбора цвета фона галереи.
Каталоги
Снимок экрана диалога Каталоги
Диалог Каталоги предназначен для выбора папки, в которую будет сохранена Галерея изображений. Также этот диалог используется для настройки используемых папок.
Окно Сохранить в предназначено для выбора папки, в которую будет сохранена html- страничка.
Все каталоги рекурсивно
Флажок Все каталоги рекурсивно включает рекурсию при выборе подкаталогов. Если флажок установлен, ползунок и окно ввода числа служат для выбора глубины рекурсии.
Копировать исходные файлы
Установка флажка Копировать исходные файлы позволяет создать папку, содержащую копии исходных изображений, отображаемых в Галерее.
Использовать файл комментариев
Флажок Использовать файл комментария вызывает окно выбора файла комментария к создаваемой Галерее.
Уменьшенные копии изображений
Снимок экрана диалога Уменьшенные копии изображений
Диалог Уменьшенные копии изображений предназначен для настройки свойств отображения эскизов в галерее.
Формат для предпросмотра
Список выбора Формат для предпросмотра предназначен для выбора формата графических файлов эскизов.
Размер уменьшенных копий
Опция Размер уменьшенных копий включает ползунок и окно ввода числа для настройки размера эскизов в создаваемой Галерее.
Установить разную глубину цвета
Флажок Установить разную глубину цвета разрешает отображение выпадающего списка выбора глубины цвета эскизов, отличной от глубины цвета исходного изображения.
