Журнал, открываемый при запуске
Определяет файл журнала, показываемый при запуске. Значение по умолчанию — либо /var/adm/messages, либо /var/log/messages, в зависимости от операционной системы.
Размер шрифта для показа журнала
Определяет размер моноширинного шрифта, используемого для показа журнала в главном окне. По умолчанию — размер шрифта терминала по умолчанию.
Высота главного окна в пикселах
Определяет высоту главного окна программы просмотра журналов в пикселах.
Ширина главного окна в пикселах
Определяет ширину главного окна программы просмотра журналов в пикселах.
Журналы, открываемые при запуске
Определяет список файлов журналов, открываемых при запуске. Список по умолчанию создаётся путём чтения /etc/syslog.conf.
