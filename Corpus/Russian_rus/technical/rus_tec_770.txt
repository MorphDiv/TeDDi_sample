Аутентификация на сервере
для задания параметров безопасности сервера.
Сервер исходящих сообщений (SMTP) требует аутентификации
Включает аутентификацию, которая необходима для отправки электронной почты с помощью SMTP.
SMTP-сервер требует отдельной аутентификации
Выберите этот параметр, если сервер SMTP требует ввода имени пользователя и пароля.
Имя пользователя
Введите имя пользователя для сервера SMTP.
Пароль
Введите пароль для этого имени пользователя.
Сервер исходящей почты использует ту же аутентификацию, что и сервер входящей почты.
При необходимости выберите чтение электронной почты перед возможностью отправки электронной почты.
Этот метод также называется &quot; SMTP после POP3 &quot; .
Имя сервера
Введите имя сервера для почтового сервера POP 3 или IMAP.
Порт
Укажите порт на сервере POP3 или IMAP.
POP 3
Указывает, что сервер входящей почты использует протокол POP 3.
IMAP
Указывает, что сервер входящей почты использует протокол IMAP.
Имя пользователя
Введите имя пользователя для сервера IMAP.
Пароль
Введите пароль.
