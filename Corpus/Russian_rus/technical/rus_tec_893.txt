Индексы
Организация индексов базы данных dBASE.
Индекс позволяет быстро получить доступ к базе данных при условии запроса данных из выделенной области, определенной с помощью указателя.
При создании таблицы индексы задаются на вкладке
Индексы
.
Таблица
Выберите таблицу базы данных, которую нужно индексировать.
Индексы таблицы
Перечисление текущих индексов для выбранной таблицы базы данных.
Чтобы удалить индекс из списка, выберите его и нажмите стрелку ВПРАВО.
Свободные индексы
Перечисление доступных индексов, которые можно назначить таблице.
Чтобы назначить индекс выбранной таблице, щелкните значок со стрелкой ВЛЕВО.
Двойная стрелка ВЛЕВО служит для назначения всех доступных индексов.
