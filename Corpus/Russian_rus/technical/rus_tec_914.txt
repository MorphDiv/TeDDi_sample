Подключение к ADO
Задает параметры для добавления базы данных ADO (Microsoft ActiveX Data Objects).
Интерфейс ADO является собственным средством Microsoft Windows для подключения к базам данных.
Для использования интерфейса ADO $[officename] требует Microsoft Data Access Components (MDAC).
Microsoft Windows 2000 и XP устанавливают эти компоненты по умолчанию.
Для более ранних версий Windows требуется дополнительно установить MDAC.
MDAC можно загрузить на сайте Microsoft.
URL-адрес источника данных
Введите URL-адрес источника данных.
Пример URL
Для соединения с файлами Access 2000 используйте следующий формат:
PROVIDER=Microsoft.Jet.OLEDB.4.0;DATA SOURCE=c:\Access\nwind2000.mdb
Для соединения по имени к каталогу Microsoft SQL сервера, имя сервера которого turner, введите:
PROVIDER=sqloledb;DATA SOURCE=turner;INITIAL CATALOG=First
Для подключения, используя ODBC драйвер:
DSN=SQLSERVER
Обзор
Чтобы открыть диалоговое окно выбора базы данных, нажмите эту кнопку.

Пароль должен содержать от 3 до 18 символов.
