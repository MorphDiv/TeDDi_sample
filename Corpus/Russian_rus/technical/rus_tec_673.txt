Мастер таблиц - Выбрать типы и формат
Задает описания для выбранных полей.
Выбранные поля
Выберите поле, чтобы изменить его описание.
-
Удаляет выбранное поле из поля со списком.
+
Добавляет в список новое текстовое поле.
Описание полей
Имя поля
Отображает имя выбранного поля данных.
При желании можно ввести другое имя.
Тип поля
Выберите тип поля.
Автозначение
Если установить &quot; Да &quot; , значения данного поля будут генерироваться базой данных.
Обязательное
Если задано значение &quot; Да &quot; , поле не может быть оставлено пустым.
Длина
Задает количество символов для поля данных.
Число десятичных знаков
Задает количество десятичных разрядов для поля данных.
Этот параметр доступен только для числовых и десятичных полей данных.
Значение по умолчанию
Задает значение по умолчанию для поля &quot; Да/Нет &quot; .
Автоинкрементная инструкция
Введите спецификатор команды SQL, задающий автоприращение определенного поля целочисленных значений в источнике данных.
Например, следующая инструкция MySQL использует оператор AUTO_INCREMENT для увеличения поля &quot; id &quot; при создании каждого нового поля данных:
CREATE TABLE &quot; table1 &quot; ( &quot; id &quot; INTEGER AUTO_INCREMENT)
В этом примере следует ввести в поле &quot; Автоинкрементная инструкция &quot; значение &quot; AUTO_INCREMENT &quot; .
