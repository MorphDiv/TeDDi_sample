Параметры печати
Открывает диалоговое окно, где можно задать параметры напечатанного документа.
Размер страниц пропорционально уменьшается.
В случае печати нескольких страниц документа на одной печатной странице, возможно, она будет заполнена не вся и останется пустое место.
Параметры печати
, будут задействованы только в том случае, если документ распечатывается с помощью значка
Печать
.
Значок
Параметры печати: вид страницы
Распределение
Определяет количество строк и столбцов для печати, а также размер при печати.
Кроме того, можно задать вывод нескольких страниц документа на одну печатную страницу.
Строки
Определяет количество рядов страниц (горизонтально расставленных страниц документа).
Столбцы
Определяет количество столбцов страниц (вертикально расставленных страниц документа).
Поля
В этой области можно задать поля печати.
Слева
Задает ширину левого поля.
Сверху
Задает высоту верхнего поля.
Справа
Задает ширину правого поля.
Снизу
Задает высоту нижнего поля.
Интервал
В этом разделе можно задать расстояния по горизонтали и по вертикали между уменьшенными страницами документа на одном листе.
По горизонтали
Задает расстояние между страницами документа по горизонтали.
По вертикали
Задает расстояние между страницами документа по вертикали.
Ориентация
Задает ориентацию страницы при печати.
альбомная
Печать страницы в альбомной ориентации.
книжная
Печать страницы в книжной ориентации.
По умолчанию
При печати просмотра страниц количество строк и столбцов не задается фиксировано, а определяется исходным предварительным просмотром.
