Справочник команд
Меню Файл
Ctrl; N Файл Создать...
Создать новый проект. Открытый в данный момент проект не будет тронут.
Ctrl; O Файл Открыть...
Открыть существующий проект. Открытый в данный момент проект не будет затронут.
Файл Загрузить примеры баз данных...
Открыть диалог программы KNewStuff, который позволит вам загрузить примеры баз данных из интернета.
Ctrl; S Файл Сохранить
Сохраняет изменения объекта в активном окне.
Файл Импорт Данные...
Импортирует данные таблицы из файла в формате CSV (Comma separated value - значения, разделённые запятыми.
Файл Экспорт Данные...
Экспортирует данные из активной таблицы или запросы в файл в формате CSV (значения, разделённые запятыми).
Ctrl; P Файл Печать...
Печатает данные из активной (открытой) таблицы или запроса.
Заметка для KDE: удостоверьтесь, что выставлена правильная печатающая система (секция Текущая печатающая система). Эта опцию можно посмотреть позже нажатием на кнопку Опцииgt; gt;.
Файл Предварительный просмотр...
Предпросмотр активной (открытой) таблицы или запроса.
Файл Настрить печать...
Шрифт... для Заголовок страницы:, Изменить... Размер страницы и поля и Добавить границы таблицы.
Файл Закрыть проект
Закрыть текущий документ, но не закрывать & kexi;.
Ctrl; Q Файл Выход
Выйти из & kexi;.
Меню Правка
Ctrl; Z Правка Отменить действие
Откат действия. Вы можете вернуться к состоянию перед изменениями.
Ctrl; & Shift; Z Правка Повторить
"Откат отката" Обращает действие отката. Возвращает изменения, которые вы изначально сделали.
Ctrl; X Правка Вырезать
Удалить выделенный текст и поместить его в буфер обмена. Эта команда недоступна, если нет выделенного текста. Используя эту функцию из дизайнера форм, вы удалите выделенный элемент интерфейса или группу элементов и поместите их в буфер обмена.
Ctrl; C Правка Копировать
Скопировть выделенный текст в буфер обмена. Эта команда недоступна, если нет выделенного текста. Если вы используете эту функцию из дизайнера форм, будут скопированы элементы интрефейса.
Ctrl; V Правка Вставить
Скопировать содержимое буфера обмена в данную точку. Эта команда недоступна, если буфер пуст. Если эта команда ипользуется из дизайнера форм и буфер содержит скопированные элементы интерфейса, то они будут вставлены в форму.
Правка Специальное копирование Данные таблицы или запроса...
Скопировать выбранные данные таблицы или запроса в буфер обмена.
Правка Специальная вставка Таблица базы данных...
Вставить данные из буфера обмена в новую таблицу текущего проекта.
Ctrl; A Правка Выбрать все
Выбрать все символы в текстовом поле ввода или все элементы интерфейса дизайнера форм.
Правка Удалить
Удалить выделенный объект.
Ctrl; Delete Правка Удалить запись
Удалить выделенную строку из таблицы.
Меню Вид
F6 Вид Представление данных
Просмотр данных.
F7 Вид Представление проектирования
Переключиться в просмотр проектирования.
F8 Вид Представление текста
Переключиться в текстовое представление. В данный момент доступно только для запросов и означает переключает в просмотр SQL в проектировщике запросов.
Alt; 1 Вид Проект
Перейти к навигатору проекта.
Alt; 2 Вид Свойства
Перейти в главную область.
Alt; 2 Вид Свойства
Перейти в панель свойств.
Вид Показать/ Скрыть свойства
Показать или скрыть панель свойств.
Вид Показать/ Скрыть навигатор проекта
Показать или скрыть панель проекта.
Меню Вставка
Вставка Таблица...
Вставка новой пустой таблицы без сохранения. Появится окно дизайнера таблиц.
Вставка Запрос...
Вставка нового пустого запроса без сохранения. Появится окно дизайнера запроса.
Вставка Форма...
Вставка новой пустой формы без сохранения. Появится окно дизайнера форм.
Вставка Сценарий...
Вставить новый сценарий без сохранения. Появится окно редактора сценариев. Эта команда доступна только если сценарии доступны.
Меню Формат
Формат Шрифт...
Изменить шрифт для выделенного объекта. Может быть использовано из дизайнера форм для установки шрифта элемента интерфейса.
Формат Привязать к сетке
Если включено, верхний левый угол элемента интерфейса будет привязан к сетке. Это ограничивает свободу перемещения элементов, однако позволяет выстроить их точнее.
Формат Разместить элементы управления
Создать новую планировку элементов. Они могут быть расположены горизонтально, вертикально, выстроены по сетке, горизонтально с разделителем или вертикально с разделителем.
Формат Разорвать макет
Отменяет выделенную планировку.
Формат Выстроить элементы управления
Выстроить выделенные элементы: слева, справа, сверху, снизу, по сетке.
Формат Настроить размер виджетов
Выставить размер выделенных элементов: подгонка, по сетке, по самому короткому, по самому длинному, по самому узкому, по самому широкому.
Формат Переместить элемент управления выше
Перенести выделенные элементы на передний план.
Формат Переместить элемент управления ниже
Отправить выделенные элементы на задний план.
Меню Данные
Shift; Return Данные Сохранить запись
Сохранить выделенную строку данных.
Данные Отменить изменение записи
Отменить изменения, сделанные с текущей строкой.
Данные Сортировать По возрастанию
Отсортировать данные в возростающем порядка (от A до Z, от 0 до 9). Данные из выделенной колонки будут использоваться для сортировки.
Данные Сортировать По убыванию
Отсортировать данные в убывающем порядке (от Z до A, от 9 до 0). Данные из выделенной колонки будут использовать для сортировки.
Меню Сервис
Сервис Импорт базы данных...
Открыть мастер импорта базы данных Импортировать существующую базу данных.
Сервис Выполнить сценарий...
Отобразить диалог открытия файла для выбора сценария.
Сервис Управление сценариями...
Отобразить диалог Управление сценариями... для выполнения различных действий со сценариями.
Сервис Сценарии
Выполнить загруженный сценарий.
