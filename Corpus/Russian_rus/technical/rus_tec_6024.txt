Описание команд
Следующая раскладка клавиш предполагает что вы не изменяли настройки по умолчанию.
Главное окно & knode;
Меню Файл
Ctrl; S Файл Сохранить
Сохраняет текущую статью в файл.
Ctrl; P Файл Печать
Печать выбранной статьи
Файл Отправить незаконченные сообщения
Все сообщение из папки Исходящее будут отправлены.
Файл Остановить работу с сетью
Отключиться от текущего сервера.
Ctrl; Q Файл Выход
Выход из & knode;.
Меню Правка
Ctrl; C Правка Копировать
Копировать выбранный текст в буфер обмена.
Ctrl; A Правка Выделить всё
Выбрать всю статью.
F4 Правка Поиск статей
Открыть диалог поиска статьи для текущей группы.
Ctrl; F Правка Найти
Правка Получить статью с идентификатором...
Позволяет загрузить статью по указанному идентификатору.
Меню Вид
Вид Показать дискуссии
Если установлен, & knode; отображает дискуссии как дерево в окне заголовков.
Вид Развернуть все дискуссии
Если установлен, & knode; отображает полную дискуссию. Это работает только если установлен параметр Показывать дискуссии.
Вид Свернуть все дискуссии
Если установлен, & knode;, не отображает дискуссии, Это работает только если установлен параметр Показывать дискуссии.
T Вид Переключить показ вложенных дискуссий
Позволяет развернуть или свернуть текущую дискуссию.
Вид Фильтр
Позволяет вам выбрать фильтр для списка статей.
Вид Сортировать
Позволяет вам сортировать список статей.
F5 Вид Обновить список
Обновить список статей.
Вид Расшифровать (ROT 13)
Если этот параметр установлен, & knode; отображает все символы статьи смещёнными на 13 символов.
Вид Проверить подпись PGP
Проверить у статьи подпись PGP.
T Вид Переключить показ вложенных дискуссий
Исходный код текущей статьи будет открыт в отдельном окне.
X Вид Использовать моноширинный шрифт
Активирует настроенный моноширинный шрифт для отображения статей.
Вид Установить кодировку
Здесь вы можете настроить кодировку, которая будет использоваться для статей.
Меню Вперёд
B Вперёд Предыдущая статья
Переключиться на предыдущую статью в списке.
N Вперёд Следующая статья
Переключиться на следующую статью в списке.
Alt; Space Вперёд Следующая непрочитанная статья
Перейти к следующей непрочитанной статье либо к непрочитанной статье в следующей группе, если необходимо.
Ctrl; Пробел Вперёд Следующая непрочитанная дискуссия
Перейти к следующей непрочитанной дискуссии либо к непрочитанной дискуссии в следующей группе, если необходимо.
- Вперёд Предыдущая группа
Переключиться на предыдущую группу новостей в окне каталогов.
+ Вперёд Следующая группа
Переключиться на следующую группу новостей в окне каталогов.
Меню Учётная запись
Учётная запись Получить новые статьи во всех группах
Подключиться с использованием текущей учётной записи и получить новые сообщения.
Учётная запись Получить новые статьи во всех группах
Подключиться с использованием текущей учётной записи и получить новые сообщения.
Учётная запись Подписаться на группы новостей
Открывает диалог подписки на телеконференции для текущей учётной записи.
Учётная запись Пометить все группы как устаревшие
Здесь вы можете отметить все группы учётной записи как устаревшие.
Учётная запись Учётная запись
Открывает диалог свойств текущей учётной записи.
Учётная запись Удалить учётную запись
Удаляет текущую учётную запись и все её телеконференции.
Меню Группа
Группа Получить новые статьи
Группа Пометить группу как устаревшую
Проверка на наличие устаревших статей и их удаление.
Группа Реорганизовать группу
Реорганизовать окно заголовков используя настройки сортировки.
Группа Отметить все статьи как прочтённые
Установка статуса Прочтённые для всех статей в текущей группе.
Группа Отметить все статьи как непрочтённые
Установка статуса Непрочтённые для всех статей в текущей группе.
Группа Отметить все статьи как непрочтённые
Группа Свойства группы
Открытие диалога свойств группы.
Группа Отписаться от группы
Отменить подписку на выбранную группу.
Меню Папка
Папка New Folder
Создаёт новую основную папку.
Папка Новая подпапка
Создаёт новую подпапку.
Папка Переименовать папку
Здесь вы можете переименовать текущую папку.
Папка Импорт папки MBox
Позволяет импортировать каталог MBox в текущий каталог.
Папка Экспорт папки MBox
Позволяет экспортировать текущий каталог в каталог формата MBox.
Папка Сжать папку
Удалить статьи, помеченные как удалённые из текущей папки.
Папка Сжать все папки
Удалить статьи, помеченные как удалённые из всех папок.
Папка Очистить папку
Удалить все статьи из текущей папки.
Папка Удалить папку
Удалить текущую папку.
Меню Статья
Ctrl; W Файл Закрыть
Открывает редактор с настройками для создания новой статьи.
R Статья Ответить
Открывает редактор для ответа на текущую статью.
A Статья Ответить по почте
Открывает редактор для ответа по электронной почте на текущую статью.
F Статья Переслать по почте
Открывает редактор для пересылки текущей статьи по электронной почте.
D Статья Отметить как прочтённую
Установить статус текущей статьи как прочтённая.
U Статья Отметить как непрочтённую
Установить статус текущей статьи как непрочтённая
Ctrl; D Статья Отметить дискуссию как прочтённую
Установить статус текущей дискуссии как прочтённая
Ctrl; U Статья Отметить дискуссию как непрочтённую
Установить статус текущей дискуссии как непрочтённая
Статья Отменить статью
Создать сообщение удаления текущей статьи из Usenet. Вы можете удалять только собственные статьи.
Статья Заменить статью
Открывает текущую статью в редакторе. Когда эта статья будет отправлена, она заменит оригинальную статью. Вы можете заменять только собственные статьи.
O Статья Открыть в отдельном окне
Текущая статья будет открыта в отдельном окне.
E Статья Редактировать статью...
Открывает редактор, для изменения текущей статьи. Вы можете использовать это только в папках Исходящее и Черновики.
Delete Статья Удалить статью
Удалить текущую статью. Вы можете удалять статьи только в папках Исходящие и Черновики.
Статья Отправить немедленно
Немедленно отправить текущую статью. Вы можете использовать это только в папках Исходящие и Черновики.
Меню Отметка
Ctrl; E Отметка Изменить правила отметки...
Открывает диалог правил отметки.
Отметка Переустановить отметки
Перезапускает и пересчитывает отметки.
Ctrl; L Отметка Убрать отметку автора...
Создаёт правило для понижения отметки всех статей автора текущей.
Ctrl; I Отметка Отметить статьи автора...
Создаёт правило для повышения отметки всех статей автора текущей.
W Отметка Следить за дискуссией
Устанавливает отметку "Следить за дискуссией" на текущую дискуссию (по умолчанию = 100).
I Отметка Игнорировать дискуссию
Устанавливает отметку "Игнорировать дискуссию" на текущую дискуссию (по умолчанию = - 100).
Меню Настройка
Настройка Показать панель инструментов
Позволяет переключать отображение панели инструментов.
Настройка Показать строку состояния
Позволяет переключать отображение строки состояния.
Ctrl; Q Файл Выход
Позволяет переключать отображение панели групп.
Ctrl; S Файл Сохранить
Позволяет переключать отображение панели заголовков.
Ctrl; S Файл Сохранить
Позволяет переключать отображение панели тела статьи.
Настройка Настройка горячих клавиш...
Открывает диалог настройки горячих клавиш.
Настройка Настроить панели инструментов...
Открывает диалог настройки панелей инструментов.
Настройка Настроить KNode...
Открывает диалог настройки & knode;.

Меню редактора.
Ctrl; Return Файл Отправить немедленно
Отправляет текущую статью немедленно.

Сохраняет текущую статью в папку Исходящее, чтобы отправить её позже.
Файл Сохранить как черновик
Сохраняет текущую статью в папку Черновики, чтобы можно было редактировать её позже.
Файл Удалить
Удаляет текущую статью и закрывает редактор.

Закрывает окно редактора.

Отменить последнее действие.
Ctrl; Z Правка Отменить действие
Повторить последнее отменённое действие.
Ctrl; X Правка Вырезать
Вырезает выделенный текст в буфер обмена, и удаляет его в окне редактора.
Ctrl; C Правка Копировать
Копировать выбранный текст в буфер обмена.
Ctrl; V Правка Вставить
Вставляет текущее содержимое буфера обмена в окно редактора.
Правка Вставить как цитату
Вставляет текущее содержимое буфера обмена в окно редактора с символами цитирования (gt;) в начале каждой строки.
Правка Выделить всё
Выделяет весь текст в окне редактора.
Ctrl; F Правка Найти
Открывает диалог поиска.
Ctrl; R Правка Заменить...
Открывает диалог замены.
Меню Вложить
Вложить Добавить подпись
Вставляет вашу подпись в конец редактируемой статьи.
Вложить Вставить файл...
Вставляет содержимое файла в окно редактора.
Вложить Вставить файл (в рамке)...
Вставляет содержимое файла в редактор и ограничивает его рамкой.
Вложить Вложить файл...
Вставить файл как вложение.
Меню Настройки
Настройки Послать на сервер новостей
Переключает отправку статей на сервер новостей.
Настройки Отправить по почте
Переключает отсылку сообщений как статьи или нет. Если установлен, будет использоваться внешний редактор.
Настройки Установить кодировку
Здесь вы можете указать кодировку, которая будет использоваться для текущей статьи. Обычно вы используете кодировку us- ASCII для англоговорящих стран.
Настройки Перенос слов
Переключает перенос слов в редакторе.
Меню Сервис
Сервис Добавить символы цитаты
Помещает gt; в начало выделенных строк.
Сервис Удалить символы цитаты
Удаляет символы цитирования из начала выделенных строк.
Сервис Добавить рамку
Помещает выбеленные строки в ASCII- рамку
Сервис Удалить рамку
Удаляет ASCII- рамку из выделенных строк.
Сервис Подписать статью ключом PGP
Подписать текущую статью с помощью PGP.
Сервис Оригинальный текст (без переносов)
Восстанавливает отправку оригинала по почте при ответе на статью.
Сервис Расшифровать (Rot- 13)
Зашифровать выбеленный текст, сдвигом каждого символа в алфавите на 13 позиций.
Сервис Запустить внешний редактор
Запускает внешний редактор (если он настроен), с содержанием текущего окна редактора.
Ctrl; Z Сервис Проверка правописания...
Открывает диалог проверки орфографии.
Настройка Показать панель инструментов
Позволяет переключать отображение панели инструментов.
Настройка Показать строку состояния
Позволяет переключать отображение строки состояния.
Настройка Настроить KNode...
Открывает диалог настройки & knode;.
Ctrl; Shift Z Правка Повторить отменённое действие
Вид Показать все заголовки
Если этот параметр установлен, & knode; отображает в окне статьи её полный заголовок.
