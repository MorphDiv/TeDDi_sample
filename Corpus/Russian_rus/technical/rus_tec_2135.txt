функция ISLEAPYEAR
определение високосного года
mw added one entry
ISLEAPYEAR
Служит для определения високосного года.
Если год високосный, функция возвращает значение 1 (TRUE); в противном случае возвращается значение 0 (FALSE).
Синтаксис
ISLEAPYEAR( &quot; Дата &quot; )
Дата
указывает, находится ли данная дата в пределах високосного года.
Параметр &quot; Дата &quot; должен представлять собой допустимую дату в соответствии с настройками локали %PRODUCTNAME.
Пример
=ISLEAPYEAR(A1) возвращает значение 1, если ячейка A1 содержит значение 1968-02-29, поскольку дата 29 февраля 1968 допустима при выбранных параметрах языка.
Также можно использовать функцию =ISLEAPYEAR( &quot; 1968-02-29 &quot; ) или =ISLEAPYEAR( &quot; 2/29/68 &quot; ).
Не следует использовать =ISLEAPYEAR(2/29/68), поскольку в этом случае сначала 29 делится на 2, затем - на 68, после чего функция ISLEAPYEAR использует полученное число в качестве даты в числовом формате.

функция ROT13
шифрование текста
mw added one entry
ROT13
Служит для шифрования строки символов путем их перемещения на 13 позиций в алфавите.
После буквы Я алфавит начинается снова (повторяется).
При повторном применении функции шифрования к полученному коду можно произвести расшифровку.
Синтаксис
ROT13(Текст)
Текст
- символьная строка для шифрования.
ROT13(ROT13 (Текст)) дешифрует код.

Дата
- любая дата в соответствующем году.
Параметр &quot; Дата &quot; должен представлять собой допустимую дату в соответствии с настройками локали %PRODUCTNAME.
Пример
WEEKSINYEAR(A1) возвращает значение 53, если ячейка A1 содержит значение 1970-02-17, которое является допустимой датой для 1970 года.
