Лист
Здесь можно установить, какие элементы будут добавлены при печати всех листов в соответствии с текущим стилем страницы.
Дополнительно можно установить порядок страниц при печати, номер первой печатаемой страницы, а также масштаб печати.
Печать
Здесь определяется, какие элементы электронной таблицы будут напечатаны.
Заголовки строк и столбцов
Здесь указывается, будут ли напечатаны заголовки строк и столбцов.
Сетка

Комментарии
Выводит на печать комментарии, введенные в электронной таблице.
Каждый комментарий печатается на отдельной странице со ссылкой на соответствующую ячейку.
Внедренные и графические объекты
Этот параметр задает печать всех внедренных объектов (если их можно напечатать) и графических объектов в документе.
Диаграммы
Этот параметр задает печать диаграмм, добавленных в электронную таблицу.
Рисованные объекты
Этот параметр задает печать всех рисованных объектов в документе.
Формулы
При выборе этого параметра в ячейках печатаются формулы, а не результаты.
Нулевые значения
При выборе этого параметра ячейки с нулевыми значениями будут напечатаны.
Порядок страниц
Здесь определяется порядок нумерации данных на листе и их печати в случае, если данные не умещаются на одной печатной странице.
Сверху вниз, затем направо
При выборе этого параметра печать выполняется вертикально с левого столбца до низа листа.
Слева направо, затем вниз
При выборе этого параметра печать выполняется горизонтально с первой строки листа до правого столбца.
Номер первой страницы
Выберите этот параметр, если требуется, чтобы номер первой страницы не был равен 1.
Введите номер первой страницы.
Масштаб
Здесь устанавливается масштаб печатаемых электронных таблиц.
Режим масштабирования
Выберите режим масштабирования из списка.
Сбоку от списка будут отображены соответствующие элементы управления.
Уменьшить или увеличить распечатку
Здесь задается коэффициент масштабирования для всех печатаемых страниц.
Коэффициент масштабирования
Введите коэффициент масштабирования.
Значения менее 100 приводят к уменьшению страницы, более высокие - к ее увеличению.
Подбор диапазона(ов) печати по ширине/высоте
Здесь указывается максимальное число страниц по горизонтали (ширина) и по вертикали (высота), на которых печатается каждый лист для текущего стиля страницы.
Диапазоны печати масштабируются всегда с сохранением пропорций, поэтому итоговое число страниц может быть меньше указанного.
Если снять один из флажков, то для неуказанного измерения будет использоваться столько страниц, сколько нужно.
Если снять оба флажка, будет установлен коэффициент масштабирования 100%.
Ширина в страницах
Введите максимальное число страниц для печати по горизонтали.
Высота в страницах
Введите максимальное число страниц для печати по вертикали.
Подбор диапазона(ов) печати по числу страниц
Здесь указывается максимальное число страниц, на которых будет печататься каждый лист для текущего стиля страницы.
При необходимости масштаб будет уменьшен, чтобы лист поместился на указанное число страниц.
Количество страниц
Введите максимальное количество печатаемых страниц.
