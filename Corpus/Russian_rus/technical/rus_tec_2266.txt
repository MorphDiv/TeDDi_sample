Параметры
решателя
используется для определения нескольких параметров.
solver engine
Выберите механизм решателя.
Если установлен только один механизм решателя, список недоступен.
Механизмы решателя могут быть установлены как расширения.
settings
Настройка текущего решателя.
edit
Если текущий элемент в списке &quot; Параметры &quot; допускает правку значения, то будет доступна кнопка &quot; Изменить &quot; .
При ее нажатии открывается диалоговое окно, в котором можно изменить значение.
edit settings spin box
Введите или измените значение.
Используйте диалоговое окно &quot; Параметры &quot; для настройки текущего механизма решателя.
Дополнительные механизмы решателя (при их наличии) можно установить в качестве расширений.
Откройте меню &quot; Сервис - Управление расширениями &quot; и выберите &quot; Веб-сайт с расширениями &quot; для поиска расширений.
Выберите из списка механизм решателя для использования и настройки.
Если установлен только один механизм решателя, список недоступен.
В поле &quot; Параметры &quot; проверьте все установки для текущей операции решения обратной задачи.
Если текущий параметр может принимать различные значения, то активируется кнопка &quot; Правка &quot; .
Щелкните &quot; Правка &quot; для открытия диалогового окна, в котором можно изменить значение.
