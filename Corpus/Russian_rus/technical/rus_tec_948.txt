Сортировка и группировка
построителя отчетов
можно определить поля для сортировки в отчете и поля для объединения в группу.
При группировке отчета по определенному полю все записи с одинаковым значением в этом поле будут объединяться в одну группу.
В поле &quot; Группы &quot; отображаются поля в порядке сверху вниз.
Можно выбрать любое поле и затем для перемещения этого поля вверх или вниз по списку нажать кнопку &quot; Вверх &quot; или &quot; Вниз &quot; .
Сортировка и группировка применяется согласно последовательности списка сверху вниз.
Вывод списка полей, которые используются для сортировки или группировки.
Верхнее поле имеет наивысший приоритет, второе поле имеет вторичный приоритет, и т. д.
Служит для открытия списка с целью выбора поля.
Перемещение выбранного поля вверх в списке.
Перемещение выбранного поля вниз по списку.
Выбор порядка сортировки.
Отображение или скрытие верхнего колонтитула группы.
Отображение или скрытие нижнего колонтитула группы.
Создание новой группы при каждом изменении значения или в зависимости от других свойств.
Используется для удаления выбранного в списке поля.
По умолчанию новая группа создается при каждом изменении значения записи из выбранного поля.
Это свойство можно изменить в зависимости от типа поля:
Для полей типа &quot; Текст &quot; можно выбрать &quot; Начальные символы &quot; и ввести количество n символов в текстовом поле ниже.
Будут группироваться записи, у которых совпадают первые n символов.
Для полей типа &quot; Дата/Время &quot; записи можно группировать по одному году, четверти, месяцу, неделе, дню, часу или минуте.
Имеется возможность дополнительного определения интервала в неделях и часах: при интервале 2 недели данные группируются в двухнедельные группы, при интервале 12 часов − в группы неполного рабочего дня.
Для полей типа &quot; Автонумерация &quot; , &quot; Валюта &quot; или &quot; Номер &quot; необходимо задать интервал.
Ввод значения интервала группы, по которому требуется группировать записи.
Выбор уровня детализации, в соответствии с которым выполняется объединение группы на одной странице.
При определении необходимости объединения нескольких записей на одной странице возможно три варианта:
Нет − границы страницы не учитываются.
&quot; Вся группа &quot; − используется для печати на одной странице верхнего колонтитула группы, раздела подробных данных и нижнего колонтитула группы.
&quot; С первым содержимым &quot; − используется для печати верхнего колонтитула группы на странице только при условии печати на этой странице первой подробной записи.
