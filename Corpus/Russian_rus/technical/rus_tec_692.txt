Параметры
Эта команда открывает диалоговое окно для изменения параметров программы.
Все параметры сохраняются автоматически.
Для развертывания элемента необходимо дважды щелкнуть по нему или щелкнуть по знаку &quot; плюс &quot; перед ним.
Для свертывания элемента необходимо щелкнуть по знаку &quot; минус &quot; или дважды щелкнуть по самому элементу.
Отображаются только элементы, применимые к текущему документу.
Если текущий документ является текстовым документом, отображается элемент %PRODUCTNAME Writer - аналогично для всех модулей %PRODUCTNAME. %PRODUCTNAME Impress и %PRODUCTNAME Draw обрабатываются в этом диалоговом окне тем же способом.
Общие элементы отображаются всегда.
Выберите элемент для изменения.
