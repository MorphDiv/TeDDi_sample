Команды меню
Чтобы использовать команды меню, необходимо выбрать окно, содержащее документ, с которым будет выполняться работа.
Подобным образом необходимо выделить объект в документе, чтобы использовать команды меню, связанные с объектом.
Эти меню зависимы от контекста.
То есть доступными будут те пункты меню, которые связаны с работой, выполняемой в текущий момент.
Если курсор установлен на тексте, будут доступны все пункты меню, необходимые для редактирования текста.
Если выделен графический объект в документе, будут доступны все пункты меню, которые можно использовать для редактирования графических объектов.
