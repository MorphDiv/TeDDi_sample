Печать
Указывает параметры печати для текстового или HTML-документа.
Параметры
в диалоговом окне
Файл - Печать
.
Содержимое
Определяет, какое содержимое документа следует печатать.
Графические объекты
Указывает, печатаются ли графические объекты этого текстового документа.
Таблицы
Указывает, печатаются ли таблицы этого текстового документа.
Рисунки (не для HTML-документов)
Указывает, печатаются ли графические объекты, созданные с помощью функций рисования $[officename].
Элементы управления
Указывает, печатаются ли элементы управления форм этого текстового документа.
Фон
Указывает, включаются ли в печатный документ цвета и объекты, вставленные в фон этой страницы (Формат - Страница - Фон).
Печать черн. цветом
Указывает, следует ли всегда печатать текст черным цветом.
Скрытый текст
Выбор этого параметра позволяет распечатывать текст, отмеченный как скрытый.
Печатается следующий скрытый текст: текст, отформатированный как скрытый в меню
Формат - Символ - Эффекты шрифта - Скрытый
, а также текстовые поля с признаком
Скрытый текст и скрытые абзацы
.
Подстан.текст в полях
Выбор этого параметра позволяет распечатывать рамки текстовых полей.
Если при печати
поля
для размещения текста должны оставаться пустыми, отключите этот режим.
Страницы
Определяет порядок печати многостраничных документов $[officename] Writer.
Левые страницы (не для HTML-документов)
Используется для определения необходимости печати всех левых (четных) страниц документа.
Правые страницы (не для HTML-документов)
Используется для определения необходимости печати всех правых (нечетных) страниц документа.
В обратном порядке
Используется для определения необходимости изменения порядка печати страниц на противоположный.
В этом случае последняя страница документа будет напечатана первой.
Брошюрой
Брошюрой
,чтобы напечатать документ в формате брошюры.
В $[officename] Writer это происходит следующим образом.
Правые страницы
, а затем снова вставить всю стопку бумаги в принтер и распечатать все левые страницы с установленным флажком
Левые страницы
.
Справа налево
Используется для печати страниц брошюры в порядке, соответствующем написанию справа налево.
Комментарии
Не печатать
Используется для определения отсутствия необходимости печати комментариев к документу.
Только комментарии
Используется для определения необходимости печати только комментариев к документу, но не самого текста.
В конце документа
Используется для определения необходимости печати комментариев в конце документа после основного текста.
В конце страницы
Используется для определения необходимости печати комментариев в конце каждой страницы, содержащей комментарии.
Прочее
Печатать автоматически вставленные пустые страницы
При выборе этой опции происходит печать пустых страниц, вставленных автоматически.
Это удобно при двухсторонней печати.
Предположим, что в руководстве установлен такой стиль абзаца главы, при котором глава всегда начинается с нечетной страницы.
Если предыдущая глава заканчивается на нечетной странице, %PRODUCTNAME вставляет четную пустую страницу.
С помощью данного параметра можно указать, следует ли выводить на печать эту четную страницу.
Каждое задание с нового листа
Указывает, что каждое новое задание печати будет начинаться с новой страницы, даже при использовании дуплексного принтера.
Если этот флажок не установлен, то при нечетном числе страниц первая страница второй копии может быть напечатана на обратной стороне последней страницы первой копии.
Подача бумаги согласно настройкам принтера
Для принтеров с несколькими лотками для бумаги параметр &quot; Подача бумаги в соответствии с параметрами принтера &quot; определяет, используется ли способ подачи бумаги, указанный в системных параметрах принтера.
Факс
Если на компьютере установлено программное обеспечение для работы с факсами и необходимо отправить факс прямо из текстового документа, выберите нужный факс.
