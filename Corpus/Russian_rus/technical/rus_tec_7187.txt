Диалоговое окно календарь
Используйте параметр
Диалоговое окно календарь поддерживает следующие параметры:
Указывает текст, который будет изображён в диалоговом окне календаря.
Указывает день, который будет выделен в диалоговом окне календаря.
Указывает месяц, который будет выделен в диалоговом окне календаря.
Указывает год, который будет выбран в диалоговом окне календаря.

Указывает формат даты, в котором она будет возвращена после выбора из диалогового окна. Формат даты по умолчанию зависит от локали.
Следующий сценарий является примером того, как создавать диалоговое окно календаря:
#!/bin/sh


 if zenity --calendar \\
 --title="Выберите Дату" \\
 --text="Щёлкните на дате, чтобы выбрать её." \\
 --day=10 --month=8 --year=2004
 then echo $?
 else echo "Дата не выбрана"
 fi

Диалоговое окно хода выполнения
Используйте параметр

Диалоговое окно индикатора хода выполнения поддерживает следующие параметры:
Указывает текст, который будет изображён в диалоговом окне.

Указывает начальное состояние индикатора хода выполнения.
--auto-close
Диалоговое окно закрывается, когда процент выполнения достигает 100%.
--pulsate
Указывает, что индикатор хода выполнения пульсирует до тех пор, пока из стандартного ввода не будет прочитан символ конца файла EOF.
Следующий сценарий является примером того, как можно создавать диалоговое окно индикатора хода выполнения:
#!/bin/sh
 (
 echo "10" ; sleep 1
 echo "# Обновление журнала почты" ; sleep 1
 echo "20" ; sleep 1
 echo "# Сброс задачи cron" ; sleep 1
 echo "50" ; sleep 1
 echo "Это строка будет просто проигнорирована" ; sleep 1
 echo "75" ; sleep 1
 echo "# Перезагрузка системы" ; sleep 1
 echo "100" ; sleep 1
 ) |
 zenity --progress \\
 --title="Обновление системного журнала" \\
 --text="Просмотр журналы почты..." \\
 --percentage=0

 if [ "$?" = -1 ] ; then
 zenity --error \\
 --text="Обновление отменено."
 fi

#!/bin/sh

 FILE=`zenity --file-selection \\
 --title="Выберите файл"`

 case $? in
 0)
 zenity --text-info \\
 --title=$FILE \\
 --filename=$FILE \\
 --editable 2gt;/tmp/tmp.txt;;
 1)
 echo "Файл не выбран.";;
 -1)
 echo "Файл не выбран.";;
 esac
