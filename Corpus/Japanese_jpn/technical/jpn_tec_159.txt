ダイアログ ハイパーリンク
の見出し
インターネット
では、ハイパーリンクの WWW アドレスまたは FTP アドレスを編集できます。
ログイン名、パスワード、匿名ユーザーのフィールドは、FTP アドレスの場合だけ使用できます。
ハイパーリンクの種類
Web
HTTP ハイパーリンクを作成します。
FTP
FTP ハイパーリンクを作成します。
Telnet
Telnet ハイパーリンクを作成します。
ジャンプ先
ハイパーリンクをクリックしたときに開くファイルの URL を入力します。ターゲットフレームを指定しない場合、ファイルは現在のドキュメントまたはフレーム内に開かれます。
WWW ブラウザ
目的の URL を読み込むことができる Web ブラウザが表示されます。
そのあと、
ターゲット
フィールドに URL をコピーして貼り付けることができます。
ドキュメントにあるターゲット
ドキュメント内のターゲット
ダイアログが開きます。
そこでドキュメントの中にあるターゲットを選択し、
挿入
ボタンでターゲット URL に追加します。
ターゲットドキュメント内でジャンプ先となる場所を指定します。
ハイパーリンク
ダイアログの
ターゲット
フィールドで指定したターゲットを挿入します。
閉じる
をクリックすると、リンクが設定され、ダイアログが終了されます。
ログイン名
FTP アドレスを使用する場合に、ログイン名を指定します。
パスワード
FTP アドレスの場合には、ここにパスワードを入力します。
匿名のユーザー
FTP アドレスに対して、匿名ユーザーとしてログインできます。
詳細設定
フレーム
リンク先のファイルを開くフレームの名前を入力するか、事前定義のフレームをリストから選択します。このボックスに何も指定しない場合、リンク先のファイルは現在のブラウザウィンドウに表示されます。
フォーム
ハイパーリンクを、テキストとして挿入するのかボタンとして挿入するのかを指定します。
イベント
マクロの割り当て
ダイアログが開きます。
テキスト
ハイパーリンクに表示するテキストまたはボタンのラベルを指定します。
名前
ハイパーリンクに付ける名前を入力します。
$[officename] はハイパーリンクに NAME タグを挿入します。
