CreateObject 関数 [実行時]
UNO オブジェクトを作成します。Windows では、OLE オブジェクトも作成できます。
このメソッドにより、パラメータとして渡されるタイプのインスタンスが作成されます。
構文:
oObj = CreateObject( type )
例:
Type address
Name1 as String
City as String
End Type
