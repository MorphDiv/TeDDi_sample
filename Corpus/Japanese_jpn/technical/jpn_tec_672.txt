メディアプレイヤー
「メディアプレイヤー」ウィンドウが表示され、ムービーファイルやサウンドファイルのプレビューを行うことができ、これらのファイルを現在のドキュメントに挿入することができます。
メディアプレイヤーは、多様なメディア形式をサポートしています。メディアプレイヤーからドキュメントにメディアファイルを挿入することもできます。
Linux または Solaris システムで、メディアプレイヤーは Java Media Framework API (JMF) を必要とします。http://java.sun.com/javase/technologies/desktop/media/jmf/index.jsp から JMF ファイルをダウンロード、インストールします。インストールされた jmf.jar のパスを「ツール → オプション → %PRODUCTNAME → Java」のクラスパスに追加します。
Windows システムでは、メディアプレイヤーは、標準でシステムにインストールされている DirectShow を使用します。
開く
プレビュー対象のムービーファイルまたはサウンドファイルを開きます。
適用
現在のムービーファイルまたはサウンドファイルが、現在のドキュメントにメディアオブジェクトとして挿入されます。
再生
現在のファイルが再生されます。
一時停止
現在のファイルのプレイバックを一時停止または再開します。
停止
現在のファイルのプレイバックを停止します。
繰り返し
ファイルが繰り返し再生されます。
ミュート
サウンドのオン/オフを切り替えます。
ボリュームスライダ
ボリュームを調整します。
ズーム
ムービープレイバックのサイズを調節します。
位置スライダ
ファイル内の別の位置に移動します。
