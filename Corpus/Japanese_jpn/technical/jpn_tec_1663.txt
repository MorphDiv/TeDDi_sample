IsDate 関数
IsDate 関数 [実行時]
日付
変数に変換できるかを判定します。
構文:
IsDate (Expression)
戻り値:
ブール型
パラメータ:
Expression:
評価する数値ないし文字列表式。この関数は、与えられた表式が日付データに変換できる場合に
True
を返し、それ以外の場合は
False
を返します。
例:
Sub ExampleIsDate
Dim sDateVar as String
sDateVar = &quot; 1997.12.12 &quot;
print IsDate(sDateVar) REM 戻り値は True
sDateVar = &quot; 19971212 &quot;
print IsDate(sDateVar) REM 戻り値は False
end sub
