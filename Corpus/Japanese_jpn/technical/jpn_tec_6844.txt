使用法: makeinfo [OPTION]... TEXINFO-FILE...
または: texi2any [OPTION]... TEXINFO-FILE...
Texinfo ソース文書を様々な形式で読めるように変換します。デフォルトでは
オンラインで Emacs や単独の GNU Info で読むのに適した Info ファイルです。
出力形式の選択 (デフォルトは Info を生成):
 --docbook Info ではなく Docbook XML を出力する。
 --html Info ではなく HTML を出力する。
 --plaintext Info ではなくプレーンテキストを出力する。
 --xml Info ではなくTexinfo XML を出力する。
 --dvi, --dvipdf, --ps, --pdf texi2dvi を呼び出して指定された出力を生成する。
Info およびプレーンテキストのためのオプション:
 --disable-encoding Info 出力に @documentencoding に基づいたアクセ
 ントおよび特殊文字を出力しない。
 --enable-encoding --disable-encoding を上書きする (デフォルト)。
 --fill-column=NUM Info を文字数 NUM で改行する (default %d)。
 --footnote-style=STYLE STYLE に基づいて Info の脚注を出力する。
 `separate' そのノード中に脚注を置く
 `end' 定義されたノードの末尾に置く (デフォルト)
 --paragraph-indent=VAL Info 段落を VAL 個の空白で字下げする (default %d)。
 VAL が `none' の場合は字下げを行わない。`asis'
 の場合は既存の字下げを保持する。
 --split-size=NUM Info ファイルをサイズ NUM で分割する (default %d)。
HTML のためのオプション:
 --css-include=FILE HTMLの <style> 出力内に FILEを含める。
 FILE に - を指定した時は標準入力から読み込む。
 --css-ref=URL CSS ファイルへの参照を生成する。
 --internal-links=FILE 内部リンクの一覧を FILE に生成する。
 --split=SPLIT SPLIT で分割する。SPLIT は例えば `chapter',
 `section', `node' など。
 --transliterate-file-names
 ファイル名を ASCII 文字変換して生成する。
 --node-files ノードとアンカーのリダイレクションファイルを生成する;
 デフォルトでは分割時にのみ設定される。
入力ファイルオプション:
 --commands-in-node-names 何もしない。互換性のために残っています。
 -D VAR 変数 VAR を定義する。@set と同じ。
 -I DIR @include 探索パスに DIR を追加する。
 -P DIR @include 探索パスの先頭に DIR を挿入する。
 -U VAR 変数 VAR を未定義にする。@clear と同じ。
@if... 条件のデフォルトは出力形式に依存します:
 Docbook を生成する場合、--ifdocbook が有効で、他は無効になります。
 HTML を生成する場合、--ifhtml が有効で、他は無効になります。
 Info を生成する場合、--ifinfo が有効で、他は無効になります。
 平文テキストを生成する場合、--ifplaintext が有効で、他は無効になります。
 XML を生成する場合、--ifxml が有効で、他は無効になります。
