Set ステートメント [実行時]
変数や属性へのオブジェクト参照を設定します。
構文:
Set ObjectVar = Object
パラメータ:
ObjectVar:
オブジェクト参照を設定する変数ないし属性。
Object:
変数や属性への参照を設定するオブジェクト。
Nothing
- 変数に
Nothing
オブジェクトを割り当て、以前の割り当てを削除します。
例:
Sub ExampleSet
Dim oDoc As Object
Set oDoc = ActiveWindow
Print oDoc.Name
End Sub
