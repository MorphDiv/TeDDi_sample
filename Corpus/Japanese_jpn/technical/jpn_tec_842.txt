Web ウィザード - 紹介
Webウィザードは、サーバー上の HTML の維持管理に使用できます。
既存の HTML を管理するために、保存済みの Web ウィザード設定を読み込むこともできます。設定には、ローカルフォルダおよび FTP サーバーに関する情報も含まれています。
Web ウィザードの設定を選択
読み込み
をクリックします。標準設定でウィザードを開始する場合は、「標準」を選択します。
削除
選択した設定が削除されます。
