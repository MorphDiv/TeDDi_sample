LDAP 接続
データベースを LDAP
(
Lightweight Directory Access Protocol)
を使用してインポートするための設定を指定します。このページは、LDAP サーバーをアドレスデータベースとして登録している場合にのみ使用可能です。
サーバー URL
LDAP サーバーの名前を「ldap.server.com」という形式で入力します。
Base DN
LDAP データベースを検索するための開始点を入力します。たとえば、「dc=com」です。
ポート番号
LDAP サーバーのポートを入力します。通常は 389 です。
保護された接続 (SSL) を使用
LDAP サーバーに対して、SSL (Secure Sockets Layer) によるセキュリティ保護された接続を作成します。
標準では、SSL 接続はポート 636 を使用します。通常の接続はポート 389 を使用します。
