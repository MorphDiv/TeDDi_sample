データベースを %PRODUCTNAME Base で使用
%PRODUCTNAME Base では、さまざまなデータベースファイル形式で格納されたデータにアクセスできます。%PRODUCTNAME Base は、dBASE 形式など、いくつかのフラットファイルデータベース形式をサポートします。%PRODUCTNAME Base を使用して、MySQL や Oracle のデータベースなど、外部のリレーショナルデータベースに接続することもできます。
次のデータベースタイプは、%PRODUCTNAME Base では読み取り専用タイプです。 %PRODUCTNAME Base 内から、データベース構造を変更したり、これらのデータベースタイプのデータベースレコードを編集、挿入、削除したりすることはできません。
