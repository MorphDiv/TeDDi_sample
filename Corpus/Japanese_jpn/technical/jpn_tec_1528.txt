ChDir ステートメント [実行時]
現在のディレクトリないしドライブを変更します。
この課題の説明
を参照してください。
構文:
ChDir Text As String
パラメータ:
Text:ディレクトリパスないしドライブを指定する文字列表式。
現在のドライブのみを変更するには、ドライブレターに続けてコロンを指定します。
例:
Sub ExampleChDir
Dim sDir1 as String , sDir2 as String
sDir1 = &quot; c:\Test &quot;
sDir2 = &quot; d:\private &quot;
ChDir( sDir1 )
msgbox CurDir
ChDir( sDir2 )
msgbox CurDir
end sub
