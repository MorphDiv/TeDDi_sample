RANK 関数
数値;順位の判断
RANK
標本に含まれる数値の順位を返します。
構文
RANK(数値;データ;タイプ)
数値
は、順位を求める数値です。
データ
は、行列または標本データの範囲です。
タイプ
(オプション) はシーケンス順です。
Type = 0 は、配列の最後の項目から最初の項目への降順を意味します (これがデフォルトです)。
Type = 1 は、範囲の最初の項目から最後の項目への昇順を意味します。
例
=RANK(A10;A1:A50)
は、値範囲 A1:A50 の A10 の値のランクを返します。
Value
が範囲内に存在しない場合は、エラーメッセージが表示されます。

TTEST

PROB 関数
PROB
ある範囲の値が 2 つの限界内にある確率密度を返します。
End

Start
の値と等しいという原則に基づいて確率密度を計算します。
構文
PROB(Data; Probability; Start; End)
データ
は、標本データの入力されているセル範囲です。
確率
とは、対応する確率の行列もしくは範囲です。
下限

下限

は、A1:A50 の範囲内の値が 50 から 60 の制限内でもある確率を返します。A1:A50 の範囲内のすべての値が、B1:B50 の範囲内にある確率を持っています。
WEIBULL 関数
WEIBULL
ワイブル分布の値を返します。
ワイブル分散は連続確率分布で、パラメータ α > 0 (shape)、β > 0 (scale) を持ちます。
C が 1 なら、WIBULL は確率密度関数を計算します。
C が 0 なら、WEIBULL は累積分散関数を計算します。
構文
WEIBULL(数値;α;β;関数形式)
数値
は、ワイブル分布の算出に用いられる値です。
α
は、ワイブル分布の α パラメータです。
β
は、ワイブル分布の β パラメータです。
関数形式
は、関数の種類です。0 を指定すると密度関数、1 を指定すると分布を計算します。
例
=WEIBULL(2;1;1;1)
は、0.86 を返します。
