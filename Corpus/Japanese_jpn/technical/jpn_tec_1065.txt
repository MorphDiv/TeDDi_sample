Web ウィザード - プレビュー
サイトのプレビューを行うだけでなく、Web サイトを公開する位置を指定します。
プレビュー
オペレーティングシステムの標準の Web ブラウザで、自分の Web ページを開きます。
ローカルディレクトリ
自分の索引ページとファイルをローカルディレクトリにアップロードします。索引ページは、指定した場所に保存されます。それ以外のファイルはすべて、索引ページが保存されたディレクトリの「myWebsite」ディレクトリに保存されます。
...
フォルダを選択できるダイアログを開きます。
アーカイブファイル
索引ページとファイルを圧縮したアーカイブファイルに追加して、その圧縮ファイルを自分の Web サイトにアップロードします。索引ページは、指定した場所に保存されます。それ以外のファイルはすべて、索引ページが保存されたディレクトリの「myWebsite」ディレクトリに保存されます。
使用中のオペレーティングシステムに応じて、使用可能なアーカイブファイルの形式は zip、gzip、および war になります。
...
アーカイブファイルの場所を指定できるダイアログが開きます。
FTP 経由の Web
自分のファイルを FTP サーバーにアップロードします。索引ページは、指定した場所に保存されます。それ以外のファイルはすべて、索引ページが保存されたディレクトリの「myWebsite」ディレクトリに保存されます。
プロキシサーバー経由でインターネットに接続している場合、FTP オプションは使用できません。
設定
FTP Connection
ダイアログを開きます。
保存設定
このウィザードで指定した設定を保存します。
名前を付けて保存
設定ファイルの名前を入力します。
