アドレス帳データソース
このウィザードでは既存のアドレス帳を $[officename] のデータソースとすることができます。
アドレス帳やその他のデータソースはいつでも $[officename] で登録できます。
外部アドレス帳の種別を選択します
すべてのシステムで、すべてのタイプが利用できるわけではありません。
Mozilla / Netscape
Mozilla または Netscape ですでにアドレス帳を使用している場合は、このオプションを選択します。
Thunderbird
Thunderbird ですでにアドレス帳を使用している場合は、このオプションを選択します。
LDAP アドレス帳データ
LDAP サーバー上のアドレス帳をすでに使用している場合は、このオプションを選択します。
Outlook アドレス帳
Microsoft Outlook (Outlook Expressではありません) ですでにアドレス帳を使用している場合は、このオプションを選択します。
Windows システムのアドレス帳
Microsoft Outlook Express ですでにアドレス帳を使用している場合は、このオプションを選択します。
KDE アドレス帳
KDE アドレス帳ですでにアドレス帳を使用している場合は、このオプションを選択します。
Mac OS X アドレス帳
Mac OS X アドレス帳 ですでにアドレス帳を使用している場合は、このオプションを選択します。
Evolution
Evolution ですでにアドレス帳を使用している場合は、このオプションを選択します。
Evolution LDAP
Evolution LDAP ですでにアドレス帳を使用している場合は、このオプションを選択します。
Groupwise
Groupwise ですでにアドレス帳を使用している場合は、このオプションを選択します。
その他の外部データソース
$[officename] のアドレス帳として別の外部データソースを登録する場合、このオプションを選択します。
キャンセル
このコマンドで変更は実行されずにウィザードを終了します。
前の編集段階に戻る
次の編集段階に進む。
完了
データソースへの接続を確立し、ダイアログを閉じます。
