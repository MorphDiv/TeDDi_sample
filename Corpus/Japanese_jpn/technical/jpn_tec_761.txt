クラスパス
Java クラスパスにフォルダおよびアーカイブを追加するために、このダイアログを使用します。どの JRE を起動しても、このパスは有効です。
割り当てられたフォルダとアーカイブ
Java クラスまたは Java クラスライブラリの場所を指定します。
新しいクラスパスは、
%PRODUCTNAME
の再起動後に有効になります。
クラスパスを介してアクセスされる Java クラスには、セキュリティチェックが行われません。
アーカイブを追加
jar 形式または zip 形式のアーカイブファイルを選択して、そのファイルをクラスパスに追加します。
フォルダを追加
フォルダを選択して、クラスパスに追加してください。
削除
リストにあるアーカイブまたはフォルダを選択してから「削除」をクリックして、選択したオブジェクトをクラスパスから削除します。
