ドキュメント変換 １ページ目
$[officename] ドキュメントまたは Microsoft Office ドキュメントのいずれを変換するか指定します。また、変換対象となるドキュメントの種類を選択します。
$[officename]
$[officename] のバイナリドキュメント (StarOffice 5.2 以前) を $[officename] で使用される OpenDocument 形式に変換します。
文書ドキュメント
$[officename] Writer の古い形式である *.sdw ドキュメントを OpenDocument の *.odt ドキュメントに変換します。
表計算ドキュメント
$[officename] Calc の古い形式である *.sdc ドキュメントを OpenDocument の *.ods ドキュメントに変換します。
図形描画/プレゼンテーションドキュメント
$[officename] Draw の古い形式である *.sda ドキュメントを OpenDocument の *.odg ドキュメントに変換し、$[officename] Impress の古い形式である *.sdd ドキュメントを OpenDocument の *.odp ドキュメントに変換します。
マスタードキュメント/数式
$[officename] Writer の古い形式である *.sgl マスタードキュメントを OpenDocument の *.odm ドキュメントに変換し、$[officename] Math の古い形式である *.smf ドキュメントを OpenDocument の *.odf ドキュメントに変換します。
Microsoft Office
Microsoft Office ドキュメントを OpenDocument 形式に変換します。
Word ドキュメント
Microsoft Word の *.doc ドキュメントを OpenDocument の *.odt ドキュメントに変換します。
Excel ドキュメント
Microsoft Excel の *.xls ドキュメントを OpenDocument の *.ods ドキュメントに変換します。
PowerPoint ドキュメント
Microsoft PowerPoint の *.ppt ドキュメントを OpenDocument の *.odp ドキュメントに変換します。
ログファイルの作成
変換されたドキュメントに関する情報を記録するログファイルを作業ディレクトリ内に作成します。
