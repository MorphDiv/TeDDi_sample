プロパティー
Report Builder
の「プロパティー」ウィンドウには常に「Report Builder」ビューで現在選択されているオブジェクトのプロパティーが表示されます。
(Shift) + (F1)
キーを押して、入力ボックスにマウスを合わせると、この入力ボックスのヘルプテキストが表示されます。
データ
タブページが表示されます。
「内容」リストからテーブルを選択し、(Tab) キーを押すか、入力ボックスの外側をクリックして、入力ボックスの選択を解除します。
表示 → フィールドの追加
を選択することもできます。
全般
タブページを使用して、特にレポートの名前を変更し、ページヘッダーエリアまたはページフッターエリアを無効にすることができます。
レポート全体の「データ」または「全般」タブページを表示するには、編集 → レポートの選択を選択します。
Group keep together
グループは、ページまたは列 (デフォルト) でまとめて維持されます。 「まとめる」も有効にする必要があります。
Page header
ページのヘッダーを印刷するコンテキスト (すべてのページに印刷するか、レポートのヘッダーまたはフッターのあるページには印刷しないか) を指定します。
Page footer
ページのフッターを印刷するコンテキスト (すべてのページに印刷するか、レポートのヘッダーまたはフッターのあるページには印刷しないか) を指定します。
Print repeated values
繰り返される値を印刷するように指定します。
全般
タブページが表示されます。
このエリアのいくつかの表示プロパティーを編集できます。
Background color
画面表示および印刷用の両方で、選択したオブジェクトの背景の色を設定します。
Visible
非表示のオブジェクトは、実行されたレポートには表示されませんが、「Report Builder」ビューには表示できます。
Height
選択したオブジェクトの高さを指定します。
Conditional Print Expression
条件付き印刷式が TRUE と評価すると、選択したオブジェクトが印刷されます。

Background transparent
選択したオブジェクトの背景を透明にするか不透明にするかを指定します。
詳細
全般
タブページが表示されます。
いくつかのプロパティーを指定して、レコードの印刷方法を微調整することができます。
Force New Page
「新規ページを強制」を選択すると、現在のセクションまたは次のセクションを新しいページに印刷するように指定されます。
New Row Or Column
複数列のデザインの場合、「新規の行または列」で現在のセクションまたは次のセクションを新しい行または列に印刷するかどうかを指定します。
Keep together
「まとめる」を選択すると、現在のオブジェクトが現在のページに収まらない場合に、新しいページの先頭から開始して印刷するように指定されます。
「詳細」エリアにデータフィールドを挿入するか、任意のエリアにその他のコントロールフィールドを挿入します。 挿入されたフィールドを選択すると、「プロパティー」ウィンドウでプロパティーを設定できます。
「ラベル」フィールドで、「ラベル」入力ボックスに表示されたテキストを変更できます。
画像の場合、画像をファイルへのリンクとして挿入するか、ベースファイルの埋め込みオブジェクトとしてのみ挿入するかを指定できます。 埋め込みを選択するとベースファイルのサイズが大きくなり、リンクを選択するとほかのコンピュータに移すことができません。
Position x
選択したオブジェクトの X 位置を設定します。
Position y
選択したオブジェクトの Y 位置を設定します。
Width
選択したオブジェクトの幅を設定します。
Font
選択したテキストオブジェクトのフォントを選択します。
