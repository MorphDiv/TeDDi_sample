XML フィルタをパッケージとして配付する
専用のパッケージ形式を使用して、XML フィルタを複数のユーザーに配布できます。
XML フィルタをパッケージとして保存するには
「XML フィルタの設定」ダイアログは、文書ドキュメントを開いているときにだけ使用可能です。
ツール → XML フィルタの設定
を選択します。
パッケージとして保存
をクリックします。
パッケージから XML フィルタをインストールするには
「XML フィルタの設定」ダイアログは、テストドキュメントを開いているときにだけ使用可能です。
ツール → XML フィルタの設定
を選択します。
パッケージを開く
をクリックして、インストールするフィルタのパッケージを選択します。
インストール済み XML フィルタを削除するには
ツール → XML フィルタの設定
を選択します。
削除
をクリックします。
