ಚಿತ್ರಗಳು ತಾವಾಗಿಯೆ ಲೋಡ್‌ ಆಗುವಂತೆ ಮಾಡು( u)
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತುಹಾಕಿದಲ್ಲಿ, ಜಾಲ ಪುಟದಲ್ಲಿ ಯಾವುದೆ ಚಿತ್ರಗಳಿದ್ದಲ್ಲಿ ಅದನ್ನು Konqueror ತಾನಾಗಿಯೆ ತೋರಿಸುತ್ತದೆ. ಇಲ್ಲದೆ ಹೋದಲ್ಲಿ, ಅದು ಚಿತ್ರಗಳ ಸ್ಥಳದಲ್ಲಿ ಪ್ಲೇಸ್‌ಹೋಲ್ಡರುಗಳನ್ನು ಇರಿಸುತ್ತದೆ, ಅದನ್ನು ಬಳಸಿಕೊಂಡು ನೀವೆ ಸ್ವತಃ ಚಿತ್ರದ ಗುಂಡಿಯ ಮೇಲೆ ಕ್ಲಿಕ್ಕಿಸುವ ಮೂಲಕ ಚಿತ್ರವನ್ನು ಲೋಡ್ ಮಾಡಬಹುದು. ನಿಮ್ಮ ಅಂತರ್ಜಾಲ ಸಂಪರ್ಕವು ಬಹಳ ನಿಧಾನ ಗತಿಯದ್ದಾಗಿರದೆ ಇದ್ದಲ್ಲಿ ನಿಮ್ಮ ವೀಕ್ಷಣೆಯನ್ನು ಇನ್ನಷ್ಟು ಉತ್ತಮಪಡಿಸಲು ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದರೆ ಒಳಿತು.
ಸಂಪೂರ್ಣವಾಗಿ ಲೋಡ್ ಆಗದ ಚಿತ್ರದ ಸುತ್ತ ಚೌಕಟ್ಟನ್ನು ಎಳೆ( w)
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತುಹಾಕಿದಲ್ಲಿ, ಇನ್ನೂ ಸಂಪೂರ್ಣವಾಗಿ ಲೋಡ್ ಆಗದ ಜಾಲ ಪುಟದಲ್ಲಿ ಅಡಕಗೊಳಿಸಲಾದ ಚಿತ್ರವು ಇರಬೇಕಾದ ಜಾಗದಲ್ಲಿ Konqueror ಒಂದು ಪ್ಲೇಸ್‌ಹೋಲ್ಡರನ್ನು ಎಳೆಯುತ್ತದೆ. ನಿಮ್ಮ ವೀಕ್ಷಣೆಯನ್ನು ಇನ್ನಷ್ಟು ಉತ್ತಮಪಡಿಸಲು ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದರೆ ಒಳಿತು, ವಿಶೇಷವಾಗಿ ನಿಮ್ಮಲ್ಲಿ ನಿಧಾನಗತಿಯ ಅಂತರ್ಜಾಲ ಸಂಪರ್ಕ ಇದ್ದಲ್ಲಿ. animations

ಸಜೀವನ ಚಿತ್ರಗಳನ್ನು Konqueror ಹೇಗೆ ತೋರಿಸುತ್ತದೆ ಎಂಬುದನ್ನು ನಿಯಂತ್ರಿಸುತ್ತದೆ: ಶಕ್ತಗೊಂಡಿದೆ: ಎಲ್ಲಾ ಸಜೀವನಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ತೋರಿಸು. ಅಶಕ್ತಗೊಂಡಿದೆ: ಸಜೀವನಗಳನ್ನು ಎಂದಿಗೂ ತೋರಿಸಬೇಡ, ಕೇವಲ ಆರಂಭದ ಚಿತ್ರವನ್ನು ಮಾತ್ರವೆ ತೋರಿಸು. ಒಮ್ಮೆ ಮಾತ್ರ ತೋರಿಸು: ಎಲ್ಲಾ ಸಜೀವನಗಳನ್ನೂ ತೋರಿಸು ಆದರೆ ಮತ್ತೆಂದೂ ಪುನರಾವರ್ತಿಸಬೇಡ.

ಸರಳ ಸ್ಕ್ರಾಲಿಂಗ್( m):
HTML ಅನ್ನು ಸ್ಕ್ರಾಲ್ ಮಾಡುವಾಗ Konqueror ಸರಳ ಕ್ರಮಗಳನ್ನು ಬಳಸಬೇಕೆ ಅಥವ, ಸಂಪೂರ್ಣ ಕ್ರಮಗಳನ್ನು ಬಳಸಬೇಕೆ ಎಂದು ನಿರ್ಧರಿಸುತ್ತದೆ: ಯಾವಾಗಲೂ: ಸ್ಕ್ರಾಲಿಂಗ್ ಮಾಡುವಾಗ ಯಾವಾಗಲೂ ಸರಳ ಕ್ರಮಗಳನ್ನೆ ಬಳಸು. ಎಂದಿಗೂ ಬೇಡ: ಎಂದಿಗೂ ಸರಳ ಸ್ಕ್ರಾಲಿಂಗ್ ಅನ್ನು ಬಳಸಬೇಡ, ಯಾವಗಲೂ ಪೂರ್ಣಕ್ರಮಗಳನ್ನೆ ಬಳಸು. ಸಮರ್ಥವಾಗಿದ್ದಾಗ: ಗಣಕದ ಕನಿಷ್ಟ ಸಂಪನ್ಮೂಲಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಸರಳ ಸ್ಕ್ರಾಲಿಂಗ್ ಮಾಡುವುದು ಎಲ್ಲಿ ಸಾಧ್ಯವೋ ಅಲ್ಲಿ ಮಾತ್ರ ಬಳಸು.
Konqueror ಅಕ್ಷರ ಶೈಲಿಗಳು, Konqueror ಜಾಲಪುಟಗಳನ್ನು ತೋರಿಸಲು ಯಾವ ಅಕ್ಷರಶೈಲಿಯನ್ನು ಬಳಸಬೇಕು ಎನ್ನುವುದನ್ನು ಈ ಪುಟದಲ್ಲಿ ನೀವು ಸಂರಚಿಸಬಹುದಾಗಿದೆ.
ಅಕ್ಷರಶೈಲಿಯ ಗಾತ್ರ( z)
ಇದು Konqueror ಜಾಲಪುಟಗಳಲ್ಲಿ ಬಳಸುವ ಅಕ್ಷರಶೈಲಿಯ ತುಲನಾತ್ಮಕ ಗಾತ್ರವಾಗಿದೆ.
ಅಕ್ಷರ ಶೈಲಿಯ ಕನಿಷ್ಟ ಗಾತ್ರ (i):
Konqueror ಇದಕ್ಕಿಂತ ಚಿಕ್ಕ ಗಾತ್ರದ ಯಾವುದೆ ಪಠ್ಯವನ್ನು ತೋರಿಸುವುದಿಲ್ಲ, ಹಾಗು ಬೇರಾವುದೆ ಸಂಯೋಜನೆಗಳಿದ್ದರೂ ಇದು ಅತಿಕ್ರಮಿಸುತ್ತದೆ.

ನಿಗದಿತ ಅಕ್ಷರಶೈಲಿ( F):

AdBlocK ಶೋಧಕಗಳನ್ನು ಶಕ್ತಗೊಳಿಸು ಅಥವ ಅಶಕ್ತಗೊಳಿಸು. ನಿರ್ಬಂಧವು ಪರಿಣಾಮಕಾರಿಯಾಗಬೇಕೆಂದರೆ, ಶಕ್ತಗೊಳಿಸಿದಾಗ ಶೋಧಕದ ಪಟ್ಟಿಯಲ್ಲಿ ತಡೆಯಲ್ಪಡಬೇಕಿರುವ ಕೆಲವು ಎಕ್ಸ್‍ಪ್ರೆಶನ್‌ಗಳನ್ನು ಸೂಚಿಸಬೇಕು.
ಶಕ್ತಗೊಳಿಸಿದಾಗ, ನಿರ್ಬಂಧಿಸಲಾದ ಚಿತ್ರಗಳನ್ನು ಪುಟದಿಂದ ಸಂಪೂರ್ಣವಾಗಿ ತೆಗೆದು ಹಾಕಲಾಗುವುದು. ಇಲ್ಲದೆ ಹೋದರೆ ನಿರ್ಬಂಧಿಸಲಾಗಿದೆ ಎಂದು ತೋರಿಸುವ ಒಂದು ಪ್ಲೇಸ್‌ಹೋಲ್ಡರನ್ನು ಬಳಸಲಾಗುತ್ತಿತ್ತು.
ಇದು URL ಶೋಧಕಗಳ ಒಂದು ಪಟ್ಟಿಯಾಗಿದ್ದು, ಲಿಂಕ್ ಮಾಡಲಾದ ಎಲ್ಲಾ ಚಿತ್ರಗಳಿಗೆ ಹಾಗು ಚೌಕಟ್ಟುಗಳಿಗೆ ಅನ್ವಯಿಸಲ್ಪಡುತ್ತದೆ. ಪಟ್ಟಿಯ ಮೇಲ್ಭಾಗದಲ್ಲಿ ಸರ್ವೆ ಸಾಮಾನ್ಯವಾದ ಶೋಧಕಗಳು ಬರುವಂತೆ ಶೋಧಕಗಳನ್ನು ಸಂಸ್ಕರಿಸಲಾಗುತ್ತದೆ.

ನೆಲೆ ಪುಟ:
ನೆಲೆ ಪುಟವನ್ನು ಆರಿಸು
ಇದು, "ನೆಲೆ" ಗುಂಡಿಯನ್ನು ಒತ್ತಿದಾಗ Konqueror (ಜಾಲ ವೀಕ್ಷಕವಾಗಿ) ತೆರಳುವ ಜಾಲಪುಟದ URL ಆಗಿದೆ. ಅದೆ Konqueror ಅನ್ನು ಒಂದು ಕಡತ ವ್ಯವಸ್ಥಾಪಕವಾಗಿ ಆರಂಭಗೊಳಿಸಿದಾಗ ಈ ಗುಂಡಿಯನ್ನು ಒತ್ತಿದಾಗ ಸ್ಥಳೀಯ ನೆಲೆ ಕಡತಕೋಶಕ್ಕೆ ನಿಮ್ಮನ್ನು ಕೊಂಡೊಯ್ಯುತ್ತದೆ.
Konqueror ವೀಕ್ಷಕ ಇಲ್ಲಿ ನೀವು Konqueror ನ ವೀಕ್ಷಕದ ಕಾರ್ಯಶೀಲತೆಯನ್ನು ಸಂರಚಿಸಬಹುದು. "ಕಡತ ವ್ಯವಸ್ಥಾಪಕ" ಸಂರಚನಾ ಮಾಡ್ಯೂಲನ್ನು ಬಳಸಿಕೊಂಡು ಕಡತ ವೀಕ್ಷಕ ಸವಲತ್ತನ್ನು ಸಂರಚಿಸಬೇಕು ಎನ್ನುವುದನ್ನು ದಯವಿಟ್ಟು ನೆನಪಿಡಿ. Konqueror ನಲ್ಲಿ ಲೋಡ್ ಆಗುವ ಜಾಲ ಪುಟಗಳಲ್ಲಿನ HTML ಸಂಜ್ಞೆಯನ್ನು ಹೇಗೆ ನಿಭಾಯಿಸಬೇಕು ಎಂಬುದನ್ನು ನೀವು ಸಂಯೋಜಿಸಬಹುದಾಗಿದೆ. ಸಾಮಾನ್ಯವಾಗಿ ಇಲ್ಲಿ ಏನನ್ನೂ ಬದಲಾಯಿಸುವ ಅಗತ್ಯವಿರುವುದಿಲ್ಲ.
ಬುಕ್‌ಮಾರ್ಕುಗಳು( k)
ಬುಕ್‌ಮಾರ್ಕುಗಳನ್ನು ಸೇರ್ಪಡಿಸುವಾಗ ಅನುಮತಿಯನ್ನು ಕೇಳು
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, ಬುಕ್‌ಮಾರ್ಕಿನ ಶೀರ್ಷಿಕೆಯನ್ನು ಬದಲಾಯಿಸಲು ಹಾಗು ಹೊಸ ಬುಕ್‌ಮಾರ್ಕನ್ನು ನೀವು ಸೇರಿಸಿದಾಗ ಅದನ್ನು ಶೇಖರಿಸಡಲು ಒಂದು ಕಡತಕೋಶವನ್ನು ಆಯ್ಕೆ ಮಾಡಲು Konqueror ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ.
ಬುಕ್‌ಮಾರ್ಕ್ ಉಪಕರಣಪಟ್ಟಿಯಲ್ಲಿ ಕೇವಲ ಗುರುತು ಹಾಕಲಾದ ಬುಕ್‌ಮಾರ್ಕುಗಳನ್ನು ಮಾತ್ರ ತೋರಿಸು
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, Konqueror, ಬುಕ್‌ಮಾರ್ಕ್ ಸಂಪಾದಕದಲ್ಲಿ ನೀವು ತೋರಿಸಲು ಗುರುತು ಹಾಕಿದ ಬುಕ್‌ಮಾರ್ಕ್ ಉಪಕರಣ ಪಟ್ಟಿಯಲ್ಲಿನ ಬುಕ್‌ಮಾರ್ಕುಗಳನ್ನು ಮಾತ್ರವೆ ತೋರಿಸುತ್ತದೆ.
ಫಾರ್ಮ್ ಪೂರ್ಣಗೊಳಿಕೆ( p)
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, ನೀವು ಜಾಲ ಫಾರ್ಮುಗಳಲ್ಲಿ ನಮೂದಿಸಿದ ಮಾಹಿತಿಯನ್ನು Konqueror ನೆನಪಿಟ್ಟುಕೊಳ್ಳುತ್ತದೆ ಹಾಗು ಅದನ್ನು ಬೇರೆ ಎಲ್ಲಾ ಪಾರ್ಮುಗಳಲ್ಲಿ ಕಂಡುಬರಬಹುದಾದ ಅದೆ ರೀತಿಯ ಬೇರೆ ಸ್ಥಳಗಳಲ್ಲಿ ಸೂಚಿಸುತ್ತದೆ.
ಗರಿಷ್ಟ ಪೂರ್ಣಗೊಳಿಕೆ( M):
ಒಂದು ಫಾರ್ಮ್ ಸ್ಥಳಕ್ಕೆ Konqueror ಎಷ್ಟು ಮೌಲ್ಯಗಳನ್ನು ನೆನಪಿಟ್ಟುಕೊಳ್ಳಬೇಕು ಎಂಬುದನ್ನು ಇಲ್ಲಿ ನೀವು ಆಯ್ಕೆ ಮಾಡಬಹುದು.
ಮೌಸ್‌ನ ವರ್ತನೆ( v)
ಕೊಂಡಿಗಳ ಮೇಲೆ ತೆರೆಸೂಚಕ ಕಾಣುವ ಬಗೆಯನ್ನು ಬದಲಾಯಿಸು (g)
ಈ ಆಯ್ಕೆಯನ್ನು ಹೊಂದಿಸಿದಲ್ಲಿ, ತೆರೆಸೂಚಕವನ್ನು ಒಂದು ಹೈಪರ್ಲಿಂಕ್‌ನ ಮೇಲೆ ಕೊಂಡೊಯ್ದಾಗ ತೆರೆಸೂಚಕದ ಆಕಾರವು ಬದಲಾಯಿಸಲ್ಪಡುತ್ತದೆ.
ಯುಆರ್‌ಎಲ್ ಮೇಲೆ ಮಧ್ಯದ ಗುಂಡಿಯನ್ನು ಕ್ಲಿಕ್ಕಿಸಿದಾಗ ಅದು ತೆರೆಯುತ್ತದೆ( i)
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, Konqueror ನೋಟದ ಮಧ್ಯದಲ್ಲಿ ಕ್ಲಿಕ್ ಮಾಡುವುದರಿಂದ ಈ URL ಅನ್ನು ತೆಗೆಯಬಹುದು.
ಬಲ ಕ್ಲಿಕ್ಕಿಸಿದಾಗ ಇತಿಹಾಸಕ್ಕೆ ಮರಳುತ್ತದೆ( b)
ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, Konqueror ನೋಟದ ಮೇಲೆ ಬಲ ಕ್ಲಿಕ್ಕಿಸಿದಾಗ ನೀವು ಇತಿಹಾಸಕ್ಕೆ ಮರಳಬಹುದಾಗಿದೆ. ಈ ಸನ್ನಿವೇಶವನ್ನು ನಿಲುಕಿಸಿಕೊಳ್ಳಲು, ಮೌಸ್‌ನ ಬಲ ಗುಂಡಿಯನ್ನು ಒತ್ತಿ ಹಾಗು ಜರುಗಿಸಿ.
ಸ್ವಯಂಚಾಲಿತವಾಗಿ ತಡವಾಗಿ ಮರುಲೋಡ್‌/ ಪುನರ್ನಿರ್ದೇಶನ ಆಗುವಿಕೆಯನ್ನು ಶಕ್ತಗೊಳಿಸು( r)
ಕೆಲವೊಂದು ಜಾಲಪುಟಗಳಿಗೆ ಒಂದು ನಿರ್ದಿಷ್ಟ ಸಮಯದ ನಂತರ ಸ್ವಯಂಚಾಲಿತ ಮರು ಲೋಡ್‌ ಮಾಡುವಿಕೆ ಅಥವ ಮರು ನಿರ್ದೇಶನದ ಅಗತ್ಯವಿರುತ್ತದೆ. ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕದೆ ಇದ್ದಲ್ಲಿ Konqueror ಇಂತಹ ಮನವಿಗಳನ್ನು ತಿರಸ್ಕರಿಸುತ್ತದೆ.
Ctrl ಕೀಲಿಯನ್ನು ಬಳಸಿಕೊಂಡು ನಿಲುಕಣಾ ಕೀಲಿಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸುವುದನ್ನು ಶಕ್ತ/ ಅಶಕ್ತಗೊಳಿಸು( y)
ಜಾಲಪುಟಗಳನ್ನು ವೀಕ್ಷಿಸುವಾಗ Ctrl ಕೀಲಿಯನ್ನು ಒತ್ತುವುದರಿಂದ KDEಯ ನಿಲುಕಣಾ ಕೀಲಿಗಳು ಶಕ್ತಗೊಳ್ಳುತ್ತವೆ. ಈ ಬಾಕ್ಸನ್ನು ಗುರುತು ಹಾಕದೆ ಇದ್ದಲ್ಲಿ ಈ ನಿಲುಕಣಾ ಸವಲತ್ತನ್ನು ಅಶಕ್ತಗೊಳಿಸುತ್ತದೆ. (ಇದು ಪರಿಣಾಮಕಾರಿಯಾಗಬೇಕೆಂದರೆ Konqueror ಅನ್ನು ಮರಳಿ ಆರಂಭಿಸುವುದು ಅಗತ್ಯ.)

ಇದು HTML ಪುಟಗಳು ಹೊಂದಿರುವ Java ದಲ್ಲಿ ಬರೆಯಲಾದ ಸ್ಕ್ರಿಪ್ಟುಗಳನ್ನು ಕಾರ್ಯಗತಗೊಳಿಸಲು ಅನುವು ಮಾಡಿಕೊಡುತ್ತದೆ. ಯಾವುದೇ ವೀಕ್ಷಕದಲ್ಲಿ ಜರುಗುವಂತೆ, ಸಕ್ರಿಯ ವಿಷಯಗಳನ್ನು ಶಕ್ತಗೊಳಿಸುವುದರಿಂದ ಸುರಕ್ಷತಾ ತೊಂದರೆಗೆ ಕಾರಣವಾಗಬಹುದು ಎಂಬುದನ್ನು ನೆನಪಿಡಿ.
ನೀವು ಯಾವ ಅತಿಥೇಯ ಹಾಗು ಡೊಮೈನ್‌ಗಳಿಗೆ ಆ ಒಂದು ನಿಶ್ಚಿತ Java ಪಾಲಿಸಿಯನ್ನು ಹೊಂದಿಸಿದ್ದೀರೊ ಅದನ್ನು ಈ ಬಾಕ್ಸ್‍ ಹೊಂದಿರುತ್ತದೆ. ಈ ಪಾಲಿಸಿಯನ್ನು, ಈ ಡೊಮೈನ್‌ಗಳು ಹಾಗು ಅತಿಥೇಯಗಳಿಂದ ಕಳಿಸಲಾದ Java ಆಪ್ಲೆಟ್‌ಗಳನ್ನು ಶಕ್ತಗೊಳಿಸಲು ಅಥವ ಅಶಕ್ತಗೊಳಿಸಲು ಬಳಸಲಾಗುವ ಪೂರ್ವನಿಯೋಜಿತ ಪಾಲಿಸಿಯ ಬದಲಿಗೆ ಬಳಸಲಾಗುವುದು. ಒಂದು ಪಾಲಿಸಿಯನ್ನು ಆಯ್ಕೆ ಮಾಡಿ ಹಾಗು ಬಲಭಾಗದಲ್ಲಿನ ನಿಯಂತ್ರಣಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಬಳಸಿಕೊಂಡು ಅದನ್ನು ಮಾರ್ಪಡಿಸಿ.
Java ಪಾಲಿಸಿಗಳನ್ನು ಒಳಗೊಂಡಿರುವ ಕಡತವನ್ನು ಆಯ್ಕೆ ಮಾಡಲು ಈ ಗುಂಡಿಯನ್ನು ಕ್ಲಿಕ್ಕಿಸಿ. ಈ ಪಾಲಿಸಿಗಳನ್ನು ಈಗಿರುವ ಪಾಲಿಸಿಗಳೊಂದಿಗೆ ಒಗ್ಗೂಡಿಸಲಾಗುವುದು. ಒಂದೇ ಬಗೆಯ ಅನೇಕ ನಮೂದುಗಳು ಇದ್ದಲ್ಲಿ ಹೆಚ್ಚಿನವುಗಳನ್ನು ಆಲಕ್ಷಿಸಲಾಗುವುದು.
Java ಪಾಲಿಸಿಗಳನ್ನು ಒಂದು ಝಿಪ್‌ ಮಾಡಲಾದ ಕಡತದಲ್ಲಿ ಇರಿಸಲು ಈ ಗುಂಡಿಯನ್ನು ಕ್ಲಿಕ್ಕಿಸಿ. java_ policy. tgz ಎಂಬ ಹೆಸರಿನ ಕಡತವನ್ನು ನೀವು ಬಯಸುವ ಸ್ಥಳದಲ್ಲಿ ಇರಿಸಲಾಗುವುದು.

ಮೇಲಿನ ಅತಿಥೇಯ ಅಥವ ಡೊಮೈನ್‌ಗಾಗಿ ಪ್ಲಗಿನ್ ಪಾಲಿಸಿಯನ್ನು ಆಯ್ಕೆ ಮಾಡಿ.
ಅತಿಥೇಯ ಅಥವ ಡೊಮೈನ್‌ನ ಹೆಸರು( H):
ಅತಿಥೇಯದ( www. kde. org ನಂತೆ) ಅಥವ ಒಂದು ಚುಕ್ಕಿಯಿಂದ ಆರಂಭಗೊಳ್ಳುವ( ಉದಾ. kde. org ಅಥವ. org) ಒಂದು ಡೊಮೈನ್‌ನ ಹೆಸರನ್ನು ನಮೂದಿಸಿ
ನೀವು ಮೊದಲು ಒಂದು ಡೊಮೈನ್ ಹೆಸರನ್ನು ನಮೂದಿಸಬೇಕು.
ಮೌಸ್‌ನ ಮಧ್ಯದ ಗುಂಡಿಯನ್ನು ಒತ್ತುವ ಮೂಲಕ ಒಂದು ಕೊಂಡಿಯನ್ನು ಅಥವ ಕಡತ ಕೋಶವನ್ನು ಆಯ್ಕೆ ಮಾಡುವಂತಹ ಹಲವಾರು ಸಂದರ್ಭಗಳಲ್ಲಿ ಇದು ಹೊಸ ವಿಂಡೋದ ಬದಲಿಗೆ ಹೊಸ ಹಾಳೆಯನ್ನು ತೆರೆಯುತ್ತದೆ.
ಕೊಂಡಿಗಳನ್ನು ಹೊಸ ವಿಂಡೋದಲ್ಲಿ ತೆರೆಯುವ ಬದಲು ಹೊಸ ಹಾಳೆಯಲ್ಲಿ ತೆರೆಯುತ್ತದೆ( l)
JavaScript ಅನ್ನು ಅನುಮತಿಸಿದಲ್ಲಿ ಅದು ಪುಟಿಕೆಗಳನ್ನು ಹೊಸ ಹಾಳೆ ಅಥವ ಹೊಸ ವಿಂಡೋದಲ್ಲಿ ತೆರೆಯಬೇಕೆ ಅಥವ ಬೇಡವೆ ಎಂದು ನಿರ್ಧರಿಸುತ್ತದೆ.
ಪುಟಿಕೆಗಳನ್ನು ಹೊಸ ವಿಂಡೋದ ಬದಲಿಗೆ ಹೊಸ ಹಾಳೆಯಲ್ಲಿ ತೆಗೆ
ಬೇರೊಂದು KDE ಪ್ರೋಗ್ರಾಂನಲ್ಲಿ ಒಂದು URL‌ ಅನ್ನು ಕ್ಲಿಕ್ ಮಾಡಿದರೆ ಅಥವ kfmclientಗೆ ಒಂದು ಯುರ್ಎಲ್ ಅನ್ನು ತೆರೆಯಲು ಆದೇಶಿಸಿದಾಗ ಈಗಿನ ಗಣಕತೆರೆಯು ಚಿಕ್ಕದಾಗಿಸಿರದ( ನಾನ್‌- ಮಿನಿಮೈಸ್) ಒಂದು ಕಾಂಕರರಿಗಾಗಿ ಹುಡುಕುತ್ತದೆ ಹಾಗು, ದೊರೆತಲ್ಲಿ, URL ಅದರಲ್ಲಿ ಒಂದು ಹೊಸ ಹಾಳೆಯಾಗಿ ತೆರೆಯಲ್ಪಡುತ್ತದೆ. ಇಲ್ಲದೆ ಹೋದಲ್ಲಿ, ಅಗತ್ಯವಿರುವ URL ‌ನೊಂದಿಗೆ ಒಂದು ಹೊಸ Konqueror ವಿಂಡೋ ತೆರೆಯಲ್ಪಡುತ್ತದೆ.
URL‌ ಹೊರಗಿನಿಂದ ತೆಗೆಯಲು ಬಯಸಿದಾಗ, ಈಗಾಗಲೆ ತೆರೆದಿರುವ ಕಾಂಕರರಿನಲ್ಲಿ ಹಾಳೆಯಾಗಿ ತೆರೆ
ಇದು ಒಂದು ಹೊಸ ಹಾಳೆಯನ್ನು ಮುಂಭಾಗದಲ್ಲಿ ತೆರೆಯುವ ಬದಲು ಹಿನ್ನಲೆಯಲ್ಲಿ ತೆರೆಯುತ್ತದೆ.
ಹೊಸ ಹಾಳೆಗಳನ್ನು ಹಿನ್ನಲೆಯಲ್ಲಿ ತೆರೆಯಿರಿ (p)
ಇದು ಹಿಂದಿನ ಹಾಳೆಯ ನಂತರದ ಹಾಳೆಯನ್ನು ತೆರೆಯುವ ಬದಲು ಈಗಿನ ಹಾಳೆಯ ನಂತರದ ಒಂದು ಪುಟದಿಂದ ಹೊಸ ಹಾಳೆಯನ್ನು ತೆರೆಯುತ್ತದೆ.
ಈಗಿನ ಹಾಳೆಯ ನಂತರ ಹೊಸ ಹಾಳೆಯನ್ನು ತೆರೆ (n)
ಎರಡು ಇಲ್ಲ ಎರಡಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಹಾಳೆಗಳು ಇದ್ದಲ್ಲಿ ಮಾತ್ರವೆ ಹಾಳೆ ಪಟ್ಟಿಯನ್ನು ಇದು ತೋರಿಸುತ್ತದೆ. ಇಲ್ಲದೆ ಹೋದಲ್ಲಿ ಅದನ್ನು ಯಾವಾಗಲೂ ತೋರಿಸಲಾಗುವುದು.
ಕೇವಲ ಒಂದು ಹಾಳೆಯು ತೆರೆಯಲ್ಪಟ್ಟಾಗ ಹಾಳೆಯ ಪಟ್ಟಿಯು ಅಡಗಿಸಲ್ಪಡುತ್ತದೆ
ಇದು ಜಾಲತಾಣದ ಚಿಹ್ನೆಯ ಬದಲಾಗಿ ಮುಚ್ಚುವ ಗುಂಡಿಗಳನ್ನು ಪ್ರತಿಯೊಂದು ಹಾಳೆಯ ಒಳಗೆಯೆ ತೋರಿಸುತ್ತದೆ.
ಮುಚ್ಚುವ ಗುಂಡಿಗಳನ್ನು ಹಾಳೆಗಳಲ್ಲಿ ತೋರಿಸು( S)
ಹಾಳೆಯ ಮೇಲೆ ಮೌಸ್‌ನ ಮಧ್ಯದ ಗುಂಡಿಯನ್ನು ಅಥವಾ ಮೌಸ್‌ನ ಚಕ್ರವನ್ನು ಕ್ಲಿಕ್ಕಿಸಿದಾಗ, ಆ ಹಾಳೆಯು ಮುಚ್ಚಲ್ಪಡುತ್ತದೆ. (ಬದಲಾವಣೆಯು ಕಾರ್ಯರೂಪಕ್ಕೆ ಬರಲು Konqueror ಮರು ಆರಂಭಗೊಳ್ಳುವುದು ಅಗತ್ಯವಾಗುತ್ತದೆ.)
ಹಾಳೆಯನ್ನು ಮುಚ್ಚಲು ಅದರ ಮೇಲೆ ಮಧ್ಯದ ಗುಂಡಿಯನ್ನು ಕ್ಲಿಕ್ಕಿಸಿ
ಒಂದು ವಿಂಡೋವು ಹಲವಾರು ಹಾಳೆಗಳನ್ನು ಹೊಂದಿದ್ದಲ್ಲಿ ಅದನ್ನು ನೀವು ಮುಚ್ಚಲು ಪ್ರಯತ್ನಿಸಿದಾಗ ಇದು ನಿಮ್ಮ ಖಚಿತತೆಯನ್ನು ಕೇಳುತ್ತದೆ.
ಅನೇಕ ಹಾಳೆಗಳನ್ನು ಹೊಂದಿರುವ ವಿಂಡೋಗಳನ್ನು ಮುಚ್ಚುವಾಗ ಅನುಮತಿ ಕೇಳು
ಇದನ್ನು ಗುರುತು ಹಾಕಿದಲ್ಲಿ, ಈ ಹಿಂದೆ ಬಳಸಲಾದ ಅಥವ ತೆರೆಯಲಾದ ಹಾಳೆಯು ಸಕ್ರಿಯಗೊಳಿಸಲ್ಪಡುತ್ತದೆ
ಈಗಿನ ಹಾಳೆಯನ್ನು ಮುಚ್ಚುವ ಮೊದಲು ಈ ಹಿಂದೆ ಬಳಸಲಾದ ಹಾಳೆಯನ್ನು ಸಕ್ರಿಯಗೊಳಿಸಿ
