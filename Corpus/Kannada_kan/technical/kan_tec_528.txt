ಅನ್‌ಲಾಕ್ ಮಾಡಲು ಪ್ರಯತ್ನಿಸಿದಾಗ ಒಂದು ಕೀಲಿ ಮಣೆಯನ್ನು ವಿಂಡೋದಲ್ಲಿ ಅಡಕಗೊಳಿಸಲು ಇದನ್ನು TRUE ಗೆ ಹೊಂದಿಸಿ. "keyboard_command" ಕೀಲಿಯು ಸೂಕ್ತವಾದ ಆಜ್ಞೆಯೊಂದಿಗೆ ಒದಗಿಸಿರಬೇಕು.
ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಸಕ್ರಿಯಗೊಂಡಾಗ ತೆರೆಯನ್ನು ಲಾಕ್ ಮಾಡಲು ಇದನ್ನು TRUE ಗೆ (ನಿಜ) ಹೊಂದಿಸಿ.
ಬೇರೊಂದು ಬಳಕೆದಾರ ಖಾತೆಗೆ ಬದಲಾಯಿಸುವ ಒಂದು ಆಯ್ಕೆಯನ್ನು ಅನ್‌ಲಾಕ್‌ ಸಂವಾದದಲ್ಲಿ ಒದಗಿಸಲು ಇದನ್ನು TRUE ಹೊಂದಿಸಿ.
ಒಂದು ಕಾಲ ವಿಳಂಬದ ನಂತರ ನಿರ್ಗಮಿಸುವ ಆಯ್ಕೆಯನ್ನು ಅನ್‌ಲಾಕ್‌ ಸಂವಾದದಲ್ಲಿ ನೀಡಲು ಇದನ್ನು TRUE ಗೆ ಹೊಂದಿಸು. ಕಾಲವಿಳಂಬವು "logout_delay" ಕೀಲಿಯಿಂದ ಸೂಚಿತಗೊಂಡಿರುತ್ತದೆ.
"embedded_keyboard_enabled" ಕೀಲಿಯು TRUE ಗೆ ಹೊಂದಿಸಲ್ಪಟ್ಟಿದ್ದಲ್ಲಿ, ಒಂದು ಕೀಲಿ ವಿಜೆಟ್‌ ಅನ್ನು ವಿಂಡೋಗೆ ಅಡಕಗೊಳಿಸಲು ಚಲಾಯಿಸಬೇಕಿರುವ ಆಜ್ಞೆ. ಈ ಕೀಲಿಯು ಸ್ಟ್ಯಾಂಡರ್ಡ್ ಔಟ್‌ಪುಟ್‌ನಲ್ಲಿ ಒಂದು XEMBED ಪ್ಲಗ್ ಸಂಪರ್ಕಸಾಧನ ಹಾಗು ವಿಂಡೋ XIDಯ ಔಟ್‌ಪುಟ್ ಅನ್ನು ಅನ್ವಯಿಸುತ್ತದೆ.
ನಿರ್ಗಮಿಸುವ ಗುಂಡಿಯನ್ನು ಒತ್ತಿದಾಗ ಕೋರಬೇಕಿರುವ ಆಜ್ಞೆ. ಈ ಆಜ್ಞೆಯು ಏನನ್ನೂ ಪ್ರಶ್ನಿಸದೆ, ಬಳಕೆದಾರರನ್ನು ಹೊರಗೆ ನಿರ್ಗಮಿಸುವಂತೆ ಮಾಡುಬೇಕು. "logout_enable" ಕೀಲಿಯು TRUE ಗೆ ಹೊಂದಿಸಿದಲ್ಲಿ ಮಾತ್ರ ಈ ಕೀಲಿಯು ಪರಿಣಾಮಕಾರಿಯಾಗುತ್ತದೆ.
ತೆರೆಯನ್ನು ಲಾಕ್ ಮಾಡುವ ಮೊದಲು, ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಅನ್ನು ಸಕ್ರಿಯವಾಗಿರಿಸಬೇಕಿರುವ ನಿಮಿಷಗಳ ಸಂಖ್ಯೆ.
ಅನ್‌ಲಾಕ್ ಸಂವಾದದಲ್ಲಿ ನಿರ್ಗಮಿಸುವ ಆಯ್ಕೆಯು ಕಾಣಿಸಿಕೊಳ್ಳುವ ಮೊದಲು, ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಅನ್ನು ಸಕ್ರಿಯವಾಗಿರಿಸಬೇಕಿರುವ ನಿಮಿಷಗಳ ಸಂಖ್ಯೆ. "logout_enable" ಕೀಲಿಯು TRUE ಗೆ ಹೊಂದಿಸಿದಲ್ಲಿ ಮಾತ್ರ ಈ ಕೀಲಿಯು ಪರಿಣಾಮಕಾರಿಯಾಗುತ್ತದೆ.
ಅಧಿವೇಶನದಲ್ಲಿ ಕೆಲಸವಿಲ್ಲ ಎಂದು ಪರಿಗಣಿಸಬೇಕಿರುವ ಮೊದಲು ಅದು ನಿಷ್ಕ್ರಿಯವಾಗಿರಬೇಕಿರುವ ನಿಮಿಷಗಳ ಸಂಖ್ಯೆ.
ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಥೀಮ್ ಅನ್ನು ಬದಲಾಯಿಸುವ ಮೊದಲು ಅದನ್ನು ಚಲಾಯಿಸಬೇಕಿರುವ ನಿಮಿಷಗಳ ಸಂಖ್ಯೆ.
ವಿದ್ಯುಚ್ಛಕ್ತಿ ನಿರ್ವಹಣೆಗೆ ಸೂಚಿಸುವ ಮೊದಲು ನಿಷ್ಕ್ರಿಯತೆಯ ಸಮಯ, ಸೆಕೆಂಡುಗಳಲ್ಲಿ. ಈ ಕೀಲಿಯನ್ನು ಅಣಿಗೊಳಿಸುವುದು ಹಾಗು ನೋಡಿಕೊಳ್ಳುವುದು ಅಧಿವೇಶನದ ವಿದ್ಯುಚ್ಛಕ್ತಿ ನಿರ್ವಹಣಾ ಮಧ್ಯವರ್ತಿಯ ಕೆಲಸವಾಗಿರುತ್ತದೆ.
ಸ್ಕ್ರೀನ್‌ಸೇವರಿನಿಂದ ಬಳಸಬೇಕಿರುವ ಆಯ್ಕೆಯ ವಿಧಾನ. ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಯಾವುದೆ ಥೀಮ್ ಅನ್ನು ಬಳಸದೆ ಇರಲು "ಕೇವಲ ಖಾಲಿ" ಅನ್ನು, ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಕೇವಲ ಒಂದು ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಅನ್ನು ಬಳಸಲು "ಒಂದು" ("ಥೀಮ್‌ಗಳು" ಕೀಲಿಯಿಂದ ಸೂಚಿಸಲಾದ), ಹಾಗು ಸಕ್ರಿಯಗೊಳಿಸಿದಾಗ ಮನಸ್ಸಿಗೆ ಬಂದಂತೆ ಸ್ಕ್ರೀನ್‌ಸೇವರ್ ಅನ್ನು ಬಳಸಲು "ಮನಸ್ಸಿಗೆ ಬಂದಂತೆ" ಅನ್ನು ಶಕ್ತಗೊಳಿಸಬಹುದಾಗಿದೆ.

ಲಾಕ್ ಸಂವಾದಕ್ಕಾಗಿ ಬಳಸಬೇಕಿರುವ ಥೀಮ್.
ಸ್ಕ್ರೀನ್‌ಸೇವರಿನಿಂದ ಬಳಸಲ್ಪಡಬೇಕಿರುವ ಥೀಮ್‌ಗಳ ಪಟ್ಟಿಯನ್ನು ಈ ಕೀಲಿಯು ಸೂಚಿಸುತ್ತದೆ. ಇದು "ಕ್ರಮ" ಕೀಲಿಯು "ಕೇವಲ ಖಾಲಿ" ಇದ್ದಲ್ಲಿ ಆಲಕ್ಷಿಸಲ್ಪಡುತ್ತದೆ, "ಕ್ರಮ" ವು "ಒಂದು" ಆಗಿದ್ದಲ್ಲಿ ಥೀಮ್ ಹೆಸರನ್ನು ನೀಡುತ್ತದೆ, ಹಾಗು "ಕ್ರಮ" ವು "ಮನಸ್ಸಿಗೆ ಬಂದಂತೆ" ಆಗಿದ್ದಲ್ಲಿ ಥೀಮ್‌ಗಳ ಒಂದು ಪಟ್ಟಿಯನ್ನು ನೀಡುತ್ತದೆ.
