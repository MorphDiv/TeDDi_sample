מדור זה כולל את השאלות הכלליות ביותר על PHP: מה היא ומה היא עושה.
1. מה זה PHP?
מתוך ה מבוא למדריך:
PHP היא שפת תיכנות משובצת בשפת HTML. רוב צורת התחביר שלה מושאלת משפות C, Java ו-Perl בתוספת כמה מאפיינים יחודיים משלה. מטרת השפה היא לאפשר למפתחי אתרים באינטרנט לכתוב עמודי אינטרנט דינמיים במהירות.
הקדמה נחמדה ל-PHP מאת Stig Sזther Bakken ניתן למצוא כאן באתר האינטרנט של Zend. כמו-כן, רוב חומר הדיון ב-PHP ניתן בחינם.
2. מה PHP מסמלת?
PHP מסמלת PHP: Hypertext Preprocessor
3. מה הקשר בין הגירסאות?
PHP / FI 2.0 היא גירסה קודמת שלא נתמכת כיום של PHP.
PHP 3 היא יורש של PHP / FI 2.0 והיא לא ממש ידידותית.
PHP 4 היא הדור האחרון של PHP, שמשתמשת ב מנוע של Zend.
4. האם ניתן להריץ כמה גירסאות של PHP בעת ובעונה אחת?
כן, בדוק את קובץ INSTALL שנמצא בתוך הפצת המקור של PHP 4. בנוסף, קרא את ה נספח.
5. מה ההבדלים בין PHP 3 ו-PHP 4?
ישנם כמה מאמרים שכתובים בנושא מאת מאת המחברים של PHP 4. הנה רשימה של כמה מהמאפיינים החדשים החשובים יותר:
מודול API מורחב
תהליך בנייה מוכלל תחת UNIX
ממשק שרתים כללי שתומך גם בשרתי web מרובי-threads.
צובע תחביר משופר
תמיכה טבעית ב-HTTP sessions
תמיכה באגירת פלט
מערכת הגדרות חזקה יותר

6. מצאתי באג! למי עלי לפנות?
פנה לבסיס הנתונים של הבאגים של PHP וודא שהבאג עוד לא ידוע. אם אינך רואה את הבאג בבסיס הנתונים, השתמש בטופס הדיווח כדי לדווח על הבאג. חשוב להשתמש בבסיס הנתונים במקום פשוט לשלוח דואר לאחת מרשימות הדואר משום שהבאג יקבל מספר מעקב מוקצה שעל-ידיו תהיה לך אפשרות לבדוק שוב מאוחר יותר את מצב הבאג. בסיס הנתונים לבאגים נמצא ב http: / /bugs.php.net /.
