פתיחת מנהל הקבצים המסופי Midnight Commander
Whether opening a terminal on the desktop opens a terminal in the home directory
If set to true, then opening a terminal on the desktop will open a terminal in the home directory. Otherwise, it will be opened in the desktop directory. Note that this key is irrelevant if the desktop directory is identical to the home directory.
Whether a Midnight Commander menu item should be displayed
If set to true, and if the terminal file manager "Midnight Commander" is installed, then an "Open in Midnight Commander" menu item will be displayed in the context menu.
