Iconv library function converts files between various encoded character sets.
The supported character sets depend on iconv() implementation on your system.
Note that iconv() function in some system is not work well as you expect.
In this case, you should install libiconv library.
You must have iconv() function in standard C library or libiconv installed on your system. libiconv library is available from http: / /www.gnu.org / software / libiconv /.
To be able to use the functions defined in this module you must compile your PHP interpreter using the configure line --with-iconv.
This extension does not define any configuration directives.
This extension does not define any resource types.
This extension does not define any constants.
