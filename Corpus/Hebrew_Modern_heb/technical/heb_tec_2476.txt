ניהול פרופילי ICC
The default gamma to use for monitors with no previous value set.
Gamma is adjusted to affect the color of the attached monitor. Traditionally Linux has used a gamma value of 1.0, but this makes monitors look washed out compared Windows XP or OS X. Apple traditionally used a value of 1.8 for a long time, but now use the same value as Microsoft. Microsoft has always used a value of 2.2.
Whether the display should be globally corrected or left to applications.
If set to TRUE then the video LUT is set with the display profile
Whether ICC profiles should be loaded from external disks.
If set to TRUE then external disks are searched for device ICC profiles at startup, for instance looking in the OSX library folder or Windows XP system folders. This may increase startup time if disks need to be spun up to be searched.
Set the _ICC_PROFILE hint for applications.
If set to TRUE then the __ICC_PROFILE hint will be set, which applications use to convert true color to screen color.
Show the fine tuning controls for display devices.
If set to TRUE then the fine tuning controls are visible. This allows the user to change the display profile outside of what the ICC profile suggests and is only recommended for advanced users.
The rendering intent to use for the display, e.g. 'perceptual'.
The rendering intent to use for softproofs, e.g. 'perceptual'.
The default RGB profile.
The RGB profile filename to be used by applications by default.
The default CMYK profile.
The CMYK profile filename to be used by applications by default.
The default calibration length.
The length of calibration, as this is proportional to the accuracy. The option 'short' creates a quick profile, 'normal' a regular one, and 'long' takes a really long time, but is more precise and creates a better profile. Use 'ask' if you want the user to choose.
The duration between sending notifications to recalibrate a display.
This is the number of seconds in between notifying the user to recalibrate each display device. Set to 0 to disable the notification.
The duration between sending notifications to recalibrate a printer.
This is the number of seconds in between notifying the user to recalibrate each printer device. Set to 0 to disable the notification.
