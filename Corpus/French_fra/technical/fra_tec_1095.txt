Pour créer un document maître
Effectuez l'une des opérations suivantes :
Texte
.

Cela permet de répercuter dans les sous-documents tous les changements apportés à un style existant dans le document maître.
Navigateur

Insérer
, puis faites l'une des actions suivantes :
Fichier - Enregistrer
.
Pour éditer un document maître
Utilisez le Navigateur pour réorganiser et éditer les sous-documents dans un document maître.
Pour ouvrir et éditer un sous-document, double-cliquez sur le nom correspondant dans le Navigateur.
Supprimer
.
L'entrée est supprimée de la liste.
Toutefois, le fichier de sous-document correspondant est conservé.
Insérer - Texte
.
Une section de texte est insérée dans le document maître juste avant l'élément sélectionné.
Vous pouvez alors saisir du texte dans cette section.
Vous ne pouvez pas insérer de section de texte à côté d'une section de texte existante dans le Navigateur.
