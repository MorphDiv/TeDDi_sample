Naviguer avec Konqueror
Le module du navigateur Konqueror de &centreConfiguration; vous permet de sélectionner différentes options concernant l'apparence et le comportement de Konqueror, le navigateur web intégré de KDE.
Options pour gérer les liens et les images dans les pages HTML.
Options concernant les polices utilisées pour afficher les pages Web.
Options pour gérer le code Java inclus dans les pages web, ainsi que la stratégie en matière de sécurité à appliquer à un tel code.
Options pour gérer le code JavaScript inclus dans les pages web.
Modules externes
Les options concernant les modules externes
HTML
La première option que vous pouvez activer sur cette page est Activer le complètement des formulaires.
Si vous cochez cette case, Konqueror essayera de se souvenir de ce que vous répondez aux questions dans les formulaires, et essayera de remplir les formulaires avec les réponses données les fois précédentes.
Vous pouvez configurer le nombre d'éléments de formulaires dont Konqueror se souvient avec la glissière intitulée Nombre maximum de complètements
L'option suivante est Changer le curseur au-dessus des liens.
Si cette option est sélectionnée, la forme du pointeur change (habituellement sous la forme d'une main) à chaque fois qu'il survole un lien hypertexte.
Cela permet d'identifier aisément les liens, spécialement lorsqu'ils sont sous la forme d'images.
Le prochain paramètre est Souligner les liens.
Vous pouvez décider de Toujours souligner les liens.
Si cette option est sélectionnée, tout texte sur les pages web qui se comporte comme un lien sera montré dans une police soulignée.
Même si de nombreuses pages web utilisent la couleur pour mettre en évidence les textes qu'on peut utiliser comme des liens, souligner rend le repérage des liens très facile.
Beaucoup de pages web utilisent des images gif animées, et elles peuvent être très ennuyeuses dans certains cas en mobilisant vos ressources système.
L'option Animations vous permet de choisir si les animations sont activées.
La valeur par défaut est Activées, mais vous pouvez la positionner sur désactivées ou ne les lancer qu'une seule fois, même si le fichier lui-même contient des instructions de les relancer indéfiniment.
Apparence
Sous cet onglet, vous pouvez sélectionner de nombreuses options relatives à l'usage des polices.
Bien que la forme et la taille des polices fassent souvent partie de la conception d'une page web, vous pouvez choisir certains réglages par défaut que Konqueror utilisera.
La première chose que vous pouvez régler ici est la taille de la police.
Il y a deux réglages qui contribuent au confort de la navigation.
Premièrement, vous pouvez régler la Taille de police minimale.
Même si la taille de la police est spécifiée dans la page affichée, Konqueror ignorera cette instruction et n'utilisera jamais des polices plus petites que ce que vous indiquez ici.
Vous pouvez ensuite régler une Taille moyenne de police.
Il ne s'agit pas que de la taille par défaut du texte, utilisée lorsque la page n'indique pas de tailles, mais il s'agit également de la taille de base utilisée pour calculer les tailles de polices relatives.
Par exemple, l'instruction HTML smaller signifie qu'il faut utiliser une taille plus petite que ce que vous indiquez ici.
Pour toutes les options, vous pouvez choisir la taille de police exacte en points en utilisant le compteur rotatif (ou simplement en l'écrivant) à côté du nom de l'option.
Ces options sont indépendantes les unes des autres.
Les pages qui ne positionnent pas une police ou demandent la police par défaut s'afficheront avec le taille que vous avez fixée dans Taille de police moyenne, alors que les pages qui demandent une taille plus petite que votre Taille de police minimale afficheront cette taille à la place.
L'une n'affecte pas l'autre.
Les options restantes concernent les polices associées à différents types de balises utilisées dans les pages HTML.
Notez que de nombreuses pages web peuvent outrepasser ces réglages.
Si vous cliquez n'importe où sur un bouton qui affiche un nom de police, une liste de noms de polices apparaît, et vous pouvez choisir une police différente si vous le désirez. (S'il y a beaucoup de polices, une barre de défilement verticale apparaît sur la liste pour vous permettre de toutes les parcourir).
Vous pouvez choisir une police pour chaque type de balises, pour chaque jeu de caractères, en changeant le jeu de caractères dans la première liste déroulante, et en choisissant une police pour chaque catégorie au dessous.
Le faire complètement prendrait du temps, vous pouvez donc le limiter à votre jeu de caractères par défaut.
La plupart des utilisateurs français utiliseront iso8859-15.
Au-dessous, vous pouvez spécifier un Ajustement de la taille de la police pour cet encodage.
Parfois les polices que vous désirez utiliser pour un encodage ou une langue particulière sont plus grandes que la moyenne, vous pouvez donc utiliser ce paramètre pour leur donner une apparence compatible avec les autres polices.
Vous pouvez régler l'encodage par défaut.
Konqueror suppose que cet encodage est utilisé par les pages lorsqu'il effectue leur rendu à l'écran.
Le réglage par défaut est Utiliser l'encodage de la langue, mais vous pouvez le remplacer par n'importe quel encodage de la liste.
Java
Les boîtes à cocher groupées sous Réglages globaux vous permettent d'activer le support de Java pour tous les sites web par défaut.
Vous pouvez également choisir d'activer ou de désactiver Java sur certains sites.
Pour ajouter une stratégie concernant un hôte donné, cliquez sur le bouton Ajouter... pour afficher un dialogue dans lequel vous pouvez écrire le nom de l'hôte puis choisir d'accepter ou de refuser du code Java provenant de cet hôte particulier, ce qui ajoutera le domaine correspondant à la liste sur la gauche de la page.
Si vous cochez l'option Afficher la Console Java, Konqueror ouvrira une fenêtre de console dans laquelle les applications Java peuvent lire ou écrire du texte.
Bien que la plupart des applications Java ne nécessitent pas une telle console, elle peut s'avérer utile pour diagnostiquer des problèmes liés à des applications Java.
Vous pouvez être amené à choisir cette dernière méthode, par exemple, si vous avez de multiples installations de Java sur votre système, et si vous voulez indiquer laquelle vous souhaitez utiliser.
Si la machine virtuelle Java que vous utilisez nécessite des options de démarrage particulières, vous pouvez les écrire dans la zone de saisie libellée Arguments Java supplémentaires.
JavaScript
Malgré son nom, JavaScript n'est pas du tout relié à Java.
La première partie de cette page fonctionne de la même façon que la page des paramètres Java.
Les boîtes à cocher groupées sous Réglages globaux vous permettent d'activer le support de JavaScript pour tous les sites web par défaut.
Vous pouvez également choisir d'activer ou de désactiver Javascript sur certains sites.
Pour ajouter une stratégie concernant un hôte donné, cliquez sur le bouton Ajouter... pour afficher un dialogue dans lequel vous pouvez écrire le nom de l'hôte puis choisir d'accepter ou de refuser du code JavaScript provenant de cet hôte particulier, ce qui ajoutera le domaine à la liste à la gauche de la page.
Le dernier groupe d'options de cette page détermine ce qui se passe lorsqu'une page utilise JavaScript pour tenter d'ouvrir une nouvelle fenêtre.
Vous pouvez régler Konqueror pour Accepter toutes les requêtes de ce type, Demander à chaque fois qu'une requête est effectuée, ou Refuser toutes les requêtes d'ouverture.
