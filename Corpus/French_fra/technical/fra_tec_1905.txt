Fonction Join [Exécution]
Renvoie une chaîne de caractères à partir d'un certain nombre de sous-chaînes issues d'une matrice de chaînes de caractères.
Syntaxe :
Join (Text As String Array, delimiter)
Valeur de retour :
Chaîne de caractères
Paramètres :
Text :
matrice de chaînes de caractères.
delimiter (facultatif) :
caractère utilisé pour délimiter les sous-chaînes dans la chaîne de caractères obtenue.
Le séparateur par défaut est le caractère espace.
Si le séparateur est une chaîne de caractères de longueur zéro ( &quot; &quot; ), les sous-chaînes sont reliées sans séparateur.
