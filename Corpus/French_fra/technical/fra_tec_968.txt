Options d'export EPS
Définit plusieurs options d'export pour les fichier EPS.
Vous devez imprimer un fichier EPS à l'aide d'une imprimante PostScript.
Les autres imprimantes n'impriment que l'aperçu incorporé.
Aperçu
Définit les options d'aperçu incorporé pour le fichier EPS.
Aperçu de l'image (TIFF)
Spécifie si un aperçu de l'image est exporté au format TIFF avec le fichier PostScript.
Interchange (EPSI)
Spécifie si une image d'aperçu monochrome au format EPSI est exportée avec le fichier PostScript.
Ce format contient uniquement des caractères imprimables issus du code ASCII 7 bits.
Version
Détermine le niveau PostScript pour l'exportation du fichier EPS.
Niveau 1
Niveau 1
si l'imprimante PostScript ne possède pas les fonctionnalités du niveau 2.
Niveau 2
Niveau 2
si le périphérique de sortie supporte les bitmaps en couleurs, les formats intégrant une palette et les images compressées.
