La boîte de dialogue de configuration de & kspread;
La sélection de Configuration Configurer KSpread... ouvre une boîte de dialogue avec plusieurs pages, sélectionnables par les icônes à gauche, qui permettent de changer plusieurs aspects des opérations de & kspread;.
Paramètres locaux
Paramètres locaux
Cette page de la boîte de dialogue de configuration de & kspread; affiche comment les éléments comme des nombres, dates, heures et monnaies sont affichés.
Si vous avez chargé un tableur qui a été généré en utilisant une variable locale différente, en actionnant le bouton Actualiser vers le système locale sur cette page, elle s'ajustera pour se conformer à vos réglages de locale.
Interface

Nombre de pages ouvertes au début & #160;:

Auto-enregistrement (min) & #160;:

Afficher les en-têtes de lignes

Divers
La page Divers de la boîte de dialogue de configuration de & kspread; contient les éléments suivants & #160;:
Mode de complètement & #160;:
Vous permet de choisir le mode d' (auto-)complètement parmi une liste d'options dans la boîte de sélection déroulante.
Actionnez Entrée pour déplacer la sélection vers & #160;:
Quand vous avez sélectionné une cellule puis actionné la touche Entrée, la sélection se déplacera d'une place vers la gauche, la droite, le haut ou le bas comme déterminé par le réglage dans la boîte de sélection déroulante.

Afficher le message d' erreur

Couleur

Vérification d' orthographe
Cette page vous permet de configurer le comportement du vérificateur d'orthographe de & kspread;.
Créer les combinaisons Radical/ Affixe absentes du dictionnaire
Si cette case est cochée, lorsque & kspread; trouve un mot dans le document qu'il ne reconnaît pas, mais consiste en une racine et un préfixe ou suffixe connus, il l'acceptera, alors que si cette case n'est pas cochée, il le rejettera.
Considérer les mots accolés comme des erreurs d' orthographe
Si cette case est cochée, les mots usuels collés ensemble seront considérés comme des erreurs, par exemple jepeux.
