Outil externe de modification
Intitulé & #160;:
Ce nom sera affiché dans le menu « & #160; Outils- > Outils externes & #160; »
Script & #160;:
Le script à lancer pour exécuter l'outil. Ce script est passé à / bin/ sh pour l'exécution. Les macros suivantes seront développées & #160;: %URL - l'URL du document actuel. %URLs - une liste des URL de tous les documents ouverts. %directory - l'URL du dossier contenant le document actuel. %filename - le nom de fichier du document actuel. %line - la ligne en cours du curseur de texte dans la vue actuelle. %column - la colonne du curseur de texte dans la vue actuelle. %selection - le texte sélectionné dans la vue actuelle. %text - le texte du document actuel.
& Exécutable & #160;:
L'exécutable utilisé par la commande, pour vérifier si un outil devra être affiché. S'il n'est pas défini, le premier mot de la commande sera employé.
Types & MIME & #160;:
Une liste de types MIME, séparés par un point-virgule, pour lesquels cet outil devra être disponible. Si ce champ reste vide, l'outil est toujours disponible. Pour faire votre choix parmi les types MIME connus, cliquez sur le bouton situé à droite.
Cliquez pour obtenir une boîte de dialogue susceptible de vous aider à créer une liste de types MIME.
& Enregistrer & #160;:

Document actuel
Tous les documents
Vous pouvez choisir d'enregistrer le document actuel ou tous les documents [modifiés] avant de lancer la commande. Cette fonction est utile si vous voulez passer des URL à une application comme un client FTP, par exemple.
Nom de la ligne de & commande & #160;:
Si vous indiquez ici un nom, vous pouvez exécuter la commande sur la la ligne de commande de la vue, avec « & #160; outil-externe-le_nom_que_vous_indiquez_ici & #160; ». Veuillez ne pas utiliser d'espaces ou de tabulations dans le nom.
