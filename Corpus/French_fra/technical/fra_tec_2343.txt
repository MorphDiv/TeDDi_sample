Styles
Quand vous élaborez des documents plus complexes qu'une lettre d'affaires, comme un livre, un journal ou autres, le document est souvent divisé en sections. chacune de ces sections peut avoir une apparence semblable.
Vous pouvez utiliser les styles pour conserver facilement une apparence homogène tout au long de votre document.
Si vous êtes familier avec les Styles dans Microsoft Word, les styles de KWord ont les mêmes les fonctions.
Vous pouvez alors sauter cette section.
Grâce à une méthode révolutionnaire, nous pouvons maintenant vous apprendre à utiliser votre ordinateur en 10 minutes.
Nous allons d'abord énumérer les parties de votre nouvel ordinateur.
Dans cet exemple, vous pouvez voir que ce manuel pour ordinateurs consiste en deux sections.
Pour le texte du titre de chaque section, nous avons changé le style de texte en Titre 1.
Le reste du texte du document est laissé comme Standard.
Une fois que nous avons fini le document, nous pouvons changer l'aspect de tout le texte libellé Titre 1 d'un coup.
KWord propose 8 styles prédéfinis.

Supprimer un style
La suppression d'un style est facile.
Ceci ouvrira une boîte de dialogue
Dans la liste des styles disponibles, sélectionnez le style que vous voulez supprimer en cliquant une fois avec le bouton gauche de la souris.
Assurez -vous que vous avez sélectionné le style à supprimer avant de cliquer sur Supprimer.
KWord ne vous demandera pas de confirmation, et vous n'aurez donc pas de possibilité de revenir.
Cliquez sur Supprimer.
Le style est maintenant supprimé.
Certains styles de KWord sont considérés comme indispensables (Standard, Titre 1, Titre 2, Titre 3, Liste numérotée, Liste alphabétique, Liste à puces) et ne peuvent pas être supprimés.
Quand vous sélectionnez n'importe lequel de ces styles pour le suppimer, le bouton Supprimer ne fonctionnera pas.

Contrôle des styles
Avec certains documents, vous voudrez vous assurer que chaque caractère dans votre document correspond exactement au style du document.
Avec d'autres documents, vous aurez besoin de plus de souplesse.
Dans ce deuxième cas, vous avez besoin de configurer KWord de telle manière qu'il ne change que certains aspects spécifiques du style avec chaque mise à jour.
Ceci ouvrira une boîte de dialogue
Maintenant, cliquez sur l'onglet libellé Update Configuration.
Cette boîte de dialogue est séparée en 7 parties principales plus une case à cocher.
Polices
Ne pas mettre à jour les polices
Si cette option est sélectionnée, KWord ne changera pas les polices (en respectant la taille, leur nom, l'italique, le gras, etc.) en appliquant le style.
Mettre à jour les familles de polices de même taille
Lorsque sélectionné, KWord ne changera pas les polices en appliquant le style, à moins que l'attribut de taille de police du texte sélectionné ne corresponde à la taille de police contenue dans le style.
Si les tailles des polices ne correspondent pas, KWord ne changera pas les attributs de polices.
Si les tailles correspondent, le texte sélectionné sera changé dans la famille de polices du style.
Aucun autre changement (gras, italique, etc.) ne sera fait.
Actualiser les familles de police de toutes les Polices de taille
Avec cette option, KWord convertira la famille de polices (mais aucun autre attribut) dans la police définie dans le style sans tenir compte de la taille des polices.
Mettre à jour les familles de polices et les attributs des polices de même taille
Lorsque sélectionné, KWord ne changera pas les polices en appliquant le style, à moins que l'attribut de taille de police du texte sélectionné ne corresponde à la taille de police contenue dans le style.
Si les tailles des polices ne correspondent pas, KWord ne changera pas les attributs de polices.
Si les tailles correspondent, le texte sélectionné sera changé pour correspondre au style.
Tous les autres attributs (gras, italique, etc.) seront changés.
Actualiser les familles de police et les attributs de toutes les polices de taille
Avec cette option, KWord changera tout le texte, sans tenir compte de la taille ou des attributs, pour correspondre aux polices spécifiées dans le style.
Couleur de la police
Cette boîte déroulante vous permet de spécifier si la couleur de police doit être changée pour correspondre au style ou non.
Ne pas actualiser les couleurs
Si cette option est sélectionnée, KWord ne changera pas les couleurs des polices.
Actualiser les couleurs
Lorsque sélectionnée, KWord changera la couleur du texte sélectionné pour correspondre à la couleur définie dans le style de police.
Indentations et espacements
Ne pas mettre à jour les indentations et espacements
Si cette option est sélectionnée, KWord ne changera pas les réglages d'indentations ni d'espacements.
Actualiser les couleurs
Lorsque sélectionné, KWord changera les réglages d'indentation ou d'espacement de paragraphe du texte sélectionné pour correspondre à ceux définis dans le style de polices.
Alignements/Flux
Cette boîte déroulante vous permet de spécifier si l'alignement des paragraphes doit être modifié pour correspondre au style ou non.
Ne pas aligner les paragraphes
Si cette option est sélectionnée, KWord ne changera pas les réglages d'alignement des paragraphes.
Actualiser les alignement
Si cette option est sélectionnée, KWord changera les réglages d'alignement des paragraphes du texte sélectionné pour correspondre à ceux définis dans le style de police.
Numérotation
Ne pas actualiser la numérotation
Si cette option est sélectionnée, KWord ne modifiera pas les réglages d'auto-numérotation.
Actualiser les couleurs
Si cette option est sélectionnée, KWord modifiera les réglages d'auto-numérotation du texte sélectionné pour correspondre à ceux définis dans le style de police.
Bords
Ne pas actualiser les bordures
Si cette option est sélectionnée, KWord ne modifiera pas les bordures des paragraphes.
Actualiser les bordures
Si cette option est sélectionnée, KWord modifiera les bordures des paragraphes pour correspondre à celles définies dans le style de police.
Tabulations
Ne pas actualiser les tabulations
Si cette option est sélectionnée, KWord ne modifiera pas les points de tabulation.
Actualiser les tabulations
lSi cette option est sélectionnée, KWord modifiera les points de tabulation pour correspondre au style des polices.
Quand vous avez fini, cliquez sur Fermer pour appliquer vos changements.
