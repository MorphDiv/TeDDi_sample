Tables dans les bases de données;ajout aux requêtes
Ajouter des tables
Indique les tables à insérer dans la fenêtre d'ébauche.
Dans la boîte de dialogue
Ajout de tables
, sélectionnez les tables nécessaires pour exécuter la tâche en cours.
Lors de la création d'une requête ou d'une nouvelle présentation de table, sélectionnez la table à laquelle la requête ou la présentation doit se référer.
Lors de l'utilisation de bases de données relationnelles, sélectionnez les tables entre lesquelles vous souhaitez établir des relations.
Dans les fenêtres d'ébauche de requête ou relationnelles, les tables insérées, ainsi que la liste de leurs champs, s'affichent dans une fenêtre séparée.
Vous pouvez définir la taille et la disposition de cette fenêtre.
Table
Affiche uniquement les tables.
Affiche uniquement les requêtes.
Nom de la table
Répertorie toutes les tables disponibles.
Pour insérer une table, sélectionnez-la dans la liste et cliquez sur
Ajouter
.
Vous pouvez également double-cliquer sur le nom de la table afin d'ouvrir une fenêtre relationnelle ou d'ébauche de requête, en haut de laquelle sont affichés tous les champs de la table sélectionnée.
