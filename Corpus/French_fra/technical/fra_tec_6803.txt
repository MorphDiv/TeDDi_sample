La commande
Fermer une image ne peut pas être annulé : une fois qu'elle est fermée tout est perdu y compris l'historique d'annulation. Si l'image a subi des modifications une confirmation vous sera demandée pour la fermer. Notez que l'image est considérée comme non modifiée dès qu'elle a été enregistrée dans un fichier même si le format choisi ne permettait pas de conserver toutes les informations possibles. Si vous souhaitez n'avoir aucun regret, enregistrez le fichier (automatiquement dans le format natif de GIMP
