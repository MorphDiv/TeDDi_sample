Vérifier les mises à jour

Gestionnaire des extensions
pour vérifier les mises à jour en ligne de toutes les extensions installées.
Pour vérifier les mises à jour d'une seule extension sélectionnée, faites un clic avec le bouton droit pour ouvrir le menu contextuel, puis choisissez
Mise à jour
.
Lorsque vous cliquez sur le bouton Vérifier les mises à jour ou choisissez la commande Mise à jour, la boîte de dialogue Mise à jour d'extension est affichée et la vérification de la disponibilité des mises à jour démarre immédiatement.
Lors de la vérification des mises à jour, vous voyez un indicateur de progression.
Attendez que des messages s'affichent dans la boîte de dialogue, ou cliquez sur Annuler pour abandonner la vérification des mises à jour.
Si aucune mise à jour n'est disponible, le message dans la boîte de dialogue vous indique qu'il n'y pas de mises à jour.
Fermez la boîte de dialogue.
Si des mises à jour sont disponibles, celles-ci peuvent être installées automatiquement ou vous devez répondre à quelques actions :
La boîte de dialogue Mise à jour des extensions peut contenir des entrées qui ne peuvent pas être sélectionnées et donc aucune mise à jour automatique ne peut être réalisée.
Les dépendances ne sont pas satisfaites (la mise à jour nécessite des fichiers supplémentaires ou plus récents pour être installée).

pour de plus amples détails.
Une mise à jour manuelle est nécessaire.
Lorsque vous cliquez sur le bouton Installer, la boîte de dialogue Téléchargement et installation est affichée.
Toutes les extensions qui peuvent être directement téléchargées, le sont maintenant.
La progression est affichée dans la boîte de dialogue Téléchargement et installation.
Si une extension n'a pas pu être téléchargée, un message est affiché.
L'opération continue pour les extensions restantes.
Certaines extensions peuvent être marquées avec la phrase &quot; mise à jour basée sur le navigateur &quot; .
Ces extensions ne peuvent pas être téléchargées par le Gestionnaire des extensions.
Un navigateur Web doit être ouvert pour télécharger la mise à jour de l'extension à partir d'un site Web particulier.
Ce site peut demander plusieurs interactions de l'utilisateur pour télécharger l'extension.
Après le téléchargement, vous devez installer l'extension manuellement, par exemple, en double-cliquant sur l'icône de l'extension dans un navigateur de fichier.
Pour les extensions marquées comme &quot; mise à jour basée sur le navigateur &quot; , le Gestionnaire des extensions s'ouvrira dans le navigateur sur le site Web correspondant.
Cela arrive lorsque vous fermez le boîte de dialogue, après avoir téléchargé n'importe quelles autres mises à jour d'extensions.
S'il n'y a pas d'extension qui puisse être directement téléchargée, alors le navigateur Web est immédiatement lancé.
Après que la dernière extension ait été téléchargée, l'installation commence.
Tout d'abord, toutes les extensions installées pour lesquelles une mise à jour a pu être correctement téléchargée sont supprimées.
Puis, les extensions mises à jour sont installées.
Si une erreur survient, un message indiquant que l'installation a échoué est affiché, mais l'opération continue.
