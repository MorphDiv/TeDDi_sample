IsObject, fonction
Fonction IsObject [Exécution]
Détermine si une variable d'objet est un objet OLE.
Si c'est le cas, la fonction renvoie la valeur True ; dans le cas contraire, elle renvoie la valeur False.
Syntaxe :
IsObject (VarObjet)
Valeur de retour :
Logique (Bool)
Paramètres :
ObjectVar :
toute variable à tester.
Si la variable d'objet contient un objet OLE, la fonction renvoie la valeur True.
