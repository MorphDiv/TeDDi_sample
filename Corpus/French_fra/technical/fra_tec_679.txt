Exporter en PDF
Enregistre le fichier actif au Portable Document Format (PDF) version 1.4.
Un fichier PDF peut être affiché et imprimé sur n'importe quelle plateforme en conservant le formatage original intact, à condition que le logiciel de prise en charge soit installé.
Général
Pages
Définit les options d'export des pages incluses dans le fichier PDF.
Tout
Exporte toutes les plages d'impression définies.
Si aucune plage d'impression n'est définie, le document entier est exporté.
Plage
Exporte les pages que vous saisissez dans le champ.
Pour exporter une plage de pages, utilisez le format 3-6.
Pour exporter des pages uniques, utilisez le format 7;9;11.
Si vous le voulez, vous pouvez exporter une combinaison de plages et de pages uniques en utilisant un format du type 3-6;8;10;12.
Sélection
Exporte la sélection active.
Images
Définit les options d'export PDF pour les images à l'intérieur du document.
Les images EPS qui ont des aperçus incorporés sont exportées uniquement comme aperçus.
Les images EPS qui n'ont pas d'aperçus incorporés sont exportées comme des substituants vides.
Compression sans perte
Sélectionne une compression sans perte des images.
Tous les pixels sont préservés.
Compression JPEG
Sélectionne une compression JPEG des images.
Pour un haut niveau de qualité, presque tous les pixels sont préservés.
Pour un bas niveau de qualité, certains pixels sont perdus et des artifacts sont introduits, mais la taille du fichier est réduite.
Qualité
Saisissez le niveau de qualité d'une compression JPEG.
Réduire la résolution des images
Sélectionne pour échantillonner de nouveau ou réduire la taille des images à un nombre inférieur de pixels par pouce.
Sélectionnez la résolution cible des images.
Général
Définit les options d'export PDF générales
PDF/A-1a
Convertit au format PDF/A-1a.
Ce format est défini comme un format de fichier de document électronique pour la préservation à long terme.
Toutes les polices qui ont été utilisées dans le document source seront incorporées dans le fichier PDF généré.
Les balises PDF seront écrites.
PDF marqué
Sélectionne pour écrire des marques PDF.
Cela peut augmenter la taille du fichier de façon importante.
Le PDF marqué contient des informations sur la structure du contenu du document.
Cela peut aider à l'affichage du document sur des périphériques ayant des écrans différents et lors de l'utilisation de logiciel de lecture d'écran.
Exporter les notes
Sélectionne pour exporter les commentaires des documents Writer et Calc comme notes PDF.
Exporter les repères de texte
Sélectionnez pour exporter tous les repères de texte d'un document Writer comme repères de texte PDF.
Les repères de texte sont créés pour tous les paragraphes numérotés (Outils - Numérotation des chapitres) et pour toutes les entrées de tables des matières pour lesquelles vous avez assigné des hyperliens dans le document source.
Créer un formulaire PDF
Sélectionnez pour créer un formulaire PDF.
Il peut être rempli et imprimé par l'utilisateur du document PDF.
Format d'envoi
Sélectionnez le format de soumission des formulaires à l'intérieur d'un fichier PDF.
Sélectionnez le format des données que vous recevrez de l'envoyeur :
FDF (Forms Data Format), PDF, HTML ou XML.
Ce paramétrage écrase les propriétés du contrôle d'URL que vous avez définies dans le document.
Exporter les pages blanches insérées automatiquement
Si activé, les pages blanches insérées automatiquement sont exportées dans le fichier PDF.
C'est indiqué si vous imprimez le PDF recto/verso.
Par exemple : dans un livre, un style de paragraphe de chapitre est défini pour toujours commencer sur une page numérotée impaire.
Le chapitre précédent finit sur une page impaire. %PRODUCTNAME insère une page blanche numérotée de gauche.
Cette option contrôle si cette page de gauche numérotée doit être importée ou non.
Créer un fichier hybride
Sun PDF Import
est installée, ce paramètre permet d'exporter le document comme fichier . pdf contenant deux formats de fichier :
PDF et ODF.
Vue initiale
Volets
Page uniquement
Sélectionnez pour générer un fichier PDF qui affiche seulement le contenu des pages.
Repères de texte et page
Sélectionnez pour générer un fichier PDF qui affiche une palette des repères de texte et le contenu des pages.
Miniatures et page
Sélectionnez pour générer un fichier PDF qui affiche une palette de miniatures et le contenu de la page.
Ouvrir à la page
Sélectionnez pour afficher la page donnée lorsque le lecteur ouvre le fichier PDF.
Agrandissement
Par défaut
Sélectionnez pour générer un fichier PDF qui affiche le contenu des pages sans facteur de zoom.
Si le logiciel de lecture est configuré pour utiliser un facteur de zoom par défaut, la page s'affiche avec ce facteur de zoom.
Adapter à la fenêtre
Sélectionnez pour générer un fichier PDF qui affiche la page agrandie pour s'adapter entièrement à la fenêtre du lecteur.
Adapter la largeur
Sélectionnez pour générer un fichier PDF qui affiche la page agrandie pour s'adapter à la largeur de la fenêtre du lecteur.
Adapter au visible
Sélectionnez pour générer un fichier PDF qui affiche le texte et les images sur la page agrandie pour correspondre à la largeur de la fenêtre du lecteur.
Facteur de zoom
Sélectionnez un facteur de zoom donné lorsque le lecteur ouvre le fichier PDF.
Mise en page
Par défaut
Sélectionnez pour générer un fichier PDF qui affiche les pages en fonction du paramètre de mise en page du logiciel de lecture.
Une page
Sélectionnez pour générer un PDF qui n'affiche qu'une page à la fois.
Continu
Sélectionnez pour générer un fichier PDF qui affiche les pages en une colonne verticale continue.
Orientation continue
Sélectionnez pour générer un fichier PDF qui affiche les pages l'une à côté de l'autre en une colonne continue.
Lorsqu'il y a plus de deux pages, la première page est affichée à droite.
Première page à gauche
Sélectionnez pour générer un fichier PDF qui affiche les pages côte à côte en une colonne continue.
Lorsqu'il y a plus de deux pages, la première page est affichée à gauche.
Vous devez activer la prise en charge des scripts complexes sous Outils - Options - Paramètres linguistiques - Langues.
Interface utilisateur
Options de fenêtre
Redimensionner la fenêtre selon la page initiale
Sélectionnez pour générer un fichier PDF qui est affiché dans une fenêtre affichant la page initiale complète.
Centrer la fenêtre sur l'écran
Sélectionnez pour générer un fichier PDF qui est affiché centré à l'écran dans la fenêtre du lecteur
Ouvrir en mode plein écran
Sélectionnez pour générer un fichier PDF qui est affiché dans une fenêtre de lecteur plein écran au-dessus de toutes les autres fenêtres.
Afficher le titre du document
Sélectionnez pour générer un fichier PDF qui est affiché avec le titre du document dans la barre de titre du lecteur.
Options d'interface utilisateur
Masquer la barre de menu
Sélectionnez pour masquer la barre de menu du lecteur lorsque le document est actif.
Masquer la barre d'outils
Sélectionnez pour masquer la barre d'outils du lecteur lorsque le document est actif.
Masquer les contrôles de la fenêtre
Sélectionnez pour masquer les contrôles du lecteur lorsque le document est actif.
Transitions
Utiliser les effets de transition
Sélectionne pour exporter les effets de transition de diapos Impress en effets PDF correspondants.
Repères de texte
Tous les niveaux de repères de texte
Sélectionner pour afficher tous les niveaux de repères de texte lorsque le lecteur ouvre le fichier PDF.
Niveaux de repères de texte visibles
Sélectionnez pour afficher les niveaux de repères de texte jusqu'au niveau sélectionné lorsque le lecteur ouvre le fichier PDF.
Liens
Spécifiez comment exporter les repères de texte et les hyperliens dans le document.
Exporter les repères de texte comme destinations nommées
Les repères de texte (cibles des références) dans un fichier PDF peuvent être définis comme des zones rectangulaires.
De plus, les repères de texte à des objets nommés peuvent être définis par leurs noms.
Activez la case à cocher pour exporter les noms des objets du document comme des cibles valides de repères de texte.
Cela permet de lier ces objets par leur nom à d'autres documents.
Convertir les références de document en cibles PDF
Activez cette case à cocher pour convertir les URL référençant d'autres fichiers ODF à des fichiers PDF ayant le même nom.
Dans les URL de référencement, les extensions .odt, .odp, .ods, .odg et .odm sont converties en extension .pdf.
Exporter les URL relatifs au système de fichiers
&quot; hyperliens relatifs &quot;
dans l'aide.
Liens entre documents
Spécifiez comment gérer les liens hypertexte du PDF vers les autres fichiers.
Mode par défaut
Les liens du document PDF vers d'autres documents seront gérés comme spécifié par votre système d'exploitation.
Ouvrir avec un lecteur de PDF
Les liens entre documents sont ouverts avec l'application de lecteur PDF qui affiche actuellement le document.
L'application de lecteur PDF doit être capable de gérer les types de fichiers spécifiés à l'intérieur de l'hyperlien.
Ouvrir avec un navigateur Internet
Les liens entre documents sont ouverts avec le navigateur Internet.
Le navigateur Internet doit être capable de gérer le type de fichier spécifié à l'intérieur du lien hypertexte.
Sécurité
Chiffrer le document PDF
Sélectionnez pour chiffrer le document PDF avec un mot de passe.
Le fichier PDF ne peut être ouvert que lorsque l'utilisateur saisit le mot de passe correct.
Déf. mot de passe d'ouverture
Cliquez pour ouvrir une boîte de dialogue dans laquelle vous saisissez un mot de passe.
Restreindre les permissions
Sélectionnez pour restreindre certaines permissions.
Les permission ne peuvent être modifiées que si l'utilisateur saisit le mot de passe correct.
Définir le mot de passe d'autorisation
Cliquez pour ouvrir une boîte de dialogue dans laquelle vous saisissez le mot de passe.
Impression
Non autorisé
L'impression du document n'est pas permise.
Faible résolution (150 dpi)
Le document ne peut être imprimé qu'en basse résolution (150 dpi).
Tous les lecteurs PDF ne prennent pas en charge ce paramétrage.
Haute résolution
Le document peut être imprimé en haute résolution.
Modifications
Non autorisé
Aucune modification du contenu n'est permise.
Insertion, suppression et rotation de pages
Seule l'insertion, la suppression et la rotation des pages est permise.
Remplissage dans les champs de formulaire
Uniquement le remplissage dans les champs du formulaire est permis.
Commentaires, remplissage dans les champs de formulaire
Uniquement les commentaires et le remplissage des champs de formulaire est permis.
Tout sauf les pages d'extraction
Toutes les modifications sont permises, excepté l'extraction des pages.
Autoriser la copie de contenu
Sélectionnez pour permettre la copie du contenu vers le presse-papiers.
Autoriser l'accès au texte des outils d'accessibilité
Sélectionnez pour permettre l'accès au texte des outils d'accessibilité.
Exporter
Exporte le fichier actif au format PDF.
