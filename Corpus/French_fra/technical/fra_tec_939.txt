Basculement entre le mode insertion et le mode écrasement
À l'aide du clavier :
Pour être activé, le curseur doit être placé dans la cellule ou dans la ligne de saisie.
À l'aide de la souris :
Dans la barre d'état, cliquez sur la zone indiquant le mode actif afin de basculer dans l'autre mode :
INS
Le curseur de texte a la forme d'une ligne verticale clignotante.
Cliquez sur la zone pour activer le mode Écrasement.
RFP
Le curseur de texte a la forme d'un bloc clignotant.
Cliquez sur cette zone pour activer le mode Insertion.
