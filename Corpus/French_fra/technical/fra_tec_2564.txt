Introduction
KPaint est un programme de dessin point par point simple qui peut être également utilisé pour retoucher des dichiers image existants et pour convertir d'un format d'image à l'autre.
Utilisation de KPaint
Quand vous lancez KPaint, une feuille blanche apparaît.
On dessine des lignes ou des formes en choisissant le type de ligne ou de forme désiré dans les icônes de barre d'outils ou le menu Outils puis en utilisant le bouton gauche de la souris ou le bouton droit de la souris pour effectuer le dessin.
Les couleurs utilisées par le boutons gauche et droit de la souris peuvent être modifiées en cliquant avec le bouton gauche de la souris sur les petits boutons affichés dans la barre d'outils se trouvant sur la droite (celle où se trouve un aperçu réduit de l'image).
Vous pouvez sélectionner une partie de l'image en choisissant l'outil Sélection d'une zone puis en délimitant cette zone en maintenant le bouton gauche de la souris enfoncé.
Une fois que c'est fait, vous pouvez Copier ou Couper la zone, puis la Coller n'importe où d'autre.
Entrez le nouveau nom de fichier dans le champ de texte Emplacement: et appuyez sur le bouton Enregistrer.
Le nom de fichier que vous saisissez doit comprendre une extension valide permettant de définir le format du fichier enregistré.

Le menu Fichiers
Commence une nouvelle image.
Si une image est déjà affichée, une nouvelle instance de KPaint est ouverte pour la nouvelle image.

Si une image est déjà affichée, une nouvelle instance de KPaint est ouverte pour la nouvelle image.
Enregistre une image modifiée sans changer son nom ni son format.
Si l'image n'a jamais encore été enregistrée, KPaint fera apparaître la boîte de dialogue Enregistrer sous pour vous permettre de saisir le nom et l'extension de fichier convenables.
Fichier Enregistrer sous...

Fichier Recharger

Ferme l'image en cours sans sortir de KPaint.

Installation
KPaint fait partie du projet d'environnement de bureau K (KDE), au sujet duquel vous pouvez trouver des détails à l'adresse http ://www.kde.org.
Pour voir si une version plus récente de KPaint est sortie, vous pouvez jeter un œ il à http ://apps.kde.com.
La plupart des distributions comprennent KPaint, mais si vous voulez le reconstruire vous-même, le code source se trouve dans le paquetage kdegraphics à l'adresse ftp ://ftp.kde.org/pub/kde/, le site FTP principal du projet KDE.
Comme KPaint utilise autoconf et automake vous ne devriez pas avoir de problèmes à la compiler.
Si vous deviez quand même rencontrer des problèmes, veuillez les signaler sur les listes de discussion de KDE.
