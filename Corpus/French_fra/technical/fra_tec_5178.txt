Introduction
& ark; est une application destinée à gérer divers formats d'archive dans l'environnement & kde;. Les archives peuvent être affichées, extraites, créées et modifiées depuis & ark;. Le programme est compatible avec plusieurs formats comme tar, gzip, bzip2, zip, rar et lha (si les programmes en ligne de commande appropriés sont installés).
Utilisation de & ark;
Ouvrir des archives
Pour ouvrir une archive dans & ark;, choisissez Ouvrir... dans le menu Fichier. Vous pouvez aussi ouvrir une archive par glisser-déposer depuis & konqueror; ou & dolphin;. Les fichiers d'archive devraient être associés à & ark;, donc vous pouvez aussi cliquer avec le bouton droit de la souris sur un fichier dans & konqueror; ou & dolphin; et sélectionner Ouvrir avec & ark; pour l'ouvrir ou choisir une option d'extraction pour ce fichier.
Travailler avec les fichiers
Une fois qu'une archive a été ouverte, vous pouvez effectuer plusieurs types d'opérations sur les fichiers qu'elle contient. En sélectionnant un fichier et en utilisant le menu Action, vous pouvez choisir ce que vous voulez faire & #160;:
Ajouter des fichiers... ajoutera à l'archive des fichiers provenant de votre disque.
Ajouter un dossier... ajoutera à l'archive un dossier provenant de votre disque.
Supprimer enlèvera à l'archive le ou les fichiers choisis.
Extraire... extraira tous les fichiers à l'emplacement que vous spécifierez sur le disque.
Aperçu ouvrira le fichier dans le programme d'affichage d'aperçu associé.
Extraire les archives
Une fois une archive ouverte dans & ark;, vous pouvez l'extraire. Pour extraire rapidement des fichiers d'une archive, vous pouvez sélectionner Extraire... dans le menu Action. Ceci ouvre un sous-menu présentant les dossiers récemment accédés, dans lesquels vous pouvez choisir d'effectuer rapidement l'extraction. La même action est possible en cliquant sur la coche située à droite du bouton Extraire situé dans la barre d'actions.
Afin d'ouvrir la boîte de dialogue Extraire, cliquez sur le bouton Extraire dans la barre d'outils ou utilisez le raccourci clavier Ctrl; E. Cette boîte de dialogue vous permet de choisir l'emplacement où extraire des fichiers. L'emplacement par défaut est le dossier contenant l'archive.
Vous pouvez spécifier d'extraire les fichiers dans un sous-dossier. Le nom par défaut de ce sous-dossier est la première partie du nom de l'archive, mais vous pouvez le modifier à votre guise. Si vous souhaitez préserver les emplacements lors de l'extraction, sélectionnez cette option. Vous pouvez également choisir d'ouvrir le dossier de destination de l'extraction dans & konqueror; ou & dolphin; une fois celle -ci terminée.
Si un fichier contenu dans l'archive est sélectionné, vous pouvez aussi sélectionner quels fichiers extraire & #160;:
Les fichiers sélectionnés seulement extrait uniquement les fichiers qui ont été choisis.
Tous les fichiers extrait tout le contenu de l'archive.
Création d'une archive et ajout de fichiers
Pour créer une archive dans & ark;, choisissez Nouveau dans le menu Fichier.
Vous pouvez alors saisir le nom de l'archive, avec l'extension appropriée (tar. gz, zip, bz2, & etc;) ou sélectionner un format pris en charge dans la liste déroulante Filtre et cocher l'option Sélectionner automatiquement l'extension de fichier. Pour ajouter des fichiers à l'archive, choisissez Ajouter des fichiers... dans le menu Action. Si vous voulez ajouter un dossier entier à une archive, sélectionnez Ajouter un dossier... dans le menu Action.
Une autre façon de procéder pour ajouter des fichiers à l'archive consiste à les glisser-déposer depuis & konqueror;, & dolphin; ou le bureau dans la fenêtre principale de & ark;, et ils seront ajoutés à l'archive actuelle.
Liste des commandes
Le menu Fichier
Ctrl; N Fichier Nouveau
Crée une nouvelle archive.
Ctrl; O Fichier Ouvrir...
Ouvre une archive.
Fichier Récemment ouvert(s)
Affiche une liste des archives récentes, pour en ouvrir une.
Ctrl; Q Fichier Quitter
Quitter & ark;.
Le menu Action
Action Ajouter des fichiers...
Affiche la boîte de dialogue Ajouter des fichiers... dans laquelle vous pouvez choisir des fichiers. Maintenez la touche & Ctrl; pressée et cliquez afin de choisir plusieurs fichiers.
Action Ajouter un dossier...
Affiche la boîte de dialogue Ajouter un dossier.... Choisissez simplement un dossier et cliquez sur le bouton Ok.
Action Supprimer
Enlève de l'archive les fichiers actuellement sélectionnés.
Ctrl; E Action Extraire... Extraction rapide vers...
Ceci ouvre un sous-menu présentant les dossiers récemment accédés, dans lesquels vous pouvez choisir d'effectuer rapidement l'extraction.
Action Aperçu
Ouvre le fichier sélectionné dans le programme d'affichage d'aperçu associé.
Le menu Configuration
Configuration Afficher & #160; la barre d' état
Affiche ou non la barre d'état.
Configuration Configurer les raccourcis clavier...
Affiche la boîte de dialogue Configurer les raccourcis. Ceci vous permet de choisir les touches de raccourcis clavier pour plusieurs éléments des menus. Pour modifier un raccourci, sélectionnez une action dans la liste, puis choisissez la combinaison de touches à utiliser. En cliquant sur le bouton pour touche de clavier qui se situe à droite, vous pouvez choisir la touche spécifique à utiliser. Appuyez simplement sur la nouvelle touche.
Configuration Configurer la barre d'outils...
Affiche une boîte de dialogue standard de & kde; où il vous est possible de configurer les icônes de la barre d'outils.

Prérequis
Pour utiliser & ark;, vous avez besoin de & kde; 4.1. & GNU; tar et une version récente de gzip sont aussi nécessaires si vous voulez permettre à & ark; de manipuler des archives tar. Pour manipuler d'autres formats d'archive, vous avez besoin des programmes en ligne de commande appropriés, comme par exemple zip, unzip, ar, rar et lha.
