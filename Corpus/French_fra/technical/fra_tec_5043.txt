Ce module est une interface vers la librairie iconv.
Pour pouvoir l'utiliser, vous devez compiler PHP avec l 'option --with-iconv.
Pour cela, vous devez avoir la fonction iconv() dans votre librairie standard C, ou bien la librairie libiconv installée sur votre système.
La librairie libiconv est disponible à http: / /www.gnu.org / software / libiconv /.
L 'extension iconv convertit des fichiers entre divers jeux de caractères.
Les jeux supportés dépendent de l'implémentation de iconv() sur votre système.
Notez que cette fonction ne fonctionne pas toujours bien sur tous les systèmes.
Dans ce cas, vous devez installer la librairie tout de même.
