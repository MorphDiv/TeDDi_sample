Assistant Web - Détails de la mise en page
Utilisez cette page de l'assistant pour personnaliser les options de mise en page de la page d'index de votre site Web.
Nom de fichier
Inclut les noms des fichiers des documents dans la page d'index.
Description
Inclut les informations résumées des documents dans la page d'index.
Auteur
Inclut les noms des personnes qui ont créé les documents dans la page d'index.
Date de création
Inclut les dates de création des documents dans la page d'index.
Date de la dernière modification
Inclut la date de la dernière modification d'un fichier dans la page d'index.
Format de fichier
Affiche le format des fichiers dans la page d'index.
Icône format de fichier
Affiche l'icône de format de fichier dans la page d'index.
N° des pages
Affiche dans la page d'index le nombre de pages que comporte votre site.
Taille (Ko)
Affiche la taille du fichier en kilo-octets dans la page d'index.
Optimiser la mise en page pour la résolution d'écran :
640 x 480
Optimise le site Web pour une résolution d'écran de 640 x 480 pixels.
800 x 600
Optimise le site Web pour une résolution d'écran de 800 x 600 pixels.
1024 x 768
Optimise le site Web pour une résolution d'écran de 1 024 x 768 pixels.
