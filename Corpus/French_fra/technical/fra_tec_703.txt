Assistant Zone de liste : liaison de champs
Indique par quels champs les tables des valeurs et les tables de listes sont liées.
La table des valeurs est la table du formulaire actif dans lequel le champ de liste est inséré.
La table de liste est la table dont les données sont affichées dans le champ de liste.
Les deux tables doivent être liées par un champ de données commun.
Ces liens doivent être saisis dans cette page de l'assistant.
Les noms de champ ne doivent pas nécessairement être les mêmes (tout dépend de la manière dont ils sont définis dans les deux tables), mais le type de champ doit être identique.
