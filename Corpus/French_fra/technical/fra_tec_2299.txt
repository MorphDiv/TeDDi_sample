Guide détaillé
Ce chapitre décrit comment effectuer certaines tâches habituelles dans KPresenter.
Créer un diaporama HTML
Créer une présentation de diapositives HTML est une manière de transformer une présentation existante en pages WWW.
Faites d'abord quelques diapos à inclure dans la présentation.
Une boîte de dialogue vous demande si vous voulez utiliser un fichier existant.
Cliquez sur Non si c'est la première fois que vous l'utilisez.
L'option Oui vous permet d'utiliser les détails que vous pavez précédemment saisis pour votre diaporama HTML.
Vous obtenez alors une boîte de dialogue qui vous demande quelques détails.
Remplissez chaque champ et assurez -vous que les détails de chemin sont corrects, car c'est l'endroit où KPresenter enregistrera vos nouveaux fichiers et images HTML.
Cliquez sur Suivant quand vous avez fini.
Vous avez alors quelques options d'affichage à régler.
Quand KPresenter crée les pages web pour vous, les couleurs du texte seront ce que vous avez fixé ici.
Cliquez sur la barre colorée pour choisir une charte de couleurs que vous convient.
Le bouton Suivant ouvre une boîte de dialogue qui vous permet de changer le nom de chaque diapo.
Enfin, enregistrez la configuration avant de fermer la boîte de dialogue.
Ce que nous avons maintenant est un dossier de HTML et d'images, créé là où nous avons précédemment positionné le chemin.
Il y a aussi un fichier index.html pour lancer votre diaporama web.
Vous pouvez voir comment ça fonctionne en utilisant votre gestionnaire de fichiers ou navigateur web pour ouvrir le fichier index.html.
Cliquez où il dit et la première page de votre présentation sera affichée.
Ça fonctionne ensuite comme un diaporama KPresenter.
Chaque clic sur l'écran vous amène à la diapo suivante.
Envoyez -le en FTP vers votre site web, en vous souvenant de respecter la structure des dossiers, et le monde pourra admirer votre travail.
Création de modèles KPresenter
Il est vraiment facile d'ajouter de nouveaux modèles à la bibliothèque de KPresenter.
Si vous avez fait une présentation dans un style que vous voudriez utiliser à nouveau, vous pouvez l'enregistrer comme modèle.
Ce didacticiel vous montrera comment nous faisons un nouveau modèle et l'enregistrons.
Ouvrir KPresenter et créer un nouveau document.
J'ai choisi un arrière-plan uni avec une boîte d'entrée de texte.
Pour cela, il faut cliquer de bouton Droit sur la zone de travail et on obtient une boîte de dialogue Arrière-plan.
Sous Format, vous trouverez Arrière-plan..., qui ouvre la même boîte de dialogue.
Je veux que l'arrière-plan soit attractif.
Dans la boîte combo du haut, vous pouvez choisir d'utiliser une image ou un clipart pour votre fond.
J'ai laissé ceci comme option par défaut Couleur/Dégradé.
Dans la boîte Combo suivante du bas, j'ai choisi Dégradé vertical, cliqué dans les deux longues boîtes dessous et choisi deux couleurs.
Vous pouvez changer les positions de début des couleurs dans les dégradés en cliquant sur la case à cocher Déséquilibré.
Ceci vous permet d'utiliser une paire de glissières X et Y.
Quelques fois, une glissière ne fera rien du tout, en fonction de quel type de gradient vous avez choisi.
Pensez à choisir les contrastes de couleurs en premier. si vous laissez les deux blancs, vous ne verrez rien.
Quand vous avez un assortiment de couleurs qui vous plaît, cliquez sur Appliquer et OK pour fermer la boîte de dialogue.
La page devrait maintenant être des couleurs que vous avez choisies. double-cliquez dans la boîte de texte pour ajouter un titre.
Quand vous êtes content de votre modèle, cliquez sur Fichier dans la barre de menus et choisissez Créer un modèle à partir de la diapo actuelle.
Cette boîte de dialogue vous permet de sélectionner où vous voulez que vos modèles soient enregistrés.
J'ai choisi d'enregistrer le mien sous le menu Présentations sur écran et l'ai appelé tutorialtemplate.
Cliquez sur OK pour clore cette boîte de dialogue.
Maintenant, fermez le document sur lequel vous avez travaillé (ne l'enregistrez pas) et KPresenter affichera la boîte de dialogue Choisir.
Vous voulez Créer un nouveau document à partir d'un modèle.
Cliquez sur l'onglet Présentations sur écran et il y a le modèle que nous avons fait plus tôt (marqué en rouge par moi).
