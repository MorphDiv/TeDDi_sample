Le fichier de relation de nom name sera ouvert. name peut être un nom de fichier, ou un chemin relatif ou absolu.
Dans tous les cas, il sera remplacé par un chemin absolu, sur une machine spécifique et sur un serveur spécifique.
En cas de succès, une relation (ressource de fichier de relation, dit aussi curseur) sera retourné.
Il sera alors utilisé avec les fonctions utilisant cette relation.
En cas d'échec, la valeur zéro est retournée, et le code d'erreur peut être demandé via la fonction dbplus_errno().
