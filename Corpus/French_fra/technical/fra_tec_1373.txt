Options d'impression
Spécifie les options des paramètres d'impression.
Réduire les données d'impression
Vous pouvez réduire la quantité de données envoyée à l'imprimante.
Ce faisant, vous accélérez l'impression, car les fichiers d'impression sont plus petits.
Pour les imprimantes dotées de peu de mémoire, l'impression est facilitée.
La réduction des données d'impression peut altérer légèrement la qualité d'impression.
Paramètres pour
Spécifie si les paramètres d'impression s'appliquent à l'impression directe ou à l'impression dans un fichier.
Réduire la transparence
Si vous activez cette case à cocher, les objets transparents seront imprimés comme des objets non transparents normaux, en fonction du bouton radio sélectionné.
La transparence ne peut être prise en charge directement par l'imprimante.
Les zones du document contenant de la transparence doivent donc être converties en bitmaps avant d'être envoyées à l'imprimante.
Selon la taille des bitmaps et la résolution d'impression, la quantité de données peut être importante.
Automatiquement
Spécifie que la transparence est uniquement imprimée si la zone transparente recouvre moins d'un quart de la page entière.
Aucune transparence
Cette option permet de ne jamais imprimer la transparence.
Réduire les bitmaps
Spécifie que les images bitmap sont imprimées avec une qualité réduite.
La résolution peut uniquement être réduite.
Elle ne peut être augmentée.
Qualité d'impression standard/optimale
La qualité d'impression optimale correspond à une résolution de 300 ppp.
La qualité d'impression standard correspond à une résolution de 200 ppp.
Résolution
Spécifie la qualité d'impression maximale en dpi.
La résolution peut uniquement être réduite.
Elle ne peut être augmentée.
Inclure les objets transparents
Si vous activez cette case, la réduction de la qualité d'impression pour les bitmaps s'applique également aux zones transparentes des objets.
Réduire les dégradés
Si vous activez cette case, les dégradés seront imprimés avec une qualité réduite.
Bandes de dégradés
Spécifie le nombre maximal de bandes de dégradés pour l'impression.
Couleur intermédiaire
Spécifie que les dégradés sont uniquement imprimés dans une couleur intermédiaire unique.
Convertir les couleurs en niveaux de gris
Indique que les couleurs doivent être imprimées uniquement en niveaux de gris.
Avertissements
Définit les avertissements s'affichant avant le début de l'impression.
Taille du papier
Activez cette case si un format de papier spécifique est nécessaire pour imprimer le document actif.
Si le format de papier utilisé dans le document n'est pas fourni par l'imprimante active, un message d'erreur s'affiche.
Orientation du papier
Activez cette case à cocher si l'impression du document requiert une orientation particulière.
Si le format utilisé pour le document actif n'est pas disponible sur l'imprimante, un message d'erreur s'affiche.
Transparence
Activez cette case à cocher si vous souhaitez toujours être averti lorsque le document contient des objets transparents.
Lorsque vous imprimez un tel type de document, une boîte de dialogue vous demande si la transparence doit être imprimée dans cette instruction d'impression.
