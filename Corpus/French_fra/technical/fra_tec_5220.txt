Introduction
& kexi; est une application de gestion de base de données. Elle peut être utilisée pour créer des bases de données, insérer des données, effectuer des requêtes et utiliser les données. Des formulaires peuvent être créés pour fournir une interface personnalisée à vos données. Tous les objets de base de donnée– tables, requêtes et formulaires– sont stockés dans la base de donnée, rendant aisé le partage de données et de structure.
& kexi; appartient à & koffice;, la suite bureautique de l'environnement de bureau KDE.
En plus d'enregistrer vos bases de données & kexi; dans des fichiers, & kexi; peut aussi stocker vos données sur un serveur de base de donnée. Utiliser un serveur de base de données vous permet de partager votre base de données avec d'autres utilisateurs et d'utiliser la base de données à plusieurs à la fois. Les serveurs de base de données supportés par & kexi; sont les suivants & #160;:
MySQL (Voyez http: / /www. mysql. com/)
PostgreSQL (Voyez http: / /www. postgresql. org/)
