IsArray, fonction
Fonction IsArray [Exécution]
Détermine si une variable est un champ de données dans une matrice.
Syntaxe :
IsArray (Var)
Valeur de retour :
Logique (Bool)
Paramètres :
Var :
toute variable à tester si elle est déclarée comme matrice.
S'il s'agit d'une matrice, la fonction renvoie la valeur
True
; dans le cas contraire, c'est la valeur
False
qui est renvoyée.
Exemple :
Sub ExampleIsArray
Dim sDatf(10) as String
print isarray(sdatf())
end Sub
