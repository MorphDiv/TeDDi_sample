Remplissage automatique de données à partir de cellules adjacentes
Grâce aux commandes de remplissage automatique et de série, vous pouvez automatiquement insérer des données dans les cellules.
Utilisation du remplissage automatique
Le remplissage automatique génère automatiquement une série de données sur la base d'un modèle prédéfini.
Cliquez sur une cellule dans la feuille, puis saisissez un nombre.
Cliquez dans une autre cellule, puis cliquez à nouveau dans la cellule qui contient le nombre que vous venez de saisir.
Faites glisser la poignée de remplissage vers le coin inférieur droit de la cellule, sélectionnez les cellules à remplir et relâchez le bouton de la souris.
Les cellules sont remplies avec des nombres croissants.

Outils - Options - %PRODUCTNAME Calc - Listes de tri

Double-cliquez sur la poignée de remplissage pour remplir automatiquement toutes les colonnes vides du bloc de données actif.
Par exemple, saisissez d'abord Jan dans la case A1 et faites glisser la poignée de remplissage jusqu'à la case A12 pour obtenir les douze mois de l'année dans la première colonne.
Saisissez ensuite des valeurs dans les cases B1 et C1.
Sélectionnez ces deux cellules, puis double-cliquez sur la poignée de remplissage.
Le bloc de données B1:C12 se remplit automatiquement.
Utilisation d'une série définie
Sélectionnez dans la feuille la plage de cellules que vous souhaitez remplir.
