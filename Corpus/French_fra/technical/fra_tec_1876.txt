Fonction EqualUnoObjects [Exécution]
Renvoie True si les deux objets Uno Basic spécifiés représentent la même instance d'objet Uno.
Syntaxe :
EqualUnoObjects (oObj1, oObj2)
Valeur de retour :
Logique (Bool)
Exemple :
// Copie d'objets - > même instance
oIntrospection = CreateUnoService( &quot; com.sun.star.beans.Introspection &quot; )
oIntro2 = oIntrospection
print EqualUnoObjects( oIntrospection, oIntro2 )
// Copie de classes struct en tant que valeurs - > nouvelle instance
Dim Struct1 as new com.sun.star.beans.Property
Struct2 = Struct1
print EqualUnoObjects( Struct1, Struct2 )
