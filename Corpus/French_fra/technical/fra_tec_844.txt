Insertion de données à partir de documents texte
Vous pouvez insérer du texte dans d'autres types de documents, tels que les classeurs et les présentations.
Notez qu'il existe une différence selon que le texte est inséré dans un cadre texte, dans une cellule de classeur ou dans le mode Plan d'une présentation.

Copie de texte par glisser-déposer
Si vous sélectionnez du texte et que vous le faites glisser dans une feuille de calcul par glisser-déposer, il sera inséré en tant que texte dans la cellule dans laquelle vous relâchez la souris.
Si vous faites glisser du texte vers une présentation en mode d'affichage normal, un objet OLE est inséré en tant que plug-in de $[officename].
Si vous faites glisser du texte vers une présentation en mode Plan, le texte sera inséré à l'emplacement du curseur.
