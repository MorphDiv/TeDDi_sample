Création d'une lettre type
Pour créer une lettre type, vous devez utiliser un document texte contenant des champs pour les données des adresses et une base de données d'adresses.
Vous pouvez ensuite regrouper ou fusionner les données des adresses et les documents texte, soit pour imprimer les lettres, soit pour les envoyer par e-mail.
Si le document est au format HTML, toute image liée ou incorporée ne sera pas envoyée avec le mail.
