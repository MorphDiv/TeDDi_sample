Internet
Internet
de la boîte de dialogue
Hyperlien
pour éditer des hyperliens vers des adresses WWW ou FTP.
Les champs relatifs au login, au mot de passe et à l'utilisateur anonyme ne sont disponibles que pour les adresses FTP.
Type d'hyperlien
Web
Crée un hyperlien HTTP.
FTP
Crée un hyperlien FTP.
Telnet
Crée un hyperlien Telnet.
Cible
Saisissez un URL pour le fichier que vous voulez ouvrir quand vous cliquez sur l'hyperlien.
Si vous ne spécifiez pas de cadre cible, le fichier s'ouvre dans le document actif ou dans le cadre.
Navigateur WWW
Ouvre un navigateur Web dans lequel vous pouvez charger l'URL souhaitée.
Vous pouvez ensuite copier et coller l'URL dans le champ
Cible
.
Cible à l'intérieur du document
Cible dans le document
Vous pouvez sélectionner la destination au sein d'un document et l'ajouter à l'URL cible en cliquant sur le bouton
Insérer
.
Spécifie la position dans le document cible à laquelle vous souhaitez passer.
Cible
de la boîte de dialogue
Hyperlien
.
Fermer
pour le définir et quitter la boîte de dialogue.
Nom de login
Spécifie votre login si vous utilisez des adresses FTP.
Mot de passe
Spécifie votre mot de passe si vous utilisez des adresses FTP.
Utilisateur anonyme
Permet de se connecter à l'adresse FTP en tant qu'utilisateur anonyme.
Autres paramètres
Cadre
Saisissez le nom du cadre dans lequel vous souhaitez ouvrir le fichier lié ou sélectionnez un cadre prédéfini dans la liste.
Si vous ne renseignez pas cette zone, le fichier lié s'ouvre dans la fenêtre du navigateur actif.
Formulaire
Indique si l'hyperlien est inséré sous forme de texte ou de bouton.
Événements
Assignation de macro
permettant de définir le code de programmation d'événements spécifiques, tels que &quot; Souris sur l'objet &quot; ou &quot; Déclencher l'hyperlien &quot; .
Texte
Spécifie le texte visible ou la légende du bouton correspondant à l'hyperlien.
Nom
Saisissez un nom pour l'hyperlien.
$[officename] insère une balise NAME dans l'hyperlien.
