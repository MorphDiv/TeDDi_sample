Assistant Web - Aperçu
Spécifiez l'emplacement souhaité pour la publication et l'aperçu de votre site Web.
Aperçu
Ouvre votre page Web dans le navigateur Web par défaut de votre système d'exploitation.
Répertoire local
Télécharge la page d'index et les fichiers dans un répertoire local.
La page d'index est enregistrée dans l'emplacement spécifié.
Tous les autres fichiers sont enregistrés dans le dossier monsiteWeb, dans le répertoire qui contient la page d'index.
...
Ouvre une boîte de dialogue de sélection de dossier.
Fichier archive
Ajoute la page d'index et les fichiers à un fichier archive compressé et télécharge le fichier sur votre site Web.
La page d'index est enregistrée dans l'emplacement spécifié.
Tous les autres fichiers sont enregistrés dans le dossier monsiteWeb, dans le répertoire qui contient la page d'index.
Selon le système d'exploitation utilisé, les formats disponibles pour les fichiers archives sont ZIP, GZIP et WAR.
...
Ouvre une boîte de dialogue dans laquelle vous pouvez spécifier l'emplacement du fichier archive.
Le Web via FTP
Télécharge vos fichiers sur un serveur FTP.
La page d'index est enregistrée dans l'emplacement spécifié.
Tous les autres fichiers sont enregistrés dans le dossier monsiteWeb, dans le répertoire qui contient la page d'index.
Si vous vous connectez à Internet via un serveur Proxy, vous ne pourrez pas utiliser l'option FTP.
Configurer
Connexion FTP
dans laquelle vous pouvez éditer et tester les paramètres de connexion du serveur FTP.
Enregistrer les paramètres
Enregistre les paramètres que vous avez spécifiés dans l'assistant.
Enregistrer sous
Saisissez le nom du fichier de paramètres.
