Insertion de données externes dans une feuille de calcul
Requête pages Web ($[officename] Calc)
permet d'insérer des tables provenant de documents HTML dans un classeur Calc.
Vous pouvez procéder de la même façon pour insérer des plages définies par nom à partir d'un classeur Calc ou Microsoft Excel.
Méthodes d'insertion :
Insertion via une boîte de dialogue
Placez le curseur au niveau de la cellule dans laquelle vous souhaitez insérer le nouveau contenu.
Insertion - Données externes
.
La boîte de dialogue
Données externes
s'ouvre.
...
pour ouvrir une boîte de dialogue de sélection de fichier.
Dans la zone de liste de la boîte de dialogue, sélectionnez les plages nommées ou les tables à insérer.
Cette boîte de dialogue vous permet également de spécifier la fréquence d'actualisation des plages ou des tables (toutes les n secondes).
Le filtre d'import peut immédiatement créer des noms pour les plages de cellules.
La mise en forme est conservée autant que possible, alors que le filtre ne charge volontairement aucune image.
Insertion via le Navigateur
Ouvrez deux documents : le classeur $[officename] Calc dans lequel insérer les données externes (document cible) et le document contenant les données externes à insérer (document source).
Ouvrez le Navigateur dans le document cible.
Dans la zone combinée présente dans la partie inférieure du Navigateur, sélectionnez le document source.
Le Navigateur affiche alors les noms des plages ainsi que les plages de base de données ou tables (feuilles) contenus dans le document source.
