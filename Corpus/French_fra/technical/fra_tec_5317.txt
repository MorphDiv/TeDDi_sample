Module de distance angulaire
Outils
Calculette astronomique
Module de distance angulaire
Le module de calcul de distance angulaire
Distance angulaire
L'outil de distance angulaire sert à mesurer l'angle entre deux points dans le ciel. Vous spécifiez simplement les coordonnées équatoriales de la paire désirée de points, et actionnez le bouton Calculer pour obtenir l'angle entre les deux points.
Il y a aussi un mode automatisé pour ce module. Dans ce mode, vous spécifiez un fichier d'entrée qui contient quatre nombre par ligne & #160;: les valeurs d'AD et de Déc pour les paires de points. D'une autre manière, vous pouvez spécifier une simple valeur pour n'importe laquelle de ces coordonnées dans le panneau de calcul (les valeurs correspondantes dans le champ d'entrée doivent être sautées si elles sont spécifiées dans le calculateur).
Une fois que vous avez spécifié le nom de fichier d'entrée et un nom de fichier de sortie, actionnez simplement le bouton Lancer pour générer le fichier de sortie.
