Utilisation des tables
Les données sont stockées dans les tables.
Par exemple, le carnet d'adresses du système qui contient les adresses e-mail de vos correspondants est une table de la base de données du carnet d'adresses.
Chaque adresse correspond à un enregistrement de données se présentant sous forme de ligne dans la table.
Les enregistrements comprennent plusieurs champs de données (champs du prénom, du nom ou de l'adresse e-mail, par exemple).

Création d'une nouvelle vue de table
Certains types de base de données prennent en charge les vues de tables.
Une vue de table est une requête enregistrée avec la base de données.
Dans une vue, vous pouvez effectuer la plupart des opérations de base de données comme vous le feriez avec une table.
Ouvrez le fichier de base de données dans lequel vous souhaitez créer la vue de table.
