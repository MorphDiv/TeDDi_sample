Éléments du menu
Chaque élément du menu est détaillé ci-dessous. Lorsque un raccourci clavier est disponible pour un élément du menu, celui -ci est indiqué.
Les menus de la fenêtre de la liste de contact
Menu Fichier
Fichier Régler l'état Connecté
Se connecte avec tous vos comptes
Fichier Régler l'état Absent
Donne l'état absent à tous vous comptes actuellement connectés
Fichier État Non connecté
Déconnecte tous vos comptes
Fichier Ajouter un contact...
Cela affiche l'assistant d'ajout de contact, qui vous permet d'ajouter un nouveau contact à votre liste
Fichier Créer un nouveau groupe...
Vous demande de saisir le nom du nouveau groupe et l'ajoute la liste de contacts.
Ctrl; Q Fichier Quitter
Vous déconnecte de tous les systèmes de & mi;, ferme toutes les fenêtres et quitte l'application.
Menu Édition
Ctrl; Z Édition Annuler
Annule le dernier changement effectué sur la liste de contacts.
Ctrl; Maj; Z Édition Refaire
Rétablit le dernier changement sur la liste de contact effectué par Édition Annuler
Édition Envoyez un message simple...
Ouvre une fenêtre vous permettant d'envoyer un seul message au contact sélectionné.
Édition Commencer une discussion...
Ouvre une fenêtre vous permettant de commencer une discussion avec le contact sélectionné.
Édition Envoyer un fichier...
Si cette fonctionnalité est géré par le système de & mi;, ouvre une fenêtre de sélection de fichier afin d'envoyer un fichier au contact sélectionné.
Édition Déplacer vers
(Disponible seulement si un contact est sélectionné) Choisissez un autre groupe à partir du sous-menu, et cela déplacera le contact vers ce groupe.
Édition Copier vers
(Disponible seulement si un contact est sélectionné) Choisissez un autre groupe à partir du sous-menu, et cela copiera le contact vers ce groupe. Les systèmes de & mi; qui permettent d'avoir des contacts dans plus d'un groupe à la fois, seront mis à jour en conséquence.
Suppr Édit Effacer le contact
(Disponible seulement si un contact est sélectionné) Efface le contact de la liste de contacts.
Édition Renommer le contact
(Disponible seulement si un contact est sélectionné) Renomme le contact sur la liste de contacts. Si vous faites cela, son nom dans la liste ne changera plus lorsque le contact changera son nom à afficher. Vous pouvez rétablir cela en modifiant les propriétés du contact.
Édition Ajouter un contact...
(Disponible seulement si un contact est sélectionné) Choisissez un autre compte à partir du sous-menu, et vous pouvez ajouter une façon de contacter cette personne en utilisant un autre compte.
Édition Ajouter à votre liste de contacts...
(Disponible seulement si un contact est sélectionné) Il arrive parfois que quelqu'un vous envoie un message sans être sur votre liste de contacts. Dans ce cas & kopete; crée une entrée temporaire pour eux, utilisez cette fonction pour les conserver à votre liste de contacts.
Édition Propriétés
La boîte de dialogue de propriétés vous permet de choisir des icônes personnalisées pour le contact sélectionné, ou de changer son nom.
Pour des contacts, vous pouvez changer leur entrée dans le carnet d'adresse de & kde;.
Édition Effacer le groupe
(Disponible seulement si un groupe est sélectionné) Efface complètement un groupe de la liste de contacts. Les contacts qui se trouvent uniquement dans ce groupe sont déplacés au niveau le plus haut de la liste de contacts.
Édition Renommer le groupe...
(Disponible seulement si un groupe est sélectionné) Renomme le groupe.

Les menus de la fenêtre de discussion
Menu Discussion
Ctrl; Enter; Discussion Envoyer un message
Envoie un message.
Ctrl; S Discussion Enregistrer
Enregistre la discussion dans un fichier.
Ctrl; P Discussion Imprimer...
Imprime la discussion.
Discussion Contacts
Ce menu liste toutes les personnes présentes dans la discussion. Vous avez accès au même menu qu'en faisant un clic droit sur le nom de la personne dans la liste de contacts, vous permettant d'effectuer des actions spécifiques, telles que envoyer un fichier, voir les informations de l'utilisateur ou la bloquer.
Ctrl; F4 Discussion Fermer
Ferme la discussion courante. S'il y a des discussions ayant lieu dans d'autres onglets de cette fenêtre, celle -ci reste ouverte.
Alt; F4 Discussion Quitter
Ferme toutes les discussions se déroulant dans cette fenêtre.
Ctrl; X Édition Couper
Couper du texte fonctionne comme dans la plupart des éditeurs & #160;: le texte sélectionné est enlevé et placé dans le presse-papiers. Noter qu'il est également possible de sélectionner du texte et de le déplacer à une nouvelle position.
Ctrl; C Édition Copier
Copier du texte fonctionne comme dans la plupart des éditeurs & #160;: le texte sélectionné est copié dans le presse-papiers. Noter qu'il est également possible de sélectionner du texte en maintenant la touche & Ctrl; enfoncée, et en l'amenant à une nouvelle position à copier.
Ctrl; V Édition Coller
Coller du texte fonctionne comme dans la plupart des éditeurs & #160;: le texte en provenance du presse-papiers est collé à la position courante du curseur.
Menu Format
Format Ajouter un smiley
Ce menu contient tous les smileys/ émoticônes que le schéma courant d'émoticônes contient. Vous pouvez changer ce schéma dans le menu de configuration de Kopete.
Format Couleur du texte...
Ouvre une boîte de dialogue de sélection de couleurs, permettant de modifier la couleur du texte.
Format Couleur d'arrière plan...
Ouvre une boîte de dialogue de sélection de couleurs, permettant de modifier la couleur de l'arrière plan.
Format Police
Ce menu vous permet de modifier la police utilisée dans la discussion.
Format Taille de la police
Ce menu vous permet de modifier la taille de la police utilisée dans la discussion.
Menu Onglets
Onglets Politique de placement des onglets
Ce menu vous permet de changer si les onglets apparaissent en haut ou en bas de la fenêtre de discussion.
Ctrl; Maj; B Onglets Détacher la discussion
Sépare la discussion courante dans sa propre fenêtre.
Onglets Déplacer l'onglet vers la fenêtre
Vous pouvez déplacer des discussions entre les fenêtres avec ce menu. Choisissez dans le menu la fenêtre de discussion où l'onglet sera déplacé.
Menu Outils
Ce menu contient les éléments ajoutés par les modules externes. Voir le chapitre sur les modules externes pour plus d'informations.

Configuration Barres d'outils État (Kopete)

Configuration Barres d'outils Barre d'outils de format (Kopete)
Affiche/ cache la barre d'outil de formatage de texte.
Configuration Liste des membres de la discussion
Ce menu permet de contrôler si la liste des personnes dans la discussion apparaît sur la droite ou sur la gauche de la fenêtre de discussion, ou si elle n'apparaît pas.
Affiche la boîte de dialogue standard de & kde; de configuration des raccourcis clavier, où vous pouvez changer les raccourcis disponible dans la fenêtre de discussion.
Affiche la boîte de dialogue standard de & kde; de configuration des barres d'outils, qui vous permet de personnaliser la barre d'outils de la liste de discussion.
