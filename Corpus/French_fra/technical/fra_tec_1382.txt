Plug-in du navigateur
Indique que le plug-in Mozilla doit être utilisé pour afficher les documents %PRODUCTNAME.
Lorsque vous cliquez sur un hyperlien pointant vers un document %PRODUCTNAME dans le navigateur Mozilla ou Netscape, l'un des événements suivants peut se produire :
Le navigateur vous demande si vous souhaitez enregistrer le document sur le disque ou si vous souhaitez utiliser une application pour ouvrir le document.
Cette boîte de dialogue de demande apparaît par défaut lorsque vous cliquez pour la première fois sur un lien hypertexte pointant vers un document %PRODUCTNAME et que le plug-in du navigateur n'a pas été activé.
Le navigateur télécharge le document dans un dossier temporaire, puis demande au système d'exécuter %PRODUCTNAME et d'ouvrir le document dans %PRODUCTNAME.
Il s'agit du paramètre par défaut si vous n'avez pas activé le plug-in du navigateur et si vous avez choisi d'ouvrir le document avec %PRODUCTNAME à la première invite.
Éditer le fichier
, une copie du document est téléchargée dans un dossier temporaire sur le disque.
Démarrez %PRODUCTNAME pour que l'application charge la copie à des fins d'édition.
Afficher les documents dans le navigateur
Active le plug-in du navigateur pour afficher les documents %PRODUCTNAME dans une fenêtre du navigateur.
Sélectionnez le plug-in, cliquez sur OK et redémarrez le navigateur.
Dans la fenêtre du navigateur, sélectionnez un lien vers un document %PRODUCTNAME.
Pour que le plug-in fonctionne, Mozilla , Netscape ou Firefox doivent être installés sur votre système.
