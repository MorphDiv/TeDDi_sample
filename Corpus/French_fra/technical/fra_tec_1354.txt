Navigateur de données
Spécifie la structure des données du document XForms actif.
Nom du modèle
Sélectionne le modèle XForms à utiliser.
Modèles
Ajoute, renomme et supprime des modèles XForms.
Ajouter
Ouvre la boîte de dialogue Ajouter un modèle dans laquelle vous pouvez ajouter un modèle XForm.
Saisissez le nom.
Les mises à jour des données du modèle changent le statut de modification du document
Lorsque activé, le statut du document sera défini sur &quot; modifié &quot; lorsque vous modifiez tout contrôle de formulaire qui est lié à des données dans le modèle.
Si désactivé, de telles modifications ne définissent pas le statut du document sur &quot; modifié &quot; .
Supprimer
Supprime le modèle XForm sélectionné.
Vous ne pouvez pas supprimer le dernier modèle.
Renommer
Renomme le modèle Xform sélectionné.
Afficher les détails
Modifie l'affichage pour afficher ou masquer les détails.
Instance
Liste les éléments qui appartiennent à l'instance active.
Envois
Liste les envois.
Liaisons
Liste les liaisons du formulaire XForm.
Instances
Ce bouton comporte des sous-menus pour ajouter, éditer ou supprimer des instances.
Ajouter
Ouvre une boîte de dialogue dans laquelle vous pouvez ajouter une nouvelle instance.
Éditer
Ouvre une boîte de dialogue dans laquelle vous pouvez modifier l'instance active.
Supprimer
Supprime l'instance active.
Vous ne pouvez pas supprimer la dernière instance.
Afficher les types de données
Modifie l'affichage pour afficher plus ou moins de détails.
Ajouter
Ouvre une boîte de dialogue permettant d'ajouter un nouvel élément (élément, attribut, envoi ou liaison) comme sous-élément de l'élément actif.
Éditer
Ouvre une boîte de dialogue permettant d'éditer l'élément sélectionné (élément, attribut, envoi ou liaison).
Supprimer
Supprime l'élément sélectionné (élément, attribut, envoi ou liaison).
