Java
Spécifie les options de prise en charge des applications Java dans %PRODUCTNAME, et notamment l'environnement d'exécution Java (JRE) à utiliser.
Options Java
Utilisation d'un environnement d'exécution Java
Permet d'exécuter des applications Java dans %PRODUCTNAME.
Lorsqu'une application Java tente d'accéder à votre disque dur, une fenêtre d'invite s'ouvre.
Environnements d'exécution Java installés :
Sélectionnez le JRE à utiliser.
Sur certains systèmes, vous devez attendre une minute avant que la liste soit peuplée.
Sur certains systèmes, vous devez redémarrer %PRODUCTNAME pour utiliser les paramètres modifiés.
Le chemin du JRE s'affiche sous la zone de liste.
Ajouter
Indiquez le chemin du dossier racine d'un JRE stocké sur votre ordinateur.
Le chemin est défini dans la boîte de dialogue ci-dessous.
