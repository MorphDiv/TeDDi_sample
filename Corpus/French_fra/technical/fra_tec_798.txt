Mode Sélection
Affiche le mode de sélection actif.
Vous pouvez basculer entre STD = Standard, EXT = Extension, AJT = Ajout, BLC = Bloc.
Chaque clic dans le champ affiche, en succession, l'une des options disponibles :
Affichage :
Mode :
Effet :
STD
Mode standard
Cliquez sur le texte pour positionner le curseur à cet emplacement ; cliquez sur une cellule pour l'activer.
La sélection effectuée est supprimée.
EXT
Mode Extension (F8)
La sélection active est étendue ou rognée en cliquant dans le texte.
AJT
Mode de sélection supplémentaire (Maj+F8)
Une nouvelle sélection est ajoutée à une sélection existante.
Il en résulte une sélection multiple.
BLC
Commande
Ctrl
+Maj+F8)
Un bloc de texte peut être sélectionné.
Alt
enfoncée pendant que vous glissez la souris pour sélectionner un bloc de texte.
Vous n'avez pas besoin d'entrer dans le mode Sélection de bloc.
