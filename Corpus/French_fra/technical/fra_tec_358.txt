Sélectionnez la plage de données.
Cliquez sur l'une des options pour des séries de données en lignes ou en colonnes.
Vérifiez si la plage de données a des étiquettes dans la première ligne, dans la première colonne ou les deux.
Organisation des séries de données
Dans la zone de liste Séries de données, vous voyez une liste de toutes les séries de données du diagramme actif.
Les séries de données en colonne sont positionnées en début de liste, les séries de données en ligne en bas de liste.
Pour organiser les séries de données, sélectionnez une entrée de la liste.
Cliquez sur Ajouter pour ajouter une autre série de données sous l'entrée sélectionnée.
La nouvelle série de données est du même type que l'entrée sélectionnée.
Cliquez sur Supprimer pour supprimer l'entrée sélectionnée de la liste des séries de données.
Utilisez les flèches de direction Haut et Bas pour déplacer vers le haut ou vers le bas l'entrée sélectionnée dans la liste.
Vous pouvez ainsi convertir des séries de données en colonne en séries de données en ligne et inversement.
Cela ne modifie pas l'ordre dans la table de source de données, seulement la présentation dans le diagramme.
Édition des séries de données
Cliquez sur une entrée de la liste pour en afficher et éditer les propriétés.
Dans la zone de liste Plages de données, vous voyez les noms de rôle et les plages de cellules des composants des séries de données.
Cliquez sur une entrée, puis éditez le contenu dans la zone de texte en-dessous.
L'étiquette près de la zone de texte indique le rôle actif sélectionné.
Sélection d'une plage de données
pour minimiser la boîte de dialogue et sélectionner une plage avec la souris.
La plage pour un rôle de données, comme les valeurs Y, ne doit pas inclure une cellule d'étiquette.
Édition des catégories ou des étiquettes de données
Saisissez ou sélectionnez une plage de cellules qui sera utilisée comme texte pour les catégories ou pour les étiquettes de données.
Les valeurs dans la plage des catégories seront affichées comme étiquettes sur l'axe x.
Insertion des éléments du diagramme
Utilisez la page Éléments du diagramme de l'assistant Diagramme pour insérer les éléments suivants :
Titres du diagramme
Légende
Lignes de grille visibles
Pour insérer des éléments supplémentaires, utilisez le menu Insertion du diagramme en mode édition.
Vous pouvez alors définir les éléments suivants :
Axes secondaires
Grilles secondaires
Étiquettes de données
Statistiques, par exemple valeurs moyennes, barres d'erreurs y et courbes de tendance
Pour paramétrer différentes étiquettes de données pour chaque série de données, utilisez la boîte de dialogue des propriétés des séries de données.
