Commandes de menu
La fenêtre contenant le document sur lequel vous souhaitez travailler doit être sélectionnéepour pouvoir utiliser les commandes de menu.
De la même façon, vous devez sélectionner un objet dans le document pour utiliser la commande de menu associée à l'objet.
Les menus sont contextuels.
Cela signifie que les entrées de menu disponibles sont liées au travail en cours.
Si le curseur se trouve sur un texte, toutes les entrées de menu disponibles concernent alors l'édition de texte.
Si vous avez sélectionné des images dans un document, vous pourrez accéder à toutes les entrées de menu permettant d'éditer des images.
