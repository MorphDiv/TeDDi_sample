Le manuel de KAtomic
Ce manuel décrit KAtomic version 2.0
Introduction
Divertissement Atomique est un petit jeu qui ressemble à Sokoban.
Le but du jeu est de construire des molécules chimiques sur un plateau ressemblant à celui de Sokoban..
Jouer à Divertissement atomique
Règles
Le but de Divertissement Atomique est de construire de molécules chimiques en utilisant les atomes de base fournis.
La molécule en construction est affichée dans un cadre de la fenêtre principale.
En cliquant sur un atome, vous ferez apparaître des flèches autour.
Ces flèches montrent les directions vers lesquelles l'atome peut se déplacer.
Après avoir cliqué sur une flèche, l'atome se déplacera dans cette direction jusqu'à ce qu'il atteigne un bord ou un autre atome.
Si deux atomes se touchent avec les connecteurs correspondants, ils forment une molécule.
Vous ne pouvez déplacer qu'un seul atome à la fois.
Le niveau est terminé lorsque la nouvelle molécule a la même structure que celle montrée dans la fenêtre de prévisualisation.
Dans les niveaux supérieurs, il vous faudra une bonne habileté pour résoudre le puzzle
Le meilleur score dans ce jeu est en fait le plus bas score, puisque le but du jeu est de résoudre un niveau avec le moins de coups possibles.
Dans la fenêtre principale, Meilleur score: affiche le plus petit nombre de coups effectués pour ce niveau.
Votre score actuel: montre le nombre de coups effectués pour le moment.
L'ascenseur en haut à droite de la fenêtre principale change le niveau de jeu.
Configuration
Le menu Jeu
Jeu Meilleurs scores
Ceci affiche les meilleurs scores du niveau.
Jeu Quitter
Sélectionner ceci mettra un terme à votre jeu en cours, et vous fera quitter KAtomic.
Le menu Configuration
Configuration Configurer les raccourcis-clavier
Ce paramètre vous permet de modifier les raccourcis clavier de Divertissement Atomique.
Veuillez consulter le paragraphe pour une liste des réglages par défaut.
Dans la boîte de dialogue qui apparaît, vous pouvez configurer la vitesse d'animation du mouvement des atomes.
Le menu Aide
F1 Aide Contenu
Invoque le système d'aide de KDE en démarrant avec le manuel de KAtomic (ce document).
Dessine un point d'interrogation (?) à côté du pointeur de la souris.
Cliquer sur un item dans KAtomic affiche alors une fenêtre d'aide (si celle-ci existe pour cet item particulier), expliquant la fonction de l'élément en question.
Aide Rapport de bogue
Ouvre une boîte de dialogue de rapport de bogue.
Vous pouvez alors envoyer un rapport de bogue ou émettre un souhait.
