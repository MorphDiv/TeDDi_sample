Programme d'amélioration
Le programme d'amélioration enregistre certaines interactions de l'utilisateur et les envoient au projet User Experience.
http://ux.openoffice.org/
.
La seconde fois que vous démarrez le logiciel %PRODUCTNAME, vous voyez une fenêtre qui donne des informations à propos du programme d'amélioration.
À ce point, vous pouvez accepter ou refuser votre participation.
Si vous acceptez, l'enregistrement des données commence immédiatement.
Lorsque vous démarrez une nouvelle session de %PRODUCTNAME, les données collectées seront envoyées.
Si vous refusez de participer, aucune donnée ne sera envoyée et l'enregistrement des données s'arrête immédiatement.
Outils - Options - %PRODUCTNAME - Programme d'amélioration
.
Cliquez sur l'icône
Info
pour parcourir une page web contenant des informations complémentaires.
Oui, je souhaite participer …
pour activer la réaction automatique.
Non, je ne souhaite pas participer
pour désactiver la réaction automatique.
Données suivies
Les données suivantes seront enregistrées et envoyées :
Clique sur les boutons de la barre d'outils, incluant les barres d'outils de l'extension
Exécution des commandes de menu, incluant les menus d'une extension
Clique sur les commandes de menu contextuel
Polices appliquées en utilisant le menu
Taille de polices appliquée en utilisant le menu
Boutons, tels que OK, Annuler, etc.
La langue du document et la langue du paragraphe appliquées en utilisant le menu ou la barre d'état
Chaque élément enregistré inclut également un horodatage et le module (Writer, Calc, Impress, ...) lorsqu'il a été appelé
Les données suivantes ne seront pas enregistrées et envoyées :
Noms d'utilisateur et mots de passe
Tout type de contenu de document
Chemins, noms de fichier
Code source de l'utilisateur
Afficher les données
Cliquez sur le bouton pour afficher les données actuellement collectées.
Ces données seront envoyées la fois suivante, avec les données qui ont été collectées jusqu'à ce que vous ayez terminé la session active.
