Assistant Web : documents
Sélectionnez les fichiers que vous souhaitez télécharger sur votre site Web.
Vous pouvez les télécharger sur un serveur ou un répertoire local.
Documents
Répertorie les documents que vous souhaitez publier sur votre site Web.
Avant de charger les documents %PRODUCTNAME, l'assistant peut les convertir aux formats HTML, PDF ou, dans certains cas, Flash.
Tous les autres fichiers sont chargés dans leur format d'origine.
Ajout
Ouvre une boîte de dialogue dans laquelle vous pouvez sélectionner les fichiers que vous souhaitez ajouter à votre site Web.
L'ordre de la liste détermine l'ordre d'affichage des hyperliens vers les documents sur la page d'index de votre site Web.
Suppression
Supprime le fichier sélectionné de la liste.
Export au format de fichier
Sélectionnez le format dans lequel vous souhaitez exporter le fichier sélectionné.
Titre
Saisissez le titre du document sélectionné.
Le titre apparaît sous forme d'hyperlien vers le document sélectionné sur la page d'index de votre site Web.
Résumé
Entrez une description pour le document sélectionné.
Auteur
Entrez le nom de l'auteur du document sélectionné.
