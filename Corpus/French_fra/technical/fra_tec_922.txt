Utilisation de bases de données dans %PRODUCTNAME Base
Dans %PRODUCTNAME Base, vous pouvez accéder à des données enregistrées dans divers formats de fichier de base de données.
Les formats de base de données simples tels que dBase sont pris en charge par %PRODUCTNAME Base en mode natif. %PRODUCTNAME Base permet également de se connecter à des bases de données relationnelles externes comme MySQL ou Oracle.
Les types de base de données suivants sont des types en lecture seule dans %PRODUCTNAME Base.
A travers %PRODUCTNAME Base, il n'est pas possible de modifier la structure de la base de données ou d'éditer, d'insérer et supprimer des enregistrements pour ces types de base de données :
