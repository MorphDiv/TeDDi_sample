Proxy
Les serveurs Proxy
permettant d'accéder à Internet peuvent être paramétrés manuellement selon vos besoins.
Paramètres
serveur Proxy
.
Serveur Proxy
Indique le type de définition de proxy.
Aucun
Accède à Internet sans serveur Proxy.
Permet d'installer une connexion directe entre votre ordinateur et un fournisseur Internet n'utilisant pas de Proxy.
Manuel
Permet de saisir manuellement le serveur Proxy.
Indiquez les serveurs Proxy selon la configuration Internet applicable.
Demandez à votre administrateur système les adresses des serveurs Proxy et les ports à saisir.
Saisissez les noms de serveur sans le préfixe du protocole.
Saisissez par exemple www.sun.com, et non http://www.sun.com.
Système
Sur les systèmes Windows ou UNIX utilisant GNOME ou KDE, cette option demande à %PRODUCTNAME d'utiliser les paramètres système.
Vous devez redémarrer %PRODUCTNAME pour appliquer ces paramètres.
Http Proxy
HTTP
Saisissez le port dans le champ de droite.
Proxy HTTPS
Saisissez le nom du serveur proxy pour HTTPS.
Saisissez le port dans le champ situé à droite
Proxy FTP
FTP
Saisissez le port dans le champ de droite.
Pas de proxy pour
Spécifie les noms des serveurs ne nécessitant pas de serveurs proxy, en les séparant par des points-virgules.
Il s'agit par exemple des serveurs faisant partie du réseau local et des serveurs utilisés pour le streaming vidéo et audio.
Vous pouvez également utiliser des substituants pour les noms d'hôtes et de domaines.
Saisissez, par exemple, *.sun.com pour ne pas utiliser de Proxy pour tous les hôtes du domaine sun.com.
Port
Saisissez le port du serveur Proxy correspondant.
La valeur maximum d'un numéro de port est fixée à 65535.
