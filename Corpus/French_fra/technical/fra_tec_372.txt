Export de fichiers texte
permet de définir les options d'export des fichiers texte.

Options de champ
Définit le séparateur de champ, le séparateur de texte et le jeu de caractères utilisés pour l'export de texte.
Jeu de caractères
Spécifie le jeu de caractères pour l'export de texte.
Séparateur de champ
Choisissez ou saisissez un séparateur de champ afin de séparer les champs de données.
Séparateur de texte
Choisissez ou saisissez un séparateur de texte incluant tous les champs de données.
Enregistrer le contenu de la cellule comme affiché
Activé par défaut, les données seront enregistrées comme affichées, incluant les formats de nombre appliqués.
Si cette case à cocher n'est pas marquée, le contenu des données brut sera enregistré, comme dans les anciennes versions de %PRODUCTNAME.
i76507, see http://specs.openoffice.org/calc/filters/csv/save-to-csv.odt
En fonction du format de nombre, enregistrer le contenu de la cellule comme affiché écrira des valeurs qui ne pourront peut-être plus être interprétées comme des données numériques lors d'un import.
Largeur de colonne fixe
Exporte tous les champs de données selon une largeur fixe.
La largeur d'un champ de données figurant dans le fichier texte exporté est définie à la largeur actuelle de la colonne correspondante.
Les valeurs sont exportées au même format que celui en cours d'affichage dans la cellule.
Si une valeur dépasse la largeur de colonne fixe, elle est exportée sous forme de chaîne ###.
Si une chaîne de texte dépasse la largeur de colonne fixe, la fin de la chaîne est supprimée.
L'alignement À gauche, Centré et À droite est simulé par des espaces insérés.
