Statistiques Adabas D
Spécifie les options d'accès à la base de données Adabas.
Taille du tampon de données
Saisissez la taille du tampon de données de la base de données.
Ce paramètre prend effet après le redémarrage de la base de données.
Incrément
Saisissez la taille dont vous voulez incrémenter automatiquement la base de données.
La taille maximum de l'incrément est de 100 Mo.
Ce paramètre prend effet après le redémarrage de la base de données.
Contrôle Nom d'utilisateur
Saisissez le nom d'un utilisateur auquel vous souhaitez donner un contrôle limité pour modifier certains paramètres de la base de données.
Contre Mot de passe
Saisissez le mot de passe du contrôle Utilisateur.
Fermeture du service en même temps que $[officename]
Quitte le serveur de base de données Adabas lorsque vous quittez $[officename].
Cette option n'est disponible que si vous démarrez le serveur de base de données depuis $[officename] avec un contrôle Utilisateur et un mot de passe.
Avancé
Statistiques de la base de données
, dans laquelle vous pouvez afficher des statistiques concernant la base de données Adabas.
