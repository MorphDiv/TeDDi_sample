Portée de l'extensions
Les utilisateurs ayant des privilèges administrateur ou root verront une boîte de dialogue où ils peuvent choisir d'installer l'extension &quot; pour tous les utilisateurs &quot; ou &quot; uniquement pour moi &quot; .
Les utilisateurs normaux, sans ces privilèges, peuvent installer, supprimer ou modifier des extensions destinées uniquement à leur propre usage.
Un utilisateur ayant les privilèges root ou administrateur peut installer une extension comme extension partagée qui est disponible à tous les utilisateurs.
Après avoir sélectionné une extension, une boîte de dialogue s'ouvre et demande si l'installation est faite pour l'utilisateur actuel ou pour tous les utilisateurs.
Un utilisateur sans les privilèges root ne peut installer une extension que pour son propre usage.
Elle est appelée une extension utilisateur.

Ajouter
Cliquez sur Ajouter pour ajouter une extension.
Une boîte de dialogue de fichier s'ouvre dans laquelle vous pouvez sélectionner l'extension que vous souhaitez ajouter.
Pour copier et enregistrer l'extension sélectionnée, cliquez sur Ouvrir.
Lisez le contrat de licence.
Cliquez sur le bouton Faire défiler pour faire défiler le texte si nécessaire.
Cliquez sur Accepter pour continuer l'installation de l'extension.
Supprimer
Sélectionnez l'extension que vous souhaitez supprimer, puis cliquez sur Supprimer.
Activer ou désactiver l'extension.
Activer
Sélectionnez l'extension que vous souhaitez activer, puis cliquez sur Activer.
Désactiver
Sélectionnez l'extension que vous souhaitez désactiver, puis cliquez sur Désactiver.
Actualiser
Cliquez pour vérifier les mises à jour en ligne pour les extensions installées.
Pour vérifier la mise à jour en ligne d'une extension sélectionnée uniquement, choisissez la commande Mise à jour à partir du menu contextuel.

Mise à jour en ligne
