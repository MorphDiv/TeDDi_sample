Version de format ODF
OpenDocument
ODF version 1.2.
Les versions antérieures de OpenOffice.org 2 et StarOffice8 supportent les formats de fichier ODF 1.0/1.1.
Ces formats de fichiers antérieurs ne peuvent stocker toutes les nouvelles fonctions du nouveau logiciel.
OpenOffice.org 3 et StarOffice 9 peuvent ouvrir des documents dans les formats ODF 1.0/1.1 et 1.2.
Lorsque vous enregistrez un document dans OpenOffice.org 3 ou StarOffice 9, vous pouvez sélectionner si vous souhaitez enregistrer le document dans le format ODF 1.2, ODF 1.2 (Étendu) ou dans le format précédent ODF 1.0/1.1.
