Spécifie les paramètres généraux de $[officename].
Années à deux chiffres
Définit une plage de dates, dans laquelle le système reconnaît une année à deux chiffres.
Années à deux chiffres
permet à l'utilisateur de définir les dates pour lesquelles les années exprimées à l'aide de deux chiffres sont ajoutées à l'année 2000.
En d'autres termes, si vous spécifiez la date du 01/01/30 (ou une date postérieure), l'entrée 01/01/20 est reconnue comme le 01/01/2020 et non comme le 01/01/1920.
Aide
Spécifie le comportement de l'aide en ligne.
Infobulles
Affiche le nom des icônes et d'autres informations sous forme d'infobulles, telles que les titres des chapitres, lorsque vous faites défiler un document contenant des chapitres.
Infoballon
Affiche un texte d'aide lorsque vous placez le curseur sur une icône, une commande de menu ou un contrôle de boîte de dialogue.
Assistant
Spécifie que l'Assistant est affiché automatiquement dans certaines situations données.
Pour afficher les rubriques d'aide contenant les informations relatives au contexte, cliquez dans la fenêtre de l'Assistant.
Réinitialiser l'assistant
Si, à trois reprises, vous n'avez pas ouvert l'Assistant dans une situation particulière, mais l'avez au contraire fermé ou laissé se fermer automatiquement, l'Assistant ne s'affiche plus dans cette situation.
Réinitialiser l'assistant
pour restaurer la liste par défaut des situations pour lesquelles l'assistant s'affiche.

Boîtes de dialogue Ouvrir/Enregistrer
Utiliser les boîtes de dialogue $[officename]
Spécifie si les boîtes de dialogue de $[officename] permettent d'ouvrir et d'enregistrer des documents.
Si ce n'est pas le cas, ce sont les boîtes de dialogue du système d'exploitation qui sont utilisées.
Lorsque vous ouvrez un fichier en indiquant un URL dans la boîte de dialogue de fichier de Windows, ce dernier ouvre une copie locale du fichier, située dans le cache d'Internet Explorer.
La boîte de dialogue de fichier de %PRODUCTNAME ouvre le fichier distant.

Statut du document
L'impression active le statut &quot; Document modifié &quot;
Spécifie si l'impression du document est considérée comme une modification.
Lorsque cette option est cochée, le système vous demandera si vous souhaitez enregistrer les modifications la prochaine fois que vous fermerez le document .
La date d'impression est alors intégrée comme une modification dans les propriétés du document.
