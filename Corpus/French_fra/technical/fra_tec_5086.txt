Les fonctions readline() implémente une interface avec la librairie GNU Readline.
Ces fonctions fournissent une ligne de commande éditable, un peu comme lorsque Bash vous permet d'utiliser les flèches de déplacement pour insérer un caractère ou passer en revue l 'historique.
A cause de l'interactivité de ces commande, elles ne seront que rarement utiles pour les applications Web, mais peuvent se révéler utiles lorsqu 'un script est exécuté depuis une commande shell.
Le site du projet GNU Readline est http: / /cnswww.cns.cwru.edu / ~chet / readline / rltop.html.
Elle est entretenue par Chet Ramey, qui est aussi l 'auteur de Bash.
