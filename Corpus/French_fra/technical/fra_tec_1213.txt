Éditer les champs
Ouvre une boîte de dialogue dans laquelle vous pouvez modifier les propriétés d'un champ.
Cliquez devant un champ, puis choisissez cette commande.
La boîte de dialogue comporte des boutons fléchés qui permettent de passer au champ suivant ou au champ précédent.
Vous pouvez aussi double-cliquer sur un champ dans le document pour l'ouvrir et le modifier.

Type de champ
Affiche le type du champ que vous modifiez.
Les éléments de boîte de dialogue suivants ne sont affichés que si le type de champ correspondant est sélectionné.
Sélectionner
Répertorie les options de champ, par exemple, &quot; fixe &quot; .
Si vous le souhaitez, cliquez sur une autre option pour le type de champ sélectionné.
Format
Sélectionnez le format à appliquer au contenu du champ.
Pour les champs de date, d'heure et d'utilisateur, vous pouvez également cliquer sur &quot; Autres formats &quot; dans la liste afin de sélectionner un format différent.
Les formats disponibles varient selon le type du champ édité.
Correction
Affiche la valeur de correction définie pour le type de champ sélectionné, par exemple, pour &quot; Page suivante &quot; , &quot; Numéro de page &quot; ou &quot; Page précédente &quot; .
Vous pouvez saisir une nouvelle valeur de correction qui sera ajoutée au numéro de page affiché.
