%PRODUCTNAME prend en charge les langages de script suivants :
%PRODUCTNAME Basic
JavaScript
BeanShell (Java)
Python (non inclus dans toutes les distributions)
Les développeurs peuvent en outre utiliser les langages de haut niveau, comme le langage de programmation Java, de façon à contrôler %PRODUCTNAME de manière externe.
