Instruction Kill [Exécution]
Supprime un fichier d'un disque.
Syntaxe :
Kill File As String
Paramètre :
File :
expression au format chaîne de caractères contenant une spécification de fichier non ambiguë.
Vous pouvez également utiliser la
notation URL
.
Exemple :
sub ExampleKill
Kill &quot; C:\datafile.dat &quot; REM Le fichier doit être créé à l'avance
end sub
