Options
Utilisez cette boîte de dialogue pour définir des options disponibles pour des types de diagrammes spécifiques.
Le contenu de la boîte de dialogue Options varie en fonction du type de diagramme.
Aligner la série de données sur :
Cette zone de groupe propose deux modes de graduation de l'axe Y.
Il n'est possible d'attribuer des propriétés aux axes (graduation et attributs) que séparément.
Axe Y primaire
Cette option est active par défaut.
Toutes les séries de données sont alignées sur l'axe Y primaire.
Axe Y secondaire
Modifie l'échelle de l'axe Y.
Cet axe n'est visible que lorsqu'une série de données au moins lui est assignée et l'affichage des axes actif.
Paramètres
Définissez dans cette zone les paramètres d'un diagramme en barres.
Les modifications s'appliquent à l'ensemble des séries de données du diagramme, et non pas uniquement aux données sélectionnées.
Écart
Définit l'espacement entre les colonnes en pourcentage.
L'espacement maximal est de 600 %.
Chevauchement
Définit les paramètres nécessaires au chevauchement des séries de données.
Vous avez le choix entre -100 et +100 %.
Lignes de connexion
Pour les diagrammes en colonnes &quot; empilées &quot; et &quot; pourcentage &quot; (barres verticales), vous devez cocher cette case afin de connecter les couches des colonnes associées aux lignes.
Affiche les barres côte à côte
Si deux axes sont affichés dans un diagramme en barres et que des séries de données sont attachées au premier axe, alors que d'autres séries de données sont attachées au second axe, alors les deux ensembles de séries de données sont affichés indépendamment, se chevauchant l'un l'autre.
Les barres provenant de différentes séries de données sont affichées comme si elles étaient attachées à un seul axe.
Dans le sens des aiguilles d'une montre
Direction dans le sens des aiguilles d'une montre
pour dessiner les morceaux dans la direction opposée.
Angle de départ
Glissez le petit point le long du cercle ou cliquez n'importe quelle position du cercle pour déterminer l'angle de départ d'un diagramme en secteurs ou en tore.
L'angle de départ est la position d'angle mathématique où le premier morceau est dessiné.
La valeur de 90 degrés dessine le premier morceau à la position de midi.
Une valeur de 0 degré débute à la position de 3 heures.
Dans les diagrammes 3D en secteurs ou en tore qui ont été créés avec des versions précédentes du logiciel, l'angle de départ est 0 degré au lieu de 90 degrés.
Pour les anciens et les nouveaux diagrammes 2D, l'angle de départ par défaut est de 90 degrés.
Lorsque vous modifiez l'angle de départ ou la direction, seule la version actuelle du logiciel affiche les valeurs modifiées.
Les versions précédentes du logiciel affichent le même document utilisant les valeurs par défaut : toujours une direction dans le sens inverse des aiguilles d'une montre et une valeur de départ à 90 degrés (pour les diagrammes 2D) ou à 0 degré (pour les diagrammes 3D en secteurs).
Degrés
Saisissez l'angle de départ entre 0 et 359 degrés.
Vous pouvez aussi cliquer sur les flèches pour modifier la valeur d'affichage.
Tracer les valeurs manquantes
Parfois, des valeurs sont absentes d'une série de données qui est affichée dans un diagramme.
Vous pouvez sélectionner, à partir de différentes options, comment tracer ces valeurs manquantes.
Les options ne sont disponibles que pour certains types de diagrammes.
~Laisser un espace
Pour une valeur manquante, aucune donnée ne sera affichée.
C'est le comportement par défaut des types de diagrammes en Colonne, Barre, Ligne, ou Toile.
~Supposer zéro
Pour une valeur manquante, la valeur y sera affichée comme zéro.
C'est le comportement par défaut d'un diagramme de type Zone.
~Continuer la courbe
Pour une valeur manquante, l'interpolation des valeurs voisines sera affichée.
C'est le comportement par défaut du diagramme de type XY.
Inclure les valeurs des cellules masquées
Cochez pour afficher également les valeurs des cellules actuellement masquées dans la plage de cellules source.
