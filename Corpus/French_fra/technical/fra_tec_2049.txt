Utilisation des Smart Tags
Les Smart Tags fournissent des informations complémentaires et des fonctionnalités permettant de spécifier les mots dans les documents Writer.
Les fonctionnalités disponibles peuvent être différentes suivant les extensions Smart Tags.
Installation des Smart Tags
extensions
de %PRODUCTNAME Writer.
Pour installer un Smart Tag, faites l'une des actions suivantes :
Outils - Gestionnaire des Extensions
pour ouvrir le gestionnaire des extensions, cliquez sur Ajouter et naviguez jusqu'au fichier.
Double cliquez sur le fichier *.oxt d'un Smart Tag lié à une page Web.
Cela nécessite un navigateur Web correctement configuré.
Menu Smart Tags
Outils - Options - %PRODUCTNAME - Apparence
.
Lorsque vous pointez sur un Smart Tag, une infobulle d'aide vous informe de faire un clic et d'appuyer sur la touche Ctrl pour ouvrir le menu des Smart Tags.
Si vous n'utilisez pas la souris, positionnez le curseur à l'intérieur du texte marqué et ouvrez le menu contextuel à l'aide de la combinaison Maj+F10.

Pour activer ou désactiver des Smart Tags
Smart Tags
dans
Outils - Options d'AutoCorrection
.
Utilisez cette boîte de dialogue pour activer ou désactiver les Smart Tags et pour gérer les balises installées.
Le texte qui est reconnu comme un Smart Tag n'est pas vérifié lors de la correction automatique.
