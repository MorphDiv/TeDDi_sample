Filtres pour appareils de poche
Vous pouvez ouvrir et enregistrer des documents dans différents formats de fichier pour équipements mobiles.
Les filtres suivants sont actuellement disponibles :
AportisDoc pour les appareils compatibles Palm OS
Pocket Excel pour les appareils compatibles Pocket PC
Pocket Word pour les appareils compatibles Pocket PC
En installant les filtres Pocket PC sous Windows, vous installez également un fichier DLL qui supporte le logiciel de synchronisation du Pocket PC.
Ces filtres pour mobiles nécessitent l'environnement d'exécution Java pour fonctionner.
