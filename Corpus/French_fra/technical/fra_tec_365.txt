Pour choisir un type de diagramme
type de diagramme
de base : cliquez sur n'importe quelle entrée étiquetée Colonne, Barre, Secteur, et ainsi de suite.
Le contenu sur le côté droit sera modifié pour offrir plus d'options en fonction du type de diagramme de base.
Facultativement, cliquez sur n'importe quelle option.
Pendant que vous modifiez les paramètres dans l'assistant, regardez l'aperçu dans le document pour voir comment le diagramme sera présenté.
Maj+F1
et pointez sur une commande pour voir un texte d'aide étendu.
Terminer
à partir de n'importe quelle page de l'assistant pour le fermer et créer le diagramme en utilisant les paramètres actuels.
Suivant
pour voir les pages suivantes de l'assistant ou cliquez sur les entrées sur le côté gauche de l'assistant pour aller à une page précise.
Précédent
pour accéder à la page précédente de l'assistant.
Annuler
pour fermer l'assistant sans créer de diagramme.
