KOnCD est un programme qui vous permets de créer des CD avec les programmes mkisofs et cdrecord.
Il peut créer des CD multisession, des CD amorçables et permet l'effacement des CD réinscriptibles.
Il peut également dupliquer les CD à la volée.

Installation
Graveur de CD ATAPI
Le protocole standart ATAPI décrit les méthodes permettant d'envoyer des commandes SCSI à travers l'IDE avec certaines limitations par rapport au "vrai" standart SCSI.
Pour cette raison, l'émulation ATAPI-SCSI est une méthode native supportant les périphériques ATAPI.
Exécutez 'make menuconfig'ou' make xconfig', choisissez alors load config from alternate file pour restaurez la configuration originale
Dans SCSI support validez SCSI support, SCSI CD-ROM support et SCSI generic support, le tout soit directement dans le kernel, soit en module
modprobe -k ide-scsi
cdrecord -scanbus
