Source de données comme table
Active une vue de table supplémentaire lorsque vous êtes en mode formulaire.
Lorsque la fonction
Source de données comme table
est activée, la table s'affiche dans une zone au-dessus du formulaire.
Icône
Source de données comme table
La vue de table et la vue de formulaire présentent les mêmes données.
Les modifications effectuées dans la table sont également visibles dans le formulaire et réciproquement.
S'il existe plusieurs formulaires logiques dans un document, la table ne peut en afficher qu'un seul à la fois.
