Ark est un gestionnaire d'archive pour KDE.
Introduction
Ark est une application destinée à gérer divers formats d'archives dans l'environnement KDE.
Les archives peuvent être visualisées, décompactées, créées et modifiées depuis Ark.
Le programme est compatible avec plusieurs formats comme tar, gzip, bzip2, zip et lha (si les programmes en ligne de commande appropriés sont installés).
Ark travaille de manière étroite avec Konqueror dans l'environnement KDE pour traiter les archives.
Utilisation de ark
Ouvrir des archives
Pour ouvrir une archive dans Ark, choisissez Ouvrir dans le menu Fichier.
Vous pouvez aussi ouvrir une archive en la déposant depuis Konqueror.
Les fichiers d'archives devraient être associés à Ark, donc vous pouvez cliquer avec le bouton droit de la souris sur un fichier dans Konqueror et sélectionner Archiveur pour l'ouvrir.
Travailler avec les fichiers
Afficher ouvrira le fichier dans le programme de visualisation par défaut.
Ouvrir avec... vous permet d'ouvrir le fichier avec le programme de votre choix.
Modifier avec... ouvrira le fichier avec le programme de votre choix.
La différence entre ceci et Ouvrir avec... est que toute modification effectuée sur le fichier sera enregistrée dans l'archive.
Supprimer enlèvera le( s) fichier( s) sélectionné( s) de l'archive.
Extraire les archives
Actuel extrait les fichiers récemment sélectionnés.
Si plusieurs fichiers ont été sélectionnés, seuls les fichiers sélectionnés en dernier seront extraits.
Tous extrait tout le contenu de l'archive.
Fichiers sélectionnés extrait tous les fichiers qui ont été sélectionnés.
Motif vous permet de spécifier quels fichiers seront extraits, correspondant à un certain motif, par exemple *.txt ou *.jpg.
Notez que vous ne pouvez utiliser qu'un motif à la fois.
Création d'une archive et ajout de fichiers
Pour créer une archive dans Ark, choisissez Nouveau dans le menu Fichier.
Vous pouvez alors saisir le nom de l'archive, avec l'extension appropriée (tar.gz, zip, bz2 etc.).
Pour ajouter des fichiers à l'archive, choisissez Ajouter un fichier... dans le menu Action.
Si vous voulez ajouter un dossier entier à une archive, sélectionnez Ajouter un dossier... dans le menu Action.
Une autre façon de procéder pour ajouter des fichiers à l'archive consiste à les déposer depuis Konqueror ou depuis le bureau dans la fenêtre principale de Ark, et ils seront ajoutés à l'archive actuelle.
Configurer Ark
Dossiers
Ceci affiche la boîte de dialogue Dossiers, afin de choisir les dossiers par défaut.
Le dernier dossier de démarrage est le dernier dossier que vous avez utilisé pour cette option particulière.
Liste des commandes
Le menu Fichier
Crée une nouvelle archive.
Ouvre une archive.
Fichier Récemment ouvert( s)
Affiche une liste des archives récentes pour en ouvrir une.
Fichier Enregistrer sous...
Vous permet d'enregistrer l'archive actuelle sous un nouveau nom.
Fichier Nouvelle fenêtre
Ouvre une nouvelle fenêtre de Ark.
Fichier Recharger
Recharge l'archive en cours.
Ceci rafraîchit la fenêtre de Ark si un autre programme a effectué des modifications sur l' archive
Fichier Fermer l' archive
Ferme l'archive actuelle.
Ferme Ark
Le menu Édition
Édition Sélectionner...
Vous permet de sélectionner des fichiers en fonction de leurs noms.
Par exemple, vous pouvez utiliser *.txt pour sélectionner tous les fichiers texte.
Notez que vous ne pouvez utiliser qu'un filtre à la fois.
Sélectionne tous les fichiers de l'archive.
Édition Désélectionner tout
Désélectionne tous les fichiers de l'archive.
Édition Inverser la sélection
Inverse la sélection des fichiers.
Les fichiers sélectionnés ne le sont plus, et ceux qui ne l'étaient pas le deviennent
Édition Afficher la sortie console
Affiche la boîte Sortie de la console, qui affiche la sortie de la ou des commandes les plus récentes exécutées par Ark.
C'est quelquefois utile si vous avez des problèmes que vous souhaitez résoudre.
Le menu Action
Action Ajouter un fichier
Affiche la boîte de dialogue Sélectionner un fichier à ajouter, où vous pouvez sélectionner des fichiers.
Maintenez enfoncée la touche Ctrl et cliquez pour sélectionner plusieurs fichiers.
Pour remonter dans le dossier parent, cliquez sur le bouton droit de la souris dans la boîte des dossiers, et choisissez Haut (ceci sera probablement amélioré bientôt).
L'onglet Avancé contient des options qui sont spécifiques au type de l'archive sur laquelle vous travaillez.
Action Ajouter un dossier...
Affiche la boîte de dialogue Ajouter un dossier... dans laquelle vous pouvez choisir un dossier à ajouter.
Sélectionnez simplement un dossier et choisissez Ouvrir.
Action Supprimer
Enlève de l'archive les fichiers actuellement sélectionnés.
Action Extraire...
Actuel extrait les fichiers sélectionnés en dernier.
Si plusieurs fichiers ont été sélectionnés, seul ceux sélectionnés en dernier seront extraits.
Tous extrait tout le contenu de l'archive.
Fichiers sélectionnés extrait tous les fichiers qui ont été sélectionnés.
Motif vous permet de spécifier quels fichiers seront extraits, en fonction d'un certain motif, par exemple *.txt ou *.jpg.
Notez que vous ne pouvez utiliser qu'un motif à la fois.
Action Afficher
Ouvre le fichier sélectionné dans le programme de visualisation associé.
Action Ouvrir avec...
Ouvre le fichier sélectionné avec le programme de votre choix.
La boîte de dialogue Ouvrir avec... vous permet de choisir le programme à utiliser.
Action Modifier avec...
Ouvre le fichier sélectionné avec le programme de votre choix.
Toutes les modifications qui seront effectuées avec le programme d'édition se répercuteront dans l'archive, puisque celle-ci sera mise à jour.
Le menu Configuration
Configuration Afficher la barre d' outils
Affiche ou non la barre d'outils.
Configuration Afficher la barre d' état
Affiche ou non la barre d'état.
Configuration Enregistrer la configuration
Enregistre les options de configuration actuelles.
Ceci signifie qu'elles seront réutilisées la prochaine fois que vous exécuterez Ark.
Configuration Configurer les raccourcis clavier...
Affiche la boîte de dialogue Configuration des raccourcis clavier.
Ceci vous permet de choisir les touches de raccourcis pour plusieurs éléments des menus.
Pour modifier un raccourci, sélectionnez une action dans la liste, puis choisissez la combinaison de touches à utiliser.
En cliquant sur la touche de clavier qui se situe en bas à droite, vous pouvez choisir la touche spécifique à utiliser.
Appuyez simplement sur une touche.
Configuration Configurer la barre d'outils...
Affiche une boîte de dialogue KDE standard où il vous est possible de configurer les icônes de la barre d'outils.
Configuration Configurer Ark...
Ceci ouvre la boîte de dialogue de configuration d'Ark.
Cette boîte de dialogue contient trois onglets.
La configuration de Ark est décrite dans la section Configuration.
Le menu Aide
F1 Aide Contenu
Invoque le système d'aide de KDE en démarrant avec le manuel de Ark (ce document).
Dessine un point d'interrogation (?) à côté du pointeur de la souris.
Cliquer sur un item dans Ark affiche alors une fenêtre d'aide (si celle-ci existe pour cet item particulier), expliquant la fonction de l'élément en question.
Aide Rapport de bogue
Ouvre une boîte de dialogue de rapport de bogue.
Vous pouvez alors envoyer un rapport de bogue ou émettre un souhait.
Affiche des informations sur la version et sur l' auteur
Aide À propos de KDE
Affiche la version de KDE et d'autres informations basiques

Installation
Comment obtenir Ark
Ark fait partie du projet KDE http ://www.kde.org/.
Ark se trouve dans le paquetage kdeutils à l'adresse ftp ://ftp.kde.org/pub/kde/, le site FTP principal du projet KDE.
Prérequis
Pour utiliser Ark, vous avez besoin de KDE 2.0.
GNU Tar v1.12 et une version récente de gzip sont aussi nécessaires.
Pour traiter d'autres formats d'archives, vous avez besoin des programmes en ligne de commande appropriés, comme par exemple zip, unzip, ar et lha.
