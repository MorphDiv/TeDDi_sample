JPEG & #160;: le format de fichier Joint Photographic Experts Group's est une bon format pour le web mais il utilise une compression avec pertes. PNG & #160;: le format Portable Network Graphics est un format de fichier extensible pour le stockage sans perte, portable pour les images raster. PNG fournit un brevet d'invention - remplaçant libre pour GIF et peut aussi remplacer beaucoup d'utilisations communes du format TIFF. PNG est conçu pour travailler aussi bien dans des applications d'aperçu en ligne, comme sur le World Wide Web, donc est utilisable en flux continu avec une option d'affichage progressif. PNG peut aussi stocker le gamma et des données chromatiques pour la gestion des couleurs améliorées correspondante sur des plates-formes hétérogènes.
