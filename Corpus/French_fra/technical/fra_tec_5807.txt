PowerDevil, un démon de gestionnaire d' énergie
Ce module est un outil de configuration pour gérer l'énergie dans & kde;. Il permet de configurer le démon PowerDevil, que vous pouvez démarrer ou arrêter depuis le module Gestionnaire de services.
Avec ce module, vous pouvez définir le comportement de votre ordinateur dans de nombreuses situations, vous permettant d'économiser autant d'énergie que possible.
Powerdevil fonctionne avec seulement avec Solid et est intégré dans votre environnement de travail. Vous ne serez pas prévenu qu'il est en marche sauf par des notifications. En outre, PowerDevil vérifie les capacités de votre système en premier, vous ne verrez donc que els options disponibles pour votre système dans ce module. Vous pouvez aussi avoir un meilleur aperçu de ces options sur la page Capacités.
Remarquez que PowerDevil ne surveille pas votre système. Cela signifie que le démon ne perdra pas de l'énergie en tentant d'en sauver, comme d'autres gestionnaires d'énergie le font.
Configuration générale
Vous pouvez configurer ici des options générales pour le module.
Configuration et profils
Verrouiller l'écran à la reprise
Si cette case est cochée, toutes les fois que l'ordinateur est mis en veille, l'écran sera verrouillé à la reprise.
Laisser PowerDevil gérer les économies d'énergie de l' écran
Si la case est cochée, PowerDevil prendra en charge les économies d'énergie de l'écran avec DPMS. Si vous voulez avoir différents délais pour les fonctionnalités de sauvegarde d'énergie de l'écran quand votre profil change, vous voulez probablement avoir cette option activée. Sinon, vous pouvez avoir des délais constants dans le module Affichagey.
Activer les notifications
Si la case est cochée, une notification apparaîtra dans une fenêtre contextuelle passive quand PowerDevil doit notifier un évènement, comme un verrouillage de l'écran ou une modification de profil.
Activer la notification des alertes
Si la case est cochée, une notification apparaîtra dans une fenêtre contextuelle passive quand PowerDevil déclenche un avertissement, comme un niveau de batterie faible ou un débranchement ou branchement d'une alimentation. Il est hautement recommandé de laisser cette option cochée.
Configurer les notifications
Si vous voulez un contrôle plus fin des notifications, vous pouvez les définir en cliquant sur ce bouton.
Avant de réaliser une mise en veille prolongée, attendre
Si cette case est cochée, quand une action de mise en veille prolongée est automatiquement déclenchée (par exemple, quand le système est inactif pendant un long moment ou qu'une paupière se ferme...) l'action ne sera pas réalisée immédiatement, mais après un délai de temps en secondes que vous spécifierez dans la boîte de saisie. Une notification sera affichée, et si celle -ci est cliquée, l'action sera annulée. Cette option est utile si vous voulez avoir la possibilité d'annuler la suspension automatique.

Modifier les profiles
Dans cette section, vous pouvez modifier, créer et supprimer les profils PowerDevil. La liste de gauche affiche tous les profils courants disponibles. Sélectionnez celui que vous voulez modifier, et personnalisez le avec les options que vous voyez sur la droite, décrites ci-dessous. Vous pouvez créer ou supprimer un profil existant grâce aux boutons que vous voyez en bas de la liste, ou vous pouvez même importer des profils depuis un fichier, ou exporter votre propre profil vers un fichier.
Il y a de nombreuses actions que vous pouvez configurer pour chaque profil, divisées en trois sections.
Actions
Désactiver la composition de KWin, si elle est active
La composition & kwin; sera désactivée, si celle -ci est activée. Elle sera réactivée quand un profil qui n'a pas cette option, sera chargé.
Réduire la luminosité lorsque le système est inactif depuis plus de
Si la case est cochée, l'affichage sera progressivement réduit, jusqu'à une luminosité de 0, dans un intervalle de temps que vous spécifiez dans la boîte de saisie.
Lorsque le système est inactif plus de
Quand vous avez spécifié l'intervalle de temps dans la boîte de saisie, l'action que vous sélectionnez dans la liste déroulante sera exécutée. Choisissez simplement « & #160; Ne rien faire & #160; » si vous voulez désactiver.
Lorsque l'écran du portable est rabattu
Définit l'action qui sera exécutée quand l'écran du portable est rabbatu.
Lorsque le bouton de mise en marche est enfoncé
Définit l'action qui sera exécutée quand le bouton de mise en marche est enfoncé.
Lorsque le bouton de mise en veille est enfoncé
Définit l'action qui sera exécutée quand le bouton de mise en veille est enfoncé.

Activer la gestion de l'énergie de l' écran
Vous pouvez définir ici la gestion de l'énergie pour votre écran. Cette section a besoin de la gestion DPMS, vous pouvez vérifier si votre système le gère dans la page des possibilités.
La case à cocher définit si la gestion du DPMS doit être activée ou désactivée. Si vous choisissez de l'activer, réglez l'intervalle de temps dans les boîtes de saisie présentes en dessous. Vous pouvez aussi empêcher l'écran de déclencher un certain état en décochant les cases correspondantes.

Possibilités
Dans cette section vous pouvez regarder les possibilités de votre système. Si l'état n'indique aucun problème, votre configuration fonctionne correctement et PowerDevil tourne avec les meilleures options possibles. Si des problèmes sont trouvés, un avertissement sera affiché avec une solution si possible.
PowerDevil est habituellement capable de résoudre le problème de lui même. Dans la plupart des cas vous aurez besoin de cliquer sur le bouton suggéré, même si quelque fois des problèmes sont le résultat d'une compilation de l'application sans la prise en charge d'un composant spécifique. Dans ce cas, PowerDevil ne fournira pas de solution mais vous alertera simplement à propos du problème.
Lanceur PowerDevil
PowerDevil peut être manuellement contrôlé par & krunner; (Alt; F2). Cela rend la modification des profils, de la luminosité, de la gestion et plus encore plus facile et rapide. Le lanceur a de nombreux mots clé que vous pouvez saisir pour accéder à ces fonctionnalités.
Les mots clés du lanceur
Voici une liste de mots clé que vous pouvez saisir dans & krunner; pour réaliser les actions relatives à PowerDevil.
Power Profile
Vous pouvez manuellement choisir un profil en utilisant ce mot clé. Tous les profils seront affichés dans le formulaire des correspondances de & krunner;.
Power Governor
Ce mot clé vous laisse manuellement définir la gestion de la fréquence du processeur. Toutes les actions de gestion seront affichées dans les correspondances de & krunner;.
Power Scheme
Vous laisse définir manuellement le modèle de gestion de l'énergie. Les modèles disponibles seront affichés dans les correspondances de & krunner;
Screen Brightness
Vous pouvez contrôler la luminosité de l'écran avec ce mot clé. Des correspondances dans & krunner; qui vous permettent de réaliser des actions communes seront affichées. Si vous saisissez une valeur après le mot clé (pex; & #160;: Screen Brightness 50), la luminosité sera définie à cette valeur.
Suspend
Vous pouvez mettre en veille votre ordinateur avec ce mot clé. Les méthodes de mise en veille disponibles seront affichées dans les correspondances de & krunner;.
