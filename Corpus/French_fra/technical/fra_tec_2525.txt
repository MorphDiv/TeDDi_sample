Gestionnaire de fichiers
Ce module vous permet de configurer les différentes caractéristiques du gestionnaire de fichiers de Konqueror.
Les fonctionnalités de navigateur web de Konqueror ont leur propre module de configuration.
Utilisation
Comportement
Cet onglet contient plusieurs options globales de Konqueror
Apparence
Cet onglet contient les options de personnalisation de l'apparence des fenêtres de Konqueror, comme par exemple la couleur du texte par défaut.
Corbeille
Les options de cet onglet permettent de définir le comportement par défaut de Konqueror lorsque vous supprimez un fichier.
Aperçus
Sur cet onglet, vous pouvez personnaliser quand Konqueror doit montrer les aperçus.
Comportement
Ouvrir les dossiers dans des fenêtres séparées
Si cette option est sélectionnée, une nouvelle fenêtre Konqueror sera créée quand vous ouvrez un dossier, plutôt que d'afficher simplement le contenu du dossier dans la fenêtre actuelle.
Afficher les opérations sur le réseau dans une fenêtre unique
Si elle est désactivée lorsque que vous procédez à un téléchargement, une boîte de dialogue apparaît, vous informant sur l'état et la progression du transfert.
Boîte de dialogue de téléchargement - Individuelle
Si vous commencez un nouveau transfert, une autre boîte de dialogue apparaît (les utilisateurs de Netscape sont habitués à ce comportement).
Boîte de dialogue de téléchargement - Commune
Afficher les informations sur les fichiers
Ici, vous pouvez décider si, lorsque la souris passe sur un fichier, vous voulez voir une petite fenêtre surgissante, donnant des informations supplémentaires.
Nombre d'entrées d'informations sur les fichiers
Ici, vous pouvez fixer combien d'items d'informations sont affichés.
URL de démarrage
Ceci vous permet de choisir l'URL (&pex; un dossier ou une page web) sur laquelle s'ouvre Konqueror lorsque vous cliquez sur le bouton Home.
Par défaut, c'est celle du dossier personnel de l'utilisateur, indiqué par un ~.
Minimiser l'utilisation de la mémoire
Jamais
Désactive le minimisation de l'utilisation de mémoire et vous permet de rendre chaque activité de navigation indépendante des autres.
Konqueror n'essayera jamais de réduire son utilisation de mémoire.
Si vous avez beaucoup de RAM, cela pourrait accélérer Konqueror.
Si vous n'avez pas assez de RAM, cela pourrait ralentir fortement Konqueror, parce que plus d'informations seront échangées avec le fichier d'échange.
Pour l'exploration locale uniquement (recommandé)
Konqueror essayera de minimiser l'utilisation de la mémoire lorsque vous parcourez des systèmes de fichiers locaux, y compris des systèmes de fichiers réseau.
Pour la navigation web seulement
Konqueror essayera de minimiser l'utilisation de la mémoire lorsque vous naviguez sur le web.
Toujours (utilisez avec précaution)
Ce réglage n'est pas conseillé, et ne devrait être utilisé uniquement si vos ressources systèmes sont très limitées.
Un mot sur la minimisation de l'utilisation de mémoire dans Konqueror
Avec cette option activée, seule une instance de Konqueror existera dans la mémoire de votre ordinateur à tout moment, sans tenir compte du nombre de fenêtres ouvertes, réduisant ainsi les besoins en ressources.
Soyez averti que cela veut aussi dire que si quelque chose tourne mal, toutes les fenêtres de navigation se fermeront en même temps.
Apparence
Les réglages suivants déterminent comment sont affichés le texte et les couleurs dans les fenêtres du mode gestionnaire de fichiers de Konqueror.
Police standard
C'est la police utilisée pour afficher le texte comme les noms de fichiers dans les fenêtres de Konqueror.
Taille des polices
Vous permet de préciser la taille du texte.
La police de caractère employée peut être choisie grâce au menu déroulant Police standard.
Couleur normale de texte
Détermine la Couleur du texte standard.
Retour à la ligne automatique
Lorsqu'elle est cochée, cette option permet d'afficher sur plusieurs lignes les noms de fichiers très longs, évitant ainsi de les tronquer.
Si cette option n'est pas cochée, on pourra néanmoins voir l'intégralité d'un nom de fichier tronqué en plaçant le curseur de la souris sur l'icône correspondante.
Noms de fichiers soulignés
Afficher les tailles des fichiers en octets
La dernière case à cocher de cette page détermine comment Konqueror affiche les tailles des fichiers, si vous êtes dans l'un des modes gestionnaire de fichiers affichant cette information.
Si vous cochez la case à cocher Afficher la taille en octets, les tailles de fichiers seront toujours affichées en octets.
Si vous la laissez décochée, les tailles de fichiers seront affichées en octets, kilooctets ou Mégaoctets, en fonction de leur taille.
Corbeille
La zone Demander confirmation pour détermine la manière de réagir de Konqueror quand vous décidez de Mettre à la corbeille, Supprimer, ou Broyer un fichier.
Si cette option est cochée, Konqueror demande confirmation avant d'effectuer l'action correspondante.
Aperçus
si vous activez les aperçus pour un de ceux-ci, les fichiers ce de type ne seront pas représentés par des icônes standards dans le gestionnaire de fichiers mais seront affichées en miniatures.
Sur les ordinateurs lents, les aperçus peuvent être lents à s'afficher, donc vous pourriez désactiver cette fonction.
Pour chaque protocole, vous pouvez indépendamment activer ou désactiver les aperçus.
Par exemple, vous pouvez activer les aperçus pour les disques montés par NFS si vous avez un réseau rapide, mais les désactiver pour le FTP du fait d'une connexion Internet lente par modem.
Vous pouvez activer et désactiver les aperçus au vol depuis Konqueror.
Par exemple, si vous voulez normalement voir les aperçus sur chaque protocole, mais trouvez un serveur en particulier qui est très lent à vous envoyer les données, vous pouvez les désactiver pour le moment, et les réactiver plus tard.
