Index
Permet d'organiser les index de base de données dBase.
Un index vous permet d'accéder rapidement à une base de données, à condition que vous demandiez des données dans la sélection définie via l'index.
Lorsque vous créez une table, vous pouvez définir les index dans l'onglet

.
Table
Sélectionnez la table de base de données à indexer.
Index de table
Répertorie les index actifs de la table de base de données sélectionnée.
Pour supprimer un index de la liste, cliquez sur cet index, puis sur la flèche vers la droite.
Index libres
Dresse la liste des index disponibles pour être assignés à une table.
Pour assigner un index à une table sélectionnée, cliquez sur l'icône représentant une flèche vers la gauche.
La double flèche vers la gauche assigne tous les index disponibles.
