Commencer un nouveau document à partir d'un modèle
Pour commencer un nouveau document basé sur un modèle, vous devez d'abord choisir votre modèle.
Rappelez -vous que les modèles sont soit orientés texte, soit disposition.
Si vous avez besoin de revoir les différences, cliquez ici.
Vous pouvez utiliser les onglets libellés Orienté Texte et Disposition pour afficher tous les modèles disponibles de ce groupe.
Une fois que vous êtes sur l'onglet convenable, on vous affiche tous les modèles disponibles, chacun avec un titre et une petite icône qui vous affiche la disposition générale du modèle.
KWord est fourni avec deux groupes de modèles standards.
Vous pouvez ajouter de nouveaux onglets en installant des modèles d'autre provenance ou créer vos propres modèles.
Pour sélectionner votre modèle, cliquez dessus avec le bouton gauche de la souris.
Le modèle sélectionné sera mis en surbrillance.
Confirmez votre choix en cliquant sur le bouton OK.
Ceci commencera un nouveau document avec ce modèle.
Pour un accès plus rapide à un modèle, double-cliquez simplement dessus et KWord chargera immédiatement ce modèle.

Ouvrir un document récent
KWord garde une trace des documents récemment édités.
Vous pouvez sélectionner un de ces documents en utilisant la boîte déroulante à droite du bouton radio libellé Ouvrir un document récent.
Une fois que le nom du fichier est sélectionné dans cette boîte déroulante, cliquez sur OK et KWord chargera le modèle.

Enregistrer un document
Une fois que vous avez saisi du texte et des données dans un document, vous voudrez habituellement enregistrer ceci dans un fichier sur votre disque dur.
La commande Enregistrer
ou en cliquant sur dans la barre d'outils.
N'importe laquelle de ces méthodes provoque la même action de KWord.
En sélectionnant la commande Enregistrer, vous dites à KWord d'enregistrer le fichier sous le nom de fichier courant.
On ne vous donnera pas la possibilité de changer le nom du fichier ou son emplacement.
Si vous n'avez pas enregistré ce fichier auparavant, il n'a pas de nom.
KWord exécute automatiquement la commande Enregistrer sous... de telle manière que vous pouvez donner un nom au fichier.
Vous pouvez vérifier que le fichier a été enregistré en vérifiant la barre de titre de KWord.
S'il y a des changements non enregistrés, la barre de titre affichera [modifié].
Ce document a toujours des changements non enregistrés
Si le fichier a été enregistré sans incident, seul le nom du fichier apparaîtra dans la barre de titres.
S'il y a eu un problème avec l'enregistrement, une boîte d'erreur apparaîtra.
Enregistrement réussi
La commande Enregistrer sous...
Général
ou en essayant d'Enregistrer un fichier qui n'a pas encore de nom.
La différence entre les commandes Enregistrer et Enregistrer sous... est que la commande Enregistrer sous... vous demande un nom de fichier, et vous permet de sélectionner un format de fichier ou un emplacement différent.
La commande Enregistrer enregistre simplement le fichier dans l'emplacement précédent.
Utilisation de la boîte de dialogue
Quand vous sélectionnez la fonction Enregistrer sous..., un dialogue apparaît.
La zone de texte étiquetée filtre montre que nous avons l'intention de l'enregistrer comme un fichier KWord.
En utilisant KWord vous pouvez enregistrer le fichier sous d'autres formats.
Pour des informations sur d'autres fichiers, voyez la section Filtres.
Il y a un bouton Annuler.
Si vous cliquez sur ce bouton, l'enregistrement sera interrompu et vous retournez en mode édition du document.
Il y a un bouton OK, qui sera utilisé quand vous aurez sélectionné l'emplacement correct, le format correct et le nom de fichier.
Un clic sur ce bouton enregistre le fichier.
En utilisant cette boîte de dialogue, vous pouvez vous déplacer dans l'arborescence des dossiers pour y enregistrer votre document.
Pour entrer dans un dossier, cliquez sur ce dossier.
Pour sortir de ce dossier, cliquez sur le bouton en flèche bleue dirigée vers le haut.
Quand vous êtes dans le dossier correct, vous pouvez saisir un nom dans la boîte Emplacement.
Il n'y a pas besoin de mettre un .kwd à la fin du nom du fichier, KWord fera ceci pour vous.

Utilisation de la boîte de dialogue
Une autre fonction de la boîte libellée Filtre est d'agir comme un masque de fichiers.
Un masque de fichiers cache tous les fichiers que KWord croit sans importance pour votre tâche en cours.
Quand le type de fichier de KWord est positionné sur les fichiers KWord, la boîte de dialogue n'affiche que les dossiers et les fichiers se terminant par .kwd.
S'il dit Tous les fichiers supportés, tous les fichiers seront affichés.
Pour entrer dans un dossier, vous double-cliquez sur ce dossier.
Pour sortir de ce dossier, cliquez sur le bouton en flèche bleue dirigée vers le haut.
Il y a un bouton OK, qui sera utilisé quand nous aurons sélectionné le nom correct du fichier.
Imprimer un document
ou en cliquant sur dans la barre d'outils.
Peu importe comment vous faites ceci, la boîte de dialogue d'impression de KDE apparaîtra.
La boîte déroulante du haut libellée Nom affiche l'imprimante actuellement sélectionnée.
Pour configurer cette imprimante, cliquez sur le bouton Propriétés.
Si vous voulez imprimer votre sortie dans un fichier postscript, dans un fichier pdf, dans un fax (pour l'envoyer en fax) ou envoyer en courrier électronique un fichier pdf, sélectionnez l'option appropriée de la boîte déroulante.
Une fois que l'option a été sélectionnée, saisissez le nom de fichier dans la boîte de texte libellée Fichier de sortie.
Si vous voulez imprimer toutes les pages de votre document en seulement un exemplaire, vous pouvez cliquer sur OK.
Si vous voulez changer le nombre de copies ou n'imprimer que les pages sélectionnées, cliquez sur Développer.
En fonction du système d'impression que votre ordinateur utilise, vous pouvez trouver que vous avez plus d'options d'impression que celles décrites ci-dessous.
Ce qui est décrit ici est disponible sur tous les systèmes.
Dans la boîte déroulante libellée Ensemble de pages, vous pouvez dire à KWord d'imprimer les pages paires, les pages impaires ou toutes les pages.
Vous pouvez faire que KDE assemble vos documents en cliquant sur la case à cocher Assembler.
Si cochée, KDE imprimera toutes les pages pour l'exemplaire 1, puis imprimera toutes les pages pour l'exemplaire 2, etc..
Si non cochée, KDE imprimera toutes les copies de la première page, puis toutes les copies de la deuxième page, etc.
Si vous cliquez sur Refermer, la boîte de dialogue se repliera à son état initial.
Si vous cliquez sur Annuler, l'impression sera annulée.
