Utilisation de documents XForms
Dans %PRODUCTNAME, les documents XForms constituent un type particulier de documents Writer.
Le mode Conception des documents XForms propose des barres d'outils et des volets supplémentaires.
Une fois le document XForms créé et enregistré, vous pouvez ouvrir le document, remplir le formulaire et envoyer les modifications à un serveur.
What's the role of UBL?
We do not officially support exporting to XHTML - but should be possible using XSLT
Pour créer un nouveau document XForms
Fichier - Nouveau - Formulaire XML
.
La fenêtre de conception XForms s'ouvre dans un document Writer vide.
Concevez le formulaire.
Insérez un contrôle, sélectionnez le modèle par défaut dans l'explorateur de propriétés et saisissez une expression de liaison.
Dans le navigateur de données, ajoutez un élément à l'instance.
Chargez une nouvelle instance à partir du fichier XML et ajoutez des contrôles aux éléments ou attributs XML appropriés.
Pour ouvrir un document XForms
Fichier - Ouvrir
et sélectionnez un document XForms.
Les documents XForms possèdent la même extension que les documents texte Writer (*.odt).
