Couleurs
Introduction
La prévisualisation du modèle de couleurs.
La glissière de contraste.
Les modèles de couleurs disponibles.
Un Widget est un terme habituel des programmeurs qui se rapporte à des éléments de l'interface utilisateur, comme les boutons, les menus ou les ascenseurs.
Vous pouvez les considérer comme les éléments fondamentaux que l'on assemble pour construire votre application.
Prévisualisation
Cette section de la boîte de dialogue montre comment un modèle de couleur s'applique sur un échantillonnage de widgets.
Elle vous fournit une prévisualisation de votre choix de couleurs.
Un modèle de couleurs consiste en un lot de 18 couleurs.
Chaque widget est étiqueté pour vous aider à vous faire une idée de la manière dont les changements retentiront sur le modèle de couleurs.
Couleur des widgets
Vous pouvez sélectionner n'importe quel élément du modèle de couleurs dans la boîte de combinaison des couleurs.
Glissière de Contraste
La glissière de Contraste vous permet de changer le contraste entre différents dégradés des couleurs choisies.
La zone de prévisualisation s'actualise en temps réel pour montrer l'effet de vos changements.
Modèles de couleurs
Vous pouvez utiliser les modèles de couleurs d'origine de KDE comme point de départ pour créer votre propre modèle.
Cliquez sur le bouton Ajouter pour enregistrer le modèle de couleurs sous un nouveau nom, puis modifiez -le.
Les modèles de couleurs intégrés d'origine à KDE ne peuvent ni être modifiés, ni être supprimés.
