http
http signifie H yper T ext T ransfer P rotocol (protocole de transfert hypertexte).
Le module d'entrées-sorties http est utilisé par toutes les applications KDE pour gérer les connexions vers les serveurs http, c'est à dire les serveurs web.
L'usage le plus courant est pour afficher les pages web dans le navigateur Konqueror.
Voir le manuel de http.
