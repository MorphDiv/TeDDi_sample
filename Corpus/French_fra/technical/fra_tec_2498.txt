KJumpingCube est un jeu tactique simple auquel vous pouvez jouer contre un ami ou contre l'ordinateur.
Introduction
KJumpingCube est un jeu tactique simple.
Vous pouvez y jouer contre l'ordinateur ou contre un ami.
Le terrain de jeu est formé de cases qui contiennent des points.
En cliquant sur les cases vous augmentez le nombre de points, et si ce nombre de points atteint le nombre maximal de points sur une case, ces points sauteront sur les cases voisines et prendront ces cases.
Le gagnant est celui qui contrôle toutes les cases.
Jouer à KJumpingCube
Règles
Un tour de jeu consiste à augmenter le nombre de points sur une case en cliquant dessus.
Vous ne pouvez déposer des points que sur une case qui n'appartient pas à votre adversaire.
En cliquant sur une case n'appartenant à personne, vous en prenez possession.
Si une case a plus de points qu'elle n'a de cases voisines, alors les points sautent sur ces cases voisines et en prennent le contrôle.
Les cases voisines sont les voisins directs, pas les cases voisines en diagonale.
Le gagnant est celui qui possède toutes les cases.

Questions, réponses et astuces
Astuces tactiques
Essayez d'éviter d'augmenter le nombre de pions sur une case lorsque votre adversaire possède une case voisine qui pourra atteindre son maximum avant votre case.

Installation
Comment obtenir KJumpingCube
KJumpingCube fait partie du projet KDE http ://www.kde.org/.
KJumpingCube se trouve dans le paquetage kdegames à l'adresse ftp ://ftp.kde.org/pub/kde/, le site FTP principal du projet KDE.

Pour compiler avec succès KJumpingCube, vous aurez besoin de KDE 2.0.
