Référence des commandes
Le menu Fichier
Ctrl; N Fichier Nouveau...
Crée un nouveau projet. N'affecte pas un projet déjà ouvert.
Ctrl; O Fichier Ouvrir...
Ouvre un projet déjà existant. N'affecte pas un projet déjà ouvert.
Fichier Télécharger les bases de données exemples...
Ouvrir la fenêtre KnewStuff, qui vous permet de télécharger des exemples de bases de données par Internet. Cela n'est pour l'instant pas disponible pour MS Windows.
Ctrl; S Fichier Enregistrer
Enregistre les changements apportés aux objets de la fenêtre active.
Fichier Importer Données de table à partir d'un fichier...
Importe les données d'une table depuis un fichier au format CSV (Comma Separated Value, c'est-à-dire Valeurs séparées par des virgules).
Fichier Exporter Table ou données de requête en tant que fichier...
Exporte les données de la table active ou les données d'une requête dans un fichier au format CSV (Comma Separated Value, c'est-à-dire Valeurs séparées par des virgules).
Ctrl; P Fichier Imprimer...
Imprime les données provenant de la table ou requête active (ouverte).
Note pour KDE & #160;: Assurez -vous que le bon système d'impression est sélectionné dans la section Système d'impression utilisé & #160;:. Cette option peut être affichée en cliquant sur le bouton Options > >.
Fichier Aperçu avant impression...
Affiche un aperçu avant impression pour la table ou la requête active (ouverte).
Fichier Configuration de la page...
Régler la police... pour le Titre de la page & #160;:, Modifier... les taille de la page et marges et Ajouter une bordure à la table.
Fichier Configuration de la page...
Ferme le projet en cours mais laisse & kexi; lancé.
Ctrl; Q Fichier Quitter
Quitte & kexi;.
Le menu Édition
Ctrl; Z Édition Annuler
Annule une action. Permet de remettre la base de données dans l'état où elle était avant votre dernier changement.
Ctrl; & Maj; Z Édition Refaire
Annule une annulation. Provoque un effet inverse à celui d'Annuler. Cela remet en place la modification que vous aviez faite à l'origine.
Ctrl; X Édition Couper
Supprime le texte sélectionné et le place dans le presse-papier. Cette commande n'est disponible que s'il y a du texte sélectionné. Si cette action est utilisée dans le Concepteur de formulaire, l'élément ou le groupe d'éléments sélectionné est supprimé et placé dans le presse-papier.
Ctrl; C Édition Copier
Copie le texte sélectionné dans le presse-papier. Cette commande n'est disponible que s'il y a du texte sélectionné. Si cette action est utilisée dans le Concepteur de formulaire, l'élément ou le groupe d'éléments sélectionné est copié dans le presse-papier.
Ctrl; V Édition Coller
Copie le contenu du presse-papier à l'endroit d'insertion. Cette commande n'est pas disponible si le presse-papier est vide. Si cette action est utilisée dans le Concepteur de formulaire et le presse-papier contient des éléments copiés, ces éléments vont être insérés dans le formulaire.
Édition Copie spéciale Table ou requête en tant que données de table...
Copie la table ou les données de la requête dans le presse-papier.
Édition Collage spécial Comme données de table...
Copie les données du presse-papier dans une nouvelle table à l'intérieur du projet actuel.
Ctrl; A Édition Tout sélectionner
Sélectionne tous les caractères de la zone de texte éditée ou tous les éléments dans le Concepteur de formulaire.
Édition Supprimer
Supprime l'objet sélectionné.
Ctrl; Suppr. Édition Supprimer la ligne
Supprime la ligne actuellement sélectionnée d'une table.
Le menu Affichage
F6 Affichage Mode données
Passe en Mode données.
F7 Affichage Mode conception
Passe en Mode conception
F8 Affichage Mode texte
Passe en mode Texte. Cela n'est possible que pour les requêtes pour l'instant, et cela signifie passer dans le mode SQL du Concepteur de requête.
Alt; 1 Affichage Navigateur de projet
Va au panneau du Navigateur de projet.
Alt; 2 Affichage Zone principale
Va à la zone principale.
Alt; 3 Affichage Éditeur de propriétés
Va à l'Éditeur de propriété.
Affichage Afficher/ Masquer & #160;: Propriétés
Affiche ou masque le panneau de l'Éditeur de propriétés.
Affichage Afficher/ Masquer & #160;: Navigateur de projet
Affiche ou masque le Navigateur de projet.
Le menu Insertion
Insertion Table...
Insère une nouvelle table vide, sans l'enregistrer. La fenêtre du concepteur de table apparaît.
Insertion Requête...
Insère une nouvelle requête vide, sans l'enregistrer. La fenêtre du concepteur de requête apparaît.
Insertion Formulaire...
Insère un nouveau formulaire vide, sans l'enregistrer. La fenêtre du concepteur de formulaire apparaît.
Insertion Script...
Insère un nouveau script vide, sans l'enregistrer. La fenêtre de l'éditeur de script apparaît. Cette commande n'est disponible que si le scriptage de & kexi; a été autorisé.
Le menu Format
Format Police...
Change la police de l'objet sélectionné. Cela peut être utilisé dans le concepteur de formulaire pour choisir la police d'un élément.
Format Aligner sur la grille
Si cette option est activée, lorsque vous déplacez des éléments sur la surface du formulaire, le coin supérieur gauche de l'élément va s'aligner sur le point de la grille le plus proche. Cela réduit votre liberté pour placer les éléments sur la surface du formulaire mais cela vous aide à les aligner précisément.
Format Éléments de disposition
Crée une nouvelle disposition pour les éléments. Les éléments peuvent être disposés horizontalement, verticalement, dans la grille, horizontalement dans un séparateur, verticalement dans un séparateur.
Format Casser la disposition
Casse la disposition sélectionnée.
Format Aligner la position des éléments
Aligne la position des éléments sélectionnés & #160;: vers la gauche, vers la droite, vers le haut, vers le bas, sur la grille.
Format Aligner la taille des éléments
Aligne la taille des éléments sélectionnés & #160;: adapter, sur la grille, pour le plus petit, pour le plus grand, pour le plus étroit, pour le plus large.
Format Amener l'élément au premier plan
Amène les éléments sélectionnés au premier plan.
Format Envoyer l'élément en arrière
Envoie les éléments sélectionnés en arrière plan.
Le menu Données
Maj; Entrée Données Enregistrer la ligne
Enregistre les données de la ligne de la table sélectionnée.
Données Annuler les changements sur la ligne
Annule les modifications effectuées sur la ligne de la table sélectionnée.
Données Tri Croissant
Trie les données dans l'ordre croissant (de A à Z et de 0 à 9). Ce sont les données de la colonne active qui sont triées.
Données Tri Décroissant
Trie les données dans l'ordre décroissant (de Z à A et de 9 à 0). Ce sont les données de la colonne active qui sont triées.
Le menu Outils
Outils Importer une base de données...
Ouvre l'assistant d'installation de base de données pour importer une base de données existante dans une base de données & kexi;.
Outils Exécuter un fichier script...
Affiche la boîte de dialogue permettant d'ouvrir un fichier de script existant.
Outils Gestionnaire de scripts...
Affiche la boîte de dialogue du Gestionnaire de scripts de & kexi; pour exécuter, charger, décharger, installer, désinstaller ou télécharger des scripts.
Outils Scripts
Exécute un script déjà chargé.
Le menu Fenêtre
Ctrl; W Fenêtre Fermer
Ferme la fenêtre active.
Fenêtre Tout fermer
Ferme toutes les fenêtres ouvertes.
Fenêtre Mode MDI Mode de cadre enfant
Passe l'interface utilisateur en mode de cadre enfant.
Fenêtre Mode MDI Mode IDEAI
Passe l'interface utilisateur en mode IDEAI.
Alt; Droite Fenêtre Fenêtre suivante
Passe à la fenêtre suivante.
Alt; Droite Fenêtre Fenêtre précédente
Passe à la fenêtre précédente.
Les derniers éléments de ce menu montrent les noms des fenêtres actuellement ouvertes.
Le menu Configuration
Configuration Barres d' outils
Montre ou cache une des barres d'outils.
Configuration Configurer les raccourcis clavier...
Configure les raccourcis claviers utilisés par & kexi;. Voyez la section sur la configuration des raccourcis claviers pour plus d'informations.
