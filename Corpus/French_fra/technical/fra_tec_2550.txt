Information sur les partitions
Cette page affiche les informations sur les partitions de vos disques durs.
L'information exacte affichée dépend du système.
Sur certains systèmes, les informations sur les partitions ne peuvent pas être affichées.
Sur Linux, cette information est lue depuis /proc/partitions, qui n'est accessible que si le pseudo-système de fichiers /proc est compilé dans le noyau (2.1.x ou supérieur).
L'utilisateur ne peut modifier aucun réglage sur cette page.
