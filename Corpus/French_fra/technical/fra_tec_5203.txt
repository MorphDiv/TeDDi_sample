Le module externe de galerie d' image
Utilisation du module externe de galerie d' image
Vous accédez au module externe de galerie d'image à partir de la barre de menus de & konqueror;, dans Outils Créer une galerie d'images..., ou avec Ctrl; I.
Quand vous lancez le module externe de galerie d'image, une boîte de dialogue s'ouvre pour vous permettre de préciser comment la galerie sera créée.
Apparence de la page
Capture d'écran de la boîte de dialogue d'apparence de la page
Capture d' écran
La boîte de dialogue d'apparence sert à régler l'apparence de la galerie.
Titre de la page
Cette option permet de modifier le titre de la galerie d'image.
Nombre d'images par ligne
Cette option permet d'ajuster le nombre d'aperçus affichés par ligne dans la galerie.
Afficher le nom du fichier image
Cette option permet d'afficher (ou pas) le nom du fichier image sous les aperçus de la galerie.
Afficher la taille du fichier image
Cette option permet d'afficher (ou pas) la taille du fichier image sous les aperçus de la galerie.
Afficher les dimensions de l' image
Cette option permet d'afficher (ou pas) les dimensions de l'image en pixels sous les aperçus de la galerie.
Nom de la police
Cette option permet de choisir la police à utiliser dans la galerie.
Taille de la police
Cette option permet de modifier la taille de la police sélectionnée.
Couleur du texte
Cette option permet de choisir la couleur du texte affiché dans la galerie.
Couleur du fond
Cette option permet de choisir la couleur de fond utilisée dans la galerie.
Dossiers
Capture d'écran de la boîte de dialogue des dossiers
La boîte de dialogue des dossiers sert à régler où enregistrer la galerie ainsi que les dossiers utilisés.
Enregistrer dans un fichier HTML
Cette option permet de choisir où enregistrer la galerie d'image sous forme d'une page html.
Parcourir les sous-dossiers
Cette option permet de choisir s'il il faut parcourir les sous-dossiers. Si cette option est activée, une glissière et un compteur vous permettent de spécifier le nombre de niveau à parcourir.
Copier les fichiers originaux
Cette option permet de choisir si la galerie utilisera un dossier contenant une copie des images originales.
Utiliser un fichier de commentaires
Cette option permet d'utiliser un fichier de commentaires à utiliser avec la galerie.
Aperçus
Capture d'écran de la boîte de dialogue des aperçus
La boîte de dialogue des aperçus sert à régler les propriétés des aperçus dans la galerie.
Format des images pour les vignettes
Cette option permet de choisir le format utilisé pour les vignettes.
Taille des aperçus
Cette option permet de préciser la taille des aperçus utilisée dans la galerie.
Déterminer une profondeur de couleurs différente
Cette option permet de déterminer une profondeur de couleurs différente pour les aperçus si vous voulez la modifier par rapport à l'image d'origine.
