Assistant Diagramme - éléments du diagramme
assistant Diagramme
vous pouvez choisir les éléments du diagramme devant être affichés.
titles title
Saisissez un titre pour le diagramme.
subtitle
Saisissez un sous-titre pour le diagramme.
x axis
Saisissez une étiquette pour l'axe x (horizontal).
y axisx axis
Saisissez une étiquette pour l'axe y (vertical).
z axis
Saisissez une étiquette pour l'axe z.
Cette option est disponible uniquement pour les diagrammes tridimensionnels.
legend
Affiche une légende dans le diagramme.
legend left
Positionne la légende à la gauche du diagramme.
top
Positionne la légende en bas du diagramme.
right
Positionne la légende à la droite du diagramme.
bottom
Positionne la légende en haut du diagramme.
grid x axis
Affiche les lignes de la grille perpendiculaires à l'axe x.
y axis
Affiche les lignes de la grille perpendiculaires à l'axe y.
z axis
Affiche les lignes de la grille perpendiculaires à l'axe z.
Cette option est disponible uniquement pour les diagrammes tridimensionnels.
secondary x axis ( &quot; 2nd x axis &quot; in spec doc)
Saisissez une étiquette pour l'axe x secondaire.
Cette option est uniquement disponible pour les diagrammes qui supportent un axe x secondaire.
secondary y axis ( &quot; 2nd y axis &quot; in spec doc)
Saisissez une étiquette pour l'axe y secondaire.
Cette option est disponible uniquement pour les diagrammes qui supportent un axe y secondaire.
Pour saisir les éléments du diagramme
Saisissez les titres ou cliquez sur les éléments que vous souhaitez voir affichés dans le diagramme actif.
Titres
Si vous saisissez le texte du titre, du sous-titre ou n'importe quel axe, l'espace nécessaire sera réservé pour afficher le texte près du diagramme.
Sinon, aucun espace ne sera réservé, laissant plus d'espace pour afficher le diagramme.
Il n'est pas possible de lier le texte du titre à une cellule.
Vous devez saisir le texte directement.
Format
.
Légende
La légende affiche les étiquettes à partir de la première ligne ou colonne, ou à partir de la plage que vous avez définie dans la boîte de dialogue Séries de données.
Si le diagramme ne contient pas d'étiquettes, la légende affiche du texte comme &quot; Ligne 1, Ligne 2, ... &quot; , ou &quot; Colonne A, Colonne B, ... &quot; en fonction du numéro de la ligne ou de la lettre de la colonne des données du diagramme.
Vous ne pouvez saisir le texte directement, il est généré automatiquement à partir de la plage de cellules Nom.
Sélectionnez l'une des options de position.
Lorsque le diagramme est terminé, vous pouvez indiquer d'autres positions à partir du menu Format.
Grilles
grid x axis
Affiche les lignes de la grille perpendiculaires à l'axe x.
y axis
Affiche les lignes de la grille qui sont perpendiculaires à l'axe y.
z axis
Affiche les lignes de la grille perpendiculaires à l'axe z.
Cette option est disponible uniquement pour les diagrammes tridimensionnels.
Les lignes visibles de la grille peuvent aider à estimer les valeurs des données du diagramme.
La distance des lignes de la grille correspond aux paramètres d'intervalle dans l'onglet Échelle des propriétés de l'axe.
Les lignes de la grille ne sont pas disponibles pour les diagrammes en secteur.
Éléments supplémentaires
Pour insérer des éléments supplémentaires, utilisez le menu Insertion du diagramme en mode édition.
Vous pouvez alors définir les éléments suivants :
Axes secondaires
Grilles secondaires
Étiquettes de données
Statistiques, par exemple valeurs moyennes, barres d'erreur y et courbes de tendance
