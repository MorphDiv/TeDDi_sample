Configurer la Structure de découpage du projet (WBS)
Le Structure de découpage du projet introduit une numérotation pour toutes les tâches du projet, la structure de la tâche. Les tâches sont numérotées d'une manière qui représente la structure du projet. Le WBS est ainsi défini comme un arbre.
Le WBS
Le WBS
Vous pouvez paramétrer le WBS de la manière dont vous voulez étudier le projet. Pour se faire, utilisez l'option Outils Définir le modèle WBS.... Cela affiche la boîte de dialogue de définition du WBS.
La boîte de dialogue de définition du WBS
Boîte de dialogue de définition du WBS
Boîte de dialogue de définition du WBS
Code
Par défaut, il s'agit d'un nombre mais vous pouvez modifier le code WBS pour un nombre romain ou une lettre, en majuscule ou en minuscule.
Séparateur
Le séparateur par défaut est un point mais il peut être paramétré en ce que vous voulez. Le séparateur définit l'apparence que va prendre l'arbre. Dans la capture d'écran ci-dessus, le séparateur est le point.
À la place d'avoir le même code pour tout les niveaux (voir exemple 3.2.1) vous pouvez définir un code différent pour chaque niveau. Pour cela, il faut choisir Utiliser des niveaux puis vous définissez le code et le séparateur de chaque niveau.
Par exemple, si vous voulez que votre arbre soit 1: II-B, vous paramétrez le niveau 0 à nombre et le séparateur à & #160;:, puis le niveau 1 en nombre romain en majuscule et le séparateur à - et enfin le niveau 2 sera en lettres majuscules.

Pour paramétrer un nouveau niveau, choisissez le numéro du niveau dans la boîte près du bouton Ajouter un niveau puis appuyez sur le bouton Ajouter un niveau. Enfin, choisissez le code pour le niveau et le séparateur.

Pour supprimer un niveau, choisissez le numéro du niveau dans la première colonne de la liste et cliquez dessus. La ligne du niveau est alors surligné et le numéro du niveau passe en gras. Cliquez alors sur le bouton Supprimer. Ceci est illustré par la capture d'écran avec le niveau 2.

Générer un code WBS
Le code WBS est généré automatiquement & #160;; choisissez simplement l'option Généré le code OT du menu Outils menu pour générer les codes WBS pour le projet.
Après avoir modifié les définitions du WBS, vous pouvez générer le nouveau WBS.
