Le programme exécute le premier Sub ou Function puis l'exécution du programme s'interrompt.
L'instruction Sub Main n'est pas prioritaire sur l'exécution du programme.
Insérez le code Basic entre les lignes de l'instruction Sub Main et End Sub que vous voyez lorsque vous ouvrez l'EDI pour la première fois.
Alternativement, supprimez toutes les lignes et entrez votre propre code Basic.
