snmp_get_quick_print() retourne la valeur courante, stockée dans la librairie UCD, de l 'option quick_print.
Par défaut, quick_print est inactivée.
?php $quickprint = snmp_get_quick_print();?
L 'exemple ci-dessus devrait retourner FALSE, si quick_print est inactivée et, TRUE si quick_print est activée.
snmp_get_quick_print() est seulement disponible avec la librairie UCD SNMP.
Cette fonction n 'est pas disponible avec la librairie Windows SNMP.
Voir: snmp_set_quick_print() pour une description complète de l 'affichage de quick_print.
