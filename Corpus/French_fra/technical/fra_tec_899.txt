Impression d'étiquettes d'adresses
Fichier - Nouveau - Étiquettes
, ouvrez la boîte de dialogue
Étiquettes
.
Étiquettes
, sélectionnez le format de la feuille d'étiquettes à imprimer
Choisissez la base de données et la table à partir de laquelle obtenir les données.
Sélectionnez un champ de base de données dont vous souhaitez imprimer le contenu.
Cliquez sur le bouton qui affiche une flèche gauche pour insérer le champ de base de données dans la zone de texte d'étiquette.
Continuez à sélectionner et à insérer des champs de base de données si vous souhaitez plus de champs sur chaque étiquette.
Vous pouvez appuyer sur Entrée pour insérer une nouvelle ligne et vous pouvez saisir des caractères pour insérer du texte fixe.
Synchroniser le contenu
sous l'onglet
Options
.
Si vous l'activez, une fois que vous quittez la boîte de dialogue de l'étiquette, une petite fenêtre s'ouvre comportant un bouton Synchroniser.
Maintenant vous n'avez besoin que de travailler sur la première étiquette du document d'étiquettes, puis de cliquer sur le bouton synchroniser pour copier le travail sur chaque étiquette du document.
Nouveau document
.
Affichage - Noms de champ
.
Cela affiche les champs d'une façon plus visible vous permettant de disposer et d'éditer le contenu de l'étiquette plus facilement.
Vous pouvez enregistrer et/ou imprimer le document d'étiquettes.
Mailing
.
Dans la boîte de dialogue Mailing, vous pouvez sélectionner les enregistrements pour lesquels vous voulez imprimer des étiquettes.
