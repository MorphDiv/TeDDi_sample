Instruction DefSng [Exécution]
Si aucun caractère de déclaration de type ou mot-clé n'est spécifié, l'instruction DefSng définit le type de variable par défaut correspondant à une plage de lettres.
Syntaxe :
Defxxx Characterrange1[, Characterrange2[,...]]
Paramètres :
Characterrange :
lettres spécifiant la plage de variables pour laquelle le type de données par défaut doit être défini.
xxx :
mot-clé définissant le type de variable par défaut :
Mot-clé :
type de variable par défaut
DefSng :
type simple
Exemple :
Définitions de préfixes REM pour les types de variables :
DefBool b
DefDate t
DefDbL d
DefInt i
DefLng l
DefObj o
DefVar v
DefSng s
Sub ExampleDefSng
sSng=Single REM sSng est une variable simple implicite
end sub
