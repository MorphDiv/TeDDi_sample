Menus contextuels des tables
Le menu contextuel du container de tables comporte diverses fonctions se rapportant à toutes les tables de base de données.
Pour éditer une table particulière dans une base de données, sélectionnez-la et ouvrez son menu contextuel.
Relations
permettant de définir des relations entre les diverses tables n'est disponible que pour les bases de données relationnelles.
En fonction du système de base de données utilisé, les menus contextuels peuvent comporter les entrées suivantes :
table est ouverte
