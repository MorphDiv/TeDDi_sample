Récemment utilisé
Sélectionnez un fichier de base de données à ouvrir dans la liste des derniers fichiers utilisés.
Cliquez sur Terminer pour ouvrir immédiatement le fichier et quitter l'assistant.
Ouvrir
Ouvre une boîte de dialogue de sélection de fichiers dans laquelle vous pouvez sélectionner un fichier de base de données.
Cliquez sur Ouvrir ou OK dans cette boîte de dialogue pour ouvrir immédiatement le fichier et quitter l'assistant.
Se connecter à une base de données existante
Cliquez ici pour créer un document de base de données pour une connexion à une base de données existante.
Type de base de données
Sélectionnez le type de base de données pour la connexion à la base de données existante.
Les bases de données de type Outlook, Evolution, KDE et Mozilla ne requièrent pas d'informations supplémentaires.
Si vous utilisez d'autres types de bases de données, l'assistant affichera des pages supplémentaires afin que vous puissiez saisir les informations requises.
