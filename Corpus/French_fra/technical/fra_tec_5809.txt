Actions pour les nouveaux périphériques
Introduction
Ce module vous permet de définir les actions que vous voulez associer quand de nouveaux périphériques sont détectés.
Capture d'écran du gestionnaire des actions périphériques
Capture d'écran du gestionnaire des actions périphériques
Vous verrez listé toutes les actions possibles qui sont disponibles quand des périphériques sont connectés. Remarquez que ces périphériques ne seront disponibles que sous certaines conditions.
Ajouter une action vous demandera de fournir un nom pour une nouvelle action, qui ouvrira ensuite cette action pour la modifier. Si vous ne fournissez pas tous les détails requis dans cette boîte de dialogue, votre action sera supprimée.
Modifier l'action vous permettra de modifier le nom, l'icône, la commande à exécuter et les conditions auxquelles l'action doit correspondre pour être affichées.
Refaire les modifications & #160; / Impossible de supprimer & #160; / Supprimer l'action. Ces trois actions réalisent trois choses différentes, cela dépend ce qui a été fourni avec votre système ou créé par vous. Vous pouvez seulement rétablir les modifications aux vastes actions du système, si il n'y a pas de modifications, ces actions ne peuvent être supprimées. Les actions créés par l'utilisateur sont complètement supprimées incluant toutes les modifications.
Ajouter une action
Le bouton Ajouter une action vous affiche une boîte de dialogue pour saisir le nouveau nom de l'action.
Capture d'écran de la boîte de dialogue « & #160; Ajouter une action & #160; »
Capture d'écran de la boîte de dialogue « & #160; Ajouter une action & #160; »
Le nom de l'action doit correctement décrire l'action. Après la saisie du nom et avoir cliqué sur Ok, La boîte de dialogue de modification s'affichera et vous permettra de définir les propriétés de l'action.
Modifier une action
La boîte de dialogue vous permet de modifier la configuration de l'action que vous avez sélectionné.

Type d' élément
Type de périphérique
Nom de la valeur
Cette fonctionnalité inclut l'icône, le nom de l'action, la commande de l'action et divers conditions prérequises. Vous pouvez modifier l'icône en cliquant dessus. Les conditions prérequises sont affichées dans une arborescence, l'élément du haut doit être « & #160; true & #160; » pour que l'action est affichée. Chaque élément peut être soit un « & #160; conteneur & #160; » ou une « & #160; conditions prérequises & #160; », qui est sélectionné quand cet élément a été ajouté. Seul les conteneurs peuvent avoir les éléments en dessous d'eux.
Les conteneurs peuvent être de deux types qui détermineront la correspondance avec les conditions. Soit tout, ou au moins une des conditions doit correspondre.
Les conditions prérequises peuvent aussi être de différents types. Si le périphérique correspond aux conditions, elles sont « & #160; true & #160; », sinon elles sont « & #160; false & #160; ».
Ajouter une condition requise
Disponible uniquement pour les conteneurs. Cela vous permet de sélectionner le type de condition requis et fournit quelque uns de ses détails de départ.
Modifier une condition requise
Cette action est réalisée seulement quand un élément valable est sélectionné. Cela vous permet de modifier les divers détails d'un type d'élément particulier.
Supprimer une condition requise
Disponible pour tous les éléments, sauf pour l'élément superutilisateur. Si un conteneur ne contient aucune conditions requises, il est alors supprimé.
Modifier une condition requise
Cette boîte de dialogue vous permet de modifier les divers détails à propos du conteneur et affichera les différentes informations, cela dépend de quel type d'élément il s'agit.
Type de conteneur
Vous pouvez seulement modifier un réglage, la correspondance logique. C'est soit « & #160; La totalité des conditions contenues doit correspondre & #160; » ou « & #160; Toute condition requise contenue doit correspondre & #160; ». Cela sera utilisé pour déterminer si un conteneur est « & #160; true & #160; ».
De nombreux réglages sont disponibles avec des réglages additionnels pour des sous-types.
Type de restriction
Seul deux réglages sont disponibles & #160;: « & #160; Est un type de valeur & #160; » et « & #160; Comparer la valeur à & #160; ». « & #160; Est un type de valeur & #160; » vérifie que le périphérique est bien le type de périphérique sélectionné. « & #160; Comparer la valeur à & #160; » compare une valeur du périphérique avec ce que vous spécifiez.
Vous pouvez sélectionner ici le type de périphérique. C'est utilisé pour limiter la sélection des valeurs pour comparer les valeurs, et les correspondances directes pour le type de périphérique sélectionné.
Seulement visible pour « & #160; Comparer la valeur à & #160; ». Cela vous permet de sélectionner une valeur de périphérique à vérifier.
Égale & #160; / N'est pas égal
Cela détermine quel type de correspondance nous recherchons à la comparaison d'une valeur. Le texte à vérifier est spécifié à coté.
