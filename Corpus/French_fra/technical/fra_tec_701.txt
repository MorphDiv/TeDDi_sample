Spécifie parmi les tables de base de données disponibles une table contenant le champ de données dont le contenu doit être affiché sous forme d'entrée de liste.
Pour les zones de liste, une table pouvant être liée à la table du formulaire actif est indiquée.
La table de liaison doit comporter au minimum un champ en commun avec celle du formulaire actif.
Ceci permet d'éviter les références ambiguës.
Dans le cas de boîtes combinées, il ne doit y avoir aucun lien entre la table du formulaire et la table qui comprend les données affichées dans la boîte combinée.
