Une introduction aux modèles
Eh bien, bonne nouvelle, passer à un traitement de textes basé sur des cadres est facile une fois que vous avez sélectionné un modèle, KWord peut agir comme le traitement de textes auquel vous êtes habitué.
Il y a des modèles disponibles pour la correspondance simple au quotidien.
Maintenant que vous voulez bien essayer, nous allons parler brièvement de ce que sont les modèles, ce que les modèles peuvent faire pour vous et comment travailler avec ces modèles.
Un modèle peut être pensé comme un moule initial pour votre document.
Avec KWord, vous pouvez utiliser la notion planifiée de comment votre document doit paraître pour rendre votre travail plus facile.
En utilisant un modèle, vous pouvez créer un jeu de cadres pour approximer la mise en page finale.
Pour vos lettres d'affaires, vous pouvez choisir un grand cadre qui remplit toute la page de haut en bas et d'un bord à l'autre (sauf les marges, bien sûr).
Pour ce journal, cependant, vous pouvez décider qu'un document de deux ou trois colonnes est plus adapté.
Une fois le modèle choisi, KWord créera les cadres pour vous.
Ce n'est pas un problème.
Si vous commencez un document sur une simple colonne et décidez plus tard que deux colonnes sont préférables, vous pouvez ajouter une autre colonne.
Le but des modèles n'est pas de limiter votre possibilité de changer votre mise en page, mais plutôt de vous décharger d'une partie du travail quand c'est possible.
En choisissant le bon modèle, vous commencez avec un document qui simplifie le formatage (contrairement à choisir le modèle par défaut et changer plus tard), mais il est aussi important de se rappeler que choisir un mauvais modèle ne limitera pas votre travail.

Les différences entre les modèles orientés texte et les modèles disposition
Avant de commencer avec ce didacticiel, il est important que les différences entre les modèles orientés texte et les modèles de mise en page soit claire dans votre esprit.
Orienté texte
Les modèles orientés texte sont conçus pour agir comme les traitements de texte orientés page.
Quand vous sélectionnez un modèle orienté texte, vous créez un modèle avec un cadre ou plus.
Le premier cadre est distinctement adapté à la taille du papier de votre document.
Quand vous augmentez ou diminuez la taille du papier ou changez les marges, la taille du cadre est automatiquement ajustée.
Vous ne pouvez pas déplacer ce cadre sur la page ni le supprimer.
Si vous avez besoin d'ajuster la taille ou la position de ce cadre, vous devez ajuster les marges.
Au-dessus du premier cadre, vous pouvez ajouter des cadres.
Ces cadres peuvent contenir des images, du texte ou ce que vous voulez d'autre.
Ces nouveaux cadres peuvent être déplacés, redimensionnés et supprimés selon vos besoins.
Si vous le voulez, KWord enroulera même le texte du premier cadre autour de toute information dans les cadres secondaires.
Disposition
Ces modèles sont conçus pour la publication.
Quand vous sélectionnez un modèle dans ce groupe, tous les cadres créés sont égaux.
Voilà la différence entre les modèles orientés texte et de mise en page.
Aucun des cadres dans un modèle mise en page n'est adapté à la page aussi fermement que le cadre principal dans un modèle orienté texte.
Chacun des cadres est indépendant de la taille de la page, et peut être déplacé, redimensionné sans changer les marges, etc.
Bien sûr, ceci signifie aussi que les modifications des marges ou de la taille de papier de votre document nécessite un redimensionnement un déplacement/redimensionnement de chaque cadre.
Comme vous pouvez le voir, le choix entre les modèles Mise en page et Orienté texte est un choix entre la souplesse (Mise en page) et la facilité (Orienté texte). vous pouvez créer tout document que vous voulez avec l'un ou l'autre type de modèle.
Si le concept de modèle reste flou, veuillez suivre le didacticiel.
Le premier document que vous créez utilisera un modèle de mise en page.
Ceci devrait vous aider à clarifier la manière les modèles fonctionnent exactement.
