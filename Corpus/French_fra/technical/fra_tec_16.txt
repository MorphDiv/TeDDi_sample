Barres d'erreur Y
pour afficher des barres d'erreur dans les diagrammes 2D.
Une barre d'erreur est une courbe indicateur qui s'étend de y - valeur_erreur_négative à y + erreur_valeur_positive.
Dans cette expression, y représente la valeur du point de données.
Lorsque &quot; écart type &quot; est sélectionnée, y est la valeur moyenne de la série de données.
Valeur_erreur_négative et valeur_erreur_positive sont les sommes calculées par la fonction de barre d'erreur ou sont données explicitement.
Insertion - Barres d'erreur Y
n'est disponible que pour les diagrammes 2D.
Catégorie d'erreur
Catégorie d'erreur
, vous pouvez sélectionner les différents types d'affichage pour la catégorie d'erreur en question.
Aucun(e)
N'affiche aucune barre d'erreur.
Valeur constante
Affiche les valeurs constantes que vous spécifiez dans la zone Paramètres.
Pourcentage
Affiche un pourcentage.
L'affichage renvoie au point de données correspondant.
Définissez le pourcentage dans la zone Paramètres.
Fonctions
Sélectionnez une fonction pour calculer les barres d'erreur.
Erreur standard : affiche l'erreur standard
Variance : affiche la variance calculée à partir du nombre de points de données et des valeurs respectives.
Écart type : affiche l'écart type (racine carrée de la variance).
Marge d'erreur : affiche la plus haute marge d'erreur en pourcentage en fonction de la plus haute valeur du groupe de données.
Définissez le pourcentage dans la zone Paramètres.
Plage de cellules
Cliquez sur la plage de cellules puis spécifiez une plage de cellules à partir de laquelle prendre les valeurs des barres d'erreur positives et négatives.
Cliquez sur le bouton pour réduire la boîte de dialogue, puis utilisez la souris pour sélectionner une plage dans la feuille.
Cliquez de nouveau sur le bouton pour restaurer la boîte de dialogue à sa taille normale.
À partir de la table de données
Pour un diagramme avec ses propres données, les valeurs de la barre d'erreur peuvent être saisies dans la table de données du diagramme.
La boîte de dialogue Table des données affiche des colonnes supplémentaires nommées Barres d'erreur Y positives et Barres d'erreur Y négatives.
Positive (+)
Saisissez la valeur à ajouter à la valeur affichée comme la valeur d'erreur positive.
Negative (-)
Saisissez la valeur à soustraire de la valeur affichée comme la valeur d'erreur négative.
Positive (+) when Cell Range is selected
Saisissez la plage d'adresses à partir de laquelle obtenir les valeurs d'erreur positives.
Utilisez le bouton Réduire pour sélectionner la plage à partir d'une feuille.
Negative (-) when Cell Range is selected
Saisissez la plage d'adresses à partir de laquelle obtenir les valeurs d'erreur négatives.
Utilisez le bouton réduire pour sélectionner la plage à partir d'une feuille.
Valeur identique pour les deux
Activez pour utiliser les valeurs d'erreur positives également comme valeurs d'erreur négatives.
Vous pouvez uniquement modifier la valeur de la boîte &quot; Positive (+) &quot; .
Cette valeur est copiée automatiquement dans la boîte &quot; Négative (-) &quot; .
Indicateur d'erreur
Spécifie l'indicateur d'erreur.
Positive et négative
Affiche les barres d'erreur positives et négatives.
Positive
Affiche uniquement les barres d'erreur positives.
Négative
Affiche uniquement les barres d'erreur négatives.
