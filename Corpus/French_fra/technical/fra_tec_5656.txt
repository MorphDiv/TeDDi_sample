Introduction
& kjots; est un petit programme qui vous permet de prendre et d'organiser facilement de petites notes.
Utilisation de & kjots;
& kjots; met à votre disposition deux éléments de base afin de classer vos notes & #160;: les carnets et les pages.
Vous pouvez créer une multitude de carnets dans & kjots;, chacun d'entre eux contenant les pages où vous inscrirez vos notes. Vous devriez donner un nom aux carnets ainsi qu'aux pages (afin de pouvoir retrouver ce que vous chercher). Bien que ce ne soit pas obligatoire, il est judicieux de procéder ainsi.
Utilisation des carnets
Pour créer un nouveau carnet, appuyez sur Ctrl; Maj; N, ou sélectionnez le menu Fichier Nouveau carnet et il vous sera alors demandé de saisir un nom pour le nouveau carnet.
Le nom du nouveau carnet devrait maintenant apparaître dans la partie gauche de la fenêtre. Si vous souhaitez que le carnet apparaisse dans vos signets, utilisez Ctrl; B ou sélectionnez le menu Signets Ajouter un signet. Vous pouvez même organiser vos signets dans des dossiers séparés de la même manière que ce que vous faites probablement avec & konqueror;.
Pour modifier l'ordre de vos carnets, vous pouvez simplement effectuer un glisser-déposer d'un carnet vers l'emplacement où vous souhaitez le voir apparaître.
Vous avez peut-être remarqué qu'il n'y a nulle part un bouton Enregistrer. Ceci est normal, & kjots; enregistre vos carnets et pages automatiquement, de telle façon que vous ne perdrez jamais cette petite note que vous avez écrite lorsque vous étiez au téléphone et aviez besoin d'écrire quelque chose à la hâte.
Il est possible d'exporter la totalité d'un carnet vers un fichier texte simple ou HTML en sélectionnant le menu Fichier Exporter le carnet.... Il vous faudra alors choisir un nom et un emplacement pour le fichier à enregistrer. L'exemple qui suit vous montre la forme qu'aura un tel fichier & #160;:
############ # Première page ############ Ne pas oublier les courses & #160;! ############# # Code utile ############# 10 PRINT "BONJOUR LE MONDE" 20 GOTO 10 ################ # Troisième page ################ Voici la troisième page & #160;; elle sert d'exemple pour KJots.
Souvenez -vous que la boîte de dialogue vous demandant de choisir un nom de fichier est une boîte de dialogue standard de & kde;. Ceci signifie qu'il est aussi simple de stocker le fichier sur votre disque local que sur un serveur web, FTP ou sur une clé USB. Vous pouvez donc exporter vers un fichier au format HTML et l'enregistrer sur un serveur web, de manière à toujours avoir vos notes à portée de main.
Utilisation des pages
La page est l'endroit où vous écrivez... tout ce que vous aurez à écrire. Dès qu'un carnet est créé, vous pouvez commencer à écrire dans la zone de saisie principale.
Pour créer des pages supplémentaires, vous pouvez utiliser Ctrl; N ou sélectionner le menu Fichier Nouvelle page, ce qui créera une nouvelle page. De manière similaire, la page courante peut être supprimée en sélectionnez le menu Fichier Supprimer la page.
Si vous avez besoin d'insérer la date et l'heure courants alors que vous écrivez une note, vous pouvez utiliser Ctrl; I.
Afin de retrouver la page que vous souhaitez consulter, vous pouvez (et devriez) donner un nom aux pages. Ceci peut se faire selon la méthode usuelle & #160;: sélectionnez la page à renommer, attendez une seconde et cliquez à nouveau dessus. Vous pouvez ensuite la renommer. Bien sûr, un raccourci clavier existe aussi, que vous pouvez à tout moment lors de l'écriture d'une note. Le raccourci clavier est & #160;: Ctrl; M.
Une autre façon de renommer la page est de sélectionner une partie de votre note et ensuite d'appuyer sur Ctrl; T & #160;: le titre de la page deviendra instantanément le texte sélectionné.
Si vous voulez un accès rapide à une page particulière, vous pouvez lui assigner un raccourci clavier. Allez dans le menu Configuration Configurer les raccourcis clavier....
Référence des commandes
Le menu Fichier
Ctrl; N Fichier Nouvelle page...
Crée une nouvelle page.
Ctrl; Maj; N Fichier Nouveau carnet...
Crée un nouveau carnet.
Fichier Exporter la page
Enregistre la page courante sous la forme d'un fichier texte ou HTML.
Fichier Exporter le carnet
Enregistre le carnet courant sous la forme d'un fichier texte ou HTML.
Fichier Effacer la page
Supprime la page courante.
Fichier Effacer le carnet
Supprime le carnet courant.
Ctrl; P Fichier Imprimer
Imprime le carnet ou la page courante.
Ctrl; Q Fichier Quitter
Quitte & kjots;.
Le menu Édition
Ctrl; X Édition Couper
Coupe le texte sélectionné - dans la zone de saisie principale - vers le presse-papiers.
Ctrl; C Édition Copier
Copie le texte sélectionné - dans la zone de saisie principale - vers le presse-papiers.
Ctrl; V Édition Coller
Colle le texte contenu dans le presse-papiers dans la zone de saisie principale.
Ctrl; T Édition Coller comme titre de page
Renomme la page avec le texte contenu dans le presse-papiers.
Ctrl; F Édition Chercher...
Recherche du texte précisé dans la zone de saisie principale. Notez que la recherche n'est effectuée que dans la page courante.
F3 Édition Poursuivre la recherche
Recherche la prochaine occurrence du texte recherché (précisée dans « & #160; Chercher & #160; »).
Ctrl; R Édition Remplacer...
Remplace un morceau de texte par un autre, dans la page en cours uniquement.
Ctrl; M Édition Renommer...
Donne la possibilité de renommer la page courante.
Ctrl; I Édition Insérer la date
Insère la date et l'heure dans la page courante.
Le menu Signets
Ctrl; B Signets Ajouter un signet
Ajoute la page courante à la liste des signets. Voir Utilisation des carnets.
Signets Modifier les signets
Modifie les signets. Vous pourrez ici chercher parmi vos signets, les commenter, les organiser dans différents dossiers et beaucoup plus encore.
Signets Nouveau dossier de signets
Crée un nouveau dossier où vous pouvez stocker vos signets.
Le menu Configuration
Configuration Afficher/ Cacher la barre d' outils
Active ou désactive l'affichage de la barre d'outils située en-dessous de la barre de menu.
Configuration Configurer les raccourcis clavier...
Permet de définir les raccourcis correspondant à certains éléments des menus.
Configuration Configurer les barres d'outils...
Permet de choisir les éléments présents dans la barre d'outils.
Configuration Configurer KJots...
Permet de configurer & kjots; notamment en ce qui concerne la police de caractères utilisée et les paramètres d'auto-enregistrement.
