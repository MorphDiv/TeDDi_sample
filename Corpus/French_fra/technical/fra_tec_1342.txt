Lecteur de médias
Ouvre la fenêtre du lecteur de médias dans laquelle vous pouvez pré-visualiser les fichiers de films et de sons et insérer ces fichiers dans le document actif.
Le lecteur de médias prend en charge beaucoup de formats de médias différents.
Vous pouvez également insérer des fichiers médias à partir du lecteur de médias dans votre document.
Sur les systèmes Linux ou Solaris, le lecteur de médias nécessite l'API Java Media Framework (JMF).
Téléchargez et installez les fichiers JMF à partir de http://java.sun.com/javase/technologies/desktop/media/jmf/index.jsp et ajoutez le chemin vers le jmf.jar installé au chemin de la classe sous Outils - Options - %PRODUCTNAME - Java.
Sur les systèmes Windows, le lecteur de médias utilise DirectShow, qui doit être installé par défaut sur votre ordinateur.
Ouvrir
Ouvre un fichier de film ou un fichier son que vous souhaitez pré-visualiser.
Appliquer
Insère le fichier de film ou de son actif comme un objet média dans le document actif.
Lancer
Jouer le fichier actif.
Suspendre
Suspend ou reprend la lecture du fichier actif.
Arrêter
Arrête la lecture du fichier actif.
Répéter
Joue le fichier en boucle.
Silencieux
Active ou désactive le son.
Curseur du volume
Ajuste le volume.
Zoom
Ajuste la taille de la lecture du film.
Positionner le curseur
Se déplace à un emplacement différent dans le fichier.
