Barre des tâches
Barre des tâches
La barre des tâches est un moyen rapide de passer d'une application à une autre.
Elle est située par défaut dans le tableau de bord, mais il est aussi possible de la placer sur le bureau.
La première option, intitulée Afficher toutes les fenêtres, permet de décider si toutes les fenêtres ouvertes sont représentées dans la barre des tâches ou non.
Par défaut, seules les fenêtres du bureau courant sont représentées.
Mais si cette option est activée, les fenêtres de tous les bureaux seront représentées dans la barre des tâches.
L'option suivante, Grouper les fenêtres similaires, vous permet d'économiser de l'espace sur votre barre des tâches, en n'affichant qu'une icône pour chaque application démarrée, sans tenir compte du nombre de fenêtres affichées.
Vous pouvez cliquer sur l'icône pour afficher un menu comportant toutes les fenêtres disponibles.
Ceci est très utile si vous avez activé l'option Afficher toutes les fenêtres.
Enfin, vous pouvez désactiver l'affichage des icônes des applications, et n'afficher que le texte.
Vous pouvez de cette façon économiser de l'espace sur votre barre des tâches, par exemple.
Actions
Le lot suivant d'options vous permet de personnaliser les actions réalisées par différents clics de souris sur les icônes de barre des tâches.
Vous pouvez sélectionner toute action de la liste pour le &BDS;, le &BGS; et le &BCS;.
Afficher la liste des tâches
Affiche la liste des tâches groupées sous l'icône sur laquelle vous avez cliqué.
S'il n'y a qu'une tâche pour cette icône, ça deviendra la fenêtre active.
C'est l'action par défaut pour le &BGS;
Naviguer parmi les fenêtres
S'il y a plusieurs tâches groupées sous l'icône, passe de l'une à l'autre jusqu'à ce que vous relâchiez le bouton (&cad; quand vous avez atteint la fenêtre que vous cherchiez).
S'il n'y a qu'une tâche ou fenêtre pour cette icône, ça deviendra la fenêtre active.
Ceci est l'action par défaut pour le &BCS; si le groupage des fenêtres est activé.
Afficher le menu des opérations
Affiche le menu des opérations pour l'application.
Ceci vous permet de réduire, agrandir, etc. déplacer les fenêtres entre les bureaux et fermer les fenêtres.
Vous pouvez effectuer ces actions sur toutes les fenêtres groupées sous cette icône, ou toute fenêtre isolée en choisissant dans le sous-menu.
C'est l'action par défaut pour le &BDS;
Activer, Restaurer, réduire une tâche
Boucle entre les états pour la fenêtre sélectionnée.
Ceci est l'action par défaut pour le &BCS; si le groupage des fenêtres n'est pas activé.
Restaurer la tâche, Réduire la tâche, Activer la tâche
Ces trois options n'ont du sens que si le groupage est désactivé.
Elles sont autoexplicatives.
Restaurer signifie la rendre active, l'amener sur le devant et lui donner le focus.
Réduire signifie l'amener derrière et donner le focus à la fenêtre qui est dessus.
