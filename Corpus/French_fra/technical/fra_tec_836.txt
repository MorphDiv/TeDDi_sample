Insertion de diagrammes
Il existe plusieurs méthodes pour réaliser un diagramme :
Insère un diagramme basé sur les données des cellules de Calc ou Writer.
Ces diagrammes s'actualisent automatiquement lorsque la source de données est modifiée.
Insérez un diagrapmme à partir d'un jeu de données par défaut, utilisez ensuite la boîte de dialogue Table de données pour saisir vos propres données pour ce diagramme.
Ces diagrammes peuvent être créés dans Writer, Impress ou Draw.
Copier un diagramme dans un autre document à partir de Calc ou Writer.
Ces diagrammes sont des instantanés pris au moment de la copie.
Ils ne s'actualisent pas lorsque les données de la source sont modifiées.
Dans Calc, un diagramme est un objet dans une feuille, il ne peut être une feuille par lui-même.
