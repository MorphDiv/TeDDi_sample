Assistant Démarrage
L'assistant Démarrage s'ouvre lorsque vous lancez %PRODUCTNAME pour la première fois.
L'assistant comporte les pages suivantes :
Bienvenue
L'assistant Démarrage vous aide à spécifier les paramètres de l'utilisateur pour %PRODUCTNAME.
Cet assistant ne s'exécute qu'une seule fois.
Licence
Lisez et acceptez le texte de la licence.
Vous devez faire défiler le texte de la licence pour pouvoir continuer.
Si vous n'acceptez pas la licence, l'installation sera annulée.
Migration
Vous pouvez utiliser cette page pour importer des données d'utilisateur d'une version de %PRODUCTNAME précédemment installée.
Copier les données personnelles
Copie le nom d'utilisateur et les autres données issues d'une précédente installation de %PRODUCTNAME.
Nom d'utilisateur
Le nom d'utilisateur est utilisé dans les propriétés du document et dans les modèles, ainsi qu'au moment de l'enregistrement de modifications apportées aux documents.
Cette page ne s'affiche que si une précédente installation de %PRODUCTNAME n'est pas trouvée sur votre ordianteur.
Saisissez vos données personnelles dans les zones de texte.
