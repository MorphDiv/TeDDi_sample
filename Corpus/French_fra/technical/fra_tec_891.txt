Ajout de points actifs cliquables à des images
Une Image Map permet de lier des URL à des zones spécifiques, appelées points actifs, sur une image de votre document.
Une Image Map est un groupe d'un ou de plusieurs points actifs.
Vous pouvez dessiner trois types de points actifs : des rectangles, des ellipses et des polygones.
Lorsque vous cliquez sur un point actif, l'URL s'ouvre dans la fenêtre ou le cadre du navigateur indiqué.
Vous pouvez également indiquer le texte à afficher lorsque la souris reste immobilisée quelques instants sur le point actif.
Pour ajouter une zone cliquable à une image
Placez le curseur à l'emplacement souhaité pour l'Image Map dans le document.
Insertion - Image - À partir d'un fichier
, puis sélectionnez une image bitmap et insérez-la.

Utilisez les icônes de l'éditeur d'Image Map pour dessiner une forme de point actif, telle qu'un rectangle, sur l'image à l'arrière-plan.
Outils - Options - %PRODUCTNAME - Général

Saisissez l'adresse URL à afficher dans le navigateur Web lorsque l'utilisateur cliquera sur le point actif.
Vous pouvez également saisir le texte qui s'affichera sous forme d'infobulle lorsque l'utilisateur placera le curseur au-dessus du point actif.
Cliquez sur le bouton Appliquer pour appliquer vos modifications, puis refermez l'éditeur d'Image Map.
Enregistrez le document au format %PRODUCTNAME ou HTML.
Vous pouvez enregistrer l'Image Map en tant que fichier, puis charger ce fichier sur un serveur Web, par exemple.
