Paramètres supplémentaires
Spécifie les options supplémentaires d'une base de données.
Édition - Base de données - Propriétés
et cliquez sur l'onglet
Paramètres supplémentaires
La disponibilité des contrôles suivants dépend du type de base de données :
Nom de l'hôte
Saisissez le nom d'hôte du serveur qui contient la base de données (ldap.server.com, par exemple).
Numéro de port
Entrez le numéro de port du serveur hébergeant la base de données.
Classe de pilote JDBC pour MySQL
Entrez le nom du pilote JDBC pour la base de données MySQL.
Jeu de caractères
Sélectionnez le jeu de caractères que vous souhaitez utiliser pour visualiser la base de données dans $[officename].
Ce choix n'affecte pas la base de données elle-même.
Pour utiliser le jeu de caractères par défaut de votre système d'exploitation, sélectionnez
Système
.
Les bases de données Texte et dBase n'acceptent que des jeux de caractères avec une longueur de caractères fixe, où tous les caractères sont codés sur le même nombre d'octets.
Classe de pilote JDBC pour Oracle
Entrez le nom du pilote JDBC pour la base de données Oracle.
Paramètres du pilote
Spécifiez des options de pilote supplémentaires.
Utiliser le catalogue pour les bases de données basées sur des fichiers
Utilise la source de données active du catalogue.
Cette option est utile lorsque la source de données ODBC est un serveur de base de données.
Ne sélectionnez pas cette option si la source de données ODBC est un pilote dBase.
DN de base
Saisissez le point de départ de la recherche à effectuer dans la base de données LDAP (dc=com, par exemple).
Nombre maximal d'enregistrements
Saisissez le nombre maximal d'enregistrements à charger lors de l'accès au serveur LDAP.
Afficher également les enregistrements supprimés
Affiche tous les enregistrements d'un fichier, y compris ceux marqués comme supprimés.
Si vous activez cette case à cocher, vous ne pourrez plus supprimer d'enregistrements.
Dans le format dBase, les enregistrements supprimés restent dans le fichier.
Pour visualiser les modifications apportées à la base de données, déconnectez-vous de cette dernière, puis reconnectez-vous.
Index
Ouvre la boîte de dialogue Index, dans laquelle vous pouvez organiser les index de table dans la base de données dBASE active.
UFI:
Commercial means Adabas
Taille du tampon de données (Mo)
Saisissez la taille du tampon de données de la base de données.
Ce paramètre prend effet après le redémarrage de la base de données.
Incrément (Mo)
Saisissez la taille d'incrément automatique de la base de données.
La taille maximale de l'incrément est de 100 Mo.
Ce paramètre prend effet après le redémarrage de la base de données.
Contrôle Nom d'utilisateur
Saisissez le nom d'un utilisateur auquel vous souhaitez donner un contrôle limité pour modifier certains paramètres de la base de données.
Contrôle Mot de passe
Saisissez le mot de passe du contrôle Utilisateur.
Fermeture du service en même temps que %PRODUCTNAME
Quitte le serveur de base de données Adabas lorsque vous quittez $[officename].
Cette option n'est disponible que si vous démarrez le serveur de base de données depuis $[officename] avec un contrôle Utilisateur et un contrôle Mot de passe.
Étendu
Statistiques de la base de données
, dans laquelle vous pouvez afficher des statistiques concernant la base de données Adabas.
Texte avec en-tête
Activez cette case à cocher si la première ligne du fichier texte contient des noms de champs.!
Séparateur de champ
Saisissez ou sélectionnez le caractère de séparation des champs de données du fichier texte.
Séparateur de texte
Saisissez ou sélectionnez le caractère d'identification des champs de texte du fichier texte.
Vous ne pouvez pas utiliser le même caractère comme séparateur de champ.
Séparateur de décimales
Saisissez ou sélectionnez le caractère utilisé comme séparateur de décimales dans le fichier texte, par exemple un point (0.5) ou une virgule (0,5).
Séparateur de milliers
Saisissez ou sélectionnez le caractère utilisé comme séparateur de milliers dans le fichier texte, par exemple une virgule (1,000) ou un point (1.000).
Extension de fichier
Sélectionnez le format du fichier texte.
L'extension sélectionnée affecte certains paramètres par défaut de cette boîte de dialogue.
