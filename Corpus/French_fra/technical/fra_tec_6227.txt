Cette liste devrait contenir les types de fichiers que votre application peut gérer. Cette liste est organisée par types MIME. MIME, Multipurpose Internet (e)Mail Extension, est un protocole standard pour identifier le type de données en fonction des extensions de nom de fichier et du type MIME correspondant. Exemple & #160;: la partie « & #160; bmp & #160; » qui se trouve après le point dans fleur. bmp indique que c'est une certaine sorte d'image, image/ x-bmp. Pour savoir quelle application doit ouvrir chaque type de fichier, le système doit être informé sur les possibilités de chaque application pour gérer ces extensions et types MIME. Si vous voulez associer cette application avec un type MIME ou plus qui ne sont pas dans la liste, cliquez sur le bouton Ajouter au-dessous.
