Table de données
dans laquelle vous pouvez éditer les données du diagramme.
Table de données
n'est pas disponible si vous insérez un diagramme basé sur une feuille Calc ou un tableau Writer.
Pour mettre à jour manuellement un diagramme lorsqu'un tableau Writer a été modifié
i81240
Certains changements ne seront visibles que lorsque vous aurez fermé, puis rouvert la boîte de dialogue.
Pour modifier les données du diagramme
Quand vous créez un diagramme basé sur des données par défaut, ou quand vous copiez un diagramme dans votre document, il est possible d'ouvrir la boîte de dialogue Table de données pour saisir vos données.
Le diagramme réagira aux données en aperçu instantané.
Édition - Annuler
pour annuler les modifications.
Insérer ou sélectionner un diagramme qui n'est pas basé sur des données de cellules existantes.
Affichage - Table de données
pour ouvrir la boîte de dialogue Table de données.
Les séries de données sont organisées en colonnes.
Le rôle de la colonne de gauche est de définir des catégories ou des étiquettes de données.
Le contenu de la colonne de gauche est toujours formaté comme du texte.
Cliquez sur une cellule de la boîte de dialogue pour en modifier le contenu.
Cliquez sur une autre cellule pour voir les modifications du contenu dans la fenêtre d'aperçu.
Saisir le nom de la série de données dans la zone de texte en haut de la colonne.
Utilisez les icônes en haut de la table pour insérer ou effacer des lignes et des colonnes.
Pour les séries de données à plusieurs colonnes, seules des séries de données entières peuvent être insérées ou effacées.
Déplacer les séries vers la droite
pour intervertir la colonne active et celle située à sa droite.
Déplacer les lignes vers le bas
pour intervertir la ligne active et celle située en dessous.
insert row
Insère une ligne sous la ligne active.
insert column
Insère une colonne après la colonne active.
delete row
Supprime la ligne active.
Il est impossible de supprimer la ligne d'étiquette.
delete column
Supprime la colonne active.
Il est impossible de supprimer la colonne d'étiquette.
move series right
Intervertit la colonne active et la colonne située à sa droite.
move row down
Intervertit la ligne active et la ligne située en dessous.
edit fields, see i72361
Saisissez le nom des séries de données.
