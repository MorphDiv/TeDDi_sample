Référence des commandes
La fenêtre principale de & appname;
Chaque élément du menu est discuté ci-dessous. Quand un raccourci clavier lance une fonctionnalité du menu, ce raccourci clavier est listé avec l'élément du menu.
Le menu Fichier
Fichier Nouveau
Cette commande crée une nouvelle collection. & appname; gère douze types de collections par défaut avec une collection vide personnalisée définie par l'utilisateur.
Ctrl; O Fichier Ouvrir..
Cette commande ouvre un fichier dans & appname;.
Fichier Récemment ouvert(s)
Cette commande ouvre un fichier depuis un sous-menu qui contient une liste des fichiers récemment ouverts.
Ctrl; S Fichier Enregistrer...
Cette commande enregistre la collection. Si le fichier est Sans titre, Enregistrer... est l'équivalent de Enregistrer sous. Seulement activé quand la collection a été modifiée.
Fichier Enregistrer sous...
Cette commande enregistre la collection vers un nouveau fichier.
Ctrl; P Fichier Imprimer...
Cette commande imprime la collection. Les options d'impression peuvent être définies dans la Boîte de dialogue de configuration.
Fichier Importer
Cette commande importe les données depuis un autre fichier. & appname; peut importer des données depuis d'autres formats.
Fichier Exporter
Cette commande exporte la collection vers un autre format. & appname; peut exporter les données dans d'autres formats.
Ctrl; Q Fichier Quitter
Cette commande ferme la fenêtre de & appname;, si la collection a des modifications non enregistrées, vous serez averti de les enregistrer avant de continuer.
Le menu Édition
Ctrl; Z Édition Annuler
Cette commande annule l'action précédente. Toutes les actions ne sont pas gérées par la commande d'annulation.
Ctrl; Maj; Z Édition Refaire
Cette commande refait la commande annulée.
Ctrl; X Édition Couper
Cette commande supprime le texte sélectionné, et copie le texte sélectionné dans le presse-papiers.
Ctrl; C Édition Copier
Cette commande copie le texte sélectionné vers le presse-papiers.
Ctrl; V Édition Coller
Cette commande colle le texte dans le presse-papiers dans l'éditeur à la position du curseur.
Ctrl; A Édition Tout sélectionner
Cette commande sélectionne toutes les entrées dans la collection, dans la Vue en colonnes.
Ctrl; Maj; A Édition Désélectionner
Cette commande désélectionne toutes les entrées dans la collection.
Ctrl; M Édition Recherche Internet
Cette commande ouvre la Boîte de dialogue des champs de collection pour rechercher et importer des entrées depuis des sites web, incluant Amazon. com.
Ctrl; J Édition Filtre avancé
Cette commande ouvre la Boîte de dialogue de filtre avancée.
Le menu Collection
Ctrl; N Collection Nouvelle entrée
Cette commande ouvre L'éditeur d'entrées pour modifier de nouvelles entrées.
Ctrl; E Collection Éditer l' entrée
Cette commande ouvre L'éditeur d'entrées pour modifier les entrées sélectionnées.
Ctrl; Y Collection Dupliquer l' entrée
Cette commande copie ou duplique les entrées sélectionnées comme de nouvelles entrées dans la collection.
Ctrl; D Collection Supprimer l' entrée
Cette commande supprime les entrées sélectionnées.
Collection Fusionner les entrées
Cette commande fusionne les entrées sélectionnées.
Collection Mettre l'entrée à jour
Ce menu contient une liste de toutes les sources disponibles, et peut être utilisé automatiquement pour lancer une requête vers une source et mettre à jour les entrées sélectionnées.
Collection Gestion des prêts...
Cette commande ouvre une boîte de dialogue pour gérer les prêts des entrées sélectionnées.
Collection Retour de prêt
Cette commande vérifie si les entrées sélectionnées sont toujours en prêt.
Ctrl; R Collection Renommer la collection
Cette commande ouvre une boîte de dialogue pour renommer la collection.
Ctrl; U Collection Champs de la collection...
Cette commande ouvre la Bopite de dialogue des champs de collection.
Collection Générer des rapports...
Cette commande ouvre la Boîte de dialogue de rapport pour générer des rapports à propos de la collection.
Collection Convertir en bibliographie
Cette commande convertit une collection de livres vers une bibliographie en ajoutant certains champs nécessaire à l'exportation vers BibTeX, et celle -ci est activée seulement quand une collection de livre est ouverte.
Collection Ligne de Macros...
Cette commande ouvre une boîte de dialogue pour modifier les macros d'entrées BibTeX dans la collection. Celle -ci est activée à l'édition d'une bibliographie.
Collection Copier au format BibTeX vers le presse-papiers
Cette commande copie une citation BibTeX vers le presse-papiers pour être collée dans un document LaTeX.
Collection Citer l'entrée dans LyX
Cette commande envoie une citation pour les entrées sélectionnées vers lyxpipe pour utiliser dans LyX, Kile, ou d'autres applications LaTeX. Celle -ci est activée seulement à l'édition de la bibliographie.
Le menu Configuration
Configuration Barre d' outils
Cette commande permute l'affichage de la barre d'outils.
Configuration Afficher la barre d' état
Cette commande permute l'affichage de la barre d'état.
Configuration Afficher la vue par groupes
Cette commande permute l'affichage de la Vue par groupes.
Configuration Afficher l'éditeur d' entrées
Cette commande permute l'affichage de l'éditeur d'entrées.
Configuration Afficher la vue par entrées
Cette commande permute l'affichage de l'éditeur d'entrées.
Configuration Options de regroupement
Cette commande modifie le champ utilisé pour grouper les entrées dans la collection.
Configuration Configurer les raccourcis clavier...
Cette commande ouvre la Boîte de dialogue de configuration des raccourcis clavier de & kde;.
Configuration Configurer les barre d'outils...
Cette commande ouvre la Boîte de dialogue de configuration de la barre d'outils de & kde;.
Configuration Configurer & appname;...
Cette commande ouvre la Boîte de dialogue de configuration de & appname;.

& help. menu. documentation;
Séparé des éléments du menu Aide standard de & kde;, décris ci-dessus, & appname; possède les entrées de menu supplémentaires & #160;:
Aide Astuce du jour
Cette commande ouvre la boîte de dialogue d'astuce du jour. Vous pouvez faire défiler toutes les astuces en utilisant les boutons de la boîte de dialogue.
