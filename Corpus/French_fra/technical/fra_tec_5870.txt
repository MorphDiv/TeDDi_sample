Introduction
Kard est un jeu de memory pour les enfants. L'écran affiche des cartes cachées. Retournez deux cartes et associez -les. Quand les cartes forment une paire, elles disparaissent. Le jeu se termine quand toutes les cartes ont disparues.
Caractérisitiques de Kard:
un nombre de cartes de 4 à 24. un mode 4 cartes (2 paires) permet aux toutspetits enfants de commencer à controller la souris.
4 thèmes différents: les couleurs, les objets de la maison, les oppositions géométriques et les syllabes.
un minuteur qui vous permet de choisir combien de temps les cartes sont affichées
un mode plein écran
Utilisation de Kard
La principale fenêtre de Kard est composée des cartes cachées, d'une barre de menu, d'une barre d'outils et d'une barre de statut. Vous devez cliquez sur 2 cartes, la carte est ensuite retournée et le dessin de la carte apparaît. Si les deux cartes sont les mêmes, elles sont retirés. Vous finissez le jeu quand toutes les paires ont été trouvées.
Voilà une capture d'écran de Kard quand vous le lancez pour la première fois
La fenêtre principale de Kard
Kard la première fois que vous le lancez: 16 cartes par défaut et le thème est "colors".
Vous pouvez choisir dans la barre d'outils ou dans Configuration Configurer Kard... combien de cartes vous voulez afficher. 4 cartes et le mode le plus simple pour les petits enfants tandis que 24 cartes le mode le plus dure.
