Définissez les assignations d'événements de la boîte de dialogue ou de l'élément de contrôle sélectionné.
Les événements disponibles dépendent du type d'élément de contrôle sélectionné.

Réception de focus
Cet événement se produit lorsqu'un contrôle reçoit le focus.
Perte de focus
Cet événement se produit lorsqu'un contrôle perd le focus.
Touche enfoncée
Cet événement se produit si l'utilisateur appuie sur une touche lorsque le focus porte sur le contrôle.
Après avoir lâché la touche
Cet événement se produit si l'utilisateur relâche une touche lorsque le focus porte sur le contrôle.
Modification
Cet événement se produit si le contenu d'un contrôle est modifié alors que le contrôle a perdu le focus.
Texte modifié
Cet événement se produit lorsque vous saisissez ou modifiez du texte dans un champ de saisie.
Statut modifié
Cet événement se produit si le statut du champ de contrôle est modifié, par exemple lorsqu'il passe de l'état actif à l'état inactif.
Souris à l'intérieur de
Cet événement se produit lorsque le curseur entre dans le contrôle.
Déplacement de la souris et touche enfoncée
Cet événement se produit lorsque vous faites glisser la souris tout en maintenant une touche enfoncée.
Déplacement de la souris
Cet événement se produit lorsque la souris se déplace sur le contrôle.
Bouton de souris enfoncé
Cet événement se produit si l'utilisateur appuie sur le bouton de la souris lorsque le pointeur est positionné sur le contrôle.
Bouton de souris relâché
Cet événement se produit si vous relâchez le bouton de la souris lorsque le pointeur est positionné sur le contrôle.
Souris à l'extérieur de
Cet événement se produit lorsque le curseur sort du contrôle.
À l'ajustement
Cet événement se produit lorsque vous faites glisser une barre de défilement.
