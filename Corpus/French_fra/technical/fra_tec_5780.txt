Ressources de & kde;
Ce module est un outil de configuration pour gérer les ressources utilisées par différentes applications de & kde;. Ceci vous permet d'ajouter, supprimer ou modifier les ressources pour les Alarmes, l'Agenda, les Notes et les Contacts.
Sélectionner un type de ressource depuis la boîte déroulante en haut de la section Ressources. La liste affiche maintenant les noms et types des ressources disponibles de ce type. Une de ces ressources est affichée comme la ressource standard.
Utiliser les boutons sur le coté droit de la liste pour ajouter, supprimer ou modifier des ressources.

Chaque type de ressources a besoin d'une ressource standard, où toutes les données doivent être enregistrées si aucune autre ressource n'est spécifiée. Si vous voulez modifier la ressource standard, utilisez le bouton Utiliser comme standard pour le sélectionner. Vous devez avoir les droits d'accès en lecture et écriture pour cette nouvelle ressource standard, sinon vous ne pourrez pas la sélectionner.
