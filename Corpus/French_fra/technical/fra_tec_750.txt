Langue pour
Interface utilisateur
Sélectionnez la langue utilisée pour l'interface utilisateur, c'est-à-dire pour les menus, les boîtes de dialogue, les fichiers d'aide, etc.
Vous devez avoir installé au moins un pack de langue supplémentaire pour une version multi-lingue de %PRODUCTNAME.
Paramétrage de l'environnement linguistique
Spécifie les paramètres linguistiques pour le pays.
Ils affectent la numérotation, la monnaie et les unités de mesure.
Toute modification dans ce champ est immédiatement appliquée.
Néanmoins, certains formatages par défaut ne sont modifiés que si le document vient d'être chargé.
Touche séparateur de décimales - Identique à l'environnement linguistique
Spécifie que la touche séparateur de décimales définie dans votre système sera utilisée lorsque vous appuierez sur la touche correspondante du clavier numérique.
Si cette option est cochée, le caractère qui s'affiche après &quot; Identique à l'environnement linguistique &quot; est inséré lorsque vous appuyez sur la touche du clavier numérique.
Si cette option n'est pas cochée, le caractère inséré est celui que fournit le logiciel de votre pilote de clavier.
Monnaie par défaut
Spécifie la monnaie par défaut utilisée pour le format monétaire et les champs monétaires.
Si vous modifiez l'environnement linguistique, la monnaie par défaut est automatiquement modifiée.
L'entrée par défaut est appliquée au format monétaire attribué à l'environnement linguistique sélectionné.
Monnaie par défaut
est transmise à tous les documents ouverts et entraîne les changements correspondants dans les boîtes de dialogue et les icônes qui commandent le format monétaire dans ces documents.
Langues par défaut des documents
Spécifie les langues pour la vérification orthographique, le dictionnaire de synonymes et la coupure des mots.
Occidental
Indique la langue utilisée pour la fonction de vérification orthographique dans les alphabets occidentaux.
Asiatique
Indique la langue utilisée pour la fonction de vérification orthographique dans les alphabets asiatiques.
Scripts complexes
Indique la langue de la vérification orthographique des scripts complexes.
Uniquement pour le document actif
Indique que le paramétrage des langues par défaut ne s'applique qu'au document actif.
