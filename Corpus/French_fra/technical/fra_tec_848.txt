Exécution de commandes SQL
Grâce aux commandes SQL, vous pouvez contrôler la base de données directement, mais aussi créer et éditer des tables et des requêtes.
Tous les types de base de données supportent les instructions SQL.
Renseignez-vous, le cas échéant, sur les instructions SQL supportées par votre système de base de données.
