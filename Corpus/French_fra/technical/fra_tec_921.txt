Impression avec des données réduites
Il est possible de réduire les données nécessaires pour imprimer un document.
Les paramètres peuvent être définis différemment selon que vous voulez imprimer le document via l'imprimante ou dans un fichier.
Outils - Options - $[officename] - Impression
.
Cliquez sur l'une des options de paramétrage suivantes :
Imprimante
:
permet de définir les options permettant de réduire les données lorsque vous imprimez un document via une imprimante
Imprimer dans un fichier
:
permet de définir les options permettant de réduire les données lorsque vous imprimez un document dans un fichier
Sélectionnez les options souhaitées, puis cliquez sur OK.
Les changements d'options effectués s'appliqueront ensuite à tous les documents que vous imprimerez.
Impression d'un document.
Vous pouvez réduire les données relatives à la transparence, les dégradés ou les bitmaps.
Sur la plupart des imprimantes, la réduction des données n'affecte pas la qualité d'impression.
L'impression connaît toutefois une nette accélération, et si vous imprimez dans un fichier, celui-ci est bien moins volumineux.
