Options de filtre ASCII
Vous pouvez spécifier les options (police standard, langue, jeu de caractères ou saut) avec lesquelles le document texte doit être importé ou exporté.
La boîte de dialogue s'affiche lorsque vous chargez un fichier ASCII avec le filtre &quot; Texte codé &quot; , lorsque le document est enregistré pour la première fois, ou lorsqu'il est enregistré sous un autre nom.
Propriétés
Définit les paramètres pour l'import ou l'export du fichier.
Seuls le jeu de caractères et le saut de paragraphe peuvent être définis lors de l'export.
Jeu de caractères
Spécifie le jeu de caractères du fichier pour l'export ou l'import.
Police de base
En définissant une police de base, vous indiquez que le texte devra être affiché dans une police spécifique.
Les polices de base ne peuvent être sélectionnées que lors de l'import.
Langue
Spécifie la langue du texte, si elle n'a pas déjà été définie.
Ce paramètre n'est disponible que lors de l'import.
Saut de paragraphe
Définit le type de saut de paragraphe pour une ligne de texte.
CR & LF
Génère un &quot; retour chariot &quot; et un &quot; saut de ligne &quot; .
Il s'agit de l'option par défaut.
CR
Génère un &quot; Retour chariot &quot; lors du saut de paragraphe.
LF
Génère un saut de ligne au niveau du saut de paragraphe.
