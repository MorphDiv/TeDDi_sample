Ces fonctions essaient de définir le type de contenu et l'encodage d 'un fichier en recherchant certaines séquences magiques, à certaines positions dans le fichier.
Même si ce n 'est pas une approche imparable, les heuristiques réalise un bon travail.
Cette extension est dérivée du moduel Apache mod_mime_magic, qui est lui-même basé sur la commande file, entretenue par Ian F.

Voyez le code source pour plus de détails sur les heuristiques et les informations de copyright.
Ces fonctions sont disponibles dans le module PHP standard, qui est toujours accessible.
L 'extension nécessite une copie de magic.mime tel que distribué avec la commande file.
Ce fichier fait partie des distributions Linux les plus récentes, et il est généralement stocké dans le dossier / usr / share / misc.
Le comportement de ces fonctions est affecté par la configuration dans le fichier php.ini.
