KNotes est une application vous permettant de créer des post-it sur votre bureau.
Introduction
KNotes est un programme qui vous permet d'écrire l'équivalent informatique des post-it.
Les notes sont sauvegardées automatiquement lorsque vous quittez le programme, et sont affichées lorsque vous démarrez le programme.
Vous pouvez imprimer vos notes et les envoyer par courrier électronique si vous configurez KNotes pour qu'il utilise des applications annexes.
Les options d'affichage des notes, telles que la couleur et la police peuvent être personnalisées pour chaque note.
Vous pouvez aussi personnaliser les paramètres par défaut.
Configuration
Configurer l'affichage dans KNotes
Cliquez avec le bouton droit de la souris sur l'icône dans le tableau de bord.
Choisissez Configuration
La boîte de dialogue Paramètres par défaut de KNotes va apparaître.
Choisissez l'icône Affichage parmi les trois icônes de gauche.
Couleur du texte
Le rectangle de couleur montre la couleur courante du texte.
En cliquant sur ce rectangle de couleur, vous ouvrirez la boîte de dialogue standard de sélection de couleur de KDE.
Couleur du fond
Le rectangle de couleur montre la couleur courante du fond.
En cliquant sur ce rectangle de couleur, vous ouvrirez la boîte de dialogue standard de sélection de couleur de KDE.
Largeur par défaut
La largeur de la note en pixels.
Modifiez ce nombre à votre convenance.
Hauteur par défaut
La hauteur de la note en pixels.
Modifiez ce nombre à votre convenance.
Configurer l'éditeur dans KNotes
Cliquez avec le bouton droit de la souris sur l'icône dans le tableau de bord.
Choisissez Configuration
La boîte de dialogue Paramètres par défaut de KNotes va apparaître.
Choisissez l'icône Éditeur parmi les trois icônes de gauche.
Longueur des tabulations
Ceci est la taille des tabulations en espaces.
Modifiez ce nombre à votre convenance.
Indentation automatique
Ceci est une case à cocher.
Si elle est cochée, l'auto-indentation est activée.
Modifier du texte enrichi
Pas encore implémenté
Cliquez sur ce bouton pour ouvrir la boîte de dialogue standard de sélection de police de KDE.
Cliquez avec le bouton droit de la souris sur l'icône de KNotes dans le tableau de bord.
Choisissez Configuration
La boîte de dialogue Paramètres par défaut - kNotes de KNotes va apparaître.
Choisissez l'icône Actions parmis les trois icônes de gauche.
Courrier électronique
Tapez une commande de courrier et tout paramètre de ligne de commande requis dans ce champ.
En utilisant %f dans la ligne de commande, vous pouvez passer le nom de fichier du corps de la note à la commande de courrier.
Utilisation de KNotes
Création d'une nouvelle note
Cliquez avec le bouton droit sur l'icône de KNotes dans le tableau de bord.
Choisissez Nouvelle note.
Écriture de votre note
Pour écrire votre note, tapez simplement votre note dans l'espace prévu à cet effet.
Les fonctions normales du clavier et d'édition à la souris sont supportées.
Annuler
Refaire
Couper
Copier
Coller
Effacer
Choisisser Tout
Insertion de la date
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Insérer la date.
Renommage d'une note
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Renommer.
Envoi d'une note par courrier électronique
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Envoyer.
Impression d'une note
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Imprimer.
Suppression d'une note
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Supprimer.
Cacher une note
Pour cacher une note, cliquez sur le X dans le coin supérieur droit de la barre de titre de la note.
La note ne sera plus affichée à l'écran.
La note en elle-même ne sera pas supprimée.
Affichage de notes
Cliquez avec le bouton droit de la souris sur l'icône de KNotes dans le tableau de bord.
Choisissez Notes.
Choisissez la note que vous voulez afficher.
Fonctions de bureau
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Vers le bureau
Choisissez le bureau désiré, ou bien Tous les bureaux
Cliquez avec le bouton droit dans la barre de titre de la note.
Choisissez Toujours visible.
Personnalisation de l'affichage d'une note
Vous pouvez personnaliser chaque note indépendamment des réglages globaux.
Les réglages que vous pouvez personnaliser sont identiques à ceux décrits dans la section sur la Configuration, mais ne s'appliqueront qu'à la note que vous modifiez.
Toutes les autres notes ouvertes, ainsi que toutes les notes que vous créerez par la suite, utiliseront les réglages par défaut.
Cliquez avec le bouton droit sur la barre de titre de la note.
Choisissez Configuration de la note....
La boîte de dialogue Paramètres locaux va s'ouvrir.
Choisissez l'icône Affichage parmi les trois icônes de gauche.
Sortie de KNotes
Cliquez avec le bouton droit de la souris sur l'icône dans le tableau de bord.
Choisissez Quitter.

Installation
Comment obtenir KNotes
KNotes fait partie du projet KDE http ://www.kde.org/.
KNotes se trouve dans le paquetage kdepim à l'adresse ftp ://ftp.kde.org/pub/kde/, le site FTP principal du projet KDE.
Programmes requis
Pour imprimer ou envoyer des notes par courrier électronique, vous avez besoin d'avoir des applications assistantes installées, telles que a2ps et KMail.
Vous allez avoir besoin de configurer KNotes pour utiliser ces applications assistantes.
