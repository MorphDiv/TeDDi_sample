mb_http_input() retourne le type d 'encodage utilisé par une requête HTTP.
Le paramètre type Le paramètre type spécifie le type d 'entrée HTTP.
Il peut prendre l'une des valeurs suivantes: "G "pour GET, "P" pour POST, "C "pour COOKIE.
Si type est omis, il prend la valeur du dernier type utilisé.
Valeur retournée: nom de l 'encodage utilisé.
Si mb_http_input() ne peut traiter ce type d 'encodage, elle retourne FALSE.
Voir aussi mb_internal_encoding(), mb_http_output() et mb_detect_order()
