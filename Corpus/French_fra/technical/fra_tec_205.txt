Mode Insertion
Affiche le mode d'insertion actif.
Vous pouvez basculer entre INS = insérer et RFP = écraser.
Ce champ n'est actif que si le curseur se trouve sur la ligne de saisie de la barre de formule ou dans une cellule.
Inser
est active).
Si le curseur est positionné dans un document, vous pouvez aussi utiliser la touche Inser pour basculer entre les différents modes.
Mode :
Effet :
INS
En mode Insertion, les nouveaux caractères sont insérés à l'emplacement du curseur ; le texte suivant est décalé vers la droite.
Le curseur est représenté par un trait vertical.
RFP
En mode Écrasement, les caractères existants sont écrasés par de nouveaux.
Le curseur est représenté par un bloc.
