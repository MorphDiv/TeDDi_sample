Couleurs;modèles
Couleurs
Permet de sélectionner une couleur dans une palette de couleurs, de modifier une couleur existante ou de définir de nouvelles couleurs.
Palette de couleurs
Nom
Spécifie le nom d'une couleur sélectionnée.
Vous pouvez également saisir un nom dans ce champ lors de la définition d'une nouvelle couleur.
Couleur
Contient une liste des couleurs disponibles.
Sélectionnez une couleur à partir de la liste des couleurs disponibles.
Modèle de couleurs
Pour apporter une modification, sélectionnez le modèle de couleurs :
RVB (Rouge Vert Bleu) ou CMJN (Cyan Magenta Jaune Noir).
%PRODUCTNAME utilise seulement le modèle de couleur RGB pour imprimer en couleur.
Les contrôles CMYK sont seulement fournis pour aider la saisie des valeurs de couleur utilisant la notation CMYK.
Si vous avez sélectionné RVB, vous pouvez voir les sigles des trois couleurs à côté desquels un compteur vous permet de définir la valeur de la couleur correspondante de 0 à 255.
R
Rouge
V
Vert
B
Bleu
Si vous avez sélectionné CMYK, vous pouvez voir les sigles des quatre couleurs à côté desquels un compteur vous permet de définir la valeur de la couleur correspondante de 0 à 255.
C
Cyan
M
Magenta
J
Jaune
N
Noir
Ajouter
Ajoute une nouvelle couleur.
Modifier
Change la couleur active.
Notez qu'aucune confirmation ne s'affiche lorsque la couleur est remplacée.
