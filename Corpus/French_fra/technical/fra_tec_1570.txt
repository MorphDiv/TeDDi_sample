Fonction Error [Exécution]
Renvoie le message d'erreur correspondant à un code d'erreur donné.
Syntaxe :
Error (Expression)
Valeur de retour :
Chaîne de caractères
Paramètres :
Expression :
Toute expression numérique qui contient le code du message d'erreur à renvoyer.
Si aucun paramètre n'est transmis, la fonction Error renvoie le message d'erreur correspondant à la dernière erreur s'étant produite lors de l'exécution du programme.
