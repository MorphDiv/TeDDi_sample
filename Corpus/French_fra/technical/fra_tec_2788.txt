cpdf_import_jpeg() ouvre une image JPG, enregistré dans le fichier file_name.
Le format de l 'image doit être JPEG.
L'image est placée dans la page courante, aux coordonnées (x-koor, y-koor).
L'image subira une rotation d 'un angle de angle degrés.
Le paramètre mode est une unité de longueur.
S'il prend la valeur de 0 (ou s'il est omis), c'est la valeur par défaut (72) qui est utilisée.
Voir aussi cpdf_place_inline_image().
