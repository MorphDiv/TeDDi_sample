Options de KWord
Configurer les raccourcis-claviers
Ceci ouvrira une boîte de dialogue.
Pour modifier les raccourcis-claviers, la première chose que vous devez faire est de trouver l'action que vous voulez modifier.
Toutes les actions possibles sont listées dans la boîte combo libellée Action.
En utilisant l'ascenseur, localisez l'action qui vous intéresse et cliquez une fois du bouton gauche de la souris.
Une fois que vous avez sélectionné l'action, vous pouvez porter votre attention sur la moitié basse de la boîte de dialogue.
Vous utilisez la moitié du bas de la boîte de dialogue pour changer les raccourcis-claviers.
Aucun raccourci
En sélectionnant cette option, l'action actuellement sélectionnée n'aura aucun raccourci-clavier.
Raccourci par défaut
En sélectionnant cette option, l'action actuellement sélectionnée utilisera le raccourci par défaut.
Une fois que cette option est cliquée, la colonne de raccourci-clavier au-dessus liste les touches par défaut.
Raccourci personnalisé
En sélectionnant cette option, l'action actuellement sélectionnée utilisera n'importe quelle combinaison de touches que vous choisirez.
Vous déterminez la combinaison de touches en plaçant les coches appropriées libellées Shift, Ctrl, et Alt.
Vous pouvez sélectionner toute combinaison de touches.
Vous pouvez aussi cliquer sur le grand bouton portant une lettre.
Maintenant, tapez la lettre ou la touche de fonction que vous voulez utiliser.
La lettre changera en fonction de la lettre tapée.
Quand la combinaison est correcte, vous pouvez soit sélectionner une autre action, soit presser OK pour enregistrer les changements.
Si vous cliquez sur le bouton Annuler, vous retournerez à l'édition de votre document, sans enregistrer les changements.
En cliquant sur le bouton Par défaut, vous changerez tous les raccourcis-claviers à leur valeur par défaut.
Configurer les barres d' outils
Ceci ouvrira une boite de dialogue.
Vous pouvez ajouter ou enlever autant de barres d'outils que vous voulez.
Vous pouvez aussi déplacer les boutons sur la barre.
Ajouter un bouton à une barre d' outils
Pour ajouter un bouton d'une barre d'outils, vous le déplacez de la boîte Actions disponibles à la colonne Actions actuelles.
Sélectionnez d'abord la barre d'outils à laquelle vous voulez ajouter un bouton en sélectionnant la barre d'outils dans la boîte déroulante libellée Barre d' outils
Maintenant, sélectionnez l'action que vous voulez ajouter dans la boîte combo libellée Actions disponibles en cliquant une fois avec le bouton gauche de la souris.
Cliquez maintenant sur l'action dans la boite combo libellée Actions actuelles sur laquelle vous voulez poser votre bouton.
Cliquez sur la flèche bleue pointant vers la droite.
Une fois que le bouton OK a été cliqué, les barres d'outils changeront.
Supprimer un bouton de la barre d' outils
Pour supprimer un bouton de la barre d'outils, vous pouvez le déplacer de la boîte Actions actuelles à la colonne Actions disponibles.
Sélectionnez d'abord le bouton de barre d'outils dans laquelle vous voulez enlever le bouton dans la boite combo libellée Actions actuelles.
Cliquez sur la flèche bleue pointant vers la gauche.
Une fois que le bouton OK a été cliqué, les barres d'outils changeront.
Déplacer un bouton sur une barre d' outils
Sélectionnez d'abord la barre d'outils dans laquelle vous voulez déplacer un bouton en sélectionnant la barre d'outils dans la boîte déroulante libellée Barre d' outils
Pour déplacer un bouton, cliquez simplement sur le bouton que vous voulez déplacer avec le bouton gauche de la souris.
Cliquez sur la flèche Haut ou Bas pour déplacer le bouton respectivement vers le haut ou vers le bas.
Une fois que le bouton OK a été cliqué, les barres d'outils changeront.
Options de KWord
Ceci ouvrira une boîte de dialogue.
Configurer la correction d'orthographe.
Pour configurer les options pour le correcteur, cliquez sur le bouton libellé Vérification de l'orthographe.
Si vous cochez la case libellée Créer les combinaison Radical/Affixe absentes du dictionnaire, quand KWord rencontre un mot racine en combinaison avec un préfixe ou un suffixe pour le mot combiné, KWord ajoutera une nouvelle entrée à votre dictionnaire.
Si elle n'est pas cochée, KWord continuera à rapporter ce mot comme fautif.
La case à cocher suivante vers le bas repérera les mots combinés.
Des exemples de tels mots sont shutout, cannot, et blackout.
Laisser cette case non cochée, empêchera KWord de corriger les sites web et les adresses email.
Ces adresses contiennent souvent des mots accolés.
La boîte déroulante Dictionnaire set à sélectionner le dictionnaire que vous voulez faire utiliser par KWord.
La boite déroulante Encodage peut servir à spécifier quel jeu de caractères vous voulez faire utiliser par le correcteur.
La boite déroulante Client sert à spécifier le programme de correction d'orthographe que KWord doit utiliser.
Quand vous êtes content de vos changements, cliquez simplement sur OK.
Si vous cliquez sur Annuler, tous les changements seront perdus.
Cliquer sur le bouton Par Défaut restaure toutes les valeurs à leur valeur par défaut.
Configurer l'interface utilisateur de KWord.
Un clic sur le bouton Interface vous permettra de changer les choses suivantes.
En utilisant la case à cocher libellée Afficher les règles pour inverser les règles sur l'écran de KWord.
Le reste des controles peuvent soit être effectués soit en utilisant les flèches de boîtes à chiffres, en saisissant la valeur directement ou en faisant glisser le curseur vers la droite de la boîte.
Auto-enregistrement
Vous pouvez utiliser ceci pour ajuster la fréquence à laquelle KWord enregistre un fichier temporaire.
Si vous fixez cette valeur à zero, KWord ne fera pas d'enregistrement automatique.
Vous pouvez régler cette valeur de zero à 60 minutes.
Nombre de fichiers récents
Vous pouvez ajuster cette valeur de 1 à 20.
Espacement de la grille en X
Espacement de la grille en Y
Indentation
Mode aperçu - Nombre de pages par ligne
Quand vous êtes content de vos changements, cliquez simplement sur OK.
Si vous cliquez sur Annuler, tous les changements seront perdus.
Cliquer sur le bouton Par Défaut restaure toutes les valeurs à leur valeur par défaut.
