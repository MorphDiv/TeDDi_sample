Périphériques
Cette boîte de dialogue affiche les systèmes de fichiers disponibles et quelques informations pour chaque périphérique.
Elle vous permet également de configurer les points de montage et le montage/démontage des périphériques.
Une icône qui décrit le type de stockage
Le format de fichier
La taille totale
Le point de montage
L'espace disque disponible
Le pourcentage d'espace disque utilisé représenté par un nombre et un graphique.
En cliquant avec le bouton droit, on obtient l'affichage d'un menu flottant.
Si le périphérique n'est pas monté, l'option Monter le périphérique est disponible.
Si le périphérique est monté, il peut être démonté en choisissant l'option Démonter le périphérique Sélectionnez Ouvrir le gestionnaire de fichiers pour ouvrir dans une nouvelle fenêtre une vue graphique des fichiers de ce périphérique.
