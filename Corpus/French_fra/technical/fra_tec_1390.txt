Imprimer
Indique les paramètres d'impression d'un document texte ou HTML.
Options
de la boîte de dialogue d'impression, accessible par la commande
Fichier - Imprimer
.
Contenu
Indique le contenu du document à imprimer.
Images
Indique si les images du document texte doivent être imprimées.
Tables
Indique si les tableaux du document texte doivent être imprimés.
Dessins (sauf documents HTML)
Indique si les images créées à l'aide des fonctions de dessin de $[officename] doivent être imprimées.
Commandes
Indique si les champs de contrôle de formulaire du document texte doivent être imprimés.
Arrière-plan
Indique si les couleurs et objets contenus dans l'arrière-plan de la page (Format - Page - Arrière-plan) doivent être inclus dans le document imprimé.
Imprimer en noir
Indique si le texte doit toujours être imprimé en noir.
Texte masqué
Activez cette option pour imprimer le texte masqué.
Le texte masqué suivant est imprimé : le texte qui a été formaté comme masqué par
Format - Caractères - Effets de caractères - Masqué
et les champs texte
Texte masqué et Paragraphes masqués
.
Substituant de texte
Activez cette option pour imprimer les substituants de texte.
Désactivez cette option pour laisser les substituants de texte en blanc lors de l'impression.
Les substituants de texte
sont des champs.
Pages
Définit l'ordre d'impression des documents $[officename] Writer comportant plusieurs pages.
Pages de gauche (sauf documents HTML)
Indique si toutes les pages de gauche (numérotées paires) du document doivent être imprimées.
Pages de droite (sauf documents HTML)
Indique si toutes les pages de droite (numérotées impaires) du document doivent être imprimées.
Inversé
Indique si l'ordre d'impression doit être inversé.
La dernière page du document est imprimée en premier.
Brochure
Brochure
pour imprimer votre document au format brochure.
Ce format est le suivant dans $[officename] Writer :
Pages de droite
, et en réinsérant ensuite la pile de papier dans l'imprimante afin d'imprimer le verso des pages après avoir coché la case
Pages de gauche
.
De droite à gauche
Cochez pour imprimer les pages de la brochure dans l'ordre correcte pour un script de droite à gauche.
Commentaires
Aucun(e)
Indique que les commentaires du document ne sont pas imprimés.
Commentaires seuls
Indique que les commentaires du document doivent être imprimés, mais non le texte même.
Fin de document
Indique que les commentaires doivent être imprimés à la fin du document, après le texte.
Fin de page
Indique que les commentaires doivent être imprimés sur une page supplémentaire ajoutée après chaque page contenant des commentaires.
Autres
Imprimer automatiquement les pages blanches insérées
Si cette option est activée, les pages blanches insérées automatiquement sont imprimées.
Cette option s'utilise en principe pour les impressions recto-verso.
Exemple : les chapitres d'un livre doivent toujours commencer sur une page impaire.
Le chapitre précédent se termine sur une page impaire. %PRODUCTNAME insère une page paire vierge.
Cette option permet de définir si cette page vierge doit être imprimée.
Création de travaux d'impression individuels
Indique que chaque nouveau travail d'impression doit commencer sur une nouvelle page, même si une imprimante recto verso est utilisée.
Si cette case n'est pas cochée, il est possible que la première page du second exemplaire s'imprime sur le verso de la dernière page du premier exemplaire, en particulier si le nombre de pages est impair.
Bac d'après les paramètres de l'imprimante
Pour les imprimantes équipées de plusieurs bacs d'alimentation, l'option &quot; Bac d'après les paramètres de l'imprimante &quot; indique si le bac utilisé est défini par les paramètres système de l'imprimante.
Fax
Si vous avez installé un logiciel de fax sur votre ordinateur et souhaitez envoyer un fax directement depuis le document texte, sélectionnez le périphérique de fax à utiliser.
