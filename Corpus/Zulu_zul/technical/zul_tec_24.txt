Tshengisa zonke izindawozokusebenzela
Uma kuyiqiniso, umqhebezi wendawo yokusebenzela uzobonisa zonke izindawo zokusebenzela. Ngaphandle kwalokho izobonisa indawo yokusebenzela yamanje.
Imidenda kumqhebezi endaweni yokusebenzela

Nika isiqiniseko kususwa kwendawo yohlu
Uma kuyiqiniso, ibhokisi lenkulumo liyaboniswa licela isiqiniseko sokuthi umsebenzisi ufuna ukususa indawo yohlu.
Umazisi yohlu lwendawo yohlu
Uhlu lomazisi kundawo yohlu. Umazisi ngamunye ubekisa indawo yohlu ephuzulu ngayinye. Izinhlelo zalezindawo zohlu ngayinye zigcinwe /apps/panel/general/$(profile)/toplevels/$(id).
Umazisi yohlu lwento yendawo yohlu
Uhlu lomazisi bento yendawo yohlu. Umazisi ngamunye ubekisa into yohlu ngayinye (jengesibonelo isiqalisi, inkinobho yokwenza noma inkinobho/isiqobolo sohlu). Izinhlelo zalezizinto ngayinye zigcinwe /apps/panel/general/$(profiles)/objects/$(id).
Indawo yohlu ebopheka ngokuphelele
Uma kuyiqiniso, indawo yohlu ingevumele inguquko kusilinganisi sendawo yohlu. Omayela ngayedwa komele baboshelwe ngokwehlukane neze. Indawo yohlu kumele iqala phansi ukuze lokhu kwenzeke.
Umazisi womayelana wokukhinendwa ekuhlohlweni
Uhlu lomazisi bomayelana indawo yohlu engezikubanaka. Ngalendlela ungakhinenda omayela abathize ekuhlohleni noma ekubonisenni uhlu. Njengesibonelo ukukhinenda umlayeli-omcane kamayelana wengeze 'OAFIID:GNOME_MiniCommanderApplet' kuloluhlu. Indawo yohlu kumele iqala phansi ukuze lokhu kwenzeke.

Isabelo sehele lesithombe esisetshenziswa njengomboniso-sithombe kunkinobho yento. Lesixhumanisi sibaluleke uma object_type lingu "drawer-object" noma "menu-object" ne use_custom_icon sithombe sishuxulelo kuyiqiniso.
Indlela yokuqukethwe uhlu
Indlela lapho okuqukethwe uhlu kwakhiwa khona. Lesishuxulelo sibalulekile uma isishuxulelo sika use_menu_path siyiqiniso object_type yesishuxulelo esingu "menu-object".

Humusha isikhundla mayelana nophansi/kwesokudla siphethweni
Uma kuyiqiniso, isikhundla sento sihunyushwa ngokuphathelene nesokudla (noma phansi uma simile thwi) esiphethelweni sendawo yohlu.
Isikhundla sento endaweni yohlu
Isikhundla sento yalendawo yohlu. Isikhundla sichazwa inani lamapixels kwesenxele (noma phezulu uma imele thwi) esiphethweni sendawoyohlu.
Igama lokusho indawo yohlu
Leligama elingafundwa umuntu ongalisebenzisa ukuchaza indawo yohlu. Ingqikithi yenhloso yalo ukusebenzanjesihloko sefasitela lendawo yohlu ezosetshenziswa umakuyaluzwa phakathi kwezindawo zohlu.
Isibuko X lapho indawo yohlu itshengiswa khona
Ngohlelo lwezibuko eziningi, ungaba nendawo yohlu kwesinye nesinye isibuko. Lesishuxulelo sisho isibuku samanje indawo yohlu itshengiswa kuso.
Imonitor yeXinerama lapho indawo yohlu itshengiswa khona
Kuhlelo leXinerama, ungaba nezindawo zohlu kuleyo-naleyo monitor. Lesishuxulelo sisho imonitor yamanje kwindawoyohlu etshengisiwe.
Nweba ukugcwalisa ububanzi besibuko
Uma kuyiqiniso, indawo yohlu izogcwala ububanzi besibuku sonke (ubude uma kuyindawo yohlu emile thwi).Kulesimo indawo yohlu ingabekwa esiphethweni sesibuko kuphela. Uma kuyiphutha, indawo yohlu izozimulika ngokwanele ukuhlalisa omayelana, iziqali nezinkinobho zendawo yohlu.
Indabuko yendawo yohlu
Indabuko yendawo yohlu. Amanani engenzeka "phezulu", "phansi", "kwesobunxele", "kwesokudla". Eimweni esinwetshiwe isishuxulelo sisho isiphethelo indawo yohlu ekuso. Esimweni esinganwetshwanga umehluko phakathi "phezulu" no"phansi" awubalulekanga-yomibili isho ukuthi kendawo yohlu eqonde dindilizi- kodwa isanika ukusikisela okusizayo ngokuthi ezinye izinto zendawo yohlo zenze njani. Njengesibonelo, ku "phezulu" ndawo yohlu inkinobho yohlu izolunguzisa uhlu lwayo ngaphansi kwendawo yohlu, kanti ku-"phansi" uhlu luzolunguzisa ngaphezu kwendawo yohlu.
Ubungakanani bendawo yohlu
Ubude (ububanzi kundawo yohlu imile thwi) bendawo yohlu. Indawo yohlu izonquma uma isebenza ubungakanani obuncane kusuka kuhlobo lombhalo nezinye izitshengiso. Ubungakanani obukhulu buphelela kwiquarter yobude besibuko (noma ububanzi).
Indawo yohlu ehlanganiswe u-x
Isabelo sendawo yohlu kumax-axis. Isishuxulela sibalulekile kuphela uma ingekho esimweni esinwetshiwe. Esimweni esinwetshiwe lesishuxulelo asinakwa nendawo yohlu ibekwa ephethelweni lwesibuko esishiyo isishuxulelo sendabuko.
Indawo yohlu enhlanganiswa u-y
Isabelo sendawo yohlu kumay-axis. Isishuxulelo sibalulekile uma singekho esimweni esinwebekile.Esimweni esinwebekile lesishuxulelo asinakwa nendawo yohlu ibekwa esiphethelweni esishiwo isishuxulelo sendabuko.

Beka maphakathi kwendawo yohlu ku x-axis
Uma kuyiqiniso, isishuxulelo x asinakwa nendawo yohlu ibekwa maphakathi ne x-axis esibukweni. Uma indawo yohlu elinganiselwa futhi izosala kulesosikhundla - kusho ukuthi indawo yohlu izokhula mahlangothi omabili. Uma kuyiphutha, isishuxulelo x isho isabelo sendawo yohlu.
Beka maphakathi indawo yohlu ku y-axis
Uma kuyiqiniso, isishuxulelo Y asinakwa nendawo yohlu ibekwa maphakathi kwe Y-axis esibukweni. Uma indawo yohlu elinganiselwa futhi ngokubukhulu izosala kuleyondawo - kusho ukuthi indawo yohlu izokhula mahlangothi omabili. Uma kuyiphutha, isishuxulelo Y siho esabelo sendawo yohlu.
Fihla indawo yohlu ekhoneni ngokuzenzakalela
Uma kuyiqiniso, indawo yohlu ifahlakala ekhoneni ngozenzakalela uma isikhombi sishiya isabelo sendawo yohlu. Ukuhambisa isikhombisi kulelokhona futhi kwenza indawo yohlu ivele futhi.
Gunyaza izinkinobho zokufihla
Uma kuyioqiniso, izinkinobho zizobekwa ezingxenyeni zendawo yohlu ngezetshenziswa ukuhambisa indawo yohlu esiphethelweni sesibuko, ishiye inkinobho kuphela ibonakala.
Gunyaza imicibisholo kuzinkinobho zokufihla
Uma kuyiqiniso, imicibisholo izobekwa phezu kwezinkinobho ezifihliwieyo. Lesishuxulelo sibalulekile uma isishuxulelo enable_buttons siyiqiniso.
Isibambezelo sefihla ngokuzenzakalela kwendawo yohlu
Isho inani lokubambezeleka ngemizuzwana emincane emuva kokuba isikhombisi sisuka esabelweni sendawo yohlu ngaphambi kokuzifihlakalela kwendawo yohlu. Lesishuxulelo sibalulekile uma isishuxulelo fihla_ngokuzenzakalela kuyiqiniso.
Isibambezelo sefihla-autoun kwendawo yohlu
Isho inani lokubambezeleka nemizuzwana emincane emuva kokuba isikhombi singene esabelweni sendawo yohlu ngaphambi kokuba indawo yohlu iphinde iziqhamukele ngokuzenzakalela. Lesishuxulelo sibalulekile uma isishuxulelo sefihla_ngokuzenzakalela siyiqniso.
Amapixels abonakalayo uma kufihliwe
