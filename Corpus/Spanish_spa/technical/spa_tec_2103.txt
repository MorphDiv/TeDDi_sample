Editar campos
Abre un diálogo donde puede editar las propiedades de un campo.
Haga clic delante de un campo y seleccione este comando.
En el diálogo puede usar las teclas de flecha para desplazarse al campo anterior o siguiente.
También puede pulsar dos veces sobre un campo del documento para abrirlo y editarlo.

Tipo de campo
Muestra el tipo de campo que está editando.
Los elementos de diálogo siguientes sólo están visibles cuando se selecciona el tipo de campo correspondiente.
Selección
Muestra las opciones del campo, como, por ejemplo, &quot; fijo &quot; .
Si lo desea, puede pulsar otra opción para el tipo de campo seleccionado.
Formato
Seleccione el formato del contenido del campo.
Para la fecha, la hora y los campos definidos por el usuario, también puede pulsar &quot; Otros formatos... &quot; en la lista y luego elija un formato diferente.
Los formatos disponibles dependen del tipo de campo que edite.
Corrección
Muestra el desplazamiento para el tipo de campo seleccionado.
Por ejemplo, para &quot; Página siguiente, &quot; &quot; Números de página &quot; o &quot; Página previa &quot; .
Se puede colocar un nuevo valor de desplazamiento que será agregado al número de página.

Seleccionar base de datos
Selecciona una base de datos registrada de la que desea insertar el campo seleccionado.

Número de registro
Muestra el número de registro de la base de datos que se inserta cuando la condición especificada en el tipo de campo &quot; Cualquier registro &quot; se cumple.
