Aplicaciones de kdeadmin
Se pueden encontrar las siguientes aplicaciones en las distribuciones de kdeadmin.
Las aplicaciones estables están relativamente libres de errores y se pueden encontrar en formato fuente y binario
ftp.kde.org / pub / kde / stable / latest / distribution / stable
Las versiones inestables cambian diariamente y sólo están disponibles en formato fuente, lo que quiere decir que usted debe compilarlas para generar una versión que funcione.
Recuerde que son inestables y que puede que ni siquiera compilen.
Si tiene un problema con una aplicación inestable informe del problema y probablemente será corregida en un versión futura.
ftp.kde.org / pub / kde / unstable / CVS / snapshots / current
Kcron
KDE ofrece un interfaz estético y cómodo para planificar tareas Cron
Kuser
Un administrador gráfico de usuarios (requiere privilegios de root para poder ser ejecutado)
KPackage
Mantiene el sistema actualizado con el administrador propio de paquetes de KDE.
Kdat
Una utilidad de copias de seguridad para propietarios de cintas.
ksysv
Un editor SysV-Init, para mantener organizadas las aplicaciones iniciadas durante el arranque del ordenador.
kwuftpd
Un interfaz gráfico para configurar el demonio FTP wuftpd.
