Formateo de párrafos
Esta sección detallará todas las opciones disponibles para dar formato a párrafos.
Para dar formato a un párrafo, ponga el cursor dentro del párrafo.
Para dar formato a más de un párrafo a la vez, simplemente seleccione los párrafos con el ratón o el teclado.
Puede dar formato a uno o más párrafos de una de estas tres formas:
Ponga el puntero del ratón en el párrafo y haga clic una vez con el right mouse button.
Aparecerá un pequeño menú.
Seleccione Párrafo....
Aparecerá un diálogo.
El diálogo Preferencias de párrafo consta de seis secciones de pestañas etiquetados Sangrado y espaciado, Alineación, Bordes, Viñetas / Numeración, Tabuladores y Sombra.
Sangrado y espaciado
Con este cuadro de diálogo se puede especificar el espaciado entre líneas y el espaciado entre párrafos.
La primera sección de este cuadro de diálogo está etiquetada Sangrado, y consta de tres partes:
Izquierda
Introduzca el valor de este cuadro para sangrar las líneas de los párrafos seleccionados fuera del margen izquierdo.
La primera línea no está afectada por esta caja.
Para alterar la primera línea, especifique ese valor en el cuadro de texto etiquetado Primera línea.
Derecha
Introduzca un valor en esta caja para sangrar todas las líneas del párrafo seleccionado (incluyendo la primera línea) lejos del margen derecho.
Primera línea
Introduzca un número en esta caja para sangrar la primera línea de un párrafo lejos del margen derecho.
La próxima sección controla cómo divide KWord los párrafos cuando un párrafo no se ajusta enteramente al marco o a la página.
La primera opción está etiquetada como Mantener las líneas juntas.
Si esta opción está seleccionada, todas las líneas del párrafo permanecerán en la misma página.
Si esto no se selecciona, KWord puede elegir mover parte del párrafo a una página o marco nuevo.
Para la mayoría del trabajo, esta opción se suele dejar sin marcar.
Las dos siguientes opciones están etiquetadas Insertar salto antes de párrafo y Insertar salto tras párrafo.
Cuando una de estas opciones se marcan, y el párrafo se mueve al marco siguiente del conjunto de marcos, se insertará un salto duro de marco antes o después del párrafo anterior (dependiendo de la opción seleccionada).
Esto le servirá para mantener el párrafo en el siguiente marco, incluso si el texto anterior de ese marco se borra.
Esta opción se suele usar en conjunto con Mantener las líneas juntas, para asegurarse de que el párrafo no retrocede en la página durante la edición.
La siguiente sección se etiqueta Espaciado de línea.
Consta de dos elementos:
Lista desplegable
Los valores seleccionados aquí se aplicarán a todas las línes de los párrafos seleccionados.
Cuadro de texto
Introduzca un valor en este cuadro de texto para especificar el espaciado entre párrafos.
Este valor sólo cambia el espaciado cuando se selecciona Personalizado en la lista desplegable de la izquierda.
Los cambios en esta entrada no afectan al espaciado entre líneas de diferentes párrafos.
La última sección está etiquetada Espacio de párrafo.
Consta de dos entradas:
Antes
Introduciendo un valor aquí, el espaciado adicional se añade antes de cada párrafo.
Después
Introduciendo un valor aquí, el espaciado adicional se añade después de cada párrafo.
A la derecha del cuadro de diálogo hay una caja de vista preliminar, que mostrará aproximadamente la distribución final de su documento.
Alineaciones
Esta sección determina cómo se posiciona el texto en la línea.
Con otras aplicaciones, puede haberse referido a esto como alineación, o justificación.
Sólo hay 4 opcione.
Si está seleccionado Izquierdo, Derecho o Centro, el texto se moverá en la línea de tal forma que se alinee con el margen izquierdo, el margen derecho o se centre entre los márgenes respectivamente.
Si se selecciona Justificar, KWord incrementará el espacio entre palabras, para que cada línea (con la excepción de la última línea de un párrafo), alcance tanto el margen izquierdo como el derecho.
Puede usar los botones de la barra de herramientas para cambiar rápidamente la justificación de uno o más párrafos.
A la derecha del cuadro de diálogo hay una caja de vista preliminar, que mostrará aproximadamente la distribución final de su documento.
Bordes
La siguiente sección se usa para definir y configurar los bordes gráficos alrededor de sus párrafos.
KWord puede envolver (en alguno o todos los lados) un párrafo con un borde.
Este borde puede ser sólido o no, de cualquier color y tamaño.
Este panel del diálogo se usa para ajustar los bordes.
Estilo
Use esta lista desplegable para seleccionar el tipo del borde nuevo.
Las opciones se ven preliminarmente en cada selección.
Ancho
Esto determina cómo de ancho será el borde resultante.
Se mide en puntos.
Color
Hacer clic sobre la barra de color le permitirá seleccionar un color usando el cuadro de diálogo de selección de color.
Ahora esto determina cómo deben aparecer los bordes, específicamente qué límites del párrafo necesitan bordes.
Es posible mezclar y unir estilos de bordes, anchos y colores en un párrafo.
Simplemente, determine el estilo del borde, en ancho y el color para un límite del borde, y seleccione el borde usando los botones de borde.
Ahora cambie el ancho o color del estilo para crear un nuevo aspecto del borde y seleccione el botón borde nuevo.
La ventana Vista preliminar le mostrará el aspecto que tienen los bordes en su párrafo.
Viñetas y numeración
Un elemento muy común de un documento es una lista de cosas.
La lista contiene varios elementos que pueden estar numerados, para hacer referencia a ellos más fácilmente.
Además, los elementos de la lista pueden ser diferenciados del resto del texto con un caracter especial que preceda a cada elemento.
Estos caracteres especiales se llaman viñetas.
Este cuadro de diálogo se usa para definir sus vieñetas o métod de numeración de listas en el documento.
Este diálogo también se puede usar para numerar capítulos y secciones.
La sección superior del cuadro de diálogo tiene tres opciones:
Ninguno, Lista y Capítulo.
Si está seleccionado Ninguno, el texto no está marcado como una lista o capítulo.
Ninguna característica de este cuadro de diálogo alterarán el texto cuando se seleccione esta opción.
Lista se usa para crear listas en el documento.
Estas listas se numeran automáticamente y el formato parece similar.
Para más información sobre listas, vea la sección denominada Listas.
Capítulo se usa para numerar capítulos y secciones de un documento.
Tabuladores
Este diálogo se usa para ajustar las tabulaciones.
Para más detalles, vea Usando tabulaciones.
Sombras
Esta pestaña se usa para crear una sombra artificial del texto del párrafo.
A la derecha del diálogo hay una caja de vista preliminar.
Esta caja se puede usar para aproximarse a la apariencia final del texto.
El color de la sombra se puede ajustar haciendo clic sobre el botón llamado Color.
KWord usará el cuadro de diálogo de selección del color para cambiar el color de la sombra.
Use el elemento denominado Distancia para seleccionar la distancia que separa la sombra del texto.
Una distancia de cero ocultará la sombra detrás del texto original.
Seleccione uno de los 8 botones de dirección para determinar en qué dirección se moverá la sombra.
Cuando esté contento con sus selecciones, haga clic sobre Aceptar para aplicar sus cambios.
