Creando plantillas, guardando un documento como una plantilla y borrando plantillas
KWord viene con varias plantillas predefinidas (tanto de distribución de página como orientadas a texto), que se diseñan para proporcionar el formateo inicial de los documentos.
Si, de todas formas, se usa el mismo formato para un documento personalizado una y otra vez, KWord puede crear una plantilla que coincida más exactamente con las necesidades del documento.
Se pueden crear un número de plantillas ilimitadas en KWord, y se pueden organizar en categorías con una simple rellamada.
Creando una plantilla nueva
Las plantillas se crean en primer lugar generando la disposición en KWord basado en una de las plantillas predefinidas, y guardando luego el archivo.
En lugar de guardarlo como un archivo tradicional, se guarda como una plantilla.
Lo que sigue, es un proceso paso a paso para guardar el documento actual como una plantilla.
Esto hace salir un diálogo:
Este diálogo le ayuda a organizar las plantillas en grupos (categorías).
Seleccione el grupo para poner dentro la nueva plantilla.
Para crear un grupo nuevo, simplemente haga clic sobre el botón denominado Añadir grupo.
Aparecerá un cuadro de diálogo que le pedirá el nombre del grupo nuevo.
Haga clic sobre Aceptar y se creará el grupo nuevo.
Teclee un nombre para la plantilla en el cuadro de texto denominado Nombre.
Seleccione el icono para su nueva plantilla usando los botones de radio de la derecha.
KWord tiene un icono predeterminado, que muestra debajo ambas opciones.
Para cambiar el icono, seleccione Personalizado y aparecerá un cuadro de diálogo.
Simplemente seleccione el icono deseado usando el cuadro de diálogo.
Haga clic sobre Aceptar cuando el icono esté seleccionado.
Clic sobre Aceptar.
Una vez que haya guardado la plantilla, la próxima vez que abra un documento, aparecerá su nueva categoría / plantilla en el diálogo de abrir archivo.
Borrando una plantilla
Para borrar una plantilla, siga estos pasos:
Esto le mostrará un diálogo
Clic sobre el pequeño signo más (+) delante del grupo de plantillas en el que se puede encontrar la plantilla.
Clic una vez sobre el nombre de la plantilla
Clic sobre el botón denominado Borrar.
Aparecerá un pequeño cuadro de diálogo para verificar que pretende borrar la plantilla.
Simplemente haga clic sobre Sí.
La plantilla ya está borrada.
Borrando un grupo de plantillas
Para borrar un grupo de plantillas, siga los siguientes pasos:
Esto le mostrará un diálogo
Clic una vez sobre el nombre del grupo
Clic sobre el botón denominado Borrar.
Aparecerá un pequeño cuadro de diálogo para verificar que pretende borrar el grupo de plantillas.
Simplemente haga clic sobre Sí.
El grupo y todas las plantillas del grupo están ahora borradas.
