Comparar & kexi; con otras aplicaciones de bases de datos
A pesar de que las aplicaciones de bases de datos tiende a proporcionar funcionalidades similares, muchas veces utilizan una terminología diferente. Por comodidad, este apéndice muestra cómo la terminología utilizada en & kexi; se corresponde con la usada por otras aplicaciones de bases de datos. Así, este capítulo puede ser práctico en la migración de bases de datos de una aplicación a otra.
Tipos de datos
La siguiente tabla muestra cómo se corresponden los tipos de datos de & kexi; con los tipos de datos de otras aplicaciones de bases de datos.
Algunos de los tipos de datos aquí listados son subtipos de otros tipos. Por ejemplo, el tipo Texto largo es un subtipo del tipo Texto. Para utilizar un subtipo en & kexi;, debería seleccionar el tipo básico correspondiente (en este caso, Texto) en el diseñador de tablas, y a continuación seleccionar subtipo utilizando Subtipo en el Editor de propiedades.
