Asistente para páginas web
El Asistente para páginas web permite mantener un sitio web en un servidor de Internet.
El Asistente para páginas web convierte los archivos de una carpeta local al formato web y los carga en el servidor.
El asistente también utiliza una de las plantillas que se proporcionan para crear una página de índice que contenga hipervínculos a los archivos cargados.
El Asistente para páginas web utiliza el protocolo de transferencia de archivos (FTP) para cargar archivos en un servidor.
El asistente no se puede utilizar para cargar los archivos si se conecta a Internet mediante un servidor proxy.
