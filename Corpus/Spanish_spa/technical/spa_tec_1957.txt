Bordes definidos por el usuario en documentos de texto
En las celdas de una tabla o en toda una tabla de Writer se pueden aplicar distintos bordes de celda.
Asimismo, el usuario también puede definir los bordes para otros objetos de documentos de texto.
Por ejemplo, puede asignar bordes a estilos de página, marcos, o a imágenes o gráficos insertados.
Seleccione una celda o un bloque de celdas en una tabla de Writer.

Configuración predeterminada
Predeterminado
para ajustar o restablecer varios bordes.
Las líneas finas grises dentro de un icono muestran los bordes que se restablecerán o suprimirán.
Las líneas oscuras dentro de un icono muestran las líneas que se ajustarán mediante el estilo de línea y el color que se haya seleccionado.
Las líneas gruesas grises dentro de un icono muestran las líneas que no se van a cambiar.
Ejemplos
Formato - Celdas - Bordes
.
símbolos predeterminados de los bordes
Haga clic en el icono de la izquierda para suprimir todas las líneas.
Esta acción quita todos los bordes exteriores y todas las líneas interiores.
Haga clic en el segundo icono de la izquierda para establecer un borde exterior y quitar todas las demás líneas.
Haga clic en el icono que está totalmente a la derecha para establecer un borde exterior.
Las líneas interiores no se modifican.
A partir de aquí, prosiga para comprobar las líneas que establecen o quitan los demás iconos.
Configuración definida por el usuario
definida por el usuario
, puede hacer clic para establecer o quitar líneas determinadas.
La vista previa muestra las líneas en tres estados.
Para alternar entre los tres, haga clic repetidamente en un extremo o una esquina.
Tipos de línea
Imagen
Significado
Línea negra
línea continua para borde
La línea negra establece la línea correspondiente de las celdas seleccionadas.
La línea se muestra discontinua si selecciona el estilo de línea de 0,05 pt.
Se muestran líneas dobles al seleccionar un estilo de línea doble.
Línea gris
línea gris para borde
Cuando la pertinente línea de las celdas seleccionadas no se va a cambiar se muestra una línea gris.
En esta posición no se agrega ni quita ninguna línea.
Línea blanca
línea blanca para borde
Cuando la pertinente línea de las celdas seleccionadas se va a quitar se muestra una línea blanca.

De forma predeterminada, todas las celdas de una tabla de Writer deben tener como mínimo una línea inferior y una izquierda.
A la mayoría de las celdas del perímetro de la tabla se les aplica más líneas de forma predeterminada.
