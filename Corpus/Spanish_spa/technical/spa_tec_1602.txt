Como ingresar cadenas de texto directamente que no se interpreta?
*
(una letra con un asterisco), el asterisco sera interpretado como un operador de multiplicación.
Para evitar esto, puede encerrar el signo con comillas dobles o agregar un espacio.
Ejemplos:
Una formula importada MathType contiene la siguiente cadena
W rSup { size 8{*} }
Si has configurado Math para convertir las formulas importadas de MathType (dentro de Herramientas - Opciones - Cargar/Guardar - Microsoft Office), verá una formula con un espacio en vez de un asterisco.
Cambiar {*} to {} * {} como en la siguiente formula:
W rSup { size 8{} * {} }
Tambien puede usar W^ &quot; * &quot; para entrar el caracter como texto directo.
Algunas formulas comienzan con un signo =.
Usa &quot; = &quot; para ingresar estos caracteres como texto directo.
