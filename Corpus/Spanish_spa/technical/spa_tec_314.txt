Barra de formato
La barra de formateo se muestra cuando la gráfica se posiciona para modo de edición.
Haga doble clic en la gráfica para ingresar en modo de edición.
Haga clic afuera de la gráfica para dejar el modo de edición.
Puede editar el formato de una gráfica usando los controles y iconos de la barra de formato.
Seleccione elementos de gráficas
Seleccionar el elemento de la gráfica que quieres dar formato.
El elemento tiene la selección de una previsualización de gráfico.
Haga clic en la selección de formato para abrir las propiedades de diálogo para los elementos de selección.
Formato de selección
Abre las propiedades de diálogo para los elementos seleccionados.
Tipos de gráficas
Abre el diálogo de tipos de gráficos.
Tabla de datos de gráficas
Abre el diálogo de la tabla de datos donde puedes modificar los datos de la gráfica.
Activar/desactivar cuadrícula horizontal
El icono de activación/desactivar de cuadricula horizontal en la barra de formato oculta la visibilidad del eje X.
Nota:
Esto solo trabaja si las cajas de verificación de la cuadricula menor en Insertar - Cuadricula están vacías.
Activar o desactivar leyenda
Para mostrar u ocultar una leyenda, haga clic en activar/desactivar en la barra de formatos.
Escala de texto
Reescala en el texto en una gráfica cuando cambias el tamaño de una gráfica.
Diseño automático
Mueve todas los elementos gráficos para la posición predeterminado dentro de la gráfica actual.
Esta función no altera el tipo de gráfica o cualquier otro atributo ademas de la posición de elementos.
