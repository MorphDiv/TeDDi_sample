KArm registra el tiempo empleado en varias tareas.
Introducción
KArm registra el tiempo que emplean varias tareas.
Es útil para registrar horas para que sean facturadas a diferentes clientes o simplemente para averiguar qué porcentaje del día pasa jugando al Doom o leyendo Barrapunto.
KArm fue escrito originalmente por Sirtaj Singh Kang.
La palabra karm significa trabajo en el Punjabi natal del autor.
Usar KArm
Iniciar KArm
Teclee karm en un prompt de comandos o seleccione Seguimiento de tiempo personal en el grupo Utilidades del menú de inicio de KDE.
Uso general
Introduzca las diversas tareas que quiera seguir.
Para empezar a cronometrar una tarea, selecciónela y haga doble clic sobre ella, seleccionando Iniciar del menú Reloj o presione el botón Iniciar de la barra de herramientas.
Para detener el recuento del tiempo, haga doble clic, seleccione Detener en el menú Reloj o presione el botón Detener de la barra de herramientas.
Ahora puede seguir el tiempo de múltiples tareas simultáneamente.
Tareas que se pueden guardar y ser restauradas la próxima vez que se use.
El interfaz de KArm
La ventana principal de KArm tiene los siguientes componentes: barra de menú, barra de herramientas, ventana de tarea / tiempo y barra de estado.
La ventana de tarea / tiempo
En esta ventana se muestran las diferentes tareas, junto con el tiempo acumulado para cada uno en la sesión actual y en total.
Las tareas de las que se sigue el tiempo tiene un pequeño icono de reloj junto al tiempo de sesión.
Se pueden crear subtareas para cada tarea.
Al hacer doble clic sobre una tarea se muestra o se contraen sus subtareas asociadas.
El tiempo total acumulado para una tarea incluye los tiempos para sus subtareas, así como su propio tiempo acumulado.
La barra de menú de KArm
El menú Archivo
Guarda la tarea actual y las subtareas con sus tiempos acumulados
Imprime la ventana de KArm
Reinicia (pone a cero) el tiempo total transcurrido para la sesión actual
Cierra KArm
El menú Reloj
S Reloj Iniciar
Inicia el seguimiento temporal de la tarea seleccionada
Esc Reloj Detener
Detiene el seguimiento temporal de la tarea seleccionada
El menú Tarea
Añade una nueva tarea
Añade una nueva subtara a la tarea seleccionada
Del Tarea Borrar
Borra la tarea o subtarea seleccionada
Cambia el nombre o el tiempo acumulado de la tarea actual
Hay dos opciones para cambiar el tiempo:
Edición absoluta, en la que los nombres de sesión y el total se pueden cambiar separadamente, y Edición relativa, en la que un cierto cambio se añade a o se sustrae tanto del tiempo de sesión como del total.
El menú Preferencias
Preferencias Configurar atajos de teclado...
Abre un diálogo para permitir al usuario personalizar los atajos de teclado
Preferencias Configurar KArm
Abre un diálogo para permitir al usuario configurar KArm
El diálogo tiene dos partes:
Guardado, que configura la ubicación del archivo guardado, si está activado el guardado automático, y su intervalo; y Detección de inactividad, que cuando está fijado muestra una alarma cuando no se detecta actividad durante un espacio de tiempo especificado.
