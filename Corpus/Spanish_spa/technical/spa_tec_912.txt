Conexión ADO
Especifica las opciones para agregar una base de datos ADO (Microsoft ActiveX Data Objects).
La interfaz ADO es un contenedor propietario de Microsoft Windows para conectar con bases de datos.
Para utilizar la interfaz ADO, $[officename] necesita Microsoft Data Access Components (MDAC).
Microsoft Windows 2000 y XP incluyen estos componentes de forma predeterminada.
Para las versiones anteriores de Windows, debe instalar MDAC por separado.
Puede descargar MDAC de la página web de Microsoft.
URL de origen de datos
Especifique el URL del origen de datos.
URL de ejemplo
Para conectar con un archivo de Access 2000, utilice el formato:
PROVIDER=Microsoft.Jet.OLEDB.4.0;DATA SOURCE=c:\Access\nwind2000.mdb
Para conectar con un nombre de un catálogo de un servidor Microsoft SQL Server con el nombre martinez, escriba:
PROVIDER=sqloledb;DATA SOURCE=martinez;INITIAL CATALOG=Primero
Para acceder a un controlador ODBC como proveedor:
DSN=SQLSERVER
