Datos de KOffice
Las aplicaciones que hacen a KOffice capaz de un buen grado de integración entre ellas.
A causa de que el proceso actual de insertar un componente varía según la aplicación, este capítulo no detallará cada paso del proceso.
Necesitará estar familiarizado con la forma básica de usar la otra aplicación para insertar correctamente un componente de esa aplicación
Diríjase a los archivos de ayuda de las otras aplicaciones para una información más específica.
Instrucciones generales
Hay dos formas de insertar un componente (hoja de cálculo, presentación, gráfico, etc) desde otra aplicación de KOffice:
Se le dará una lista de aplicaciones de KOffice para que elija.
Seleccione la aplicación que desee.
o haciendo clic sobre en la barra de herramientas.
Aparecerá un cuadro de diálogo con una lista de aplicaciones de KOffice para que seleccione.
Elija la aplicación que desee.
KWord le pedirá ahora que defina un marco para este objeto.
Ponga el puntero del ratón donde quiera que vaya la esquina superior izquierda del marco y haga clic una vez con el left mouse button.
Los detalles de cada aplicación son diferentes.
Vea los manuales de la aplicación de KOffice para detalles sobre el uso de esa aplicación. (En otras palabras, desde aquí en adelante, si inserta una hoja de cálculo desde KSpread, entonces las ventanas actuarán de la misma que en KSpread).
Trabajando con el componente insertado
Para editar los datos del componente, simplemente haga doble clic con el puntero del ratón y KWord cambiará las barras de herramientas, los elementos del menú y la estructura de las aplicaciones para que coincidan on la aplicación de KOffice.
De esta forma, puede usar las herramientas específicas de la aplicación para refinar su componente.
El marco que contiene los datos empotrados se puede manipular de la misma forma que cualquier otro marco de KWord.
Cuando guarde su documento, el componente se guarda dentro del documento de KWord.
