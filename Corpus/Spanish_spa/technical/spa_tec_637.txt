Conexión LDAP
Especifica la configuración para importar una base de datos utilizando LDAP
(
Lightweight Directory Access Protocol)
.
Esta página sólo está disponible si se ha registrado un servidor LDAP como base de datos de direcciones.
URL del servidor
Escriba el nombre del servidor LDAP utilizando el formato &quot; ldap.server.com &quot; .
Base DN
Escriba el punto inicial para buscar en la base de datos LDAP, por ejemplo, &quot; dc=com &quot; .
Número de puerto
Escriba el puerto del servidor LDAP, que suele ser 389.
Usar conexión segura (SSL)
Crea una conexión segura con el servidor LDAP mediante Secure Sockets Layer (SSL).
De modo predeterminado, una conexión SSL utiliza el puerto 636.
Una conexión regular utiliza el puerto 389.
