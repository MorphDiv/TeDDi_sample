El menú Archivo
El menú Archivo de la ventana principal
La ventana principal tiene las siguientes entradas:
Archivo Abrir...
Abre un archivo de pruebas existente.
Archivo Abrir reciente
Abre un archivo de pruebas utilizado recientemente.
Archivo Cerrar
Cierra el archivos de pruebas abierto actualmente.
Archivo Salir

El menú Archivo de la ventana de edición
Archivo Nuevo
Crea una nueva prueba
Archivo Abrir...
Abre un archivo de pruebas existente.
Archivo Abrir reciente
Abre un archivo de pruebas utilizado recientemente.
Archivo Guardar
Guarda la prueba abierta actualmente.
Archivo Guardar como...
Guardar el juego abierto actualmente con un nuevo nombre.
Archivo Cerrar

El menú Construir
El menú Construir existe únicamente en la ventana principal de KEduca.
Construir Crear / Modificar...
Crea o modifica una prueba en la ventana Crear o modificar una prueba.
El menú Opciones
Las primeras entradas están disponibles en las dos ventanas principales de KEduca.
Opciones Mostrar barra de herramientas
Conmuta la visión de la barra de herramientas.
De forma predeterminada esta se encuentra activada.
Opciones Configurar accesos rápidos
Personaliza las asignaciones de teclas de KEduca.
Opciones Configurar barras de herramientas...
Personaliza las barras de herramientas de KEduca.
La siguiente entrada está sólo disponible en la ventana principal.
Personaliza KEduca.
Las opciones disponibles están descritas en el capítulo Configuración.
El menú Ayuda
F1 Help Contenidos...
Inicia el sistema de ayuda de KDE mostrando las páginas de información de KEduca. (este documento).
Modifica la forma del puntero transformandola en una combinación de flecha e interrogación.
Al pulsar sobre elementos dentro de KEduca se abrirá una ventana de ayuda (si existe para ese caso particular) describiendo la función de dicho elemento.
Help Informar de fallo...
Abre un cuadro de diálogo de informe de fallos donde puede informar de un fallo o solicitar una mejora o nueva propiedad deseable.
Mostrará información sobre la versión y el autor.
Ayuda Acerca de KDE
Muestra la versión de KDE y otra información básica.
