Formateando caracteres
Esta sección cubre los cambios que se pueden hacer a caracteres individuales o a bloques de caracteres, incluyendo:
Tamaño del caracter
Tipografía
Estilo de tipografía (cursiva, negrita, etc.)
Subrayados, superíndice, subíndice, tachado, etc.
Color del caracter y de fondo.
Mayús / minús de caracter.
Cambiando el tamaño, la tipografía, el estilo de tipografía y los colores del texto y del fondo
Usando el teclado o las barras de herramientas
Hay varios atajos de teclado que puede usar para hacer cambios comunes al formateo de caracteres.
Despues de seleccionar el texto, puede usar estas combinaciones de teclas para cambiar cada atributo.
La barra de herramientas Formateo básico tiene botones que puede usar para cambiar estos mismos atributos básicos.
La tabla siguiente muestra detalles de cada atributo, los botones de la barra de herramientas y los atajos de teclado.
Reduce el tamaño de la tipografía:
Ctrl <
Aumenta el tamaño de la tipografía:
Ctrl >
No puede cambiar el juego de caracteres ni añadir doble subrayado usando el teclado no la barra de herramientas.
Debe usar el cuadro de diálogo.
Usando el cuadro de diálogo
Si tiene muchos cambios que hacer, también puede usar un cuadro de diálogo para fijar estos mismos atributos.
Puede abrir el cuadro de diálogo Formateo de caracteres en una de estas tres formas:
Clic sobre el texto seleccionado con el right mouse button.
Aparecerá un pequeño menú.
Seleccione Tipografía... desde el menú.
Puede usar el atajo de teclado:
Cada método abrirá un cuadro de diálogo
Usando este cuadro de diálogo, puede seleccionar las opciones de tipografía, tamaño, juego de caracteres, cursiva y negrita desde las tres listas superiores.
La caja de vista preliminar del medio del diálogo le mostrará el aspecto de sus preferencias actuales.
Si la cadena de vista preliminar predeterminada no es apropiada, puede cambiarla haciendo clic una vez dentro de la caja de vista preliminar con el left mouse button.
Aparecerá un cursor de texto.
Puede editar el texto para que se ajuste a sus necesidades.
La frase de ejemplo volverá a ser la predeterminada la próxima vez que abra este diálogo.
Usando las cajas de verificación ubicadas bajo la caja de vista preliminar, puede seleccionar superíndice o subíndice.
Haciendo clic sobre el botón etiquetado Cambiar color... le permitirá cambiar el color del texto usando el cuadro de diálogo de selección del color.
Hacer clic sobre el botón etiquetado Cambiar color de fondo... le permitirá cambiar el color del fondo usando el cuadro de diálogo de selección del color.
Si hace clic en la pestaña llamada Efectos de tipografía, el diálogo cambia a:
La subsección denominada Subrayado tiene dos listas desplegables.
En la lista más ala izquierda seleccione entre no subrayado (Sin), Sencillo, Negrita simple o Doble subrayado.
Use la lista derecha para seleccionar el estilo de subrayado (línea sólida, punteada, etc.).
Use el botón Cambiar color para seleccionar un color para el subrayado usando el cuadro de diálogo de selección de color.
La subsección denominada Tachado tiene dos listas.
Use la lista izquierda para seleccionar entre no tachado (Sin), Sencillo o Doble tachado.
Use la lista derecha para seleccionar el estilo de la línea.
Al hacer clic sobre Aceptar se confirmarán sus cambios al documento.
Al hacer clic sobre Cancelar, se abortarán los cambios que haya hecho.
Cambiando mayús / minús de las tipografías
KWord puede cambias las mayúsculas o minúsculas de grandes bloques de texto a uno de los cinco estilos predefinidos.
Comience seleccionando el bloque de texto que quiera cambiar.
Un pequeño cuadro de diálogo aparecerá con cuatro opciones:
Mayúscula
Convertirá todos los caracteres a mayúscula.
Ejemplo:
KWORD CAMBIARÁ LAS MAYÚS / MINÚS AUTOMÁTICAMENTE.
Minúscula
Convertirá todos los caracteres a minúscula.
Ejemplo: kword cambiará las mayús / minús automáticamente.
Mayús / minús de título
Capitalizará cada palabra.
Ejemplo:
Kword Cambiará Las Mayús / Minús Automáticamente.
Intercambiar mayús / minús
Convierte todas las letras minúsculas en mayúsculas y convierte todas las letras mayúsculas en minúsculas.
Mayús / minús de frase
Capitaliza el primer caracter tras un punto.
Cuando haya seleccionado la opción que le guste, haga clic sobre Aceptar y KWord hará los cambios.
