Instrucción Close [Runtime]
Cierra un archivo especificado que se abrió con la instrucción Open.
Sintaxis:
Close NúmeroArchivo As Integer[, NúmeroArchivo2 As Integer[,...]]
Parámetros:
NúmeroArchivo:
Cualquier expresión entera que especifique el número de canal de datos que se abrió con la instrucción
Open
.
Ejemplo:
Sub EjemploTrabajoConArchivo
Dim iNumero As Integer
Dim sLinea As String
Dim aArchivo As String
Dim sMensaje as String
aArchivo = &quot; c:\data.txt &quot;
sMensaje = &quot; &quot;
iNumero = Freefile
Open aArchivo For Output As #iNumero
Print #iNumero, &quot; Primera línea de texto &quot;
Print #iNumero, &quot; Otra línea de texto &quot;
Close #iNumero
iNumero = Freefile
Open aArchivo For Input As iNumero
While not eof(iNumero)
Line Input #iNumero, sLinea
If sLinea < > &quot; &quot; then
sMensaje = sMensaje & sLinea & chr(13)
end if
wend
Close #iNumero
Msgbox sMensaje
End Sub
