odbc_num_rows() devolvera el numero de registros de un ODBC.
Esta funcion devolvera -1 en caso de error.
Para declaraciones INSERT, UPDATE y DELETE odbc_num_rows() devolvera el numero de registros afectados.
Para una clausula SELECT esta puede ser el numero de registros permitidos.
Nota:
El uso de odbc_num_rows() para determinar el numero de registros permitidos despues de un SELECT devolvera -1.
