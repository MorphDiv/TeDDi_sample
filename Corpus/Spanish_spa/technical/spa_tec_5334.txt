kjs
1
Intérprete ECMAScript compatible con & kde;
Sinopsis
kjs --help -help -h -e statement script
Descripción
kjs es un motor de ECMAScript/ JavaScript para KDE.
Opciones de las aplicaciones
--help, -help, -h
Muestra el uso de kjscmd.
-e statement
Ejecuta la sentencia y sale. Por ejemplo:
kjs -e "print('hello world ')"
script
Ejecuta la sentencia incluida en el archivo script.
Utilización
kjs file, como otros muchos interpretes, cuando se le llama sin parámetros utiliza un editor de líneas para introducir sentencias, mostrando JS > Para salir del editor de líneas teclee quit() o pulse Ctrl+D.
