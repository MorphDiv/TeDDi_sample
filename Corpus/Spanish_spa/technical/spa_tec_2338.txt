Enlaces
KWord tiene la capacidad de insertar direcciones de Internet, de correo electrónico y localizaciones de archivos con un descriptor.
Esto se usa mayoramente para crear páginas web en KWord.
Una vez insertado en el documento, el descriptor será visible en KWord, pero al guardar como un documento web, el descriptor se convertirá en el hiperenlace para la ubicación especificada con la dirección.
Insertar un nuevo enlace
Esto le llevará a un pequeño cuadro de diálogo.
Usando la barra de iconos de la derecha, seleccione el tipo de enlace que desee insertar.
En el campo de comentario, teclee el texto que quiere que aparezca en su documento web (por ejemplo, el sitio web de KOffice).
En el campo de dirección, teclee la dirección de internet, dirección de correo o la ubicación del archivo que quiere enlazar con el comentario (por ejemplo, http: / /www.koffice.org)
Clic sobre Aceptar.
Por defecto, KWord muestra los enlaces subrayados (como en la mayoría de navegadores).
Se puede encontrar más información aquí.
Abriendo un enlace
Puede usar un enlace desde KWord.
Simplemente haga clic una vez sobre el enlace con el right mouse button.
Aparecerá un submenú.
Seleccione Abrir enlace.
Un enlace a un sitio web abrirá una nueva ventana del navegador y le llevará directamente a ese sitio web.
Un enlace a una dirección de correo abrirá un nuevo mensaje en su programa de correo electrónico.
Un enlace a un archivo abrirá el visor apropiado o el editor para ese tipo de archivo.
Cambiando un enlace
Para cambiar los detalles de un enlace, simplemente haga clic una vez sobre el enlace con el right mouse button.
Aparecerá un submenú.
Seleccione Cambiar enlace....
Aparecerá el cuadro de diálogo de enlaces, con la configuración actual del enlace mostrado.
Haga cualquier cambio conveniente y haga clic sobre Aceptar para realizar efectivamente sus cambios.
Borrando un enlace
Puede borrar un enlace como cualquier otro texto de su documento.
Simplemente coloque el cursor al final del enlace, y presione Backspace.
El enlace se borra completamente.
