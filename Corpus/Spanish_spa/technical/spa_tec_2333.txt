El manual de KChart

KChart es una aplicación para dibujar gráficas de barras, de tarta y de líneas.
Se recomienda su uso con KSpread.
KChart aún está bajo desarrollo, así que algunas características no funcionan.
Introducción
KChart se recomienda principalmente que sea usado con KSpread.
Está actualmente en fase de revisión y se debe considerar software beta.
Trazando con KChart
Para trazar con KChart, cuando está empotrado en KSpread, haga lo siguiente:
Resalte la segunda fila de números.
Ahora haga clic sobre el botón Gráfica para crear la gráfica.
Seleccione las opciones que quiera del asistente de gráficas.
El asistente de gráficas surgirá tras insertar la gráfica.
Para hacer una gráfica de barra para elementos individuales, ponga los números en una columna, como se muestra abajo.
La orientación de los números determina cómo se hará el trazado.
Una fila de números define las ordenadas para un trazado sencillo.
Una columna de números define las alturas para cada barra, en una gráfica de barras.
También determina el tamaño de los pedazos en una gráfica de tarta.

Sale de KChart
El menú Editar
Editar Personalizar con asistente...
Le permite especificar el tipo de gráfica.
También puede insertar títulos y cambiar las tipografías, con el asistente.
Editar Editar datos...
Tabla para introducir los datos.
Editar Configurar...
Diálogos para configurar la gráfica.
Las pestañas de configuración
Colores
Seleccione colores de gráficos, de cuadrícula y de título aquí.
Tipografía
Seleccione el estilo y el tamaño de la tipografía aquí.
La serie de colores también se puede seleccionar aquí.
La serire se refiere a los gráficos individuales.
Cada conjunto de datos de un gráfico es una serie.
Fondo
Selecciona el color de fondo de su gráfico.
Los fondos de imagen no funcionan por ahora.
Leyenda
Cambia la ubicación de la leyenda.
El color del cuadro de la leyenda también se puede cambiar.
La leyenda contiene la clave de colores de las series de los gráficos.
Ejes
Los ejes se pueden activar o desactivar aquí.
No todas las características funcionan aquí.
Parámetros 3D
Se puede fijar aquí la perspectiva para las barras de gráficos 3d.
Subtipo de gráfica
Cambia el tipo de barra del gráfico.
Encabezado / Pie
Introduzca los títulos que quiera para su gráfico aquí.

Abajo hay un ejemplo de cómo configurar un atajo para abrir el asistente de gráficas.

Luego clic sobre el botón Primario.
El atajo de teclado se ha introducido ya.
Configurando las barras de herramientas
Para añadir un botón a la barra de herramientas Archivo.
Asegúrese de que Archivo <KOfficeShell> se muestra arriba del cuadro de selección múltiple.
Clic sobre uno de los elementos del lado izquierdo.
Este elemento ahora tendrá un fondo azul que muestra que está seleccionado.
Luego haga clic sobre el botón de la Flecha derecha para ponerlo a la derecha.
Clic sobre Aplicar y luego clic sobre Aceptar
Los elementos nuevos deben estar en la barra de herramientas.

Preguntas y respuestas
¿Puede aceptar KChart archivos de texto de datos, como columnas de números, para hacer el trazado?
No.
Por ahora, KChart sólo puede hacer gráficos desde dentro de KSpread.
¿Por qué algunos comandos del menú Editar Configurar... no funcionan?
KChart aún se está desarrollando y algunas características no funcionan.

Instalación
Cómo obtener KChart
KChart forma parte del proyecto KDE http: / /www.kde.org /.
KChart está incluido en el paquete koffice en ftp: / /ftp.kde.org / pub / kde /, el servidor FTP del proyecto KDE.
Requisitos
Para usar KChart con éxito, necesita KOffice 1.2 y Qt-3.0.5.
Puede encontrar una lista de cambios en http: / /www.koffice.org / kchart /.
