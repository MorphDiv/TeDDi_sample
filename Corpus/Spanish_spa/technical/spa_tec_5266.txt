Introduzca aquí la expresión que desea buscar. Si no está marcado « expresión regular », cualquier carácter distinto de un espacio será « escapado » con un carácter de barra inversa. Los metacaracteres posibles son:. - representa cualquier carácter ^ - representa un inicio de línea $- representa un fin de línea \\lt; - representa el principio de una palabra \\gt; - representa el final de una palabra También dispone de los siguientes operadores de repetición:? - el elemento precedente puede aparecer una vez como máximo * - el elemento precedente puede aparecer ninguna o más veces + - el elemento precedente puede aparecer una o más veces {n} - el elemento precedente debe aparecer exactamente n veces {n,} - el elemento precedente debe aparecer n o más veces {, n} - el elemento precedente debe aparecer un máximo de n veces {n, m} - el elemento precedente debe aparecer al menos n veces, pero un máximo de m veces. También se permiten referencias a subexpresiones entre paréntesis mediante la notación \\#. Vea la documentación de grep(1) para más información.
