Introducción
& kjots; es un pequeño programa que le ayuda a escribir algunas notas cortas y las mantiene organizadas.
Usando & kjots;
& kjots; dispone de dos elementos básicos para organizar sus notas: libros y páginas.
Puede crear varios libros en & kjots;, y cada libro puede contener varias páginas, donde escribirá sus notas. Cada libro debe tener un nombre (de manera que pueda localizar lo que está buscando), al igual que cada página. Aunque no está obligado a dar un nombre a una página, es muy recomendable que lo haga.
Uso de libros
Para crear un nuevo libro, use Ctrl; Shift; N o la entrada del menú Archivo Nuevo libro, y se le pedirá que introduzca un nombre para el nuevo libro.
El nombre del nuevo libro no debe aparecer en la parte izquierda de la ventana. Si quiere que el libro esté en sus Marcadores, utilice Ctrl; B o la entrada del menú Marcadores Añadir marcador. También puede organizar sus marcadores en carpetas separadas, como seguramente hará en & konqueror;.
Si desea tener sus libros en un orden distinto, solo tiene que arrastrarlos y soltarlos en el lugar donde quiera que estén.
Tal vez haya notado que no hay una opción para guardar en ninguna parte. Esto no es un error: & kjots; guarda sus libros y páginas de forma automática, de modo que nunca perderá esa pequeña nota que escribió cuando hablaba por teléfono y necesitaba anotar algo urgentemente.
También puede exportar un libro completo como archivo de texto plano o html usando Archivo Exportar libro.... En tal caso se le pedirá que introduzca una ubicación para guardar el archivo. El formato de un libro de ejemplo guardado se muestra abajo:
############ # Primera página ############ ¡Recuerda ir de tiendas! ############# # Código útil ############# 10 PRINT "HOLA MUNDO" 20 GOTO 10 ################ # La tercera página ################ Esta es la tercera página y es un ejemplo para KJots.
Recuerde que el diálogo donde se le pregunta por un nombre de archivo es un diálogo estándar de & kde;. Esto significa que es tan fácil guardar el archivo en su disco duro local como guardarlo en un servidor web o ftp, o en un lápiz de memoria USB. Así, puede exportar un archivo como html y guardarlo en un servidor web, de modo que siempre tenga disponibles sus notas.
Usando páginas
Las páginas están donde actualmente escriba lo que quiera que desee escribir. Una vez que haya creado un libro, puede escribir texto en el cuadro de texto principal.
Para crear páginas adicionales, puede usar Ctrl; N o la entrada del menú Archivo Nueva página, que creará una nueva página. De igual modo, la página actual puede ser eliminada usando Archivo Eliminar página.
Si necesita insertar la fecha y hora actuales mientras está escribiendo una nota, puede usar Ctrl; Shift; I.
Para que pueda ir a la página que desea ver, las páginas pueden (y deben) tener un nombre. Esto se puede hacer del modo usual: seleccione la página que quiere renombrar, espere un segundo y pulse una vez más, entonces podrá renombrarla. También dispone de un acceso rápido de teclado que puede usar mientras está escribiendo una nota: Ctrl; M.
Otra forma de renombrar la página consiste en seleccionar una parte de su nota y luego pulsar Ctrl; T, con lo que el texto seleccionado se convertirá en el título de la página.
Si desea acceder rápidamente a una determinada página, puede asignar un acceso rápido de teclado a dicha página. Vaya a Preferencias Configurar accesos rápidos....
Referencia de órdenes
El menú Archivo
Ctrl; N Archivo Nueva página...
Crea una nueva página.
Ctrl; Shift; N Archivo Nuevo libro...
Crea un libro nuevo.
Archivo Exportar página...
Guarda la página actual en un archivo de texto plano o html.
Archivo Exportar libro...
Guarda el libro actual en un archivo de texto plano o html.
Archivo Eliminar página
Borra la página actual.
Archivo Eliminar libro
Borra el libro actual.
Ctrl; P Archivo Imprimir
Imprime el libro o la página actual.
Ctrl; Q Archivo Salir
Sale de & kjots;.
El menú Edición
Ctrl; X Edición Cortar
Corta el texto seleccionado actualmente en la ventana de texto principal y lo coloca en el portapapeles.
Ctrl; C Edición Copiar
Copia el texto seleccionado actualmente en la ventana de texto principal y lo coloca en el portapapeles.
Ctrl; V Edición Pegar
Pega el texto del portapapeles en la ventana de texto principal.
Ctrl; T Edición Pegar como título de página
Renombra la página con el texto del portapapeles.
Ctrl; F Edición Buscar...
Busca texto en la ventana de texto principal. Tenga en cuenta que sólo busca en la página actual.
F3 Edición Buscar siguiente
Busca la siguiente coincidencia del texto que se quiere encontrar (especificado en Buscar).
Ctrl; R Edición Sustituir...
Sustituye un trozo de texto por otro, sólo en la página actual.
Ctrl; M Edición Renombrar...
Da la posibilidad de renombrar la página actual.
Ctrl; I Edición Insertar fecha
Inserta la fecha y la hora en la página actual.
El menú Marcadores
Ctrl; B Marcadores Añadir marcador
Añade la página actual a la lista de marcadores. Vea Uso de libros.
Marcadores Editar marcadores
Edita los marcadores. Aquí podrá buscar marcadores, hacer comentarios, organizarlos en diferentes carpetas y mucho más.
Marcadores Nueva carpeta de marcadores
Crea una nueva carpeta donde puede guardar sus marcadores.
El menú Preferencias
Preferencias Ocultar/ Mostrar barra de herramientas
Muestra u oculta la barra de herramientas bajo el menú.
Preferencias Configurar atajos de teclado...
Elegir atajos de teclado para algunos elementos del menú.
Preferencias Configurar barras de herramientas...
Selecciona qué elementos están presentes en la barra de herramientas.
Preferencias Configurar KJots...
Configura las opciones de & kjots;, incluida la fuente que se usará y las opciones de auto guardado.
