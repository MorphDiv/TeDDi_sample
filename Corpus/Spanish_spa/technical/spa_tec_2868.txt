La función cpdf_page_init() crea una nueva página de altura height y profundidad width.
La página tiene el número page number y orientación orientation. orientation puede ser 0 para retrato y 1 para paisaje.
El último parámetro opcional unit define la unidad del sistema de coordenadas.
El valor debería ser el número de puntos postscript por unidad.
Como el valor de una pulgada el igual a 72 puntos, un valor de 72 sería la unidad para una pulgada.
Por defecto es 72.
Vea también cpdf_set_current_page().
