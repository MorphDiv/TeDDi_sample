Asistente para páginas web - Vista previa
Especifique dónde desea publicar la página web y obtener una vista previa de ella.
Vista previa
Abre la página web en el explorador web predeterminado del sistema operativo.
Directorio local
Carga los archivos y la página de índice en un directorio local.
La página de índice se guarda en la ubicación especificada.
Todos los demás archivos se guardan en el directorio &quot; myWebsite &quot; del directorio que contiene la página de índice.
...
Abre un diálogo para seleccionar una carpeta.
Archivo histórico
Agrega los archivos y la página de índice a un archivo histórico comprimido y carga el archivo en la página web.
La página de índice se guarda en la ubicación especificada.
Todos los demás archivos se guardan en el directorio &quot; myWebsite &quot; del directorio que contiene la página de índice.
En función del sistema operativo, los formatos del archivo histórico disponibles son zip, gzip y war.
...
Abre un diálogo que permite especificar la ubicación del archivo histórico.
Web vía FTP
Carga los archivos en un servidor FTP.
La página de índice se guarda en la ubicación especificada.
Todos los demás archivos se guardan en el directorio &quot; myWebsite &quot; del directorio que contiene la página de índice.
La opción FTP no se puede utilizar si se conecta a Internet mediante un servidor proxy.
Configurar
Conexión FTP
, que permite editar y probar la configuración de conexión para el servidor FTP.
Guardar configuración
Guarda la configuración especificada en este asistente.
Guardar como
Escriba un nombre para el archivo de configuración.
