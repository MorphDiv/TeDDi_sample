Menú contextual de un campo de control
El menú contextual de un campo de control contiene, entre otros, los comandos siguientes:
Sustituir por
Abre un submenú en el que se puede seleccionar un tipo de control para reemplazar el control que está seleccionado en el documento.
Se adoptan todas las propiedades posibles.
Campo de texto
El control seleccionado se transforma en un cuadro de texto.
Botón
El control seleccionado se transforma en un botón.
Etiqueta
El control seleccionado se transforma en una etiqueta.
Listado
El control seleccionado se transforma en un cuadro de lista.
Casilla de verificación
El control seleccionado se transforma en una casilla de verificación.
Campo de opción
El control seleccionado se transforma en un botón de opción.
Campo combinado
El control seleccionado se transforma en un cuadro combinado.
Botón de imagen
El control seleccionado se transforma en un botón gráfico.
Selección de archivo
El control seleccionado se transforma en una selección de archivos.
Campo de fecha
El control seleccionado se transforma en un campo de fecha.
Campo horario
El control seleccionado se transforma en un campo de hora.
Campo numérico
El control seleccionado se transforma en un campo numérico.
Campo de moneda
El control seleccionado se transforma en un campo de moneda.
Campo enmascarado
El control seleccionado se transforma en un campo enmascarado.
Control de imagen
El control seleccionado se transforma en un control de imagen.
Campo formateado
El control seleccionado se transforma en un campo con formato.
