El diálogo Opciones del proyecto
Este diálogo le permite definir algunas propiedades para el proyecto. Puede acceder al diálogo usando el menú Proyecto Editar proyecto principal....
El diálogo Opciones del proyecto
El diálogo Opciones del proyecto
Id:
Introduzca un número. Define la identificación del proyecto y debe ser único.
Nombre:
Introduzca el nombre del proyecto.
Líder:
Introduzca el nombre del líder del proyecto. Al hacer clic en el botón Elegir... aparece la lista de contactos de KAddressBook y allí puede elegir un nombre.
Planificación
Aquí puede definir cuándo debe comenzar o finalizar el proyecto. Elija una fecha de comienzo o de finalización.
Fecha de comienzo:
Si se define una fecha de comienzo, el proyecto se planifica hacia el futuro a partir de esa fecha. Cuando se calcula el proyecto, la fecha de finalización define cuándo está planificado que termine el proyecto.
Fecha de finalización:
Si se define una fecha de finalización, el proyecto se planifica hacia el pasado desde esa fecha. Cuando se calcula el proyecto, la fecha de comienzo muestra cuándo debe comenzar el proyecto para que termine a tiempo.
Notas del proyecto y resumen:
Aquí puede escribir varias notas asociadas con el proyecto y/ o el resumen del proyecto.
Al hacer clic en el botón Aceptar se aplican los cambios y se cierra el diálogo, y al hacer clic en el botón Cancelar se cancelan los cambios que acaba de hacer.

El diálogo Opciones de calendario
Este diálogo le permite definir un calendario para su proyecto. Puede acceder a este diálogo usando el menú Proyecto Editar calendario....
El diálogo Opciones de calendario
El diálogo Opciones de calendario
Puede tener varios calendarios para su proyecto. Primero añada un nombre para el primer calendario usando el botón Añadir. La vista del calendario se activa con el día presente marcado en una esquina cuando hace clic en el nombre del calendario.
Trabajar con un calendario
Trabajar con un calendario
Puede definir los días como Laborale o No laborable. Haga clic en los días que quiera que sean laborables para su proyecto y elija Trabajando en lugar de Indefinido. Defina luego el intervalo de trabajo seleccionando las fechas de comienzo y de finalización y haciendo clic en el botón Añadir intervalo para añadir el horario laboral. Haga clic en el botón Aplicar para validar sus opciones.
Días laborables y no laborables
Días laborables y no laborables
Aquí puede ver los días laborables en azul y los días no laborables en rojo.
El diálogo Editar cuentas
Las cuentas se utilizan para evaluar el coste del proyecto. Este diálogo le permite configurar varias cuentas para ser utilizadas en su proyecto. Puede acceder a ese diálogo usando el menú Proyecto Editar cuentas....
El diálogo Editar cuentas
El diálogo Editar cuentas
Para crear una cuenta, haga clic en el botón Nuevo. Luego puede introducir el nombre de la cuenta, que puede validar utilizando la tecla Enter. Puede añadir una descripción breve de esta cuenta haciendo doble clic en el campo descripción.
Puede crear subcuentas de la cuenta principal o una cuenta nueva. Puede eliminar cualquier cuenta resaltada usando el botón Eliminar.
También puede definir una cuenta seleccionando el nombre de la cuenta en la lista desplegable.

El diálogo Recursos
Este diálogo le permite definir los recursos para el proyecto actual. Puede acceder a este diálogo utilizando el menú Proyecto Editar recursos....
El diálogo Recursos
El diálogo Recursos
Primero tiene que definir un grupo de recursos haciendo clic en el botón Nuevo en el área de grupo de recursos. Haga clic sobre el nombre de este grupo y añada un recursos nuevo haciendo clic sobre el botón Nuevo en el área de recursos. Muestra el diálogo Opciones del recurso.
El diálogo Opciones de recursos
El diálogo Opciones de recursos
El diálogo Configurar & kplato;...
Este diálogo le permite establecer las opciones globales predefinidas de & kplato;. Puede acceder a este diálogo utilizando el menú Opciones Configurar & kplato;....
Sobre cualquier widget de este diálogo puede obtener una ayuda del tipo ¿Qué es esto? con Shift; F1 o utilizando el icono Ayuda.
Opciones predefinidas de tareas
Establece las opciones que se deben utilizar de forma predefinida cada vez que añada una tarea nueva.
Opciones predefinidas de tareas
Opciones predefinidas de tareas
Responsable
Inserta el nombre de la persona responsable para esta tarea. No está limitado a las personas que hay disponibles en un grupo de recursos, sino que puede ser cualquiera.
Al hacer clic sobre el botón Seleccionar... puede insertar una persona de la libreta de direcciones.
Direcciones
Direcciones
Temporización
Planificación:
Elija la restricción de planificación que quiere utilizar de forma predefinida. Según esta restricción, los campos de fecha estarán disponibles si es necesario rellenarlos. Estas opciones afectan a la planificación real de la tarea.
Estimado:
Estima las predicciones de los resultados cuantitativos. La estimación se puede basar en esfuerzo o en duración. Si se basa en esfuerzo, la duración final depende de los recursos asignados a la tarea. Para una estimación basada en la duración, los recursos asignados no afectan a la duración fija de la tarea, sino sólo al coste.
Nota:
Escriba aquí una nota que quiera mostrar de forma predefinida para cada tarea nueva.
Al hacer clic en el botón Opciones predefinidas se reincian las opciones del diálogo a sus valores predefinidos, y al hacer clic en Aplicar se aplican los cambios y se deja abierto el diálogo. Al hacer clic en el botón Aplicar se aplican los cambios y se cierra el diálogo, y el botón Cancelar cancela todos los cambios que ha hecho.
