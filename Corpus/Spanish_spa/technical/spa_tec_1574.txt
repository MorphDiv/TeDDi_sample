Efecto

Dirección
Especifica la dirección del efecto.
Inicio acelerado
Activar este opción para asignar una velocidad que aumenta gradualmente, cuando comienza el efecto.
Final decelerado
Activa este opción para asignar una velocidad que disminuye gradualmente para terminar el efecto.
Mejoras
Especifica las mejoras del efecto actual.
Sonido
Seleccione un sonido en la Galería o una de las entradas especiales.
Sin sonido
: no se reproduce ningún sonido durante la animación del efecto.
Detener sonido anterior
: el sonido del efecto anterior se detiene en cuanto se ejecuta el efecto actual.
Otro sonido
: muestra un diálogo de apertura de archivos que permite seleccionar un archivo de sonido.
Botón Sonido
Reproduce el archivo de sonido seleccionado.

Atenuar con color
- después de la animación, la forma se rellena con un color atenuado.
No atenuar
: no se ejecuta ningún efecto posterior.
Ocultar tras la animación
: oculta la forma cuando finaliza la animación.
Ocultar tras la animación
: oculta la forma en la siguiente animación.
Atenuar color

Palabra por palabra

Letra por letra
