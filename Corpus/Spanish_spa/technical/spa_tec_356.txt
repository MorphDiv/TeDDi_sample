Burbuja
Una gráfica que muestra la relación de tres variables.
Dos variables son usadas por la posición del eje X y Y, mientras que la tercer variable se muestra con un tamaño relativo a cada burbuja.
El diálogo de series de datos para un gráfico de burbuja posee una entrada para definir el rango de datos para los tamaños de Burbujas.
