kmenuedit permite la edición del menú principal de KDE.
Introducción
kmenuedit permite la edición del menú principal de KDE.
kmenuedit se puede iniciar bien pulsando con el botón derecho del ratón sobre el botón K, o eligiendo Editar menú desde el submenú Sistema del menú Principal.
kmenuedit le permite:
Ver y editar el menú principal actual
Cortar, Copiar y pegar elementos del menú
Crear y borrar submenús
Utilizando kmenuedit
El panel izquierdo de aplicaciones muestra la estructura del menú principal.
Cuando usted navega por elementos del panel izquierdo, el panel derecho muestra información detallada del elemento del menú resaltado.
Información general del programa
Nombre
Este es el nombre de su programa tal y como aparece en el menú Principal.
Puede ser diferente del nombre real del ejecutable.Por ejemplo el nombre del ejecutable de mc es "Midnight Commander".
Comentario
Describa en este campo el programa con mayor detalle.
Esto es totalmente opcional.
Comando
Este es el nombre del programa ejecutable.
Asegurese de que tiene permiso para ejecutar este programa.
Tipo
Este campo es similar al campo de Comentario.
Describa aquí el tipo de su programas
Ruta de trabajo
Especifique la ruta de trabajo del programa.
Esta será la ruta actual cuando el programa se inicie.
No necesita ser igual a la ubicación del ejecutable.
Lista de iconos
Pulse sobre este icono para mostrar una selección de iconos.
Elija un icono para su programa.
Ejecutar en terminal
Debe seleccionar esta opción si su programa requiere emulador de terminal para ejecutarse.
Esto se aplica a aplicaciones de consola.
Opciones del terminal
Ponga todas las opciones de terminal en este campo.
Ejecutar como un usuario diferente
Si desea ejecutar este programa como un usuario diferente (usted no), seleccione esta casilla, y proporcione el nombre de usuario en el campo Usuario
Puede asignar un acceso rápido especial de teclado para iniciar su programa.
Pulse el botón vacío a la derecha de la casilla tecla actual y pulse la combinación de teclas de su teclado que desea asociar con su programa.
Aparecerá un cuadro de diálogo, permitiendole asignar una segunda asociación de teclas al mismo elemento seleccionando el botón Alternar.
Esto puede resultar útil, por ejemplo, si cambia los mapas de teclado, y algunos accesos rápidos no son tan cómodos de introducir en todo momento
Pulse la x para borrar el acceso rápido, por ejemplo si cometió un error.
Seleccione la casilla Multi-tecla si desea asignar una acceso rápido que utilice más de una tecla.
Por omisión la casilla Auto cerrar está seleccionada, y el diálogo se cerrará cuando haya seleccionado una asociación de teclas.
Deseleccionela si desea que el diálogo permenzca abierto.
Menú Referencia
Archivo Nuevo elemento
Añade un nuevo elemento
Archivo Nuevo submenú
Añade un nuevo submenú
Sale de kmenuedit.
Corta el elemento acual de elemento hasta el portapapeles.
Si desea mover el elemento del menú, debería primero cortarlo hasta el portapapeles, moverlo al lugar de destino usando el panel izquierdo, y usar la función Pegar para pegar el elemento de menú desde el portapapeles.
Copia el elemento de menú actual hasta el portapapeles.
Más tarde puede usar la función Pegar para pegar el elemento del menú pegado desde el portapapeles hasta su destino.
Puede pegar el mismo elemento muchas veces.
Pegar el elemento del menú desde el portapapeles a la ubicación actualmente seleccionada en el menú Principal.
Puede utilizar primero Cortar o Copiar antes de que pueda Pegar.
Editar Borrar
Borra el elemento de menú actualmente seleccionado.
