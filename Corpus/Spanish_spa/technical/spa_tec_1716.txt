Función Join [Ejecución]
Devuelve una cadena a partir de varias subcadenas de una matriz.
Sintaxis:
Join (Texto As String Array, delimitador)
Valor de retorno:
Cadena
Parámetros:
Texto:
Una matriz de cadenas.
Delimitador (opcional):
Un carácter que se utiliza para separar las subcadenas en la cadena resultante.
El delimitador predeterminado es el carácter espacio.
Si el delimitador es una cadena de longitud cero &quot; &quot; , las subcadenas se unen sin utilizar ningún separador.
