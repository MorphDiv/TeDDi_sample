Configuración adicional
Especifica opciones adicionales para un origen de datos.
Editar - Base de datos - Propiedades - ficha Configuración adicional
La disponibilidad de los siguientes controles depende del tipo de base de datos:
Nombre de host
Escriba el nombre de host del servidor que contiene la base de datos, por ejemplo ldap.server.com.
Número de puerto
Escriba el número de puerto para el servidor que contiene la base de datos.
Clase de controlador MySQL JDBC
Escriba el nombre del controlador JDBC para la base de datos MySQL.
Juego de caracteres
Seleccione el conjunto de caracteres que desee usar para ver la base de datos en $[officename].
Esta opción no afecta a la base de datos.
Elija &quot; Sistema &quot; para utilizar el conjunto de caracteres predeterminado del sistema operativo.
Bases de datos de Texto y dBASE estan restringidas a juegos de caracteres de tamaño fijo, donde todos los caracteres están codificados con el mismo número de bytes.
Clase de controlador Oracle JDBC
Escriba el nombre del controlador JDBC para la base de datos Oracle.
Configuración del controlador
Especifique opciones de controlador adicionales.
Usar catálogo con bases de datos basadas en archivo
Usa el actual origen de datos del catálogo.
Esta opción es util cuando el origen de datos ODBC es un servidor de base de datos.
No seleccione esta opción si el origen de datos ODBC es un controlador dBASE.
Base DN
Escriba el punto de inicio para buscar en la base de datos LDAP, por ejemplo dc=com.
Número máximo de registros
Escriba el número máximo de registros que desee cargar cuando acceda al servidor LDAP.
Mostrar también registros eliminados
Muestra todos los registros de un archivo, incluidos los marcados como borrados.
Si selecciona esta opción, no podrá borrar registros.
En formato dBASE, los registros borrados permanecen en e archivo.
Para ver los cambios efectuados en la base de datos, cierre la conexión con ella y vuelva a conectar.
Índices
Abre el diálogo de índices, donde puede organizar la tabla de índices en la base de datos actual dBASE.
UFI:
Commercial means Adabas
Tamaño de memoria intermedia (MB)
Escriba el tamaño de la memoria intermedia de datos para la base de datos.
Esta configuración surte efecto tras reiniciar la base de datos.
Incremento de datos (MB)
Escriba el tamaño en el que desea incrementar automáticamente la base de datos.
El tamaño de incremento máximo es de 100 MB.
Esta configuración surte efecto tras reiniciar la base de datos.
Nombre de control de usuario
Escriba el nombre de un usuario al que desee conceder un control limitado para modificar algunos de los parámetros de la base de datos.
Contraseña de control
Escriba la contraseña del control de usuario.
Cerrar servicio al cerrar %PRODUCTNAME
Sale del servidor de base de datos Adabas al salir de $[officename].
Esta opción sólo está disponible si inicia el servidor de base de datos desde $[officename] con un control de usuario y contraseña.
Extendido
Estadísticas de base de datos
, en el que se pueden ver estadísticas de la base de datos Adabas.
El texto contiene encabezados
Seleccione esta casilla de verificación si la primera línea del archivo de texto contiene los nombres de los campos.
Separador de campo
Escriba o seleccione el carácter de separación de los campos de datos en el archivo de texto.
Separador de texto
Escriba o seleccione el carácter que identifica un campo de texto en el archivo de texto.
No se puede utilizar el mismo carácter usado como separador de campos.
Separador decimal
Escriba o seleccione el carácter utilizado como separador decimal en el archivo de texto; por ejemplo, el punto (0.5) o la coma (0,5).
Separador de miles
Escriba o seleccione el carácter utilizado como separador de miles en el archivo de texto; por ejemplo, el punto (1.000) o la coma (1,000).
Extensión del archivo
Seleccione el formato del archivo de texto.
La extensión seleccionada influye sobre algunos parámetros de configuración predeterminados de este diálogo.
