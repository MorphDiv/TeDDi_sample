Soporte de la plataforma Java
$[officename] soporta la plataforma Java para correr aplicaciones y componentes basados en la arquitectura JavaBeans.
Para que $[officename] admita la plataforma Java deberá instalar el software del Entorno de ejecución Java 2 de Sun Microsystems.
Durante la instalación de $[officename] se ofreció la opción de instalar dichos archivos, si no estaban ya instalados.
Si lo necesita, puede instalarlos ahora.
El archivo ejecutable de configuración se encuentra en la carpeta $[officename]/program y en el CD de $[officename].
Esta versión se denomina, en Windows, &quot; j2re-1_4_1_01-windows-i586-i.exe &quot; .
El nombre cambia de la forma correspondiente para otros sistemas operativos.
La admisión de plataforma Java se debe activar en $[officename] para poder ejecutar aplicaciones de Java.
Herramientas - Opciones - $[officename] - Seguridad
.
Antes de poder utilizar un controlador de JDBC, debe agregar la ruta de su clase.
Seleccione Herramientas - Opciones - %PRODUCTNAME- Java y haga clic en el botón Ruta de clases.
Después de agregar la información de ruta, reinicie %PRODUCTNAME.
Herramientas - Opciones - $[officename] - Seguridad
se aplicarán aunque la Máquina Virtual Java (JVM, una máquina virtual para la plataforma Java) ya se haya iniciado.
Después de efectuar modificaciones en ClassPath deberá reiniciar $[officename].
También deberá hacerlo en el caso de efectuar modificaciones en
Herramientas - Opciones - Internet - Proxy
.
Únicamente los cuadros &quot; Proxy Http &quot; y &quot; Proxy Ftp &quot; y sus puertos no requieren reiniciar, ya que se evalúan al pulsar Aceptar.
