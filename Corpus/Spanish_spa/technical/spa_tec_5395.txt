Comportamiento de Konqueror Aquí puede configurar el comportamiento de Konqueror como gestor de archivos.
Opciones varias
Abrir carpetas en & ventanas separadas
Si esta opción está activada, Konqueror abrirá una ventana nueva cuando abra una carpeta, en lugar de mostrar los contenidos de la carpeta en la ventana actual.
Mostrar entradas de « Borrar » en el menú contextual que ignoren la papelera
Marque esta opción si quiere que se muestren órdenes « Borrar » en los menús contextuales del escritorio y del gestor de archivos. Siempre puede borrar los archivos manteniendo pulsada la tecla de mayúsculas cuando elija « Mover a la papelera ».
Rutas Este módulo le permite elegir en que parte del sistema de archivos se almacenan los archivos del escritorio Use « ¿Qué es esto? » (Mayúsculas+F1) para obtener ayuda sobre las opciones específicas.
Ruta del escritorio:
Esta carpeta contiene todos los archivos que ve en su escritorio. Se puede cambiar la ubicación de esta carpeta si así lo desea, y los contenidos también serán transferidos automáticamente a la nueva dirección.
Ruta de autoinicio:
Esta carpeta contiene aplicaciones o enlaces a aplicaciones (teclas rápidas) que serán iniciados automáticamente cuando arranque KDE. Se puede cambiar la ubicación de este directorio si así lo desea, y los contenidos también serán transferidos automáticamente a la nueva dirección.
Ruta para documentos:
Esta carpeta será la predeterminada para cargar o guardar documentos.
Ruta para descargas:
Esta carpeta será la predeterminada para guardar los elementos que descargue.
Ruta para películas:
Esta carpeta será la predeterminada para cargar o guardar películas.
Ruta para imágenes:
Esta carpeta será la predeterminada para cargar o guardar imágenes.
Ruta para música:
Esta carpeta será la predeterminada para cargar o guardar música.
