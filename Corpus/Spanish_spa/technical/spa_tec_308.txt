Solver
Abre el diálogo de Solver.
Sover te permite resolver ecuaciones con multiples variables desconocidas por el metodo de busqueda de metas.
target cell
Ingrese o de clic en la referencia de la celda o la celda meta.
Este campo toma la dirección de la celda del cual el valor sera optimizado.
maximum
Intenta resolver la ecuación para un valor maximo de la celda usada.
minimum
Trate de solucionar la ecuación para un valor mínimo de la celda objetivo.
value of
Intenta resolver la ecuación para acercar el valor dado de la celda usada.
value of text field
Ingresa el valor o una referencia de celda.
by changing cells
Ingresa el rango de celdas que pueden ser modificadas.
cell reference
Ingrese una referencia de celda.
operators
Selecciona un operador de la lista.
value
Ingresa el valor de una celda de referencia.
shrinker
Haga clic aquí para reducir o restablecer el diálogo.
Puede hacer clic o seleccionar las celdas en la hoja.
Usted puede introducir manualmente una referencia de celda en el cuadro de entrada.
delete row
De clic para eliminar la fila de la lista.
Cualquier fila abajo de esta fila se mueve hacia arriba.
options button
Abre el diálogo de opciones.
solve button
Haga clic para solventar la ecuación con la configuración actual.
El diálogo de configuración se retiene hasta que se cierre el documento actual.
Para resolver ecuaciones con solver
celda objetvo
.
Usted puede elegir si el valor en la celda objetivo debe ser el máximo, mínimo, o acercarse a un determinado valor.
para cambiar celdas
caja.
Puede definir una serie de condiciones limitantes para definir los limites de algunas celdas.
Por ejemplo, puedes definir las limitantes de una variable o celda que no deba ser mas que otra variable, o no sea mas que un valor dado.
Puedes tambien definir las limitantes que una o mas variables debe ser enteros (valores sin decimales), o valores binarios (donde solo 0 y 1 son permitidos).
El motor predeterminado de solver soporta solo ecuaciones lineales.
