Acciones
Defina asignaciones de acciones para el campo de control o diálogo seleccionados.
Las acciones disponibles dependen del tipo de campo de control seleccionado.

Al recibir el enfoque
Este evento tiene lugar si un control recibe el foco.
Al cargar el enfoque
Este evento tiene lugar si un control pierde el foco.
Tecla pulsada
Este evento tiene lugar cuando el usuario pulsa una tecla cuando el campo de control tiene el foco.
Tecla suelta
Este evento tiene lugar cuando el usuario suelta una tecla mientras el control tiene el foco.
Modificado
Este evento tiene lugar si el control pierde el foco y el contenido del control cambia desde que perdió el foco.
Texto modificado
Este evento tiene lugar si se introduce o modifica un texto en un campo de entrada.
Estado modificado
Este evento tiene lugar si el estado del campo de control ha cambiado, por ejemplo, de marcado a desmarcado.
Ratón dentro
Este evento tiene lugar cuando el ratón entra en el control.
Ratón movido con la tecla pulsada
Este evento tiene lugar cuando el ratón se arrastra con una tecla pulsada.
Ratón movido
Este evento tiene lugar cuando el ratón se mueve sobre el control.
Botón del ratón pulsado
Este evento tiene lugar cuando se suelta el botón del ratón mientras el puntero se encuentra sobre el control.
Botón del ratón soltado
Este evento tiene lugar cuando se suelta el botón del ratón mientras el puntero se encuentra sobre el control.
Ratón fuera
Este evento tiene lugar cuando el ratón sale del control.
Al ajustar
