El elemento gráfico Webslice le permite mostrar parte de una página web en su escritorio o en un panel. Webslice es totalmente interactivo. Indique el URL de la página web en el campo URL. Introduzca un identificador CSS en el campo Elemento a mostrar (por ejemplo #mybox para elementos con el identificativo "mybox"). Este es el método preferido, porque funciona mejor ante cambios en la disposición de los elementos de la página web. Como alternativa, puede indicar un rectángulo de la página web a usar como porción. Utilice « x, y, anchura, altura » en píxeles. Por ejemplo, « 100,80,300,360 ». Este método es un último recurso para páginas web que no proporcionan suficientes etiquetas semánticas para el mecanismo anterior. Si se usan ambos métodos (elemento y geometría), el elemento tendrá precedencia.
