Método de autenticación
Elija su mecanismo de seguridad para acceder al servidor LDAP.
Las opciones son Sencillo o Kerberos.
Nombre del usuario
Si usa el método de autenticación &quot; sencillo &quot; , debe escribir el nombre del usuario en el servidor LDAP aquí.
Si usa &quot; Kerberos &quot; , sólo podrá ver el nombre de usuario, pero no editarlo.
Contraseña
Si usa el método de autenticación &quot; sencillo &quot; , debe escribir la contraseña para acceder al servidor LDAP aquí.
Si usa &quot; Kerberos &quot; , este cuadro no estará activo.
Guardar contraseña
Si usa el método de autenticación &quot; sencillo &quot; , marque este cuadro para que la contraseña sea permanente.
Si la contraseña es permanente y el mismo usuario inicia posteriormente $[officename], el nombre del usuario y la contraseña no se volverán a solicitar.
