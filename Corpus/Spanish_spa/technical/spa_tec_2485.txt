Aplicaciones multimedia
Puede encontrar las siguientes aplicaciones en las distribuciones de kdemultimedia.
Las aplicaciones estables están relativamente libres de errores y se pueden encontrar en formato fuente y binario
ftp.kde.org / pub / kde / stable / latest / distribution / stable
Las versiones inestables cambian diariamente y sólo están disponibles en formato fuente, lo que quiere decir que usted debe compilarlas para generar una versión que funcione.
Recuerde que son inestables y que puede que ni siquiera compilen.
Si tiene un problema con una aplicación inestable informe del problema y probablemente será corregida en un versión futura.
ftp.kde.org / pub / kde / unstable / CVS / snapshots / current
KMix
Un panel mezclador de sonidos
aKtion
Un reproductor de películas y animaciones, con soporte para muchos formatos.
Noatun
Un reproductor para archivos multimedia, con soporte para muchos formatos, listas de reproducción e incluso videos.
Kaboodle
Un reproductor multimedia de aRts ligero y versatil - no incluye listas de reproducción, ni propiedades extravagantes para enlentecerlo, simplemente reproduce rápidamente un sólo archivo.
Kscd
Un reproductor de CD para KDE, con soporte CDDB.
KMid
Un reproductor MIDI con soporte para la lírica... ¡su propia máquina de Karaoke!.
KMidi
Un reproductor MIDI agradable pero que demanda bastante CPU
