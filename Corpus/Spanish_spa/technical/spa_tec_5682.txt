La barra de herramientas principal contiene los botones comunes utilizadas. Pulsando con el botón izquierdo sobre un icono se activará la acción correspondiente. Algunos de los botones mostrarán deslizadores cuando se pulsan. Los deslizadores le permiten cambiar varios controles del reproductor: volumen de sonido, contraste de vídeo, brillo, color y saturación. Los controles de vídeo solo estarán disponibles para los archivos de vídeo.
La barra de herramientas de la lista de reproducción muestra el título multimedia cargado o que se va a reproducir, ofrece órdenes para ir al elemento elemento anterior o al siguiente de la lista de reproducción, y también le permite ver la lista de reproducción completa que está formada por los elementos que se están reproduciendo o que se va a reproducir. Si selecciona un elemento diferente de la lista, KPlayer lo cargará y comenzará la reproducción. La barra de herramientas también contiene opciones para repetir o barajar la lista de reproducción.
La barra de herramientas de progreso y búsqueda se muestra cuando se conoce la duración de un medio. Muestra el tiempo de reproducción y también permite buscar, esto es, mover el punto de reproducción adelante y atrás a lo largo del archivo. Para buscar una posición específica, arrastre el deslizador hasta la posición deseada con el botón izquierdo del ratón, o pulse en esta posición con el botón central. Para moverse hacia delante y hacia atrás en pasos, pulse el botón izquierdo sobre la barra a la derecha o a la izquierda del deslizador, o pulse los botones avanzar y retroceder.
La barra de herramientas de volumen tiene un deslizador que muestra el volumen de sonido actual y le permite cambiarlo. Éste es el mismo deslizador al que se puede acceder cuando aparece el deslizador de volumen tras pulsar el botón de volumen de la barra de herramientas principal, pero no desaparece cuando pulse en alguna parte. Puede mostrar u ocultar la barra de herramientas utilizando la opción Mostrar barra de herramientas de volumen en el menú Preferencias. Pulsar el botón Volumen en esta barra de herramientas también lo ocultará. Adicionalmente la barra de herramientas tiene un botón Silencio que activa o desactiva el sonido.
La barra de herramientas de Contraste tiene un deslizador para el contraste que muestra el contraste del vídeo actual y le permite cambiarlo. Éste es el mismo deslizador al que se puede acceder cuando aparece el deslizador de Contraste tras pulsar el botón de Contraste de la barra de herramientas principal, pero no desaparece cuando pulse en alguna parte. Puede mostrar u ocultar la barra de herramientas utilizando la opción Mostrar barra de herramientas de Contraste en el menú Preferencias. Pulsar el botón Contraste en esta barra de herramientas también lo ocultará.
La barra de herramientas de Brillo tiene un deslizador para el brillo que muestra el brillo del vídeo actual y le permite cambiarlo. Éste es el mismo deslizador al que se accede cuando aparece el deslizador de Brillo tras pulsar el botón de Brillo de la barra de herramientas principal, pero no desaparece cuando pulse en alguna parte. Puede mostrar u ocultar la barra de herramientas utilizando la opción Mostrar barra de herramientas de brillo en el menú Preferencias. Pulsar el botón Brillo en esta barra de herramientas también lo ocultará.
La barra de herramientas de Color tiene un deslizador para el color y le permite cambiarlo. Éste es el mismo deslizador al que se accede cuando aparece el deslizador de Color tras pulsar el botón de Color de la barra de herramientas principal, pero no desaparece cuando pulse en alguna parte. Saltar & marcos
Habilita o deshabilita el salto de imágenes para mantener la sincronización
La orden « Saltar fotogramas » activa o desactiva la opción saltar fotogramas. Si su sistema es demasiado lento para reproducir un archivo, MPlayer puede saltar algunos fotogramas, de forma que la reproducción no sea tan lenta. La opción Saltar fotogramas es menos agresiva que la opción Saltar más fotogramas, y no debería causar problemas de reproducción. La opción Saltar fotogramas también puede configurarse a través de la página Avanzada de forma global en las Preferencias de KPlayer o para un archivo de en particular a través de las propiedades del archivo.
Saltar más marcos
Habilita o deshabilita el mayor salto de imágenes para mantener la sincronización
La orden « Saltar más fotogramas » activa o desactiva la opción para saltar un número más grande de fotogramas. Si su sistema es demasiado lento para reproducir un archivo, MPlayer puede saltar algunos fotogramas para que la reproducción no sea tan lenta. Esta opción salta los fotogramas de una forma más agresiva y puede romper la codificación. La opción Saltar fotogramas también puede configurarse a través de la página Avanzada de forma global en las Preferencias de KPlayer o para un archivo de en particular a través de las propiedades del archivo.

La biblioteca multimedia está formada por una ventana en la que puede organizar sus archivos, transmisiones, dispositivos, gestionar sus listas de reproducción, y elegir qué elementos reproducir. Salto de & fotogramas
Elija cuándo y cómo MPlayer debería saltar fotogramas
Si su sistema es demasiado lento para reproducir este archivo, MPlayer puede saltarse algunos fotogramas para que la reproducción no sea tan lenta. De forma predeterminada se utiliza la configuración de KPlayer: Ninguno no realiza saltos, Suave salta algunos fotogramas, y Fuerte se salta más fotogramas y puede romper la decodificación. Si selecciona una opción de salto de fotogramas desde el menú Reproductor mientras mantiene pulsada la tecla Mayúsculas, KPlayer recordará esta elección para el archivo cargado.

Auto es la elección recomendada, y permite que MPlayer decida qué codec utilizar automáticamente. Si necesita decirle a MPlayer que utilice un códec en particular para un archivo o transmisión dada, modifique esta opción en las propiedades del archivo.
Ajuste de tamaño
Cambia el tamaño del vídeo cuando realice un escalado por software
Opción para especificar el tamaño del vídeo que se utilizará para realizar el escalado por software. El escalado por software consume una cantidad considerable de los recursos del sistema, por tanto, a menos que deba realizar el escalado por software para conseguir una mejor calidad de imagen, debe elegir una salida de vídeo que utilice un escalado por hardware.

Habilitar doble memoria intermedia
Activa una doble memoria intermedia para la salida de vídeo
Opción que indica si se utilizará una doble memoria intermedia para utilizarse en la salida de vídeo. Se recomienda esta opción, ya que se mostrará más suave en muchos casos.
Habilitar generación de vídeo directo
Activa la generación de vídeo directa para la salida de vídeo
Opción para indicar si se activará la generación de vídeo directa para utilizarse a la salida de vídeo. Esta opción puede mejorar el rendimiento, pero también puede dar problemas al mostrar el vídeo, por ejemplo, cuando se utiliza con la opción de doble memoria intermedia o cuando se reproduce con subtítulos.
