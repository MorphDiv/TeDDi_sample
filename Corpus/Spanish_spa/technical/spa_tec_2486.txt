Aplicaciones de red
Se pueden encontrar las siguientes aplicaciones en las distribuciones de kdenetwork.
Las aplicaciones estables están relativamente libres de errores y se pueden encontrar en formato fuente y binario
ftp.kde.org / pub / kde / stable / latest / distribution / stable
Las versiones inestables cambian diariamente y sólo están disponibles en formato fuente, lo que quiere decir que usted debe compilarlas para generar una versión que funcione.
Recuerde que son inestables y que puede que ni siquiera compilen.
Si tiene un problema con una aplicación inestable, informe del problema y probablemente será corregida en un versión futura.
ftp.kde.org / pub / kde / unstable / CVS / snapshots / current
KMail
Un cliente de correo electrónico completamente equipado, que permite el uso de múltiples servidores e identidades.
KNode
Un potente cliente de noticias usenet, con soporte para múltiples servidores NNTP.
Kit
Un cliente de mensajería instantánea de AOL, que utiliza el protocolo TOC.
KSirc
Un cliente IRC
KOrn
Un applet Biff.
Korn comprueba si hay correo nuevo y en caso de que lo haya lo indica para que pueda descargarlo de su( s) servidor(es).
kppp
Un marcador de Internet.
KTalkd
Los usuarios en su red utilizando el clásico talk pueden llamar su atención incluso en KDE utilizando KTalkd.
