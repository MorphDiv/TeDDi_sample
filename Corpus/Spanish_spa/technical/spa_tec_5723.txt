Está ejecutando una versión de Parley que todavía no se ha liberado oficialmente. Tenga en cuenta que esta versión puede ser inestable, colgarse o perder sus cambios. Informe de cualquier cosa que considere útil para mejorar Parley! http: / /bugs. kde. org
Parley le permite editar muchas propiedades adicionales de las palabras que puede introducir. Utilice « Preferencias » - > « Herramientas » para activar las diferentes herramientas que le ayudarán a establecer información adicional para su vocabulario. Esto incluye imágenes, sonido, tipos de palabra y mucho más. Elija aquellas que desee.
Para buscar una palabra, solo tiene escribirla en la barra de búsqueda.
Si no desea ver la respuesta después de pulsar entrar cuando está practicando, puede configurar Parley para saltar directamente a la siguiente entrada. También puede mostrar la respuesta durante un tiempo fijo. utilice « Configurar práctica » - > « Avanzada » - > « Mostrar solución después de la respuesta ».
Si desea ver todas las palabras de un determinado tipo (ejem. sustantivos), seleccione el tipo de palabra en la lista. Para mostrar la lista de tipos de palabra, utilice « Preferencias » - > « Herramientas » - > « Tipos de palabra » en el menú.
Puede arrastrar y soltar su lección para cambiar el orden e incluso hacerla depender de otra.
Puede arrastrar una palabra en la lista de tipos de palabra para cambiar su tipo. También puede seleccionar varias palabras a la vez.
Preferencias de las imágenes Una forma rápida de asignar una imagen a una palabra es soltarla en el área de la imagen.
Tipos de palabras Puede asignar una etiqueta especial a algunos tipos de palabra para permitir que Parley sepa su significado gramatical. Para los verbos, por ejemplo, puede introducir las conjugaciones.
Elección múltiple Para algunos vocabularios son mejores las pruebas de elección múltiple. Puede añadir opciones a una palabra que siempre se mostrará en el modo de elección múltiple. Si no, se generarán elecciones aleatorias a partir de otras entradas del documento (utilice para ello la herramienta de elección múltiple).
¿Desea introducir símbolos de pronunciación rápidamente? Realice una doble pulsación sobre el símbolo que necesite para añadirlo directamente.
Puede utilizar el diálogo « Descargar vocabulario nuevo » o visitar el sitio web en el que puede encontrar muchos documentos de vocabulario: http: / /edu. kde. org/ contrib/ kvtml2. php.
