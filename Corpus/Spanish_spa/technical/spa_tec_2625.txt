KNotes es una aplicación de notas de recordatorio para el escritorio.
Introducción
KNotes es un programa que le permite escribir el equivalente en el ordenador a las notas de recordatorio.
Las notas se guardan automáticamente cuanto sale del programa, y se muestran cuando abre el programa.
Puede imprimir y enviar por correo sus notas si configura KNotes para usar aplicaciones de apoyo.
Muestra características de las notas como el color y la tipografía que se pueden personalizar para cada nota.
También puede personalizar los valores predeterminados.
Configuración
Configurando la visualización en KNotes
Para configurar la visualización:
Clic derecho sobre el icono del panel.
Seleccione Preferencias...
Seleccione el icono Visualizar de los tres iconos de la izquierda.
Las siguientes opciones están disponibles:
Color del texto
El cuadrado de color muestra el color del texto actual.
Haciendo clic sobre este cuadrado de color, abre el diálogo de selección de color estándar de KDE.
Color de fondo
El cuadrado de color muestra el color de fondo actual.
Haciendo clic sobre este cuadrado de color, abre el diálogo de selección de color estándar de KDE.
Anchura predeterminada
La anchura de la nota en píxels.
Edite este número como desee.
Altura predeterminada
La altura de la nota en píxels.
Edite este número como desee.
Configurando el editor en KNotes
Para personalizar el editor:
Clic derecho sobre el icono del panel.
Seleccione Preferencias...
Seleccione el icono Editor de los tres iconos de la izquierda.
Las siguientes opciones están disponibles:
Tamaño de la tabulación
Este es el tamaño de la tabulación en espacios.
Edite este número como desee.
Sangrado automático
Esto es un cuadro de selección.
Si está marcado, el sangrado automático está activo.
Editar texto rico
Aún no implementado
Tipografía del título: clic para cambiar
Puede cambiar la tipografía usada para el título de sus notas.
Clic sobre este botón para abrir el diálogo de selección de tipografías estándar de KDE.
Tipografía del texto: clic para cambiar
Puede cambiar la tipografía usada para el texto de sus notas.
Clic sobre este botón para abrir el diálogo de selección de tipografías estándar de KDE.
Configurando acciones en KNotes
Para configurar acciones en KNotes:
Clic derecho en el icono del panel de KNotes.
Seleccione Preferencias...
El diálogo Predeterminado - KNotes de KNotes se abrirá.
Elija el icono Acciones de los tres iconos de la izquierda.
Las siguientes opciones están disponibles:
Acción de correo
Teclee un comando de correo y las opciones de línea de comando que se precisen en esta caja.
Usando %f en la línea de comando puede pasar el nombre de archivo del cuerpo de la nota para el comando de correo.
Usando KNotes
Creando una nota nueva
Para crear una nota nueva:
Clic derecho sobre el icono del panel de KNotes.
Seleccione Nueva nota.
Escribiendo su nota
Para escribir su nota, simplemente teclee la nota en el espacio proporcionado.
Se admiten las funciones de edición normales del teclado y el ratón.
Dar clic derecho sobre el espacio de edición proporciona las siguientes opciones de menú:
Deshacer
Rehacer
Cortar
Copiar
Pegar
Limpiar
Seleccionar todo
El texto se puede seleccionar manteniendo pulsado el botón izquierdo del ratón y moviendo el ratón, o manteniendo pulsada la tecla Shift y usando las teclas de flecha.
Insertando la fecha
Para insertar la fecha actual en la nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Insertar fecha.
La fecha y hora actual se insertará en el texto de la nota, en la posición del cursor.
Cambiando el nombre de una nota
Para cambiar el nombre de una nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Renombrar.
Teclee el nuevo nombre de la nota en el diálogo que aparece.
Para aceptar el nuevo nombre, presione el botón Aceptar.
Para salir del diálogo sin cambiar el nombre de la nota, pulse el botón Cancelar.
Para limpiar lo que ha tecleado y comenzar de nuevo, haga clic sobre el botón Limpiar.
Enviando una nota
Para enviar una nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Correo.
Lo próximo que ocurra variará en función de cómo haya configurado la acción de correo en el diálogo Preferencias....
Imprimiendo una nota
Para imprimir una nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Imprimir.
Se abrirá un diálogo de impresión estándar de KDE.
Borrando una nota
Para borrar una nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Eliminar.
Ocultando una nota
Para ocultar una nota, haga clic sobre la X en la esquina superior derecha de la barra de título de la nota.
La nota no volverá a mostrarse en pantalla.
La propia nota no se borrará.
Mostrando notas
Cuando inicia KNotes, todas las notas se mostrarán en la pantalla.
Si oculta una nota y luego quiere mostrarla:
Clic derecho sobre el icono del panel de KNotes.
Seleccione Notas.
Elija la nota que quiera mostrar.
Funciones de escritorio
Para enviar una nota a un escritorio específico:
Clic derecho sobre la barra de título de la nota.
Seleccione A escritorio
Elija el escritorio deseado, o en su caso, Todos los escritorios
Para asegurarse de que la nota permanezca por encima de las otras ventanas:
Clic derecho sobre la barra de título de la nota.
Seleccione Siempre encima.
Para devolver la nota al comportamiento de una ventana más normal, simplemente repita este proceso.
Personalizando una visualización de nota
Puede personalizar cada nota independientemente de la configuración global.
Las preferencias que puede personalizar son idénticas a las descritas en la sección Configuración, pero sólo se aplicarán a las notas que está cambiando.
Cualesquiera otras notas abiertas, y cualquier nota nueva que cree, usarán la configuración predeterminada.
Para personalizar la visualización de una nota:
Clic derecho sobre la barra de título de la nota.
Seleccione Preferencias de la nota....
Aparecerá el diálogo Configuración local.
Seleccione el icono Visualizar de los tres iconos de la izquierda.
Saliendo de KNotes
Para salir de KNotes:
Clic derecho sobre el icono del panel.
Seleccione Salir.

Instalación
Cómo obtener KNotes
KNotes forma parte del proyecto KDE http: / /www.kde.org /.
KNotes está incluido en el paquete kdepim en ftp: / /ftp.kde.org / pub / kde /, el servidor FTP del proyecto KDE.
Requisitos
Para imprimir o enviar por correo notas, necesita tener aplicaciones de apoyo instaladas, como a2ps y KMail.
Necesitará configurar KNotes para usar esas aplicaciones de apoyo.
