Kghostview muestra e imprime PostScript (.ps, .eps) y archivos Portable Document Format (.pdf).
Este documento describe la versión 0.13pre2 de Kghostview.
Introducción
Kghostview muestra e imprime archivos PostScript (.ps, .eps) y archivos con formato de documento portátil (.pdf).
Es una versión para KDE del programa de Tim Theisen llamado Ghostview que utiliza el programa Alladin Ghostscript para ver documentos preparados en el leguaje de descripción de página PostScript de Adobe.
PostScript es el leguaje de descripción de página más común para imprimir en los sistemas UNIX y esta aplicación sirve para previsualizar el material antes de imprimirlo y para leer documentos en línea.
Puede utilizar Kghostview con todas las versiones recientes de Ghostscript.
La versiones más modernas ofrencen un rendimiento muy superior del que se puede sacar ventaja con Kghostview.
Si un documento no cumple con la convención de estructura del documento de Adobe, la funcionalidad del visor es limitada.
Por ejemplo, si no hay tabla de contenidos, no es posible saltar de una página a otra ni seleccionar las páginas.
Fundamentos
Esta sección describe lo que se ve en la ventana principal de Kghostview, las acciones de los botones de la barra de herramientas y cómo utilizar la lista de páginas.
Es posible abrir varias sesiones de Kghostview de forma simultánea para visualizar diferentes documentos.
La barra de título en la parte superior de la ventana muestra el nombre del documento en esa ventana.
Nótese que la barra de herramientas y los campos de texto que ayudan en la navegación a través del documento puede ser ocultados a través del menú Opciones para tener más en espacio en la pantalla para la zona del contenido.
El área principal muestra una página del documento actual.
Si la página es demasiado grande para entrar en la ventana, automáticamente se añaden barras de desplazamiento a los lados de la imagen, aunque pueden ser desactivadas a través del menú Opciones.
También puede desplazarse por una página con las flechas del teclado arriba y abajo.
Los botones de la barra de herramientas página siguiente y página anterior hacen lo que indican, o también se puede ir inmediatamente a cualquier página pinchando con el botón izquierdo del ratón en la lista de páginas.
Si la página es demasiado alta para mostrarse completamente en la ventana, los botones leer hacia arriba el documento y leer hacia abajo el documento pueden ser utilizados para desplazarse arriba y abajo por todo el documentos.
Si se activa en el menú Opciones, se mostrará una lista de páginas en la parte izquierda de la ventana.
Esta lista tiene dos columnas: la primera contiene una bandera indicando que esa página ha sido marcada, la segunda contiene los números de página.
Puede utilizar esta lista de páginas para navegar por el documentos o para marcar páginas para la impresión.
También puede limpiar o cambiar las marcas de páginas con el menú desplegable que se activa pinchando con el botón derecho del ratón en cualquier parte del área de la lista de páginas o en el menú Marcas de páginas.

El menú Marcas de páginas
También se puede abrir este menú pinchando con el botón derecho del ratón en cualquier lugar de la lista de páginas.
Cambia la marca de la página actual.
Marcas de páginas Marcar todas las páginas
Cambiar la marca de todas las páginas del documento.
Marcas de páginas Marcar páginas pares
Activa la marca de todas las páginas numeradas pares en el documento.
Usado en unión de Marcar páginas impares o Cambiar marcas de páginas, esto proporciona un método sencillo de imprimir un documento a doble cara en una impresora de una sola cara.

Configuración
Consta de dos páginas, Opciones generales y Configuración de Ghostscript
Opciones generales
Puede seleccionar Activar antialiasing de fuentes e imágenes para obtener texto de bordes más suaves, pero teniendo en cuenta que el antialiasing consume mucha memoria y es más lento que el sistema de fuentes normal.
Las versiones antiguas de Ghostscript no tienen la opción de antialiasing.
Seleccione Utilizar fuentes de plataforma si desea utilizar sus fuentes nativas del sistema en vez de las que vienen con Ghostscript.
Active la opción Mostrar mensajes en una ventana separada si desea que se le informe de cualquier salida o mensaje de error que genere el intérprete Ghostscript.
Puede elegir si Kghostview utiliza una Paleta Monocromo, Escala de grises o Color para su funcionamiento.
Configuración de Ghostscript
En este cuadro de diálogo puede establecer el nombre del ejecutable del Intérprete Ghostscript y los argumentos de no antialiasing y de antialiasing que se le comunicarán.
La configuración predeterminada debería ser la adecuada para la mayoría de los sistemas.

Instalación
Kghostview es parte del proyecto KDE 3, se pueden encontrar detalles del mismo en http: / /www.kde.org.
Para utilizar Kghostview, debe tener instalados tanto el programa Ghostscript como KDE 3.
La página web de Ghostscript es http: / /www.cs.wisc.edu / ~ghost /
La mayoría de las distribuciones incluyen Kghostview, pero si desea compilar el código fuente usted mismo, puede encontralo en el paquete kdegraphics en ftp: / /ftp.kde.org / pub / kde /, el servidor FTP principal del proyecto KDE.
Para comprobar si existe una versión más moderna de Kghostview, puede mirar en http: / /apps.kde.com.
