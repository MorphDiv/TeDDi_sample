Este capítulo describe cómo realizar algunas tareas comunes en & kpresenter;.
Crear una presentación de diapositivas en HTML
Convierte una presentación existente en páginas HTML estándar con ayudas para la navegación.
En primer lugar, haga una presentación nueva o abra una existente. Luego, haga clic en Archivo Crear presentación HTML...
Opción del menú archivo para Crear presentación HTML...
Opción del menú archivo para Crear presentación HTML...
Un cuadro de diálogo le preguntará si quiere usar una configuración guardada anteriormente:
Cuadro de diálogo de configuración anterior
Cuadro de diálogo de configuración anterior
Haga clic en No si esta es la primera vez que utiliza esta característica, o si quiere cambiar las opciones específicas. La opción Sí le permite usar detalles que ha introducido con anterioridad para sus presentaciones HTML, para que pueda crear rápidamente la presentación sin tener que ajustar las opciones.
Obtendrá entonces un cuadro de diálogo que le solicitará unos pocos detalles:
Rellenar los detalles para su presentación HTML
Rellenar los detalles para su presentación HTML
Rellene cada campo (en caso de existir, el cuadro de diálogo muestra la información que haya almacenada en & kcontrolcenter;) y asegúrese de que los detalles de la ruta correctos, como dónde debe guardar & kpresenter; sus nuevos archivos HTML e imágenes. Pulse el botón Siguiente cuando haya terminado.
Si la ruta que seleccione en esta etapa no existe, & kpresenter; le preguntará que si quiere crearla.
Ahora puede configurar las opciones específicas de HTML:
Cumplimente los detalles adicionales para su presentación HTML
Cumplimente los detalles adicionales para su presentación HTML
Aquí se pueden cambiar los detalles de la codificación y del formato, junto con el nivel de zoom de las diapositivas. Por ejemplo, si originalmente diseñó sus diapositivas para que se mostraran a pantalla completa con una resolución de 1280x1024, podría escalar las diapositivas al 50% para permitir que se vean fácilmente en una ventana del navegador mucho más pequeña.
En la mayoría de los casos, las opciones predeterminadas están bien, y puede pulsar Siguiente tranquilamente para ir a la siguiente página.
Ahora puede personalizar los colores de las páginas web:
Configurar el estilo de visualización de su presentación HTML
Configurar el estilo de visualización de su presentación HTML
Cuando & kpresenter; crea para usted las páginas web, los colores del texto para las ayudas de la navegación serán los que defina aquí. Para cambiar cualquiera de los colores, haga clic en la barra de colores que hay junto a la correspondiente etiqueta del texto. Los colores que utilice en las diapositivas permanecerán sin cambios.
Elija colores que sean complementarios al contenido de su presentación.
El botón Siguiente presenta un cuadro de diálogo que le permite cambiar el nombre usado en la página HTML para cada diapositiva:
Configurar los nombres de las diapositivas
Configurar los nombres de las diapositivas
Si pretende que su presentación se reproduzca automáticamente (ie; avanzar automáticamente a la siguiente diapositiva, & etc;), también puede hacer que la presentación HTML equivalente se exhiba con el mismo comportamiento:
Presentaciones automáticas
Presentaciones automáticas
Aquí puede forzar el avance de una diapositiva a la siguiente tras un tiempo, y también hacer que la presentación vuelva a comenzar cuando llegue al final.
Es la hora de producir efectivamente su presentación HTML, cuyo progreso se mostrará en este cuadro de diálogo:
Guardar su configuración
Guardar su configuración
Por último, puede guardar la configuración que acaba de usar antes de cerrar el cuadro de diálogo, pulsando el botón Guardar configuración....
Lo que tenemos ahora son las carpetas html/ y pics/, que se han creado en la ruta que definimos anteriormente:
Las nuevas carpetas para la presentación HTML
Las nuevas carpetas para la presentación HTML
También hay un archivo index. html para lanzar su presentación web.
Puede ver cómo funciona usando su gestor de archivos o su navegador web para abrir el archivo index. html. Haga clic donde se le pida, y se mostrará la primera página de su presentación. Ahora se comportará como una presentación de & kpresenter;.
Cada pulsación en los iconos de la flecha le envían a la diapositiva anterior o siguiente. Para volver al inicio de la presentación pulse en el icono diapositiva.
Para compartir su presentación con el mundo, transfiera los archivos que se han producido a una ubicación accesible desde la red (ie; espacio web), asegurándose de mantener intacta la estructura de directorio.
Exportar presentaciones de & kpresenter; a una Memory Stick.
Algunos proyectores de Sony tienen la capacidad de hacer funcionar directamente una presentación desde un Memory Stick, sin necesidad de conectar un computador al proyector. & kpresenter; puede exportar las presentaciones directamente al formato necesario para estos proyectores, y este tutorial le mostrará cómo exportar su presentación.
Cada presentación de diapositivas consta de varios archivos de imagen (uno por cada diapositiva de la presentación original, más dos diapositivas de título) y un archivo de índice. Se crean en una estructura de directorios relativamente compleja, que permite que se guarde más de una presentación en una sola memory stick. Abajo se muestra un ejemplo sencillo, que se compone de dos presentaciones.
Estructura de directorios de Memory Stick
Si la presentación que quiere exportar no está abierta todavía, vaya a Archivo Abrir... y seleccione la presentación que quiera exportar.
Ahora seleccione el menú Archivo y elija Crear presentación en Memory Stick....
La selección del menú Crear presentación en Memory Stick... de & kpresenter;.
La selección del menú Crear presentación en Memory Stick... de & kpresenter;.
Esto hace aparecer el cuadro de diálogo Crear presentación para Memory Stick.
El cuadro de diálogo Crear presentación en Memory Stick.
El cuadro de diálogo Crear presentación en Memory Stick.
La Ruta es el directorio en el que se va a exportar su presentación como directorios (DCIM y MSSONY, consulte el diagrama de la estructura de directorio). Si es posible, puede querer exportar directamente a Memory Stick, en cuyo caso debe introducir aquí la ubicación del medio.
El Título se usa tanto para el nombre del archivo de índice como para la etiqueta de texto de la diapositiva del título. Tenga en cuenta que este título no afecta a los títulos de ninguna de las diapositivas normales de la presentación. La diapositiva del título se ve durante la configuración del proyector previa al comienzo de la visualización de la presentación real.
Seleccione la opción Definir colores para expandir el cuadro de diálogo:
El cuadro de diálogo Crear presentación en Memory Stick (extendido).
El cuadro de diálogo Crear presentación en Memory Stick (extendido).
Estos dos selectores de color sólo se usan para la diapositiva de título. Puede ser útil para cambiar las diapositivas del título para hacer más fáciles las presentaciones para identificarlas cuando funcionen en el proyector, aunque es perfectamente válido dejar los valores predefinidos.
Puede pulsar el botón Aceptar para seguir adelante con el proceso de exportación actual.
Su presentación se exportará ahora en el formato correcto, con el siguiente cuadro de diálogo que le informa del progreso del proceso:
El cuadro de diálogo de progreso Crear presentación en Memory Stick de & kpresenter;.
El cuadro de diálogo de progreso Crear presentación en Memory Stick de & kpresenter;.
Cuando se haya completado la exportación, puede pulsar el botón Hecho para cerrar el cuadro de diálogo.
Si no exportó la presentación directamente a una Memory Stick, tendrá que copiar los archivos adecuados a la Memory Stick antes de poder utilizar el proyector. Como se indicó antes, tiene que copiar los directorios adecuados y los archivos que comienzan por DCIM y MSSONY en la « raíz », o / de la Memory Stick. Es crítico que se mantenga la estructura del directorio, y posiblemente pueda querer copiar los directorios DCIM y MSSONY completos.
Si fuera necesario, puede cambiar el nombre al archivo de índice (que de forma predefinida es Slideshow. SPP). Sin embargo, no puede cambiar el nombre a las imágenes de las diapositivas, ni tampoco puede cambiar el nombre de los directorios, porque están codificados en el interior del archivo de índice.
Creandos plantillas de & kpresenter;
Es muy fácil añadir nuevas plantillas a la biblioteca de & kpresenter;.
Si ha hecho una presentación con un estilo que le gustaría volver a usar, puede guardarlo como una plantilla.
Este tutorial le mostrará cómo podemos hacer una plantilla nueva y guardarla.
Abra & kpresenter; y cree un documento nuevo.
Para añadir un pequeño destello a la diapositiva vacía, podemos cambiar el fondo de la diapositiva (para eso, vaya a Formato > Fondo de la diapositiva... o seleccione el mismo elemento del menú & RMB;:
El cuadro de diálogo Fondo de diapositiva de & kpresenter;.
El cuadro de diálogo Fondo de diapositiva de & kpresenter;.
Queremos que el fondo sea atractivo, así que vamos a añadir un gradiente al fondo usando el menú desplegable, y luego seleccione los colores del gradiente haciendo clic en los cuadros de colores. Por supuesto, puede querer usar una imagen como fondo y hacer clic en la solapa Imagen: experimente, pero déjelo bonito.
Seleccionar un fondo de diapositiva
Seleccionar un fondo de diapositiva
La página debiera tener ahora el fondo que desee. Cree un cuadro de texto y teclee un encabezado para la diapositiva:
Añadir un encabezado a la nueva plantilla
Añadir un encabezado a la nueva plantilla
Cuando esté conforme con su plantilla, vaya a Archivo Gestor de plantillas:
Archivo Gestor de plantillas
Archivo Gestor de plantillas
Este cuadro de diálogo le permite seleccionar dónde le gustaría guardar su plantilla:
Guardar la plantilla nueva
Guardar la plantilla nueva
Le recomiendo encarecidamente que guarde la plantilla en el grupo Presentaciones de pantalla. Puede llamarlo de la forma que quiera. En mi caso, la he llamado Plantilla_Tutorial. Cuando esté contento con las opciones, presione Aceptar para añadir su propia plantilla personalizada a la colección de plantillas.
Ahora puede cerrar el documento en el que ha estado trabajando (no es necesario que lo guarde). La próxima vez que se le presente el diálogo de inicio, su propia plantilla estará disponible para seleccionarla.
