Aplicaciones de utilidades
Se pueden encontrar las siguientes aplicaciones en las distribuciones de kdeutils.
Las aplicaciones estables están relativamente libres de errores y se pueden encontrar en formato fuente y binario
ftp.kde.org / pub / kde / stable / latest / distribution / stable
Las versiones inestables cambian diariamente y sólo están disponibles en formato fuente, lo que quiere decir que usted debe compilarlas para generar una versión que funcione.
Recuerde que son inestables y que puede que ni siquiera compilen.
Si tiene un problema con una aplicación inestable informe del problema y probablemente será corregida en un versión futura.
ftp.kde.org / pub / kde / unstable / CVS / snapshots / current
kcharselect
Selector de caracteres (muestra todos los caracteres disponibles de un determinado tipo de letra, incluyendo aquellos para los que no hay teclas en el teclado)
kdepasswd
Un interfaz KDE para el comando UNIX passwd.
kdessh
Un interfaz KDE para ssh.
kdf
Un interfaz KDE para la utilidad df, con funcionalidades añadidas
Kfloppy
Un formateador de disquetes
KEdit
Un editor pequeño y sencillo, como el conocido Notepad.
Es adecuado para pequeñas tareas de edición, o como navegador de archivos de textos sencillos.
KArm
Un seguidor de tiempos.
Puede introducir una lista de tareas, iniciar el reloj de KArm, y contará el número de minutos que usted emplea en la tarea resaltada.
KHexEdit
Un editor hexadecimal completo
KJots
Un pequeño programa que toma notas.
klaptopdaemon
Monitoriza las baterías y proporciona gestión para portátiles.
KLjettool
Un panel de control para las impresoras HP LaserJet
Klpq
Klpq es un interfaz para lpq, lprm y lpc.
KNotes
Una utilidad manejable y de buen aspecto para tomar notas.
