Propiedades del archivo
Introducción
El diálogo Propiedades del archivo le permite elegir varias opciones que solo se aplicarán a un archivo, emisión, dispositivo, pista, título o canal en particular. Muchas de las opciones son las mismas o similares a las que se pueden elegir globalmente en el diálogo Preferencias de & kplayer;. Las opciones establecidas en las propiedades del archivo sobreescribirán a las globales. Las propiedades del archivo le dan un control completo sobre la forma en que se reproducirá el archivo, incluyendo las opciones que se pasarán a & mplayer;. Otras propiedades muestran información sobre & kplayer; que se encuentra fuera del archivo o emisión, como la duración, el tamaño del vídeo y los ratios de bit de audio y vídeo.
Seleccione Archivo Propiedades... para abrir el diálogo para el archivo o URL cargados. Puede abrir las propiedades para cualquier elemento en la biblioteca multimedia pulsando con el botón derecho sobre el elemento y seleccionando Propiedades... en el menú contextual de la biblioteca o seleccionando Biblioteca Propiedades.
El diálogo contiene varias secciones que se pueden seleccionar con el botón izquierdo de los elementos en la parte izquierda del diálogo.
Sección General
Muestra información general sobre el elemento multimedia y le permite establecer algunas propiedades esenciales.
Tamaño sección
Si el archivo o emisión contiene vídeo, le indica la resolución y le permite controlar el tamaño del área de vídeo durante la reproducción.
Sección Vídeo
Contiene propiedades que muestran información sobre el vídeo y le permite controlar la salida de vídeo al reproducir el archivo, si tiene un componente de vídeo.
Sección Audio
Contiene propiedades que muestran información técnica sobre el audio y le permiten controlar la salida del audio cuando reproduce el archivo, si tiene un componente de audio.
Sección Subtítulos
Contiene las propiedades que controlas la carga de subtítulos y cómo se muestran, si el título se ha reproducido con subtítulos.
Sección Avanzadas
Contiene varias propiedades que afectan a los parámetros avanzados que se pasa a través de la línea de órdenes a & mplayer;, así como la interacción con el esclavo E/ S & kde; si se utiliza para transferir el archivo.
Los botones de la parte inferior del diálogo le permite hacer elecciones sobre los contenidos de todas las secciones del diálogo.
Botón Aceptar
Acepta los cambios y cierra el diálogo.
Botón Cancelar
Cierra el diálogo sin guardar los cambios.
Botón Aplicar
Aplica los cambios sin cerrar el diálogo.
Botón Predeterminados
Restaura todas las configuraciones con los valores originales predefinidos y los aplica.
Botón Cerrar
Después de elegir Aplicar o Predeterminados, el botón Cancelar se cambiará por el de Cerrar. Pulsándolo se cerrará el diálogo y se descartarán los cambios que haya hecho desde la última vez que se aplicaron.
Botón Ayuda
Abre este manual de usuario y muestra la sección que describe la parte seleccionada del diálogo.
También puede pulsar el pequeño botón ¿Qué es esto? con un signo de interrogación en la parte derecha del diálogo y pulsar sobre una propiedad para obtener una breve explicación sobre ella.
Pulsando el botón situado en la parte superior derecha del diálogo éste se cerrará descartando cualquier cambio que no se haya aplicado.
Sección General
Esta sección muestra información sobre el elemento multimedia y le permitirá establecer algunas propiedades esenciales.
Nombre
Esta propiedad muestra y le permite cambiar el nombre del elemento que se está mostrando en el desplegable de la lista de reproducción, la biblioteca multimedia y también en la barra de título de & kplayer; durante la reproducción.
Ruta o dirección
Esta propiedad muestra la ruta al archivo para los archivos locales o el URL para los demás archivos. El URL identifica el archivo o emisión, y todas las demás propiedades están asociadas a él. Esta propiedad es de solo lectura. No es aplicable y no se muestra para las pistas de CD, títulos de DVD y canales de los dispositivos de sintonización.
Tipo
Esta propiedad muestra el tipo de elemento multimedia. Para un dispositivo, pista, título o canal es el tipo de dispositivo. Para los demás elementos es el tipo MIME si se conoce. Esta propiedad es de solo lectura.
Lista de reproducción
Esta propiedad le permite elegir si este elemento es un archivo de lista de reproducción. & mplayer; no puede detectar automáticamente y reproducir una archivo de lista de reproducción, por eso & kplayer; debe indicárselo al analizar un archivo como una lista de reproducción. Auto le dice a & kplayer; que utilice la extensión de del archivo para adivinar si es un archivo de lista de reproducción. Las extensiones de las listas de reproducción que reconoce KPlayer son ram, rpm, smi, smil, asx, m3u, pls y strm. Si un archivo con una de estas extensiones no es una lista de reproducción, o un archivo de lista de reproducción tiene una extensión diferente, necesitará corregir esta propiedad para poder reproducir el archivo o la emisión.
Esta propiedad no se aplica y no se muestra para los dispositivos, discos, títulos, pistas y canales.
Duración
Esta propiedad muestra la duración del archivo. & kplayer; intenta detectarla con la precisión de una décima de segundo, y suele ajustarse bien a la duración real. Esta propiedad es de sólo lectura. No es aplicable y no se muestra para los dispositivos y canales.
Los dispositivos sintonizadores como DVB, PVR y TV, y sus canales disponibles tienen algunas propiedades específicas que no se aplican a otros elementos.
Frecuencia
Esta propiedad muestra y le permite cambiar la frecuencia en MHz de un canal DVB, PVR o TV. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Canales
Esta propiedad muestra y le permite elegir la lista de canales disponibles en un dispositivo TV o PVR. Para los demás elementos esta propiedad no se aplica y no se muestra.
Controlador
Esta propiedad muestra y le permite elegir el controlador de entrada para un dispositivo de TV o PVR. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Archivo de canal
Esta propiedad muestra y le permite cambiar la ruta a un archivo que lista los canales disponibles en un dispositivo DVB y sus propiedades. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Sección Tamaño
Esta sección muestra la resolución de vídeo y le permite controlar el tamaño del área de vídeo cuando reproduzca este archivo, si contiene un vídeo.
Resolución
Esta propiedad muestra la resolución del vídeo en puntos. Es de sólo lectura, y estará vacía para los archivos que solo contengan audio.
Tamaño original
Esta propiedad muestra el tamaño original del vídeo en puntos. El tamaño del vídeo original es la resolución de vídeo ajustada para corregir la relación de aspecto. Es de sólo lectura, y estará vacía para los archivos que solo contengan audio.
Tamaño actual
Esta propiedad muestra el tamaño original del vídeo en puntos. El tamaño de vídeo actual es el tamaño de vídeo después de aplicar cualquier filtro. Es de sólo lectura, y estará vacía para los archivos que solo contengan audio.
Tamaño de visualización
Esta propiedad le permite elegir el tamaño inicial del vídeo en & kplayer; que se utilizará cuando comience la reproducción del archivo, a menos que haya desactivado la opción para Ajustar el tamaño de la ventana automáticamente en la página General de las Preferencias de KPlayer. La configuración Automático utilizará la opción Tamaño inicial de vídeo mínimo de las configuración General. La opción ajustar tamaño se utilizará para fijar un tamaño inicial que haya especificado. La opción ajustar aspecto utilizará nuevamente la opción Tamaño inicial de vídeo mínimo, pero aplicará el aspecto inicial que haya especificado.
Pantalla completa
Esta propiedad le permite elegir el modo de pantalla completa cuando se reproduce este archivo. El valor predeterminado mantendrá el modo de pantalla completo del archivo anterior.
Maximizada
Esta propiedad le permite elegir el estado de ventana maximizado cuando este archivo no se reproduce a pantalla completa. El valor predeterminado mantendrá el estado maximizado del archivo anterior.
Mantener aspecto
Esta propiedad le permite establecer la opción Mantener aspecto específicamente para este archivo. El valor predeterminado mantendrá la opción Mantener aspecto que haya elegido globalmente.
Sección Vídeo
Esta sección contiene propiedades que muestran información técnica sobre el vídeo y le permiten controlar la salida de vídeo cuando se está reproduciendo el archivo, si éste tiene un componente de vídeo.
Pista
Este propiedad le permite elegir la pista de vídeo que se utilizará para reproducir este archivo o emisión. La opción auto le indica a & mplayer; que elija automáticamente una pista de vídeo. La opción ajustar ID le permite introducir un ID de la pista de vídeo a reproducir. Las demás opciones eligen una pista de vídeo de la lista de pistas detectadas. Una forma sencilla de establecer esta propiedad para un archivo o emisión que tiene más de una pista de vídeo es seleccionar una pista de vídeo desde el submenú Vídeo del menú Reproductor.
Contraste
Esta propiedad controla el contraste del vídeo para este archivo. La opción predeterminada utiliza el mismo contraste que para los demás archivos. La opción ajustar a establece el contraste inicial especificado cuando se carga el archivo. Las opciones añadir y restar ajustarán el contraste respecto a las configuraciones predeterminadas.
Brillo
Esta propiedad controla el brillo del vídeo para este archivo. La opción predeterminada utiliza el mismo contraste que para los demás archivos. La opción ajustar a establece el contraste inicial especificado cuando se carga el archivo. Las opciones añadir y restar ajustarán el contraste respecto a las configuraciones predeterminadas.
Color
Esta propiedad controla el color del vídeo para este archivo. La opción predeterminada utiliza el mismo contraste que para los demás archivos. La opción ajustar a establece el contraste inicial especificado cuando se carga el archivo. Las opciones añadir y restar ajustarán el contraste respecto a las configuraciones predeterminadas.
Saturación
Esta propiedad controla la saturación del vídeo para este archivo. La opción predeterminada utiliza el mismo contraste que para los demás archivos. La opción ajustar a establece el contraste inicial especificado cuando se carga el archivo. Las opciones añadir y restar ajustarán el contraste respecto a las configuraciones predeterminadas.
Codificador
Esta propiedad especifica los codificadores de vídeo que se utilizarán cuando se reproduzca este archivo. La opción predeterminada utilizará los codificadores elegidos en la página Vídeo en las Preferencias de & kplayer;, normalmente automático. Automático le indicará a & mplayer; que elija el codificador que piense que es el mejor para reproducir este archivo.
Tasa de bits
Esta propiedad muestra la tasa de bits para el vídeo de este archivo. Es de sólo lectura, y está vacía para los archivos que solo contienen audio.
Tasa de fotogramas
Esta propiedad muestra la tasa de fotogramas de vídeo para este archivo. Es de sólo lectura, y está vacía para los archivos que solo contienen audio.
Los dispositivos sintonizadores como DVB, PVR y TV tiene algunas propiedades específicas que no se aplican a los demás elementos.
Entrada
Esta propiedad le permite elegir la entrada de vídeo que se utilizará cuando se reproduce desde un dispositivo sintonizador. La opción auto le indica a & mplayer; que elija automáticamente una entrada de TV o DVB. La opción ajustar a permite introducir el ID de la entrada que se utilizará. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Formato
Esta propiedad le permite elegir el formato de vídeo que se usará cuando se reproduzca desde un dispositivo PVR o TV. La opción auto le indica a & mplayer; que elija un formato de salida de vídeo automáticamente. Las demás opciones le permiten especificar un formato de vídeo particular. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Norma
Esta propiedad le indica la norma de vídeo que se utilizará para reproducir desde el dispositivo de PVR o TV. La opción auto le indica a & mplayer; que elija automáticamente una norma de vídeo. La opción ajustar ID le permite introducir la ID de norma que se utilizará. Las demás opciones le permiten elegir una norma de vídeo particular. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Sección Audio
Esta sección contiene propiedades que muestran información técnica sobre el audio y le permiten controlar la salida de audio cuando se está reproduciendo el archivo, si éste tiene un componente de audio.
Esta propiedad le permite elegir la pista de audio que se utiliza cuando se reproduce este archivo o emisión. La opción auto permite a & mplayer; elegir la pista de audio automáticamente. La opción ajustar ID le permite introducir el ID de la pista de audio que desea reproducir. Las demás opciones eligen una pista de audio de la lista de pistas detectadas. Una forma sencilla de establecer esta propiedad para un archivo o emisión que tenga más de una pista de audio es seleccionar una pista de audio en el submenú Audio del menú Reproductor.
Volumen
Esta propiedad controla el volumen de sonido para este archivo. La opción predeterminada utiliza el mismo volumen que los demás archivos. La opción ajustar a establece el volumen inicial cuando se carga el archivo. Las opciones añadir y restar ajustarán el volumen en relación con la opción predefinida.
Retraso
Esta propiedad controla el retraso del sonido respecto al vídeo. De forma predeterminada esta opción recuerda automáticamente cuando se utilizaron las órdenes Aumentar retraso y Reducir retraso en el submenú Audio del menú Reproductor. La configuración predeterminada reiniciará el retraso a cero al cargar el archivo.
Esta propiedad especifica los codificadores de audio que se utilizarán cuando se reproduzca este archivo. La opción predeterminada utilizará los codificadores elegidos en la página Audio en las Preferencias de & kplayer;, normalmente Automático. Automático le indicará a & mplayer; que elija el codificador que piense que es el mejor para reproducir este archivo.
Esta propiedad muestra la tasa de bits para el audio de este archivo. Es de sólo lectura, y está vacía para los archivos que solo contienen audio.
Tasa de muestreo
Esta propiedad muestra la tasa de muestreo para el audio de este archivo. Es de sólo lectura, y está vacía para los archivos sin pista de audio.
Modo
Esta propiedad le permite elegir el modo de audio que se utilizar para reproducir desde dispositivos PVR o TV. La opción auto le indica a & mplayer; que elija automáticamente un modo de audio. Las demás opciones le permitirán especificar un modo de audio particular. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Esta propiedad le permite elegir la entrada de audio que se utilizará para la reproducción desde el dispositivo sintonizador. La opción auto le indica a & mplayer; que elija automáticamente una entrada de audio de TV o DVB. La opción ajustar a le permite introducir el ID de entrada que se utilizará. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Modo inmediato
Esta propiedad le permite elegir el modo inmediato para reproducir desde un dispositivo PVR o TV. Cuando se desactiva, & mplayer; capturará audio desde el mismo dispositivo como vídeo. Cuando se activa, le indica a & mplayer; que capture el audio a través del cable que va desde la tarjeta de TV hasta la tarjeta de sonido. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Captura
Esta propiedad le permite elegir el sistema de captura de audio, ya sea ALSA u OSS, para reproducir desde el dispositivo PVR o TV. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Dispositivo de captura
Esta propiedad le permite especificar el dispositivo de captura de audio para reproducir desde el dispositivo PVR or TV. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Sección Subtítulos
Esta sección contiene las propiedades que controlan la cargan y representación de los subtítulos, si el archivo se va a reproducir con subtítulos.
Esta propiedad le permite elegir los subtítulos que se utilizará cuando se reproduce este archivo o emisión. La opción ninguna desactiva los subtítulos. La opción externo le indica a & mplayer; que muestre los subtítulos cargados desde un archivo externo dado por la propiedad Ruta externa. La opción ajustar ID le permite introducir un ID de pista de subtítulo interna para reproducir. Las demás opciones permite elegir una pista de subtítulo interno de la lista de pistas detectadas. La forma más sencilla es establecer esta propiedad seleccionando una pista o archivo de subtítulo o la opción Ninguna del submenú Subtítulos del menú Reproductor.
Ruta externa
Esta propiedad especifica la ruta al archivo de subtítulos externo para utilizarlo con este archivo. Esta propiedad se establece automáticamente cuando utiliza las órdenes Cargar subtítulos del menú Archivo. La orden también establece el valor de la Pista a Externa.
Tipo externo
Esta propiedad le permite especificar el tipo de subtítulos externos. La opción auto le indica a & kplayer; que determine el tipo basándose en la extensión del archivo de subtítulos.
Codificación
Esta propiedad le permite especificar la codificación del texto en el archivo externo de subtítulos. La opción predeterminada le indicará a & kplayer; que utilice la codificación especificada en las Preferencias de & kplayer;.
Esta propiedad le permite especificar la tasa de fotogramas del archivo externo de subtítulos. La opción auto utiliza la tasa de fotogramas del archivo de vídeo.
Cargar automáticamente
Si es un archivo local, esta propiedad le indica a & kplayer; que intente cargar automáticamente los subtítulos. La configuración predeterminada utiliza la propiedad Cargar automáticamente de la página Subtítulos en las Preferencias de & kplayer;.
Posición
Esta propiedad le permite establecer la posición vertical de los subtítulos. La configuración predeterminada utilizará la misma posición que para los demás archivos. Una forma sencilla de establecer esta propiedad es utilizar los accesos rápidos de la orden que posiciona los subtítulos en el submenú Subtítulos del menú Reproductor mientras mantiene pulsada la tecla & Shift;.
Esta propiedad controla el retardo de los subtítulos respecto al vídeo. De forma predeterminada esta propiedad recordará automáticamente cuándo utiliza las órdenes Aumentar retraso y Reducir retraso en el submenú Subtítulos del menú Reproductor. La configuración predeterminada siempre reiniciará el retraso de los subtítulos a cero cuando se carga el archivo.
Leyendas
Esta propiedad le permite elegir si mostrar o no los subtítulos de las leyendas en un dispositivo o disco específico o para cada título individual en un DVD. Esta propiedad no es válida para los archivos o emisiones remotas.
Sección Avanzadas
Esta sección contiene varias propiedades que afectan a los parámetros avanzados de la línea de órdenes que se pasarán a & mplayer;, así como la interacción con el esclavo E/ S de & kde; si se utiliza para transferir el archivo.
Parámetros adicionales de la línea de comandos
Esta propiedad le permite especificar los argumentos adicionales que se pasarán a & mplayer;. Puede añadir los argumentos bien en la página Avanzadas de las Preferencias de & kplayer; o utilizar ajustar a para sobreescribirlos. La opción predeterminada utiliza los argumentos proporcionados en las Preferencias de & kplayer;.
Demultiplexor
Esta propiedad especifica los codificadores de vídeo que se utilizarán cuando se reproduzca este archivo. La opción predeterminada utilizará los codificadores elegidos en la página Vídeo de las Preferencias de & kplayer;, normalmente Automático. Automático le indicará a & mplayer; que elija el codificador que piense que es el mejor para reproducir este archivo.
Salto de fotogramas
Esta propiedad permite indicarle a & mplayer; los fotogramas que ha de saltarse si su sistema es demasiado lento para reproducir el archivo correctamente. La opción suave se salta menos fotogramas que la opción duro, y debería probarla primero. La opción predeterminado utiliza la configuración de Saltar fotograma de la página Avanzadas de las Preferencias de & kplayer;. También puede cambiar esta opción mientras reproduce un archivo utilizando las órdenes del del menú Avanzada del menú Reproductor. Si mantiene pulsada la tecla & Shift;, & kplayer; utilizará de de forma predeterminada la configuración solo para el archivo actual, y recordará esta propiedad.
Caché
Esta propiedad le permite elegir si utilizará la memoria caché para este archivo, pudiendo establecer su tamaño. La configuración predeterminada utiliza las opciones de la memoria caché de la página Avanzadas en las Preferencias de & kplayer;, auto le indica a & mplayer; que utilice su propia configuración, ninguna le indica a & mplayer; que no utilice memoria caché, y ajustar tamaño le permite especificar un tamaño.
Construir nuevo índice
Esta propiedad le permite elegir si & mplayer; creará un índice nuevo para el desplazamiento. El valor predeterminado utiliza la opción de las Preferencias de & kplayer;, Sí creará un índice si el archivo no lo tenía, No le indica a & mplayer; que no cree un índice, y Forzar indica que se cree siempre un índice.
Usar KIOSlave
Esta propiedad le permite elegir si se utilizará un esclavo E/ S de & kde; para transferir y reproducir este archivo. La opción auto le indicará a & kplayer; que decida automáticamente utilizar las opciones establecidas en las Preferencias de & kplayer;.
Usar archivo temporal
Esta propiedad le permite elegir si se utilizará un archivo temporal para reproducir desde un esclavo E/ S de & kde;. La configuración predeterminada utiliza las opciones de las Preferencias de & kplayer;. Esta propiedad no tendrá efecto para las URL que se pasen directamente a & mplayer;.
Los dispositivos PVR y TV tienen algunas propiedades específicas que no son aplicables a los demás elementos.
Compresión por hardware
Esta propiedad le permite decidir si se utilizará la comprensión por hardware MJPEG para un dispositivo PVR o TV y le permite especificar el tamaño de la imagen. Para los demás elementos esta propiedad no es aplicable y no se muestra.
Calidad de compresión
Esta propiedad le permite especificar la calidad de compresión entre 0 y 100 cuando se activa la compresión por software para un dispositivo PVR o TV. Para los demás elementos esta propiedad no es aplicable y no se muestra.
