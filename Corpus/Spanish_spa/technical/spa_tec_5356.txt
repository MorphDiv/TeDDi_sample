Introducción
KVpnc es una interfaz de KDE para varios clientes vpn. Soporta Cisco VPN (vpnc), IPSec (FreeS/ WAN (OpenS/ WAN), racoon), PPTP (pptpclient) y OpenVPN.

Importar certificado (si fuera necesario X.509 y si es PKCS12 cuando termina en. p12).
Configuración de una conexión nueva (pasarela,...)

Descarga
Obténgalo de la sección de archivos (http: / /download. gna. org/ kvpnc/).
CVS
El último código desarrollado puede obtenerse del cvs. Solo debe hacer lo siguiente:
cd kvpnc
make
make
su
make install
exit
Requisitos
Para utilizar & kvpnc; correctamente necesitará una versión de & kde; superior a la 3.1.
Las siguientes herramientas son opcionales pero para un uso normal se requiere un mí­nimo de ellas.
vpnc > = 0.2-rm+zomb-pre9 (se recomienda encarecidamente la versión 0.3. x) - Cisco VPN
FreeS/ WAN (también OpenS/ WAN) - IPSec VPN Linux 2.4/ 2.6
racoon - (ipsec-tools) > = 1. x IPSec VPN Linux 2.6
pptpclient - PPTP VPN
OpenVPN. - VPN basado en SSL

Para compilar kvpnc haga:
tar xvfj kvpnc-lt; versióngt;. tar. bz2
cd kvpnc-0.5
make
Configuración
Inicie kvpnc desde KDE - Internet - kvpnc. Cambie las configuraciones globales en el diálogo de Preferencias. Pulse en "Perfil nuevo... Â" para añadir un perfil nuevo. Introduzca un nombre nuevo en el diálogo que aparece, rellene los campos vacíos y guarde el perfil pulsando en "Guardar perfil...". Después introduzca sus datos VPN, pulse en "conectar" para conectarse a su servidor VPN. De forma predeterminada, kvpnc se minimiza en la bandeja del sistema después de que la conexión se haya realizado correctamente.
