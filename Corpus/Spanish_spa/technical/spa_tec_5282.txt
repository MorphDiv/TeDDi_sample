Las opciones de menú
El menú Archivo
Ctrl; N Archivo Nuevo proyecto
Crea un nuevo proyecto. Debe elegir el tipo de proyecto (CD de audio, DVD de datos,...).
Ctrl; O Archivo Abrir...
Abre un proyecto existente que puede seleccionarse con el diálogo Abrir archivo de KDE.
Ctrl; R Archivo Abrir reciente
Esto es un acceso rápido a re-abrir proyectos recientemente abiertos. El sub-menú al que pertenece este elemento contiene una lista de esos proyectos, al pulsar sobre un archivo específico se abrirá de nuevo.
Ctrl; S Archivo Guardar
Guarda el proyecto actual. Si ya se había guardado una copia del documento entonces se sobreescribirá el archivo previamente guardado sin pedir autorización al usuario. Si es la primera vez que se guarda un nuevo documento, se accederá al diálogo Guardar como.
Archivo Guardar como...
Guarda el proyecto actual con un nuevo nombre de archivo. Se muestra el dialogo Guardar como para especificar el nombre del directorio del nuevo archivo de proyecto.
Archivo Guardar todo
Guarda todos los proyectos abiertos. Es lo mismo que seleccionar Archivo Guardar para cada uno de ellos.
Ctrl; C Archivo Cerrar
Cierra el proyecto actual. Si se ha modificado un proyecto pero no ha sido guardado todavía entonces & k3b; le preguntará que hacer. Puede elegir entre guardar o descartar los cambios, y también tendrá la oportunidad de cancelar el cierre y mantener el proyecto abierto.
Archivo Cerrar todo
Cierra todos los proyectos abiertos. Para cada proyecto sin guardar & k3b; le preguntará que hacer, igual que cuando elige Archivo Cerrar.
Ctrl; Q Archivo Salir
Salir de & k3b; después de cerrar todos los proyectos abiertos. Para cada proyecto sin guardar & k3b; le preguntará que hacer, igual que cuando elige Archivo Cerrar.
El menú Proyecto
Proyecto Añadir archivos...
Cuando se selecciona este elemento, el diálogo que se muestra le permite elegir uno o mas archivos a añadir al proyecto. Esto tiene el mismo efecto que arrastrar archivos directamente desde la vista de contenidos a la vista de proyectos. Si el proyecto actual es un proyecto de disco de datos, los archivos añadidos aparecerán directamente en el directorio raíz del disco.
Proyecto Limpiar proyecto
Elimina todos los archivos y directorios del proyecto actual. El proyecto permanece abierto.
Ctrl; B Proyecto Grabar...
Abre el diálogo de grabación para el proyecto actual. Este es de echo el mismo que el diálogo de propiedades - la única diferencia en que hay un botón adicional de grabación que hace que & k3b; grabe un disco a partir de los datos del proyecto. El diálogo de grabación no se abrirá si el proyecto no contiene archivos.
Ctrl; P Proyecto Propiedades...
Abre el diálogo de propiedades para el proyecto actual. Aquí puede especificar un montón de opciones relativas al proyecto. Cada tipo de proyecto tiene un conjunto distinto de opciones, la mayor parte de las cuales se explican usando Que es esto (accesible pulsando el botón derecho del ratón).
Proyecto Importar sesión
Este elemento solo aparece si un proyecto de CD/ DVD de datos o DVD de vídeo está activo. Hará que K3b Importe las entradas del archivo de una sesión anterior en la actual. Puede usarlo cuando recopile archivos para discos multisesión. Por lo tanto los datos de la sesión anterior se incluyen siempre, incluso si no elige hacer uso de esta orden. Solo ayuda a saber que hay ya en el disco.
Proyecto Limpiar sesión importada
Este elemento solo aparece cuando un proyecto de CD/ DVD de datos o de DVD de vídeo está activo. Provoca que K3b elimine las entradas de archivos del proyecto actual que hayan sido importadas mediante Proyecto Importar sesión mientras que mantiene el resto de los datos del proyecto. Por lo tanto el uso de esta orden no elimina realmente nada del disco que se va a grabar, solo oculta de nuevo los archivos.
Proyecto Editar imágenes de arranque
Este elemento solo aparece cuando un proyecto de CD/ DVD de datos o de DVD de vídeo está activo. Aquí puede especificar imágenes de arranque para crear CD o DVD auto arrancables. Se pude copiar una imagen de arranque directamente de un disquete o disco duro (por ejemplo creada con la orden dd) así como de otra imagen de arranque. En cualquier caso se trata de un único archivo que contiene un sistema completo y arrancable que se graba como un archivo normal. Para que el ordenador sepa que el disco contiene una imagen de arranque, la aplicación de grabación crea un catálogo de arranque cuyo nombre puede determinarse en la ventana del diálogo.
El menú Herramientas
Herramientas Copiar CD...
Abre el diálogo de copia de CD. Proporciona la posibilidad de copiar el contenido de un CD a otro disco sin necesitar abrir un archivo de proyecto, Alternativamente, puede elegir crear una imagen del CD original que puede ser grabada en cualquier momento. También existe una opción de clonar el CD en lugar de realizar una copia normal, lo cual será mejor cuando se copien CD con sectores defectuosos o CD de vídeo.
Herramientas Copiar DVD...
Abre el diálogo de copia de DVD. No se necesita un proyecto de DVD. Proporciona la posibilidad de copiar los contenidos de un DVD a otro disco. Además, puede elegir crear sólo una imagen de un DVD que puede grabarse en cualquier momento. La transcodificación de vídeo con el diálogo de copia de DVD no es posible todavía, por lo que el disco de destino debe ser suficientemente grande para contener todos los datos (video) del disco origen en su formato original.
Herramientas Borrar CD-RW...
Abre el diálogo de borrado de CD-RW. Esto le ayuda a borrar los contenidos de un CD-RW, o una parte de ellos.
Herramientas Formatear DVD±RW...
Abre el diálogo de formateo de un DVD. Esto le ayuda a dar formato a un DVD-R(W) o un DVD+R(W), lo que provocará que los contenidos del disco se borren. & k3b; da la opción entre los modos « Sobre escritura » e « Incremental ».
Herramientas Grabar imagen de CD...
Graba una imagen de CD creada con anterioridad. El diálogo Grabar imagen de CD pide seleccionar un archivo *. iso, *. cue o *. toc como fuente de datos que usted puede grabar inmediatamente pulsando el botón Iniciar. (En este momento no se admiten archivos Nero *. nrg, por lo que usted tendrá que emplear otras herramientas como Nrg2Iso.)
Herramientas Grabar imagen ISO de DVD...
Graba una imagen de DVD creada con anterioridad. El diálogo Grabar imagen Iso9660 pide seleccionar un archivo *. iso como origen de datos, el cual puede grabar inmediatamente pulsando el botón Iniciar.
Herramientas Codificar vídeo...
Puede codificarse un DVD de video que haya sido previamente extraído con alguna ayuda del diálogo Codificar video. Esta caja de diálogo se abre habitualmente después de extraer el DVD desde la vista de contenido, pero también puede usarse en solitario. Contiene información sobre el DVD de vídeo extraído, opciones de codificación e incluso la posibilidad de redimensionar y recortar vídeos.
Herramientas Información del disco
Muestra información sobre el disco insertado. Esta información se muestra en la vista de contenidos y abarca propiedades como tipo, tamaño y longitud de las pistas o el CD o DVD que se encuentra en su unidad.
El menú Configuración
Preferencias Barra de herramientas
Muestra una lista de barras de herramientas de & k3b;. Si una entrada en una barra de herramientas está marcada, significa que la barra de herramientas es visible.
Preferencias Mostrar/ ocultar barra de estado
Esto le permite mostrar u ocultar la pequeña barra de la parte inferior de la ventana principal que contiene información sobre el estado y actividad de & k3b;.
Preferencias Mostrar directorios
Alterna la visibilidad de la vista de directorio. Esto le permite seleccionar directorios y unidades de disco. Cuando está seleccionada, sus contenidos se mostrarán en la vista de contenidos, las unidades de disco también tienen un menú contextual que proporciona funciones tales como información del disco, (des)montar la unidad o expulsar el medio.
Preferencias Mostrar contenidos
Alterna la visibilidad de la vista de contenidos. Esta vista le permite seleccionar archivos que pueden arrastrarse a la vista de proyectos para añadirlos al proyecto. La vista de contenidos también actúa como una interfaz para extraer CD de audio y discos de vídeo cuando se selecciona en la vista de directorio una unidad de discos que contiene un CD/ DVD adecuado.
Preferencias Mostrar cabecera del documento
Alterna la visibilidad de la cabecera del documento que pertenece a la vista del proyecto. Se trata de una pequeña barra que solo es visible si la vista de proyectos contiene algún proyecto abierto. La cabecera del documento no tiene funcionalidad, pero es agradable y aumenta la claridad mediante la separación de la vista de proyectos de otras vistas.
Preferencias Configurar accesos rápidos...
Esta orden abre una caja de diálogo en la que pueden cambiarse las asociaciones de teclas para las órdenes de menús de & k3b;. Después de seleccionar una de las órdenes disponibles en la parte superior del diálogo, el acceso rápido para esta acción se puede cambiar en la parte inferior.
Preferencias Configurar barras de herramientas...
Esta orden abre una caja de diálogo en la cual pueden personalizarse las barras de herramientas. El cuadro desplegable en la parte superior del diálogo determina que barra de herramientas puede editarse en cada momento. La lista de opciones disponibles situada a la izquierda contiene las órdenes que pueden añadirse a la barra de herramientas, la lista de acciones actuales de la derecha muestra las que ya se han añadido. Los elementos pueden añadirse seleccionando la orden adecuada de la lista de acciones disponibles y pulsando el botón derecho par moverlas a la lista de acciones actuales. Para eliminar un elemento se sigue el camino opuesto. Los botones arriba y abajo cambian la posición de las órdenes en la barra de herramientas.
Preferencias Configurar & k3b;
Esto abre & k3bsetup; que ayuda a configurar los permisos correctos que necesita & k3b; para grabar CD y DVD. La gestión de derechos de usuario de Linux permite la ejecución de programas y el acceso a unidades de disco si no se han concedido permisos por el administrador. & k3bsetup; no pude configurar permisos sin privilegios de administrador, por lo cual tiene que proporcionar la contraseña de root cuando arranque.
Preferencias Configurar & k3b;...
Abre el diálogo de opciones en el cual pueden configurarse las opciones generales. Aunque la mayor parte de las funciones de & k3b; deberían funcionar recién instaladas, este diálogo le permite personalizar y ajustar el programa.
