Estas funciones permiten acceder a bases de datos Oracle8 y Oracle7.
Estas usan la Oracle8 Call-Interface (OCI8).
Necesitará las librerías clientes de Oracle8 para usar esta extensión.
Esta extensión es más flexible que las estándar de Oracle.
Soporta el enlace de variables locales y globales de PHP con placeholders de Oracle, tiene soporte completo para LOB, FILE y ROWID y le permiten usar las variables definidas por el usuario.
