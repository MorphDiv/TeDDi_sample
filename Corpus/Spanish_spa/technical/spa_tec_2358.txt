Opciones de KWord
Esta sección de la documentación le mostrará cómo configurar KWord para que se adapte a su estilo de trabajo personal y a sus preferencias.
KWord se puede modificar de tres formas distintas:
Configurar atajos
Configurar barras de herramientas
Opciones diversas
Configurar atajos
Esto le mostrará un cuadro de diálogo.
Para editar el atajo de teclado, la primera cosa que necesita hacer es buscar la acción que quiere editar.
Todas las acciones posibles se listan en el cuadro etiquetado Acción.
Usando la barra de desplazamiento, localice la acción en la que esté interesado y luego haga clic una vez con el left mouse button.
Una vez que haya seleccionado la acción, debe dirigir su atención a la mitad inferior del cuadro de diálogo.
Use la mitad inferior del cuadro de diálogo para cambiar el atajo de teclado.
Ninguno
Al selecciona esta opción, la acción seleccionada actualmente no tendrá ningún atajo de teclado.
Predeterminado
Al seleccionar esta opción, la acción seleccionada actualmente usará el atajo de teclado predeterminado.
Una vez que haga clic sobre esta opción, se listará debajo el atajo de teclado predeterminado.
Personalizado
Al seleccionar esta opción, puede elegir cualquier combinación de teclas como atajo de teclado para esta acción.
Definiendo atajos de teclado personalizados
Determine la combinación de teclas haciendo clic sobre el botón con la combinación de teclado actual.
Aparecerá un segundo diálogo:
Este cuadro de diálogo le permite determinar dos combinaciones de teclas diferentes que dispararán la acción deseada:
Principal y Alternativo.
Definiendo atajos de teclados sencillos
Comience por decidr si quiere espeificar el atajo Principal o Alternativo, poniendo una marca en el botón de radio apropiado.
El botón limpiará el atajo de teclado actual.
Ahora teclee el atajo en el teclado.
KWord usará esta combinación de teclas como el nuevo atajo de teclado.
La ventana se cerrará automáticamente.
Para evitar que la ventana se cierre automáticamente, simplemente borre la marca del cuadro de verificación llamado Cerrar automáticament.
Una vez que haya completado sus entradas, simplemente haga clic sobre Aceptar para aceptar los cambios o Cancelar para cancelarlos.
Definiendo atajos de teclado multitecla
Los atajos de teclado multitecla se pueden usar cuando ya no le quedan atajos de teclado sencillos para asignar.
Los atajos de teclado multitecla se editan de la misma forma que los atajos de teclado sencillos, salvo que se pone una marca en la casilla de verificación etiquetada Multitecla antes de introducir el atajo de teclado.
Ahora puede introducir múltiples caracteres para el atajo de teclado.
Esta es una combinación multitecla.
Una vez que haya completado sus entradas, simplemente haga clic sobre Aceptar para aceptar los cambios o Cancelar para cancelarlos.
Configurar las barras de herramientas
Esto le llevará a un cuadro de diálogo.
Puede añadir o borrar tantos botones de la barra de herramientas como quiera.
También puede mover los botones por la barra de herramientas usando este diálogo.
Añadiendo un botón a una barra de herramientas
Para añadir un botón desde una barra de herramientas, muévalo desde el cuadro de Aciones disponibles hasta la columna Acciones actuales.
Seleccione primero la barra de herramientas a la que quiera añadir un botón seleccionando la barra de herramientas de la lista desplegable llamada Barra de herramientas
Ahora seleccione la acción que quiera añadir del cuadro de selección etiquetado Acciones disponibles haciendo clic una vez con el botón izquierdo del ratón.
Clic sobre la flecha azul que apunta a la derecha.
Una vez que el botón Aceptar o el botón Aplicar se han pulsado, las barras de herramientas cambian.
Borrando un botón de una barra de herramientas
Para borrar un botón de una barra de herramientas, muévalo del cuadro de Acciones actuales a la columna de Acciones disponibles.
Seleccione primero la barra de herramientas de la que quiera borrar un botón seleccionando la barra de herramientas de la lista desplegable llamada Barra de herramientas
Seleccione el botón de la barra de herramientas que quiera borrar de la lista de selección múltiple llamada Acciones actuales.
Clic sobre la flecha azul que apunta a la izquierda.
Una vez que el botón Aceptar o el botón Aplicar se han pulsado, las barras de herramientas cambian.
Moviendo un botón en una barra de herramientas
Primero seleccione la barra de herramientas en la que quiera mover un botón seleccionando la barra de herramientas de la lista desplegable llamada Barra de herramientas
Para mover un botón, simplemente haga clic sobre el botón que quiere mover con el botón izquierdo del ratón.
Clic sobre las flechas hacia arriba o abajo para mover el botón arriba o abajo en la barra de herramientas respectivamente.
Una vez que el botón Aceptar o el botón Aplicar se han pulsado, las barras de herramientas cambian.
Opciones de KWord
Esto le mostrará un cuadro de diálogo.
Configurar el interfaz de usuario de KWord.
Hacer clic sobre Interfaz se le permitirá cambiar lo siguiente.
Mostrar la barra de estado
Poner una marca en este cuadro de verificación hace que la barra de estado sea visible en KWord.
Mostrar barra de desplazamiento
Poner una marca en este cuadro de verificación hace visible la barra de desplazamiento en KWord.
RePág / AvPág mueve el signo de intercalación
Si este cuadro de verificación tiene una marca, cuando pulse los botones AvPág y RePág del teclado, KWord moverá el cursor de texto (el signo de intercalación) abajo una página.
Si no hay marca en este cuadro de verificación, KWord mueve la vista, pero no cambia la posición del cursor de texto.
Número de archivos recientes:
Puede ajustar este valor entre 1 y 20.
Tamaño de la cuadrícula horizontal:
Determina el tamaño horizontal de la cuadrícula.
Cuando los marcos y las tabulaciones se ponen en la página, se ubican en un punto de la cuadrícula.
Esto le da a su documento un aspecto más profesional, porque los elementos se alinean efectivamente.
Puede reducir el tamaño de la cuadrícula si necesita un control más fino de su distribución.
Tamaño de la cuadrícula vertical:
Determina el tamaño vertical de la cuadrícula.
Cuando los marcos y las tabulaciones se ponen en la página, se ubican en un punto de la cuadrícula.
Esto le da a su documento un aspecto más profesional, porque los elementos se alinean efectivamente.
Puede reducir el tamaño de la cuadrícula si necesita un control más fino de su distribución.
Sangrado de párrafo con los botones de la barra de herramientas:
Use este elemento para determinar la lejanía del sangrado de párrafos al usar el botón de incrementar () y el de reducir sangrado ().
Modo de vista preliminar - Número de páginas por fila:
Determina el número de páginas por fila en modo de vista preliminar.
Esto determina el tamaño de las páginas en modo de vista preliminar.
Cuando esté contento con los cambios, simplemente haga clic sobre Aceptar.
Si hace clic sobre Cancelar, todos los cambios se perderán.
Haciendo clic en el botón Predeterminados, se restauran todos los valores a sus valores predeterminados.
Configurar las opciones del documento
Predeterminados del documento
Espaciado de columnas predeterminado
Use esto para ajustar el espaciado predeterminado entre columnas.
Para más información sobre columnas, vea Columnas.
Tipografía predeterminada
Clic sobre el botón Seleccionar, y aparecerá un nuevo diálogo.
Use este diálogo para elegir la tipografía predeterminada.
Esta preferencia determina la tipografía predeterminada usada por KWord hasta que se formatee más adelante.
Preferencias del documento
Guardar automáticamente cada (min):
Puede usar esto para ajustar la frecuencia en la que KWord guarda un archivo temporal.
Si fija este valor a cero, KWord no guardará automáticamente.
Puede ajustar el guardado automático de cero a 60 minutos.
Número de página inicial
Use este cuadro de texto para determinar el número de página inicial.
Para más información sobre numeración de páginas, vea Numeración de páginas.
Esto es útil si tiene que dividir un documento en múltiples archivos.
Tabulación:
Cada documento de KWord tiene un juego de tabulaciones predeterminadas.
Si añade tabulaciones a su documento, las tabulaciones nuevas añadidas se superponen a las predeterminadas.
Puede usar este cuadro de texto para definir el espaciado entre las tabulaciones.
Por ejemplo, si introducir 1.5 en este cuadro de texto, y la unidad de medida es el centímetro, la primera tabulación predeterminada estará ubicada 1,5 cm a la derecha del margen izquierdo del marco.
La segunda tabulación predeterminada se localizará a 3 cm desde el margen izquierdo, etc..
Preferencias del cursor
Esta sección del cuadro de diálogo tiene un cuadro de verificación llamado Cursor en área protegida.
Cuando hay una marca en este cuadro de verificación y hace clic en un marco protegido de su documento, aparece un cursor.
Cuando la marca se borra de este cuadro de verificación y hace clic en un marco protegido, no es visible el cursor.
Cuando esté contento con los cambios, simplemente haga clic sobre Aceptar.
Si hace clic sobre Cancelar, todos los cambios se perderán.
Haciendo clic en el botón Predeterminados, se restauran todos los valores a sus valores predeterminados.
Configurar ortografía.
Para configurar las opciones para el revisor ortográfico, haga clic en el botón llamado Ortografía.
Cada opción se detalla debajo.
Crear combinaciones de raíz / afijo que no estén en el diccionario
Si está marcado, cuando KWord encuentra una palabra raíz en combinación con bien un prefijo o un sufijo formando una palabra combinada, KWord añadirá la palabra combinada como una entrada nueva a su diccionario.
Si no está marcado, KWord continuará informando de esta palabra como incorrecta.
Considerar palabras juntas como errores ortográficos
Si este cuadro está marcado, y KWord encuentra dos palabras del diccionario una junto a la otra y sin separación, las marcará como incorrectas.
Si no está marcado, las palabras combinadas se ignorarán.
Ejemplos de tales palabras son shutout, cannot, y blackout (Nota del traductor: estar formas no son demasiado frecuentes en el castellano).
Dejar este cuadro de verificación sin marcar ayudará a prevenir que KWord considere las direcciones web y de correo electrónico como errores ortográficos.
Estas direcciones suelen contener palabras juntas.
Diccionario:
La lista desplegable se usa para seleccionar el diccionario que quiera que use KWord para todos los diccionarios disponibles.
Codificación:
La lista desplegable se puede usar para especificar qué juego de caracteres quiere que use el corrector.
Cliente:
La lista desplegable se usa para especificar el programa de verificación ortográfica que debe usar KWord de una lista de programas disponibles.
Ignorar palabras en mayúsculas.
Si está marcado, KWord no marcará PGE como incorrecto, pero marcará así Pge y pge.
Esto es útil cuando su documento contiene una gran cantidad de acrónimos.
Ignorar las mayúsculas en el título.
Si está marcado, KWord no marcará Pge como incorrecto, pero sí marcará PGE y pge.
Mostrar las palabras incorrectas en el documento.
Si está marcado, KWord subrayará todas las palabras incorrectas en rojo mientras escribe.
Para eliminar este comportamiento, borre la marca de este cuadro de verificación.
Limpiar todas las palabras de la historia de ignorar...
Si está seleccionado, KWord borrará todas las palabras que hayan sido previamente marcadas como el botón Ignorar todo.
Cuando esté contento con los cambios, simplemente haga clic sobre Aceptar.
Si hace clic sobre Cancelar, todos los cambios se perderán.
Haciendo clic en el botón Predeterminados, se restauran todos los valores a sus valores predeterminados.

Configurar opciones varias
Varios
Unidades
Seleccione su unidad de medida preferida.
KWord usará estas unidades para todas las medidas.
Vea Usando las reglas para más información.
Límite de deshacer / rehacer
Use este elemento o desplazador para determinar cuántas acciones mantiene KWord en su búfer de deshacer.
Cualquier acción que supere este número se olvidará.
Mostrar enlaces
Poner una marca en este cuadro de verificación hace que los enlaces a documentos sean visibles en KWord.
Si no hay marca en este cuadro de selección, los enlaces a documentos se ocultarán.
Vea Enlaces a documentos para más información.
Subrayar todos los enlaces
Poner una marca en este cuadro de verificación hará que KWord subraye automáticamente un enlace al documento cuando se cree.
Vea Enlaces a documentos para más información.
Mostrar comentarios
Poner una marca en este cuadro de verificación hace que los comentarios de documentos sean visibles en KWord.
Vea Comentarios del documento para más información.
Mostrar código de campo
Si hay una marca en este cuadro de verificación, KWord muestra el nombre de las variables del documento, en vez del contenido de la variable en la pantalla de KWord.
Si este cuadro de verificación no tiene una marca, KWord muestra los contenidos de la variable.
Esta opción no afecta a la salida impresa.
Los contenidos de las variables se imprimen cualquiera que sea el estado de este cuadro de verificación.
Ver formato
Puede usar estos cuatro cuadros de verificación para determintar qué caracteres de formato se muestran cuando le pida a KWord que muestre los caracteres de formato.
Ver formato de fin de párrafo
Visualiza u oculta los caracteres de nueva línea.
Ver formato de espacio7
Visualiza u oculta los espacios individuales.
Ver formato de tabulaciones
Visualiza u oculta las tabulaciones.
Ver formato de salto
Visualiza u oculta las marcas de párrafo.
