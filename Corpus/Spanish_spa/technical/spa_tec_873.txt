Menús contextuales de tablas
El menú contextual del contenedor de tablas ofrece diversas funciones que hacen referencia a todas las tablas de la base de datos.
Para editar una tabla específica dentro de una base de datos, seleccione la tabla correspondiente y abra su menú contextual.
Relaciones
para la definición de relaciones entre diversas tablas sólo está disponible en bases de datos relacionales.
Según el sistema de base de datos utilizado, los menús contextuales contendrán las entradas siguientes:
tabla abierta
, diversas funciones estarán disponibles para editar los datos.
