¿Dónde está la página web de KDE?
La página web oficial de KDE está en http: / /www.kde.org.
Si prefiere un mirror local, hay varios para escoger.
Para ver una lista actualizada de mirrors según su ubicación, por favor vaya a http: / /www.kde.org / webmirrors.html.
¿Hay una lista de correo de KDE?
En realidad, hay más de una.
Cada una se enfoca en un determinado aspecto de KDE.
Algunas de ellas son para desarrolladores, por lo que no se discutirá en detalle sobre ellas aquí.
Algunas de las listas más importantes en que los usuarios pueden interesarse son:
kde
Esta es la principal lista de correo de KDE, para discusiones en general.
kde-announce
Esta lista se utiliza para anunciar nuevas versiones de KDE, así como nuevas herramientas y aplicaciones.
kde-look
Esta lista está relacionada con cuestiones sobre aspecto y comportamiento de la interfaz y consideraciones generales sobre la interfaz de usuario.
Para la lista completa de foros disponibles, por favor consulte http: / /www.kde.org / maillinglists.html.
Por favor observe que no es una buena idea hacer preguntas que ya han sido contestadas en este FAQ.
¿Cómo puedo suscribirme a estas listas o cancelar mi suscripción?
Envíe un mensaje a una de las siguientes direcciones según la lista a la que desee suscribirse:
kde-request@kde.org
kde-announce-request@kde.org
kde-user-request@kde.org
kde-look-request@kde.org
Envie un mensaje a una de las siguientes direcciones según lalista en la que quiera cancelar su suscripción:
kde-request@kde.org
kde-announce-request@kde.org
kde-user-request@kde.org
kde-look-request@kde.org
¡Nunca envíe una petición para suscribirse o cancelar una suscripción directamente a las listas de correo!
Utilice sólo las direcciones mencionadas anteriormente.
Existe también una lista completa de todas las listas de correo relacionadas con KDE y una interfaz web para suscribirse y cancelar suscripciones, en http: / /master.kde.org / mailman / listinfo.
¿Existe un archivo de las listas de correo?
Sí, existe un archivo de las distintas listas de correo que es mantenido por Progressive ComputerConcepts.
Su dirección es http: / /lists.kde.org.
Aquí se puede ver un listado por directorios de las listas de correo.
Observe que en este nivel sólo se pueden hacer búsquedas de artículos según su tema y / o autor.
Usted probablemente no encuentre lo que desea mediante este tipo de búsqueda.
Para buscar según el contenido de los mensajes, tendrá que entrar en una de las listas de correo.
Sólo haga click en la carpeta en la que desea buscar (e.g., kde o kde-user) lo que hará que el menú que permite escoger el tipo de búsqueda contenga una ítem para búsquedas según el contenido del mensaje Body).
Para una búsqueda más minuciosa debería buscar en las carpetas kde, kde-user, y kde-devel.
