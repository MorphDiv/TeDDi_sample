Uso: hunspell [OPCIÓN]... [ARCHIVO]...
Verifica ortografía de cada ARCHIVO. Sin ARCHIVO, verifica la entrada estándar.
-1 comprueba sólo el primer campo en cada línea (delimitador = tabulador)
-a interfaz de la tubería de Ispell
-d d[,d2,...] usar los diccionarios d (d2 etc.)
-D mostrar diccionarios disponibles
-G imprime solamente palabras o líneas correctas
-h, --help muestra esta ayuda y sale
-F Formato de entrada de archivo HTML
-i enc entrada codificada
-l mostrar palabras mal deletreadas
-L mostrar líneas con palabras mal deletreadas
-m analizar las palabras del texto entrante
-n formato de archivo de entrada nroff/troff
-p dic configurar el diccionario personalizado dict
-r avisa de los posibles errores (palabras raras)
-P password establece la contraseña para diccionarios cifrados
-s buscar las raíces de las palabras de entrada
-t formato de archivo de entrada TeX/LaTeX
-v, --versión imprime el número de la versión
-vv imprime el número de la versión compatible con Ispell
