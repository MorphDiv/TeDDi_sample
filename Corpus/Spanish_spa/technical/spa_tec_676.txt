El Administrador de extensiones agrega, quita, habilita, inhabilita y actualiza extensiones de %PRODUCTNAME.
A continuación se incluyen ejemplos de extensiones de %PRODUCTNAME:
Componentes UNO (módulos de software compilados)
Datos de configuración (para comandos de menú)
Bibliotecas de %PRODUCTNAME Basic
Bibliotecas de diálogos de %PRODUCTNAME
Archivos de extensiones (archivos *.oxt que contengan una o más extensiones)
Alcance de la extensión
Usuarios con privilegios de administrador verán un diálogo donde podrán elegir instalar extensiones &quot; para todos los usuarios &quot; o &quot; solo para mí &quot; .
Usuarios normales sin estos privilegios pueden instalar, quitar, o modificar extensiones únicamente para su propio uso.
Un usuario como administrador o con privilegios de administrador puede instalar una extensión para ser compartida y disponible entre todos los usuarios.
Después de seleccionar una extensión, un diálogo se muestra y le solicita el lugar donde se instalará para el usuario actual o todos los usuarios.
Un usuario sin privilegios de administrador solamente podrá instalar extensiones para su propio uso.
Esto se conoce como una extensión de usuario

Herramientas- Administrador de Extensiones
y presione
Agregar
.
Para instalar una extensión compartida en modo de texto.
Como administrador, abra una terminal o línea de comandos.
\
/
.
Escriba los siguientes comandos, utilizando la ruta y el nombre de la extension del documento:
unopkg agregar --directorio compartido del archivo.oxt
Seleccione la extensión que desea quitar, habilitar, o deshabilitar.
Para algunas extensiones también podra abrir el diálogo Opciones.
Agregar
Presione clic en Agregar para agregar una extensión.
Se abre un cuadro de diálogo de archivo en el que puede seleccionar las extensiones que desee agregar.
Para copiar y registrar las extensiones seleccionadas, haga clic en Abrir.
Lea la licencia.
Presione el botón de desplazamiento para descender lo necesario.
Haga clic en Aceptar para continuar la instalación de la extensión.
Quitar
Seleccione la extension que desee eliminar y haga clic en Quitar.
Habilita o deshabilita una extensión.
Habilitar
Seleccione la extensión que desee habilitar y haga clic en Habilitar.
Desactivar
Seleccione la extensión que desee desactivar y haga clic en desactivar.
Actualización
Haga clic para comprobar las actualizaciones en línea de todas las extensiones instaladas.
Para comprobar sólo las actualizaciones de las extensiones seleccionadas, elija el comando Actualizar del menú contextual.
La verificación de actualizaciones se inicia de inmediato.
Usted verá el diálogo
Actualización de extensiones
.
Opciones
Seleccione una extensión instalada, luego haga clic para abrir el diálogo de opciones de dicha extensión.
