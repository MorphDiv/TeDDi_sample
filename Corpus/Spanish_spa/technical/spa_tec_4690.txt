El cuadro de diálogo de configuración de & kspread;
Seleccionando Preferencias Configurar & kspread;... se abre un cuadro de diálogo con algunas páginas, seleccionados con los iconos en el cuadro de diálogo de la izquierda, que le permite cambiar muchos aspectos de la operación de & kspread;.
Opciones locales
Opciones de configuración locales.
Esta página del diálogo de configuración de & kspread; muestra cómo se visualizan elementos como números, fecha, hora y dinero.
Si ha cargado una hoja de cálculo que se generó usando un locale diferente, pulsando el botón Actualizar opciones locales en esta página se actualizará para que se amolde a las preferencias locales que están definidas en & kcontrolcenter;.
Interfaz
Configuración de la interfaz.
Esta página del diálogo de configuración de & kspread; (obtenida seleccionando Configurar & kspread;... desde el menú Preferencias) controla algunas características más de & kspread;:
Número de hojas abiertas al comenzar:
Controla cuántas hojas de trabajo se crearán si la opción Hoja de cálculo vacía se selecciona al iniciar & kspread;.
Número de archivos que se muestran en lista de recientes:
Controla el número máximo de nombres de archivos que se muestran cuando selecciona Archivo Abrir reciente.
Intervalo para guardar automáticamente (minutos):
Aquí puede seleccionar el tiempo entre guardados automáticos, o desactivar esta característica completamente si selecciona No guardar automáticamente (arrastre la barra deslizable totalmente hacia la izquierda)
Crear archivos de respaldo
Marque esta casilla si quiere crear algunos archivos de respaldo. De forma predefinida está marcada.
Mostrar barra de desplazamiento vertical
Marque o desmarque este cuadro para mostrar u ocultar la barra de desplazamiento vertical en todas las hojas.
Mostrar barra de desplazamiento horizontal
Marque o desmarque este cuadro para mostrar u ocultar la barra de desplazamiento horizontal en todas las hojas.
Mostrar encabezado de columna
Marque esta caja para mostrar las letras de la columna encima de una hoja de trabajo.
Mostrar encabezado de fila
Marque esta caja para mostrar los números de fila al lado izquierdo.
Mostrar solapas
Esta casilla de verificación controla si se muestran las solapas al final de la hoja de trabajo.
Mostrar barra de herramientas de fórmulas
Aquí dice si quiere mostrar u ocultar la barra Fórmula.
Mostrar barra de estado
Desmarque esta casilla si quiere ocultar la barra de estado.
Misc
Configuraciones varias.
La página Misc del cuadro de diálogo de configuración de & kspread; 's contiene los siguiente elementos;
Modo de terminación:
Le permite elegir el modo de terminación (automático) de texto de entre varias opciones en la lista desplegable.
Al presionar intro se mueve el cursor de celda:
Cuando haya seleccionado una celda pulse la tecla Enter y la selección se moverá Abajo, Arriba ,a la Derecha, Izquierdao Abajo, primera columna, como determina la opción de esta lista desplegable.
Método de cálculo:
La selección de esta lista desplegable se usa para elegir el cálculo realizado por la función Resumen de la barra de estado.
Paso de sangrado (cm):
Le permite definir la cantidad de sangrado usada por la opción Aumentar sangrado del menú Formato.
La unidad que utiliza aquí es la que está definida en el diálogo Configurar & kspread;, en la solapa Diseño de página, en la opción Unidades de página predefinidas:.
Mostrar mensaje de error para fórmulas no válidas
Si esta caja está marcada, se mostrará un cuadro de mensaje de error cuando haya introducido ua celda que & kspread; no entienda.
Color
Configuración del color.
Esta página del cuadro de diálogo de configuración de & kspread; le permite elegir el color de la cuadrícula de la hoja. Si no quiere que aparezca ninguna cuadrícula, quite la marca de la casilla Mostrar cuadrícula en el diálogo Formato Hoja Propiedades de la hoja.
Esta página también le permite seleccionar el color de las líneas usadas para indicar los bordes de la página impresa cuando se marca el cuadro Bordes de página en el menú Ver.
Clic en el color actual para mostrar el diálogo Seleccionar color, estándar de KDE.
Color de cuadrícula:
Clic aquí para cambiar el color de la cuadrícula, & ie; el color de los bordes de cada celda.
Bordes de página:
Cuando está marcada la opción de menú Ver Mostrar bordes de página, se muestran los bordes de la página. Clic aquí para elegir otro color para los bordes distinto del rojo predefinido.
Diseño de página
Configuración del diseño de página.
Esta página del diálogo de configuración de & kspread; le permite configurar el tamaño de página predeterminado, la orientación y las unidades usadas por la impresora y por el cuadro de diálogo Disposición de página (obtenido al seleccionar Disposición de página... desde el menú de Formato).
Tamaño predefinido de página:
Elija el tamaño de página predefinido para la hoja de trabajo, de entre los tamaños de página más habituales. Tenga en cuenta que puede sobreescribir el tamaño de página para la hoja actual usando el diálogo Formato Diseño de página...
Orientación predefinida de página:
Elige la orientación predefinida de la hoja: retrato o apaisado. Tenga en cuenta que puede sobreescribir la orientación para la hoja actual usando el diálogo Formato Diseño de página....
Unidad predefinida de página:
Elija la unidad predefinida que se debe usar en todas las hojas. Tenga en cuenta que pueda sobreescribir la unidad para la hoja actual usando el diálogo Formato Diseño de página....
Ortografía
Configuración de ortografía.
Esta página le permite configurar el comportamiento del verificador ortográfico de & kspread;.
Crear combinaciones de raíz/ afijo que no estén en el diccionario
Si este cuadro está marcado, cuando & kspread; encuentra una palabra en el documento que no se reconoce pero que consta de una palabra raíz reconocida más un refijo o sufijo reconocido, se aceptará, mientras que si el cuadro no está marcado, el verificador ortográfico lo rechazará.
Considerar las palabras que-van-juntas como errores ortográficos
Si esta marca está puesta las palabras comunes que se escriben juntas se considerarán errores ortográficos. (N. del traductor: en castellano no es frecuente esta construcción; en inglés, podríamos encontrar ejemplos como cannot).
Diccionario:
Esta lista desplegable se puede usar para seleccionar diccionarios alternativos.
Codificación:
Para selecionar la codificación de los caracteres que se va a usar.
Cliente:
Esta lista desplegable le permite seleccionar entre los diferentes programas para revisar ortografía que pueden estar presentes en su computador.
Evitar todas las palabras en mayúscula
Marque este casilla si quiere que el verificador ortográfico ignore las palabras en mayúsculas, que suelen ser acrónimos como & kde;.
No comprobar las mayúsculas iniciales del título
Marque esta casilla si quiere que el verificador ortográfico ignore la mayúscula inicial de las palabras del título, por ejemplo Mi Propia Hoja De Cálculo o Mi propia hoja de cálculo. Si no está marcada, el verificador ortográfico pide una letra en mayúscula en los nombres del título.
Configurar opciones de texto a voz
Consulte la sección Texto a voz en el capítulo sobre accesibilidad para obtener más detalles.
