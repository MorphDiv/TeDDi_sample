Navegador de datos
Especifica la estructura de datos del documento XForms actual.
Nombre de modelo
Selecciona el modelo XForms que se desea usar.
Modelos
Agrega, cambia el nombre y quita modelos XForms.
Agregar
Abre el cuadro de diálogo Agregar modelo, que permite agregar un modelo XForm.
Escriba el nombre.
Actualizaciones de cambio de modelo de datos de las modificaciones del estado del documento
Cuando está activada, el estado del documento establecerá la condición de &quot; modificado &quot; al cambiar cualquier forma de control que este obligado en el modelo de datos.
Cuando no está activado, este cambio no establece el documento la condición de &quot; modificado.
Quitar
Borra el modelo XForm seleccionado.
No es posible borrar el último modelo.
Cambiar nombre
Cambia el nombre del modelo XForm seleccionado.
Mostrar detalles
Muestra u oculta los detalles.
Instancia
Muestra una lista de los elementos que pertenecen a la instancia actual.
Envíos
Muestra una lista de los envíos.
Enlaces
Muestra una lista para los enlaces de XForm.
Instancias
Este botón tiene submenús para agregar, editar o eliminar instancias.
Agregar
Abre un cuadro de diálogo en el que se puede agregar una instancia.
Editar
Abre un cuadro de diálogo en el que puede modificar la instancia actual.
Quitar
Borra la instancia actual.
No es posible borrar la última instancia.
Mostrar tipos de datos
Configura la visualización para mostrar una cantidad de detalles especificada.
Agregar
Abre un cuadro de diálogo para agregar un componente (elemento, atributo, envío o enlace) como subcomponente del componente actual.
Editar
Abre un cuadro de diálogo que permite editar el componente seleccionado (elemento, atributo, envío o enlace).
Eliminar
Elimina el componente seleccionado (elemento, atributo, envío o enlace).
