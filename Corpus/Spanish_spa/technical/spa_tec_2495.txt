Diccionario visual de términos de KDE
Este proyecto se inició para ayudar a unificar el vocabulario utilizado por la documentación de KDE.
Le proporcionamos este diccionario aquí como una ayuda para aprender a manejarse a través del interfaz gráfico de usuario de KDE.
Estos nombres se han normalizado en la documentación de KDE, de modo que siempre encuentre la misma denominación para cada uno de los elementos de interfaz existentes.
El término Widget (objeto visual) se refiere al bloque básico de construcción de un interfaz de un programa de ordenador.
Cada widget se comporta de un modo similar cada vez que se utiliza, pero el efecto que ese objeto visual produce, depende de su lugar en el programa.
