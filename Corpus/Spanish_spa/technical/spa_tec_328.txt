Categoría de error
Categoría de Error
, se puede escoger diferentes formas de mostrar la categoría de error.
Ningún
No mostrar barras de error.
Valor constante
Muestra valores constantes que Usted puede especificar en el área de parámetros.
Porcentaje
Muestra un porcentaje.
Se muestra los datos correspondientes al punto elegido.
Defina el porcentaje usando el botón de selección.
Funciones
Seleccione una función para calcular the las barras de error.
Error Estándar:
Muestra el error estándar.
Varianza:
Muestra la varianza calculada desde el número de puntos de datos y los respectivos valores.
Desviación Estándar:
Muestra la desviación estándar (raíz cuadrada de la varianza).
Margen de Error:
Muestra el margen de error máximo expresado en porcentaje de acuerdo al valor máximo del grupo de datos.
Configura el porcentaje en el área de Parámetros.
Rango de celdas
Presione rango de Celdas y luego especifique un rango de celdas desde donde tomar los valores de las barras de error positivas y negativas.
Haga clic en un botón para reducir el tamaño del cuadro de diálogo, a continuación, utilice el ratón para seleccionar el rango de celdas en la hoja de cálculo.
Haga clic en el botón otra vez para restablecer el diálogo a su tamaño completo.
Desde Tabla de Datos
Para un diagrama con sus propios datos, los valores de la barra de error pueden ingresarse en la tabla los valores de la barra de error pueden ser ingresados en la tabla de datos del diagrama.
El dialogo de la tabla de datos muestra columnas adicionales tituladas Barras de Error Y Positivas y Barras de Error Y negativas.
Positive (+)
Ingrese el valor a agregar a los valores visualizados como el valor de error positivo
Negative (-)
Ingrese el valor a restar del valor visualizado como el valor de error negativo.
Positive (+) when Cell Range is selected
Ingrese el rango de direcciones desde donde recibir los valores de error positivos.
Use el botón central de su ratón (si lo tiene) para seleccionar el rango desde una hoja de cálculo
Negative (-) when Cell Range is selected
Ingrese el rango de direcciones desde donde recibir los valores de error negativo.
Use el botón central de su ratón (si lo tiene) para seleccionar el rango desde una hoja.
Igual valor para ambos
Habilita a usar tanto los valores de error positivos como los valores de error negativosa.
Usted solamente puede cambiar el valor del cuadro &quot; Positivo (+).
El valor recibido será copiado al cuadro &quot; Negativo (-) &quot; automaticamente.
