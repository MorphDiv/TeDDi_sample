Usando etiquetas inteligentes
Etiquetas inteligentes proveen información y funcionalidad para especificar palabras en un documento de Writer.
Las caracteristicas disponibles pueden ser diferentes para diferentes extensiones de etiquetas inteligentes.

Este se requiere un navegador internet correctamente configurado.
Menú de etiquetas inteligentes
Herramientas - Opciones - %PRODUCTNAME - Representación
.
Cuando quieres resaltar una etiqueta inteligente, un tip de ayuda te informa a oprimir Ctrl-click para abrir un menú de etiquetas inteligentes.
Si no usas un ratón, posicionar el cursor dentro de texto marcado y abre el menu contextual oprimiendo Shift+F10.
