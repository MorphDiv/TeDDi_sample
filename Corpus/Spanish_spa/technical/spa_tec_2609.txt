Estilo
Introducción
Este módulo se usa para configurar el modo en el que KDE dibuja los widgets individuales.
Un Widget (objeto visual) es un término de uso común entre los programadores para referirse a elementos del interfaz de usuario tales como botones, menús, y barras de desplazamiento.
Puede pensar en ellos como las piezas fundamentales que se ensamblan para hacer su aplicación.
Con este módulo puede configurar como se dibujan los widgets, pero para cambiar el color de los widgets, consulte la sección titulada Colores.
Este panel está dividio en tres secciones.
Estilos de widgets y temas
La lista superior, contiene una lista de estilos predefinidos.
Cada estilo tiene un nombre, y una breve descripción.
Para cambiar los estilos, simplemente pulse sobre el nombre del estilo, y pulse Aplicar para ver el aspecto de los nuevos widgets.
Otras configuraciones para dibujar:
Esta pequeña sección sólo tiene una opción.
Barra de menú en la parte superior de la pantalla al estilo de MacOS
Esto activará una barra de menú en la parte superior de la pantalla.
Esta barra de menñu reflejará las opciones de menú de la aplicación activa.
