Parámetros de inicio de Java
Puede utilizar este diálogo para especificar parámetros de inicio opcionales para el entorno de ejecución de Java (JRE).
Las opciones que especifique en este diálogo serán válidas para cualquier JRE que inicie.
Parámetro de inicio de Java
Escriba un parámetro de inicio para un entorno JRE, como lo haría en una línea de comandos.
Haga clic en Asignar para agregar el parámetro a la lista de parámetros de inicio disponibles.
Los nombres de ruta no admiten caracteres de escape ni comillas.
Por ejemplo, para que la propiedad del sistema &quot; myprop &quot; apunte a una carpeta, escriba el parámetro siguiente:
-Dmyprop=c:\Archivos de programa\java
Para habilitar la depuración en JRE, escriba los parámetros siguientes:
-Xdebug
-Xrunjdwp:transport=dt_socket,server=y,address=8000
Estos cambios surten efecto al reiniciar %PRODUCTNAME.
Parámetros de inicio asignados
Quitar
.
Asignar
Agrega el parámetro de inicio de JRE actual a la lista.
Quitar
Borra el parámetro de inicio de JRE seleccionado.
