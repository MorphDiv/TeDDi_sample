Botón
Icono
Añade un botón de comando.
Los botones de comando se pueden usar para ejecutar un comando para una acción definida, como una pulsación del ratón.
Si lo desea, puede agregar texto o un gráfico al botón.

Icono

Casilla de verificación
Icono
Añade una casilla de verificación que puede usarse para activar o desactivar una función.
Campo de opción
Icono
Añade un botón que permite al usuario seleccionar entre varias opciones.
Los botones de opción agrupados deben tener índices de tabulación consecutivos.
Normalmente están rodeados por un cuadro de grupo.
Si se dispone de dos grupos de botones de opción, debe insertarse un índice de tabulación entre los índices de los dos grupos en el marco del grupo.
Etiqueta
Icono
Añade un campo para mostrar etiquetas.
Estas etiquetas son sólo para mostrar texto predefinido y no para introducir texto.

Icono

Listado
Icono

Cuadro combinado
Icono
Añade un cuadro combinado.
Un cuadro combinado es un listado de una línea en el que el usuario puede pulsar y elegir una entrada de la lista.
Si se desea, se pueden convertir las entradas del cuadro combinado como &quot; de sólo lectura &quot; .

Icono
Añade un marco que puede usarse para agrupar visualmente campos de control similares, como botones de opción.
Para definir dos grupos distintos de botones de opción, asegúrese que el índice de tabulación del marco de grupo esté entre los de los dos grupos.

Campo enmascarado
Icono
Agrega un campo enmascarado.
Un campo enmascarado consta de una máscara de entrada y de una máscara literal.
La máscara de entrada determina los datos de usuario que se pueden entrar.
La máscara literal define el estado del campo enmascarado al cargar el formulario.
