Usar el editor
Esto debería ir en la sección del editor.
Globales
Muestra las llamadas & DCOP; global y setGlobal para proporcionarle variables globales para el guión
& DCOP;
Muestra cómo usar las llamadas & DCOP;, locales y externas, para comunicarse con aplicaciones externas.
Slots
Muestra cómo usar conexiones y slots para gestionar eventos. Se usan tanto los slots de población como los estándares.
Preferencias
Le muestra cómo usar las funciones @readSetting y @writeSetting para escribir y restaurar el contenido del widget. También le muestra cómo usar el slot populate() para inicializar el contenido del widget.
Añadir
Le muestra cómo puede añadir texto a TextEdit y cómo puede usarlo para mostrar texto con formato.
Línea de órdenes
Muestra cómo pasar parámetros al cuadro de diálogo de & kommander; mediante la línea de órdenes. También muestra cómo cambiar contenido de listas y texto de botones.
Inicializar
Muestra cómo usar la « inicialización » para « destruir » guiones del cuadro de diálogo principal para inicializar y almacenar algunas opciones.
