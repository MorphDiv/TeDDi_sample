Estilos de texto
Introducción a estilos de texto
Cuando construye documentos que son más complejos que una carta de negocios, como un libro, carta de noticias o un resumen, el documento se divide a menudo en secciones.
Cada una de estas secciones pueden tener una apariencia similar.
Puede usar estilos para mantener fácilmente un aspecto consistente para todo su documento.
Si está familiarizado con los estilos de Microsoft Word, los estilos de texto de KWord realizan las mismas funciones.
Puede pasar la siguiente sección.
Como ejemplo, si miramos la sección de un documento, y asignamos estilos de texto diferentes a las partes apropiadas:
En este ejemplo, puede ver varios estilos de texto diferentes para el trabajo:
El título está centrado, subrayado y en negrita
El título de la sección (Income) está en negrita
El título para la figura 1-1 es más pequeño que el texto normal y está en negrita.
El resto del documento está en una tipografía estándar.
No necesitamos saber cómo queremos exactamente que aparezcan el texto y los párrafos de estas secciones.
Todo lo que necesitamos hacer es identificar estas secciones de texto como Títulos de sección, Texto normal, etc..
Una vez que haya finalizado el documento, puede cambiar la apariencia de todo el texto etiquetado como Título de la sección de una vez.
Esto le asegurará que todos los títulos de la sección serán consistentes en su documento.
KWord tiene 8 estilos de texto predefinidos.
Los estilos de texto no se deben confundir con los estilos de tabla ni estilos de marcos.
Los estilos de texto determinan cómo aparecen los caracteres y los párrafos.
Los estilos de marcos controlan los bordes y el color de fondo del marco.
Los estilos de tablas determinan qué aspecto tienen las tablas en el doumento terminado.
Cambiando el estilo de texto del texto
Para cambiar el estilo de texto del texto, primero, seleccione el texto al que quiera aplicar los cambios.
Ahora puede cambiar el estilo del texto de una de estas dos formas:
Esto abrirá la lista de estilos de texto disponibles.
Selecione el estilo de la lista.
Puede seleccionar el estilo de texto usando la lista desplegable de la barra de herramientas de formateo de caracteres.
Esta lista desplegable tiene el siguiente aspecto:.
Simplemente, seleccione el estilo de texto nuevo de la lista indicada.
Creando un estilo de texto
Si planea usar ampliamente los estilos de texto en un documento grande, probablemente querrá crear nuevos estilos específicos para sus necesidades.
Hay dos formas principales para crear un estilo de texto nuevo:
Crear un estilo de texto basado en otro estilo de texto usando un diálogo.
Formatear un bloque de texto al formato deseado, creando luego un estilo de texto basado en ese texto.
Creando un nuevo estilo de texto basado en un estilo de texto actual
Esto le mostrará un cuadro de diálogo
En la lista de la izquierda hay una lista de todos los estilos de texto definidos actualmente.
Seleccione el estilo actual que se parezca más al estilo nuevo.
Haga clic sobre ese estilo una vez con el left mouse button.
Clic sobre el botón Nuevo.
KWord creará un estilo de texto nuevo basado en el estilo seleccionado.
Le asignará un nombre temporal [Plantilla de estilo nuevo (8)].
Teclee un nombre descriptivo en la caja llamada Nombre.
Este será el nombre del estilo de texto.
El resto del diálogo se usa para hacer los cambios necesarios para crear el nuevo estilo de texto.
La caja de vista preliminar le mostrará la apariencia de su nuevo estilo de texto.
Use los botones denominados Tipografía, Sangrado y espaciado, Alineación >, Bordes, Viñetas / Numeración, Tabuladores y Sombra para crear el estilo de texto.
Clic sobre Aceptar para crear su estilo de texto nuevo.
Clic sobre Cancelar para detener la creación del estilo de texto.
Cambiando el tamaño de la tipografía, el tipo, etc.
La pestaña llamada Tipografía se usa para fijar el tipo de tipografía, el estilo, el formato, etc..
Las funciones de esta pestaña son idénticas a las del diálogo de cambio de tipografía usado para editar el texto general.
Los cambios en esta pestaña afectan a todo el texto formateado con este estilo de texto.
Cambiando el espaciado y el sangrado de los párrafos
La pestaña llamada Sangrado y espaciado se usa para ajustar el espaciado entre líneas, sangrado de párrafos y el espaciado entre párrafos.
Las funciones de esta pestaña son idénticas a las del diálogo de cambio de tipografía usado para editar el texto general.
Los cambios en esta pestaña afectan a todo el texto formateado con este estilo de texto.
Cambiando la alineación del párrafo
La pestaña llamada Alineación determina cómo se ubica el texto en la línea.
Con otras aplicaciones, puede haberse referido a esto como Alineación o Justificación.
Las funciones de esta pestaña son idénticas a las del diálogo de Formatear párrafo.
Los cambios en esta pestaña afectan a todo el texto formateado con este estilo de texto.
Cambiando los párrafos del borde
La pestaña llamada Bordes se usa para definir y configurar los bordes gráficos alrededor de sus párrafos.
Las funciones de esta pestaña son idénticas a las del diálogo de Formatear párrafo.
Los cambios en esta pestaña afectan a todo el texto formateado con este estilo de texto.
Cambiando la numeración / viñetas del párrafo
La pestaña llamada Viñetas / Numeración se usa para hacer que todo el texto se formatee con este estilo de texto en una lista.
Esta pestaña funciona de forma idéntica a la misma pestaña del diálogo Formatear párrafo.
Para más información, vea la sección titulada Listas.
Cambiando las tabulaciones.
Usando la pestaña llamada Tabuladores, es posible definir los tabuladores para el nuevo estilo de texto.
Esta pestaña funciona de forma idéntica a la misma pestaña del diálogo Formatear párrafo.
Para más información, vea la sección titulada Tabulaciones.
Cambiando la sombra del estilo de texto.
Usando la pestaña llamada Shadow, es posible cambiar el sombreado artificial del texto en el estilo de texto.
Esta pestaña funciona de forma idéntica a la misma pestaña del diálogo Formato de párrafo.
Creando un estilo de texto basado en un texto con formato
KWord le pedirá un nombre para su estilo de texto.
Introduzca el nombre en el cuadro de texto y haga clic sobre Aceptar.
Se crea un estilo de texto nuevo con la tipografía, espaciado de párrafo, alineación de párrafo, bordes o sombras del texto actualmente seleccionado.
Los futuros cambios de formato a este texto seleccionado no cambiará automáticamente el estilo de texto que acaba de crear.
Si quiere que los cambios sean parte del estilo de texto, debe editar el estilo de texto.
Borrando un estilo de texto
Borrar un estilo de texto no necesario es fácil.
Esto hará surgir un cuadro de diálogo.
De la lista de estilos de texto disponibles, seleccione el estilo que quiera borrar haciendo clic una vez con el left mouse button.
Asegúrese de que ha seleccionado el estilo de texto correcto antes de hacer clic sobre Borrar.
KWord no le pedirá confirmación, así que no tendrá la posibilidad de volver atrás.
Clic sobre Borrar.
El estilo de texto se ha borrado.
Editando un estilo de texto
El verdadero poder de los estilos de texto está en la capacidad para editar las opciones de formato de ese estilo de texto después de que el estilo de texto esté definido.
Por cambiar el formato del estilo de texto, KWord cambiará inmediatamente cada párrafo con ese estilo de texto, y mantiene un aspecto consistente en el documento.
KWord usa los mismos interfaces para editar las opciones de formato de un estilo de texto, que se usa para crear el estilo de texto en primer lugar.
Para editar un estilo de texto actual:
Esto hará surgir un cuadro de diálogo.
Seleccione el estilo de texto que quiere editar de la lista de la izquierda haciendo clic con el left mouse button.
Ahora puede hacer los cambios que quiera a este estilo de texto.
La caja de vista preliminar le mostrará la apariencia de su nuevo estilo de texto.
No cambie el nombre de su plantilla.
Haciendo esto, borrará su estilo de texto actual de la lista (y crea uno nuevo con el mismo nombre).
Cualquier párrafo que se formateara originalmente con este estilo de texto volverá al Estándar.
Use los botones denominados Tipografía, Sangrado y espaciado, Alineación >, Bordes, Viñetas / Numeración, Tabuladores y Sombra para alterar la apariencia del estilo de texto.
Clic en Aceptar para realizar con éxito sus cambios.
Clic sobre Cancelar para abortar todos los cambios a este estilo de texto.
Importar un estilo de texto
KWord tiene la capacidad de importar un estilo de texto de un documento de KWord e incluirlo en la lista de estilos de texto de otro documento de KWord.
Esto hará surgir un cuadro de diálogo vacío.
Clic sobre el botón Cargar.
Esto le llevará al diálogo de selección de archivo.
Seleccione el archivo de KWord desde el que quiera importar el estilo de texto y haga clic sobre Aceptar.
El cuadro de diálogo se llenará ahora con todos los estilos de texto disponibles para la importación.
Si KWord encuentra un nombre de estilo de texto duplicado en el archivo seleccionado, añadirá un número al final del nombre del estilo de texto para identificar el estilo importado.
Por ejemplo, si importa el estilo de texto Estándar desde otro archivo de KWord, KWord cambiará el nombre del estilo de texto a Estándar-1.
Seleccione todos los estilos de texto que quiera importar.
Luego haga clic sobre Aceptar.
