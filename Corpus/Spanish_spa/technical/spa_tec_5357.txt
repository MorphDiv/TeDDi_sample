& kalcul; es una aplicación matemática para niños pequeños (a partir de 6 años) para hacer pruebas de sumas, restas, multiplicaciones y divisiones. También proporciona ejercicios mezclados. Se pueden realizar dos tipos de ejercicios con & kalcul;: operaciones en las que puede configurar el tipo de operación y el rango y el programa le hará preguntas aleatorias sobre los números, y niveles preseleccionados (puede proporcionar pruebas o configurar sus propios niveles a través de Herramientas -gt; Editor de niveles).

Introducción
& kalcul; es una aplicación matemática para niños pequeños (a partir de 6 años) para hacer pruebas de sumas, restas, multiplicaciones y divisiones. Se pueden realizar dos tipos de ejercicios con & kalcul;: operaciones en las que puede configurar el tipo de operación y el rango y el programa le hará preguntas aleatorias sobre los números, y niveles preseleccionados en los que establecerá las preguntas a realizar.
& kalcul; pretende ser simple de utilizar para los niños pero suficientemente potente con muchas opciones y configuraciones para los adultos (padres o profesores). & kalcul; también guarda el resultado de cualquier prueba en un archivo asociado al nombre de usuario actual y planeamos hacer que estos resultados estén disponibles para KSpread y KChart (o cualquier otra aplicación de oficina).
Uso de & kalcul;
La ventana principal de & kalcul; está formada por un área de texto en la parte izquierda y un cronómetro en la derecha. El cronómetro es un reloj que le permite averiguar la duración de una prueba o hacerla en una determinada cantidad de tiempo.
Veamos una captura de pantalla de la ventana principal de & kalcul;
Ventana principal de & kalcul;
Pulse el botón Inicio para comenzar la prueba. El reloj comienza a andar y se muestra una pregunta. Éste es el Modo de prácticas en el que las preguntas se seleccionan aleatoriamente. Algunos parámetros pueden seleccionarse en el menú Herramientas -gt; Diálogo de ejercicios. Los valores predeterminados son: operación de suma con dos términos y cada término inferior a 10.

Iniciar una prueba
Si pulsa el botón Detener la prueba se parará y se reiniciarán los valores para iniciar una prueba nueva.
Si pulsa el botón Pausa, se detendrá la prueba, la pregunta continuará mostrándose, y se parará el cronómetro. Pulse el botón Continuar para continuar con la prueba.
Se muestran dos barras de progreso que le indicarán el número de preguntas que ha respondido y cuántas de ellas son correctas.

Herramientas
Este capítulo explica las dos principales herramientas de & kalcul;: el diálogo de ejercicio y el editor de niveles.
El diálogo de ejercicio
El diálogo de ejercicio le permite seleccionar qué tipo de ejercicio desea que haga el niño/ alumno.
Cuando seleccione el menú Herramientas -gt; Diálogo ejercicio, verá primero este diálogo:
El diálogo de ejercicio
El diálogo de ejercicio
« Práctica » es un prueba cronometrada con preguntas aleatorias de un tipo especificado.
« Nivel » es una prueba cronometrada con un nivel que puede seleccionar.
Cuando seleccione « Práctica », se mostrará la siguiente pantalla:
Seleccionar el tipo
Seleccionar el tipo
Pulse en el tipo deseado para su próxima prueba. Puede seleccionar el número máximo para cada término de la operación así como otros parámetros en Preferencias (***Ver si he traducido esto igual en los mensajes del programa) -gt; Configurar & kalcul;...
Si selecciona nivel en la primera pantalla del diálogo ejercicio, obtendrá el siguiente diálogo que lista todos los archivos. kalcul presentes en su sistema.
Seleccionar el nivel
Seleccionar el nivel
Seleccionar el nivel. Las características del nivel se muestran en la parte derecha. Pulsando en ¡Comenzar! (***¿He traducido eso igual en los mensajes?) se cargará el nivel seleccionado.
El editor de niveles
El editor de niveles le permitirá escribir sus propias pruebas. Estas pruebas se guardarán en archivos. kalcul.
Editor de niveles
Editor de niveles
En el campo « Descripción », describa las características del nivel, por ejemplo: Grado 3 - Fácil para 6 años - Medio o número de clase 102 - Sumar hasta 10.
El área « Opciones » incluye dos cuadros desplegables. Configure el nivel de dificultad a Fácil, Medio, Difícil o Experto, y configure el tipo de operación para el nivel: suma, resta, multiplicación, división o mezcla. Mezcla es una mezcla de estas operaciones, por ejemplo: 2 + 4 - 5.
El área Preguntas es el lugar en el que se editan las preguntas. Las preguntas son las que se realizarán al niño. Introduzca la operación en Pregunta y el resultado de la operación en Resultado.
Pulse el botón Añadir pregunta y se incluirá una línea nueva. Una doble pulsación en cada campo (Pregunta y Resultado) le permitirá teclear su pregunta y su respuesta. Haga una doble pulsación para editar el campo pregunta e introducir la operación: 2+5 ó 5-3, por ejemplo. No importa si añade espacios. Entrar validará el campo. Una doble pulsación en el campo Resultado le permitirá introducir el resultado a la pregunta. Entrar validará sus datos.
Pulsar el botón Borrar pregunta borrará la línea seleccionada.
En la parte inferior del diálogo, existen cuatro botones. Nuevo crea un nuevo nivel. No guarda el nivel actual y vacía todos los campos. Abrir le permite abrir un archivo de nivel (con la extensión. kalcul) y editarlo. Guardar guarda la prueba actual. Abre un diálogo de archivos y le permite guardar el nivel con la extensión. kalcul. Cerrar cierra el diálogo sin guardar los datos.
Preferencias
Este capítulo explica el diálogo Preferencias -gt; Configurar & kalcul;...
La pestaña General
La pestaña General
La pestaña General
Aquí podrá configurar el nombre del usuario. Este nombre se utilizará para guardar los resultados de todas las pruebas para este usuario.
Aquí también podrá configurar el número de preguntas para cada prueba.
La pestaña Cronómetro
La pestaña Cronómetro
La pestaña Cronómetro
Si marca Utilizar un cronómetro, la prueba se cronometrará; esto es, se detendrá después de que el tiempo haya finalizado. El tiempo irá disminuyendo.
Configure el tiempo para el cronómetro utilizando el cursor de abajo.
La pestaña Avanzadas
La pestaña Avanzadas
La pestaña Avanzadas
Puede seleccionar el máximo que desee para el resultado. Por ejemplo, un niño de 6 años estará comenzando a aprender a sumar hasta 5, luego hasta 6, después hasta 7 y así sucesivamente. Configurar este valor a N asegurará que las operaciones aleatorias no tengan un resultado mayor de N. Poner este valor a 0 implicará que no existe límite.
Configurar el máximo para el primer término a N le asegurará que el primer término sea inferior a N. Por ejemplo, si asigna el valor 10, el primer término será un número entre 0 y 10. Lo mismo se puede aplicar al segundo término.
Guía para profesores/ padres de & kalcul;
Estadísticas
El resultado para un niño (usuario) se guarda en el archivo $KDEHOME/ share/ apps/ kalcul/ nombre_usuario. txt.
Las futuras características le permitirán visualizar estos resultados con una aplicación KOffice, KSpread o KChart.
