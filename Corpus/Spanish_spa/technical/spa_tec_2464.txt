Sugerencias y trucos para el trabajo diario.
Cualquiera puede hacer una tortilla con huevos.
El truco es hacerla sin ninguno.
Prácticamente todo el mundo puede utilizar KDE esta es la razón para la que los desarrolladores lo crearon.
No existen selecciones crípticas, y pocos archivos de configuración requieren ser editados en ASCII.
Sin embargo, hay otros modos en los se puede hacer el trabajo de modo más elegante, reservando tiempo para las tareas realmente importantes, como el KSirtet.

Circular por los escritorios

Mini línea de comandos

Menú de ventana
Alt F4
Cerrar la ventana actual
Ctrl F{ 1..8}
Cambiar al escritorio especificado
Ctrl Alt Esc
Destructor de ventanas (todas las ventanas sobre las que pulse serán destruidas).
Pulse esta combinación de teclas para cancelar el cursor asesino.
Ctrl Alt Retroceso
Salir de KDE (sin guardar los cambios).
Utilice esta opción sólo como último recurso.
Ctrl Alt Teclado numérico +
Selecciona la siguiente resolución de pantalla.
Es necesario haber configurado el servidor X para que funcione.
Ctrl Alt Teclado numérico -
Selecciona la resolución de pantalla previa.
Es necesario haber configurado el servidor X para que funcione.
