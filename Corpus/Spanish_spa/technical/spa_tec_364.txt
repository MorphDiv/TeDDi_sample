Asistente de Gráfico - Elementos del Gráfico
Asistente para gráficos
puede escoger los elementos del gráfico que se van a mostrar.
titles title
Introduzca el título para su gráfico.
subtitle
Introduzca un subtítulo para su gráfico.
x axis
Introduce una etiqueta para el eje-x (horizontal).
y axisx axis
Introduce una etiqueta para el eje-y (vertical).
z axis
Introduce una etiqueta para el eje-z.
Esta posición está disponible únicamente para gráficos tridimensionales.
legend
Muestra una leyenda en su gráfico.
legend left
Posiciones de la leyenda a la izquierda del gráfico.
top
Posiciona la leyenda arriba del gráfico.
right
Posiciones de la leyenda a la derecha del gráfico.
bottom
Posiciona la leyenda al fondo del gráfico.
grid x axis
Muestra las líneas de las cuadrículas que están paralealas al eje-x.
y axis
Muestra la cuadricula que son paralelos al eje y.
z axis
Muestra las líneas de las cuadrículas que son paralelas al eje-z.
Esta opción está disponible únicamente para gráficos tridimensionales.
secondary x axis ( &quot; 2nd x axis &quot; in spec doc)
Introduzca una etiqueta para el eje X secundario.
Esta opción solo está disponible para gráficos que soportan un eje X secundario.
secondary y axis ( &quot; 2nd y axis &quot; in spec doc)
Introduzca una etiqueta para el eje Y secundario.
Esta opción sólo está disponible para gráficos que soportan un eje Y secundario.
Para ingresar los elementos del gráfico
Escriba los títulos o haga clic en los elementos que quiera que se muestren en el gráfico actual.
Títulos
Si escribe texto como título, subtítulo, o cualquier eje, se reservará el espacio necesario para mostrar el texto junto al gráfico.
Si no escribe ningún texto, no se reserva espacio, dejando mas espacio para mostrar el gráfico.
No es posible para ligar el titulo de texto a la celda.
Debes ingresar el texto directamente.
Formato
.
Leyenda
La leyenda muestra las etiquetas desde la primera fila o columna, o desde el rango fijado en el dialogo de Serie de Datos.
Si su gráfico no contiene etiquetas, la leyenda muestra texto como &quot; Fila 1, Fila 2, ... &quot; , o &quot; Columna A, Columna B, ... &quot; de acuerdo con la letra del número de filas o columnas de los datos del gráfico.
No puede escribir el texto directamente, se genera automáticamente desde el rango de celdas de Nombre.
Seleccione una de las opciones de posición.
Cuando el gráfico esté concluido, puede especificar otras posiciones usando el menú Formato.
Cuadrícula
grid x axis
Muestra las líneas de las cuadrículas que están paralealas al eje-x.
y axis
Muestra la cuadricula que son paralelos al eje y.
z axis
Muestra las líneas de las cuadrículas que son paralelas al eje-z.
Esta opción está disponible únicamente para gráficos tridimensionales.
La cuadrícula visible puede ayudar a estimar los valores de los datos en el gráfico.
La distancia entre la cuadricula corresponde al la configuracion del intervalo en la escala del axis y propiedades.
La cuadrícula no está disponible para los diagramas de círculo.
Elementos adicionales
Para elementos adicionales puede utilizar el menu Insertar del gráfico en modo edición.
Allí puede definir los siguientes elementos:
Eje secundario
Cuadrícula auxiliar
Etiquetas de datos
Estadísticas, por ejemplo valores promedios, barras de error Y y curvas de regresión
