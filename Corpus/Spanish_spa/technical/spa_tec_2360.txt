Almacenamiento e impresión del documento
Esta sección de la documentación cubrirá todo lo que necesite saber sobre iniciar un nuevo documento, guardar un documento, recuperar un documento guardado e imprimir un documento.
Iniciando un documento nuevo
Se puede comenzar un documento nuevo de 5 formas:
Puede usar el atajo de teclado:
Ctrl N
o hacer clic sobre en la barra de herramientas.
De cualquier forma, si comienza un documento nuevo aparecerá un cuadro de diálogo:
Este cuadro de diálogo le permite:
Comenzar un documento nuevo desde una plantilla
Abrir un documento existente
Abrir un documento reciente
Comenzar con un documento vacío. (esta es la predeterminada)
KWord recuerda su elección anterior.
Esa elección previa será la opción predeterminada actual.
Puede usar los siguientes atajos de teclado:
Si usa los atajos de teclado, necesitará pulsar Enter en el teclado o hacer clic sobre Aceptar para confirmar su selección.
Comenzando un documento nuevo desde una plantilla
Para comenzar un documento nuevo basado en una plantilla, debe primero seleccionar su plantilla.
Recuerde que las plantillas pueden ser orientadas a texto o a distribución de página.
Si necesita revisar las diferencias, haga clic aquí.
Puede usar las pestañas denominadas Orientada a texto y Distribución de página para mostrar todas las plantillas disponibles de ese grupo.
Una vez que tenga la pestaña orrecta, se le mostrarán todas las plantillas disponibles, cada una con un título y un icono pequeño que le muestra la distribución general de la plantilla.
KWord viene con dos grupos estándares de plantillas.
Puede añadir pestañas nuevas instalando plantillas externas o creando sus propias plantillas.
Para seleccionar su plantilla, haga clic sobre ella con el botón izquierdo.
La plantilla seleccionada se resaltará.
Confirme su elección haciendo clic sobre el botón Aceptar.
Esto comenzará un documento nuevo con esa plantilla.
Para un acceso más rápido a una plantilla, simplemente haga doble clic sobre la plantilla y KWord cargará inmediatamente la plantilla.
Abrir un documento existente
Clic sobre el botón Elegir y se mostrará un nuevo cuadro de diálogo.
Para más detalles sobre este cuadro de diálogo, revise Usando el diálogo de selección de archivos.
Abrir un documento reciente
KWord recuerda la mayoría de los documentos editados recientemente.
Puede seleccionar uno de estos documentos usando la lista desplegable a la derecha del botón de radio denominado Abrir un documento reciente.
Una vez que seleccione el nombre de archivo en esta lista desplegable, haga clic sobre Aceptar y KWord cargará la plantilla.
Cargando un documento vacío
Esto comienza un documento nuevo en modo orientado a texto, con un marco por página.
Esta es una elección excelente para la correspondencia diara, o para el procesamiento de textos general.
Simplemente haga clic sobre el botón Aceptar para comenzar.
Guardando un documento
Una vez que haya introducido texto y datos en un documento, normalmente querrá guardar esto en un archivo de su disco duro.
El comando Guardar
El comando Guardar se puede invocar de 3 formas:
Puede usar el atajo de teclado:
Ctrl S
o haciendo clic sobre en la barra de herramientas.
Cualquier de estos métodos hace que KWord realice la misma acción.
Seleccionando el comando Guardar le está indicando a KWord que guarde el archivo con el nombre de archivo actual.
No se le dará la opción de cambiar ni el nombre de archivo ni su ubicación.
Si no ha guardado este archivo antes, no tiene un nombre de archivo.
KWord ejecutará automáticamente el comando Guardar como... para que pueda proporcionar un nombre de archivo.
KWord no informa de un guardado con éxito.
Por tanto, si no hubo ningún incidente en el guardado, KWord le devolverá a la edición de su documento.
Puede verificar que el archivo se guardó mirando a la barra de título.
Si hay algún cambio no guardado, la barra de título contendrá [modificado].
Si el guardado tuvo éxito, sólo aparecerá en la barra de título el nombre del archivo.
Si hay algún problema con el guardado, aparecerá un cuadro de diálogo de error.
El comando Guardar como...
General
El comando Guardar como... se puede invocar de dos formas:
o intentando Guardar un archivo que aún no tiene un nombre de archivo.
La diferencia entre los comandos Guardar y Guardar como... es que el comando Guardar como... le pide un nombre para el archivo, y le deja seleccionar un formato de archivo diferente.
El comando Guardar simplemente guarda el archivo en la ubicación previa.
Usando el cuadro de diálogo
Cuando seleccione Guardar como... aparece un diálogo.
Para más información sobre usar este diálogo, vea la sección titulada Usando el diálogo de selección de archivo.
Cuando esté en la carpeta en la que quiera guardar el archivo, teclee un nombre de archivo en el cuadro Localización.
No necesita poner un .kwd al final de su nombre de archivo; KWord lo hará por usted.
Los nombres de archivos de UNIX son más flexibles que en muchos otros sistemas operativos.
Los nombres de archivos pueden:
ser de casi cualquier longitud
ser cualquier combinación de letras mayúsculas y minúsculas
incluir espacios y puntuación
Los nombres de archivos no deben:
comenzar con un espacio o punto
acabar con una extensión de archivo común (.ps, .pdf, etc).
Una vez que haya introducido la información correcta, puede hacer clic sobre Aceptar para completar el guardado.
KWord no informa del guardado con éxito.
Por tanto, si el archivo se guardó sin incidentes, KWord le devolverá a la edición de su documento.
Si hay un problema con el guardado, aparecerá un cuadro de error.
KWord sólo le permitirá guardar su archivo si los permisos del sistema operativo le dejan.
Si intenta guardar fuera de ese ámbito, KWord generará un error.
Recuperando un documento guardado
El comando Abrir se puede invocar de 4 maneras:
Haciendo clic sobre Abrir un documento guardado anteriormente y el botón Elegir cuando esté Abriendo un documento nuevo
Puede usar el atajo de teclado:
Ctrl O
o haciendo clic sobre en la barra de herramientas.
Cualquier de estos métodos hace que KWord realice la misma acción.
Usando el cuadro de diálogo
Al intentar abrir un archivo, aparece un diálogo.
Para más información sobre usar este diálogo, vea la sección titulada Usando el diálogo archivo.
Use este diálogo para localizar el documento que quiere cargar.
Una vez localizado, haga clic una vez con el left mouse button sobre el nombre del archivo.
Clic sobre Aceptar.
El archivo se cargará.
Hay un botón Cancelar.
Si hace clic sobre este botón, la carga se abortará.
Imprimiendo un documento
El comando Imprimir se puede invocar de 3 formas:
Puede usar el atajo de teclado:
Ctrl P
o hacer clic sobre en la barra de herramientas.
No importa cómo lo haga, aparecerá el diálogo de impresión de KDE.
La lista desplegable de arriba llamada Nombre: muestra la impresora seleccionada actualmente.
Para configurar esta impresora, haga clic sobre el botón Propiedades....
Para seleccionar una impresora diferente, clic sobre la lista desplegable y seleccione la impresora deseada.
Si quiere imprimir su salida a un archivo postscript, a un archivo PDF, a un módem fax (para enviarlo como un fax) o enviar por correo como un archivo PDF, seleccione la opción apropiada de la lista desplegable.
Una vez que esa opción se haya seleccionado, introduzca el nombre de archivo en el cuadro de texto llamado Archivo de salida:.
La línea llamada Sistema de impresión usado actualmente: determina cómo interactúa KWord con su computador y cualquier impresora conectada.
La configuración completa del sistema de impresión está fuera del alcance de este documento.
Para más información, vea el sitio web de impresión de KDE en http: / /printing.kde.org.
Si quiere imprimir todas las páginas de su documento y sólo una copia, puede hacer clic en el botón Aceptar.
Si quiere cambiar el número de copias o sólo imprimir las páginas seleccionadas, haga clic sobre Expandir
Dependiendo de cómo use su computador el sistema de impresión, puede encontrar que tiene más opciones de impresión que las descritas debajo.
Lo que se describe aquí está disponible en la mayoría de sistemas.
Una vez que haya hecho clic sobre Expandir, el cuadro de diálogo pasa a:
Bajo Selección de página, puede elegir entre imprimir el documento completo (seleccione Todo), imprimir la página donde está el cursor (simplemente seleccione Actual), o imprimir un rango de páginas.
Si selecciona la opción Imprimir rango, introduzca su rango de páginas en el cuadro de texto que se proporciona. (Ejemplo:
3-13)
En la lista desplegable llamada Fijar página:, puede decirle a KWord que imprima las Páginas pares, las Páginas impares o Todas las páginas.
En el cuadro de la derecha llamado Copias:, debe fijar el número de copias que quiera imprimr.
Puede o bien introducir el número directamente en la caja, o usar las flechas para ajustar el número.
KWord puede imprimir un máximo de 999 copias cada vez.
Puede hacer que KDE intercale sus documentos haciendo clic sobre el cuadro de verificación Intercalar.
Si está seleccionado, KDE imprimirá todoas las páginas para la copia 1, luego imprimirá todas las páginas para la copia 2, etc..
Si no está seleccionado, KDE imprimirá todas las copias de la primera página, luego todas las copias de la segunda página, etc.
También puede determinar si el documento se imprime en orden (imprime la primera página primero), o en orden inverso (imprime la última página primero).
Si hace clic sobre Contraer, el diálogo de impresión contraerá la vista anterior.
Cuando esté satisfecho con sus selecciones, puede hacer clic sobre Aceptar para imprimir.
Si hace clic sobre Cancelar, el diálogo de impresión se abortará.
