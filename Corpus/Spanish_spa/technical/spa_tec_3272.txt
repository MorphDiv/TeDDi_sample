Mueve los objetos cuyas id se especifican en el segundo parámetro desde la colección con id id origen hasta la colección con el id id destino.
Si el id de destino es 0, los objetos serán disociados de la colección origen.
Si esta fuese la última instancia del objeto, este sería borrado.
Si desea borrar todas las instancias de una vez, utilice hw_deleteobject().
El valor devuelto es el número de objetos movidos.
Vea también hw_cp(), hw_deleteobject().
